## Introducción: Las Últimas Semanas del Curso Importan

**Desafío de junio**: Alumnos desmotivados, problemas de comportamiento, tiempo desaprovechado en las últimas dos semanas

**Investigación** (McCombs & Pope, 1994): La reflexión estructurada mejora la retención un **35%** (repasar consolida el aprendizaje)

**Cierre tradicional**: Películas, fiestas, cero trabajo académico
```
Realidad común:
- 2 semanas: Actividades no académicas
- Pérdida de aprendizaje: Los alumnos olvidan contenido (el retroceso veraniego empieza antes)
- Problemas de conducta: Sin estructura = caos en el aula
```

**Alternativa significativa**: Reflexión + celebración + repaso sistemático
```
Estructura propuesta:
- Semana 1: Repaso de todo el aprendizaje del año (sistemático)
- Semana 2: Actividades de reflexión + creación de portafolios
- Últimos días: Celebración con conexión académica

Resultado: El aprendizaje continúa, los alumnos se marchan orgullosos de su crecimiento
```

## Portafolios de Crecimiento: Evidencia Tangible del Progreso

**Evidencia visual del trabajo de todo el año**

### Contenido del Portafolio

**Comparación septiembre-junio**:
```
Sección inicial: "Dónde Empecé"
- Muestra de escritura de septiembre (primera semana)
- Evaluación de matemáticas de septiembre (línea base)
- Coloreo de septiembre (motricidad fina inicial)

Sección final: "Dónde Estoy Ahora"
- Muestra de escritura de junio (última semana)
- Evaluación de matemáticas de junio (fin de curso)
- Proyecto artístico de junio (progreso motricidad fina)

Sección intermedia: "El Camino"
- Fotos del año
- Fichas favoritas (seleccionadas por el alumno)
- Certificados de logros
- Gráfico de crecimiento (fluidez lectora, etc.)
```

**Actividad de montaje** (Semana 1 de las últimas dos):
```
Lunes-Martes: Los alumnos recopilan muestras de trabajo de sus archivos
Miércoles: Organizan el portafolio (orden cronológico)
Jueves: Decoran la portada, escriben reflexión
Viernes: Comparten con un compañero o en pequeño grupo
```

## Actividades de Repaso (Últimas 2 Semanas)

**Repaso sistemático del contenido del año**

### Programación Semanal Lunes-Viernes

**Semana 1: Repaso de Contenidos**
```
Lunes: Repaso espiral de matemáticas (todas las destrezas del año, ficha de 30 problemas)
Martes: Sopa de letras de vocabulario (100 palabras de todo el año)
Miércoles: Crucigrama de ciencias (términos clave de todas las unidades)
Jueves: Línea temporal de sociales (eventos/personajes importantes estudiados)
Viernes: Comprensión lectora (pasajes de textos favoritos)

Formato: Fichas + juegos + debates
Objetivo: Refrescar la memoria antes del verano
```

**Semana 2: Reflexión + Celebración**
```
Lunes: Crear portafolios (organizar muestras de trabajo)
Martes: Escribir ensayos de reflexión (repaso del año)
Miércoles: Grabar mensajes en vídeo (consejos para la clase del próximo año)
Jueves: Libros de autógrafos (los compañeros firman libros de recuerdos)
Viernes: Celebración (juegos académicos, ceremonia de premios, fiesta)
```

## Ceremonia de Premios y Reconocimiento

**Celebrar a cada alumno**

### Certificados Individualizados

**Un certificado por alumno** (fortaleza personalizada):
```
Ejemplos:
"Alumno con Mayor Progreso en Matemáticas"
"Campeón de la Perseverancia" (nunca se rindió)
"Embajador de la Amabilidad" (siempre ayudando)
"Pensador Creativo" (resolución única de problemas)
"Estrella de la Lectura" (leyó más de 50 libros)

Principio: TODOS los alumnos reciben reconocimiento por ALGO
Objetivo: Todos los alumnos se marchan sintiéndose valorados
```

**Uso de generadores**: Crear 30 certificados personalizados (21 minutos = 42 seg cada uno)

## Cuadernillo de Aprendizaje Veraniego

**Prevenir el retroceso veraniego**

### Contenido del Cuadernillo

**Suministro para 6-8 semanas**:
```
Semana 1: 5 fichas de repaso (todas las asignaturas)
Semana 2: 5 fichas
...
Semana 8: 5 fichas

Total: 40 fichas (mezcla de asignaturas)

Tiempo de generación: 28 minutos (40 × 42 seg)
Tiempo de impresión: 45 minutos (1.200 páginas para 30 alumnos)

Incluir:
- Instrucciones para los padres (20 min/día, 3-5 días/semana)
- Claves de respuestas (los padres pueden corregir)
- Registro de lectura (seguimiento de lectura veraniega)
```
