## Passo 4: Personalizar a Sua Ficha

**Definir parâmetros**

### Exemplo Passo a Passo: Caça-Palavras

**Opções de personalização**:

**1. Introduzir palavras de vocabulário**:
```
Campo de entrada: Digite ou cole as palavras

Exemplo (unidade de ciências):
- fotossíntese
- clorofila
- dióxido de carbono
- oxigénio
- luz solar
- água
- energia
- glicose

Dica: Uma palavra por linha ou separadas por vírgula
```

---

**2. Escolher tamanho da grelha**:
```
Opções:
- 10×10: Fácil (alunos mais novos, palavras simples)
- 15×15: Médio (dificuldade padrão)
- 20×20: Difícil (alunos mais velhos, mais palavras)

Recomendação: Comece com 15×15 (ajuste se for muito fácil/difícil)
```

---

**3. Definir nível de dificuldade**:
```
Fácil: Palavras apenas horizontal e vertical (sem diagonais)
Médio: Horizontal, vertical, diagonal
Difícil: Todas as direções incluindo ao contrário

Escolha com base em: Ano escolar e experiência dos alunos com caça-palavras
```

---

**4. Opções adicionais**:
```
☑ Incluir chave de respostas (recomendado)
☑ Incluir lista de palavras (banco de palavras a encontrar)
☐ Esconder lista de palavras (mais desafiante)

Título: "Vocabulário da Fotossíntese"
Instruções: "Encontre as 8 palavras relacionadas com fotossíntese"
```

## Passo 5: Pré-visualizar e Ajustar

**Verificar antes de finalizar**

### Pré-visualização em Tempo Real

**O que rever**:
```
Verificação de layout:
- Palavras cabem na grelha? ✓
- Título está claro? ✓
- Instruções são legíveis? ✓
- Aparência geral é profissional? ✓

Precisa de ajustes?
- Grelha muito cheia: Aumentar tamanho (10×10 → 15×15)
- Muito fácil: Aumentar dificuldade (médio → difícil)
- Palavra em falta: Adicionar à lista, regerar

Processo iterativo: Ajustar → Pré-visualizar → Ajustar → Pré-visualizar até perfeito

Tempo: 30 segundos a 2 minutos (dependendo dos ajustes necessários)
```

## Passo 8: Imprimir com Eficiência

**Configurações de impressão são importantes**

### Configurações Ideais de Impressão

**Preferências da impressora**:
```
Qualidade: Normal (não rascunho, não foto)
- Rascunho: Muito claro (difícil de ler)
- Normal: Claro, usa quantidade apropriada de tinta
- Foto: Desnecessário (desperdiça tinta, lento)

Cor: Preto e branco (a menos que especificamente necessário)
- Maioria das fichas: P&B suficiente
- Páginas para colorir: Cor necessária
- Poupança: P&B custa 80% menos por página

Duplex (frente e verso): Quando apropriado
- Fichas de prática: Lado único (alunos escrevem)
- Textos de leitura: Duplex OK (poupa papel)
- Importante: Verificar qual lado fica "para cima" (orientação de viragem importa)

Papel: Papel de cópia padrão 75g/m²
- Evitar cartolina: Caro, desnecessário para maioria das fichas
- Evitar reciclado: Por vezes encrava, pode ser menos nítido
```

---

### Dicas para Impressão em Grande Quantidade

**Para turma de 30 alunos**:
```
Trabalho de impressão: 30 cópias da ficha

Configurações:
1. Encher completamente o tabuleiro de papel (200+ folhas)
2. Selecionar "Agrupar" (se for agrafar depois)
3. Escolher "Imprimir tudo de uma vez" (não uma de cada vez)
4. Enviar trabalho de impressão
5. Afastar-se (deixar funcionar)

Tempo: 30 cópias = ~3 minutos de impressão
Evitar: Ficar à espera junto à impressora (usar tempo para outras tarefas)
```

## Erros Comuns a Evitar

**Aprenda com os erros dos outros**

### Erro 1: Demasiados Parâmetros

**Problema**:
```
Utilizador novo: Ajusta cada opção individual (sobrecarregado pelas escolhas)
Resultado: Paralisia por análise (demora 10 minutos a gerar uma ficha)
```

**Solução**:
```
Comece simples: Use configurações padrão primeiro
Gere: Veja o resultado
Ajuste: Apenas mude o que precisa de mudar
Itere: Explore gradualmente mais opções

Primeira ficha: 2 minutos (a aprender)
Quinta ficha: 30 segundos (confiante)
```

---

### Erro 2: Nível de Dificuldade Errado

**Problema**:
```
Professor: Cria ficha "difícil" para alunos com dificuldades
Resultado: Frustração, desistência
```

**Solução**:
```
Comece mais fácil: Se tiver dúvidas, desça um nível
Teste: Dê primeiro a um aluno
Ajuste: Mais fácil ou difícil conforme resposta
Refine: Acerte a dificuldade ao longo do tempo
```

---

### Erro 3: Não Manter Ficheiros Organizados

**Problema**:
```
Descargas: Todas vão para pasta padrão "Transferências"
Meses depois: "Onde está aquele caça-palavras de fotossíntese que criei?"
Resultado: Recriar fichas desnecessariamente
```

**Solução**:
```
Imediatamente: Mover ficha descarregada para pasta apropriada
Nomeação: Usar nomes descritivos (não apenas "ficha1.pdf")
Reutilizar: Encontrar e reutilizar no próximo ano (poupar tempo)
```

## Resolver Problemas Comuns

**Soluções rápidas**

### Problema: "A ficha não gera"

**Possíveis causas**:
```
1. Demasiadas palavras para tamanho da grelha
   Solução: Reduzir número de palavras ou aumentar tamanho da grelha

2. Ligação à internet perdida
   Solução: Verificar WiFi, atualizar página

3. Caracteres inválidos na entrada
   Solução: Remover símbolos especiais, usar letras padrão
```

---

### Problema: "Qualidade de impressão é fraca"

**Soluções**:
```
1. Verificar configurações da impressora (deve ser "Normal" não "Rascunho")
2. Garantir níveis adequados de tinta (substituir cartucho se baixo)
3. Limpar cabeças de impressão (menu de manutenção da impressora)
4. Usar papel de qualidade (papel barato pode parecer desfocado)
```

---

### Problema: "Não consigo encontrar o ficheiro descarregado"

**Solução**:
```
1. Verificar pasta padrão de Transferências
2. Pesquisar no computador por PDF criado hoje
3. Navegador: Verificar histórico de descargas (mostra localização do ficheiro)
4. No futuro: Guardar imediatamente em pasta organizada
```

## Conclusão

Domine os geradores de fichas em **30 minutos** - configuração (5 min) + primeira ficha (5 min) + prática (20 min).

**Guia de geração**: Selecionar gerador → personalizar (introduzir conteúdo, tamanho grelha, dificuldade) → pré-visualizar (verificar layout) → gerar (2-5 seg) → descarregar (PDF pronto)

**Chaves de respostas**: Geração automática (checkbox durante criação), página 2 do PDF, zero tempo extra

**Avançado**: Geração em lote (5 fichas em 5 min), criação de modelos (guardar configurações personalizadas), variações sazonais (mesma competência, tema diferente)
