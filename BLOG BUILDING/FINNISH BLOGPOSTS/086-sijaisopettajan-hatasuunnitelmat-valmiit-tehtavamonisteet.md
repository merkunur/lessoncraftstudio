## Johdanto: Sijaisongelman todellisuus

**Opettajan arki**: Tarvitset sijaista lyhyellä varoitusajalla (sairastuminen, hätätilanne, täydennyskoulutus)

**Ongelma**: Sijaissuunnitelmien laatiminen kello 5.30 sairaana
```
Klo 6.00: Heräät flunssaan
Klo 6.15: Soitat sijaiseen
Klo 6.30: Yrität kirjoittaa oppituntisuunnitelmia (voit todella huonosti)
Klo 7.00: Luovutat, jätät yleisen "antakaa lukea" -lapun
Tulos: Hukattu koulupäivä (sijainen ei tiedä mitä opettaa)
```

**Ratkaisu**: Etukäteen tehdyt hätäsuunnitelmat (kun olet terve)

**Tutkimus** (Weems & Rogers, 2007): Laadukkaat sijaissuunnitelmat ylläpitävät **90 % normaalista oppimisesta** vs **40 %** huonoilla tai olemattomilla suunnitelmilla

**Paras käytäntö**: Pidä 3–5 päivää valmiita sijaissuunnitelmia aina valmiina

## Sijaisen hätäkansio

**Fyysinen kansio luokassa** (sijainen voi ottaa heti)

### Kansion sisältö

**Osio 1: Luokan tiedot**
```
- Paikkakartta (nimillä merkitty)
- Päivärytmi (kunkin aineen ajat)
- Käyttäytymisen hallintajärjestelmä (luokan säännöt, palkinnot)
- Erityistä tukea tarvitsevat oppilaat (mukautusten lista)
- Hätätilanteiden ohjeet (paloharjoitus, lukitus)
- Avuliaat oppilaat (luotettavat oppilaat, joilta sijainen voi kysyä apua)
```

**Osio 2: Päivä 1 kokonaisopetus**
```
Aamu:
- 8.30–9.00: Aamutoimet + nimenhuuto (sijainen tekee itsenäisesti)
- 9.00–9.30: Matematiikan tehtävämoniste (30 tehtävää, vastausavain mukana)
- 9.30–10.00: Sanahaku (sanaston sanoja käynnissä olevasta aiheesta)

Keskipäivä:
- 10.00–10.30: Lukeminen (itsenäinen lukeminen + ymmärtämisen tehtävämoniste)
- 10.30–11.00: Välitunti
- 11.00–11.30: Väritystehtävä (rauhoittava toiminta)

Iltapäivä:
- 11.30–12.00: Lounas
- 12.00–12.30: Luonnontieteen sanahaku (käynnissä olevan luonnontieteen aiheen termejä)
- 12.30–13.00: Ristisanatehtävä (opetussanastoa)
- 13.00–13.30: Matematiikkapulma (kriittistä ajattelua, kiinnostavaa)
- 13.30–14.00: Vapaa valinta: Viivojen piirtäminen, värittäminen tai itsenäinen lukeminen

Kotiinlähtö: klo 14.00

Huomautus sijaiselle: "Kaikkien tehtävien vastausavaimet ovat osiossa 5. Oppilaiden tulee tarkistaa omat työnsä oltuaan valmiita."
```

**Osiot 3–5: Päivät 2, 3, 4** (samanlainen rakenne, erilaiset tehtävät)

**Osio 6: Vastausavaimet** (kaikki tehtävät)

## Itsestään selvät tehtävämonisteet

**Perusperiaate**: Sijaisen ei tarvitse opettaa uutta sisältöä

**Tehtäväominaisuudet sijaissuunnitelmiin**:

### Selkeät ohjeet tehtävämonisteessa

**Esimerkki matematiikan tehtävän otsikko**:
```
Nimi: _________________ Päivämäärä: _______

OHJEET:
1. Ratkaise jokainen yhteenlaskutehtävä
2. Näytä laskutoimituksesi
3. Tarkista vastauksesi vastausavaimesta (opettajan pöydällä)
4. Jos saat 8 tai enemmän väärin, pyydä apua sijaiselta
5. Kun olet valmis, laita tehtävä palautuskoriin

Arvioitu aika: 30 minuuttia
```

**Sijaisystävällinen**: Ohjeet OVAT tehtävämonisteessa (sijaisen ei tarvitse selittää)

---

### Tutut tehtävätyypit

**Käytä tehtäviä, jotka oppilaat jo tuntevat**:
```
✓ Laskutoimitukset (harjoiteltu viikoittain - oppilaat tuntevat muodon)
✓ Sanahaut (tehty säännöllisesti - itsestään selviä)
✓ Ristikot (tuttu palapelimuoto)
✓ Värittäminen (kaikki tietävät miten väritään)

✗ Uudet pelisäännöt (sijainen ei voi opettaa tuntematonta toimintaa)
✗ Monimutkaiset projektit (vaativat opettajan tukea)
✗ Ryhmätyö (käyttäytymisen hallinnan haaste sijaiselle)
```

## Käyttäytymisen hallinta sijaisille

**Haaste**: Oppilaat testaavat rajoja sijaisen kanssa

**Ratkaisu**: Selkeät odotukset + kiinnostava työ

### Etukäteen kirjoitettu viesti oppilaille

**Sisällytä sijaissuunnitelmaan**:
```
Hyvät oppilaat,

Olen pois tänään, mutta odotan, että osoitatte sijaisopettajalle samaa kunnioitusta kuin minulle. Tässä mitä odotan:

✓ Noudattakaa kaikkia luokan sääntöjä
✓ Tehkää kaikki annetut tehtävät
✓ Auttakaa sijaista, jos hänellä on kysymyksiä
✓ Olkaa ystävällisiä ja kunnioittavia

Kysyn sijaiselta raporttia palatessani. Oppilaat, jotka täyttävät odotukset, saavat [palkinnon]. Ne, jotka eivät täytä, saavat [seurauksen].

Hyvää päivää!
[Opettajan nimi]
```

**Vastuullisuus**: Oppilaat tietävät, että opettaja seuraa asiaa jälkikäteen

---

### Kiinnostava työ (estää tylsistymisen)

**Tylsistyminen = käytösongelmia**

**Strategia**: Sekoitus helppoa + haastavaa + hauskaa
```
Aamu: Tavallinen matematiikka (tuttua, rakentaa itseluottamusta)
Aamupäivä: Sanahaku (pelin kaltaista, kiinnostavaa)
Ennen lounasta: Värittäminen (rauhoittavaa, vähän stressiä)
Lounaan jälkeen: Ristikko (kohtalainen haaste)
Päivän loppu: Matematiikkapulma TAI vapaa valinta (motivoiva lopetus)

Tulos: Oppilaat sitoutuneita (vähemmän aikaa huonoon käytökseen)
```

## Sisältöön sidotut sijaissuunnitelmat

**Ihannetilanne**: Sijaissuunnitelmat kertaavat käynnissä olevaa sisältöä (eivät ole vain aikaa vieviä)

### Sisältöön sidottujen suunnitelmien laatiminen kuukausittain

**Syyskuu (vuoden alku)**:
```
Luo sijaissuunnitelmat syyskuun sisällön perusteella:
- Matematiikka: Yhteen- ja vähennyslaskun kertaus (syyskuussa opetetut taidot)
- Sanasto: Syyskuun oikeinkirjoitussanat (sanahaussa/ristikossa)
- Luonnontiede: Syyskuun luonnontieteen termit (sanahaussa)

Tulos: Vaikka sijainen opettaa, oppilaat harjoittelevat relevanttia sisältöä
```

**Lokakuu**:
```
Päivitä sijaissuunnitelmat:
- Matematiikka: Kertolasku (lokakuun painopiste)
- Sanasto: Lokakuun sanat
- Luonnontiede: Lokakuun aiheen termit

Korvaa syyskuun sijaissuunnitelmien tehtävät lokakuun versioilla
Aika: 10 minuuttia luodaksesi 5 uutta tehtävää
```

**Jatkuva**: Päivitä sijaissuunnitelmat kuukausittain (aina ajankohtaista sisältöä)

## Eriyttäminen sijaissuunnitelmissa

**Haaste**: Sijainen ei voi eriyttää kuten varsinainen opettaja

**Ratkaisu**: Etukäteen eriytetyt materiaalit

### Porrastetut tehtävälaatikot

**Aseta etukäteen**:
```
Laatikko 1 (Sininen): Mukautetut tehtävät
- 10 tehtävää 20:n sijaan
- Kuvatuki
- Kenelle: Tukea tarvitseville oppilaille

Laatikko 2 (Vihreä): Tavalliset tehtävät
- 20 tehtävää
- Luokka-aste
- Kenelle: Tasoaan vastaaville oppilaille

Laatikko 3 (Punainen): Haastavat tehtävät
- 30 tehtävää
- Monimutkaiset sovellukset
- Kenelle: Edistyneille oppilaille

Huomautus sijaiselle: "Oppilaat tietävät mistä värilaatikosta ottaa. He valitsevat itse sopivan tason."

Tulos: Eriyttäminen tapahtuu automaattisesti (sijaisen ei tarvitse tehdä päätöksiä)
```

## Opeta oppilaat sijaisopettajapäiviin

**Valmistelu**: Opeta oppilaille miten käyttäytyä sijaisten kanssa

**Syyskuun harjoitus**:
```
Opettaja: "Joskus olen poissa ja meillä on sijainen. Tässä mitä odotan:"

Roolileikki:
- Oppilas vapaaehtoinen "sijaiseksi"
- Luokka harjoittelee:
  * Sijaisen ohjeiden noudattamista
  * Tehtävien ottamista oikeasta laatikosta
  * Avun pyytämistä sijaiselta sopivasti
  * Siivoamista päivän lopussa

Harjoituskerta: "Tänään teeskelemme, että on sijaispäivä"
Oppilaat: Noudattavat sijaisen hätäsuunnitelmaa harjoituksena
Tulos: Kun oikea sijainen tulee, oppilaat tuntevat rutiinin
```

## Yhteenveto

Laadukkaat sijaissuunnitelmat ylläpitävät **90 % normaalista oppimisesta** (Weems & Rogers, 2007) – valmistelu estää hukatut koulupäivät.

**Hätäkansio**: 3–5 täyttä päivää (luokan tiedot + päivittäiset oppitunnit + vastausavaimet + avuliaat oppilaat)

**Itsestään selvät tehtävät**: Selkeät ohjeet TEHTÄVÄMONISTEESSA (sijaisen ei tarvitse opettaa)

**Tutut toiminnot**: Laskutoimitukset, sanahaut, ristikot, värittäminen (oppilaat tuntevat muodon)

**Käyttäytymisen hallinta**: Etukäteen kirjoitettu viesti (odotukset + vastuullisuus), kiinnostava työ (estää tylsistymisen)

**Sisältöön sidottu**: Päivitä kuukausittain (aina vastaa käynnissä olevaa opetussuunnitelmaa, ei vain aikaa vieviä)

**Generaattoritehokkuus**: 3 päivää luotu 45 minuutissa (15 luominen + 30 tulostus)

**Digitaalinen varmuuskopio**: Google Classroom hätäkansio (nolla varoitusajan poissaolo katettu)

**Eriyttäminen**: Etukäteen porrastetut laatikot (sininen/vihreä/punainen, oppilaat valitsevat itse tason)

**Pitkäaikainen poissaolo**: 10 päivää 2–3 tunnissa (pitkä loma katettu)

**Sijaisen palaute**: Lomake raportointiin (käytös, suorittaminen, haasteet)

**Oppilaiden harjoitus**: Syyskuun roolileikki (harjoittele sijaispäivän rutiinia)

**Tutkimus**: Laadukkaat suunnitelmat = 90 % oppimista vs 40 % ilman (Weems & Rogers, 2007)
