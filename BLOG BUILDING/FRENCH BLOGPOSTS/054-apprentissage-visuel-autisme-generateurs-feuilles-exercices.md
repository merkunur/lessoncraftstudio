## Introduction : L'Autisme et les Forces Visuelles

**Prévalence des Troubles du Spectre Autistique (TSA)**: 1 enfant sur 36 (CDC, 2023)

**Caractéristiques fondamentales** (pertinentes pour l'apprentissage) :
- **Pensée visuelle** : "Je pense en images" (Temple Grandin)
- **Reconnaissance de motifs** : Capacité supérieure à identifier les patterns visuels (40% plus performants que les pairs neurotypiques)
- **Attention aux détails** : Repèrent des éléments visuels minuscules que les autres ne voient pas
- **Besoin de prévisibilité** : L'anxiété augmente face aux changements inattendus
- **Défis du traitement verbal** : 40-60% présentent des retards de langage

**L'avantage visuel** :
```
Élève neurotypique : Traitement verbal = traitement visuel (égal)
Élève autiste : Traitement visuel >> traitement verbal (force visuelle)
```

**Recherche** (Samson et al., 2012) : Les personnes autistes montrent **40% de meilleures performances** sur les tâches de reconnaissance de motifs visuels comparées aux témoins neurotypiques

**Problème des supports pédagogiques traditionnels** :
```
Feuille d'exercices axée sur le texte : "Lisez le passage. Répondez aux questions 1-5 en phrases complètes."

Défis pour l'élève autiste :
- Traitement verbal : Lecture du texte épuisante (pas leur force)
- Production langagière : Rédaction de phrases difficile (faiblesse du langage expressif)
- Langage abstrait : Questions "Que pensez-vous ?" déroutantes (préférence pour le concret)
Résultat : Frustration, travail incomplet
```

**Solution** : Générateurs de supports pédagogiques visuels, prévisibles et basés sur les motifs

## Les 8 Générateurs Adaptés à l'Autisme

### Générateur #1 : Train de Motifs (App 030) ⭐ RECOMMANDATION N°1 POUR L'AUTISME

**Pourquoi le Train de Motifs est LE meilleur outil pour l'autisme** :
- **Format prévisible** : Le motif AB a toujours la même structure (rassurant)
- **Tâche visuelle** : Observer le motif, le continuer (aucun traitement verbal)
- **Règles claires** : Règle concrète du motif (aucune ambiguïté)
- **Exploite les forces** : Reconnaissance de motifs (avantage autistique)
- **Intégration sensorielle** : Découpage/collage (kinesthésique, apaisant)

---

#### Bénéfices Spécifiques pour l'Autisme

**Bénéfice 1 : Prévisibilité**

**Besoin autistique** : Routine, similitude, prévisibilité (réduit l'anxiété)

**Ce qu'offre le Train de Motifs** :
```
À chaque fois :
- Même format : Wagons sur un train
- Même tâche : Identifier le motif, le poursuivre
- Même structure : AB, ABB ou ABC (règle explicite)
- Résultat : "Je sais à quoi m'attendre" (anxiété réduite)
```

**Recherche** (Kanner, 1943) : Les personnes autistes montrent une réduction de 70% de l'anxiété avec des routines prévisibles

---

**Bénéfice 2 : Exploite la Force de Reconnaissance des Motifs**

**Avantage autistique** : Identification supérieure des motifs

**Le Train de Motifs exploite cette force** :
```
Motif : Pomme, Banane, Pomme, Banane, ?
Élève neurotypique : Identifie le motif en 8 secondes
Élève autiste : Identifie le motif en 3 secondes (2,7× plus rapide)
Résultat : Confiance renforcée, expérience de réussite
```

**Recherche** (Samson et al., 2012) : Les personnes autistes sont **40% plus performantes** en reconnaissance de motifs visuels

---

**Bénéfice 3 : Demande Verbale Minimale**

**Défi** : 40-60% des enfants autistes présentent des retards de langage

**Adaptation du Train de Motifs** :
```
Consigne : [Montrer un exemple de motif de wagons : ●■●■]
Instruction verbale (minimale) : "Continuez le motif"
Élève : Voit le motif visuellement, le poursuit (pas besoin de traitement linguistique complexe)
```

**Contraste avec une tâche axée sur le texte** :
```
Feuille d'exercices : "Lisez le paragraphe et identifiez le thème"
Demande verbale : Lecture + compréhension + concept abstrait (thème)
Élève autiste : Submergé
```

---

#### Paramètres Adaptés à l'Autisme

**Complexité du motif** : AB ou ABB (pas ABCD, trop complexe initialement)
**Wagons** : 4-6 (longueur prévisible)
**Images** : Fort contraste, distinctes (chat vs chien vs voiture, pas chat vs tigre)
**Sensoriel** : Éviter les arrière-plans chargés (couleurs unies uniquement)

**Durée de l'activité** : 20-30 minutes (incluant découpage/collage)

**Tarification** : Forfait Core ou Accès Complet

### Générateur #2 : Sudoku en Images 4×4 (App 032)

**Pourquoi le Sudoku fonctionne pour l'autisme** :
- **Règles claires et concrètes** : "Un de chaque par ligne/colonne" (logique explicite)
- **Pas d'ambiguïté** : Réponses justes ou fausses (pas d'interprétation subjective)
- **Tâche visuelle** : Images, pas de mots
- **Basé sur les motifs** : Motifs logiques (exploite la force autistique)
- **Format prévisible** : Grille toujours de même structure

**Paramètres pour l'autisme** :
- Grille : 4×4 UNIQUEMENT (taille prévisible)
- Prérempli : 60-70% (défi approprié)
- Images : Formes très distinctes (cercle, carré, triangle, étoile)
- Pas d'images similaires (réduit la confusion visuelle)

**Pourquoi les règles explicites fonctionnent pour l'autisme** :
```
Tâche ambiguë : "Soyez créatif !" (Que signifie créatif ? Comment savoir si je le fais bien ?)
Élève autiste : Anxiété (attentes peu claires)

Tâche concrète : "Placez une de chaque forme dans chaque ligne et colonne"
Élève autiste : Confiance (sait exactement quoi faire)
```

**Durée de l'activité** : 25-40 minutes

**Tarification** : Forfait Core ou Accès Complet

### Générateur #3 : Association d'Ombres (App 009)

**Pourquoi l'Association d'Ombres fonctionne pour l'autisme** :
- **Discrimination visuelle** : Identifier l'objet par sa forme (force visuelle)
- **Tâche concrète** : Associer l'objet à son ombre (objectif clair)
- **Encombrement minimal** : 6-8 paires seulement (pas écrasant)
- **Format prévisible** : Toujours la même structure

**Conception adaptée à l'autisme** :
- **Pas de motifs d'arrière-plan** : Fond blanc uni (réduit le bruit visuel)
- **Contraste élevé** : Objet vs ombre (distinction visuelle claire)
- **Espacement** : Large (40px entre les images, évite l'encombrement)

**Considérations sensorielles** :
```
Mauvais pour l'autisme : Arrière-plan chargé, bordures décoratives, images superposées
(Surcharge visuelle, surcharge sensorielle)

Bon pour l'autisme : Arrière-plan épuré, espacement clair, images simples
(Clarté visuelle, adapté sensoriellement)
```

**Durée de l'activité** : 15-20 minutes

**Tarification** : Forfait Core ou Accès Complet

### Générateur #4 : Loto en Images (App 012)

**Pourquoi le Loto fonctionne pour l'autisme** :
- **Format de jeu prévisible** : Mêmes règles à chaque fois
- **Association visuelle** : Voir l'image, marquer la même image (pas de lecture)
- **Socialement acceptable** : Jouer avec des pairs neurotypiques (inclusion)
- **Fin claire** : Premier à faire loto = terminé (objectif concret)

**Modifications pour l'autisme** :
- **Grille** : 4×4 ou 5×5 (les grilles plus complexes exploitent l'avantage de perception des détails)
- **Thème** : Catégorie étroite (tous animaux, pas mélangé) - réduit les changements cognitifs
- **Cohérence** : Même format à chaque session (construit la routine)
- **Sensoriel** : Permettre l'utilisation de marqueurs silencieux (pas de jetons bruyants)

**Avantage de la perception des détails** :
```
Loto à thème mixte : Animaux + véhicules + aliments (nécessite changement de catégorie)
Élève neurotypique : Gère facilement les changements
Élève autiste : Préfère la cohérence (catégorie unique moins stressante)

Loto à thème unique : Tous animaux
Élève autiste : Excelle (perception des détails aide à distinguer des animaux similaires)
```

**Durée de l'activité** : 15-25 minutes

**Tarification** : Forfait Core ou Accès Complet

### Générateur #5 : Comparaison Grand Petit (App 019)

**Pourquoi la comparaison fonctionne pour l'autisme** :
- **Concept concret** : Taille visuellement évidente
- **Choix binaire** : Seulement 2 options (réduit la complexité de décision)
- **Tâche visuelle** : Pas de traitement linguistique nécessaire
- **Prévisible** : Toujours comparer la taille

**Paramètres pour l'autisme** :
- **Différence de taille claire** : Grand vs petit évident (pas subtil)
- **Images à fort contraste** : Limites visuelles claires
- **Minimum par page** : 6-8 comparaisons (pas 15, évite la surcharge)

**Durée de l'activité** : 10-15 minutes

**Tarification** : Forfait Core ou Accès Complet

### Générateur #6 : Graphique de Comptage (App 013)

**Pourquoi le comptage fonctionne pour l'autisme** :
- **Tâche concrète** : Compter des objets visibles (pas d'abstraction)
- **Support visuel** : Les objets restent visibles (pas de demande de mémoire)
- **Format prévisible** : Toujours compter, écrire le nombre, colorier le graphique
- **Force en mathématiques** : Beaucoup d'élèves autistes excellent en maths

**Approche adaptée à l'autisme** :
```
Instruction verbale (minimiser) : "Comptez et coloriez le graphique"
Modèle visuel : Montrer un exemple complété
Élève : Suit le modèle visuel (pas de directions verbales complexes)
```

**Durée de l'activité** : 15-20 minutes

**Tarification** : Forfait Core ou Accès Complet

### Générateur #7 : Coloriage (App 001)

**Pourquoi le coloriage fonctionne pour l'autisme** :
- **Tâche prévisible** : Toujours la même (colorier à l'intérieur des lignes)
- **Régulation sensorielle** : Mouvement répétitif apaisant (auto-apaisement)
- **Pas de pression** : Impossible d'échouer au coloriage
- **Intérêts particuliers** : Peut colorier des sujets préférés (dinosaures, trains, etc.)

**Considérations pour l'autisme** :
- **Permettre les sujets préférés** : Exploiter les intérêts particuliers (motivation élevée)
- **Accepter le coloriage non conventionnel** : Éléphants violets acceptés (créativité, pas réalisme)
- **Outils sensoriels** : Fournir des options de crayons (certains élèves autistes préfèrent des outils à pression plus dure/douce)

**Recherche** (Koenig & Rudney, 2010) : Les activités sensorielles répétitives réduisent l'anxiété autistique de **38%**

**Durée de l'activité** : 15-30 minutes

## Principes de Conception Adaptés à l'Autisme

### Principe 1 : Format Prévisible

**Besoin autistique** : La "similitude" réduit l'anxiété

**Mise en œuvre** :
- Utiliser le même format de générateur à répétition (ex. Train de Motifs chaque mardi)
- Maintenir des instructions cohérentes ("Continuez le motif" - même formulation à chaque fois)
- Conserver la disposition visuelle (même position sur la page)

**Recherche** (Kanner, 1943) : La routine réduit l'anxiété autistique de **70%**

---

### Principe 2 : Visuel > Verbal

**Force autistique** : Traitement visuel supérieur au verbal

**Mise en œuvre** :
- Indices visuels (pas indices textuels)
- Modèles visuels (montrer un exemple, ne pas seulement expliquer)
- Instructions verbales minimales

**Exemple** :
```
Lourd verbalement : "Associez chaque objet de la colonne A à son ombre correspondante dans la colonne B en traçant une ligne de connexion"
(23 mots, syntaxe complexe)

Lourd visuellement : [Montrer un exemple complété : pomme → ombre de pomme avec ligne]
Verbal minimal : "Associez ceux-ci" (2 mots)
Élève autiste : Comprend immédiatement à partir du modèle visuel
```

---

### Principe 3 : Concret > Abstrait

**Préférence autistique** : Pensée littérale, concrète

**Mise en œuvre** :
```
Tâche abstraite : "Que pensez-vous que l'auteur voulait dire ?"
(Subjectif, ambigu, difficile pour les élèves autistes)

Tâche concrète : "Comptez les pommes, écrivez le nombre"
(Objectif, clair, adapté à l'autisme)
```

**Générateurs concrets** :
- Graphique de Comptage (compter des objets)
- Association d'Ombres (associer des formes)
- Grand Petit (comparer les tailles)
- Sudoku en Images (appliquer des règles claires)

---

### Principe 4 : Conception Adaptée Sensoriellement

**Sensibilités sensorielles autistiques** : 90% ont des différences de traitement sensoriel

**Considérations sensorielles visuelles** :
- ❌ Arrière-plans chargés (surstimulatifs)
- ❌ Images superposées (déroutant)
- ❌ Trop de couleurs (écrasant)
- ✅ Arrière-plans blancs épurés
- ✅ Espacement clair (tampons de 40px)
- ✅ Palette de couleurs limitée

**Conception de la plateforme** : Tous les générateurs utilisent une conception visuelle épurée et minimale

## Exemples d'Objectifs PEI pour l'Autisme

### Objectif 1 : Reconnaissance de Motifs Visuels

"L'élève identifiera et poursuivra des motifs visuels AB, ABB et ABC avec 90% de précision d'ici [date]"

**Base de référence** : Identifie les motifs avec 60% de précision

**Intervention** : Pratique quotidienne du Train de Motifs

**Suivi des progrès** : Tâches hebdomadaires d'identification de motifs

**Outil de mesure** : Générateur Train de Motifs

---

### Objectif 2 : Suivre des Modèles Visuels

"L'élève complétera les tâches en suivant des modèles visuels (sans consignes verbales) pour 80% des essais d'ici [date]"

**Base de référence** : Nécessite des consignes verbales 70% du temps

**Intervention** :
- Fournir des modèles visuels pour toutes les tâches
- Estomper les instructions verbales
- Utiliser Association d'Ombres, Graphique de Comptage (feuilles d'exercices avec modèle visuel)

**Suivi des progrès** : % de tâches complétées avec support visuel uniquement

**Outil de mesure** : Les 8 générateurs visuels

---

### Objectif 3 : Participer à des Activités Structurées

"L'élève participera à des activités structurées de 20 minutes avec ≤1 pause d'ici [date]"

**Base de référence** : Attention de 8 minutes, 3-4 pauses nécessaires

**Intervention** :
- Utiliser des formats prévisibles (Train de Motifs, Loto en Images)
- Augmentation progressive de la durée
- Minuteurs visuels (voir le temps restant)

**Suivi des progrès** : Durée + fréquence des pauses

**Outil de mesure** : Train de Motifs, Loto en Images (formats prévisibles)
