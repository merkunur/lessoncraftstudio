## Introduction : La Réalité des Classes Hétérogènes

**Exemple typique d'une classe de CE2** :

```
Niveaux de lecture :
- 5 élèves : niveau CE1 (un an de retard)
- 12 élèves : niveau CE2 (niveau attendu)
- 8 élèves : niveau CM1-CM2 (un à deux ans d'avance)
- 5 élèves : allophones (niveaux variés)

Niveaux en mathématiques :
- 6 élèves : consolidation des acquis CE1
- 15 élèves : travail sur les compétences CE2
- 9 élèves : prêts pour les défis CM1

Résultat : 30 élèves avec 5+ niveaux d'apprentissage différents
```

**Recherches** (Tomlinson & Imbeau, 2010) : La différenciation améliore les résultats de **tous les élèves** (pas seulement ceux en difficulté ou les élèves avancés)

**Problème traditionnel** : L'enseignement uniforme
```
L'enseignant distribue : La même fiche à tous les 30 élèves
Résultat :
- Élèves en difficulté : Trop difficile (frustration, abandon)
- Élèves de niveau attendu : Adapté (travail productif)
- Élèves avancés : Trop facile (ennui, problèmes de comportement)

Efficacité : 33 % (atteint seulement le tiers médian de la classe)
```

**Solution par différenciation** : Plusieurs versions, mêmes objectifs
```
L'enseignant distribue : Trois versions (adaptée, standard, enrichie)
Résultat :
- En difficulté : Défi approprié (réussite, confiance)
- Niveau attendu : Travail au niveau de la classe (maîtrise)
- Avancés : Activités d'enrichissement (engagement, progression)

Efficacité : 100 % (atteint tous les élèves à leur niveau)
```

## Système de Fiches à Niveaux Différenciés

**Stratégie centrale** : Même concept, complexité variable

### Modèle à Trois Niveaux

**Exemple : Pratique de la multiplication**

**Niveau 1 (Adapté)** - Pour les élèves en difficulté :
```
Nombre de problèmes : 10 (au lieu de 20)
Tables : Seulement 0 à 5 (2 × 3, 4 × 5, etc.)
Aide : Images en disposition rectangulaire fournies
Format : Police plus grande, plus d'espace

Exemples de problèmes :
1. 2 × 3 = ___ [image de 2 rangées × 3 colonnes]
2. 3 × 4 = ___ [image de 3 rangées × 4 colonnes]

Objectif : Développer la confiance, maîtriser les tables de base
```

**Niveau 2 (Standard)** - Pour les élèves de niveau attendu :
```
Nombre de problèmes : 20
Tables : 0 à 10 (toutes les multiplications de base)
Aide : Aucune (pratique autonome)
Format : Espacement standard

Exemples de problèmes :
1. 6 × 7 = ___
2. 8 × 9 = ___

Objectif : Pratiquer les compétences du niveau de classe
```

**Niveau 3 (Enrichi)** - Pour les élèves avancés :
```
Nombre de problèmes : 15
Portée : Nombre à deux chiffres × nombre à un chiffre (12 × 7, 23 × 4)
Défi : Problèmes en contexte inclus
Format : Opérations mixtes (× et ÷)

Exemples de problèmes :
1. 15 × 6 = ___
2. Si une boîte contient 24 crayons, combien de crayons dans 8 boîtes ?
3. 72 ÷ 8 = ___ (vérification : ___ × 8 = 72)

Objectif : Dépasser le niveau de classe, appliquer aux situations concrètes
```

**Principe clé** : Les trois niveaux travaillent la multiplication (même compétence), mais à différents degrés de complexité

## Stratégies de Groupements Flexibles

**Les groupes changent selon la compétence/le sujet** (pas de groupes fixes par niveau)

### Rotation Basée sur les Compétences

**Lundi : Évaluation diagnostique** (tous font le même test de 10 problèmes)
```
Résultats :
- 8 élèves : 0 à 4 corrects (besoin de réenseignement)
- 15 élèves : 5 à 7 corrects (besoin de pratique)
- 7 élèves : 8 à 10 corrects (prêts pour l'enrichissement)
```

**Mardi-Jeudi : Groupes différenciés**
```
Groupe A (Réenseignement) :
- Enseignement en petit groupe avec l'enseignant
- Matériel de manipulation fourni
- Fiches progressives (Niveau 1)
Durée : 30 minutes avec l'enseignant

Groupe B (Pratique) :
- Travail autonome (fiches Niveau 2)
- Vérification entre pairs avec corrigés
- L'enseignant circule pour les questions
Durée : 30 minutes en autonomie

Groupe C (Enrichissement) :
- Activités d'enrichissement (fiches Niveau 3)
- Problèmes défis, énigmes
- Option : Enseigner au Groupe A plus tard
Durée : 30 minutes en autonomie

Vendredi : Post-évaluation (mesurer les progrès)
```

**Semaine suivante** : Nouvelle compétence, NOUVEAUX groupes (les élèves sont regroupés selon la nouvelle évaluation)

---

### Groupements Basés sur les Intérêts

**Toute différenciation n'est pas par niveau** - parfois par intérêt

**Exemple : Pratique du vocabulaire** (mêmes 20 mots, formats différents)
```
Choix 1 : Mots mêlés (apprenants visuels, faible stress)
Choix 2 : Mots croisés (apprenants verbaux, défi modéré)
Choix 3 : Phrases (rédacteurs créatifs, application)
Choix 4 : Illustrations (apprenants artistiques, mémoire visuelle)

Tous les élèves : Apprennent les mêmes 20 mots de vocabulaire
Différenciation : Le format correspond à l'intérêt/force
Résultat : Engagement accru, même résultat d'apprentissage
```

## Tâches Parallèles

**Différents chemins vers le même objectif d'apprentissage**

### Exemple en Mathématiques : Les Fractions

**Objectif d'apprentissage** : Comprendre que les fractions représentent des parties d'un tout

**Tâche A** (Concret) :
```
Matériel : Fractions manipulables (pièces physiques)
Activité : « Utilise les cercles de fractions pour montrer 1/2, 1/4, 1/8 »
Fiche : Dessine ce que tu as construit, nomme les fractions
Niveau : Pour les élèves ayant besoin d'expérience concrète
```

**Tâche B** (Visuel) :
```
Matériel : Fiche avec des formes
Activité : « Colorie 1/2 de ce rectangle, 3/4 de ce cercle »
Fiche : Plusieurs formes à colorier selon les fractions
Niveau : Pour les élèves prêts pour la représentation visuelle
```

**Tâche C** (Abstrait) :
```
Matériel : Fiche avec droite graduée
Activité : « Place 1/4, 1/2, 3/4 sur la droite graduée »
Fiche : Droite graduée + fractions équivalentes
Niveau : Pour les élèves prêts pour la pensée abstraite
```

**Évaluation commune** : Tous les élèves complètent le même ticket de sortie (représenter 1/2 d'une manière quelconque)

## Stations d'Apprentissage Autonome

**Les élèves progressent dans les séquences à leur propre rythme**

### Configuration des Stations

**Exemple : Unité de géométrie** (4 stations, 2 semaines)

**Station 1 : Les Formes** (Jours 1-3)
```
Tâche : Identifier les formes 2D (cercle, carré, triangle, rectangle, pentagone, hexagone)
Fiches : 3 niveaux disponibles
- Niveau 1 : Associer les formes aux noms (support visuel)
- Niveau 2 : Identifier les formes dans les images (monde réel)
- Niveau 3 : Classifier les formes par propriétés (côtés, angles)

Suivi des progrès : L'élève coche quand complété
Passage : Quand réussite à 80 % sur l'auto-vérification, passer à la Station 2
```

**Station 2 : Propriétés** (Jours 4-6)
```
Tâche : Apprendre les propriétés des formes (côtés, sommets, angles)
Fiches : 3 niveaux
- Niveau 1 : Compter les côtés et sommets (base)
- Niveau 2 : Comparer les formes (plus/moins de côtés)
- Niveau 3 : Créer des formes avec propriétés spécifiques

Passage : Réussir l'auto-vérification (80 %+) → Station 3
```

**Station 3 : Aire et Périmètre** (Jours 7-9)
```
Tâche : Calculer les mesures
Fiches : 3 niveaux
- Niveau 1 : Compter les unités pour l'aire (quadrillage fourni)
- Niveau 2 : Utiliser les formules (A = L × l, P = 2L + 2l)
- Niveau 3 : Problèmes en contexte (application)

Passage : Réussir l'auto-vérification → Station 4
```

**Station 4 : Application Réelle** (Jours 10-12)
```
Tâche : Appliquer la géométrie à des problèmes authentiques
Fiches : 3 niveaux
- Niveau 1 : Concevoir un jardin (dimensions données)
- Niveau 2 : Concevoir une chambre (choisir les dimensions)
- Niveau 3 : Maximiser l'espace (problème d'optimisation)

Achèvement : Les 4 stations = Unité complétée
```

**Différenciation intégrée** :
- Rythme : Les élèves rapides finissent en 8 jours, les plus lents prennent 12 jours
- Niveau : Les élèves choisissent la difficulté appropriée à chaque station
- Soutien : L'enseignant travaille avec de petits groupes à la Station 1 (fondamentale)

## Compactage pour les Élèves Avancés

**Éliminer la pratique redondante** pour les élèves qui ont déjà maîtrisé le contenu

### Protocole de Pré-évaluation

**Lundi : Pré-test** (contenu de toute l'unité)
```
Unité : Addition à deux chiffres avec retenue (20 problèmes)

Résultats :
- La plupart des élèves : 0 à 10 corrects (besoin d'enseignement)
- 4 élèves : 18 à 20 corrects (déjà maîtrisé)

Décision : Compacter pour ces 4 élèves avancés
```

**Mardi-Vendredi : Différencié**
```
Classe principale (26 élèves) :
- Reçoit l'enseignement sur l'addition à deux chiffres
- Complète les fiches de pratique (programme standard)

Groupe compacté (4 élèves) :
- Saute l'enseignement de base (déjà maîtrisé)
- Travaille sur l'enrichissement : Addition à trois chiffres + problèmes
- Complète des fiches défis (Niveau 3, générées en 42 secondes)

Résultat :
- Classe principale : Maîtrise la compétence du niveau
- Groupe compacté : Va au-delà du niveau (prévient l'ennui)
```

## Stratégie RAFT (Produits Différenciés)

**Les élèves démontrent leur apprentissage dans différents formats**

### Composantes RAFT

**R**ôle : Qui es-tu ?
**A**uditoire : À qui t'adresses-tu ?
**F**ormat : Quelle forme cela prendra-t-il ?
**T**hème : Quel est le sujet ?

**Exemple : Unité sur le système solaire**

**Option 1** (Verbal/Linguistique) :
```
Rôle : Astronaute
Auditoire : Citoyens terriens
Format : Journal de bord
Thème : Décrire une visite sur Mars
Fiche : Modèle de journal (questions guides fournies)
```

**Option 2** (Visuel/Spatial) :
```
Rôle : Scientifique de la NASA
Auditoire : Élèves
Format : Schéma annoté
Thème : Expliquer la taille des planètes par rapport à la Terre
Fiche : Modèle de schéma avec étiquettes à compléter
```

**Option 3** (Logique/Mathématique) :
```
Rôle : Ingénieur spatial
Auditoire : Contrôle de mission
Format : Tableau de données
Thème : Calculer les temps de voyage vers chaque planète
Fiche : Modèle de tableau avec espace de calcul
```

**Option 4** (Interpersonnel) :
```
Rôle : Guide touristique
Auditoire : Touristes spatiaux
Format : Brochure
Thème : Créer une brochure « Visitez Saturne »
Fiche : Modèle de brochure (plan triptyque)
```

**Toutes les options** : Abordent le même contenu (connaissance du système solaire)
**Différenciation** : Le format correspond à la force/intérêt de l'élève

## Activités Relais pour les Élèves Rapides

**Travail significatif quand les élèves terminent tôt** (pas du travail supplémentaire inutile)

### Menu d'Activités Relais

**Toujours disponibles** (les élèves connaissent la routine) :
```
✓ Choix 1 : Énigmes mathématiques (Sudoku, grilles logiques, mots croisés mathématiques)
✓ Choix 2 : Lecture (lecture autonome + fiche de lecture)
✓ Choix 3 : Vocabulaire (mots mêlés avec mots défis)
✓ Choix 4 : Fiches d'enrichissement (problèmes défis optionnels)
✓ Choix 5 : Aider un camarade (tutorat entre pairs, quand approprié)

Règles :
- Terminer d'abord le travail assigné
- Choisir dans le menu relais (pas de question « Qu'est-ce que je fais ? »)
- Travailler calmement et de façon autonome
- Toutes les activités relais sont significatives (pas des récompenses, mais un apprentissage précieux)

Matériel généré : 50+ fiches relais par matière, prêtes à imprimer
Temps de création : 50 × 42 sec = 35 minutes pour l'approvisionnement de l'année
```

## Différenciation par Niveau de Préparation

**Ajuster selon où sont les élèves MAINTENANT**

### Niveaux selon la Taxonomie de Bloom

**Même contenu, différents niveaux de pensée**

**Thème : Histoire « La Toile de Charlotte »**

**Se rappeler/Comprendre** (Niveau 1) :
```
Questions de la fiche :
1. Qui sont les personnages principaux ?
2. Où se déroule l'histoire ?
3. Quel problème Wilbur rencontre-t-il ?
4. Comment Charlotte aide-t-elle Wilbur ?

Niveau cognitif : Rappel et compréhension
Pour : Élèves développant encore la compréhension de base
```

**Appliquer/Analyser** (Niveau 2) :
```
Questions de la fiche :
1. Compare Wilbur au début et à la fin de l'histoire
2. Pourquoi Charlotte a-t-elle choisi les mots qu'elle a tissés ?
3. Que se passerait-il si Fern n'avait jamais visité la grange ?
4. Comment l'amitié change-t-elle les personnages ?

Niveau cognitif : Application et analyse
Pour : Élèves prêts pour une réflexion plus profonde
```

**Évaluer/Créer** (Niveau 3) :
```
Questions de la fiche :
1. Évalue : Le plan de Charlotte était-il la meilleure solution ? Qu'est-ce qui aurait pu sauver Wilbur autrement ?
2. Crée : Écris une fin alternative où Wilbur aide Charlotte
3. Juge : Quel personnage a montré le plus de croissance ? Défends ta réponse avec des preuves
4. Conçois : Si cette histoire continuait, quels défis les personnages affronteraient-ils ?

Niveau cognitif : Évaluation et création
Pour : Élèves prêts pour le plus haut niveau de pensée
```

## Systèmes de Gestion pour la Différenciation

**L'organisation est la clé d'une différenciation réussie**

### Bacs Codés par Couleur

**Système visuel** (les élèves se servent eux-mêmes) :
```
Station de mathématiques :
- Bac bleu : Adapté (moins de problèmes, plus de soutien)
- Bac vert : Standard (niveau de la classe)
- Bac rouge : Défi (enrichissement)

Les élèves savent : Prendre la fiche de leur niveau
L'enseignant sait : Quels élèves ont besoin de quel niveau (observation au fil du temps)
Ajustement : Les élèves peuvent changer de niveau en progressant (pas fixe)

Temps de préparation : 10 minutes pour imprimer et trier dans les bacs (hebdomadaire)
```
