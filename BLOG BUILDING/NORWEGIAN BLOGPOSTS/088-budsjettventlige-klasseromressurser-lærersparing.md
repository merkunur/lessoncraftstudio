## Innledning: Lærernes budsjettutfordring

**Virkeligheten for norske lærere**: Mange lærere bruker **egne penger på klasseromsutstyr** hvert år

**Typisk budsjettfordeling**:
```
Vanlig årlig forbruk:
- Arbeidsbøker og forbruksmateriell: kr 3.000
- Undervisningsmateriell: kr 1.500
- Klasseromspynt: kr 1.000
- Kontorrekvisita: kr 2.000
Total: kr 7.500 av egen lomme
```

**Problemet**: Arbeidsbøker kan kun brukes én gang
```
Kjøp: 30 mattebøker × kr 100 = kr 3.000
Bruk: August-juni (elevene skriver i bøkene)
Juni: Bøkene følger med elevene hjem (kan ikke gjenbrukes)
Neste år: Kjøp 30 nye arbeidsbøker (kr 3.000 igjen)
5-års kostnad: kr 15.000 på forbruksbøker
```

**Løsningen**: Digital generering + utskrift = gjenbrukbare ressurser
```
Kostnad: kr 1.440/år abonnement
Generer: Ubegrensede arbeidsark (tilsvarer 100+ arbeidsbøker)
Gjenbruk: Lagre PDF-filer, skriv ut igjen neste år (null tilleggskostnad)
5-års kostnad: kr 7.200 (spar kr 7.800 sammenlignet med arbeidsbøker)
```

## Strategier for gjenbrukbare materialer

**Reduser papirforbruk**

### Tørr-viskelommer

**Engangs investering**: kr 500 for 30 lommer
```
Slik fungerer det:
1. Skriv ut arbeidsark (én kopi)
2. Sett inn i tørr-viskelomme
3. Eleven løser med tusj for tavle
4. Visk ut, gjenbruk 100+ ganger

Eksempel:
Morgenøvelse i matte (daglig 5-oppgave ark):
- Uten lommer: Skriv ut 30 kopier × 180 dager = 5.400 sider
- Med lommer: Skriv ut 30 kopier én gang, gjenbruk hele året = 30 sider
Papir spart: 5.370 sider (kr 2.685 til kr 0,50/side)

ROI: kr 500 investering sparer kr 2.685/år (5,4× avkastning)
```

---

### Laminerte arbeidsark

**For hyppig brukte aktiviteter**:
```
Laminer: Gangetabeller, alfabettavler, tallinje
Kostnad: kr 200 laminator + kr 100 lamineringsark = kr 300
Holdbarhet: Varer 5+ år

Bruk: Elevene bruker med tavletusj
Eksempel: "Strek over bokstavene" arbeidsark
- Skriv ut én gang, laminer, bruk 200+ ganger over år
- Kostnad per bruk: kr 0,50 ÷ 200 = kr 0,0025 (praktisk talt gratis)
```

## Papirbesparende strategier

**Reduser utskriftskostnader**

### Tosidig utskrift

**Enkel endring, stor besparelse**:
```
Ensidig: 180 arbeidsark = 180 sider
Tosidig (duplex): 180 arbeidsark = 90 sider
Besparelse: 90 sider per elev = 2.700 sider/klasse = kr 1.350/år

Innstillinger: Skriverpreferanser → Tosidig utskrift → "Kort kant" (vend)
```

---

### Visning + notatbok

**Eliminer fotokopiering**:
```
Tradisjonelt: Skriv ut 30 kopier av arbeidsark
Alternativ: Vis ETT arbeidsark på skjerm, elevene skriver av til notatbok

Eksempel:
Matteoppgaver: Vis 20 oppgaver på prosjektor
Elever: Skriv oppgaver + svar i mattebok
Papir brukt: 0 ark

Når passende: Regnestykker, ordforrådsdefinisjoner, forståelsesspørsmål
Når ikke passende: Komplekse diagrammer, lange tekster
```
