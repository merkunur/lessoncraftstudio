## Introduction : L'écriture comme outil de pensée

**Vision traditionnelle** : L'écriture = matière enseignée en français uniquement

**Compréhension actuelle** : L'écriture = outil de réflexion pour TOUTES les disciplines

**Recherche** (Graham & Perin, 2007 - Writing Next) : Écrire sur les contenus disciplinaires **améliore la compréhension** de 0,30 à 0,50 taille d'effet (équivalent à 12-19 points de percentile)

**Pourquoi l'écriture améliore l'apprentissage** :
```
L'élève résout un problème de maths : 3/4 + 2/4 = 5/4
Enseignant : "Maintenant EXPLIQUE par écrit comment tu as fait"
Réponse de l'élève : "J'ai additionné les numérateurs (3+2=5) et gardé le même dénominateur (4) parce que..."

Processus cognitif activé :
Étape 1 : Résoudre le problème (procédural)
Étape 2 : Expliquer sa démarche (métacognitif)
Étape 3 : Organiser l'explication (traitement linguistique)

Résultat : Compréhension approfondie (expliquer = vraiment comprendre)
```

**Principe fondamental** : Écrire pour APPRENDRE (pas seulement pour montrer qu'on a appris)

## Applications en mathématiques

**Objectif** : Les élèves explicitent leur raisonnement mathématique par écrit

### Stratégie 1 : "Montre ton travail" enrichi

**Approche classique** : Montrer uniquement les étapes numériques
```
47 + 28 = ?
  47
+ 28
----
  75

Travail montré : Calculs seuls (procédural)
```

**Approche avec écriture intégrée** : Expliciter la démarche en phrases
```
Problème : 47 + 28 = ?

Mes calculs :
  47
+ 28
----
  75

Mon explication :
D'abord, j'ai additionné les unités : 7 + 8 = 15.
J'ai écrit le 5 et retenu le 1.
Ensuite j'ai additionné les dizaines : 4 + 2 = 6, plus la retenue 1 = 7.
Donc le résultat est 75.

Bénéfice : L'élève doit COMPRENDRE (pas juste mémoriser la procédure)
```

**Modification des fiches générées** : Ajouter une ligne "Explique ton raisonnement" sous chaque exercice

---

### Stratégie 2 : Explicitation des problèmes à énoncé

**Après résolution d'un problème** :
```
Énoncé : "Sarah possède 3/4 d'une pizza. Elle en donne 1/4 à son amie. Quelle portion de pizza reste-t-il à Sarah ?"

Solution de l'élève : 3/4 - 1/4 = 2/4 = 1/2

Explication rédigée :
Sarah avait 3 parts sur 4. Elle en a donné 1 part. Il lui reste donc 2 parts sur 4, ce qui équivaut à la moitié (1/2).

Compétences développées :
- Calcul mathématique (soustraction)
- Équivalence de fractions (2/4 = 1/2)
- Communication écrite (expliciter la démarche)
- Compréhension de texte (interpréter l'énoncé)
```

---

### Stratégie 3 : Création d'énoncés par les élèves

**Activité sur fiche** :
```
Consigne : Tu as appris à additionner des fractions cette semaine. INVENTE maintenant un problème qui utilise l'addition de fractions.

Exemple d'élève :
"Tom a couru 2/5 de kilomètre lundi et 3/5 de kilomètre mardi. Quelle distance totale Tom a-t-il parcourue ?"

Solution : 2/5 + 3/5 = 5/5 = 1 kilomètre

Exigence cognitive : Rétro-ingénierie (créer le problème, pas seulement le résoudre)
Compétence rédactionnelle : Création narrative, contextualisation mathématique
```

## Applications en sciences

**Objectif** : Expliquer des concepts scientifiques, documenter des observations

### Stratégie 1 : Vocabulaire scientifique en contexte

**Approche classique** : Définir le vocabulaire
```
Photosynthèse = processus par lequel les plantes fabriquent leur nourriture grâce à la lumière
(Définition apprise par cœur, compréhension superficielle)
```

**Approche avec écriture intégrée** : Utiliser le vocabulaire dans une explication
```
Consigne : "Explique comment une plante obtient son énergie. Utilise ces mots : photosynthèse, chlorophylle, lumière solaire, dioxyde de carbone, oxygène."

Réponse de l'élève :
Les plantes fabriquent leur propre nourriture grâce à la photosynthèse. La chlorophylle présente dans les feuilles capte la lumière solaire. La plante absorbe le dioxyde de carbone de l'air et l'eau du sol. Elle utilise l'énergie de la lumière pour transformer le dioxyde de carbone et l'eau en glucose (nourriture). Lors de ce processus, la plante rejette de l'oxygène dans l'atmosphère.

Mots de vocabulaire : Les 5 utilisés correctement en contexte
Compréhension : L'élève maîtrise le PROCESSUS (pas seulement les définitions)
```

**Application avec générateurs** : Créer des mots mêlés avec termes scientifiques, puis exercice d'écriture utilisant les mots trouvés

---

### Stratégie 2 : Rédaction de comptes rendus d'expérience

**Fiche compte rendu simplifié** :
```
Expérience : _______________________________

QUESTION : Que cherchons-nous à découvrir ?
[L'élève formule la question de recherche]

HYPOTHÈSE : Que penses-tu qu'il va se passer ?
[Prédiction de l'élève avec justification]

PROTOCOLE : Qu'avons-nous fait ?
1. _______________
2. _______________
3. _______________

OBSERVATIONS : Qu'as-tu vu/mesuré ?
[L'élève enregistre les données]

CONCLUSION : Qu'as-tu appris ?
[L'élève interprète les résultats, fait le lien avec l'hypothèse]

Compétences scientifiques : Démarche expérimentale
Compétences rédactionnelles : Écriture procédurale, rapport de données, analyse
```

### Stratégie 3 : "Explique-le à un enfant de 5 ans"

**Défi** : Expliquer un concept complexe de façon simple

**Consigne sur fiche** :
```
Cette semaine nous avons étudié le cycle de l'eau. Explique le cycle de l'eau à un enfant de maternelle (mots simples, sans vocabulaire technique).

Exemple d'élève :
L'eau des lacs et des océans monte dans le ciel (ça s'appelle évaporation). En haut dans les nuages, l'eau devient froide et se transforme en petites gouttes (condensation). Quand il y a trop de gouttes, elles tombent comme de la pluie (précipitation). La pluie retombe dans les lacs et les océans, et tout recommence !

Compétences rédactionnelles :
- Adaptation au public (maternelle = langage simple)
- Reformulation du vocabulaire (évaporation expliqué entre parenthèses)
- Organisation logique (ordre du cycle clair)
- Vérification de la compréhension (on ne peut simplifier que si on comprend vraiment)
```

## Applications en histoire-géographie et sciences humaines

**Objectif** : Relier les événements historiques à la compréhension personnelle

### Stratégie 1 : Écriture selon une perspective historique

**Après avoir étudié un événement historique** :
```
Thème : La Révolution française

Consigne : "Tu es un enfant parisien en 1789. Écris une page de journal intime sur ce que tu vis."

Exemple d'élève :
14 juillet 1789
Cher journal,
Aujourd'hui, les gens ont pris la Bastille. J'ai entendu beaucoup de bruit dans les rues. Maman m'a dit de rester à l'intérieur. Papa dit que tout va changer, que le roi ne pourra plus faire ce qu'il veut. J'ai peur mais aussi j'ai de l'espoir. Les gens disent qu'on sera plus libres maintenant.

Contenu historique : Prise de la Bastille, climat social, espoir de changement
Compétences rédactionnelles : Adoption d'un point de vue, format journal, expression émotionnelle
Empathie : Connexion personnelle à l'expérience historique
```

---

### Stratégie 2 : Comparaison passé/présent

**Mots croisés avec vocabulaire historique/civique, puis exercice de rédaction** :

```
Après avoir complété les mots croisés (démocratie, droits, vote, liberté, citoyen) :

Consigne : "Compare le droit de vote en 1800 et aujourd'hui. Qu'est-ce qui est similaire ? Qu'est-ce qui est différent ?"

Réponse d'élève :
En 1800, seuls les hommes riches pouvaient voter. Aujourd'hui, tous les citoyens de 18 ans et plus peuvent voter, hommes et femmes. Dans les deux cas, les gens considèrent le vote comme un droit important. La différence, c'est qu'on a élargi qui peut voter. Ça montre que la démocratie peut s'améliorer avec le temps.

Pensée historique : Changement dans le temps, cause/effet
Compétences rédactionnelles : Structure comparaison/contraste, raisonnement argumenté
```

---

### Stratégie 3 : Analyse écrite de cartes

**Fiche** : Fournir une carte, demander une analyse écrite

```
Carte : Carte de l'Empire romain à son apogée

Consigne : "Que te révèle cette carte sur la puissance romaine ? Utilise des preuves spécifiques de la carte."

Réponse d'élève :
Cette carte montre que l'Empire romain était très puissant. Je peux le dire car leur territoire s'étendait sur trois continents : l'Europe, l'Afrique et l'Asie. Ils contrôlaient toute la mer Méditerranée, ce qui leur donnait le contrôle des routes commerciales. La capitale, Rome, est au centre, ce qui facilitait le gouvernement d'un si grand empire. La taille de leur territoire prouve qu'ils avaient une armée forte.

Pensée historique : Analyse fondée sur des preuves, raisonnement géographique
Compétences rédactionnelles : Structure affirmation + preuves, lecture cartographique
```

## Développement du vocabulaire disciplinaire par l'écriture

**Principe** : Écrire avec le nouveau vocabulaire pour se l'approprier

### Exercices progressifs avec vocabulaire

**Niveau 1 (Base)** : Utiliser le vocabulaire dans une phrase
```
Vocabulaire : photosynthèse, écosystème, habitat

Consigne : Écris une phrase pour chaque mot.
Élève : "Les plantes utilisent la photosynthèse pour fabriquer leur nourriture."

Niveau de compétence : Usage basique, vérification de la compréhension
```

**Niveau 2 (Intermédiaire)** : Utiliser plusieurs mots dans des phrases liées
```
Consigne : Écris un paragraphe utilisant les trois mots de vocabulaire.

Réponse d'élève :
Les plantes utilisent la photosynthèse pour fabriquer leur propre nourriture. C'est important pour l'écosystème car d'autres animaux mangent les plantes pour obtenir de l'énergie. Chaque être vivant a besoin d'un habitat où il peut trouver nourriture et abri.

Niveau de compétence : Relier les concepts, structure de paragraphe
```

**Niveau 3 (Avancé)** : Enseigner le concept en utilisant le vocabulaire
```
Consigne : Explique comment l'énergie circule dans un écosystème. Utilise tous les mots de vocabulaire ET deux autres termes appris cette semaine.

Réponse d'élève :
L'énergie circule dans un écosystème selon une chaîne alimentaire. Les plantes utilisent la photosynthèse pour capter l'énergie du soleil. Les herbivores vivent dans des habitats où ils trouvent des plantes à manger, récupérant ainsi l'énergie stockée dans les plantes. Ensuite, les carnivores mangent les herbivores, transmettant l'énergie vers le haut de la chaîne. Chaque organisme dans l'écosystème dépend des autres pour l'énergie. Cette interconnexion montre l'importance de chaque habitat.

Niveau de compétence : Synthèse, enseignement aux autres, intégration multiple du vocabulaire
```

## Questions "ticket de sortie" écrites

**Écriture de fin de séance** (5 minutes) :

**Séance de maths** : Fractions
```
Ticket de sortie : "Explique en 2-3 phrases ce que tu as appris sur les fractions aujourd'hui."

Réponse d'élève : "J'ai appris que les fractions représentent des parties d'un tout. Le numérateur indique combien de parts j'ai, et le dénominateur indique combien de parts il y a en tout. Maintenant je comprends que 2/4 c'est pareil que 1/2."

Évaluation : L'enseignant lit rapidement (30 secondes par élève), identifie qui a compris vs qui est confus
```

**Séance de sciences** : États de la matière
```
Ticket de sortie : "Nomme les trois états de la matière. Donne un exemple de chacun."

Réponse d'élève : "Les trois états sont solide, liquide et gaz. Un solide c'est la glace, un liquide c'est l'eau, et un gaz c'est la vapeur. C'est toujours la même chose (H2O) mais sous différentes formes."

Évaluation : Vérifie la compréhension de base + application
```

## Grilles d'évaluation pour l'écriture disciplinaire

**Grille simple sur 3 points** :

```
Compréhension du contenu :
3 = Concept expliqué avec précision et détails
2 = Concept expliqué avec erreurs mineures
1 = Compréhension non démontrée

Qualité de l'écriture :
3 = Phrases complètes, organisées, peu d'erreurs
2 = Phrases majoritairement complètes, organisation partielle
1 = Phrases incomplètes, difficile à suivre

Usage du vocabulaire :
3 = Vocabulaire requis utilisé correctement en contexte
2 = Une partie du vocabulaire utilisée, erreurs mineures
1 = Vocabulaire non utilisé ou utilisé incorrectement

TOTAL : ___/9 points
```

**Correction rapide** : 30 secondes par élève (3 vérifications rapides)
