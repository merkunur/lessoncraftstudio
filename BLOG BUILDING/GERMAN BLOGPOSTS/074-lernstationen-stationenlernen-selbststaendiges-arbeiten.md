## Einleitung: Die Herausforderung der Kleingruppen-Förderung

**Effektive Förderung erfordert**: Lehrkraft arbeitet mit Kleingruppen (5-7 Kinder), während andere selbstständig arbeiten

**Das Problem**: Was machen die anderen 20 Kinder, während die Lehrkraft beschäftigt ist?

**Klassische Situation** (funktioniert nicht):
```
Lehrkraft: "Ich arbeite jetzt mit der roten Gruppe. Alle anderen lesen leise."
Realität nach 5 Minuten:
- 3 Kinder lesen tatsächlich
- 5 Kinder unterhalten sich
- 4 Kinder spitzen Stifte
- 8 Kinder fragen "Darf ich auf die Toilette?"
Ergebnis: Lehrkraft wird ständig unterbrochen, Kleingruppen-Förderung unmöglich
```

**Die Lösung**: Lernstationen (Stationenlernen mit Rotation)

**Forschungsgrundlage** (Bohl, 2009): Strukturierte Lernstationen ermöglichen **45-60 Minuten ungestörte Kleingruppen-Förderung** (gegenüber 10-15 Minuten ohne klare Struktur)

## Das 6-Stationen-Rotations-System

**Aufbau**: 6 Stationen, Wechsel alle 20 Minuten, Lehrkraft betreut Station 1

### Stationenübersicht

**Station 1: Lehrkraft-geleitete Fördergruppe**
```
Inhalt: Lese-Förderung, gezielte Mathematik-Förderung, Intervention
Gruppengröße: 5-7 Kinder
Rolle der Lehrkraft: Aktive Instruktion (neue Kompetenz einführen)
Dauer: 20 Minuten
```

**Station 2: Selbstständige Arbeitsblatt-Übung**
```
Inhalt: Mathematik-Arbeitsblatt, Leseverständnis, Wiederholung
Gruppengröße: 5-7 Kinder
Rolle der Lehrkraft: Keine (Kinder arbeiten eigenständig)
Material: Vorbereitete Arbeitsblätter, Lösungsblätter zur Selbstkontrolle
Dauer: 20 Minuten
```

**Station 3: Partner-Arbeit**
```
Inhalt: Rechen-Bingo, Wörtersuche-Wettbewerb, gemeinsames Kreuzworträtsel
Gruppengröße: 5-7 Kinder (2-3 Paare)
Rolle der Lehrkraft: Keine (Kinder arbeiten zusammen)
Material: Bild-Bingo-Karten, Wörtersuche-Rätsel, Stoppuhr
Dauer: 20 Minuten
```

**Station 4: Feinmotorik / Kreativ-Station**
```
Inhalt: Ausmalbilder, Muster-Zug (schneiden/kleben), Linien nachspuren
Gruppengröße: 5-7 Kinder
Rolle der Lehrkraft: Keine (beruhigende, selbstständige Tätigkeit)
Material: Ausmalbilder, Scheren, Klebestifte, Buntstifte
Dauer: 20 Minuten
```

**Station 5: Digital-Station / Computer**
```
Inhalt: Computer-Lernspiele, Tipp-Training, digitales Lesen
Gruppengröße: 5-7 Kinder (wenn 6 Computer vorhanden)
Rolle der Lehrkraft: Keine (digitale Übung)
Material: Computer/Tablets mit Lernsoftware
Dauer: 20 Minuten
```

**Station 6: Hör-Station / Lese-Ecke**
```
Inhalt: Hörbuch anhören, freies Lesen mit Lesekontrolle
Gruppengröße: 5-7 Kinder
Rolle der Lehrkraft: Keine (selbstständiges Hören/Lesen)
Material: Hörbücher, Kopfhörer, Lesetagebücher
Dauer: 20 Minuten
```

## Tagesablauf der Rotation

**Klassengröße**: 30 Kinder, aufgeteilt in 6 Gruppen à 5 Kinder

**Montag-Stationen** (120 Minuten gesamt):

**Durchgang 1** (9:00-9:20):
- Gruppe A: Station 1 (Lehrkraft)
- Gruppe B: Station 2 (Arbeitsblätter)
- Gruppe C: Station 3 (Partner)
- Gruppe D: Station 4 (Kreativ)
- Gruppe E: Station 5 (Computer)
- Gruppe F: Station 6 (Lesen)

**Durchgang 2** (9:20-9:40):
- Gruppe F: Station 1 (Lehrkraft) ← Wechsel
- Gruppe A: Station 2
- Gruppe B: Station 3
- Gruppe C: Station 4
- Gruppe D: Station 5
- Gruppe E: Station 6

**Durchgang 3** (9:40-10:00):
- Gruppe E: Station 1 (Lehrkraft)
- (Alle Gruppen wechseln weiter)

**Durchgang 4** (10:00-10:20):
- Gruppe D: Station 1 (Lehrkraft)

**Durchgang 5** (10:20-10:40):
- Gruppe C: Station 1 (Lehrkraft)

**Durchgang 6** (10:40-11:00):
- Gruppe B: Station 1 (Lehrkraft)

**Ergebnis**: Alle 6 Gruppen erhalten 20 Minuten Förderung bei der Lehrkraft, alle Kinder arbeiten durchgängig 120 Minuten konzentriert

## Station 2: Selbstständige Arbeitsblatt-Station (Generator-Einsatz)

**Zweck**: Sinnvolle eigenständige Übung (keine Beschäftigungstherapie)

**Wochenbedarf**: 30 Arbeitsblätter (6 Gruppen × 5 Tage)

**Montag - Station 2 Inhalt**:
```
Schwerpunkt: Mathematik-Wiederholung
Arbeitsblatt: 20 Additions-Aufgaben (Klassenstufe)
Selbstkontrolle: Lösungsblatt verfügbar (Kinder kontrollieren selbst)
Dauer: 20 Minuten (Kinder schaffen ~15 Aufgaben)
```

**Dienstag - Station 2 Inhalt**:
```
Schwerpunkt: Wortschatz-Übung
Arbeitsblatt: Wörtersuche (aktuelle Wochenwörter)
Selbstkontrolle: Alle 15 Wörter finden
Dauer: 20 Minuten
```

**Mittwoch - Station 2 Inhalt**:
```
Schwerpunkt: Logisches Denken
Arbeitsblatt: Kreuzworträtsel (Fachwortschatz)
Selbstkontrolle: Lösungsblatt verfügbar
Dauer: 20 Minuten
```

**Donnerstag - Station 2 Inhalt**:
```
Schwerpunkt: Logik/Problemlösung
Arbeitsblatt: Mathematik-Rätsel (2-3 Unbekannte)
Selbstkontrolle: Lösung vorhanden
Dauer: 20 Minuten
```

**Freitag - Station 2 Inhalt**:
```
Schwerpunkt: Wiederholung/Lernkontrolle
Arbeitsblatt: Gemischte Wiederholung (alle Wochenthemen)
Abgabe: Lehrkraft korrigiert (formative Beurteilung)
Dauer: 20 Minuten
```

**Generator-Workflow**:
```
Sonntag-Vorbereitung (15 Minuten):
- 5 Mathe-Arbeitsblätter erstellen: 3,5 Min
- 5 Wörtersuche-Rätsel erstellen: 3,5 Min
- 5 Kreuzworträtsel erstellen: 3,5 Min
- 5 Mathe-Rätsel erstellen: 3,5 Min
- Alles ausdrucken (je 30 Kopien): Drucker erledigt das

Gesamtzeit: 15 Minuten für komplettes Wochenmaterial Station 2
```

## Station 3: Partner-Arbeit-Station

**Zweck**: Soziales Lernen, gegenseitige Unterstützung, Motivation

**Montag bis Freitag Material**:

**Montag**: Bild-Bingo (Rechnen)
```
Material: 15 Bingo-Karten (erstellt in 1 Minute)
Ablauf: Paare spielen zusammen, wechseln sich beim Aufrufen ab
Dauer: 20 Minuten (2-3 Spiele)
```

**Dienstag**: Wörtersuche-Wettbewerb
```
Material: Gleiche Wörtersuche wie Station 2
Wettbewerb: Wer findet alle Wörter zuerst?
Dauer: 20 Minuten
```

**Mittwoch**: Kreuzworträtsel gemeinsam
```
Material: Gleiches Kreuzworträtsel wie Station 2
Zusammenarbeit: Partner arbeiten zusammen (Hinweise besprechen)
Dauer: 20 Minuten
```

**Donnerstag**: Sudoku-Partner
```
Material: 15 Sudoku-Rätsel (4×4 oder 6×6)
Zusammenarbeit: Gemeinsam lösen
Dauer: 20 Minuten
```

**Freitag**: Geheimschrift-Herausforderung
```
Material: 15 Kryptogramm-Arbeitsblätter
Wettbewerb: Welches Paar ist zuerst fertig?
Dauer: 20 Minuten
```

## Station 4: Kreativ-/Feinmotorik-Station

**Zweck**: Beruhigende Tätigkeit, Feinmotorik-Training, Entspannung

**Wochenmaterial**:

**Montag-Dienstag**: Ausmalbilder
```
Erstellen: 15 Ausmalbilder (mathematische Kunst, Jahreszeiten-Themen)
Erstellzeit: 3,5 Minuten
Dauer: 20 Minuten (Kinder malen in eigenem Tempo)
```

**Mittwoch**: Muster-Zug (schneiden/kleben)
```
Erstellen: 15 Muster-Arbeitsblätter
Material: Scheren, Klebestifte
Dauer: 20 Minuten (Muster ausschneiden, in Reihenfolge kleben)
```

**Donnerstag-Freitag**: Linien nachspuren (Labyrinthe)
```
Erstellen: 15 Labyrinth-Arbeitsblätter (verschiedene Schwierigkeiten)
Dauer: 20 Minuten (2-3 Labyrinthe lösen)
```

**Vorteil**: Stressfreie Station (alle haben Erfolg, stärkt Selbstvertrauen)

## Stationen-Organisations-Systeme

**Herausforderung**: Kinder müssen wissen WOHIN sie gehen, WAS sie tun

**Lösung**: Visuelles Organisations-System

### Visuelle Stations-Tafel

**Aufbau** (an Klassenzimmerwand):
```
Stationenrotations-Tafel

Durchgang 1 (9:00-9:20):
☐ Gruppe A → Station 1 (Lehrkraft-Tisch)
☐ Gruppe B → Station 2 (Blauer Tisch - Arbeitsblätter)
☐ Gruppe C → Station 3 (Roter Tisch - Partner-Arbeit)
☐ Gruppe D → Station 4 (Grüner Tisch - Kreativ)
☐ Gruppe E → Station 5 (Computer)
☐ Gruppe F → Station 6 (Lese-Ecke)

[Gong um 9:20 - im Uhrzeigersinn wechseln]
```

**Aus Kindersicht**:
```
Kind in Gruppe B:
- Schaut auf Tafel: "Ich bin an Station 2 (Blauer Tisch)"
- Geht zum Blauen Tisch, sieht Arbeitsblatt + Lösungsblatt
- Bearbeitet Arbeitsblatt selbstständig
- Gong erklingt, wechselt zu Station 3 (Roter Tisch)
- Schaut auf Tafel, sieht Partner-Arbeit Anweisung
- Arbeitet mit anderem Gruppe-B-Kind zusammen, spielt Bingo
```

**Vorteil für Lehrkraft**: Keine Fragen wohin (System ist visuell + vorhersehbar)

### Material-Organisation der Stationen

**Physischer Aufbau**:

**Blauer Tisch (Station 2 - Selbstständige Arbeitsblätter)**:
```
Plastikbox beschriftet "Station 2 - Montag":
- 30 Arbeitsblätter (bereits sortiert)
- 6 Lösungsblätter
- Bleistifte
- "Anleitung"-Karte: "Arbeitsblatt bearbeiten, Antworten prüfen, abgeben"
```

**Roter Tisch (Station 3 - Partner-Arbeit)**:
```
Plastikbox beschriftet "Station 3 - Montag":
- 15 Bingo-Karten
- 1 Aufruf-Blatt
- Stoppuhr
- "Anleitung"-Karte: "Bingo mit Partner spielen, abwechselnd aufrufen"
```

**Grüner Tisch (Station 4 - Kreativ)**:
```
Plastikbox beschriftet "Station 4 - Montag":
- 30 Ausmalbilder
- Buntstifte, Farbstifte
- "Anleitung"-Karte: "Bilder ausmalen, leise arbeiten"
```

**Ergebnis**: Kinder organisieren sich selbst (Lehrkraft nicht für Vorbereitung nötig)

## Verbindlichkeits-Systeme

**Herausforderung**: Woher weiß man, dass Kinder tatsächlich gearbeitet haben?

**Lösungen**:

### Strategie 1: Abgabe-Box

**Station 2** (Arbeitsblätter):
```
Anforderung: Kinder müssen fertige Arbeitsblätter abgeben
Lehrkraft: Prüft schnell auf Vollständigkeit (5 Min nach Stationen)
Rückmeldung: "Prima Arbeit!" ODER "Bitte Aufgaben 15-20 noch fertig machen"
```

---

### Strategie 2: Stations-Laufzettel

**Jedes Kind hat Laufzettel**:
```
Name: _____________

Montag Stations-Laufzettel:
☐ Station 1: Mit Lehrkraft gearbeitet (Lehrkraft-Kürzel: ____)
☐ Station 2: Arbeitsblatt erledigt #___
☐ Station 3: Bingo gespielt mit Partner: _______
☐ Station 4: __ Seiten ausgemalt
☐ Station 5: Tipp-Lektion abgeschlossen #___
☐ Station 6: ___ Minuten gelesen

Eltern-Unterschrift (Freitag): ___________
```

**Vorteil**: Verbindlichkeit für Kinder + Elternkommunikation

---

### Strategie 3: Selbsteinschätzung

**Freitag-Reflexion**:
```
Diese Woche an den Stationen:
Meine beste Arbeit war an Station ___: weil __________
Meine Anstrengung diese Woche war: ☐ Sehr gut ☐ Gut ☐ Verbesserungsbedarf
Nächste Woche werde ich: _________________________

Lehrkraft-Rückmeldung: _________________________
```

**Ergebnis**: Metakognition (Kinder denken über eigenes Lernen nach)

## Differenzierung innerhalb der Stationen

**Herausforderung**: Station 2 hat Kinder auf verschiedenen Niveaus

**Lösung**: Gestufte Materialien

### Station 2 Gestuftes Arbeitsblatt-System

**Drei Boxen am Blauen Tisch**:

**Box 1: "Herausforderung"** (über Klassenstufe)
```
Inhalt: 25 Aufgaben, komplex
Beschriftung: "Für Kinder, die eine Herausforderung möchten"
Selbstwahl: Leistungsstarke Kinder wählen dies
```

**Box 2: "Standard"** (Klassenstufe)
```
Inhalt: 20 Aufgaben, klassenstufen-angemessen
Beschriftung: "Standard-Übung"
Meiste Kinder: Wählen dies
```

**Box 3: "Unterstützung"** (unter Klassenstufe)
```
Inhalt: 15 Aufgaben, mit Hilfestellung (Bild-Modus)
Beschriftung: "Extra Unterstützung"
Kinder mit Schwierigkeiten: Wählen dies ODER Lehrkraft weist zu
```

**Generator-Workflow**:
```
3 Versionen erstellen (2 Minuten gesamt):
Version 1: Zahlenbereich 1-100, 25 Aufgaben (42 Sek)
Version 2: Zahlenbereich 1-50, 20 Aufgaben (42 Sek)
Version 3: Zahlenbereich 1-20, Bild-Modus, 15 Aufgaben (42 Sek)

Ergebnis: Sofortige Differenzierung, Kinder wählen selbst
```

## Lautstärke-Management

**Herausforderung**: 30 Kinder an 6 Stationen = potentielles Chaos

**Lösungen**:

### Lautstärke-Ampel

**An jeder Station ausgehängt**:
```
Station 1 (Lehrkraft): Stufe 0 (still - Lehrkraft spricht)
Station 2 (Arbeitsblätter): Stufe 1 (Flüstern zu sich selbst okay)
Station 3 (Partner): Stufe 2 (leises Gespräch mit Partner)
Station 4 (Kreativ): Stufe 1 (stilles Ausmalen)
Station 5 (Computer): Stufe 1 (Flüstern wenn Hilfe nötig)
Station 6 (Lesen): Stufe 0 (stilles Lesen)
```

**Durchsetzung**:
```
Wenn Station zu laut wird:
Warnung 1: Lehrkraft zeigt Finger mit Stations-Nummer (visuelles Signal)
Warnung 2: Diese Station verliert 2 Minuten bei nächster Rotation
Ergebnis: Selbstregulation (Kinder achten auf eigene Lautstärke)
```

## Zusammenfassung
