## Introduktion: Begränsningarna med traditionella prov

**Traditionell bedömning**:
```
Format: Flervalsfrågor + ifyll-i-lucka-prov
Vad det mäter: Igenkänning och återgivning
Vad det missar: Tillämpning, kreativitet, problemlösning, kommunikation

Exempel:
Fråga: "Vad är 8 × 7?"
Elev: Skriver "56"
Bevis: Eleven kan denna tabell

MEN: Provet visar inte om eleven förstår VAD multiplikation betyder eller KAN TILLÄMPA det i verkliga situationer
```

**Forskning** (Wiggins, 1990): Autentisk bedömning förutsäger bättre **verklig prestation** än traditionella prov

**Alternativ bedömning**:
```
Format: Prestationsuppgift, projekt, demonstration
Vad det mäter: Tillämpning, syntes, problemlösning
Vad det avslöjar: Djup förståelse + färdigheter

Exempel:
Uppgift: "Designa en trädgård som är 8 meter × 7 meter. Beräkna arean. Rita i skala. Planera vad du ska plantera."
Elev: Tillämpar multiplikation, använder rumsligt tänkande, fattar beslut, kommunicerar visuellt

Bevis: Eleven FÖRSTÅR multiplikation som areaberäkning OCH kan tillämpa i autentiskt sammanhang
```

**Nyckelprincip**: Bedömning ska spegla hur kunskap används i verkligheten

## Prestationsuppgifter

**Elever visar lärande genom ATT GÖRA**

### Matteprestationsuppgift

**Kursplan**: Mäta area och omkrets (årskurs 4)

**Traditionellt prov**:
```
Fråga 1: En rektangel är 6 cm × 4 cm. Vad är arean?
Fråga 2: Vad är omkretsen?

Mäter: Kan eleven beräkna med formler
Saknar: Förståelse för vad area/omkrets betyder i verkliga livet
```

**Prestationsuppgift som alternativ**:
```
Uppgift: "Designa ditt drömrum"

Instruktioner:
1. Välj mått (rummet måste vara 10-15 meter × 10-15 meter)
2. Beräkna area (visa uträkning)
3. Beräkna omkrets (visa uträkning)
4. Rita rummet i skala (1 cm = 2 meter)
5. Lägg till möbler med mått (säng, skrivbord, byrå)
6. Förklara: "Får möblerna plats? Hur mycket golvyta blir kvar?"

Bedömningskriterier:
- Korrekt areaberäkning (2 poäng)
- Korrekt omkrets (2 poäng)
- Skalenlig ritning (3 poäng)
- Realistiska möbelmått (2 poäng)
- Utrymmesresonemang (3 poäng)
Totalt: 12 poäng

Vad det avslöjar:
- Kan tillämpa formler korrekt (beräkning)
- Förstår rumsligt tänkande (möbelplacering)
- Kan kommunicera matematiskt tänkande (förklaring)
- Gör verkliga kopplingar (praktisk tillämpning)
```

**Arbetsbladsdel**: Planeringsmall (genererad på 42 sekunder)
```
Rumsplaneringsblad

Mina rumsmått: ____ meter × ____ meter

Areaberäkning:
____ × ____ = ____ kvadratmeter

Omkrets:
(____ + ____) × 2 = ____ meter

Möbellista:
Föremål: _______ Storlek: ____ × ____
Föremål: _______ Storlek: ____ × ____
Föremål: _______ Storlek: ____ × ____

Får allt plats? Förklara:
_________________________________________________
```

### NO-prestationsuppgift

**Kursplan**: Förstå växters behov (årskurs 3)

**Traditionellt prov**:
```
Flervalsfråga:
"Vad behöver växter för att växa?"
a) Vatten och jord
b) Vatten och solljus
c) Vatten, solljus och näring
d) Bara vatten

Mäter: Kan eleven återge fakta
Saknar: Förståelse för VARFÖR växter behöver detta
```

**Prestationsuppgift som alternativ**:
```
Uppgift: "Växtexperiment"

Vecka 1: Sätt upp experiment
- Plantera 3 bönfrön i koppar
- Kopp A: Vatten + solljus + jord
- Kopp B: Vatten + mörker + jord
- Kopp C: Inget vatten + solljus + jord

Vecka 2-4: Observera och dokumentera (mätarbetsblad)
Datum | Kopp A höjd | Kopp B höjd | Kopp C höjd | Anteckningar
...

Vecka 4: Analysarbetsblad
1. Vilken växt växte högst? Varför?
2. Vad hände med Kopp B (inget ljus)? Varför?
3. Vad hände med Kopp C (inget vatten)? Varför?
4. Dra slutsatser: Vad behöver växter? Bevisa med din data.

Bedömning:
- Korrekta mätningar dokumenterade (3 poäng)
- Detaljerade observationer (2 poäng)
- Slutsatser stödda av data (4 poäng)
- Tydligt naturvetenskapligt resonemang (3 poäng)
Totalt: 12 poäng

Vad det avslöjar:
- Kan genomföra experiment (naturvetenskaplig process)
- Kan samla data korrekt (mätfärdigheter)
- Kan analysera resultat (kritiskt tänkande)
- Förstår orsak-verkan (konceptuell kunskap)
```

## Portfoliobedömning

**Samla arbetsprover över tid för att visa utveckling**

### Läsportfolio

**Innehåll** (September - Maj):
```
Del 1: Läsloggar (spåra lästa böcker)
- September: 2 böcker
- Januari: 5 böcker
- Maj: 8 böcker
Synlig utveckling: Ökad läsvolym

Del 2: Förståelseprov
- September: Enkel sammanfattning av bilderbok
- Januari: Karaktärsanalys av kapitelbok
- Maj: Jämförande uppsats om två romaner
Synlig utveckling: Ökad komplexitet

Del 3: Ordförråd
- September: 20 synord behärskade
- Januari: 50 synord + 30 ämnesord
- Maj: 100+ ord, kan använda i sammanhang
Synlig utveckling: Utökat ordförråd

Del 4: Läsflyt (inspelningar)
- September: 40 ord/min, hackigt
- Januari: 65 ord/min, flytande
- Maj: 90 ord/min, uttrycksfullt
Synlig utveckling: Förbättrat läsflyt

Årsreflektion (eleven fyller i arbetsblad):
"När jag jämför mitt septemberarbete med majarbete har jag utvecklats genom..."
"Jag är mest stolt över..."
"Nästa år vill jag förbättra..."
```

**Bedömningsvärde**: Portfolio visar UTVECKLING (inte bara nuvarande nivå)

### Matteportfolio

**Utvecklingsdokumentation**:
```
Post 1 (September): Enkel addition (20 uppgifter, 80% korrekt, 15 minuter)
Post 2 (November): Tvåsiffrig addition (20 uppgifter, 90% korrekt, 12 minuter)
Post 3 (Januari): Multiplikation intro (10 uppgifter, 60% korrekt, 20 minuter)
Post 4 (Mars): Multiplikationsträning (20 uppgifter, 85% korrekt, 10 minuter)
Post 5 (Maj): Flerstegsordproblem (10 uppgifter, 90% korrekt, 15 minuter)

Portfolio omslag (elevifylld):
Mattefärdigheter jag behärskar i år:
☑ Ensiffrig addition
☑ Tvåsiffrig addition med växling
☑ Multiplikationstabell 0-10
☑ Ordproblemslösning
☐ Division (tränar nästa år!)

Bevis: "Titta på Post 1 och Post 5. I september kunde jag bara göra enkel addition. Nu kan jag lösa komplexa ordproblem med flera steg!"

Föräldragranskning: Föräldrar ser konkreta bevis på årets utveckling (inte bara betygsbokstäver)
```

## Projektbaserad bedömning

**Flerveckorsprojekt visar djup förståelse**

### SO-projekt

**Tema**: Lokal samhälle (årskurs 2)

**Traditionell bedömning**:
```
Prov: 20 frågor om samhällshjälpare, kartfärdigheter, medborgarskap
Format: Ifyll-i-lucka, matchning
Tid: 30 minuter
```

**Projektalternativ**:
```
Uppgift: "Skapa en samhällsguide"

Vecka 1: Forskning
- Intervjua 2 samhällshjälpare (brandman, bibliotekarie, etc.)
- Arbetsblad: Mall för intervjufrågor
- Dokumentera: Vad gör de? Varför är deras jobb viktigt?

Vecka 2: Kartskapande
- Rita karta över lokala samhället (viktiga landmärken)
- Märk ut: skola, bibliotek, brandstation, polisstation, park
- Arbetsblad: Kartplaneringsrutnät (skala angiven)

Vecka 3: Skrivande
- Skriv "Om vårt samhälle"-häfte
- Delar: Historia, Viktiga personer, Platser att besöka, Hur man blir en god medborgare
- Arbetsblad: Skrivorganisatör för varje del

Vecka 4: Presentation
- Dela samhällsguiden med klassen
- Förklara: Vad gör vårt samhälle speciellt?

Bedömningskriterier:
- Forskning komplett och korrekt (4 poäng)
- Karta tydlig och märkt (4 poäng)
- Skrivande informativt och organiserat (5 poäng)
- Presentation tydlig och engagerande (3 poäng)
Totalt: 16 poäng

Vad det bedömer:
- SO-innehåll (samhällskunskap)
- Forskningsfärdigheter (intervjuer, källor)
- Geografi (kartskapande)
- Skrivande (informativ text)
- Kommunikation (presentation)

Tidsinvestering: 4 veckor
Djup: Mycket djupare än 30-minuters prov
```

## Exitbiljetter (Snabba formativa bedömningar)

**3-5 minuters förståelsekontroller**

### Dagliga exitbiljettexempel

**Efter matteLektion om bråk**:
```
Exitbiljett (arbetsblad - tar 42 sek att generera):

Namn: _____________ Datum: _______

Idag lärde vi oss om att jämföra bråk.

1. Vilket är större: 1/2 eller 1/4? Förklara hur du vet.
   _________________________________________________

2. Rita en bild för att visa varför 3/4 är större än 1/2.
   [Plats för ritning]

3. En sak jag fortfarande inte förstår:
   _________________________________________________

Läraranvändning: Läs biljetter, gruppera elever efter förståelse
- Tydlig förståelse → Självständig träning imorgon
- Förvirring → Liten grupp omundervisning imorgon
```

---

### NO-exitbiljett

**Efter lektion om materiens tillstånd**:
```
Exitbiljett:

Dagens ämne: Materiens tillstånd

1. Ge ett exempel på:
   Fast ämne: _______  Vätska: _______  Gas: _______

2. Vad händer när du värmer is?
   _________________________________________________

3. Rita: Visa vatten som fast ämne, vätska och gas
   [Tre rutor för ritningar]

4. Bedöm din förståelse: ☐ Fattar  ☐ Behöver hjälp  ☐ Förvirrad

Läraranvändning: Sortera efter självbedömning
- "Fattar" → Fördjupningsaktivitet imorgon
- "Behöver hjälp" → Fler exempel och träning
- "Förvirrad" → Omundervisning med visuellt stöd
```

## Autentiska demonstrationer

**Verklig färdighetstillämpning**

### Läsning: Bokrekommendation

**Bedömningsuppgift**:
```
Istället för: Traditionell bokrapport (sammanfatta handling)

Prova: Bokrekommendationspresentation

Uppgift: "Övertyga dina klasskamrater att läsa din bok"

Krav:
1. Engagerande sammanfattning (avslöja inte slutet!)
2. Varför du gillade den (personlig koppling)
3. Vem skulle gilla den (målgruppsidentifiering)
4. Läs favoritpassage högt (läsflytdemonstration)
5. Skapa visuellt stöd (affisch, ritning, diorama)

Arbetsblad: Planeringsguide
- Boktitel/författare: ______________
- Genre: ______________
- Huvudkaraktärer: ______________
- Anledning 1 att läsa den: ______________
- Anledning 2 att läsa den: ______________
- Anledning 3 att läsa den: ______________
- Passage att läsa högt (sida ___): Märk med post-it

Bedömning:
- Förståelse demonstrerad? (förstod berättelsen)
- Kommunikation tydlig? (presentationsfärdigheter)
- Läsflyt lämpligt? (högläsning)
- Övertalning effektiv? (övertygande argument)

Autentiskt: Så här delar människor verkligen böcker (inte konstgjord bokrapport)
```

## Bedömningsmatriser för alternativ bedömning

**Tydliga kriterier säkerställer rättvis bedömning**

### Projektbedömningsmall

**NO-mässa projekt** (genererad som arbetsblad):
```
Bedömningsmall: NO-mässa projekt

Forskningsfråga (4 poäng)
4 = Tydlig, testbar fråga
3 = Fråga finns men kunde vara tydligare
2 = Frågan för bred eller inte testbar
1 = Ingen tydlig fråga

Metod (4 poäng)
4 = Detaljerade, reproducerbara steg
3 = Steg finns, vissa detaljer saknas
2 = Metod oklar
1 = Ingen metod dokumenterad

Datainsamling (4 poäng)
4 = Korrekt data, organiserad tabell/graf
3 = Data finns, organisation kunde förbättras
2 = Data ofullständig
1 = Ingen data insamlad

Slutsats (4 poäng)
4 = Tydlig slutsats stödd av data
3 = Slutsats finns, svag koppling till data
2 = Slutsats matchar inte data
1 = Ingen slutsats

Presentation (4 poäng)
4 = Tydlig, engagerande, välorganiserad
3 = Begriplig, lämplig organisation
2 = Svår att följa
1 = Förvirrande eller ofullständig

Totalt: _____ / 20 poäng

Elevnytta: Vet exakt vad som förväntas innan projektet startar
```

## Sammanfattning

Autentisk bedömning förutsäger bättre **verklig prestation** (Wiggins, 1990) - elever visar djup förståelse genom tillämpning.

**Prestationsuppgifter**: "Designa ett sovrum" (area/omkrets tillämpning), växtexperiment (naturvetenskaplig metod demonstration)

**Portfoliobedömning**: Samla arbetsprover sept-maj (visa utveckling över tid), elevreflektion över framsteg

**Projektbaserad**: Flerveckorsprojekt (Samhällsguide), bedöm flera kunskapsmål i en autentisk uppgift

**Exitbiljetter**: 3-5 minuters dagliga kontroller (sortera elever efter förståelse, informera nästa dags gruppering)

**Självbedömning**: Lärandemålschecklistor (elever utvärderar egen behärskning), projektreflektioner (metakognition)

**Kamratbedömning**: Kamratgranskningsprotokoll (elever ger feedback, lär sig utvärderingsfärdigheter)

**Autentiska demonstrationer**: Bokrekommendationer (verklig färdighet), presentationer (kommunikationsbedömning)

**Bedömningsmatriser**: Tydliga kriterier (elever vet förväntningar), rättvis bedömning (konsekventa standarder)

**Forskning**: Alternativ bedömning = bättre förutsägelse av verklig prestation (Wiggins, 1990)
