# Grid Match App - Feature Analysis

## Source: REFERENCE APPS/grid match.html

## Verified Features

### 1. Page Setup Accordion
- **Language Selection** (11 languages): EN, DE, FR, ES, PT, IT, NL, SV, DA, NO, FI
- **Page Size Options**:
  - Letter Portrait (8.5×11")
  - Letter Landscape (11×8.5")
  - A4 Portrait (210×297mm)
  - A4 Landscape (297×210mm)
  - Custom size (width/height in pixels)
- **Background Options**:
  - Fallback color picker
  - Background theme selection
  - Background opacity slider (0-1)
- **Border Options**:
  - Border theme selection
  - Border gallery preview
  - Border opacity slider (0-1)

### 2. Text Tools Accordion
- **Add New Text**: Input field + "Add Text to Worksheet" button
- **Font Family Options** (7 fonts):
  - Lexend Deca
  - Baloo 2
  - Nunito
  - Quicksand
  - Fredoka
  - Arial
  - Verdana
- **Text Properties**:
  - Text color picker
  - Font size (min 8)
  - Outline color
  - Outline width (0-10, step 0.5)

### 3. Grid Options Accordion
- **Rows**: 2-4 (default 3)
- **Columns**: 2-4 (default 3)
- **Clue Cells**: 1-5 (default 1)

### 4. Image Library Accordion
- Theme-based selection
- Search function for images
- Available images gallery
- Selected image preview
- 3000+ child-friendly images

### 5. Upload Custom Images Accordion
- Multi-file upload (accepts image/*)
- Preview of uploaded images
- Click to use uploaded images

### 6. Tabs
- Worksheet tab
- Answer Key tab

### 7. Toolbar (Object Context)
- **Layers**: Bring to Front, Bring Forward, Send Backward, Send to Back
- **Align**: Left, Center H, Right, Top, Center V, Bottom, Center on Page H/V
- **Lock/Unlock**: Lock objects in place
- **Delete**: Remove selected objects

### 8. Download Options
- Worksheet (JPEG)
- Answer Key (JPEG)
- Worksheet (PDF)
- Answer Key (PDF)
- Grayscale toggle

### 9. Top Right Actions
- Undo/Redo
- Clear Canvas
- Generate Worksheet
- Generate Answer Key

## App Description
Grid Match is a visual puzzle worksheet generator where children match pieces to complete a picture. A large image is divided into a grid, with some cells removed. Children must identify which pieces belong where. This develops visual discrimination, spatial reasoning, and problem-solving skills.
