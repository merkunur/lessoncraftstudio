## Stap 4: Je Werkblad Aanpassen

**Parameters instellen**

### Woordzoeker Voorbeeld Uitwerking

**Aanpassingsmogelijkheden**:

**1. Voer woordenlijst in**:
```
Invoerveld: Typ of plak woorden

Voorbeeld (aardrijkskunde):
- hoofdstad
- provincie
- Nederland
- rivier
- gebergte
- klimaat
- continent
- oceaan

Tip: Eén woord per regel, of gescheiden door komma's
```

---

**2. Kies rastergrootte**:
```
Opties:
- 10×10: Makkelijk (jongere leerlingen, eenvoudige woorden)
- 15×15: Gemiddeld (standaard moeilijkheidsgraad)
- 20×20: Moeilijk (oudere leerlingen, meer woorden)

Aanbeveling: Begin met 15×15 (pas aan als te makkelijk/moeilijk)
```

---

**3. Stel moeilijkheidsgraad in**:
```
Makkelijk: Woorden alleen horizontaal en verticaal (geen diagonaal)
Gemiddeld: Horizontaal, verticaal, diagonaal
Moeilijk: Alle richtingen inclusief achterwaarts

Kies op basis van: Leerjaar en ervaring met woordzoekers
```

---

**4. Extra opties**:
```
☑ Antwoordblad toevoegen (aangeraden)
☑ Woordenlijst tonen (lijst met te zoeken woorden)
☐ Woordenlijst verbergen (uitdagender)

Titel: "Aardrijkskunde Woordenschat"
Instructies: "Vind alle 8 woorden over Nederland"
```

## Stap 8: Efficiënt Printen

**Print instellingen zijn belangrijk**

### Optimale Print Instellingen

**Printer voorkeuren**:
```
Kwaliteit: Standaard (niet concept, niet foto)
- Concept: Te licht (moeilijk leesbaar)
- Standaard: Helder, gebruikt juiste hoeveelheid inkt
- Foto: Onnodig (verspilt inkt, langzaam)

Kleur: Zwart-wit (tenzij specifiek nodig)
- Meeste werkbladen: Z/W voldoende
- Kleurplaten: Kleur vereist
- Besparing: Z/W kost 80% minder per pagina

Dubbelzijdig: Waar geschikt
- Oefenwerkbladen: Enkelzijdig (leerlingen schrijven erop)
- Leesteksten: Dubbelzijdig OK (bespaart papier)
- Belangrijk: Controleer welke kant "boven" is (draairichting belangrijk)

Papier: Standaard 80 grams kopieerpapier
- Vermijd dik papier: Duur, onnodig voor meeste werkbladen
- Vermijd gerecycled: Kan soms vastlopen, minder scherp
```

---

### Bulk Printen Tips

**Voor klas van 30 leerlingen**:
```
Print opdracht: 30 kopieën van werkblad

Instellingen:
1. Vul papierlade volledig (200+ vellen)
2. Selecteer "Sorteren" (als je later gaat nieten)
3. Kies "Alles tegelijk printen" (niet één voor één)
4. Verstuur print opdracht
5. Loop weg (laat het draaien)

Tijd: 30 kopieën = ~3 minuten printtijd
Vermijd: Bij printer blijven staan (gebruik tijd voor andere taken)
```

## Veelgemaakte Fouten Voorkomen

**Leer van andermans ervaringen**

### Fout 1: Te Veel Parameters Aanpassen

**Probleem**:
```
Nieuwe gebruiker: Past elke optie aan (overweldigd door keuzes)
Resultaat: Verlamt door keuzes (duurt 10 minuten voor één werkblad)
```

**Oplossing**:
```
Begin eenvoudig: Gebruik eerst standaard instellingen
Genereer: Bekijk resultaat
Pas aan: Wijzig alleen wat echt moet
Herhaal: Ontdek geleidelijk meer opties

Eerste werkblad: 2 minuten (leren)
Vijfde werkblad: 30 seconden (zelfverzekerd)
```

---

### Fout 2: Verkeerde Moeilijkheidsgraad

**Probleem**:
```
Leerkracht: Maakt "moeilijk" werkblad voor zwakke leerlingen
Resultaat: Frustratie, opgeven
```

**Oplossing**:
```
Begin makkelijker: Bij twijfel, kies één niveau lager
Test: Geef eerst aan één leerling
Pas aan: Makkelijker of moeilijker op basis van reactie
Verfijn: Krijg moeilijkheidsgraad precies goed na verloop van tijd
```

---

### Fout 3: Bestanden Niet Georganiseerd

**Probleem**:
```
Downloads: Alles naar standaard "Downloads" map
Maanden later: "Waar is die aardrijkskunde woordzoeker die ik maakte?"
Resultaat: Werkbladen onnodig opnieuw maken
```

**Oplossing**:
```
Direct: Verplaats gedownload werkblad naar juiste map
Naamgeving: Gebruik beschrijvende namen (niet alleen "werkblad1.pdf")
Hergebruik: Vind en hergebruik volgend jaar (bespaart tijd)
```

## Veelvoorkomende Problemen Oplossen

**Snelle oplossingen**

### Probleem: "Werkblad wordt niet gegenereerd"

**Mogelijke oorzaken**:
```
1. Te veel woorden voor rastergrootte
   Oplossing: Verminder aantal woorden of vergroot raster

2. Internetverbinding verbroken
   Oplossing: Controleer WiFi, ververs pagina

3. Ongeldige tekens in invoer
   Oplossing: Verwijder speciale symbolen, gebruik standaard letters
```

---

### Probleem: "Printkwaliteit is slecht"

**Oplossingen**:
```
1. Controleer printer instellingen (moet "Standaard" zijn, niet "Concept")
2. Zorg voor voldoende inkt (vervang cartridge bij laag niveau)
3. Reinig printkoppen (printer onderhoudsmenu)
4. Gebruik kwaliteitspapier (goedkoop papier kan wazig zijn)
```

---

### Probleem: "Kan gedownload bestand niet vinden"

**Oplossing**:
```
1. Controleer standaard Downloads map
2. Zoek computer naar PDF aangemaakt vandaag
3. Browser: Bekijk download geschiedenis (toont bestandslocatie)
4. Voortaan: Bewaar direct in georganiseerde map
```
