## Introductie: Zelfstandigheid Opbouwen in de Klas

**Herkenbare lerarenuitspraak**: "Ik kan niet iedereen tegelijk helpen!"

**Realiteit van een klas met 28 leerlingen**:
```
Leerkracht werkt met instructiegroep (5 leerlingen, 20 minuten)
Overige 23 leerlingen: Moeten productief zelfstandig werken

Probleem: Leerlingen onderbreken
- "Ik snap de opdracht niet"
- "Ik ben klaar, wat nu?"
- "Kun je mijn werk nakijken?"
- Resultaat: Instructiegroep constant verstoord
```

**Oplossing**: Leerlingen systematisch zelfstandig leren werken
```
Systemen ingericht:
- Duidelijke verwachtingen (leerlingen weten wat ze moeten doen)
- Zelfinstructerende materialen (kunnen zonder leerkracht werken)
- Zelfcontrolemogelijkheden (niet wachten op nakijken)
- Keuzemenu's (als ze klaar zijn, duidelijke vervolgstappen)

Resultaat: Leerkracht kan zich focussen op instructiegroep (23 leerlingen productief bezig)
```

**Onderzoek** (Zimmerman, 2002): Zelfregulerende leervaardigheden verhogen prestaties met **0,4-0,6 standaarddeviaties**

**Kernprincipe**: Zelfstandigheid wordt AANGELEERD, niet verondersteld

## Zelfinstructerende Werkbladen

**Ontwerp materialen die leerlingen alleen kunnen uitvoeren**

### Duidelijke Instructies OP het Werkblad

**Zwak ontwerp** (vraagt om uitleg van leerkracht):
```
Werkblad header:
Naam: ___________

[20 rekensommen zonder context]

Probleem: Leerlingen weten niet wat ze moeten doen
- "Moet ik mijn rekenwerk laten zien?"
- "Mag ik de rekenmachine gebruiken?"
- "Hoeveel tijd heb ik?"
→ Leerlingen vragen de leerkracht (verstoort instructiegroep)
```

**Sterk ontwerp** (zelfinstructerend):
```
Naam: ___________ Datum: ________ Verwachte tijd: 20 minuten

INSTRUCTIE:
1. Maak alle vermenigvuldigingssommen
2. Laat je rekenwerk zien in de uitwerkruimte
3. Je mag GEEN rekenmachine gebruiken
4. Check je antwoorden met de QR-code (scan rechtsonder)
5. Bij 5+ fouten: vraag de leerkracht om hulp
6. Alles goed? Ga dan naar de PLUSBAK

[20 rekensommen, elk met uitwerkruimte]

[QR-code rechtsonder] ← Scan om antwoorden te controleren

Duidelijke verwachtingen: Leerling weet precies wat te doen (geen vragen nodig)
```

## Tijdmanagementtools

**Help leerlingen hun zelfstandige werktijd indelen**

### Visuele Timer Weergave

**Klassikale systeem**:
```
Hulpmiddel: Grote digitale timer op digibord (of fysieke timer zichtbaar voorin)

Leerkracht kondigt aan: "Je hebt 20 minuten voor het rekenwerk. De timer gaat nu starten."

Voordelen:
- Leerlingen zien resterende tijd (tempo aanpassen)
- "Ik heb 10 minuten over en nog 5 sommen" → Leerling past tempo aan
- Geen verrassingen als tijd op is (visueel aftellen)

Zelfstandige vaardigheid: Leren eigen tijd bewaken (essentiële levensvaardigheid)
```

---

### Persoonlijke Tijdregistratie

**Werkbladfunctie**:
```
Bovenaan werkblad:

Starttijd: ______ Eindtijd: ______ Totale tijd: ______

Doel: Afmaken in 20 minuten

Leerling noteert:
Start: 9:05    Eind: 9:22    Totaal: 17 minuten ✓ Doel gehaald!

OF

Start: 9:05    Eind: 9:30    Totaal: 25 minuten ✗ Langer dan doel

Leerkrachtgebruik: Identificeer leerlingen die consistent moeite hebben met tempo
Leerlinggebruik: Zelfbewustzijn van werksnelheid (metacognitie)
```

## Zelfcontrolesystemen

**Directe feedback zonder leerkracht**

### Toegang tot Antwoordmodel

**Systeemopties**:

**Optie 1: QR-Code** (digitaal nakijken)
```
Rechtsonder op werkblad: QR-code
Leerling scant: Antwoordmodel verschijnt op device
Controleert zelf: Markeert fouten met rode pen
Verbetert: Herstelt fouten
Levert in: Verbeterd werkblad

Leerkrachtvoordeel: Krijgt grotendeels correct werk (niet tijd besteden aan simpele fouten nakijken)
Leerlingvoordeel: Directe feedback (leert meteen van fouten)
```

**Optie 2: Antwoordmodellenstation** (fysiek nakijken)
```
Klasopstelling: Antwoordmodellenmap op tafel achterin

Leerling maakt werkblad af → loopt naar station → controleert antwoorden → gaat terug om te verbeteren

Protocol:
1. Maak eerst ALLE opgaven (niet één voor één controleren)
2. Gebruik andere kleur pen om na te kijken (toont wat verbeterd is)
3. Herstel fouten
4. Leg antwoordmodel terug

Voordeel: Leerlingen wachten niet in de rij tot leerkracht werk nakijkt
```

**Optie 3: Partnercontrole** (samenwerkend)
```
Partnersysteem: Leerlingen wisselen werkbladen uit
Ieder kijkt werk van partner na met antwoordmodel
Geeft feedback: "Opgave 5 is fout. Check je aftreksommen."

Dubbel voordeel:
- Beide leerlingen herhalen leerstof (nakijker versterkt eigen leren)
- Directe feedback (niet wachten op leerkracht)
```

## Keuzeborden voor Snelle Werkers

**"Ik ben klaar! Wat nu?" → Duidelijke vervolgstappen**

### Gestructureerd Keuzemenu

**Opgehangen in klas**:
```
"Als Je Klaar Bent Met Je Werkblad..."

MOET JE:
☐ 1. Je antwoorden controleren (QR-code of antwoordmodellenstation)
☐ 2. Fouten verbeteren

DAN KIES JE:
☐ Rekenpuzzels (Sudoku-bak)
☐ Lezen (zelfstandig lezen + leeslogboek invullen)
☐ Schrijven (creatieve schrijfopdrachten map)
☐ Pluswerkblad (uitdagende opdrachten bak)
☐ Vriend helpen (als deze vraagt om hulp)

MAG NIET:
☒ Leerkracht storen tijdens instructie
☒ Rondlopen in de klas
☒ Niets doen

Duidelijke verwachtingen: Leerlingen weten altijd wat te doen (geen onderbrekingen)
```

## "Vraag Drie Voordat Je Mij Vraagt" Protocol

**Probleem oplossen voordat leerkracht vragen**

### Klassenposter

**Regel**: Voordat je de leerkracht vraagt, probeer deze 3 dingen:
```
1. Lees de instructie opnieuw zorgvuldig
   (Vaak staat het antwoord in de opdracht)

2. Raadpleeg je hulpmiddelen
   (Rekenen: Rekenkaart, formuleblad)
   (Taal: Woordenlijst, woordbank)
   (Spelling: Checklist, schrijfwijzer)

3. Vraag een buurman/buurvrouw
   (Fluisterend: "Kun je me helpen bij opgave 5?")

DAN: Als je het nog niet snapt, steek je hand op voor de leerkracht

Voordelen:
- Minder onderbrekingen leerkracht (veel vragen zelfstandig opgelost)
- Bouwt probleemoplossend vermogen (vindingrijkheid)
- Creëert samenwerkende sfeer (elkaar helpen)
```

## Zelfstandig Tempo Werken

**Leerlingen gaan verder als ze klaar zijn**

### Stationswerken

**Systeem**: 4 stations, leerlingen werken in eigen tempo
```
Station 1: Optellen met overschrijden
- Werkblad afmaken (20 sommen)
- Controleren met antwoordmodel (minimaal 16 goed = 80%)
- Als geslaagd: Naar Station 2

Station 2: Aftrekken met lenen
- Werkblad afmaken
- Controleren (minimaal 80%)
- Als geslaagd: Naar Station 3

Station 3: Gemengd optellen/aftrekken
- Werkblad afmaken
- Controleren (minimaal 80%)
- Als geslaagd: Naar Station 4

Station 4: Tekstsommen (toepassing)
- Werkblad afmaken
- Controleren (minimaal 80%)
- Als geslaagd: Thema compleet! Start volgend thema.

Differentiatie ingebouwd:
- Snelle leerlingen: Alle 4 stations af in 1 week
- Gemiddelde leerlingen: Af in 1,5 week
- Zwakke leerlingen: Af in 2 weken
- Elke leerling werkt in passend tempo (niet gehaast of afgeremd)
```

## Zelfstandigheid Geleidelijk Aanleren

**Stapsgewijs naar meer zelfstandigheid**

### September: Veel Ondersteuning

**Leerkracht doet**:
```
- Leest instructies hardop voor
- Maakt voorbeeldopgaven samen
- Kijkt ieders werk na
- Vertelt leerlingen wat ze moeten doen als ze klaar zijn

Zelfstandig werk: 10 minuten
Leerkrachtondersteuning: Veel
```

---

### November: Gemiddelde Ondersteuning

**Leerkracht doet**:
```
- Leerlingen lezen instructies zelf (leerkracht beschikbaar voor vragen)
- Leerlingen werken werkblad alleen af
- Leerlingen controleren zelf met antwoordmodel (leerkracht steekproefsgewijs)
- Leerlingen kiezen uit 2-3 opties als ze klaar zijn

Zelfstandig werk: 20 minuten
Leerkrachtondersteuning: Gemiddeld
```

---

### Maart: Weinig Ondersteuning

**Leerlingen doen**:
```
- Lezen en volgen instructies zelfstandig
- Maken volledig werkblad af zonder leerkrachtinteractie
- Controleren grondig zelf
- Kiezen vervolgactiviteit van keuzebord
- Werken productief 30+ minuten

Leerkrachtrol: Werkt met instructiegroep, scant af en toe de klas
Zelfstandigheid bereikt: Leerlingen zelfsturend
```

**Progressie is geleidelijk**: Vaardigheden opbouwen over maanden (niet onmiddellijke zelfstandigheid verwachten)

## Fysieke Organisatie voor Zelfstandigheid

**Klasinrichting ondersteunt autonomie**

### Materiaalstations

**Zelfbedieningsstations**:
```
Station 1: "Extra Werkbladen"
- School gemist? Pak gemiste opdrachten
- Werkblad kwijt? Pak vervanging
- Leerlingtoegang: Altijd (niet vragen aan leerkracht)

Station 2: "Materialen"
- Potloden, gummen, stiften, scharen
- Pak wat nodig is (niet vragen)

Station 3: "Inlevermanden"
- Eén mand per vak (gelabeld: Rekenen, Taal, Spelling)
- Leerling maakt werk af → legt in juiste mand
- Niet wachten tot leerkracht ophaalt

Station 4: "Nagekeken Werk"
- Nagekeken opdrachten hier teruggelegd
- Leerlingen halen op tijdens zelfstandige werktijd

Voordeel: Leerlingen pakken zelfstandig wat ze nodig hebben (leerkracht niet gestoord)
```

## Verwachtingen Zelfstandig Werken

**Expliciet gedrag aanleren**

### Ankergrafiek: "Zo Ziet Zelfstandig Werken Eruit"

**Samen opstellen met leerlingen**:
```
ZO ZIET HET ERUIT:
- Werken aan tafel of aangewezen werkplek
- Ogen op eigen werk
- Stil materialen gebruiken
- Werk zelfstandig controleren

ZO KLINKT HET:
- Fluisterstemmen (als je hulp vraagt aan buurman)
- Potloden schrijven
- Bladzijden omslaan
- Grotendeels stil

ZO ZIET HET ER NIET UIT:
- Rondlopen in de klas
- Hardop praten
- Leerkracht onderbreken
- Niets zitten doen

Modellen: Leerkracht demonstreert goed en fout (leerlingen benoemen verschil)
Oefenen: Klas oefent zelfstandig werken terwijl leerkracht observeert
Bekrachtigen: Leerkracht prijst specifiek gedrag ("Ik zie iedereen stil werken!")
```

## Zelfstandig Werk Monitoren

**Leerkracht controleert zonder instructiegroep te storen**

### Snelle Scan Systeem

**Elke 5 minuten**:
```
Leerkracht pauzeert instructie kort (10 seconden)
Scant klas: Zijn zelfstandig werkenden bezig?

Signalen:
- Duim omhoog: "Je doet het goed, ga zo door!" (geen woorden nodig)
- Wijs naar timer: "Check hoeveel tijd je nog hebt"
- Houd werkblad omhoog: "Zorg dat je het goede werk doet"

Geen onderbreking: Instructiegroep gaat door, zelfstandig werkenden bijgestuurd indien nodig
```

---

### Inleverprotocol

**Afrondingsroutine**:
```
Als leerling werkblad afmaakt:

Stap 1: Antwoorden zelfstandig controleren
Stap 2: "Kwaliteitschecklist" invullen (achterkant werkblad)
   ☑ Ik heb alle opgaven gemaakt
   ☑ Ik heb mijn antwoorden gecontroleerd
   ☑ Ik heb fouten verbeterd
   ☑ Ik heb mijn rekenwerk laten zien
   ☑ Mijn naam staat op het blad

Stap 3: Inleveren in mand
Stap 4: Volgende activiteit kiezen van keuzebord

Voordeel: Kwaliteitscontrole (leerlingen reviewen eigen werk voor inleveren)
```

## Veelvoorkomende Problemen Oplossen

**Oplossingen voor zelfstandigheidsproblemen**

### Probleem: "Leerlingen onderbreken nog steeds tijdens instructie"

**Oplossing: Visueel signaal**
```
Rode beker op leerkrachtentafel: "Niet storen - alleen bij nood"
Groene beker: "Ik kan snelle vragen beantwoorden"

Regel: Bij rode beker moeten leerlingen eerst "Vraag 3 Voordat Je Mij Vraagt" protocol proberen
Resultaat: Onderbrekingen met 80%+ gereduceerd
```

---

### Probleem: "Leerlingen jagen door werk heen zonder zorgvuldigheid"

**Oplossing: Kwaliteitsnormen**
```
Minimum tijdseis: "Dit werkblad moet minstens 15 minuten duren"

Plus: Zelfcontrole onthult fouten (leerling ziet gevolgen van jachten)

Plus: "Als je binnen 10 minuten klaar bent, kijk ik je werk na. Bij 3+ fouten moet je opnieuw." (verantwoordelijkheid)

Resultaat: Leerlingen leren zorgvuldig werken (jachten wordt niet beloond)
```

## Conclusie

Zelfregulerende leervaardigheden verhogen prestaties **0,4-0,6 SD** (Zimmerman, 2002) - zelfstandigheid geeft leerlingen kracht.

**Zelfinstructerende werkbladen**: Duidelijke instructie OP blad (verwachte tijd, stappen, controlemethode), leerlingen maken af zonder vragen

**Tijdmanagement**: Visuele timers (leerlingen zien aftellen), persoonlijke registratie (noteer start/eindtijd), bouwt tempobewustzijn

**Zelfcontrole**: QR-code antwoordmodellen (directe feedback), antwoordmodellenstations (fysiek nakijken), partnercontrole (samenwerkend)

**Keuzeborden**: Moet-doen stappen (controleren, verbeteren) + dan-kiezen opties (puzzels, lezen, plus), duidelijke vervolgstappen

**Vraag 3 Voordat Je Mij Vraagt**: Herlaas instructie → raadpleeg hulpmiddelen → vraag buurman → dan leerkracht (reduceert onderbrekingen 80%+)

**Stations in eigen tempo**: Verder als klaar (80%+ om door te gaan), differentiatie automatisch (snelle/gemiddelde/zwakke leerlingen)

**Geleidelijke afbouw**: September veel ondersteuning → maart weinig ondersteuning (bouw zelfstandigheid op over maanden)

**Fysieke organisatie**: Materiaalstations (zelfbediening materialen, inlevermanden), leerlingen pakken zelfstandig

**Verwachtingen**: Ankergrafiek (zo ziet/klinkt het), modelleren en oefenen, bekrachtig positief gedrag
