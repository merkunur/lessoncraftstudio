# Wordsearch Features Analysis (Norwegian Page)

## PRICING VERIFICATION
**App Name:** Word Search
**Subscription Tier:** FREE (with watermark on worksheets)
**Limitations:** Personal use only, NO commercial license
**Premium Version:** Core Bundle ($144/year) or Full Access ($240/year) removes watermark and adds commercial license

## KEY FEATURES FOUND IN wordsearch.html

### 1. Language Settings (line 840-862)
- Image Library Language selector
- 11 languages: en, de, fr, es, pt, it, nl, sv, da, no, fi
- Critical for Norwegian content creation

### 2. Page & Scene (line 863-905)
- **Page Size Options:**
  - Letter Portrait (8.5×11")
  - Letter Landscape (11×8.5")
  - A4 Portrait (210×297mm)
  - A4 Landscape (297×210mm)
  - Custom size (px inputs)
- **Background:**
  - Fallback color selector
  - Background theme selector
  - Background opacity slider
- **Border:**
  - Border theme selector
  - Border opacity slider

### 3. Text Tools (line 906-928)
- Add new text with custom content
- **Text Properties:**
  - Color picker
  - Size (number input, min 8)
  - Font family: Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana
  - Outline color
  - Outline width (0-10 range slider)

### 4. Puzzle Settings (line 929-953)
- **Grid Size:**
  - Rows (number input)
  - Columns (number input)
- **Puzzle Options (checkboxes):**
  - Allow Diagonal Words (checked by default)
  - Allow Reverse Words
  - Show Word/Image List (checked by default)
  - Show only images
  - Show only words

### 5. Image Library (line 954-975)
- **Image Source for Puzzle:**
  - Theme selector with "Use Random Theme" option
- **Individual Image Selection:**
  - Filter by theme dropdown
  - Search images text input
  - Available Images display (max 8 images)
  - Selected Images preview area

### 6. Upload Custom Images (line 976-990)
- Multi-file upload button
- File selection indicator
- Preview of uploaded images for current session

### 7. Manual Image Name Editing (line 991-1009)
- Enable manual edit checkbox
- Edit image names before generating
- Click to add images from library
- Edit names in the preview area

### 8. Custom Word List (line 1010-1027)
- Use custom word list checkbox
- Text area for entering words (one per line, max 8)
- Alternative to using image library

### 9. Layer Controls (line 1039-1074)
- **Layer Order:**
  - Bring to Front
  - Bring Forward
  - Send Backward
  - Send to Back
- **Alignment - Selected Objects:**
  - Align selected items relative to each other
- **Alignment - Page:**
  - Align selected items to page edges

### 10. Download and Generate (line 1079-1123)
- **Tabs:**
  - Worksheet tab
  - Answer Key tab
- **Unlock All Button:** Unlock all locked objects
- **Generate Dropdown:**
  - New Worksheet
  - Answer Key
- **Download Dropdown:**
  - Worksheet (JPEG)
  - Answer Key (JPEG)
  - Worksheet (PDF)
  - Answer Key (PDF)
- **Grayscale Toggle:** Export in grayscale to save ink

### 11. 11 Language Support
Critical feature - content language selector allows creating Norwegian word searches using Norwegian image names/words.
