## Introdução: Encerrar o Ano Letivo com Intencionalidade Pedagógica

**O desafio das últimas semanas**: Alunos mentalmente nas férias, problemas comportamentais, tempo desperdiçado

**Investigação** (McCombs & Pope, 1994): A reflexão estruturada melhora a retenção em **35%** (rever conteúdos consolida aprendizagens)

**Final de ano tradicional**: Filmes, festas, ausência de trabalho académico
```
Realidade observada:
- 2 semanas: Atividades não-académicas
- Perda de aprendizagem: Alunos esquecem matéria (regressão de verão começa cedo)
- Problemas comportamentais: Falta de estrutura = caos na sala de aula
```

**Alternativa pedagógica eficaz**: Reflexão + celebração + consolidação
```
Estrutura proposta:
- Semana 1: Revisão sistemática de todo o ano letivo
- Semana 2: Atividades de reflexão + criação de portfólio
- Dias finais: Celebração com ligação académica

Resultado: Aprendizagem contínua até ao último dia, alunos orgulhosos do seu crescimento
```

## Construção do Portfólio de Crescimento

**Evidência tangível do progresso ao longo do ano**

### Conteúdos do Portfólio

**Comparação setembro-junho**:
```
Secção inicial: "Onde Comecei"
- Amostra de escrita de setembro (primeira semana)
- Avaliação diagnóstica de matemática (linha de base)
- Trabalho de expressão plástica de setembro (coordenação motora inicial)

Secção final: "Onde Estou Agora"
- Amostra de escrita de junho (última semana)
- Avaliação final de matemática (fim de ano)
- Projeto artístico de junho (evolução motora fina)

Secção intermédia: "A Minha Jornada"
- Fotografias do ano letivo
- Fichas favoritas (selecionadas pelo aluno)
- Certificados de conquistas
- Gráfico de progresso (fluência leitora, tabelas de multiplicação, etc.)
```

**Atividade de montagem** (Semana 1 das últimas duas semanas):
```
Segunda-Terça: Alunos recolhem trabalhos dos dossiers
Quarta: Organizam portfólio (ordem cronológica)
Quinta: Decoram capa, escrevem reflexão
Sexta: Partilham com par ou pequeno grupo
```

## Atividades de Consolidação (Últimas 2 Semanas)

**Revisão sistemática dos conteúdos do ano**

### Calendarização Segunda-Sexta de Revisão

**Semana 1: Revisão de Conteúdos**
```
Segunda: Revisão espiral de matemática (todas as competências do ano, ficha de 30 exercícios)
Terça: Caça-palavras de vocabulário (100 palavras de todo o ano)
Quarta: Palavras-cruzadas de ciências (termos-chave de todas as unidades)
Quinta: Linha do tempo de história (acontecimentos/figuras estudadas)
Sexta: Compreensão leitora (excertos dos textos favoritos)

Formato: Fichas + jogos + discussões
Objetivo: Refrescar memória antes das férias de verão
```

**Semana 2: Reflexão + Celebração**
```
Segunda: Criar portfólios (organizar amostras de trabalho)
Terça: Escrever textos de reflexão (balanço do ano)
Quarta: Gravar mensagens vídeo (conselhos para a turma do próximo ano)
Quinta: Livros de autógrafos (colegas assinam cadernos de memórias)
Sexta: Celebração (jogos académicos, cerimónia de prémios, festa)
```
