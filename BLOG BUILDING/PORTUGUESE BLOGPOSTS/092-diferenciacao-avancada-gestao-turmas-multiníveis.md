## Introdução: A Realidade das Turmas com Diferentes Níveis

**Turma típica do 3º ano - Distribuição real**:
```
Níveis de leitura:
- 5 alunos: nível de 1º ano (2 anos de atraso)
- 12 alunos: nível de 3º ano (ao nível esperado)
- 8 alunos: nível de 4º-5º ano (1-2 anos adiantados)
- 5 alunos: aprendentes de português como língua adicional (níveis variados)

Níveis de matemática:
- 6 alunos: ainda consolidam competências do 2º ano
- 15 alunos: trabalham objetivos do 3º ano
- 9 alunos: preparados para desafios do 4º ano

Resultado: 30 alunos com 5+ níveis de instrução diferentes
```

**Investigação** (Tomlinson & Imbeau, 2010): A diferenciação aumenta o desempenho de **todos os alunos** (não apenas os que têm dificuldades ou os avançados)

**Problema tradicional**: Ensino uniformizado
```
Professor distribui: Mesma ficha para os 30 alunos
Resultado:
- Alunos com dificuldades: demasiado difícil (frustração, desistem)
- Alunos ao nível: adequado (trabalho produtivo)
- Alunos avançados: demasiado fácil (aborrecimento, problemas comportamentais)

Eficácia: 33% (só alcança o terço médio da turma)
```

**Solução através da diferenciação**: Múltiplas versões, mesmos objetivos
```
Professor distribui: Três versões (modificada, padrão, enriquecida)
Resultado:
- Com dificuldades: desafio apropriado (sucesso, confiança)
- Ao nível: trabalho adequado ao ano (domínio)
- Avançados: tarefas de extensão (envolvidos, em crescimento)

Eficácia: 100% (alcança todos os alunos no seu nível)
```

## Sistema de Fichas em Três Níveis

**Estratégia central**: Mesmo conceito, diferentes níveis de complexidade

### Modelo de Três Níveis

**Exemplo: Prática de multiplicação**

**Nível 1 (Modificado)** - Para alunos com dificuldades:
```
Problemas: 10 (vs 20)
Intervalo: factos 0-5 apenas (2 × 3, 4 × 5, etc.)
Apoio: imagens de arranjos retangulares fornecidas
Formato: letra maior, mais espaço

Exemplos de problemas:
1. 2 × 3 = ___ [imagem de 2 filas × 3 colunas]
2. 3 × 4 = ___ [imagem de 3 filas × 4 colunas]

Objetivo: construir confiança, dominar factos fundamentais
```

**Nível 2 (Padrão)** - Para alunos ao nível esperado:
```
Problemas: 20
Intervalo: factos 0-10 (toda a tabuada básica)
Apoio: nenhum (prática independente)
Formato: espaçamento padrão

Exemplos de problemas:
1. 6 × 7 = ___
2. 8 × 9 = ___

Objetivo: praticar objetivos do ano de escolaridade
```

**Nível 3 (Enriquecido)** - Para alunos avançados:
```
Problemas: 15
Intervalo: dois dígitos × um dígito (12 × 7, 23 × 4)
Desafio: problemas com contexto incluídos
Formato: operações mistas (× e ÷)

Exemplos de problemas:
1. 15 × 6 = ___
2. Se uma caixa contém 24 lápis, quantos lápis há em 8 caixas?
3. 72 ÷ 8 = ___ (verificação: ___ × 8 = 72)

Objetivo: estender para além do ano, aplicar a situações
```

**Princípio fundamental**: Os três níveis abordam multiplicação (mesmo objetivo curricular), mas com diferentes níveis de complexidade

## Estratégias de Grupos Flexíveis

**Os grupos mudam conforme a competência/tópico** (não são grupos fixos por capacidade)

### Rotação por Competências

**Segunda-feira: Avaliação diagnóstica** (todos fazem o mesmo teste de 10 problemas)
```
Resultados:
- 8 alunos: 0-4 corretos (precisam de reensino)
- 15 alunos: 5-7 corretos (precisam de prática)
- 7 alunos: 8-10 corretos (preparados para extensão)
```

**Terça a quinta-feira: Grupos diferenciados**
```
Grupo A (Reensino):
- Instrução orientada pelo professor (grupo pequeno, ensino direto)
- Materiais manipulativos fornecidos
- Fichas com apoio (Nível 1)
Duração: 30 minutos com o professor

Grupo B (Prática):
- Trabalho autónomo (fichas Nível 2)
- Verificação entre pares com chaves de respostas
- Professor circula para questões
Duração: 30 minutos independente

Grupo C (Extensão):
- Atividades de enriquecimento (fichas Nível 3)
- Problemas desafio, puzzles
- Opcional: ensinar depois ao Grupo A
Duração: 30 minutos independente

Sexta-feira: Avaliação sumativa (medir crescimento)
```

**Semana seguinte**: Nova competência, NOVOS grupos (alunos reagrupam com base em nova avaliação diagnóstica)

---

### Grupos por Interesse

**Nem toda a diferenciação é por capacidade** - às vezes é por interesse

**Exemplo: Prática de vocabulário** (mesmas 20 palavras, formatos diferentes)
```
Escolha 1: Sopa de letras (aprendentes visuais, baixo stress)
Escolha 2: Palavras cruzadas (aprendentes verbais, desafio moderado)
Escolha 3: Frases criativas (escritores criativos, aplicação)
Escolha 4: Ilustrações (aprendentes artísticos, memória visual)

Todos os alunos: aprendem as mesmas 20 palavras de vocabulário
Diferenciação: formato corresponde ao interesse/força
Resultado: maior envolvimento, mesmo resultado de aprendizagem
```

## Tarefas Paralelas

**Caminhos diferentes para o mesmo objetivo de aprendizagem**

### Exemplo de Matemática: Frações

**Meta de aprendizagem**: Compreender que frações representam partes de um todo

**Tarefa A** (Concreta):
```
Materiais: manipulativos de frações (peças físicas)
Atividade: "Usa círculos de frações para mostrar 1/2, 1/4, 1/8"
Ficha: desenha o que construíste, identifica frações
Nível: para alunos que precisam de experiência manipulativa
```

**Tarefa B** (Visual):
```
Materiais: ficha com formas
Atividade: "Pinta 1/2 deste retângulo, 3/4 deste círculo"
Ficha: múltiplas formas para colorir frações
Nível: para alunos preparados para representação visual
```

**Tarefa C** (Abstrata):
```
Materiais: ficha com reta numérica
Atividade: "Marca 1/4, 1/2, 3/4 na reta numérica"
Ficha: reta numérica + frações equivalentes
Nível: para alunos preparados para pensamento abstrato
```

**Avaliação comum**: Todos os alunos completam o mesmo bilhete de saída (mostrar 1/2 de qualquer forma)

## Estações de Aprendizagem Autorregulada

**Alunos trabalham progressões ao seu próprio ritmo**

### Configuração das Estações

**Exemplo: Unidade de geometria** (4 estações, 2 semanas)

**Estação 1: Formas** (Dias 1-3)
```
Tarefa: identificar formas 2D (círculo, quadrado, triângulo, retângulo, pentágono, hexágono)
Fichas: 3 níveis disponíveis
- Nível 1: associar formas a nomes (apoio com imagem)
- Nível 2: identificar formas em imagens (mundo real)
- Nível 3: classificar formas por propriedades (lados, ângulos)

Acompanhamento de progresso: aluno marca quando completado
Progressão: quando passar 80% na autoavaliação, avança para Estação 2
```

**Estação 2: Propriedades** (Dias 4-6)
```
Tarefa: aprender propriedades das formas (lados, vértices, ângulos)
Fichas: 3 níveis
- Nível 1: contar lados e vértices (básico)
- Nível 2: comparar formas (mais/menos lados)
- Nível 3: criar formas com propriedades específicas

Progressão: passar autoavaliação (80%+) → Estação 3
```

**Estação 3: Área e Perímetro** (Dias 7-9)
```
Tarefa: calcular medidas
Fichas: 3 níveis
- Nível 1: contar unidades para área (grelha fornecida)
- Nível 2: usar fórmulas (A = c × l, P = 2c + 2l)
- Nível 3: problemas contextualizados (aplicação)

Progressão: passar autoavaliação → Estação 4
```

**Estação 4: Aplicação no Mundo Real** (Dias 10-12)
```
Tarefa: aplicar geometria a problemas autênticos
Fichas: 3 níveis
- Nível 1: desenhar um jardim (dimensões dadas)
- Nível 2: desenhar um quarto (escolher dimensões)
- Nível 3: maximizar espaço (problema de otimização)

Conclusão: todas as 4 estações = unidade completa
```

**Diferenciação integrada**:
- Ritmo: alunos rápidos terminam em 8 dias, alunos mais lentos levam 12 dias
- Nível: alunos escolhem dificuldade apropriada em cada estação
- Apoio: professor trabalha com grupos pequenos na Estação 1 (fundacional)

## Compactação para Alunos Avançados

**Eliminar prática redundante** para alunos que já dominam o conteúdo

### Protocolo de Avaliação Prévia

**Segunda-feira: Pré-teste** (todo o conteúdo da unidade)
```
Unidade: adição de dois dígitos com reagrupamento (20 problemas)

Resultados:
- Maioria dos alunos: 0-10 corretos (precisam de instrução)
- 4 alunos: 18-20 corretos (já dominam)

Decisão: compactar para os 4 alunos avançados
```

**Terça a sexta-feira: Diferenciado**
```
Turma principal (26 alunos):
- Recebe instrução sobre adição de dois dígitos
- Completa fichas de prática (currículo padrão)

Grupo compactado (4 alunos):
- Salta instrução básica (já dominam)
- Trabalha em enriquecimento: adição de três dígitos + problemas contextualizados
- Completa fichas desafio (Nível 3, geradas em 42 segundos)

Resultado:
- Turma principal: domina objetivo do ano
- Grupo compactado: estende para além do ano (previne aborrecimento)
```

## Estratégia RAFT (Produtos Diferenciados)

**Alunos demonstram aprendizagem em formatos diferentes**

### Componentes RAFT

**R**ole (Papel): Quem és tu?
**A**udience (Audiência): Para quem estás a escrever/criar?
**F**ormat (Formato): Que forma vai ter?
**T**opic (Tópico): Qual é o assunto?

**Exemplo: Unidade do sistema solar**

**Opção 1** (Verbal/Linguística):
```
Papel: Astronauta
Audiência: Cidadãos da Terra
Formato: Entrada de diário
Tópico: Descrever visita a Marte
Ficha: modelo de entrada de diário (com sugestões)
```

**Opção 2** (Visual/Espacial):
```
Papel: Cientista da NASA
Audiência: Estudantes
Formato: Diagrama legendado
Tópico: Explicar tamanhos dos planetas comparados à Terra
Ficha: modelo de diagrama com legendas para completar
```

**Opção 3** (Lógica/Matemática):
```
Papel: Engenheiro espacial
Audiência: Controlo de missão
Formato: Tabela de dados
Tópico: Calcular tempos de viagem para cada planeta
Ficha: modelo de tabela com espaço para cálculos
```

**Opção 4** (Interpessoal):
```
Papel: Guia turístico
Audiência: Turistas espaciais
Formato: Brochura
Tópico: Criar uma brochura "Visite Saturno"
Ficha: modelo de brochura (esboço de tríptico)
```

**Todas as opções**: abordam o mesmo conteúdo (conhecimento do sistema solar)
**Diferenciação**: formato corresponde à força/interesse do aluno

## Diferenciação por Prontidão

**Ajustar com base em onde os alunos estão AGORA**

### Organização por Taxonomia de Bloom

**Mesmo conteúdo, diferentes níveis de pensamento**

**Tópico: História "A Teia de Carlota"**

**Recordar/Compreender** (Nível 1):
```
Questões da ficha:
1. Quem são os personagens principais?
2. Onde se passa a história?
3. Que problema enfrenta o Wilbur?
4. Como é que a Carlota ajuda o Wilbur?

Nível cognitivo: recordação e compreensão
Para: alunos ainda a construir compreensão básica
```

**Aplicar/Analisar** (Nível 2):
```
Questões da ficha:
1. Compara o Wilbur no início e no fim da história
2. Porque escolheu a Carlota as palavras que teceu?
3. O que aconteceria se a Fern nunca visitasse o celeiro?
4. Como é que a amizade muda os personagens?

Nível cognitivo: aplicação e análise
Para: alunos preparados para pensamento mais profundo
```

**Avaliar/Criar** (Nível 3):
```
Questões da ficha:
1. Avaliar: Foi o plano da Carlota a melhor solução? O que mais poderia ter salvado o Wilbur?
2. Criar: Escreve um final alternativo onde o Wilbur ajuda a Carlota
3. Julgar: Que personagem mostrou mais crescimento? Defende a tua resposta com evidências
4. Desenhar: Se esta história continuasse, que desafios enfrentariam os personagens?

Nível cognitivo: avaliação e criação
Para: alunos preparados para pensamento de nível superior
```

## Sistemas de Gestão para Diferenciação

**A organização é fundamental para diferenciação bem-sucedida**

### Caixas Codificadas por Cores

**Sistema visual** (alunos autosselecionam):
```
Estação de matemática:
- Caixa azul: modificado (menos problemas, mais apoio)
- Caixa verde: padrão (nível do ano)
- Caixa vermelha: desafio (enriquecimento)

Alunos sabem: pegar ficha do seu nível
Professor sabe: que alunos precisam de que nível (observação ao longo do tempo)
Ajuste: alunos podem mudar de nível à medida que crescem (não são fixos)

Tempo de preparação: 10 minutos para imprimir e organizar em caixas (semanal)
```

**Grelha de opções diferenciadas**
