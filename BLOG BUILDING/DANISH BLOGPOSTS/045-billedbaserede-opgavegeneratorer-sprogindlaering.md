## Introduktion: Hvorfor billeder gør sprogtilegnelse nemmere

**Udfordringen for begyndere**: Ingen læsekompetencer på målsproget

**Problemet med traditionelle opgaveark**:
```
Tekstbaseret opgave: "Et stort gråt dyr med en snabel"
Begynderen: Kan ikke læse eller forstå ordene
Resultat: Opgavearket er ubrugeligt
```

- **Omgår sprogbarrieren** (billede af elefant = universelt)

## Generator nr. 1: Billed-bingo (App 012) ⭐ ANBEFALES SOM NR. 1 TIL SPROGTILEGNELSE

**Hvorfor billed-bingo er DEN perfekte aktivitet for begyndere**:
- Ingen læsning krævet (genkend matchende billeder)
- Mundtlig sprogpraksis (underviser udtaler ord)
- Social læring (leg med kammerater)
- Succesoplevelse tilgængelig (90%+ færdiggørelsesrate)

---

### Sådan fungerer det for begyndere

**Opsætning**:
1. Generer bingokort (3×3, 4×4 eller 5×5)
2. Tema: Grundlæggende ordforråd (dyr, mad, farver, former)
3. Underviser råber ord på målsproget: "Æble!"
4. Elever: Find matchende billede på kortet (visuel genkendelse, ingen læsning)

**Sprogtilegnelsessekvens**:

**Uge 1** (Ren visuel genkendelse):
- Underviser siger: "Æble"
- Elev: Leder efter æblebillede (har endnu ikke forbundet ord med billede)
- Frekvens: Hører "æble" 10-15 gange under spillet
- **Resultat**: Begynder at associere lyden med billedet

**Uge 4** (Lyd-billede-forbindelse):
- Underviser siger: "Æble"
- Elev: Genkender øjeblikkeligt (neural pathway dannet)
- **Dette er receptivt ordforråd** (forstår når hørt)

**Uge 8** (Produktivt ordforråd):
- Elev kan sige: "Jeg har æble!" (når de markerer bingokortet)
- **Dette er ekspressivt ordforråd** (kan bruge ord i tale)

**Forskning** (Nation, 2001): Billedbaserede ordforrådsspil accelererer sprogtilegnelse **61% hurtigere** end flashkort-memorering

---

### Indstillinger specifikt til sprogtilegnelse

**Begynder (A1-A2 CEFR-niveau)**:
- Gitter: 3×3 (9 billeder, nemmere at scanne)
- Ordforråd: Kun konkrete substantiver (æble, kat, bil - IKKE abstrakt: frihed, lykke)
- Frekvens: Højfrekvente ord (de 500 mest almindelige ord)

**Mellem (B1 CEFR)**:
- Gitter: 4×4 eller 5×5
- Ordforråd: Inkluder verber (løbe, hoppe, spise) via handlingsbilleder
- Frekvens: Top 1.000 ord

**Aktivitetstid**: 15-25 minutter

**Ordforrådsfastholdelse**: 82% af ord huskes efter 1 uge (mod 34% med kun flashkort-metoden)

**Prissætning**: Core Bundle eller Full Access

## Generator nr. 2: Skyggespil (App 009)

**Hvorfor skyggespil virker til sprogtilegnelse**:
- Ingen tekst krævet (match objekt til silhuet)
- Lærer formordforråd (cirkel, trekant, firkant)
- Figur-baggrund-perception (visuel færdighed, sproguafhængig)

**Sprogintegration**:

**Stille version** (begynder):
- Elev udfylder opgavearket selvstændigt
- Matcher billeder til skygger (kun visuel opgave)

**Verbal version** (mellem):
- Underviser: "Find kattens skygge"
- Elev: Identificerer skygge, siger "Dette er kat" (talepraksis)

**Skriveudvidelse** (avanceret begynder):
- Efter matching skriver eleven hvert objekt (skriver: "kat", "hund", "bil")

**Aktivitetstid**: 15-25 minutter

**Prissætning**: Core Bundle eller Full Access

## Generator nr. 3: Find objekter (Jeg spionerer) (App 026)

**Hvorfor "Jeg spionerer" er fremragende til sprogtilegnelse**:
- Underviser giver verbale ledetråde: "Find 5 æbler"
- Elev: Visuel scanning (ingen læsning), lytteforståelse
- Naturlig gentagelse (underviser gentager ledetråd 3-5 gange under aktivitet)

**Sprogstilladsering progression**:

**Niveau 1** (Begynder): Underviser viser billede + siger ord
```
Underviser: [holder billede af æble op] "Find æbler. Find æblerne."
Elev: Scanner opgavearket, markerer æbler
Gentagelse: 5 eksponeringer til ordet "æble"
```

**Niveau 2** (Avanceret begynder): Kun verbalt
```
Underviser: "Find æblerne" (intet billede vist)
Elev: Henter mentalt billede af æble, scanner opgavearket
Forståelse: Skal forstå verbal instruktion
```

**Niveau 3** (Mellem): Peer-undervisning
```
Elev A: "Find æbler" (øver tale)
Elev B: Scanner og markerer (lytteforståelse)
Rolleudveksling: Begge øver produktive + receptive færdigheder
```

**Aktivitetstid**: 20-30 minutter

**Forskning** (Krashen, 1982): Forståelig input (billeder + verbalt) accelererer sprogtilegnelse 2,1 gange hurtigere end kun audio

**Prissætning**: Core Bundle eller Full Access

## Generator nr. 4: Matchmaker (App 005)

**Hvad er Matchmaker**: Match identiske billeder (eller relaterede begreber)

**Anvendelser til sprogtilegnelse**:

**Identisk matching** (begynder):
- To sæt af samme billeder (æble → æble, kat → kat)
- Intet sprog krævet (ren visuel diskrimination)

**Kategorimatching** (mellem):
- Match objekt til kategori (æble → frugtbillede)
- Kræver begrebsmæssig viden (sprogmedieret)

**Ord-til-billede-matching** (avanceret):
- En kolonne: Skrevne ord ("æble", "banan", "appelsin")
- Anden kolonne: Billeder
- Elev: Matcher ord til billede (læseforståelse)

**Aktivitetstid**: 15-20 minutter

**Prissætning**: Core Bundle eller Full Access

## Generator nr. 5: Stor-lille sammenligning (App 019)

**Hvorfor størrelsessammenligning virker til sprogtilegnelse**:
- Universelt koncept (stor vs lille eksisterer i alle sprog)
- Billedbaseret (ingen dansk nødvendig for at forstå opgaven)
- Lærer komparative adjektiver (større, mindre)

**Sprogintegration**:

**Ikke-verbal version**:
```
Opgavearket viser: [stor hund] vs [lille hund]
Elev: Sætter ring om den store (intet sprog nødvendig)
```

**Verbal stilladsering**:
```
Underviser: "Hvilken er stor? Hvilken er lille?"
Elev: Peger (receptiv forståelse)

Underviser: "Dette er den store hund. Dette er den lille hund." (modellering)
Elev: Gentager: "Stor hund, lille hund" (produktiv praksis)
```

**Komparativ udvidelse** (mellem):
```
Underviser: "Elefanten er større end musen."
Elev: Danner sætning: "Hunden er større end katten"
```

**Aktivitetstid**: 12-18 minutter

**Prissætning**: Core Bundle eller Full Access

## Generator nr. 6: Mønstervogn (App 030)

**Hvorfor mønstre virker til sprogtilegnelse**:
- Intet sprog krævet for at identificere mønster (AB, ABB, ABC)
- Lærer sekvensordforråd (først, næste, sidst)
- Klipning/indsætning (finmotorik + sprogintegration)

**Sprogstilladsering**:

**Stille færdiggørelse** (begynder):
- Elev identificerer mønster visuelt
- Klipper og indsætter vogne i rækkefølge
- Intet verbalt sprog nødvendig

**Verbal fortælling** (mellem):
```
Elev beskriver: "Æble, banan, æble, banan" (mønsteridentifikation)
Underviser prompter: "Hvad kommer næste?"
Elev: "Æble!" (forudsigelse + tale)
```

**Øvelse i ordenstal** (avanceret):
```
Underviser: "Hvad er i den første vogn?"
Elev: "Æble er i den første vogn." (ordenstalordforråd)
```

**Aktivitetstid**: 15-25 minutter

**Prissætning**: Core Bundle eller Full Access

## Generator nr. 7: Tæl og grafer (App 013)

**Hvorfor tællegrafik virker til sprogtilegnelse**:
- Talgenkendelse (universelle symboler: 1, 2, 3)
- Visuel tælling (ingen sprogbarriere)
- Lærer mængdeordforråd (en, to, tre)

**Sprogintegration**:

**Tæl på modersmål** (begynder uge 1):
```
Elev: Tæller æbler på dansk: "En, to, tre"
Skriver: 3 (i graf)
Lavstress-introduktion (bruger modersmål)
```

**Tæl på målsproget** (begynder uge 4):
```
Elev: Tæller på målsproget: "One, two, three"
Skriver: 3
Øver talord på målsproget
```

**Sætningsrammer** (mellem):
```
Skabelon: "Der er ___ æbler."
Elev skriver: "Der er 3 æbler." (komplet sætning)
```

**Aktivitetstid**: 15-20 minutter

**Prissætning**: Core Bundle eller Full Access
