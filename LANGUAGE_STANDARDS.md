## üéØ TRANSLATION PHILOSOPHY

### Core Principles

1. **Native Feel First**
   - Translations should sound like the app was originally written in that language
   - Avoid word-for-word translations that sound unnatural
   - Use idioms and expressions native speakers would use

2. **Educational Context Matters**
   - This is educational software for teachers and publishers
   - Use established educational terminology, not generic tech terms
   - Consider the pedagogical appropriateness of language

3. **Cultural Sensitivity**
   - Respect regional conventions (paper sizes, date formats, measurement systems)
   - Consider cultural differences in educational systems
   - Use culturally appropriate examples

4. **Clarity Over Brevity**
   - In educational software, clarity is more important than saving space
   - Better to use a longer, clearer phrase than a short ambiguous one
   - However, still respect UI space constraints

5. **Consistency is King**
   - Use the same translation for the same concept across ALL apps
   - Reference this glossary for every translation decision
   - Document any deviations with rationale

6. **Professional Quality**
   - No machine translation feel
   - Proper grammar, spelling, and punctuation
   - Sound like a professional educational software company

- **Decimal Separator**: Use comma (,) not period (.) for decimals

### When to Use This Glossary
1. **Always** consult this glossary before translating any text
2. **Document** any deviations in the app's translation rationale file
3. **Update** this glossary when introducing new terms
4. **Reference** existing translations for consistency

### When to Deviate
Sometimes context requires a different translation than the standard glossary term:
- **Document why** in the `translations/{app}/{APP}-{LANG}-TRANSLATION-RATIONALE.md` file
- **Consider if** the new translation should become the standard (update glossary)
- **Ensure consistency** - if you deviate once, deviate the same way everywhere in that app

### Maintaining This Document
- This is a **living document** - update as needed
- When adding new terms, add to ALL 11 languages
- Get native speaker confirmation for new terms when possible
- Keep this file in sync with actual translations
