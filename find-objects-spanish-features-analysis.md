## Section 5: Object Selection

### **11 Language Support for Image Library:**
- English, German, French, Spanish, Portuguese, Italian, Dutch, Swedish, Danish, Norwegian, Finnish
- Language selector at top of sidebar
- Image filenames match selected language

### Activity Mode Selection:

#### **Mode 1: I Spy - Find Hidden Objects** (default)

**Distractors (Select 8–12):**
- Manual selection from image library OR
- **Distractor Theme Select** dropdown (choose theme to auto-fill)
- Displays selected distractors

**Hidden Objects (Select 1–5):**
- Manual selection from image library OR
- **Hidden Object Theme Select** dropdown (choose theme to auto-fill)
- Displays selected hidden objects

#### **Mode 2: Odd One Out - Find Unpaired Images**

**Pairs:**
- **Number of Pairs** dropdown:
  - 8 pairs (16 images)
  - 9 pairs (18 images)
  - 10 pairs (20 images) - default
  - 11 pairs (22 images)
  - 12 pairs (24 images)
- **Pairs Theme Select** dropdown OR manual selection
- Displays selected pairs

**Odd Images (Unpaired):**
- **Number of Odd Images** dropdown:
  - 1 odd image
  - 2 odd images - default
  - 3 odd images
- **Odd Images Theme Select** dropdown OR manual selection
- Displays selected odd images

## Canvas Editing Features

### Context Toolbar (appears when object selected):

**Layer Controls:**
- Bring to Front
- Bring Forward
- Send Backward
- Send to Back

**Alignment Tools:**
- Align Left/Center/Right
- Align Top/Center/Bottom
- Center on Page Horizontally
- Center on Page Vertically

**Lock/Unlock:**
- Lock individual objects
- Unlock All button (appears when objects locked)

**Delete:**
- Delete selected objects

### Tab Row:
- **Worksheet Tab** (main canvas)
- **Answer Key Tab** (shows solution)

### Zoom Controls:
- Zoom In
- Zoom Out
- Reset Zoom
- Zoom Percentage display

### History Controls:
- Undo (Ctrl+Z)

### Generate Dropdown:
- **New Worksheet** - generates new random worksheet

### Download Dropdown:
- Multiple download options available
- PDF and JPEG formats
