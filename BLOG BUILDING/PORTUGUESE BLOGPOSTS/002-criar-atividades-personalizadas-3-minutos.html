Passo 2: Gerar Automaticamente (3-10 segundos)

Clique em "Gerar Atividade"

O que acontece (algoritmo funciona em 3 segundos):

  1. Embaralhamento Fisher-Yates: Embaralha cada palavra com zero vies
  2. Deteccao de colisao: Garante que embaralhado seja diferente do original (forca pelo menos 1 letra movida)
  3. Calculadora de pistas fracionarias: Revela floor(comprimento_palavra / 2) letras para modo Facil
  4. Correspondencia de imagens: Busca imagens climaticas da biblioteca de 3.000+ imagens
  5. Motor de layout: Organiza elementos em layout retrato de 2 colunas
  6. Sistema de canvas duplo: Cria atividade + gabarito simultaneamente
Tempo de processamento: 1,8 segundos para 6 palavras

O Que Voce Pode Editar (Pos-Geracao)

1. Mover Elementos

Como: Clique em qualquer texto/imagem, arraste para nova posicao

Caso de uso: Aluno com disturbio de processamento visual precisa de mais espacamento

  • Selecione as palavras embaralhadas
  • Arraste mais distante (reduza o amontoamento visual)
  • Tempo: 15 segundos

2. Redimensionar Texto

Como: Selecione texto Ajuste o controle deslizante de tamanho de fonte (12-48px)

Caso de uso: PEI exige fonte 150% maior para dislexia

  • Selecione todo o texto (Ctrl+A)
  • Aumente de 18px para 27px (1,5x)
  • Tempo: 10 segundos

Conformidade com PEI: Concluida em 10 segundos vs 30 minutos recriando no Word

3. Girar Elementos

Como: Selecione elemento Clique na alca de rotacao, arraste

Caso de uso: Adicionar variedade visual para engajamento

  • Gire imagens 5-10 graus (visual ludico, nao institucional)
  • Tempo: 5 segundos por imagem

4. Editar Texto

Como: Clique duas vezes em qualquer texto, digite alteracoes

Casos de uso:

  • Corrigir erro de digitacao ("relapago" "relampago") - 5 segundos
  • Mudar titulo ("Embaralhamento de Palavras" "Desafio das Palavras do Clima") - 10 segundos
  • Adicionar campo de nome ("Nome: _______") - 15 segundos

5. Deletar Elementos

Como: Selecione elemento Pressione Delete OU clique no icone da lixeira

Caso de uso: Simplificar para aluno com TDAH (reduzir desordem visual)

  • Delete borda decorativa
  • Delete 2 imagens (mantenha as 4 essenciais)
  • Tempo: 10 segundos

6. Trocar Imagens

Como: Clique na imagem Botao "Alterar Imagem" Selecione da biblioteca OU faca upload personalizado

Caso de uso: Substitua arco-iris generico por foto da excursao

  • Clique na imagem do arco-iris
  • Clique em "Fazer Upload de Imagem Personalizada"
  • Selecione foto da visita a estacao meteorologica de ontem
  • Tempo: 15 segundos
Engajamento do aluno: 3x maior com suas proprias fotos

7. Adicionar Elementos

Como: Clique no botao "Adicionar Texto" ou "Adicionar Imagem"

Casos de uso:

  • Adicionar texto de instrucao: "Circule cada palavra quando encontra-la"
  • Adicionar desafio bonus: "Use cada palavra em uma frase no verso"
  • Adicionar campo de nome no topo
  • Tempo: 15-30 segundos

8. Ajustar Layouts

Como: Alterne entre 1 coluna e 2 colunas, ou retrato paisagem

Caso de uso: Aluno tem dificuldade com varredura visual entre 2 colunas

  • Mude para layout de coluna unica (fluxo vertical)
  • Reduz movimento dos olhos esquerda-direita
  • Tempo: 5 segundos (um clique)

O Sistema de Desfazer/Refazer com 20 Estados

Cometeu um erro? Mudou de ideia?

  • Ctrl+Z: Desfazer (20 estados para tras)
  • Ctrl+Y: Refazer (20 estados para frente)

Cada acao rastreada: mover, redimensionar, deletar, adicionar, girar, edicoes de texto

Beneficio

Experimente sem medo. Tente mover 3 imagens, nao gostou, desfaca 3 vezes.

Tempo Decorrido: 2 minutos e 30 segundos no total
