## Johdanto: Kiinteä vai kehittyvä ajattelutapa

**Carol Dweckin tutkimus** (2006): Oppilaan asenne älykkyyttä kohtaan vaikuttaa dramaattisesti oppimistuloksiin

**Kiinteä ajattelutapa**:
```
Uskomus: "Älykkyys on kiinteä – olet joko älykäs tai et"

Kohdatessaan haasteita:
Oppilaan ajatus: "Tämä on vaikeaa = en ole tarpeeksi älykäs"
Reaktio: Luovuttaa nopeasti (välttää näyttämästä tyhmältä)
Tulos: Rajallinen kasvu (välttelee haasteita)
```

**Kasvuajattelutapa**:
```
Uskomus: "Älykkyys kehittyy ponnistelemisen ja harjoittelun kautta"

Kohdatessaan haasteita:
Oppilaan ajatus: "Tämä on vaikeaa = minun täytyy yrittää kovemmin"
Reaktio: Ponnistella vaikeuksista huolimatta (aivot ovat kuin lihas)
Tulos: Jatkuva kasvu (hyväksyy haasteet)
```

**Tutkimus** (Dweck, 2006): Kasvuajattelutapaa omaavat oppilaat osoittavat **0,3-0,5 standardipoikkeaman** parempia oppimistuloksia kuin kiinteää ajattelutapaa omaavat ikätoverinsa (vastaa useita kuukausia lisäoppimista)

**Luokkahuoneen ongelma**: Monella oppilaalla on kiinteä ajattelutapa
```
Oppilas näkee vaikean työarkin: "En osaa tehdä tätä"
Opettaja: "Yritä vaan!"
Oppilas: Kieltäytyy, sulkeutuu
Tulos: Oppimista ei tapahdu
```

**Ratkaisu**: Rakenna kasvuajattelutapaa järjestelmällisesti haasteprogression avulla

## Haasteprogressioiden malli

**Periaate**: Aloita helposta (rakenna itseluottamusta), lisää vaikeutta vähitellen (rakenna sinnikkyyttä)

### Viikko viikolta -haasteportaat

**Esimerkki: Sudoku-progressio (8 viikkoa)**

**Viikko 1: 4×4 Sudoku**
```
Ruudukon koko: 4×4 (helpoin)
Annetut vihjeet: 10 (paljon vinkkejä)
Odotettu aika: 8-10 minuuttia
Onnistumisprosentti: 95%+ (melkein kaikki onnistuvat)

Oppilaan kokemus: "Onnistuin! Sudoku on hauskaa!"
Kasvuajattelutavan viesti: VOIT ratkaista pulmia (rakenna itseluottamusta)
```

**Viikko 2: 4×4 Sudoku (vähemmän vihjeitä)**
```
Ruudukon koko: 4×4 (sama koko)
Annetut vihjeet: 8 (vähemmän vinkkejä, hieman vaikeampi)
Odotettu aika: 10-12 minuuttia
Onnistumisprosentti: 85% (useimmat onnistuvat, jotkut kamppailevat)

Oppilaan kokemus: "Vaikeampi kuin viime viikolla, mutta keksin sen!"
Kasvuajattelutavan viesti: Ponnistelu johtaa menestykseen (varhaista sinnikkyyden harjoittelua)
```

**Viikko 3: 6×6 Sudoku**
```
Ruudukon koko: 6×6 (isompi ruudukko, uusi haaste)
Annetut vihjeet: 18
Odotettu aika: 15-18 minuuttia
Onnistumisprosentti: 75% (huomattavaa kamppailua joillekin)

Oppilaan kokemus: "Tässä meni pidempään, mutta en luovuttanut!"
Kasvuajattelutavan viesti: Sinnikkyyys vaikeuksissa (tuottavaa ponnistelemista)
```

**Viikko 4: 6×6 Sudoku (vähemmän vihjeitä)**
```
Ruudukon koko: 6×6
Annetut vihjeet: 15
Odotettu aika: 18-22 minuuttia
Onnistumisprosentti: 65% (merkittävä haaste)

Oppilaan kokemus: "Tein virheitä, mutta pyyhkin ja yritin uudelleen!"
Kasvuajattelutavan viesti: Virheet = oppimismahdollisuuksia
```

**Viikot 5-8: Asteittainen eteneminen 9×9:ään**
```
Viikko 5: 9×9 monilla vihjeillä (35+)
Viikko 6: 9×9 kohtalaisilla vihjeillä (30)
Viikko 7: 9×9 vähemmillä vihjeillä (25)
Viikko 8: 9×9 tavallinen vaikeus (25 vihjettä, monimutkainen)

Viikolla 8:
Onnistumisprosentti: 50-60% (haastavaa mutta saavutettavissa)
Oppilaan kokemus: "Tämä on vaikeaa, mutta tiedän että osaan vaikeita asioita!"
Kasvuajattelutapa sisäistetty: Haasteet = mahdollisuuksia kasvaa
```

**Avainperiaate**: Älä koskaan hyppää viikosta 1 viikkoon 8 (portaiden on oltava asteittaiset)

## Tuottava ponnisteleminen vs turhautuminen

**Kriittinen ero**: Jonkin verran ponnistelemista rakentaa resilienssiä, liikaa aiheuttaa luovuttamisen

**Tuottava ponnisteleminen** (toivottavaa):
```
Piirteet:
- Oppilas sitoutunut (kokeilee strategioita)
- Näkyvää ponnistelua (pyyhkii, yrittää uudelleen)
- Tunnetila: Keskittynyt (ei järkyttynyt)
- Ajankäyttö: 15-30 minuuttia (kestävää)
- Lopputulos: Lopulta onnistuu TAI pyytää apua sopivasti

Esimerkki: Oppilas työskentelee ristikoiden parissa 20 minuuttia, juuttuu 2 sanaan, kysyy kaverolta
Tulos: Kasvu (harjoitteli sinnikkyyttä + itsepuolustusta)
```

**Tuhoisa turhautuminen** (vältettävä):
```
Piirteet:
- Oppilas ei sitoutunut (pää alhaalla, ei yritä)
- Ei näkyvää ponnistelua (tuijottaa tyhjänä)
- Tunnetila: Järkyttynyt, vihainen, itkee
- Aika: Luovuttaa 2-3 minuutissa
- Lopputulos: Kieltäytyy jatkamasta

Esimerkki: Oppilas katsoo kryptogrammia, sanoo "Tämä on mahdotonta", lopettaa
Tulos: Kiinteä ajattelutapa vahvistui ("En osaa tätä")
```

**Opettajan rooli**: Seurata ja puuttua ENNEN kuin turhautuminen muuttuu tuhoisaksi

## 80% onnistumisen sääntö

**Optimaalinen haastetaso** (Vygotskin lähikehityksen vyöhyke):
```
Liian helppoa (95%+ onnistuminen): Ei kasvua (tylsää)
Optimaalinen (70-85% onnistuminen): Maksimaalinen kasvu (haastava mutta saavutettavissa)
Liian vaikeaa (50% onnistuminen): Turhautuminen (ylivoimaista)
```

**Sovellus työarkkeihiin**:

**Maanantai**: Esittele uusi taito (opettajajohtoinen, 90% onnistumisen odotus)
```
Työarkki: 20 tehtävää, paljon tukea
Oppilaan onnistuminen: 18/20 oikein (90%)
Tarkoitus: Rakenna itseluottamusta uuden taidon kanssa
```

**Tiistai-keskiviikko**: Harjoittele (80-85% onnistuminen)
```
Työarkki: 20 tehtävää, kohtalainen vaikeus
Oppilaan onnistuminen: 16-17/20 oikein (80-85%)
Tarkoitus: Vahvista oppimista (optimaalinen haastavuusvyöhyke)
```

**Torstai**: Haaste (70-75% onnistuminen)
```
Työarkki: 20 tehtävää, monimutkaisia sovelluksia
Oppilaan onnistuminen: 14-15/20 oikein (70-75%)
Tarkoitus: Venytä ajattelua (tuottavaa ponnistelemista)
```

**Perjantai**: Arviointi (75-80% odotettu)
```
Työarkki: 25 tehtävää, vaihteleva vaikeus
Luokan keskiarvo: 19/25 oikein (76%)
Tarkoitus: Osoita kasvua viikon aikana
```

**Tulos**: Oppilaat kokevat sekä onnistumisia ETTÄ haasteita (tasapainoinen ruokavalio)

## Virheiden juhlimisen protokolla

**Perinteinen reaktio virheisiin**:
```
Oppilas tekee tehtävän väärin
Opettaja: Merkitsee X:llä (punaisella kynällä)
Oppilaan tunne: Häpeä (epäonnistuin)
Kiinteä ajattelutapa: "Olen huono matematiikassa"
```

**Kasvuajattelutavan reaktio**:
```
Oppilas tekee tehtävän väärin
Opettaja: "Näytä ajattelusi – selvitetään mitä tapahtui!"
Oppilas: Selittää prosessin
Opettaja: "Ymmärrän! Ymmärsit vaiheet 1 ja 2, mutta vaihe 3 kaipaa säätöä. Yritä uudelleen!"
Oppilaan tunne: Oppimismahdollisuus (ei epäonnistuminen)
Kasvuajattelutapa: "Virheet auttavat minua oppimaan"
```

### "Suosikkivirhe"-strategia

**Viikoittainen rutiini**:

**Perjantain jakaminen** (10 minuuttia):
```
Opettaja: "Mikä oli tällä viikolla suosikkivirheesi – sellainen, joka opetti sinulle jotain?"

Oppilas A: "Laskemani jatkuvasti osoittajat JA nimittäjät yhteen murtoluvuissa. Sitten tajusin, että nimittäjät pysyvät samana! Se virhe auttoi minua ymmärtämään murtolukuja paremmin."

Oppilas B: "Kiiruhdim kertolaskun läpi ja tein huolimattomuusvirheitä. Opin hidastamaan ja tarkastamaan työni."

Oppilas C: "En ymmärtänyt kryptogrammeja ennen kuin tein virheen olettamalla, että 'Q' on aina 'A'. Sitten opin kirjainten yleisyydestä!"

Luokka: Suosionosoitukset jokaiselle jaetulle virheelle

Kasvuajattelutavan viesti: Virheet ovat arvokkaita oppimishetkiä (eivät epäonnistumisia)
```

### Virheenseurantaarkki

**Oppilaan työarkki**:
```
Oppimispäiväkirjani

Tehtävä jonka tein väärin: 47 + 28 = 65
Mitä tein: Laskin 7+8=15, kirjoitin 5, sitten laskin 4+2=6
Mikä meni pieleen: Unohdin ryhmitellä 1:n
Mitä opin: Minun täytyy siirtää kymmenen seuraavaan sarakkeeseen
Nyt ymmärrän: [oppilas piirtää ryhmittäymismallin]

Kasvuajattelutavan reflektio:
Ennen tätä virhettä en ymmärtänyt ryhmittämistä. Nyt ymmärrän!
Tämä virhe teki minusta älykkäämmän. ✓
```

**Hyöty**: Metakognitio (ajatteleminen ajattelemisesta) + virheiden normalisointi

## Ponnistelun ylistäminen vs kyvyn ylistäminen

**Tutkimus** (Dweck, 2006): MITEN kehtaat vaikuttaa ajattelutavan kehitykseen

**Kyvyn ylistäminen** (heikentää kasvuajattelutapaa):
```
Oppilas ratkaisee pulman
Opettaja: "Vau, olet niin älykäs!"

Vastaanotettu viesti: Onnistuminen = älykkyyden oleminen (synnynnäinen)
Seuraava haaste: Oppilas huolissaan (Entä jos en osaa ratkaista sitä? Ehkä en olekaan älykäs?)
Tulos: Välttelee haasteita (suojaa "älykäs"-leimaa)
```

**Ponnistelun ylistäminen** (rakentaa kasvuajattelutapaa):
```
Oppilas ratkaisee pulman
Opettaja: "Vau, sinnittelit tämän haasteen läpi! Näin sinun kokeilevan kolmea eri strategiaa!"

Vastaanotettu viesti: Onnistuminen = ponnistelu ja strategia
Seuraava haaste: Oppilas itsevarma (Jos työskentelen kovasti, voin selvittää sen)
Tulos: Omaksuu haasteet (ponnistelu on polku)
```

```
❌ Älä sano: "Se on okei, matematiikka ei ole kaikille"
✅ Sano: "Näen sinun pyyhkivän ja yrittävän uudelleen – juuri näin oppiminen tapahtuu!"
```

## "Vielä"-strategia

**Kiinteän ajattelutavan kieli**: "En osaa tehdä tätä"

**Kasvuajattelutavan kieli**: "En osaa tehdä tätä VIELÄ"

**Opettajan mallinnus**:
```
Oppilas: "En osaa ratkaista Sudokua"
Opettaja: "Et osaa ratkaista Sudokua VIELÄ. Osaat harjoittelemalla!"

Oppilas: "En ymmärrä murtolukuja"
Opettaja: "Et ymmärrä murtolukuja VIELÄ. Työskentelemme sen parissa yhdessä!"

Sana "vielä": Viittaa siihen, että oppiminen on mahdollista (tulevaisuuteen suuntautunut)
```

**Luokkahuoneen juliste**:
```
VIELÄn voima

En osaa tehdä tätä... VIELÄ
En ymmärrä... VIELÄ
En ole hyvä tässä... VIELÄ

[Kuva: Aivot joissa "kasvuvyöhyke" laajenee]

Joka kerta kun harjoittelet, aivosi vahvistuvat!
```

## Eriytetyt haastevaihtoehdot

**Ongelma**: Yksi haastetaso ei sovi kaikille oppilaille

**Ratkaisu**: Porrastettu haastavalikko

**Maanantain haastavalikko**:
```
Valitse YKSI haaste tänään:

TASO 1: Aloittelija (Taitojen rakentaminen)
- 4×4 Sudoku (10 min)
- Sanahaku 10 sanalla (12 min)
- Matematiikkatehtävä, 15 tehtävää (15 min)

TASO 2: Tutkimusmatkailija (Kohtalainen haaste)
- 6×6 Sudoku (20 min)
- Ristikko, 15 sanaa (20 min)
- Matematiikkapulmat, 2 tuntematonta (20 min)

TASO 3: Mestari (Edistynyt haaste)
- 9×9 Sudoku (40 min)
- Kryptogrammi, 80 kirjainta (45 min)
- Matematiikkapulmat, 4 tuntematonta (40 min)

SÄÄNTÖ: Voit valita MINKÄ TAHANSA tason!
Kasvuajattelutapa: Valitse missä SINÄ tarvitset kasvua (ei opettajan määräämä)
```

**Oppilaan reflektio suorittamisen jälkeen**:
```
Tänään valitsin: Taso ___
Se oli: ☐ Liian helppo ☐ Sopiva ☐ Liian vaikea
Seuraavalla kerralla: ☐ Sama taso ☐ Tason nousu ☐ Tason lasku

Kasvuajattelutavan kysymys: Opitko jotain tänään? ☐ Kyllä ☐ Ei
Jos kyllä, mitä? ___________
```
