## Bildende Kunst: Ausmalen & Design

**Mehr als „nur Ausmalen"**

### Achtsame Mandalas (Stressabbau + Feinmotorik)

**Aktivität**:
```
Arbeitsblatt: Komplexes Mandala-Design (druckbar)

Anleitung:
1. Farben bewusst auswählen (Farblehre-Praxis)
2. Innerhalb der Linien bleiben (feinmotorische Präzision)
3. Muster erstellen (Wiederholung, Symmetrie)
4. Aufmerksamkeit fokussieren (Achtsamkeit, Ruhe)

Dauer: 15-20 Minuten

Vorteile:
- Stressreduktion: Meditative Aktivität (beruhigt Angst)
- Feinmotorik: Handkraft, Stiftkontrolle
- Fokus: Übung für anhaltende Aufmerksamkeit
- Kreative Wahl: Farbkombinationen, Muster

Forschung: Ausmalen reduziert Angst um 12% (Curry & Kasser, 2005)
```

**Fächerübergreifende Verbindung**:
```
Mathematik: Formen im Mandala identifizieren (Kreise, Dreiecke, Quadrate)
Geometrie: Symmetrie diskutieren (Achsensymmetrie, Rotationssymmetrie)
Kultur: Ursprung der Mandalas (Hindu- und buddhistische Traditionen)
```

### Farblehre-Arbeitsblätter

**Farbbeziehungen verstehen**:
```
Primärfarben-Arbeitsblatt

Aktivität: Farbkreis-Übung

Primärfarben (können nicht gemischt werden):
- Rot [Kreis zum Ausmalen]
- Blau [Kreis zum Ausmalen]
- Gelb [Kreis zum Ausmalen]

Sekundärfarben (zwei Primärfarben mischen):
- Violett = Rot + Blau [mischen und ausmalen]
- Grün = Blau + Gelb [mischen und ausmalen]
- Orange = Rot + Gelb [mischen und ausmalen]

Tertiärfarben (Primär + Sekundär):
- 6 Tertiärfarben erstellen und benennen

Herausforderung: Farbverlauf erstellen (hell bis dunkel)

Anwendung: Farben zur Illustration von Vokabeln nutzen
- „Fröhlich" (helle Gelb-, Orangetöne)
- „Traurig" (Blau-, Grautöne)
- „Wütend" (Rot-, Schwarztöne)

Verbindung: Kunsttechnik + emotionales Vokabular
```

## Musikintegration

**Rhythmus, Muster und Klang**

### Rhythmusmuster-Arbeitsblätter

**Mathematik trifft Musik**:
```
Rhythmus verstehen = Brüche verstehen

Arbeitsblatt: Musiknotation als Mathematik

Ganze Note: ○ = 4 Schläge (4/4)
Halbe Note: ♩ = 2 Schläge (2/4)
Viertelnote: ♪ = 1 Schlag (1/4)
Achtelnote: ♫ = 1/2 Schlag (1/8)

Aktivität: Schläge addieren
♩ + ♩ = ___ Schläge (2 + 2 = 4)
♪ + ♪ + ♪ + ♪ = ___ Schläge (1+1+1+1 = 4)
○ = ♩ + ♩ (4 Schläge = 2 Schläge + 2 Schläge)

Herausforderung: Eigenes Rhythmusmuster erstellen
Ziel: 8 Gesamtschläge
Beliebige Kombination verwenden: ___________________

Ausklatschen: Rhythmus für die Klasse vorführen

Mathematische Verbindung: Brüche, Addition, Äquivalenz (○ = ♩♩ = ♪♪♪♪)
Musikalische Verbindung: Notenlesen, Rhythmus
```

## Zeichnen & Illustration

**Akademische Inhalte durch Kunst**

### Wissenschaftliche Illustration

**Naturwissenschaften + detailliertes Zeichnen**:
```
Beobachtungszeichnung: Lebenszyklus einer Pflanze

Materialien: Echte Pflanze (oder Foto), Bleistift, Arbeitsblatt

Anleitung:
1. Pflanze sorgfältig beobachten (2 Minuten)
2. Zeichnen, was Sie sehen (nicht, wie Sie denken, dass Pflanzen aussehen)
3. Teile beschriften: Wurzeln, Stängel, Blätter, Blüte
4. Details hinzufügen: Blattadern, Rindenstruktur

Genauigkeit zählt: Wissenschaftliche Zeichnungen müssen präzise sein

Entwickelte Fähigkeiten:
- Beobachtung: Kleine Details bemerken
- Feinmotorik: Kontrollierte Stiftbewegung
- Naturwissenschaft: Pflanzenanatomie verstehen
- Geduld: Sich Zeit zum sorgfältigen Zeichnen nehmen

Erweiterung: Pflanze in verschiedenen Lebensphasen zeichnen (Samen → Keimling → Erwachsen → Blüte)

Vorteil: Kunst macht Naturwissenschaft einprägsam (visuelles Gedächtnis stärker als Text)
```

### Geschichte-Illustration

**Leseverständnis + visuelle Interpretation**:
```
Text lesen, Szene zeichnen:

Textabschnitt:
„Die alte Eiche stand am Rand der Wiese. Ihre verdrehten Äste streckten sich zum Himmel wie knorrige Finger. An ihrem Fuß wuchsen Wildblumen in einem Regenbogen von Farben."

Zeichnen Sie, was Sie gelesen haben:
[Großer leerer Bereich für Illustration]

Beschriften Sie Ihre Zeichnung mit Wörtern aus dem Text:
- alte Eiche
- verdrehte Äste
- Wiese
- Wildblumen

Verständnisprüfung:
- Wie sahen die Äste aus? __________
- Wo stand der Baum? __________

Vorteil: Visualisierung (Verständnisstrategie), künstlerischer Ausdruck, Textnachweis
```

## Geometrie & Kunst-Verbindung

**Mathematik schön gemacht**

### Parkettierungen-Arbeitsblätter

**Wiederholende Muster (keine Lücken, keine Überlappungen)**:
```
Was ist eine Parkettierung?
Muster aus Formen, die eine Fläche ohne Lücken oder Überlappungen bedecken

Beispiele in der Natur:
- Bienenwabe (Sechsecke)
- Fischschuppen (Ovale)
- Schildkrötenpanzer (unregelmäßige Vielecke)

Eigene Parkettierung erstellen:

Schritt 1: Form wählen (Dreieck, Quadrat, Sechseck funktionieren am besten)
Schritt 2: Form transformieren (von einer Seite ausschneiden, zur gegenüberliegenden kleben)
Schritt 3: Wiederholt nachzeichnen, um Muster zu erstellen
Schritt 4: Ausmalen, um Design zu verbessern

Arbeitsblatt: Rasterpapier bereitgestellt, nachzeichnen und ausmalen

Mathematische Konzepte:
- Transformationen (Rotation, Translation, Spiegelung)
- Symmetrie
- Geometrische Formen

Künstlerverbindung: M.C. Escher (berühmter Parkettierungskünstler)
```

---

### Symmetrie-Kunst

**Spiegelbilder**:
```
Achsensymmetrisches Zeichnen

Arbeitsblatt: Halbiertes Bild bereitgestellt (Schmetterlingsflügel, Gesicht, Baum)

Aufgabe: Andere Hälfte vervollständigen (Spiegelbild)

Schwierigkeitsstufen:
Leicht: Einfache Formen, klares Raster
Mittel: Detailliertes Bild, kleines Raster
Schwer: Komplexe Kurven, kein Raster

Mathematische Fähigkeit: Symmetrie verstehen (Spiegelachse)
Künstlerische Fähigkeit: Präzises Zeichnen, Detailgenauigkeit

Erweiterung: Symmetriebeispiele in der Natur finden (fotografieren und Symmetrieachse markieren)
```
