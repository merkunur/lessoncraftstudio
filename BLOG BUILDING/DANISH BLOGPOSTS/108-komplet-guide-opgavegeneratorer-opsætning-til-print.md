## Trin 4: Tilpas Dit Arbejdsark

**Indstil parametre**

### Ordsøgning Eksempel Gennemgang

**Tilpasningsmuligheder**:

**1. Indtast ordforrådsord**:
```
Inputfelt: Skriv eller indsæt ord

Eksempel (naturfag-emne):
- fotosyntese
- klorofyl
- kuldioxid
- ilt
- sollys
- vand
- energi
- glukose

Tip: Ét ord per linje, eller kommasepareret
```

---

**2. Vælg gitterstørrelse**:
```
Muligheder:
- 10×10: Let (yngre elever, simple ord)
- 15×15: Mellem (standard sværhedsgrad)
- 20×20: Svært (ældre elever, flere ord)

Anbefaling: Start med 15×15 (juster hvis for let/svært)
```

---

**3. Indstil sværhedsgrad**:
```
Let: Ord kun vandret og lodret (ingen diagonaler)
Mellem: Vandret, lodret, diagonal
Svært: Alle retninger inklusive baglæns

Vælg baseret på: Elevernes klassetrin og erfaring med ordsøgning
```

---

**4. Yderligere indstillinger**:
```
☑ Inkluder facitliste (anbefalet)
☑ Inkluder ordliste (liste over ord at finde)
☐ Skjul ordliste (mere udfordrende)

Titel: "Fotosyntese Ordforråd"
Instruktioner: "Find alle 8 ord relateret til fotosyntese"
```

## Trin 5: Forhåndsvis og Juster

**Tjek før færdiggørelse**

### Realtids Forhåndsvisning

**Hvad skal gennemgås**:
```
Layout-tjek:
- Passer ord i gitter? ✓
- Er titel tydelig? ✓
- Er instruktioner læsbare? ✓
- Ser det professionelt ud? ✓

Justeringer nødvendige?
- Gitter for tæt: Forøg størrelse (10×10 → 15×15)
- For let: Forøg sværhedsgrad (mellem → svært)
- Manglende ord: Tilføj til liste, regenerer

Iterativ proces: Juster → Forhåndsvis → Juster → Forhåndsvis indtil perfekt

Tid: 30 sekunder til 2 minutter (afhængigt af tilpasninger)
```

## Trin 8: Effektiv Udskrivning

**Printerindstillinger betyder noget**

### Optimale Printerindstillinger

**Printerindstillinger**:
```
Kvalitet: Standard (ikke kladde, ikke foto)
- Kladde: For lys (svær at læse)
- Standard: Klar, bruger passende blæk
- Foto: Unødvendig (spilder blæk, langsom)

Farve: Sort-hvid (medmindre specifikt nødvendigt)
- De fleste arbejdsark: Sort-hvid tilstrækkelig
- Farvelægningssider: Farve påkrævet
- Besparelser: Sort-hvid koster 80% mindre per side

Duplex (dobbeltsidet): Når passende
- Øvelsesark: Enkeltsidet (elever skriver på)
- Læsetekster: Duplex OK (sparer papir)
- Vigtigt: Tjek hvilken side er "op" (vendeorientering betyder noget)

Papir: Standard 80g kopipapir
- Undgå karton: Dyrt, unødvendigt for de fleste arbejdsark
- Undgå genbrugspapir: Kan sommetider sidde fast, kan være mindre skarpt
```

---

### Masseudskrivningstips

**Til 30-elevers klasse**:
```
Printjob: 30 kopier af arbejdsark

Indstillinger:
1. Fyld papirbakke helt (200+ ark)
2. Vælg "Sortér" (hvis hæftning senere)
3. Vælg "Print alle på én gang" (ikke én ad gangen)
4. Send printjob
5. Gå væk (lad det køre)

Tid: 30 kopier = ~3 minutters printtid
Undgå: At stå ved printer (brug tid til andre opgaver)
```

## Avancerede Teknikker

**Løft dine generatorfærdigheder**

### Batch-generering

**Opret flere arbejdsark på én gang**:
```
Scenarie: Har brug for 5 forskellige matematikark til differentieret øvelse

Proces:
1. Generer Arbejdsark A (let: 10 opgaver)
2. Download og gem
3. Juster parametre (mellem: 15 opgaver)
4. Generer Arbejdsark B
5. Fortsæt for C, D, E

Tid: 5 arbejdsark × 1 min = 5 minutter i alt
Resultat: Hel uges differentieret øvelse klar
```

---

### Skabelonoprettelse

**Gem brugerdefinerede indstillinger**:
```
Hvis du ofte opretter lignende arbejdsark:

Eksempel: Ugentlig staveordsøgning (altid 15×15, mellem sværhedsgrad)

Proces:
1. Indstil alle præferencer (gitterstørrelse, sværhedsgrad, format)
2. Gem som skabelon (hvis tilgængelig i din plan)
3. Næste uge: Indlæs skabelon, indtast kun nye ord

Tid sparet: 30 sekunder per arbejdsark (spring parametervalg over)
```

---

### Sæsonvariationer

**Samme indhold, forskelligt tema**:
```
Færdighed: Tocifret addition (20 opgaver)

September: Æble-tema overskrift og grafik
Oktober: Græskar-tema
November: Høst-tema

Samme matematik, frisk udseende (holder elever engagerede gennem året)

Metode: Skift titel, juster grafik-tema (hvis tilgængeligt)
Tid: Tilføj 15 sekunder per arbejdsark (minimal indsats, stor engagement-effekt)
```

## Almindelige Fejl at Undgå

**Lær af andres fejl**

### Fejl 1: For Mange Parametre

**Problem**:
```
Ny bruger: Justerer hver eneste indstilling (overvældet af valg)
Resultat: Analyse-lammelse (tager 10 minutter at generere ét arbejdsark)
```

**Løsning**:
```
Start simpelt: Brug standardindstillinger først
Generer: Se resultat
Juster: Skift kun det der behøver ændring
Iterer: Udforsk gradvist flere muligheder

Første arbejdsark: 2 minutter (læring)
Femte arbejdsark: 30 sekunder (selvsikker)
```

---

### Fejl 2: Forkert Sværhedsgrad

**Problem**:
```
Lærer: Opretter "svært" arbejdsark til elever med udfordringer
Resultat: Frustration, giver op
```

**Løsning**:
```
Start lettere: Hvis usikker, gå ét niveau ned
Test: Giv til én elev først
Juster: Lettere eller sværere baseret på respons
Forfin: Find den rigtige sværhedsgrad over tid
```

---

### Fejl 3: Ikke at Holde Organiserede Filer

**Problem**:
```
Downloads: Alle går til standard "Downloads" mappe
Måneder senere: "Hvor er den fotosyntese-ordsøgning jeg lavede?"
Resultat: Genskabelse af arbejdsark unødvendigt
```

**Løsning**:
```
Med det samme: Flyt downloadet arbejdsark til passende mappe
Navngivning: Brug beskrivende navne (ikke bare "arbejdsark1.pdf")
Genbrug: Find og genbrug næste år (spar tid)
```

## Fejlfinding af Almindelige Problemer

**Hurtige løsninger**

### Problem: "Arbejdsark vil ikke genereres"

**Mulige årsager**:
```
1. For mange ord til gitterstørrelse
   Løsning: Reducer ordantal eller forøg gitterstørrelse

2. Internetforbindelse tabt
   Løsning: Tjek WiFi, genindlæs side

3. Ugyldige tegn i input
   Løsning: Fjern specialsymboler, brug standardbogstaver
```

---

### Problem: "Printkvalitet er dårlig"

**Løsninger**:
```
1. Tjek printerindstillinger (skal være "Standard" ikke "Kladde")
2. Sikr tilstrækkelige blækniveauer (udskift patron hvis lav)
3. Rens printhoved (printer vedligeholdelsesmenu)
4. Brug kvalitetspapir (billigt papir kan se sløret ud)
```

---

### Problem: "Kan ikke finde downloadet fil"

**Løsning**:
```
1. Tjek standard Downloads mappe
2. Søg computer efter PDF oprettet i dag
3. Browser: Tjek downloadhistorik (viser filplacering)
4. Fremadrettet: Gem straks til organiseret mappe
```

## Konklusion

Mestre opgavegeneratorer på **30 minutter** - opsætning (5 min) + første arbejdsark (5 min) + øvelse (20 min).

**Opsætning**: Opret konto (2 min), indstil præferencer (klassetrin, sprog, enheder), udforsk dashboard (33 generatorer organiseret efter kategori)

**Genereringsgennemgang**: Vælg generator → tilpas (indtast indhold, gitterstørrelse, sværhedsgrad) → forhåndsvis (tjek layout) → generer (2-5 sek) → download (PDF klar)

**Print effektivt**: Standard kvalitet, sort-hvid medmindre farve nødvendigt, duplex når passende, masseudskrivningsindstillinger (sortér, print alle på én gang)

**Facitlister**: Automatisk generering (afkrydsningsfelt under oprettelse), side 2 af PDF, nul ekstra tid

**Avanceret**: Batch-generering (5 arbejdsark på 5 min), skabelonoprettelse (gem brugerdefinerede indstillinger), sæsonvariationer (samme færdighed, forskelligt tema)

**Undgå fejl**: Start med standardindstillinger (juster trinvist), match sværhedsgrad til elever (start lettere), organiser filer med det samme (tab ikke arbejde)

**Fejlfinding**: Gitterstørrelse vs. ordantal, printkvalitetsindstillinger, find downloadede filer

**Support**: Tutorials, vidensbase, e-mail support (24-timers svar for Core/Full), videogennemgange

**Priser**: Gratis 0 kr. (vandmærke), Core 1.080 kr./år (lærere), Full 1.800 kr./år (kommerciel)
