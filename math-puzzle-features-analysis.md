1. **Math Puzzle Grid**: 2×2 to 4×4 grid with addition/subtraction problems
2. **11 Languages**: Full UI and content support
3. **Custom Images**: Theme library + upload your own
4. **Two Canvases**: Separate worksheet and answer key
5. **Full Editability**: Drag, scale, rotate, layer all elements
6. **Professional Downloads**: JPEG and PDF at 300 DPI
7. **Canvas Editing**: Undo/redo, zoom, alignment, locking
8. **Page Customization**: Size, color, backgrounds, borders
9. **Text Tools**: Add and customize text elements
10. **Answer Keys**: Automatic answer key generation and download
