## Introdução: O Desafio da Coordenação entre Docentes

**Modelo de co-docência**: Professor titular + professor de educação especial = sala partilhada

**Prevalência**: 65% das escolas portuguesas e brasileiras adotam modelos de ensino colaborativo (modelo de inclusão)

**Desafio de coordenação**:
```
Professor titular: Planifica currículo, cria materiais
Professor educação especial: Fornece acomodações, adaptações
Problema: Dois professores, planeamento separado → esforço duplicado OU desalinhamento
```

**Problema tradicional com fichas**:
```
Professor titular: Cria ficha nível turma (30 exercícios, complexos)
Professor educação especial: Precisa adaptar (reduzir para 15 exercícios, adicionar imagens)
Duplicação: 2× trabalho para essencialmente o mesmo conteúdo
Resultado: Ineficiência total
```

**Solução com geradores**: Uma plataforma, diferenciação instantânea

**Investigação** (Friend & Cook, 2017): Ensino colaborativo eficaz requer **planeamento partilhado** + **recursos comuns** (reduz tempo preparação 40%)

## Modelos de Co-Docência e Aplicação de Fichas Educativas

### Modelo 1: Um Ensina, Outro Apoia

**Estrutura**: Professor titular leciona, professor de apoio circula prestando suporte

**Aplicação de fichas**:
```
Professor titular: Distribui ficha matemática (20 exercícios, nível turma)
Professor de apoio: Circula, identifica alunos com dificuldades
Estratégia: Professor de apoio entrega versão adaptada (10 exercícios, apoio visual) aos alunos que necessitam acomodação
```

**Fluxo de trabalho com geradores**:
```
Domingo planeamento (10 minutos):
Professor titular: Gera ficha matemática padrão (20 exercícios, simbólicos)
Professor apoio: Gera versão adaptada (10 exercícios, modo imagem)
Ambos: Partilham via Google Drive

Segunda-feira aula:
Professor titular: Distribui versão padrão à maioria alunos
Professor apoio: Distribui versão adaptada aos alunos NEE (5-8 alunos)
Resultado: Diferenciação sem reuniões planeamento separadas
```

**Tempo poupado**: 30 minutos reunião planeamento semanal (podem coordenar via pasta partilhada)

---

### Modelo 2: Ensino por Estações

**Estrutura**: Turma dividida em estações, cada professor lidera uma estação

**Aplicação de fichas**:
```
Estação 1 (Professor titular): Palavras cruzadas nível turma
Estação 2 (Professor apoio): Palavras cruzadas adaptadas (menos palavras, pistas visuais)
Estação 3 (Autónomo): Desafios alunos rápidos
Estação 4 (Autónomo): Manipulativos concretos
```

**Fluxo de trabalho com geradores**:
```
Professor titular gera:
- Estação 1 palavras cruzadas (15 palavras, pistas texto)
- Estação 3 desafios (criptogramas, sudoku avançado)

Professor apoio gera:
- Estação 2 palavras cruzadas (8 palavras, pistas imagem)
- Chaves resposta todas estações

Tempo: 12 minutos total (ambos professores gerando)
Estações preparadas semana inteira
```

---

### Modelo 3: Ensino Paralelo

**Estrutura**: Turma dividida ao meio, ambos professores ensinam mesmo conteúdo

**Aplicação de fichas**:
```
Grupo A (Professor titular): Dificuldade padrão
Grupo B (Professor apoio): Dificuldade adaptada (grupo menor = mais suporte)
```

**Vantagem geradores**: Mesmo gerador, definições diferentes = versões paralelas instantâneas

**Exemplo**:
```
Aula matemática: Prática adição

Ficha Grupo A:
- Intervalo: 1-100
- Exercícios: 25
- Formato: Apenas simbólico
- Gerada: 42 segundos

Ficha Grupo B:
- Intervalo: 1-20
- Exercícios: 15
- Formato: Modo imagem
- Gerada: 42 segundos

Preparação total: 84 segundos duas versões diferenciadas
Ambos grupos praticam mesma competência (adição) nível apropriado
```

---

### Modelo 4: Ensino Alternativo

**Estrutura**: Professor titular ensina grupo grande, professor apoio trabalha grupo pequeno para pré-ensino ou reforço

**Aplicação de fichas**:
```
Grupo grande (Professor titular): Introdução novo conteúdo + prática nível turma
Grupo pequeno (Professor apoio): Pré-ensino com ficha estruturada OU reforço com ficha simplificada
```

**Exemplo pré-ensino**:
```
Sexta-feira (apoio grupo reduzido):
- Professor apoio: Pré-ensina vocabulário próxima semana (caça-palavras simplificado)
- Alunos: Expostos termos antes aula segunda-feira
- Resultado: Mais sucesso durante aula turma completa segunda

Segunda-feira (titular turma completa):
- Professor titular: Ensina vocabulário (palavras cruzadas padrão)
- Alunos pré-ensinados: Já familiarizados termos (confiança aumentada)
```

## Aplicações Sala de Apoio

**Sala de apoio**: Espaço separado onde alunos NEE recebem instrução especializada

**Horário**: Alunos saem sala turma 30-60 minutos

**Desafio curricular**: Deve alinhar com currículo turma fornecendo apoio especializado

---

### Coordenação Curricular Sala de Apoio

**Problema**: Sala apoio ensinar conteúdo diferente = aluno perde instrução turma

**Solução**: Currículo paralelo (mesmos tópicos, dificuldade adaptada)

**Exemplo**:
```
Sala turma regular (Semana 15 Outubro):
- Matemática: Tabuadas multiplicação 1-12
- Português: Estudo obra 4º ano
- Ciências: Vocabulário estados matéria

Sala de apoio (Mesma semana):
- Matemática: Tabuadas multiplicação 1-5 (intervalo simplificado)
- Português: Compreensão adaptada (mesma obra, perguntas ajustadas)
- Ciências: Caça-palavras estados matéria (mesmo vocabulário, atividade diferente)

Resultado: Aluno aprende mesmo conteúdo colegas (apenas apoio adaptado)
```

**Fluxo trabalho geradores**:
```
Professor educação especial: Coordena com professores titulares

Segunda manhã:
- Professores titulares: Publicam fichas semana Google Drive partilhado
- Professor apoio: Abre fichas, anota competências ensinadas
- Professor apoio: Gera versões adaptadas (dificuldade reduzida)
- Tempo total: 15 minutos (vs 2-3 horas criar do zero)
```

---

### Monitorização Progresso Objetivos PEI

**Finalidade sala apoio**: Instrução direcionada objetivos PEI (Programa Educativo Individual)

**Aplicação geradores**: Ferramenta monitorização progresso consistente

**Exemplo objetivo PEI**: "Aluno resolverá problemas adição um dígito com 80% precisão"

**Monitorização progresso semanal**:
```
Semana 1: Gera ficha adição (20 problemas, 1-10)
          Resultado aluno: 12/20 (60%)

Semana 2: Gera NOVA ficha adição (mesma competência, problemas diferentes)
          Resultado aluno: 14/20 (70%)

Semana 3: Gera NOVA ficha
          Resultado aluno: 16/20 (80%) ✓ OBJETIVO ALCANÇADO

Documentação: 3 fichas pasta PEI (prova progresso)
```

**Vantagem geradores**: Problemas frescos cada semana (sem memorização, avaliação competência real)

### Reunião Planeamento Semanal (Reduzida)

**Co-planeamento tradicional**: 90 minutos/semana (discutir currículo, criar materiais, coordenar)

**Com geradores**: 30 minutos/semana (discutir apenas instrução, materiais pré-gerados)

**Tempo poupado**: 60 minutos/semana × 36 semanas = **36 horas/ano por dupla docentes**

**Agenda reunião** (30 minutos):
```
1. Rever progresso alunos semana passada (10 min)
2. Discutir objetivos próxima semana (10 min)
3. Atribuir papéis (quem ensina o quê) (5 min)
4. Confirmar estratégia diferenciação (quais alunos recebem quais materiais) (5 min)

Nota: Materiais já gerados (não discutidos reunião)
Resultado: Reunião foca INSTRUÇÃO, não preparação materiais
```

## Boas Práticas Inclusão Escolar

### Diferenciação Invisível

**Objetivo**: Acomodar alunos NEE sem estigma

**Estratégia**: Usar mesmo gerador, diferenças subtis

**Exemplo**:
```
Turma completa: "Todos fazem ficha matemática"

Realidade:
- 22 alunos: Ficha 25 exercícios (padrão)
- 8 alunos NEE: Ficha 15 exercícios (adaptada)

Semelhança visual:
- Mesmo título ("Prática Adição")
- Mesmo formato (exercícios linhas)
- Mesmo layout página
- Única diferença: Número exercícios (menos óbvio que "ADAPTADO" carimbado topo)

Resultado: Alunos NEE sentem-se incluídos (não "diferentes")
```

---

### Desenho Universal Aprendizagem (DUA)

**Princípio DUA**: Desenhar para todos aprendentes desde início (não adaptar posteriormente)

**Aplicação geradores**:
```
Em vez de:
1. Criar ficha padrão
2. Adaptar para alunos NEE (trabalho extra)

Abordagem DUA:
1. Gerar 3 versões simultaneamente (15 min):
   - Nível 1: Abaixo nível turma (alunos dificuldades)
   - Nível 2: Nível turma (maioria)
   - Nível 3: Acima nível turma (alunos avançados)

2. Alunos escolhem nível OU professor atribui discretamente

Resultado: Todos alunos têm materiais apropriados, sem estigma "adaptado"
```
