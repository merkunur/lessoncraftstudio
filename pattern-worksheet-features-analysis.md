## App-Specific Features

### Pattern Types (9 total)
1. **AB** - 2 images, alternating (simplest)
2. **AAB** - 2 images (2 same, 1 different)
3. **ABB** - 2 images (1, then 2 same)
4. **ABC** - 3 images (three different)
5. **AABB** - 2 images (pairs alternating)
6. **ABBC** - 3 images (complex)
7. **AABC** - 3 images (complex)
8. **ABCC** - 3 images (complex)
9. **ABCD** - 4 images (most complex)

### Question Types
1. **Blank Box** - Empty space for child to fill in
2. **Choose from Options** - Multiple choice format

### Exercise Settings
- **Number of Exercises:** 1-8 per worksheet
- **Random blank box position:** Toggle option
- **Include Puzzle Numbers:** Toggle option
- **Start from random element:** Toggle option
- **Individual puzzle configuration:** Each puzzle can have different settings

### Overall Theme Options
- Can apply one theme to entire worksheet
- Or configure each puzzle individually
