## Indledning: Lærernes Budgetudfordring

**Lærerens virkelighed**: Den gennemsnitlige danske lærer bruger **5.600 kr./år** af egne penge på klasseværelsesmaterialer

**Budgetfordeling**:
```
Typiske udgifter:
- Arbejdsbøger: 2.250 kr. (engangsbrugsmaterialer, kan ikke genbruges)
- Undervisningsmaterialer: 1.125 kr.
- Klasseværelsesudsmykning: 750 kr.
- Diverse materialer: 1.475 kr.
Total: 5.600 kr. ud af egen lomme
```

**Problemet**: Arbejdsbøger er til engangsbrug (kan ikke genbruges næste år)
```
Køb: 30 matemati karbejdsbøger × 75 kr. = 2.250 kr.
Brug: August-maj (eleverne skriver i bøgerne)
Juni: Bøgerne går hjem (kan ikke genbruges)
Næste år: Køb 30 nye arbejdsbøger (2.250 kr. igen)
5-års omkostning: 11.250 kr. på engangsbrug arbejdsbøger
```

**Løsningen**: Digital generering + print = genanvendelige ressourcer
```
Pris: 1.080 kr./år abonnement
Generer: Ubegrænsede arbejdsark (svarende til 100+ arbejdsbøger)
Genbrug: Gem PDF'er, print igen næste år (nul ekstra omkostning)
5-års omkostning: 5.400 kr. (spar 5.850 kr. i forhold til arbejdsbøger)
```

## Genanvendelige Materialer Strategier

**Reducer papirforbrug**

### Tørbar Overflade Lommer

**Engangsinvestering**: 375 kr. for 30 lommer
```
Sådan virker det:
1. Print arbejdsark (ét eksemplar)
2. Indsæt i tørbar overflade lomme
3. Eleven udfylder med tørbar tusch
4. Slet, genbrug 100+ gange

Eksempel:
Morgen matematik opvarmning (daglig 5-problem arbejdsark):
- Uden lommer: Print 30 kopier × 180 dage = 5.400 sider
- Med lommer: Print 30 kopier én gang, genbrug hele året = 30 sider
Papir sparet: 5.370 sider (2.013 kr. ved 0,37 kr./side)

ROI: 375 kr. investering sparer 2.013 kr./år (5,4× afkast)
```

---

### Laminerede Arbejdsark

**Til hyppigt anvendte aktiviteter**:
```
Laminer: Tabeller, alfabetstavler, tallinjer
Pris: 150 kr. lamineremaskine + 75 kr. laminerings ark = 225 kr.
Holdbarhed: Holder 5+ år

Brug: Elever bruger med tørbare tuscher
Eksempel: "Tegn bogstaverne" arbejdsark
- Print én gang, laminer, brug 200+ gange over årene
- Pris pr. brug: 0,37 kr. ÷ 200 = 0,002 kr. (praktisk gratis)
```

## Papirbesparende Strategier

**Reducer printomkostninger**

### Duplex Print

**Enkel ændring, stor besparelse**:
```
Enkeltsidet: 180 arbejdsark = 180 sider
Duplex (dobbeltsidet): 180 arbejdsark = 90 sider
Besparelse: 90 sider pr. elev = 2.700 sider/klasse = 1.013 kr./år

Indstillinger: Printer indstillinger → Tosidet print → "Kort kant" (vend)
```

---

### Display + Hæfte Papir

**Eliminer fotokopier**:
```
Traditionelt: Print 30 kopier af arbejdsark
Alternativ: Projektér ÉT arbejdsark på skærm, eleverne kopierer til hæfte

Eksempel:
Matematik opgaver: Vis 20 opgaver på projektor
Elever: Skriv opgaver + svar i matematik hæfte
Papir brugt: 0 ark

Når passende: Beregningsøvelser, ordforrådsdefinitioner, forståelsesspørgsmål
Når ikke passende: Komplekse diagrammer, lange tekster
```
