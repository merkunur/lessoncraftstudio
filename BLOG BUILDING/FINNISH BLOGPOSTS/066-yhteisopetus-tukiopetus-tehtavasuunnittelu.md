## Johdanto: Yhteisopetuksen koordinoinnin haaste

**Yhteisopetuksen rakenne**: Luokanopettaja + erityisopettaja = yhteinen opetusryhmä

**Yleisyys**: Yli 60 % suomalaisista kouluista käyttää yhteisopetusta osana kolmiportaista tukea

**Koordinointiongelma**:
```
Luokanopettaja: Suunnittelee opetuskokonaisuuden ja luo perusmateriaalit
Erityisopettaja: Eriyttää materiaalit ja toteuttaa tukitoimet
Ongelma: Kaksi opettajaa, erilliset suunnitelmat → tuplatyö TAI epäjohdonmukaisuus
```

**Perinteinen tehtävämonisteongelma**:
```
Luokanopettaja: Luo 30 tehtävän matematiikkamonisteen (haasteellinen)
Erityisopettaja: Muokkaa monisteen (15 tehtävää, lisää kuvia ja tukia)
Tuplatyö: 2× työmäärä olennaisesti samoista sisällöistä
Tulos: Tehottomuus
```

**Generaattorien ratkaisu**: Yksi alusta, välitön eriyttäminen

**Tutkimus** (Friend & Cook, 2017): Tehokas yhteisopetus vaatii **yhteistä suunnittelua** + **yhteisiä resursseja** (vähentää valmistelua 40 %)

## Yhteisopetuksen mallit ja tehtävämonisteiden käyttö

### Malli 1: Toinen opettaa, toinen avustaa

**Rakenne**: Luokanopettaja opettaa, erityisopettaja kiertää antaen tukea

**Tehtävämonisteiden käyttö**:
```
Luokanopettaja: Jakaa matematiikkamonisteen (20 tehtävää, ikätasolle sopiva)
Erityisopettaja: Kiertää luokassa, tunnistaa tukea tarvitsevat oppilaat
Ratkaisu: Erityisopettaja antaa eriytetyn version (10 tehtävää, kuvatuella) HOJKS-oppilaille
```

**Generaattorityökulku**:
```
Sunnuntai-suunnittelu (10 minuuttia):
Luokanopettaja: Luo perusversion matematiikkamoniste (20 tehtävää, symbolinen)
Erityisopettaja: Luo eriytetyn version (10 tehtävää, kuvallinen tila)
Molemmat: Jakavat Google Drivessa

Maanantain oppitunti:
Luokanopettaja: Jakaa perusversion suurimmalle osalle oppilaita
Erityisopettaja: Jakaa eriytetyn version tukea tarvitseville (5-8 oppilasta)
Tulos: Eriyttäminen ilman erillistä suunnittelupalaveria
```

**Säästetty aika**: 30 minuuttia viikkosuunnittelua (koordinointi Google Drive -kansion kautta)

---

### Malli 2: Pienryhmäopetus (station teaching)

**Rakenne**: Luokka jaettu pisteisiin, kumpikin opettaja johtaa yhtä pistettä

**Tehtävämonisteiden käyttö**:
```
Piste 1 (Luokanopettaja): Ikätason mukainen ristikko
Piste 2 (Erityisopettaja): Mukautettu ristikko (vähemmän sanoja, kuvavihjet)
Piste 3 (Itsenäinen): Nopeiden tekijöiden haasteet
Piste 4 (Itsenäinen): Konkreettiset välineet
```

**Generaattorityökulku**:
```
Luokanopettaja luo:
- Piste 1 ristikko (15 sanaa, tekstivihje)
- Piste 3 haasteet (kryptogrammi, vaikea sudoku)

Erityisopettaja luo:
- Piste 2 ristikko (8 sanaa, kuvavihje)
- Vastausavaimet kaikkiin pisteisiin

Aika: 12 minuuttia yhteensä (molemmat opettajat luovat materiaalia)
Viikon pisteet valmiina
```

---

### Malli 3: Rinnakkaisopetus

**Rakenne**: Luokka puoliksi, molemmat opettavat samaa sisältöä

**Tehtävämonisteiden käyttö**:
```
Ryhmä A (Luokanopettaja): Perustaso
Ryhmä B (Erityisopettaja): Mukautettu taso (pienempi ryhmä = enemmän tukea)
```

**Generaattorihyöty**: Sama generaattori, eri asetukset = välittömät rinnakkaisversiot

**Esimerkki**:
```
Matematiikan oppitunti: Yhteenlaskun harjoittelu

Ryhmä A -moniste:
- Lukualue: 1-100
- Tehtäviä: 25
- Muoto: Vain symbolit
- Luotu: 42 sekuntia

Ryhmä B -moniste:
- Lukualue: 1-20
- Tehtäviä: 15
- Muoto: Kuvatila
- Luotu: 42 sekuntia

Valmistelu yhteensä: 84 sekuntia kahdelle eriyttävälle versiolle
Molemmat ryhmät harjoittelevat samaa taitoa (yhteenlasku) sopivalla tasolla
```

---

### Malli 4: Ennakko- ja jälkiopetus

**Rakenne**: Luokanopettaja opettaa isoa ryhmää, erityisopettaja vetää pienryhmän ennakko- tai jälkiopetuksessa

**Tehtävämonisteiden käyttö**:
```
Iso ryhmä (Luokanopettaja): Uuden sisällön esittely + ikätasoinen harjoittelu
Pieni ryhmä (Erityisopettaja): Ennakko-opetus tuella TAI jälkiopetus yksinkertaistetulla moniste
```

**Ennakko-opetuksen esimerkki**:
```
Perjantai (erityisopettajan pienryhmä):
- Erityisopettaja: Ennakko-opettaa ensi viikon sanaston (yksinkertaistettu sanahaku)
- Oppilaat: Tutustuvat termeihin ennen maanantain oppituntia
- Tulos: Parempi menestys maanantain koko ryhmän oppitunnilla

Maanantai (luokanopettajan koko ryhmä):
- Luokanopettaja: Opettaa sanaston (perinteinen ristikko)
- Ennakko-opetetut oppilaat: Tunnistavat termit jo valmiiksi (lisääntynyt itseluottamus)
```

## Tukiopetuksen sovellukset

**Tukiopetus**: Erillinen tila, jossa oppilaat saavat yksilöllistä tukiopetusta

**Aikataulu**: Oppilaat nostetaan luokanopetuksesta 30-60 minuutiksi

**Opetussuunnitelmahaaste**: Tukiopetuksen pitää olla linjassa luokan opetussuunnitelman kanssa, mutta tarjota yksilöllistä tukea

---

### Tukiopetuksen opetussuunnitelman koordinointi

**Ongelma**: Tukiopetus eri sisällöstä = oppilas jää paitsi luokanopetuksesta

**Ratkaisu**: Rinnakkainen opetussuunnitelma (samat aiheet, mukautettu vaikeustaso)

**Esimerkki**:
```
Luokanopetus (Viikko 15.10):
- Matematiikka: Kertotaulut 1-12
- Äidinkieli: 4. luokan kirja
- Ympäristöoppi: Aineen olomuodot, sanasto

Tukiopetus (Sama viikko):
- Matematiikka: Kertotaulut 1-5 (yksinkertaistettu lukualue)
- Äidinkieli: Sama kirja, mukautetut kysymykset
- Ympäristöoppi: Aineen olomuodot sanahaku (sama sanasto, eri aktiviteetti)

Tulos: Oppilas oppii samaa sisältöä kuin ikätoverit (vain mukautetulla tuella)
```

**Generaattorityökulku**:
```
Erityisopettaja: Koordinoi luokanopettajien kanssa

Maanantaiaamuna:
- Luokanopettaja jakaa viikon monisteet Google Drivessa
- Erityisopettaja: Avaa monisteet, huomioi opetettavat taidot
- Erityisopettaja: Luo mukautetut versiot (alennettu vaikeustaso)
- Yhteensä aikaa: 15 minuuttia (vs. 2-3 tuntia luomiseen tyhjästä)
```

---

### HOJKS-tavoitteiden seuranta

**Tukiopetuksen tarkoitus**: Kohdennettu HOJKS-tavoitteiden opetus

**Generaattoreiden käyttö**: Johdonmukainen edistymisen seurannan työkalu

**Esimerkki HOJKS-tavoite**: "Oppilas ratkaisee yhden luvun yhteenlaskutehtäviä 80 % tarkkuudella"

**Viikoittainen edistymisen seuranta**:
```
Viikko 1: Luo yhteenlaskumoniste (20 tehtävää, 1-10)
         Oppilaan tulos: 12/20 (60 %)

Viikko 2: Luo UUSI yhteenlaskumoniste (sama taito, eri tehtävät)
         Oppilaan tulos: 14/20 (70 %)

Viikko 3: Luo UUSI moniste
         Oppilaan tulos: 16/20 (80 %) ✓ TAVOITE SAAVUTETTU

Dokumentointi: 3 monistetta HOJKS-kansiossa (todiste edistymisestä)
```

**Generaattorihyöty**: Tuoreet tehtävät joka viikko (ei ulkomuistia, aito taidon arviointi)

## Yhteistyön suunnittelutyökulku

### Jaettu Google Drive -kansio

**Rakenne**:
```
2024-2025 Yhteisopetuksen materiaalit/
├── Matematiikka/
│   ├── Perustaso/
│   └── Mukautettu_taso/
├── Aidinkieli/
│   ├── Perustaso/
│   └── Mukautettu_taso/
├── Arvioinnit/
│   ├── Perustestit/
│   └── Mukautetut_testit/
└── Vastausavaimet/
```

**Työkulku**:
```
Sunnuntai (Luokanopettaja):
- Luo perustason monisteet viikolle
- Lataa "Perustaso"-kansioihin
- Yhteensä aikaa: 15 minuuttia

Sunnuntai (Erityisopettaja):
- Tarkistaa luokanopettajan lataukset
- Luo mukautetut versiot samoista taidoista
- Lataa "Mukautettu_taso"-kansioihin
- Yhteensä aikaa: 15 minuuttia

Maanantai-perjantai:
- Molemmat opettajat käyttävät sopivia versioita oppilaille
- Ei päivittäistä koordinointia (materiaalit ennakkovalmiita)
```

---

### Viikoittainen suunnittelupalaveri (lyhennetty)

**Perinteinen yhteissuunnittelu**: 90 minuuttia/viikko (keskustelu opetussuunnitelmasta, materiaalien luonti, koordinointi)

**Generaattoreiden kanssa**: 30 minuuttia/viikko (keskustelu vain opetuksesta, materiaalit ennakkoluotuna)

**Säästetty aika**: 60 minuuttia/viikko × 36 viikkoa = **36 tuntia/vuosi opettajaparia kohden**

**Kokousagenda** (30 minuuttia):
```
1. Edellisen viikon oppilasedistyksen tarkistus (10 min)
2. Tulevan viikon tavoitteiden keskustelu (10 min)
3. Roolien jako (kuka opettaa mitä) (5 min)
4. Eriyttämisstrategian vahvistus (ketkä oppilaat saavat mitä materiaalia) (5 min)

Huom: Materiaalit jo luotuna (ei keskustella kokouksessa)
Tulos: Kokous keskittyy OPETUKSEEN, ei materiaalien valmisteluun
```

## Inklusiiviset parhaat käytännöt

### Näkymätön eriyttäminen

**Tavoite**: HOJKS-oppilaiden tukeminen ilman leimaamista

**Strategia**: Käytä samaa generaattoria, hienovaraiset erot

**Esimerkki**:
```
Koko luokka: "Kaikki tekevät matematiikkamonisteen"

Todellisuus:
- 22 oppilasta: 25 tehtävän moniste (perustaso)
- 8 HOJKS-oppilasta: 15 tehtävän moniste (mukautettu)

Visuaalinen samankaltaisuus:
- Sama otsikko ("Yhteenlaskuharjoitus")
- Sama muoto (tehtävät riveissä)
- Sama sivuasettelu
- Ainoa ero: Tehtävien määrä (vähemmän silmäänpistävä kuin "MUKAUTETTU" leima ylhäällä)

Tulos: HOJKS-oppilaat tuntevat kuuluvansa joukkoon (eivät "erilaisia")
```

---

### Universaali oppimisen suunnittelu (UDL)

**UDL-periaate**: Suunnittele kaikille oppijoille alusta alkaen (älä jälkikäteen mukautuksia)

**Generaattoreiden käyttö**:
```
Sen sijaan, että:
1. Luot perusmonisteen
2. Mukauta HOJKS-oppilaille (lisätyö)

UDL-lähestymistapa:
1. Luo 3 versiota samanaikaisesti (15 min):
   - Taso 1: Alle luokkatason (oppimisvaikeuksissa olevat)
   - Taso 2: Luokka-aste (enemmistö)
   - Taso 3: Yli luokkatason (edistyneet)

2. Oppilaat valitsevat tason TAI opettaja jakaa huomaamattomasti

Tulos: Kaikilla oppilailla sopivat materiaalit, ei "mukautettu"-leimaa
```
