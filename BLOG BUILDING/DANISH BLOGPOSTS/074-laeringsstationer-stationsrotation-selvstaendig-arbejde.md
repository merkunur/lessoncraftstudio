## Indledning: Udfordringen med smågruppe undervisning

**Effektiv undervisning kræver**: Læreren arbejder med små grupper (5-7 elever) mens resten arbejder selvstændigt

**Problemet**: Hvad laver de andre 20 elever, mens læreren er optaget?

**Traditionelt scenarie** (virker ikke):
```
Lærer: "Jeg arbejder med den røde gruppe. Resten læser stille for sig selv."
Virkeligheden efter 5 minutter:
- 3 elever læser
- 5 elever snakker
- 4 elever sliber blyanter
- 8 elever spørger "Må jeg gå på toilet?"
Resultat: Læreren bliver afbrudt konstant, smågruppe undervisning er umulig
```

**Løsningen**: Læringsstationer (stationsrotationssystem)

**Forskning** (Fountas & Pinnell, 1996): Strukturerede læringsstationer muliggør **45-60 minutters uafbrudt smågruppe undervisning** (mod 10-15 minutter med ustruktureret selvstændigt arbejde)

## 6-stations rotationssystem

**Struktur**: 6 stationer, eleverne roterer hver 20. minut, læreren leder Station 1

### Stationsopgaver

**Station 1: Lærer-ledet (Smågruppe undervisning)**
```
Indhold: Guidet læsning, målrettet matematikundervisning, intervention
Gruppestørrelse: 5-7 elever
Lærerrolle: Aktiv undervisning (underviser i nye færdigheder)
Varighed: 20 minutter
```

**Station 2: Selvstændige arbejdsark**
```
Indhold: Matematik arbejdsark, læseforståelse, faglig gennemgang
Gruppestørrelse: 5-7 elever
Lærerrolle: Ingen (eleverne arbejder selvstændigt)
Materialer: Færdigtrykte arbejdsark, facitlister til selvkontrol
Varighed: 20 minutter
```

**Station 3: Makkerarbejde**
```
Indhold: Matematik bingo, ordsøgningskapløb, krydsord samarbejde
Gruppestørrelse: 5-7 elever (2-3 par)
Lærerrolle: Ingen (eleverne spiller sammen)
Materialer: Billede-bingo plader, ordsøgninger, stopur
Varighed: 20 minutter
```

**Station 4: Finmotorik / Kreativt**
```
Indhold: Farvelægningsopgaver, mønsterto (klip/lim), tegne linjer
Gruppestørrelse: 5-7 elever
Lærerrolle: Ingen (rolig, selvstændig aktivitet)
Materialer: Farvelægningssider, sakse, lim, blyanter
Varighed: 20 minutter
```

**Station 5: Teknologi / Digital træning**
```
Indhold: Computerbaserede matematikspil, skrivepraksis, digital læsning
Gruppestørrelse: 5-7 elever (hvis 6 computere tilgængelige)
Lærerrolle: Ingen (digital praksis)
Materialer: Computere/tablets med læringssoftware
Varighed: 20 minutter
```

**Station 6: Lytning / Læsning**
```
Indhold: Lydbog lytning, selvstændig læsning med forståelseskontrol
Gruppestørrelse: 5-7 elever
Lærerrolle: Ingen (selvstændig lytning)
Materialer: Lydbøger, hovedtelefoner, læselogs
Varighed: 20 minutter
```

## Daglig rotationsplan

**Klassestørrelse**: 30 elever, opdelt i 6 grupper á 5

**Mandag stationer** (120 minutter i alt):

**Periode 1** (9:00-9:20):
- Gruppe A: Station 1 (Lærer)
- Gruppe B: Station 2 (Arbejdsark)
- Gruppe C: Station 3 (Makker)
- Gruppe D: Station 4 (Kreativt)
- Gruppe E: Station 5 (Teknologi)
- Gruppe F: Station 6 (Læsning)

**Periode 2** (9:20-9:40):
- Gruppe F: Station 1 (Lærer) ← Roterer
- Gruppe A: Station 2
- Gruppe B: Station 3
- Gruppe C: Station 4
- Gruppe D: Station 5
- Gruppe E: Station 6

**Periode 3** (9:40-10:00):
- Gruppe E: Station 1 (Lærer)
- (Alle grupper fortsætter rotation)

**Periode 4** (10:00-10:20):
- Gruppe D: Station 1 (Lærer)

**Periode 5** (10:20-10:40):
- Gruppe C: Station 1 (Lærer)

**Periode 6** (10:40-11:00):
- Gruppe B: Station 1 (Lærer)

**Resultat**: Alle 6 grupper ser læreren i 20 minutter hver, alle elever er engagerede i fulde 120 minutter

## Station 2: Selvstændige arbejdsarks-station (Brug af generatorer)

**Formål**: Meningsfuld selvstændig træning (ikke travlt arbejde)

**Ugentlige materialer nødvendige**: 30 arbejdsark (6 grupper × 5 dage)

**Mandag Station 2 indhold**:
```
Fokus: Regnefærdigheder gennemgang
Arbejdsark: 20 plusopgaver (klassetrin-niveau)
Selvkontrol: Facitliste tilgængelig (elever tjekker eget arbejde)
Varighed: 20 minutter (elever udfylder ~15 opgaver)
```

**Tirsdag Station 2 indhold**:
```
Fokus: Ordforrådspraksis
Arbejdsark: Ordsøgning (ugens ordforråd)
Selvkontrol: Find alle 15 ord
Varighed: 20 minutter
```

**Onsdag Station 2 indhold**:
```
Fokus: Kritisk tænkning
Arbejdsark: Krydsord (fagligt ordforråd)
Selvkontrol: Facitliste tilgængelig
Varighed: 20 minutter
```

**Torsdag Station 2 indhold**:
```
Fokus: Logik/problemløsning
Arbejdsark: Matematikpuslespil (2-3 ubekendte)
Selvkontrol: Facitliste medfølger
Varighed: 20 minutter
```

**Fredag Station 2 indhold**:
```
Fokus: Gennemgang/vurdering
Arbejdsark: Blandet gennemgang (hele ugens færdigheder)
Aflevering: Lærer retter (formativ vurdering)
Varighed: 20 minutter
```

**Generator arbejdsgang**:
```
Søndags forberedelse (15 minutter):
- Generer 5 matematik arbejdsark: 3,5 min
- Generer 5 ordsøgninger: 3,5 min
- Generer 5 krydsord: 3,5 min
- Generer 5 matematikpuslespil: 3,5 min
- Print alle (30 kopier af hver): Håndteres af printer

Samlet tid: 15 minutter for hele ugens Station 2 materialer
```

## Station 3: Makkerarbejds-station

**Formål**: Social læring, peer-støtte, engagement

**Mandag-fredag materialer**:

**Mandag**: Billede-bingo (matematiske facts)
```
Materialer: 15 bingo plader (genereret på 1 minut)
Opsætning: Par spiller sammen, skiftes til at kalde numre
Varighed: 20 minutter (2-3 spil)
```

**Tirsdag**: Ordsøgningskapløb
```
Materialer: Samme ordsøgning som Station 2
Konkurrence: Hvem kan finde alle ord først?
Varighed: 20 minutter
```

**Onsdag**: Krydsord samarbejde
```
Materialer: Samme krydsord som Station 2
Samarbejde: Partnere arbejder sammen (diskuterer ledetråde)
Varighed: 20 minutter
```

**Torsdag**: Matematik sudoku partnere
```
Materialer: 15 sudoku puslespil (4×4 eller 6×6)
Samarbejde: Arbejd sammen om at løse
Varighed: 20 minutter
```

**Fredag**: Kryptogram udfordring
```
Materialer: 15 kryptogram arbejdsark
Konkurrence: Hvilket par bliver færdigt først?
Varighed: 20 minutter
```

---

## Station 4: Kreativ/finmotorisk station

**Formål**: Beroligende aktivitet, finmotorisk udvikling, hjernepause

**Ugentlige materialer**:

**Mandag-tirsdag**: Farvelægningssider
```
Generer: 15 farvelægningssider (matematisk kunst, sæsonmæssige temaer)
Tid til at oprette: 3,5 minutter
Varighed: 20 minutter (elever farvelægger i eget tempo)
```

**Onsdag**: Mønsterto (klip/lim)
```
Generer: 15 mønster arbejdsark
Materialer: Sakse, limstifter
Varighed: 20 minutter (klip mønstre ud, lim i rækkefølge)
```

**Torsdag-fredag**: Tegne linjer (labyrinter)
```
Generer: 15 labyrint arbejdsark (varierende sværhedsgrad)
Varighed: 20 minutter (udfyld 2-3 labyrinter)
```

**Fordel**: Stressfri station (alle lykkes, opbygger selvtillid)

## Stationsstyringssystemer

**Udfordring**: Elever skal vide HVOR de skal gå, HVAD de skal lave

**Løsning**: Visuelt styringssystem

### Visuelt stationstavle

**Opsætning** (på klasseværelsets væg):
```
Stationsrotations tavle

Periode 1 (9:00-9:20):
☐ Gruppe A → Station 1 (Lærerbord)
☐ Gruppe B → Station 2 (Blåt bord - Arbejdsark)
☐ Gruppe C → Station 3 (Rødt bord - Makkerarbejde)
☐ Gruppe D → Station 4 (Grønt bord - Kreativt)
☐ Gruppe E → Station 5 (Computere)
☐ Gruppe F → Station 6 (Læsekrog)

[Klokken ringer kl 9:20 - roter med uret]
```

**Elevperspektiv**:
```
Elev i Gruppe B:
- Ser på tavlen: "Jeg er ved Station 2 (Blåt bord)"
- Går til Blåt bord, ser arbejdsark + facitliste
- Udfylder arbejdsark selvstændigt
- Klokken ringer, går til Station 3 (Rødt bord)
- Ser på tavlen, ser Makkerarbejde instruktioner
- Går sammen med en anden Gruppe B elev, spiller Bingo
```

**Lærer fordel**: Ingen spørgsmål om hvor man skal gå (systemet er visuelt + forudsigeligt)

## Stationsmaterialer organisering

**Fysisk opsætning**:

**Blåt bord (Station 2 - Selvstændige arbejdsark)**:
```
Plasticboks mærket "Station 2 - Mandag":
- 30 arbejdsark (allerede adskilt)
- 6 facitlister
- Blyanter
- "Instruktioner" kort: "Udfyld arbejdsark, tjek svar, aflever"
```

**Rødt bord (Station 3 - Makkerarbejde)**:
```
Plasticboks mærket "Station 3 - Mandag":
- 15 bingo plader
- 1 opkaldsliste
- Stopur
- "Instruktioner" kort: "Spil bingo med makker, skiftes til at kalde"
```

**Grønt bord (Station 4 - Kreativt)**:
```
Plasticboks mærket "Station 4 - Mandag":
- 30 farvelægningssider
- Farver, farveblyanter
- "Instruktioner" kort: "Farvelæg billeder, ingen snak"
```

**Resultat**: Elever selv-administrerer (ingen lærer nødvendig til opsætning)

## Ansvarligheds systemer

**Udfordring**: Hvordan ved du, at eleverne rent faktisk arbejdede?

**Løsninger**:

### Strategi 1: Afleveringsboks

**Station 2** (arbejdsark):
```
Krav: Elever skal aflevere udfyldt arbejdsark
Lærer: Tjekker hurtigt for fuldførelse (5 min efter stationer)
Feedback: "Godt arbejde!" ELLER "Venligst færdiggør opgave 15-20"
```

---

### Strategi 2: Stationslog

**Hver elev har logark**:
```
Navn: _____________

Mandag stationslog:
☐ Station 1: Arbejdede med lærer (lærer initialer: ____)
☐ Station 2: Udfyldte arbejdsark #___
☐ Station 3: Spillede bingo med makker: _______
☐ Station 4: Farvede __ sider
☐ Station 5: Fuldførte skrivelektion #___
☐ Station 6: Læste ___ minutter

Forældreunderskrift (fredag): ___________
```

**Fordel**: Elevansvar + forældrekommunikation

---

### Strategi 3: Selvevaluering

**Fredag refleksion**:
```
Denne uge ved stationer:
Mit bedste arbejde var ved Station ___: fordi __________
Min indsats denne uge var: ☐ Fremragende ☐ God ☐ Skal forbedres
Næste uge vil jeg: _________________________

Lærer feedback: _________________________
```

**Resultat**: Metakognition (elever tænker over egen læring)

## Differentiering inden for stationer

**Udfordring**: Station 2 har elever på forskellige niveauer

**Løsning**: Opdelt materialer

### Station 2 opdelt arbejdsarkssystem

**Tre bokse ved Blåt bord**:

**Boks 1: "Udfordring"** (over klassetrin niveau)
```
Indhold: 25 opgaver, komplekse
Mærket: "For elever der ønsker en udfordring"
Selvvalg: Avancerede elever vælger dette
```

**Boks 2: "Standard"** (på klassetrin niveau)
```
Indhold: 20 opgaver, klassetrin-passende
Mærket: "Standard praksis"
Fleste elever: Vælger dette
```

**Boks 3: "Støtte"** (under klassetrin niveau)
```
Indhold: 15 opgaver, stilladseret (billede tilstand)
Mærket: "Ekstra støtte"
Kæmpende elever: Vælger dette ELLER lærer tildeler
```

**Generator arbejdsgang**:
```
Generer 3 versioner (2 minutter i alt):
Version 1: Område 1-100, 25 opgaver (42 sek)
Version 2: Område 1-50, 20 opgaver (42 sek)
Version 3: Område 1-20, billede tilstand, 15 opgaver (42 sek)

Resultat: Øjeblikkelig differentiering, elever selvvælger
```

## Støjstyring

**Udfordring**: 30 elever ved 6 stationer = potentielt kaos

**Løsninger**:

### Stemmeniveau diagram

**Opsat ved hver station**:
```
Station 1 (Lærer): Niveau 0 (tavs - lærer taler)
Station 2 (Arbejdsark): Niveau 1 (hviske til sig selv ok)
Station 3 (Makker): Niveau 2 (stille samtale med makker)
Station 4 (Kreativt): Niveau 1 (tavs farvelægning)
Station 5 (Teknologi): Niveau 1 (hviske hvis i tvivl)
Station 6 (Læsning): Niveau 0 (tavs læsning)
```

**Håndhævelse**:
```
Hvis station bliver for højlydt:
Advarsel 1: Lærer holder fingre op der viser stationsnummer (visuelt signal)
Advarsel 2: Den station mister 2 minutter næste rotation
Resultat: Selvregulering (elever overvåger eget volumen)
```

## Konklusion

1. Lærer-ledet (smågruppe undervisning, 5-7 elever, 20 min)
