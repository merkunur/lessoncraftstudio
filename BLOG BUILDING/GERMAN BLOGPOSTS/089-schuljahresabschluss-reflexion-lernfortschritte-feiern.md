## Einleitung: Die letzten Schulwochen sinnvoll nutzen

**Herausforderung der letzten Schulwochen**: Schüler sind bereits gedanklich in den Ferien, Konzentrationsprobleme häufen sich, wertvolle Unterrichtszeit geht verloren.

**Forschungsergebnis** (McCombs & Pope, 1994): Strukturierte Reflexionsphasen verbessern die Behaltensleistung um **35%** – systematisches Wiederholen festigt bereits Gelerntes nachhaltig.

**Traditioneller Schuljahresabschluss**: Filme schauen, Eisessen, kaum fachliche Arbeit
```
Realität vieler Klassenzimmer:
- Letzte 2 Wochen: Vorwiegend außerunterrichtliche Aktivitäten
- Lernverlust: Schüler vergessen Inhalte (Sommerlernverlust beginnt früh)
- Verhaltensauffälligkeiten: Fehlende Struktur führt zu Unruhe
```

**Sinnvolle Alternative**: Reflexion + Wertschätzung + Systematische Wiederholung
```
Strukturierter Ablauf:
- Woche 1 (vorletzte Schulwoche): Wiederholung aller Jahresinhalte
- Woche 2 (letzte Schulwoche): Reflexionsaktivitäten + Portfolioarbeit
- Letzter Schultag: Feier mit Bezug zum Gelernten

Ergebnis: Lernen geht weiter, Schüler verlassen stolz das Schuljahr
```

## Lernportfolio erstellen

**Greifbare Dokumentation des Jahresfortschritts**

### Portfolio-Inhalte

**September-Juni-Vergleich**:
```
Vorderer Teil: "So habe ich angefangen"
- Schreibprobe September (erste Schulwoche)
- Mathe-Diagnose September (Ausgangslage)
- Malarbeit September (feinmotorische Ausgangslage)

Hinterer Teil: "So weit bin ich gekommen"
- Schreibprobe Juni (letzte Schulwoche)
- Mathe-Abschlusstest Juni (Lernstand)
- Kunstwerk Juni (feinmotorische Entwicklung)

Mittelteil: "Mein Lernweg"
- Fotos aus dem Schuljahr
- Lieblingsarbeitsblätter (Schüler wählt aus)
- Urkunden und Auszeichnungen
- Lernfortschritts-Diagramm (z.B. Leseflüssigkeit)
```

**Portfolio-Zusammenstellung** (Woche 1 der letzten beiden Wochen):
```
Montag-Dienstag: Schüler sammeln Arbeitsmaterialien aus Mappen
Mittwoch: Portfolio ordnen (chronologisch)
Donnerstag: Deckblatt gestalten, Reflexion schreiben
Freitag: Präsentation in Partnerarbeit oder Kleingruppe
```

## Wiederholungsaktivitäten (Letzte 2 Wochen)

**Systematische Aufarbeitung der Jahresinhalte**

### Wochenplan Montag-Freitag

**Woche 1: Inhaltliche Wiederholung**
```
Montag: Mathe-Spiralwiederholung (alle Jahreskompetenzen, 30-Aufgaben-Blatt)
Dienstag: Wortschatz-Wörtersuche (100 Begriffe aus dem gesamten Schuljahr)
Mittwoch: Sachunterricht-Kreuzworträtsel (Schlüsselbegriffe aller Themen)
Donnerstag: Zeitstrahl Geschichte (wichtige Ereignisse/Personen)
Freitag: Leseverständnis (Texte aus Lieblingslektüren)

Format: Arbeitsblätter + Lernspiele + Klassengespräche
Ziel: Auffrischung vor den Sommerferien
```

**Woche 2: Reflexion + Wertschätzung**
```
Montag: Portfolios erstellen (Arbeitsmaterialien ordnen)
Dienstag: Reflexionsaufsätze schreiben (Jahresrückblick)
Mittwoch: Video-Botschaften aufnehmen (Tipps für nächste Klasse)
Donnerstag: Freundschaftsbücher (Mitschüler schreiben Einträge)
Freitag: Abschlussfeier (Lernspiele, Urkundenverleihung, Fest)
```

## Urkundenverleihung & Würdigung

**Jedes Kind wertschätzen**

### Individualisierte Urkunden

**Eine Urkunde pro Schüler** (personalisierte Stärke):
```
Beispiele:
"Größter Lernfortschritt in Mathematik"
"Durchhaltevermögen-Champion" (nie aufgegeben)
"Hilfsbereitschafts-Meister" (immer unterstützend)
"Kreativ-Denker" (originelle Problemlösungen)
"Lese-Star" (über 50 Bücher gelesen)

Prinzip: JEDES Kind wird für ETWAS gewürdigt
Ziel: Alle Schüler verlassen das Schuljahr mit Erfolgserlebnis
```

**Generator-Nutzung**: 30 individuelle Urkunden erstellen (21 Minuten = 42 Sek. pro Stück)
