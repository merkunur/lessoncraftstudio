## Inleiding: Van Docentgestuurd naar Leerlinggestuurd

**Traditioneel leren**: Leerkracht bepaalt doelen, volgt voortgang, beslist vervolgstappen
```
Leerkracht: "Deze week leren we de tafels van 7"
Leerling: Passieve ontvanger (doet wat de leerkracht zegt)
Resultaat: Externe motivatie (werken voor de leerkracht, niet voor jezelf)
```

**Leerlinggestuurd leren**: Leerling stelt doelen, volgt voortgang, evalueert groei
```
Leerling: "Mijn doel is om vrijdag de tafel van 7 te beheersen"
Leerling: Volgt dagelijkse voortgang (maandag: 40%, dinsdag: 55%, woensdag: 70%)
Leerling: Bereikt doel vrijdag (85% beheersing)
Resultaat: Intrinsieke motivatie (werken aan eigen doel)
```

**Onderzoek** (Hattie & Timperley, 2007): Leerling-zelfmonitoring heeft **0,62 effectgrootte** (equivalent aan 1,5 jaar extra leerwinst)

**Kernbevinding**: Leerlingen die hun eigen voortgang bijhouden presteren beter dan leeftijdsgenoten waarbij de leerkracht dit doet

## SMART Doelen voor Basisschoolleerlingen

**SMART framework** (Doran, 1981):
- **S**pecifiek: Helder, gedefinieerd doel
- **M**eetbaar: Kan bijhouden met cijfers/data
- **A**cceptabel: Uitdagend maar haalbaar
- **R**elevant: Belangrijk voor de leerling
- **T**ijdgebonden: Heeft een deadline

### Leeftijdsgeschikte SMART Doelen

**Groep 4-5** (7-8 jaar):
```
Vaag doel: "Beter worden in rekenen"
SMART doel: "Ik ga 8 van de 10 optellingen goed maken voor vrijdag"

Specifiek: Optellingen ‚úì
Meetbaar: 8/10 (80%) ‚úì
Acceptabel: Nu 6/10, dus 8/10 is realistisch ‚úì
Relevant: Leerling wil beter worden in rekenen ‚úì
Tijdgebonden: Voor vrijdag ‚úì
```

**Groep 6-7** (10-11 jaar):
```
Vaag doel: "Beter lezen"
SMART doel: "Ik ga 90 woorden per minuut (WPM) lezen met 95% nauwkeurigheid aan het eind van de maand"

Huidig: 75 WPM
Doel: 90 WPM (15 WPM toename over 4 weken = realistisch)
Meting: Wekelijkse leestests
```

## Feedback van Medeleerlingen en Zelfcorrectie

**Leerlingen controleren eigen werk, daarna peer review**

### Tweefasige Correctie Protocol

**Fase 1: Zelfcontrole**
```
Leerling maakt werkblad (20 opgaven)
Leerkracht geeft antwoordenblad
Leerling: Markeert eigen werk (cirkelt foute antwoorden)
Leerling: Probeert fouten zelfstandig te verbeteren
```

**Fase 2: Peer Review**
```
Leerling wisselt met klasgenoot
Klasgenoot: Controleert correcties (zijn fouten goed verbeterd?)
Klasgenoot: Geeft feedback ("Goed gedaan!" of "Controleer opgave 7 nog eens")
Leerling: Maakt laatste correcties
```

**Voordelen**:
- Zelfregulering (controleer eigen werk voor inleveren)
- Foutenanalyse (waarom was ik fout?)
- Peer leren (uitleggen aan elkaar)
- Tijdwinst leerkracht (leerlingen doen eerste correctieronde)

## Het Zelfmonitoring Proces Aanleren

**Expliciete instructie nodig** (ga er niet vanuit dat leerlingen het weten)

### Week 1: Introduceer Doelen Stellen

**Leerkracht modelleert**:
```
"Mijn doel deze week is om binnen 24 uur op alle ouderemails te reageren. Ik ga elke ochtend en middag mijn email checken. Aan het eind van elke dag markeer ik of ik mijn doel heb gehaald. Op vrijdag zie ik of ik het de hele week heb bereikt."

Leerlingen: Kijken naar leerkracht model (zie proces)
```

### Week 2: Begeleide Oefening

**Leerkracht en leerlingen samen**:
```
Klasdoel: Iedereen maakt 4/5 dagen huiswerk af
Elke dag: Leerlingen noteren (Heb ik huiswerk gemaakt? Ja/Nee)
Vrijdag: Tellen (Heb ik 4/5 dagen doel gehaald?)

Leerkracht: Begeleidt reflectie ("Zo niet, wat stond in de weg?")
```

### Week 3: Zelfstandige Oefening

**Leerlingen stellen individuele doelen**:
```
Leerling kiest eigen doel uit opties:
- Tafels beheersing
- Leessnelheid
- Spelling nauwkeurigheid
- Huiswerk afmaken

Leerling: Volgt voortgang zelfstandig
Leerkracht: Checkt in op vrijdag (bewaakt maar micromanaget niet)
```

### Week 4: Volledig Eigenaarschap

**Leerlingen leiden eigen doelstelling**:
```
Leerling: Stelt doel zonder input leerkracht
Leerling: Besluit hoe voortgang bij te houden
Leerling: Reflecteert op bereiken
Leerling: Stelt nieuw doel voor volgende week

Leerkracht: Faciliteert alleen (leerling-gedreven)
```
