## UNIQUE APP CHARACTERISTICS

**Math Puzzle Focus:**
- Creates visual math puzzles using images/symbols
- Each symbol represents a number
- Users solve for the symbol values
- Supports addition and/or subtraction operations
- Difficulty controls number of different symbols (2-4)
- Can show/hide answers in worksheet

**Image-Based Math:**
- Uses themed images as symbols in equations
- Can use individual images or full themes
- Custom uploaded images also work
- Min/max value range customizable
- Optional negative results

**Answer Key Generation:**
- Separate answer key tab
- Shows solutions to the puzzles
- Downloadable separately from worksheet
