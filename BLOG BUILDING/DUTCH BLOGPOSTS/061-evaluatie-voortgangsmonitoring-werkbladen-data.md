## Inleiding: Werkbladen als Evaluatie-instrument

**Soorten evaluatie**:
- **Formatief**: Tijdens het leerproces (voortgang monitoren, instructie aanpassen)
- **Summatief**: Na het leerproces (beheersing evalueren, cijfers geven)
- **Diagnostisch**: Voor het leerproces (leemtes identificeren, interventie plannen)

**Voordeel van werkbladen**: Automatische documentatie (tastbaar bewijs van leerlingprestaties)

**Probleem bij traditionele evaluatie**:
```
Leerkracht mondelinge toets: "Wat is 5 + 3?"
Leerling: "8"
Leerkracht: ✓ (mentale notitie: leerling beheerst optellen)
Probleem: Geen blijvend bewijs, voortgang niet meetbaar over tijd
```

**Werkblad-evaluatie oplossing**:
```
Week 1: Optelwerkblad (10 sommen), score: 60% (6/10 correct)
Week 4: Optelwerkblad (10 sommen), score: 80% (8/10 correct)
Week 8: Optelwerkblad (10 sommen), score: 100% (10/10 correct)
Documentatie: Duidelijke groeicurve (60% → 80% → 100%)
```

**Onderzoek** (Black & Wiliam, 1998): Formatieve toetsing verbetert leerprestaties met **0,4-0,7 standaarddeviaties** (gelijk aan verbetering van 50ste naar 65-75ste percentiel)

**Kernvereiste**: Systematische dataverzameling (werkbladen faciliteren dit)

## Formatieve Toetsing met Werkbladgeneratoren

### Strategie 1: Wekelijkse Voortgangsmonitoring

**Concept**: Dezelfde vaardigheid, wekelijkse meting

**Implementatie (Voorbeeld rekenen)**:
```
Vaardigheid: Optellen binnen 20

Week 1: Genereer optelwerkblad (20 sommen, bereik 1-20)
        Leerling behaalt: 12/20 correct (60%)

Week 2: Genereer NIEUW optelwerkblad (andere sommen, zelfde vaardigheid)
        Leerling behaalt: 15/20 correct (75%)

Week 3: Genereer NIEUW werkblad
        Leerling behaalt: 18/20 correct (90%)

Week 4: Genereer NIEUW werkblad
        Leerling behaalt: 20/20 correct (100%) ✓ BEHEERSING

Data: Duidelijke stijgende lijn (interventie werkt)
```

**Generatorvoordeel**: Elke week nieuwe opgaven (geen memoriseren, echte vaardigheidstoetsing)

**Beste generatoren voor voortgangsmonitoring**:
- Rekenwerkblad Generator (App 010): Wekelijkse rekenvaardigheden meten
- Optellen/Aftrekken (App 002/007): Specifieke bewerkingen volgen
- Woordzoeker (App 003): Woordenschatherkenning + voltooiingstijd meten

---

### Strategie 2: Foutpatroonanalyse

**Concept**: Specifieke misvattingen identificeren (niet alleen "fout")

**Implementatie (Voorbeeld rekenen)**:
```
Leerling maakt aftreksommen:
- 15 - 7 = 12 (onjuist, moet zijn 8)
- 14 - 6 = 12 (onjuist, moet zijn 8)
- 13 - 5 = 12 (onjuist, moet zijn 8)

Patroon geïdentificeerd: Leerling komt altijd op 12 uit (optellen in plaats van aftrekken)
Misvatting: Verwisseling van bewerkingen (+/−)
Gerichte interventie: Aftrekken opnieuw aanleren met concreet materiaal
```

**Foutanalysecategorieën**:

**Rekenfout**: Fout antwoord, maar correct proces
- 7 + 8 = 14 (moet zijn 15, telfout)

**Conceptuele fout**: Fundamenteel begripsgebrek
- 7 + 8 = 78 (cijfers achter elkaar, niet optellen)

**Procedurele fout**: Verkeerde werkwijze
- Trekt kleinste van grootste af ongeacht positie (42 - 28 = 26, moet zijn 14)

**Generatorvoordeel**: Consistente opmaak vergemakkelijkt patroonherkenning

---

### Strategie 3: Gedifferentieerde Evaluatie

**Concept**: Toetsen op leerlingniveau (niet one-size-fits-all)

**Probleem bij standaardtoetsen**:
```
Klassikale toets: 20 optelsommen (bereik 1-100)
- Sterke leerling: Klaar in 3 minuten (te makkelijk, geen uitdaging)
- Gemiddelde leerling: Klaar in 10 minuten (passend)
- Zwakke leerling: Kan niet afmaken (te moeilijk, frustratie)
Resultaat: Toets niet bruikbaar voor 2/3 van de klas
```

**Generatoroplossing**:
```
Zelfde vaardigheid (optellen), verschillende niveaus:

Zwakke leerling: Optellen 1-20, plaatjesmodus, 10 sommen
                 Succesvol afgerond (passende uitdaging)

Gemiddelde leerling: Optellen 1-100, symbolisch, 20 sommen
                     Passende moeilijkheidsgraad

Sterke leerling: Optellen 100-1000, meercijferig, 25 sommen
                 Adequaat uitgedaagd

Resultaat: Alle leerlingen getoetst op eigen niveau (zinvolle data voor iedereen)
```

**Tijd voor gedifferentieerde toetsen maken**:
- Handmatig: 3 toetsen × 30 min = 90 minuten
- Generatoren: 3 toetsen × 42 sec = 126 seconden (2,1 minuten)
- **Tijdwinst**: 87,9 minuten per toetsmoment

## IHP/OPP Doelen Monitoren

**IHP-vereiste**: Voortgang naar doelen documenteren (driemaandelijks of vaker)

**Generatorvoordeel**: Consistent meetinstrument

**Voorbeeld IHP-doel**: "Leerling lost 15/20 enkelvoudige optelsommen op met 80% nauwkeurigheid voor mei 2025"

**Dataverzameling met generatoren**:

**September nulmeting**:
```
Genereer: Optelwerkblad (20 sommen, bereik 1-10)
Leerlingprestatie: 8/20 correct (40%)
IHP-notitie: Onder doel (40% versus 80% streefwaarde)
```

**November voortgangscheck**:
```
Genereer: NIEUW optelwerkblad (20 sommen, zelfde bereik)
Leerlingprestatie: 12/20 correct (60%)
IHP-notitie: Verbetering (40% → 60%, +20 procentpunten)
```

**Januari voortgangscheck**:
```
Genereer: NIEUW werkblad
Leerlingprestatie: 15/20 correct (75%)
IHP-notitie: Bijna op doel (75% versus 80% streefwaarde)
```

**Maart voortgangscheck**:
```
Genereer: NIEUW werkblad
Leerlingprestatie: 17/20 correct (85%)
IHP-notitie: DOEL BEHAALD (85% overtreft 80% streefwaarde) ✓
```

**Jaarlijkse IHP-bespreking**: Presenteer alle 4 werkbladen (visuele documentatie van vooruitgang)

**Wettelijke naleving**: Blijvend document (vereist voor IHP-documentatie)

## Handelingsgericht Werken: Data voor de Cyclus

**Drie-stappenmodel handelingsgericht werken**:
- **Stap 1**: Basisondersteuning voor alle leerlingen (80% slaagt)
- **Stap 2**: Extra ondersteuning in kleine groep (15% heeft dit nodig)
- **Stap 3**: Intensieve individuele begeleiding (5% heeft gespecialiseerde hulp nodig)

**Vereiste**: Frequente voortgangsmonitoring (wekelijks of tweewekelijks)

**Generatortoepassing**:

**Stap 2 leerling** (ontvangt rekenondersteuning in kleine groep):
```
Week 1: Optelwerkblad, 50% nauwkeurigheid (interventie start)
Week 2: Optelwerkblad, 55% nauwkeurigheid (lichte verbetering)
Week 3: Optelwerkblad, 60% nauwkeurigheid (consistente groei)
Week 4: Optelwerkblad, 65% nauwkeurigheid (reageert op interventie)
Week 5: Optelwerkblad, 70% nauwkeurigheid
Week 6: Optelwerkblad, 75% nauwkeurigheid
Week 7: Optelwerkblad, 80% nauwkeurigheid
Week 8: Optelwerkblad, 85% nauwkeurigheid (terug naar stap 1 mogelijk)

Data: 8 meetpunten over 8 weken (voldoende voor handelingscyclus)
Uitkomst: Interventie succesvol, leerling kan terug naar stap 1
```

**Frequentie**: Wekelijks (generatoren maken dit haalbaar, handmatig maken zou 8 × 40 min = 320 minuten kosten)

**Generatortijd**: 8 × 42 sec = 5,6 minuten totaal

## Tussenevaluaties (Benchmarks)

**Benchmark**: 3× per jaar meting (begin, midden, einde schooljaar)

**Doel**: Groei over het schooljaar meten

**Implementatie**:

**September (Begin schooljaar)**:
```
Groep 4 rekenbenchmark:
- Optellen 1-20: Genereer 20 sommen
- Aftrekken 1-20: Genereer 20 sommen
- Leerlingscores: Optellen 60%, Aftrekken 50%
Nulmeting vastgesteld
```

**Januari (Halverwege schooljaar)**:
```
Zelfde vaardigheden, NIEUWE opgaven (generatoren maken verse toets):
- Optellen 1-20: NIEUWE 20 sommen
- Aftrekken 1-20: NIEUWE 20 sommen
- Leerlingscores: Optellen 85%, Aftrekken 75%
Voortgang: +25% optellen, +25% aftrekken (op schema)
```

**Mei (Einde schooljaar)**:
```
Zelfde vaardigheden, NIEUWE opgaven:
- Optellen 1-20: NIEUWE 20 sommen
- Aftrekken 1-20: NIEUWE 20 sommen
- Leerlingscores: Optellen 95%, Aftrekken 90%
Jaargroei: +35% optellen, +40% aftrekken (sterke groei)
```

**Datavisualisatie**:
```
Optelvoortgang: 60% → 85% → 95% (duidelijke groeilijn)
Aftrekvoortgang: 50% → 75% → 90% (sterke verbetering)
Conclusie: Leerling heeft uitstekende voortgang geboekt (rapportage voor ouders/directie)
```

## Data-Gedreven Instructiebeslissingen

**Evaluatiedata gebruiken om onderwijs aan te passen**:

**Scenario 1**: 80% van de klas worstelt met lenen bij aftrekken
```
Data: Klasgemiddelde op aftreksommen: 45%
Foutpatroon: Meeste fouten bij sommen die lenen vereisen (23 - 17, 42 - 28)
Beslissing: Leenconcept opnieuw aanleren (niet alleen meer oefenen)
Actie: Gebruik concreet materiaal, genereer daarna nieuw werkblad voor hertoetsing
```

**Scenario 2**: Sterke leerling verveelt zich
```
Data: Leerling maakt groep 4 rekenwerkblad in 3 minuten, 100% nauwkeurigheid
Beslissing: Leerling heeft gevorderder materiaal nodig
Actie: Genereer groep 6 niveau werkblad (passende uitdaging)
Resultaat: Leerling betrokken, werkt op geschikt niveau
```

**Scenario 3**: Leerlingachteruitgang
```
Data: Leerlingscores dalen (Week 1: 80%, Week 2: 75%, Week 3: 65%)
Waarschuwing: Achteruitgang duidt op probleem (niet leren, of externe factor)
Beslissing: Onderzoek (gesprek met leerling, ouders, schoolmaatschappelijk werk)
Actie: Mogelijk interventie, aanpassingen, of ondersteuning nodig
```

**Data informeert beslissingen**: Generatoren leveren snel en consistent data
