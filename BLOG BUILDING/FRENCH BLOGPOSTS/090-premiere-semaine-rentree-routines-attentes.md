## Introduction : La Semaine Décisive

**Recherche** (Wong & Wong, 2009) : Les deux premières semaines déterminent la réussite de **toute l'année scolaire**

**Ce qui se joue la première semaine** :
```
Bon départ :
- Procédures clairement enseignées
- Attentes établies dès le début
- Routines pratiquées activement
- Climat de classe construit
Résultat : Année fluide (95% de problèmes comportementaux en moins)

Départ chaotique :
- Aucune procédure enseignée
- Attentes floues
- Désordre toléré
Résultat : Gestion de comportements problématiques toute l'année
```

**Principe fondamental** : « On n'a jamais une deuxième chance de faire une bonne première impression »

## Jour 1 : Établir les Attentes

**Objectifs** : Sécurité, procédures, cohésion

### Matin : Chasse au Trésor dans la Classe

**Activité** : Les élèves complètent une fiche de découverte de la classe
```
Chasse au Trésor en Classe

Nom : _____________ Date : _______

Trouve ces éléments et note leur emplacement :

1. Où rendre les travaux terminés ? ________
2. Où se trouvent les crayons de réserve ? __________
3. Où se range le cartable ? ___________
4. Où est située la bibliothèque de classe ? ______________
5. Où sont les fournitures communes ? __________
6. Où est la sortie de secours ? _____________
7. Où trouver l'emploi du temps du jour ? _______
8. Où ranger le matériel d'arts plastiques ? _________

Bonus : Quelle règle de classe vois-tu affichée ?
_________________________________________________
```

**Objectif pédagogique** : Les élèves découvrent l'organisation spatiale activement (pas de visite passive)

## Jour 2 : Pratiquer les Procédures

**Objectif** : Enseigner COMMENT fonctionne la classe

### Matin : Liste de Vérification des Procédures

**Activité** : Les élèves complètent pendant la pratique
```
Pratique des Procédures de Classe

Aujourd'hui nous pratiquons :

ROUTINE DU MATIN :
☐ Accrocher mon cartable
☐ Sortir les devoirs (si applicable)
☐ Tailler mes crayons (2 crayons)
☐ Lire le message du matin au tableau
☐ Commencer l'activité rituelle

PENDANT LES LEÇONS :
☐ Lever la main pour parler
☐ Écouter quand les autres parlent
☐ Suivre les consignes dès la première fois
☐ Demander de l'aide de manière appropriée

TRANSITIONS :
☐ Ranger mon espace de travail
☐ Ranger ma chaise
☐ Me mettre en rang calmement
☐ Marcher en silence dans le couloir (niveau 0)

J'ai pratiqué ces procédures aujourd'hui ! ✓
Signature de l'enseignant(e) : ____________
```

**Méthode** : PRATIQUER réellement chaque procédure (pas seulement en parler)

---

### Après-midi : Fiche de Candidature aux Responsabilités

**Activité** : Postuler pour les responsabilités de classe
```
Candidature aux Responsabilités

Nom : _____________ Date : _______

Je souhaite postuler pour cette responsabilité : __________

Responsabilités disponibles :
- Chef de rang (guide la classe dans les couloirs)
- Responsable de la porte (tient la porte pour la classe)
- Distributeur (distribue les feuilles)
- Nettoyeur de tableau (efface le tableau en fin de journée)
- Tailleur de crayons (taille les crayons de la classe)
- Bibliothécaire (organise la bibliothèque de classe)
- Aide-secrétariat (apporte les documents au secrétariat)

Pourquoi je serais bon(ne) dans ce rôle :
_________________________________________________
_________________________________________________

Expérience antérieure (à la maison ou à l'école) :
_________________________________________________

Enseignant(e) : Examiner les candidatures, attribuer les rôles mercredi
```

## Jour 3 : Évaluation Diagnostique

**Objectif** : Évaluer le niveau de départ des élèves

### Matin : Évaluation en Lecture

**Activité** : Vérification de la fluidité en lecture + compréhension
```
Lis ce texte silencieusement (l'enseignant chronomètre) :

[Texte adapté au niveau, 100 mots]

Temps : _____ secondes = _____ mots/minute

Maintenant réponds à ces questions :
1. De quoi parle ce texte ? (idée principale)
2. Pourquoi [personnage] a fait [action] ? (inférence)
3. Que signifie [mot] ? (vocabulaire)

Enseignant(e) : Enregistre les données (base de référence pour suivre les progrès)
```

---

### Après-midi : Évaluation Diagnostique en Mathématiques

**Activité** : Fiche de révision (contenus de l'année précédente)
```
Évaluation Diagnostique en Mathématiques

20 problèmes couvrant :
- Addition/soustraction (niveau antérieur)
- Multiplication de base (si approprié)
- Problèmes écrits
- Formes géométriques/mesures

Objectif : Pas de notation ! Cela aide simplement l'enseignant(e) à comprendre ce dont tu te souviens.

Fais de ton mieux, mais ne stresse pas si tu ne sais pas quelque chose !
```

## Jour 4 : Cohésion d'Équipe et Communauté

**Objectif** : Construire une culture de classe positive

### Matin : Trouve Quelqu'un Qui...

**Activité** : Bingo de découverte des camarades
```
Trouve Quelqu'un Qui...

Circule dans la classe, trouve des camarades qui correspondent à chaque description. Obtiens leur signature !

☐ A la même couleur préférée que toi : _______
☐ Est né(e) le même mois : ________
☐ Aime la même série de livres : _____________
☐ A visité un autre pays : ___________
☐ Joue d'un instrument de musique : ____________
☐ A plus de 2 frères et sœurs : ______________
☐ Aime les mathématiques : ____________________________
☐ Adore lire : _________________________

Objectif : Remplir toute la fiche, rencontrer tous les élèves de la classe !
```

---

### Après-midi : Affiche des Objectifs de Classe

**Activité** : Brainstorming collaboratif
```
Nos Objectifs de Classe Cette Année

Comment voulons-nous que notre classe soit ?

Fiche individuelle :
J'espère que notre classe sera : __________________
Une chose que je peux faire pour aider : ________________

Classe : Partager les idées, l'enseignant(e) écrit sur l'affiche
Résultat : « Notre Charte de Classe »
Tous les élèves signent (engagement)
```

## Jour 5 : Pratique et Révision

**Objectif** : Consolider les routines apprises lundi-jeudi

### Matin : Simulation Complète d'une Journée

**Activité** : Pratiquer une journée complète comme si c'était la Semaine 2
```
Matin :
- Compléter l'activité rituelle (exercice de calcul mental)
- Sortir les devoirs (aucun pour l'instant, mais pratiquer le geste)
- Déposer dans la corbeille

Mi-journée :
- Transition vers la lecture (pratiquer la procédure de rangement)
- Travailler sur une fiche de lecture en autonomie
- Rendre quand c'est terminé

Après-midi :
- Transition vers les mathématiques (pratiquer encore)
- Compléter une fiche de mathématiques
- Auto-correction avec le corrigé (pratiquer la procédure)

Fin de journée :
- Préparer les cartables (pratique)
- Ranger son espace (pratique)
- Se mettre en rang pour la sortie (pratique)

Retour : « Aujourd'hui nous avons fait chaque routine parfaitement ! »
```

## Fiches Essentielles pour la Première Semaine

**Matériel nécessaire** (à générer en 15 minutes au total) :

### Activités « Apprendre à se Connaître »

- Tout sur Moi (Jour 1)
- Chasse au Trésor en Classe (Jour 1)
- Trouve Quelqu'un Qui (Jour 4)
- Candidatures aux Responsabilités (Jour 2)
**Temps** : 3 minutes pour générer 4 fiches

---

### Évaluations Diagnostiques

- Compréhension de lecture (Jour 3)
- Révision mathématiques (Jour 3)
- Échantillon d'écriture (Jour 2)
**Temps** : 2 minutes pour 3 évaluations

---

### Pratique des Procédures

- Liste de vérification des procédures (Jour 2)
- Fiche de pratique des routines (Jour 5)
**Temps** : 1,5 minute pour 2 fiches

---

### Réflexions

- Réflexion quotidienne (Jours 1-5, 5 fiches)
- Bilan Semaine 1 (Jour 5)
**Temps** : 4 minutes pour 6 fiches de réflexion

**Génération totale** : 10,5 minutes pour tout le matériel de la première semaine

## Plan de Remplacement pour la Première Semaine

**Plan de secours** : Si l'enseignant est absent de manière imprévue pendant la première semaine

**Plan d'Urgence Jour 1** :
```
Si remplacement nécessaire Jour 1 :

Matin :
- Distribuer la Chasse au Trésor en Classe (réponses dans le dossier)
- Les élèves complètent de manière autonome (auto-explicatif)

Après-midi :
- Distribuer la fiche Tout sur Moi
- Les élèves complètent et décorent

Sortie :
- Collecter les fiches, laisser pour l'enseignant
- Les élèves se mettent en rang (le remplaçant suit les procédures de sortie affichées)

Matériel : Pré-imprimé, dans le « Dossier Urgence Semaine 1 »
```
