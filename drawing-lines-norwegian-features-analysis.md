## How It Works - Step by Step

1. **Select Template:** Choose from 8 drawing line patterns (curves, diagonals, horizontal, vertical)
2. **Assign Image Pairs:**
   - Manually select images for left/right sides
   - OR use auto-fill to randomly assign from selected theme
3. **Customize Settings:**
   - Choose page size (Letter/A4, Portrait/Landscape)
   - Add borders and backgrounds
   - Include name/date fields if needed
4. **Generate Worksheet:** Click Generate to create worksheet on canvas
5. **Edit on Canvas:**
   - Add text labels or instructions
   - Move, resize, rotate any element
   - Change colors, fonts, styles
   - Add more images from library
   - Upload and add custom images
6. **Download:** Export as JPEG or PDF in professional 300 DPI quality

## Educational Purpose

**Pre-writing and Fine Motor Skills Development:**
- Helps preschool/kindergarten children practice drawing lines
- Develops hand-eye coordination
- Builds pencil control and grip strength
- Prepares children for writing letters
- Engaging visual elements (images) motivate tracing practice

**Norwegian Education Context:**
- Perfect for barnehage (kindergarten/preschool, ages 1-5)
- Excellent for 1. trinn (1st grade, ages 6-7) preparing for writing
- Supports finmotorikk (fine motor skills) development
- Aligns with Norwegian curriculum focus on skriving (writing) preparation
