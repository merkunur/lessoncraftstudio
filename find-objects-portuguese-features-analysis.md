# Find Objects App - Features Analysis

## Source: REFERENCE APPS/find objects.html

## Pricing Tier
**Find Objects = Full Access ($240/year = $25/month)**

## Activity Modes (2 modes available)
1. **I Spy - Find Hidden Objects**: Classic hidden object activity where children search for specific items among distractors
2. **Odd One Out - Find Unpaired Images**: Matching activity where children find images that don't have a pair

## Accordion Sections (5 sections)

### 1. Page & Scene
- **Page Size Options**: Letter Portrait (8.5x11"), Letter Landscape (11x8.5"), A4 Portrait (210x297mm), A4 Landscape (297x210mm), Custom (width/height in px)
- **Background**:
  - Use Child-Friendly Decorations checkbox
  - Fallback Color picker
  - Background Theme dropdown
  - Background Opacity slider
- **Border**:
  - Border Theme dropdown
  - Border Opacity slider

### 2. Text & Content
- **Text Tools**:
  - Add New Text input field
  - Add Text button
- **Selected Text Properties**:
  - Color picker
  - Size (number input, default 48)
  - Font selection: Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana
  - Outline Color picker
  - Outline Width slider (0-10)
- **Other Content**:
  - Include Name/Date Fields checkbox

### 3. Image Library
- Filter by Theme dropdown
- Search Images input field
- Available Images display grid

### 4. Upload Custom Images
- Multi-file upload input
- Session-based uploaded images preview area

### 5. Object Selection
**I Spy Mode Controls:**
- Distractors section (Select 8-12 images)
  - Theme selection dropdown
  - Selected items display
- Hidden Objects section (Select 1-5 images)
  - Theme selection dropdown
  - Selected items display

**Odd One Out Mode Controls:**
- Pairs section
  - Number of Pairs: 8, 9, 10, 11, or 12 pairs
  - Theme selection dropdown
  - Selected pairs display
- Odd Images section (Unpaired)
  - Number of Odd Images: 1, 2, or 3
  - Theme selection dropdown
  - Selected odd images display

## Toolbar Features
- **Layers**: Bring to Front, Bring Forward, Send Backward, Send to Back
- **Alignment**:
  - Align Selected: Left, Center H, Right, Top, Center V, Bottom
  - Align to Page: Center on Page Horizontally, Center on Page Vertically
- **Lock/Unlock** button
- **Delete** button

## Tabs
1. Worksheet tab
2. Answer Key tab

## Top Right Actions
- **Zoom Controls**: Zoom In, Zoom Out, Reset, Percentage display
- **History**: Undo (Ctrl+Z), Redo (Ctrl+Y)
- **Unlock All** button (appears when objects are locked)
- **Create dropdown**:
  - New Worksheet button
  - Answer Key button
- **Download dropdown**:
  - Worksheet (JPEG)
  - Answer Key (JPEG)
  - Worksheet (PDF)
  - Answer Key (PDF)
- **Grayscale toggle** checkbox
- **Clear All** button

## Languages Supported (11)
English, German (Deutsch), French (Français), Spanish (Español), Portuguese (Português), Italian (Italiano), Dutch (Nederlands), Swedish (Svenska), Danish (Dansk), Norwegian (Norsk), Finnish (Suomi)

- 3000+ child-friendly images
- Theme-based organization
- Search functionality
- Image browsing

- JPEG format

- Attention to detail
- Vocabulary building (image names)
