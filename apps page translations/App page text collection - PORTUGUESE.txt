1. NATURALIDADE: O texto soa como português original, não traduzido
2. ADAPTADO AO PÚBLICO: Linguagem apropriada para professores e pedagogos portugueses
3. COERÊNCIA: Terminologia uniforme em todo o texto
4. CONCISÃO: Formulações claras e significativas
5. MOTIVADOR: Botões CTA motivadores e precisos

Exemplos de decisões de tradução:

- "Word Search" → "Sopa de Letras" (termo português padrão, não "Procura de palavras")

- "Core Bundle" → "Pacote Essencial" (mais atrativo que "Pacote básico")
- "Full Access" → "Acesso Completo" (claro e direto)
- "MatchUp Maker" → "Jogo de Pares" (descritivo funcional)
- "Word Scramble" → "Palavras Baralhadas" (termo português natural)
- "Coloring Pages" → "Páginas para Colorir" (expressão portuguesa comum)
- "Bingo" → "Bingo de Imagens" (o termo "Bingo" mantém-se por ser universal)
- "Pattern" → "Padrões" (termo apropriado para este contexto educativo)
- "Fine Motor Skills" → "Motricidade Fina" (termo pedagógico português)
- "Language Arts" → "Português" (adaptado ao contexto português)
- "Odd One Out" → "O Intruso" (expressão portuguesa natural)
- "Shadow Match" → "Combinar Sombras" (descritivo e claro)
- "English Writing Practice" → "Exercícios de Escrita" (generalizado para o contexto português)
