## Como Criar Atividades para Imprimir Grid Match em 5 Passos Simples

### Passo 1: Escolha a Imagem Base para suas Atividades de Alfabetização e Coordenação Motora

O primeiro passo é selecionar a imagem que será dividida em peças. Você tem duas opções principais para criar atividades de alfabetização e coordenação motora. Use a biblioteca integrada com mais de 3000 imagens ou faça upload de suas próprias fotos.

Para usar a biblioteca, abra o painel "Biblioteca de Imagens" no menu lateral. Selecione um tema no menu suspenso para filtrar as opções. Temas incluem animais, alimentos, transportes e muito mais. Use a busca por palavras-chave para encontrar imagens específicas rapidamente.

Para upload de imagens próprias, clique em "Enviar Imagens Personalizadas". Selecione uma ou mais imagens do seu computador. Formatos aceitos incluem JPEG, PNG e GIF. Suas imagens aparecem na área de pré-visualização instantaneamente.

Clique na imagem desejada para selecioná-la. A imagem selecionada aparece destacada no painel. Esta será a base do seu quebra-cabeça Grid Match. Escolha imagens com elementos visuais claros para melhores resultados.

### Passo 2: Configure o Tamanho da Grade para Atividades 1º Ano e Atividades 2º Ano

Ajuste as configurações da grade para adequar a dificuldade ao nível dos alunos. Atividades 1º ano funcionam melhor com grades menores e menos peças removidas. Atividades 2º ano podem usar configurações mais desafiadoras.

Abra o painel "Opções de Grade" no menu lateral esquerdo. Configure o número de linhas entre 2 e 4. Configure o número de colunas entre 2 e 4. Uma grade 2x2 cria 4 peças, ideal para iniciantes. Uma grade 4x4 cria 16 peças para alunos avançados.

Defina o número de células de pista entre 1 e 5. Células de pista são peças que permanecem visíveis na imagem principal. Mais células de pista tornam a atividade mais fácil. Menos células aumentam o desafio de raciocínio espacial.

No painel "Configuração de Página", escolha o tamanho do papel. Opções incluem Carta Retrato, Carta Paisagem, A4 Retrato e A4 Paisagem. Dimensões personalizadas estão disponíveis para projetos especiais.

Cada geração cria um layout diferente de peças. Gere múltiplas vezes para criar versões variadas da mesma atividade. Isso evita que alunos copiem respostas uns dos outros. Perfeito para avaliações e atividades diferenciadas.
