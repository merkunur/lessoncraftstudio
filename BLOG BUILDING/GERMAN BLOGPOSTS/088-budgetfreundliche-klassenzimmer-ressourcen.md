## Einleitung: Die Budgetrealität im deutschen Schulalltag

**Lehreralltag in Deutschland**: Durchschnittlich geben Lehrkräfte **€600-800 pro Jahr** aus eigener Tasche für Klassenzimmermaterialien aus.

**Typische Ausgabenverteilung**:
```
Übliche Ausgaben pro Schuljahr:
- Arbeitshefte und Kopiervorlagen: €300 (Verbrauchsmaterial, nicht wiederverwendbar)
- Anschauungsmaterial: €150
- Klassenraumdekoration: €100
- Schreibwaren und Bastelmaterial: €200
Summe: €750 aus eigener Tasche
```

**Das Problem**: Arbeitshefte sind Einweg-Material (keine Wiederverwendung möglich)
```
Anschaffung: 30 Mathematik-Arbeitshefte × €10 = €300
Verwendung: September bis Juli (Schüler schreiben direkt rein)
Schuljahresende: Hefte gehen nach Hause (nicht wiederverwendbar)
Nächstes Jahr: Erneuter Kauf von 30 Arbeitsheften (wieder €300)
5-Jahres-Kosten: €1.500 für Verbrauchsmaterialien
```

**Die Lösung**: Digitale Generierung + Ausdruck = wiederverwendbare Ressourcen
```
Kosten: €144/Jahr Abonnement
Generierung: Unbegrenzte Arbeitsblätter (entspricht 100+ Arbeitsheften)
Wiederverwendung: PDFs speichern, nächstes Jahr erneut drucken (keine Zusatzkosten)
5-Jahres-Kosten: €720 (Ersparnis: €780 gegenüber Arbeitsheften)
```

## Strategien für wiederverwendbare Materialien

**Papierverbrauch reduzieren**

### Folienhüllen mit Whiteboard-Markern

**Einmalinvestition**: €50 für 30 Folienhüllen
```
Funktionsweise:
1. Arbeitsblatt ausdrucken (eine Kopie)
2. In Folienhülle einlegen
3. Schüler bearbeiten mit Whiteboard-Marker
4. Abwischen, 100+ Mal wiederverwenden

Beispiel:
Tägliche Mathe-Kopfrechenübung (5 Aufgaben):
- Ohne Hüllen: 30 Kopien × 180 Tage = 5.400 Blätter
- Mit Hüllen: 30 Kopien einmal, ganzes Jahr nutzen = 30 Blätter
Papierersparnis: 5.370 Blätter (€268,50 bei €0,05/Blatt)

ROI: €50 Investition spart €268,50/Jahr (5,4-fache Rendite)
```

---

### Laminierte Arbeitsblätter

**Für häufig verwendete Materialien**:
```
Laminieren: Einmaleins-Tafeln, ABC-Poster, Zahlenstrahle
Kosten: €20 Laminiergerät + €10 Laminierfolien = €30
Haltbarkeit: 5+ Jahre

Verwendung: Schüler nutzen mit Whiteboard-Markern
Beispiel: "Buchstaben nachspuren"-Arbeitsblatt
- Einmal drucken, laminieren, 200+ Mal verwenden über Jahre
- Kosten pro Verwendung: €0,05 ÷ 200 = €0,00025 (praktisch kostenlos)
```

## Papierspar-Strategien

**Druckkosten senken**

### Duplexdruck (beidseitig)

**Einfache Änderung, große Ersparnis**:
```
Einseitig: 180 Arbeitsblätter = 180 Blätter
Duplex (beidseitig): 180 Arbeitsblätter = 90 Blätter
Ersparnis: 90 Blätter pro Schüler = 2.700 Blätter/Klasse = €135/Jahr

Einstellung: Druckereigenschaften → Beidseitiger Druck → "Kurzkante" (Umschlagen)
```

---

### Projektion + Heftarbeit

**Kopieren eliminieren**:
```
Traditionell: 30 Kopien eines Arbeitsblatts ausdrucken
Alternative: EIN Arbeitsblatt am Beamer zeigen, Schüler übertragen ins Heft

Beispiel:
Rechenaufgaben: 20 Aufgaben am Projektor anzeigen
Schüler: Aufgaben + Lösungen ins Matheheft schreiben
Papierverbrauch: 0 Blätter

Geeignet für: Rechenübungen, Vokabelarbeit, Leseverständnisfragen
Nicht geeignet für: Komplexe Diagramme, lange Texte
```
