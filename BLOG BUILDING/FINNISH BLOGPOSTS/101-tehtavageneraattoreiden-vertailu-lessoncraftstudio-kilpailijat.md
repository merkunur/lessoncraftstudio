## Nopeusvertailu

**Aika yhden tehtävämonisteen luomiseen**

### LessonCraftStudio: 42 sekuntia

**Prosessi**:
```
1. Valitse generaattori (5 s)
2. Syötä parametrit (20 s)
   - Aihe/sanasto
   - Vaikeustaso
   - Ulkoasuvalinnat
3. Luo tehtävä (2 s)
4. Lataa PDF (15 s)

Yhteensä: keskimäärin 42 sekuntia
Sisältää: Täysin räätälöidyn, tulostusvalmiita tehtävämonisteen

Esimerkki: Matematiikan sanallisia tehtäviä
- Syötä: 20 tehtävää, yhteen- ja vähennyslasku, 2-numeroiset luvut
- Luo: 42 sekuntia
- Tulos: Räätälöity tehtävämoniste ainutlaatuisilla tehtävillä
```

---

### Education.com: 2-3 minuuttia

**Prosessi**:
```
1. Selaa kirjastoa (30-60 s sopivan tehtävän löytämiseen)
2. Esikatsele tehtävä (15 s)
3. Räätälöi tarvittaessa (30-45 s)
4. Lataa (15 s)

Yhteensä: 2-3 minuuttia
Rajoitus: Valinta valmiista valikoimasta (ei täysin räätälöitävissä)
```

---

### Teachers Pay Teachers: 5-15 minuuttia

**Prosessi**:
```
1. Etsi tehtävätyyppi (2-5 min selausta)
2. Tutki useita vaihtoehtoja (3-5 min)
3. Osta (1 min)
4. Lataa (30 s)
5. Usein muokkaus tarpeen (3-5 min PowerPointissa)

Yhteensä: 5-15 minuuttia
Edut: Korkealaatuista, opettajien tekemää
Haitat: Aikaa vievää etsintää, maksaa jokaisesta materiaalista
```

---

### Twinkl: 1-2 minuuttia

**Prosessi**:
```
1. Hae tietokannasta (30 s)
2. Esikatselu (15 s)
3. Räätälöi pohjaa (30-45 s)
4. Lataa (15 s)

Yhteensä: 1-2 minuuttia
Huom: Valmiita pohjia (vähemmän joustava kuin generaattori)
```

---

### WorksheetWorks.com: 1 minuutti

**Prosessi**:
```
1. Valitse generaattori (10 s)
2. Aseta parametrit (30 s)
3. Luo + lataa (20 s)

Yhteensä: ~1 minuutti
Vertailukelpoinen nopeus: Samankaltainen kuin LessonCraftStudio
```

---

### Super Teacher Worksheets: 1-2 minuuttia

**Prosessi**:
```
1. Selaa luokka-asteen/aineen mukaan (30-60 s)
2. Valitse tehtävä (15 s)
3. Tulosta/lataa (15 s)

Yhteensä: 1-2 minuuttia
Rajoitus: Kiinteät tehtävät (ei räätälöintiä)
```

**Nopeuden voittaja**: LessonCraftStudio & WorksheetWorks (nopein räätälöintivaihtoehdoilla)

## Valikoiman ja ominaisuuksien vertailu

**Generaattoreiden määrä ja tyypit**

### LessonCraftStudio: 33 generaattoria

**Kategoriat**:
```
Matematiikka (8 generaattoria):
- Yhteenlasku, vähennyslasku, kertolasku, matematiikkapulmat, sudoku jne.

Lukutaito (12 generaattoria):
- Sanahaku, ristikko, sanasto, kirjoitus, oikeinkirjoitus jne.

Visuaalinen oppiminen (7 generaattoria):
- Kuvabingo, etsi esineitä, väritys, viivojen piirtäminen jne.

Logiikka ja kriittinen ajattelu (6 generaattoria):
- Hahmontunnistus, yhdistäminen, lajittelu, sokkelot jne.

Yhteensä: 33 ainutlaatuista generaattorityyppiä
Räätälöinti: Täysi hallinta sisällöstä, vaikeustasosta ja ulkoasusta
```

---

### Education.com: Kirjastopohjainen (yli 10 000 valmista)

**Muoto**:
```
Ei generaattoreita: Valmiiden tehtävämonisteiden kirjasto
Järjestys: Luokka-asteen, aineen, taidon mukaan
Räätälöinti: Rajoitettu (jonkin verran muokkausta mahdollista)

Edut: Valtava valikoima valmiita tehtäviä
Haitat: Ei voi luoda uutta sisältöä tarpeen mukaan
```

---

### Teachers Pay Teachers: Markkinapaikka (miljoonia resursseja)

**Muoto**:
```
Ei generaattoreita: Opettajien luomien materiaalien markkinapaikka
Valikoima: Tehtävämonisteet, oppituntisuunnitelmat, aktiviteetit, projektit
Räätälöinti: Riippuu myyjästä (osa muokattavissa, osa ei)

Edut: Erittäin monipuolinen, korkealaatuinen
Haitat: Pitää ostaa erikseen, epäyhtenäinen laatu
```

---

### Twinkl: Yhdistelmä (generaattoreita + kirjasto)

**Muoto**:
```
Generaattoreita: ~20 perusgeneraattoria
Kirjasto: Yli 1 000 000 valmista resurssia
Räätälöinti: Pohjia voi muokata

Edut: Yhdistelmälähestymistapa (joustavuus + valmis)
Haitat: Tilaus vaaditaan täyteen pääsyyn
```

---

### WorksheetWorks.com: Yli 400 generaattoria

**Muoto**:
```
Vain generaattoreita: Yli 400 erikoistunutta generaattoria
Valikoima: Matematiikka, kielet, maantiede, logiikka, pulmat
Räätälöinti: Laajat parametrit jokaiseen generaattoriin

Edut: Eniten generaattoreita saatavilla
Haitat: Vanhanaikainen käyttöliittymä, ylivoimainen määrä vaihtoehtoja
```

---

### Super Teacher Worksheets: Yli 20 000 valmista

**Muoto**:
```
Ei generaattoreita: Valmiiden tehtävämonisteiden kirjasto
Järjestys: Luokka-asteen ja aineen mukaan
Räätälöinti: Ei lainkaan (tulostetaan sellaisenaan)

Edut: Yksinkertainen, suoraviivainen
Haitat: Ei räätälöintiä, kiinteä sisältö
```

**Valikoiman voittaja**: WorksheetWorks (eniten generaattoreita), LessonCraftStudio (33 korkealaatuista generaattoria erinomaisella käyttökokemuksella)

**Laatuominaisuudet**:
```

Fontit: Ammattimaiset, ikätasoiset
Grafiikat: Korkealaatuiset kuvat (tarvittaessa)
Tyhjä tila: Tasapainoinen (ei ahtautunut)
Tulostuslaatu: Terävä, selkeä PDF

Opettajapalaute: "Näyttää kustantajan julkaisemalta"
Oppilaiden reaktio: Kiinnostava, ammattimainen ulkoasu
```

---

### Education.com: Korkealaatuinen, kuvitettu

**Vahvuudet**:
```
Visuaalit: Värikäs, kiinnostava grafiikka
Ammattimainen: Koulutustiimin suunnittelema
Haitat: Jotkut tehtävät liian täynnä (liikaa grafiikkaa)
```

---

### Teachers Pay Teachers: Vaihteleva

**Laatujakauma**:
```
Paras: Poikkeuksellinen, ammattimainen opettajasuunnittelijat
Huonoin: Amatöörimainen, epäjohdonmukainen muotoilu
Todellisuus: Pitää arvioida jokainen resurssi erikseen

Etu: Voi lukea arvostelut ennen ostamista
```

---

### Twinkl: Ammattimainen, brittiläinen tyyli

**Ominaisuudet**:
```

```

Haitta: Näyttää vähemmän ammattimaiselta kuin kilpailijat
```

### Super Teacher Worksheets: Selkeä, yksinkertainen

**Muotoilu**:
```
Tyyli: Suoraviivainen, selkeä
Grafiikat: Minimaalinen, sopiva

Etu: Oppilaan on helppo keskittyä sisältöön (ei häiriöitä muotoilusta)

**Laadun voittaja**: LessonCraftStudio, Education.com, Twinkl (moderni, ammattimainen)

## Räätälöinnin joustavuus

**Kuinka paljon opettajilla on hallintaa?**

### LessonCraftStudio: Laaja räätälöinti

**Vaihtoehdot per tehtävä**:
```
Sisältö: Syötä oma sanasto, tehtävät jne.
Vaikeustaso: Säädä monimutkaisuustasoa
Ulkoasu: Ruudukon koko, tehtävien määrä, muoto
Kuvat: Valitse kirjastosta (kuvageneraattorit)
Kieli: Useita kieliä tuettu

Esimerkki: Sanahaku
- Oma sanalistasi (mitkä tahansa sanat)
- Ruudukon koko (10×10, 15×15, 20×20)
- Vaikeustaso (helppo/keskitaso/vaikea)
- Tulostusvaihtoehdot (sanalistan kanssa/ilman)

Joustavuus: Korkea (luo juuri mitä tarvitset)
```

---

### Education.com: Rajoitettu

**Räätälöinti**:
```
Useimmat tehtävät: Tulosta sellaisenaan (ei muutoksia)
Jotkut tehtävät: Muokkaa vain tekstikenttiä
Yleisesti: Vähäinen räätälöinti
```

---

### Teachers Pay Teachers: Riippuu myyjästä

**Vaihtelee**:
```
Jotkut: Täysin muokattavissa PowerPoint/Google Docs
Toiset: Vain PDF (ei muokkausta)
Pitää tarkistaa: Tuotekuvaus muokattavuudesta
```

---

### Twinkl: Kohtalainen

**Muokkaus**:
```
Pohjat: Voi muokata tekstiä, joitakin ulkoasumuutoksia
Generaattorit: Parametrien säätöjä
Yleisesti: Kohtalainen räätälöinti
```

---

### WorksheetWorks.com: Laaja

**Parametrit**:
```
Satoja vaihtoehtoja per generaattori:
- Numerovälit
- Laskutoimitusten tyypit
- Muotoiluasetukset
- Vaikeustason säädöt

Joustavuus: Erittäin korkea (joskus ylivoimainen)
```

---

### Super Teacher Worksheets: Ei lainkaan

**Kiinteä**:
```
Tulosta sellaisenaan: Ei räätälöintiä saatavilla
Etu: Nopea (ei päätöksiä tarvita)
Haitta: Ei välttämättä vastaa tarkkoja tarpeitasi
```

**Räätälöinnin voittaja**: LessonCraftStudio & WorksheetWorks (erittäin joustava)
