# Wordsearch App - Feature Analysis (Source: REFERENCE APPS/wordsearch.html)

## Accordion Sections (8 Total)

### 1. Language Settings
- **11 Content Languages**: English, German, French, Spanish, Portuguese, Italian, Dutch, Swedish, Danish, Norwegian, Finnish
- Image library language selector
- Image names and themes display in selected language

### 2. Page & Scene
**Page Setup:**
- Letter Portrait (8.5×11")
- Letter Landscape (11×8.5")
- A4 Portrait (210×297mm)
- A4 Landscape (297×210mm)
- Custom (user-defined dimensions in pixels)

**Background:**
- Fallback color picker
- Background theme selection
- Background opacity slider (0-1)

**Border:**
- Border theme selection
- Border opacity slider (0-1)

### 3. Text Tools
**Add New Text:**
- Text input field
- Add text button

**Selected Text Properties:**
- Color picker
- Font size (min 8px)
- Font family: Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana
- Outline color picker
- Outline width slider (0-10)

### 4. Puzzle Settings
**Grid Size:**
- Rows: 5-30 (default 12)
- Columns: 5-30 (default 12)

**Puzzle Options (Checkboxes):**
- Allow Diagonal Words (checked by default)
- Allow Reverse Words
- Show Word/Image List (checked by default)
- Show only images
- Show only words

### 5. Image Library
**Image Source for Puzzle:**
- Theme selection (random or specific theme)

**Individual Image Selection:**
- Filter by theme dropdown
- Search images input field
- Available images display (max 8)
- Selected images preview

**3000+ child-friendly images**
- Theme-based organization
- Search functionality

### 6. Upload Custom Images
- Multi-file upload button
- Accepts: image/* (all common image formats)
- Uploaded images preview (session-based)

### 7. Manual Image Name Editing
- Enable/disable checkbox
- Click to select images from library
- Edit names before generating puzzle
- Manual edit images preview area

### 8. Custom Word List
- Enable/disable checkbox
- Text-only puzzles (no images)
- Textarea for word input (one per line, max 8 words)
- Only letters allowed - special characters/numbers removed

## Canvas Editing Features
- Drag, rotate, scale, delete any element
- Full editability of all objects
- Layer management (bring to front/forward, send backward/back)
- Alignment tools (align selected, align to page)
- Undo/Redo functionality
- Lock/Unlock all objects

## Generate Options
- Generate New Worksheet
- Generate Answer Key (automatically created)

## Download Options
**Formats:**
- Worksheet (JPEG)
- Worksheet (PDF)
- Answer Key (JPEG)
- Answer Key (PDF)

**Quality:**
- 300 DPI professional quality
- Grayscale toggle (save ink)

## Tab System
- Worksheet tab (main editing)
- Answer Key tab (preview answer key)

## Key Value Propositions
1. **11 Language Support** - Critical for language learning and multilingual education
2. **3000+ Image Library** - Theme-based, searchable
3. **Custom Image Upload** - Multi-file, any format
4. **Manual Image Editing** - Edit names before generating
5. **Custom Word Lists** - Text-only puzzles
6. **Full Canvas Editability** - Drag, rotate, scale everything
7. **Professional Quality** - 300 DPI exports
8. **Flexible Grid** - 5x5 to 30x30
9. **Answer Key Auto-Generation** - Included with every puzzle
10. **Multiple Export Options** - JPEG/PDF for worksheet and answer key
