## Introduction : Planification Traditionnelle vs Planification à Rebours

**La planification traditionnelle** (conception linéaire) :
```
Étape 1 : Choisir des activités intéressantes (fiches ludiques, manipulations)
Étape 2 : Animer les séances
Étape 3 : Créer l'évaluation en fin de séquence
Problème : Les activités ne correspondent pas forcément à l'évaluation
Résultat : Les élèves font des activités mais ne maîtrisent pas les compétences visées
```

**La planification à rebours** (Understanding by Design - Wiggins & McTighe) :
```
Étape 1 : Définir les objectifs d'apprentissage (que doivent SAVOIR les élèves ?)
Étape 2 : Déterminer les preuves d'apprentissage (comment DÉMONTRER la maîtrise ?)
Étape 3 : Concevoir les activités d'apprentissage (quelles activités pour Y PARVENIR ?)
Résultat : Chaque activité construit méthodiquement la compétence attendue
```

**Les recherches** (Wiggins & McTighe, 2005 - Understanding by Design) démontrent que la planification à rebours augmente la réussite des élèves de **15 à 25 %** en garantissant la cohérence entre objectifs, enseignement et évaluation.

**Le principe fondamental** : Commencer par la FIN (l'évaluation), puis remonter vers les apprentissages.

## Les 3 Étapes de la Planification à Rebours

### Étape 1 : Identifier les Résultats Attendus

**La question centrale** : Que doivent savoir et savoir-faire les élèves ?

**Exemple de séquence** : Les fractions en CM1

**Attendus du programme** (Bulletin Officiel) :
```
• Comprendre la notion de fraction
• Repérer et placer des fractions sur une droite graduée
• Encadrer une fraction par deux nombres entiers consécutifs
• Établir des égalités entre des fractions usuelles
• Comparer des fractions de même dénominateur
```

**Compétences durables** (savoirs essentiels que les élèves retiendront) :
```
1. Les fractions représentent des parts d'un tout
2. Plusieurs fractions peuvent représenter la même quantité (équivalence)
3. Les fractions se comparent en trouvant un dénominateur commun
4. Les fractions servent dans la vie quotidienne (recettes, partages, mesures)
```

**Questions essentielles** :
```
1. Comment deux fractions différentes peuvent-elles être égales ?
2. Quand utilise-t-on les fractions au quotidien ?
3. Comment savoir quelle fraction est la plus grande ?
4. Pourquoi 1/2 = 2/4 = 4/8 ?
```

---

### Étape 2 : Déterminer les Preuves d'Apprentissage Acceptables

**La question centrale** : Comment les élèves démontreront-ils leur maîtrise ?

**L'évaluation finale** (conçue AVANT l'enseignement) :

**Partie 1 : Démonstration des compétences** (20 points)
```
1. Dessine un schéma montrant que 2/4 = 1/2 (preuve visuelle)
2. Compare 3/5 et 2/3 (montre ton raisonnement avec un dénominateur commun)
3. Place 7/4 sur une droite graduée (précision du placement)
4. Cite 3 situations de la vie courante où tu utilises des fractions (application)
```

**Partie 2 : Résolution de problèmes** (10 points)
```
Problème : "Tu as mangé 2/6 d'une pizza. Ton camarade a mangé 3/9 de la même pizza.
Qui a mangé le plus ? Prouve ta réponse avec un schéma."

Barème :
- 4 points : Réponse correcte avec preuve visuelle
- 3 points : Réponse correcte sans justification
- 2 points : Réponse incorrecte mais démarche cohérente
- 0-1 point : Aucun raisonnement visible
```

**Critère de maîtrise** : 80 % de réussite = compétence acquise (24/30 points)

**Maintenant** : Concevoir l'enseignement pour que les élèves PUISSENT réussir ces tâches.

---

### Étape 3 : Planifier les Expériences d'Apprentissage

**La question centrale** : Quelles activités prépareront les élèves à l'évaluation ?

**Semaine 1 : Introduction aux fractions**
```
Lundi : Manipulation avec des parts de gâteaux (concret)
Mardi : Fiche - colorier des fractions (représentation)
Mercredi : Mots mêlés de vocabulaire (numérateur, dénominateur, équivalent)
Jeudi : Coloriage fractionné (1/2 en rouge, 1/4 en bleu)
Vendredi : Évaluation formative (5 exercices - vérification de la compréhension)

Objectif : Les élèves savent identifier et représenter des fractions
```

**Semaine 2 : Fractions équivalentes**
```
Lundi : Pliage de papier (1/2 = 2/4 = 4/8 physiquement)
Mardi : Fiche - entraînement aux fractions équivalentes (15 exercices)
Mercredi : Mots croisés illustrés (vocabulaire : équivalent, simplifier)
Jeudi : Puzzle mathématique (trouver les fractions équivalentes dans le système)
Vendredi : Évaluation formative (5 exercices - équivalence uniquement)

Objectif : Les élèves comprennent pourquoi 2/4 = 1/2
```

**Semaine 3 : Comparaison de fractions**
```
Lundi : Bandes fractionnées (comparaison visuelle)
Mardi : Fiche - comparer des fractions avec dénominateurs identiques (20 exercices)
Mercredi : Fiche - comparer des fractions (trouver le dénominateur commun)
Jeudi : Problèmes écrits (laquelle est la plus grande ?)
Vendredi : Évaluation formative (5 exercices - comparaison)

Objectif : Les élèves peuvent comparer n'importe quelles deux fractions
```

**Semaine 4 : Addition et soustraction**
```
Lundi : Addition de fractions avec supports visuels
Mardi : Fiche - addition avec dénominateurs identiques (20 exercices)
Mercredi : Fiche - soustraction avec dénominateurs identiques (20 exercices)
Jeudi : Entraînement aux opérations mixtes
Vendredi : ÉVALUATION DE SÉQUENCE (l'évaluation conçue à l'Étape 2)

Objectif : Les élèves maîtrisent l'addition/soustraction, prêts pour l'évaluation
```

**Vérification de l'alignement** : Chaque activité développe les compétences nécessaires pour l'évaluation finale.

### Progression Annuelle en Mathématiques (Exemple CM1)

**Septembre : Numération et calcul mental**
```
Compétences : Addition/soustraction de grands nombres, arrondis
Fiches clés :
- Semaine 1 : Addition à 4 chiffres (20 exercices quotidiens)
- Semaine 2 : Soustraction à 4 chiffres (20 exercices quotidiens)
- Semaine 3 : Arrondis à la dizaine, centaine (mots mêlés avec nombres)
- Semaine 4 : Problèmes écrits (opérations mixtes)
Évaluation : Test de 25 exercices (objectif de maîtrise : 80 %)
```

**Octobre : Tables de multiplication et stratégies**
```
Compétences : Multiplier jusqu'à 100, modèles en aires
Fiches clés :
- Semaine 1 : Tables de 0 à 5 (bingo illustré pour l'entraînement)
- Semaine 2 : Tables de 6 à 10 (fiches d'entraînement quotidiennes)
- Semaine 3 : Modèles en aires (multiplication sur quadrillage)
- Semaine 4 : Problèmes écrits (applications de la multiplication)
Évaluation : Test chronométré (100 calculs en 5 minutes, objectif 80 %)
```

**Novembre : Division**
```
Compétences : Division jusqu'à 100, restes
Fiches clés :
- Semaine 1 : Tables de division (inverse de la multiplication)
- Semaine 2 : Division posée (2 chiffres ÷ 1 chiffre)
- Semaine 3 : Division avec reste
- Semaine 4 : Problèmes de division
Évaluation : Test de 30 divisions
```

**Décembre : Fractions (introduction)**
```
Compétences : Comprendre les fractions comme parts d'un tout
Fiches clés :
- Semaine 1 : Identifier les fractions (colorier des modèles)
- Semaine 2 : Fractions équivalentes (modèles visuels)
- Semaine 3 : Comparer des fractions (même dénominateur)
Évaluation : Test visuel sur les fractions (dessiner et comparer)
```

**Janvier : Fractions (opérations)**
```
Compétences : Additionner/soustraire des fractions de même dénominateur
Fiches clés :
- Semaines 1-2 : Addition de fractions (modèles + symbolique)
- Semaines 3-4 : Soustraction de fractions
Évaluation : Test de 25 opérations sur les fractions
```

**Février : Nombres décimaux**
```
Compétences : Écriture décimale, valeur positionnelle aux centièmes
Fiches clés :
- Semaine 1 : Lire/écrire les décimaux
- Semaine 2 : Comparer les décimaux
- Semaine 3 : Additionner les décimaux
- Semaine 4 : Soustraire les décimaux
Évaluation : Test d'opérations décimales
```

**Mars : Grandeurs et mesures**
```
Compétences : Conversions d'unités, tableaux de données
Fiches clés :
- Semaine 1 : Conversion d'unités (cm/m, g/kg)
- Semaine 2 : Diagrammes en bâtons (créer à partir de données)
- Semaine 3 : Analyse de données (mots mêlés avec termes de mesure)
Évaluation : Test de mesures + interprétation de données
```

**Avril : Géométrie**
```
Compétences : Angles, figures, droites, symétrie
Fiches clés :
- Semaine 1 : Mesure d'angles (pratique avec rapporteur)
- Semaine 2 : Types de triangles/quadrilatères (classification)
- Semaine 3 : Droites (parallèles, perpendiculaires, sécantes)
- Semaine 4 : Symétrie (tracer des axes de symétrie)
Évaluation : Test de géométrie + constructions
```

**Mai : Révisions et approfondissement**
```
Focus : Révision spiralée du contenu annuel
Fiches clés :
- Semaine 1 : Révision des opérations (entraînement mixte)
- Semaine 2 : Révision fractions/décimaux
- Semaine 3 : Révision mesures/géométrie
- Semaine 4 : Préparation évaluations nationales (si applicable)
Évaluation : Examen final complet
```

   nécessitent la comparaison de fractions

## L'Enseignement Guidé par l'Évaluation

**Le principe** : Les activités d'entraînement doivent ressembler à l'évaluation

**L'incohérence traditionnelle** :
```
Entraînement : Fiches à réponse ouverte (écrire la réponse)
Évaluation : QCM (cocher des cases)
Résultat : Choc du format le jour de l'évaluation (charge cognitive gaspillée)
```

**L'entraînement aligné** :
```
Entraînement : Même format que l'évaluation
Exemple : Si l'évaluation est un QCM, s'entraîner avec des QCM
Résultat : Familiarité avec le format (charge cognitive centrée sur le contenu)
```

## Développement du Guide de Progression

**Le guide de progression** : Calendrier pour couvrir le programme

**Objectif** : S'assurer de terminer le programme avant la fin de l'année

### Créer un Guide de Progression

**Étape 1 : Compter les jours d'enseignement**
```
Année scolaire : 162 jours (France)
- Vacances scolaires : déjà déduites
- Jours d'évaluations nationales : 5 jours
- Sorties scolaires/événements : 7 jours
- Jours tampons (absences, imprévus) : 10 jours
Jours d'enseignement effectifs : 140 jours
```

**Étape 2 : Répartir les jours par compétences**
```
Mathématiques CM1 : 35 compétences principales
140 jours ÷ 35 compétences = 4 jours par compétence (moyenne)

Ajuster selon la complexité :
- Compétences simples (arrondis) : 2 jours
- Compétences complexes (opérations sur fractions) : 8 jours
```

**Étape 3 : Créer le planning hebdomadaire**
```
Semaine 1 (2-6 sept) : Numération et valeur positionnelle
Semaine 2 (9-13 sept) : Addition de grands nombres
Semaine 3 (16-20 sept) : Soustraction de grands nombres
...

Résultat : Chaque compétence a un temps dédié
```

**Avantage des générateurs** : Créer rapidement les supports pour chaque compétence
```
Focus semaine 2 : Addition de grands nombres
Générer : 5 fiches d'addition (3,5 minutes)
Imprimer : 30 exemplaires chacune (géré par l'imprimante)
Prêt : Supports de la semaine 2 terminés
```

## La Différenciation dans la Planification à Rebours

**Le défi** : Tous les élèves ne maîtrisent pas au même rythme

**La solution** : Objectifs échelonnés

### Niveaux de Maîtrise Échelonnés

**Niveau 1 : Maîtrise de base** (80 % des élèves)
```
Objectif : Comparer des fractions de même dénominateur
Évaluation : 3/4 vs 2/4 (laquelle est la plus grande ?)
```

**Niveau 2 : Maîtrise satisfaisante** (60 % des élèves)
```
Objectif : Comparer des fractions de dénominateurs différents
Évaluation : 3/4 vs 2/3 (utiliser le dénominateur commun)
```

**Niveau 3 : Maîtrise approfondie** (20 % des élèves)
```
Objectif : Comparer trois fractions ou plus, expliquer le raisonnement
Évaluation : Ranger 2/3, 3/4, 5/6 du plus petit au plus grand, justifier
```

**L'enseignement** : Fournir des supports pour les trois niveaux
```
Lundi : Tous les élèves - Comparaison de base (dénominateurs identiques)
Mardi : Tous les élèves - Introduction dénominateurs différents
Mercredi : Niveau 1 - Plus d'entraînement dénominateurs identiques (Atelier 1)
          Niveau 2 - Entraînement dénominateurs différents (Atelier 2)
          Niveau 3 - Comparaison avancée de multiples fractions (Atelier 3)
```

**Utilisation des générateurs** : Créer 3 niveaux instantanément
```
Fiche niveau 1 : Comparer des fractions (même dénominateur uniquement)
Fiche niveau 2 : Comparer des fractions (trouver PPCM)
Fiche niveau 3 : Ordonner 4+ fractions (complexe)

Temps : 126 secondes (42 sec chacune) pour les 3 niveaux
```

## Protocole de Réflexion et Révision

**Après la séquence** : Réfléchir et améliorer

### Analyse Post-Séquence

**Questions à se poser** :
```
1. Les élèves ont-ils maîtrisé les objectifs ? (vérifier les résultats d'évaluation)
   - Si oui : Conserver la séquence telle quelle
   - Si non : Qu'est-ce qui nécessite un réenseignement ?

2. Quelles activités ont été les plus efficaces ?
   - Engagement maximum : [lister les activités]
   - Meilleurs résultats d'apprentissage : [lister les activités]
   - Conserver celles-ci, éliminer les inefficaces

3. Le rythme était-il adapté ?
   - Trop rapide : Ajouter 1-2 jours l'année prochaine
   - Trop lent : Réduire de 1-2 jours l'année prochaine

4. Que vais-je modifier l'année prochaine ?
   - [révisions spécifiques]
```

**Exemple de réflexion** :
```
Séquence : Les fractions (4 semaines)
Résultats d'évaluation : 85 % de maîtrise (au-dessus de l'objectif de 80 %) ✓

Activité la plus efficace : Pliage de papier (semaine 2, lundi)
- 100 % d'engagement
- Les élèves ont eu un "déclic"
- À CONSERVER

Activité la moins efficace : Mots mêlés de fractions (semaine 1, mercredi)
- Élèves ennuyés (trop simple)
- À REMPLACER par sudoku de fractions (plus stimulant)

Rythme : Juste ce qu'il faut (terminé vendredi comme prévu)

Modifications pour l'année prochaine :
- Remplacer mots mêlés par sudoku
- Ajouter une vérification formative en semaine 3
  (certains élèves encore confus jeudi)
```
