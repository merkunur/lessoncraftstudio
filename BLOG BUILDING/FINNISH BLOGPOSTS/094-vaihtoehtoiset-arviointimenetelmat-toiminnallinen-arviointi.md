## Johdanto: Perinteisten kokeiden rajoitukset

**Perinteinen arviointi**:
```
Muoto: Monivalinta + täydennä tyhjät kohdat -koe
Mitä se mittaa: Tunnistamista ja muistamista
Mitä se ei näytä: Soveltamista, luovuutta, ongelmanratkaisua, viestintää

Esimerkki:
Kysymys: "Mikä on 8 × 7?"
Oppilas: Kirjoittaa "56"
Todiste: Oppilas tietää tämän laskutoimituksen

MUTTA: Koe ei näytä, ymmärtääkö oppilas MITÄ kertolaskutoimitus tarkoittaa tai OSAAKO soveltaa sitä todellisiin tilanteisiin
```

**Tutkimus** (Wiggins, 1990): Autenttinen arviointi ennustaa paremmin **käytännön osaamista** kuin perinteiset kokeet

**Vaihtoehtoinen arviointi**:
```
Muoto: Toiminnallinen tehtävä, projekti, osaamisen osoitus
Mitä se mittaa: Soveltamista, synteesiä, ongelmanratkaisua
Mitä se paljastaa: Syvällistä ymmärrystä + taitoja

Esimerkki:
Tehtävä: "Suunnittele puutarha, jonka koko on 8 metriä × 7 metriä. Laske pinta-ala. Piirrä mittakaavaan. Suunnittele, mitä istutettaisiin."
Oppilas: Soveltaa kertolaskua, käyttää avaruudellista päättelyä, tekee päätöksiä, viestii visuaalisesti

Todiste: Oppilas YMMÄRTÄÄ kertolaskun pinta-alan laskemisena JA osaa soveltaa sitä autenttisessa yhteydessä
```

**Keskeinen periaate**: Arvioinnin tulisi heijastaa sitä, miten tietoa käytetään todellisessa maailmassa

## Toiminnalliset tehtävät

**Oppilaat osoittavat oppimisen TEKEMÄLLÄ**

### Matematiikan toiminnallinen tehtävä

**Tavoite**: Pinta-alan ja ympärysmitan mittaaminen (4. luokka)

**Perinteinen koe**:
```
Kysymys 1: Suorakulmion sivut ovat 6 cm × 4 cm. Mikä on pinta-ala?
Kysymys 2: Mikä on ympärysmitta?

Mittaa: Osaako oppilas laskea kaavoilla
Puuttuu: Ymmärrys siitä, mitä pinta-ala/ympärysmitta tarkoittavat käytännössä
```

**Toiminnallinen vaihtoehto**:
```
Tehtävä: "Suunnittele unelmiesi makuuhuone"

Ohjeet:
1. Valitse mitat (huoneen tulee olla 10-15 metriä × 10-15 metriä)
2. Laske pinta-ala (näytä laskutoimitus)
3. Laske ympärysmitta (näytä laskutoimitus)
4. Piirrä huone mittakaavaan (1 cm = 2 metriä)
5. Lisää huonekalut mittojen kanssa (sänky, työpöytä, lipasto)
6. Selitä: "Mahtuuko huonekalut? Kuinka paljon lattiapinta-alaa jää?"

Arviointikriteerit:
- Oikea pinta-alan laskeminen (2 pistettä)
- Oikea ympärysmitan laskeminen (2 pistettä)
- Mittakaavaan piirretty pohjapiirros (3 pistettä)
- Realistiset huonekalujen mitat (2 pistettä)
- Tilankäytön selitys (3 pistettä)
Yhteensä: 12 pistettä

Mitä se paljastaa:
- Osaa soveltaa kaavoja oikein (laskutoimitukset)
- Ymmärtää avaruudellista päättelyä (huonekalujen sijoittelu)
- Osaa viestiä matemaattista ajattelua (selitys)
- Tekee käytännön yhteyksiä (soveltaminen)
```

**Tehtäväpohja**: Suunnittelupohja (luotu 42 sekunnissa)
```
Huoneen suunnittelulomake

Huoneeni mitat: ____ metriä × ____ metriä

Pinta-alan laskeminen:
____ × ____ = ____ neliömetriä

Ympärysmitan laskeminen:
(____ + ____) × 2 = ____ metriä

Huonekalulista:
Esine: _______ Koko: ____ × ____
Esine: _______ Koko: ____ × ____
Esine: _______ Koko: ____ × ____

Mahtuuko kaikki? Selitä:
_________________________________________________
```

### Luonnontieteen toiminnallinen tehtävä

**Tavoite**: Kasvien kasvutarpeet (3. luokka)

**Perinteinen koe**:
```
Monivalinta:
"Mitä kasvit tarvitsevat kasvaakseen?"
a) Vettä ja multaa
b) Vettä ja auringonvaloa
c) Vettä, auringonvaloa ja ravinteita
d) Vain vettä

Mittaa: Osaako oppilas muistaa faktan
Puuttuu: Ymmärrys MIKSI kasvit tarvitsevat näitä asioita
```

**Toiminnallinen vaihtoehto**:
```
Tehtävä: "Kasvien kasvukoe"

Viikko 1: Kokeen aloitus
- Kylvä 3 pavunsiementä kuppeihin
- Kuppi A: Vesi + auringonvalo + multa
- Kuppi B: Vesi + pimeys + multa
- Kuppi C: Ei vettä + auringonvalo + multa

Viikko 2-4: Havainnointi ja kirjaaminen (mittauspohja)
Päivämäärä | Kuppi A korkeus | Kuppi B korkeus | Kuppi C korkeus | Huomiot
...

Viikko 4: Analyysipohja
1. Mikä kasvi kasvoi pisimmäksi? Miksi?
2. Mitä tapahtui kupille B (ei valoa)? Miksi?
3. Mitä tapahtui kupille C (ei vettä)? Miksi?
4. Tee johtopäätökset: Mitä kasvit tarvitsevat? Todista tiedoillasi.

Arviointi:
- Tarkat mittaukset kirjattu (3 pistettä)
- Yksityiskohtaiset havainnot (2 pistettä)
- Johtopäätökset tuettu tiedoilla (4 pistettä)
- Tieteellinen päättely selkeää (3 pistettä)
Yhteensä: 12 pistettä

Mitä se paljastaa:
- Osaa suorittaa kokeen (tieteellinen prosessi)
- Osaa kerätä tietoa tarkasti (mittaustaidot)
- Osaa analysoida tuloksia (kriittinen ajattelu)
- Ymmärtää syy-seuraussuhteet (käsitteellinen tieto)
```

## Portfolio-arviointi

**Kerää työnäytteitä ajan mittaan osoittaaksesi edistymisen**

### Lukemisen portfolio

**Sisältö** (syyskuu - toukokuu):
```
Osio 1: Lukupäiväkirjat (seuraa luettuja kirjoja)
- Syyskuu: 2 kirjaa
- Tammikuu: 5 kirjaa
- Toukokuu: 8 kirjaa
Näkyvä kehitys: Lukumäärä kasvoi

Osio 2: Ymmärtämisnäytteet
- Syyskuu: Yksinkertainen kuvakirjan tiivistelmä
- Tammikuu: Hahmoanalyysi kertauskirjasta
- Toukokuu: Vertailu kahden romaanin välillä
Näkyvä kehitys: Monimutkaisuus lisääntyi

Osio 3: Sanasto
- Syyskuu: 20 näköä-sanaa hallittu
- Tammikuu: 50 näköä-sanaa + 30 sisältösanaa
- Toukokuu: 100+ sanaa, osaa käyttää asiayhteydessä
Näkyvä kehitys: Sanasto laajeni

Osio 4: Sujuvuustallenteet
- Syyskuu: 40 sanaa/min, katkonaista
- Tammikuu: 65 sanaa/min, sujuvampaa
- Toukokuu: 90 sanaa/min, ilmeikästä
Näkyvä kehitys: Sujuvuus parani

Lukuvuoden lopun reflektio (oppilas täyttää pohjan):
"Kun vertaan syyskuun töitä toukokuun töihin, kehityin näin..."
"Olen ylpein tästä..."
"Ensi vuonna haluan parantaa..."
```

**Arviointiarvo**: Portfolio näyttää KEHITYKSEN (ei vain nykyistä tasoa)

---

### Matematiikan portfolio

**Kehityksen dokumentointi**:
```
Näyte 1 (syyskuu): Yksinkertainen yhteenlasku (20 tehtävää, 80% oikein, 15 minuuttia)
Näyte 2 (marraskuu): Kaksinumeroinen yhteenlasku (20 tehtävää, 90% oikein, 12 minuuttia)
Näyte 3 (tammikuu): Kertolaskun aloitus (10 tehtävää, 60% oikein, 20 minuuttia)
Näyte 4 (maaliskuu): Kertolaskuharjoittelu (20 tehtävää, 85% oikein, 10 minuuttia)
Näyte 5 (toukokuu): Monivaiheiset sanallinen tehtävät (10 tehtävää, 90% oikein, 15 minuuttia)

Portfolion kansisivu (oppilaan täyttämä):
Matemaattiset taidot, jotka hallitsen tänä vuonna:
☑ Yksinnumeroinen yhteenlasku
☑ Kaksinumeroinen yhteenlasku lainauksella
☑ Kertolaskutaulut 0-10
☑ Sanallisten tehtävien ratkaisu
☐ Jakolasku (harjoittelen ensi vuonna!)

Todiste: "Katso näyte 1 ja näyte 5. Syyskuussa osaan vain yksinkertaista yhteenlaskua. Nyt osaan ratkaista monimutkaisia sanallisia tehtäviä useilla vaiheilla!"

Vanhempien arvio: Vanhemmat näkevät konkreettista todistetta vuoden kehityksestä (ei vain todistuksen kirjainarvosana)
```

## Projektipohjainen arviointi

**Monivuotiset projektit osoittavat syvällistä ymmärrystä**

### Yhteiskuntaopin projekti

**Aihe**: Paikallinen yhteisö (2. luokka)

**Perinteinen arviointi**:
```
Koe: 20 kysymystä yhteisön työntekijöistä, karttaidoista, kansalaisuudesta
Muoto: Täytä tyhjät, yhdistä parit
Aika: 30 minuuttia
```

**Projektivaihtoehto**:
```
Tehtävä: "Luo yhteisöopas"

Viikko 1: Tutkimus
- Haastattele 2 yhteisön työntekijää (palomies, kirjastonhoitaja, jne.)
- Tehtäväpohja: Haastattelukysymysten malli
- Dokumentoi: Mitä he tekevät? Miksi heidän työnsä on tärkeää?

Viikko 2: Kartan luominen
- Piirrä kartta paikallisesta yhteisöstä (tärkeimmät maamerkit)
- Merkitse: koulu, kirjasto, paloasema, poliisiasema, puisto
- Tehtäväpohja: Kartan suunnitteluruudukko (mittakaava annettu)

Viikko 3: Kirjoittaminen
- Kirjoita "Tietoja yhteisöstämme" -vihko
- Osiot: Historia, Tärkeät ihmiset, Vierailukohteet, Kuinka olla hyvä kansalainen
- Tehtäväpohja: Kirjoitusjäsennin jokaiselle osiolle

Viikko 4: Esitys
- Jaa yhteisöopas luokalle
- Selitä: Mikä tekee yhteisöstämme erityisen?

Arviointikriteerit:
- Tutkimus täydellinen ja tarkka (4 pistettä)
- Kartta selkeä ja merkitty (4 pistettä)
- Kirjoittaminen informatiivista ja jäsenneltyä (5 pistettä)
- Esitys selkeä ja kiinnostava (3 pistettä)
Yhteensä: 16 pistettä

Mitä se arvioi:
- Yhteiskuntaopin sisältö (yhteisötietous)
- Tutkimustaidot (haastattelut, lähteet)
- Maantieto (kartan luominen)
- Kirjoittaminen (informatiivinen teksti)
- Viestintä (esitys)

Aikainvestointi: 4 viikkoa
Syvyys: Paljon syvempi kuin 30 minuutin koe
```

## Autenttiset osoitukset

**Käytännön taitojen soveltaminen**

### Lukeminen: Kirjasuositus

**Arviointitehtävä**:
```
Sen sijaan että: Perinteinen kirjaraportti (tiivistä juoni)

Kokeile: Kirjasuositusesitys

Tehtävä: "Vakuuta luokkatoverisi lukemaan kirjasi"

Vaatimukset:
1. Kiinnostava tiivistelmä (älä paljasta loppua!)
2. Miksi pidit siitä (henkilökohtainen yhteys)
3. Kenelle se sopii (yleisön tunnistaminen)
4. Lue lempikohta ääneen (sujuvuuden osoitus)
5. Luo visuaalinen (juliste, piirros, dioraama)

Tehtäväpohja: Suunnitteluopas
- Kirjan nimi/kirjailija: ______________
- Genre: ______________
- Päähenkilöt: ______________
- Syy 1 lukea se: ______________
- Syy 2 lukea se: ______________
- Syy 3 lukea se: ______________
- Ääneen luettava kohta (sivu ___): Merkitse muistilapulla

Arviointi:
- Ymmärrys osoitettu? (ymmärsi tarinan)
- Viestintä selkeää? (esitystaidot)
- Sujuvuus sopiva? (ääneen lukeminen)
- Vakuuttaminen tehokasta? (vakuuttavat perustelut)

Autenttinen: Näin ihmiset todella jakavat kirjoja (ei keinotekoinen kirjaraportti)
```

## Arviointikriteerit vaihtoehtoiselle arvioinnille

**Selkeät kriteerit varmistavat oikeudenmukaisen arvioinnin**

### Projektiarviointikriteerit

**Tiedemessuprojekti** (luotu tehtäväpohjana):
```
Arviointikriteerit: Tiedemessuprojekti

Tutkimuskysymys (4 pistettä)
4 = Selkeä, testattavissa oleva kysymys
3 = Kysymys olemassa mutta voisi olla selkeämpi
2 = Kysymys liian laaja tai ei testattavissa
1 = Ei selkeää kysymystä

Menetelmä (4 pistettä)
4 = Yksityiskohtaiset, toistettavat vaiheet
3 = Vaiheet olemassa, joitain yksityiskohtia puuttuu
2 = Menetelmä epäselvä
1 = Ei dokumentoitua menetelmää

Tiedon kerääminen (4 pistettä)
4 = Tarkka tieto, järjestetty taulukko/kaavio
3 = Tieto olemassa, järjestely voisi parantua
2 = Tieto puutteellista
1 = Ei kerätty tietoa

Johtopäätös (4 pistettä)
4 = Selkeä johtopäätös tuettu tiedolla
3 = Johtopäätös olemassa, heikko yhteys tietoon
2 = Johtopäätös ei vastaa tietoa
1 = Ei johtopäätöstä

Esitys (4 pistettä)
4 = Selkeä, kiinnostava, hyvin järjestetty
3 = Ymmärrettävä, riittävä järjestely
2 = Vaikea seurata
1 = Sekava tai puutteellinen

Yhteensä: _____ / 20 pistettä

Oppilaan hyöty: Tietää tarkalleen mitä odotetaan ennen projektin aloittamista
```
