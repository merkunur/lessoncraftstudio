## Inleiding: Voorwaartse vs Achterwaartse Lesvoorbereiding

**Traditionele lesvoorbereiding** (voorwaarts ontwerpen):
```
Stap 1: Leuke activiteiten kiezen (werkbladen, knutselactiviteiten)
Stap 2: Lessen uitvoeren
Stap 3: Aan het einde een toets maken
Probleem: Activiteiten sluiten niet aan bij de toets
Resultaat: Leerlingen doen activiteiten maar beheersen de leerdoelen niet
```

**Achterwaartse planning** (Wiggins & McTighe, 2005):
```
Stap 1: Bepaal de gewenste resultaten (wat moeten leerlingen WETEN?)
Stap 2: Bepaal het bewijs van leren (hoe TONEN ze hun begrip?)
Stap 3: Plan leeractiviteiten (welke activiteiten BRENGEN hen daar?)
Resultaat: Elke activiteit draagt doelgericht bij aan beheersing
```

**Onderzoek** (Wiggins & McTighe, 2005 - Understanding by Design): Achterwaartse planning verhoogt leerresultaten met **15-25%** door afstemming tussen leerdoelen, instructie en toetsing

**Kernidee**: Begin met het EINDRESULTAAT (toets) en werk achteruit naar de instructie

## De 3 Fasen van Achterwaartse Planning

### Fase 1: Bepaal de Gewenste Resultaten

**Vraag**: Wat moeten leerlingen kennen en kunnen?

**Voorbeeldthema**: Breuken voor groep 6

**Kerndoelen** (Nederlandse context):
```
Kerndoel 26: Leerlingen leren vlot rekenen met breuken
Kerndoel 27: Leerlingen kunnen breuken vergelijken en ordenen
Kerndoel 28: Leerlingen begrijpen gelijkwaardige breuken
```

**Duurzaam begrip** (concepten die beklijven):
```
1. Breuken vertegenwoordigen delen van een geheel
2. Verschillende breuken kunnen dezelfde hoeveelheid voorstellen (gelijkwaardigheid)
3. Breuken kunnen vergeleken worden door gelijke noemers te vinden
```

**Essentiële vragen**:
```
1. Hoe kunnen twee verschillende breuken gelijk zijn?
2. Wanneer gebruiken we breuken in het dagelijks leven?
3. Hoe weten we welke breuk groter is?
```

---

### Fase 2: Bepaal het Bewijs van Leren (Toetsing)

**Vraag**: Hoe tonen leerlingen aan dat ze de leerdoelen beheersen?

**Eindtoets** (ontworpen VOOR de instructie):

**Deel 1: Vaardighedendemonstratie** (20 punten)
```
1. Teken een model dat laat zien dat 2/4 = 1/2 (visueel bewijs)
2. Vergelijk 3/5 en 2/3 (toon je werk met gelijke noemers)
3. Tel op: 2/8 + 3/8 (los op met model)
4. Noem 3 situaties waarin je breuken gebruikt (toepassing)
```

**Deel 2: Probleemoplossing** (10 punten)
```
Vraagstuk: "Je hebt 2/6 van een pizza opgegeten. Je vriend at 3/9 van dezelfde pizza.
Wie at er meer? Bewijs je antwoord."

Rubric:
- 4 punten: Correct antwoord met visueel bewijs
- 3 punten: Correct antwoord, geen bewijs
- 2 punten: Fout antwoord, maar redelijke poging
- 0-1 punten: Geen werk getoond
```

**Beheersingsnorm**: 80% nauwkeurigheid = beheersing (24/30 punten)

**Nu**: Ontwerp instructie zodat leerlingen deze taken KUNNEN uitvoeren

---

### Fase 3: Plan Leeractiviteiten (Instructie)

**Vraag**: Welke activiteiten bereiden leerlingen voor op de toets?

**Week 1: Introductie breuken**
```
Maandag: Pizza-breukenmodellen (concreet)
Dinsdag: Werkblad - breuken inkleuren (picturaal)
Woensdag: Woordzoeker breuken (woordenschat: teller, noemer, gelijkwaardig)
Donderdag: Breuken inkleuren (1/2 rood, 1/4 blauw)
Vrijdag: Formatieve toets (5 opgaven - begrip checken)

Doel: Leerlingen kunnen breuken identificeren en tekenen
```

**Week 2: Gelijkwaardige breuken**
```
Maandag: Papier vouwen (1/2 = 2/4 = 4/8 fysiek ervaren)
Dinsdag: Werkblad - gelijkwaardige breuken oefenen (15 opgaven)
Woensdag: Kruiswoordpuzzel breuken (woordenschat: gelijkwaardig, vereenvoudigen)
Donderdag: Wiskundepuzzel (vind gelijkwaardige breuken in systeem)
Vrijdag: Formatieve toets (5 opgaven - alleen gelijkwaardigheid)

Doel: Leerlingen begrijpen waarom 2/4 = 1/2
```

**Week 3: Breuken vergelijken**
```
Maandag: Breukenstroken (visuele vergelijking)
Dinsdag: Werkblad - vergelijk breuken met gelijke noemers (20 opgaven)
Woensdag: Werkblad - vergelijk breuken (vind gelijke noemers)
Donderdag: Tekstsommen (welke is groter?)
Vrijdag: Formatieve toets (5 opgaven - vergelijken)

Doel: Leerlingen kunnen twee willekeurige breuken vergelijken
```

**Week 4: Optellen/aftrekken**
```
Maandag: Breuken optellen met modellen
Dinsdag: Werkblad - optellen met gelijke noemers (20 opgaven)
Woensdag: Werkblad - aftrekken met gelijke noemers (20 opgaven)
Donderdag: Gemengde bewerkingen oefenen
Vrijdag: EINDTOETS (de toets ontworpen in Fase 2)

Doel: Leerlingen kunnen breuken optellen/aftrekken, klaar voor toets
```

**Alignmentcheck**: Elke activiteit bouwt aan vaardigheden voor de eindtoets

## Curriculum Mapping: Het Jaaroverzicht

**Curriculum map**: Overzicht van de lesstof voor het hele schooljaar

**Doel**: Zorgen voor volledige dekking (geen gaten in de leerlijn)

### Maandelijkse Curriculum Map (Groep 6 Rekenen Voorbeeld)

**September: Getalbegrip & plaatswaarde**
```
Kerndoelen: Optellen/aftrekken meerciijferige getallen, afronden
Belangrijkste werkbladen:
- Week 1: 3-cijferig optellen (20 opgaven dagelijks)
- Week 2: 3-cijferig aftrekken (20 opgaven dagelijks)
- Week 3: Afronden op tientallen, honderdtallen (woordzoeker met getallen)
- Week 4: Tekstsommen (gemengde bewerkingen)
Toetsing: 25-opgaven toets (80% beheersingsnorm)
```

**Oktober: Tafels & vermenigvuldigingsstrategieën**
```
Kerndoelen: Vermenigvuldigen tot 100, rechthoeksmodellen
Belangrijkste werkbladen:
- Week 1: Tafels 0-5 (Afbeelding Bingo voor oefening)
- Week 2: Tafels 6-10 (dagelijkse oefenwerkbladen)
- Week 3: Rechthoeksmodellen (rasterrekenen)
- Week 4: Tekstsommen (vermenigvuldigingstoepassingen)
Toetsing: Tijdtoets tafels (100 opgaven in 5 minuten, 80% norm)
```

**November: Deling**
```
Kerndoelen: Delen tot 100, rest bij deling
Belangrijkste werkbladen:
- Week 1: Deelsommen (omgekeerde van vermenigvuldigen)
- Week 2: Staartdeling (2-cijferig ÷ 1-cijferig)
- Week 3: Deling met rest
- Week 4: Tekstsommen deling
Toetsing: 30-opgaven delingstoets
```

**December: Breuken (introductie)**
```
Kerndoelen: Breuken begrijpen als delen van een geheel
Belangrijkste werkbladen:
- Week 1: Breuken herkennen (modellen inkleuren)
- Week 2: Gelijkwaardige breuken (visuele modellen)
- Week 3: Breuken vergelijken (zelfde noemer)
Toetsing: Visuele breukentoets (tekenen en vergelijken)
```

**Januari: Breuken (bewerkingen)**
```
Kerndoelen: Optellen/aftrekken breuken met gelijke noemers
Belangrijkste werkbladen:
- Week 1-2: Breuken optellen (modellen + symbolisch)
- Week 3-4: Breuken aftrekken
Toetsing: 25-opgaven breuken bewerkingen toets
```

**Februari: Decimalen**
```
Kerndoelen: Decimale notatie, plaatswaarde tot honderdsten
Belangrijkste werkbladen:
- Week 1: Decimalen lezen/schrijven
- Week 2: Decimalen vergelijken
- Week 3: Decimalen optellen
- Week 4: Decimalen aftrekken
Toetsing: Decimalen bewerkingen toets
```

**Maart: Meten & data**
```
Kerndoelen: Eenheden omrekenen, grafieken, data-analyse
Belangrijkste werkbladen:
- Week 1: Eenheden omrekenen (cm/m, ml/l)
- Week 2: Lijngrafieken (maken van data)
- Week 3: Data-analyse (woordzoeker met meetkundige termen)
Toetsing: Meten toets + data-interpretatie
```

**April: Meetkunde**
```
Kerndoelen: Hoeken, vormen, lijnen, symmetrie
Belangrijkste werkbladen:
- Week 1: Hoeken meten (gradenboog oefenen)
- Week 2: Soorten driehoeken/vierhoeken (classificeren)
- Week 3: Lijnen (evenwijdig, loodrecht, snijdend)
- Week 4: Symmetrie (symmetrielijnen tekenen)
Toetsing: Meetkunde toets + constructieopdrachten
```

**Mei: Herhaling & verdieping**
```
Focus: Spiraalgewijs herhalen van alle lesstof
Belangrijkste werkbladen:
- Week 1: Bewerkingen herhaling (gemengde oefening)
- Week 2: Breuken/decimalen herhaling
- Week 3: Meten/meetkunde herhaling
- Week 4: CITO-voorbereiding (indien van toepassing)
Toetsing: Uitgebreide eindtoets
```

## Protocol voor Afstemming op Leerdoelen

**Probleem**: Activiteiten sluiten niet aan bij kerndoelen

**Oplossing**: Expliciete alignmentcheck

### Alignment Checklist

**Voor elk werkblad**:
```
☐ Kerndoel(en) behandeld: [schrijf kerndoelcode]
☐ Leerdoel: Leerlingen kunnen [specifiek, meetbaar doel]
☐ Succescriteria: [hoe weet je of het beheerst is?]
☐ Verbinding met toets: [welke toetsvraag bereidt dit voor?]
```

**Voorbeeld**:
```
Werkblad: Breuken vergelijken (20 opgaven)

☐ Kerndoel: Kerndoel 27 (Breuken vergelijken)
☐ Leerdoel: Leerlingen kunnen twee breuken met verschillende noemers vergelijken
   door gelijke noemers te gebruiken met 80% nauwkeurigheid
☐ Succescriteria: 16/20 opgaven correct
☐ Verbinding met toets: Eindtoets vraag 7-10 vereisen breuken vergelijken

Alignment geverifieerd ✓
```

## Toetsgerichte Instructie

**Principe**: Oefenmateriaal moet lijken op de toets

**Traditionele mismatch**:
```
Oefening: Open werkbladen (antwoord opschrijven)
Toetsing: Meerkeuzevragen (bolletjes inkleuren)
Resultaat: Formatshock op toetsdag (cognitieve belasting verspild)
```

**Afgestemde oefening**:
```
Oefening: Zelfde format als toets
Voorbeeld: Als toets meerkeuze is, oefen met meerkeuze
Resultaat: Formatvertrouwdheid (cognitieve focus op inhoud)
```

## Een Pacing Guide Ontwikkelen

**Pacing guide**: Tijdlijn voor het behandelen van leerdoelen

**Doel**: Zorgen dat je het curriculum afrondt voor het einde van het schooljaar

### Een Pacing Guide Maken

**Stap 1: Tel instructiedagen**
```
Schooljaar: 180 dagen
- Vakanties: 20 dagen
- Toetsdagen: 10 dagen
- Uitjes/evenementen: 10 dagen
- Bufferdagen (ziekte, sneeuw): 10 dagen
Werkelijke instructiedagen: 130 dagen
```

**Stap 2: Wijs dagen toe aan leerdoelen**
```
Groep 6 rekenen: 30 kerndoelen
130 dagen ÷ 30 kerndoelen = 4,3 dagen per kerndoel (gemiddeld)

Aanpassen op complexiteit:
- Eenvoudige kerndoelen (afronden): 2 dagen
- Complexe kerndoelen (breuken bewerkingen): 8 dagen
```

**Stap 3: Maak weekplanning**
```
Week 1 (15-19 aug): Plaatswaarde & afronden (Kerndoel 24-25)
Week 2 (22-26 aug): Optellen meercijferig (Kerndoel 26)
Week 3 (29 aug-2 sep): Aftrekken meercijferig (Kerndoel 26)
...

Resultaat: Elk kerndoel heeft toegewezen tijd
```

**Generator-voordeel**: Maak snel materialen voor elk kerndoel
```
Week 2 focus: Meercijferig optellen
Genereren: 5 optelverwerkbladen (3,5 minuten)
Printen: 30 exemplaren elk (printer regelt)
Klaar: Week 2 materialen compleet
```

## Differentiatie in Achterwaartse Planning

**Uitdaging**: Niet alle leerlingen bereiken beheersing in hetzelfde tempo

**Oplossing**: Gelaagde leerdoelen

### Gelaagde Beheersingsniveaus

**Niveau 1: Basis beheersing** (80% van leerlingen)
```
Leerdoel: Breuken vergelijken met gelijke noemers
Toetsing: 3/4 vs 2/4 (welke is groter?)
```

**Niveau 2: Vaardig** (60% van leerlingen)
```
Leerdoel: Breuken vergelijken met verschillende noemers
Toetsing: 3/4 vs 2/3 (gebruik gelijke noemers)
```

**Niveau 3: Gevorderd** (20% van leerlingen)
```
Leerdoel: Drie of meer breuken vergelijken, redenering uitleggen
Toetsing: Orden 2/3, 3/4, 5/6 van klein naar groot, motiveer
```

**Instructie**: Bied materialen voor alle drie niveaus
```
Maandag: Alle leerlingen - Basisvergelijking (gelijke noemers)
Dinsdag: Alle leerlingen - Verschillende noemers introductie
Woensdag: Niveau 1 - Meer oefening gelijke noemers (Station 1)
          Niveau 2 - Oefenen verschillende noemers (Station 2)
          Niveau 3 - Gevorderde meervoudige vergelijking (Station 3)
```

**Generator-gebruik**: Maak 3 niveaus direct
```
Niveau 1 werkblad: Vergelijk breuken (alleen gelijke noemer)
Niveau 2 werkblad: Vergelijk breuken (vind kleinste gemene noemer)
Niveau 3 werkblad: Orden 4+ breuken (complex)

Tijd: 126 seconden (42 sec elk) voor alle 3 niveaus
```

## Reflectie & Revisieprotocol

**Na afronding thema**: Reflecteer en verbeter

### Post-Thema Analyse

**Vragen om te stellen**:
```
1. Hebben leerlingen de leerdoelen beheerst? (check toetsdata)
   - Zo ja: Behoud thema zoals het is
   - Zo nee: Wat moet opnieuw behandeld worden?

2. Welke activiteiten waren het meest effectief?
   - Hoogste betrokkenheid: [lijst activiteiten]
   - Beste leerresultaten: [lijst activiteiten]
   - Behoud deze, elimineer ineffectieve

3. Werkte de pacing?
   - Te gehaast: Voeg 1-2 dagen toe volgend jaar
   - Te langzaam: Verlaag 1-2 dagen volgend jaar

4. Wat ga ik volgend jaar veranderen?
   - [specifieke aanpassingen]
```

**Voorbeeldreflectie**:
```
Thema: Breuken (4 weken)
Toetsresultaten: 85% beheersing (boven 80% norm) ✓

Meest effectieve activiteit: Papier vouwen (Week 2, Maandag)
- 100% betrokkenheid
- Leerlingen hadden "aha!" moment
- BEHOUDEN

Minst effectief: Breuken woordzoeker (Week 1, Woensdag)
- Leerlingen verveeld (te eenvoudig)
- VERVANGEN door breuken sudoku (meer uitdagend)

Pacing: Precies goed (afgerond vrijdag zoals gepland)

Wijzigingen volgend jaar:
- Vervang woordzoeker door sudoku
- Voeg één extra formatieve check toe in Week 3 (sommige leerlingen nog verward op donderdag)
```
