## Inleiding: Het Schooljaar Zinvol Afsluiten

**Uitdaging laatste schoolweken**: Leerlingen zijn afwezig met hun hoofd, gedragsproblemen, verspilde leertijd

**Onderzoek** (McCombs & Pope, 1994): Gestructureerde reflectie verbetert retentie met **35%** (terugkijken versterkt het leren)

**Traditioneel einde schooljaar**: Films kijken, feestjes, geen academische activiteiten
```
Realiteit:
- 2 weken: Niet-academische activiteiten
- Leerverlies: Leerlingen vergeten lesstof (zomerdip begint vroeg)
- Gedragsproblemen: Geen structuur = chaos
```

**Zinvol alternatief**: Reflectie + viering + herhaling
```
Structuur:
- Week 1: Herhaling van alle lesstof van het jaar (systematisch)
- Week 2: Reflectieactiviteiten + portfolio samenstellen
- Laatste dagen: Viering met academische verbinding

Resultaat: Leren gaat door, leerlingen vertrekken trots op hun groei
```

## Groeiportfolio Samenstellen

**Tastbaar bewijs van de vooruitgang van het jaar**

### Portfolio Inhoud

**September-juni vergelijking**:
```
Voorste gedeelte: "Waar Ik Begon"
- September schrijfvoorbeeld (eerste week)
- September rekentoets (nulmeting)
- September kleurwerk (fijne motoriek nulmeting)

Achterste gedeelte: "Waar Ik Nu Ben"
- Juni schrijfvoorbeeld (laatste week)
- Juni rekentoets (eindejaarsniveau)
- Juni kunstproject (fijne motoriek vooruitgang)

Middengedeelte: "Mijn Reis"
- Foto's van het jaar
- Favoriete werkbladen (leerling kiest zelf)
- Certificaten van prestaties
- Groeigrafiek (leesvloeiendheid grafiek, etc.)
```

**Samenstellactiviteit** (Week 1 van de laatste twee weken):
```
Maandag-dinsdag: Leerlingen verzamelen werkvoorbeelden uit mappen
Woensdag: Portfolio ordenen (chronologische volgorde)
Donderdag: Kaft versieren, reflectie schrijven
Vrijdag: Delen met maatje of kleine groep
```

## Herhalingsactiviteiten (Laatste 2 Weken)

**Systematische herhaling van de jaarinhoud**

### Maandag-Vrijdag Herhalingsschema

**Week 1: Inhoud Herhalen**
```
Maandag: Rekenspiraal alle jaarvaardigheden (30-sommen werkblad)
Dinsdag: Woordzoeker woordenschat (100 woorden van het hele jaar)
Woensdag: Kruiswoordpuzzel wereldoriëntatie (kernbegrippen van alle thema's)
Donderdag: Geschiedenistijdlijn (belangrijke gebeurtenissen/personen)
Vrijdag: Begrijpend lezen (teksten van favoriete verhalen)

Format: Werkbladen + spelletjes + gesprekken
Doel: Geheugen opfrissen voor de zomervakantie
```

**Week 2: Reflectie + Viering**
```
Maandag: Portfolio's maken (werkvoorbeelden ordenen)
Dinsdag: Reflectie-essays schrijven (terugblik op het jaar)
Woensdag: Videoboodschappen opnemen (advies voor volgjarige klas)
Donderdag: Vriendenboekjes (klasgenoten schrijven in memory books)
Vrijdag: Viering (academische spelletjes, prijsuitreiking, feest)
```

## Certificaten en Erkenningsceremonie

**Vier elke leerling**

### Gepersonaliseerde Certificaten

**Eén certificaat per leerling** (gepersonaliseerde sterke punten):
```
Voorbeelden:
"Meest Verbeterde Rekenen"
"Doorzetter van het Jaar" (gaf nooit op)
"Vriendelijkheidsambassadeur" (altijd behulpzaam)
"Creatieve Denker" (unieke probleemoplossing)
"Leesster/-held" (50+ boeken gelezen)

Principe: ELKE leerling wordt ergens voor erkend
Doel: Alle leerlingen vertrekken met waardering
```

**Generator gebruik**: Maak 30 gepersonaliseerde certificaten (21 minuten = 42 sec elk)
