## Introduktion: Udfordringen ved Samundervisning

**Samundervisningsmodellen**: Almenundervisningslærer + specialundervisningslærer = fælles klasserum

**Udbredelse**: 60% af danske skoler anvender samundervisning (inklusionsmodel)

**Koordineringsudfordring**:
```
Almenundervisningslærer: Planlægger fagindhold, laver materialer
Specialundervisningslærer: Tilbyder tilpasninger, modificerer indhold
Problem: To lærere, separat planlægning → dobbeltarbejde ELLER manglende sammenhæng
```

**Traditionelt arbejdsarkproblem**:
```
Almenundervisningslærer: Laver klassetrin-arbejdsark (30 opgaver, udfordrende)
Specialundervisningslærer: Skal tilpasse (reducér til 15 opgaver, tilføj billeder)
Dobbeltarbejde: 2× arbejde for i princippet samme indhold
Resultat: Ineffektivitet
```

**Generatorløsning**: Én platform, øjeblikkelig differentiering

**Forskning** (Friend & Cook, 2017): Effektiv samundervisning kræver **fælles planlægning** + **fælles ressourcer** (reducerer forberedelsestid med 40%)

## Samundervisningsmodeller & Arbejdsarkanvendelse

### Model 1: Den Ene Underviser, Den Anden Assisterer

**Struktur**: Almenundervisningslærer underviser, specialundervisningslærer cirkulerer og yder støtte

**Arbejdsarkanvendelse**:
```
Almenundervisningslærer: Tildeler matematikark (20 opgaver, klassetrin-niveau)
Specialundervisningslærer: Cirkulerer, identificerer elever med udfordringer
Strategi: Specialundervisningslærer giver tilpasset version (10 opgaver, billedstøtte) til elever der har brug for tilpasning
```

**Generator arbejdsgang**:
```
Søndag planlægning (10 minutter):
Almenundervisningslærer: Genererer standard matematikark (20 opgaver, symbolsk)
Specialundervisningslærer: Genererer modificeret version (10 opgaver, billedtilstand)
Begge: Deler via Google Drev

Mandag lektion:
Almenundervisningslærer: Uddeler standardversion til de fleste elever
Specialundervisningslærer: Uddeler modificeret version til IUP-elever (5-8 elever)
Resultat: Differentiering uden separate planlægningsmøder
```

**Tid sparet**: 30 minutters ugentligt planlægningsmøde (kan koordineres via delt mappe i stedet)

---

### Model 2: Stationsundervisning

**Struktur**: Klassen deles i stationer, hver lærer leder én station

**Arbejdsarkanvendelse**:
```
Station 1 (Almenundervisningslærer): Klassetrin-krydsord
Station 2 (Specialundervisningslærer): Modificeret krydsord (færre ord, billedspor)
Station 3 (Selvstændigt): Opgaver til tidlig færdige
Station 4 (Selvstændigt): Konkrete materialer
```

**Generator arbejdsgang**:
```
Almenundervisningslærer genererer:
- Station 1 krydsord (15 ord, tekstspor)
- Station 3 udfordringer (kryptogrammer, avanceret sudoku)

Specialundervisningslærer genererer:
- Station 2 krydsord (8 ord, billedspor)
- Facitlister til alle stationer

Tid: 12 minutter i alt (begge lærere genererer)
Stationer klar til ugen
```

---

### Model 3: Parallel Undervisning

**Struktur**: Klassen deles i to, begge lærere underviser samme indhold

**Arbejdsarkanvendelse**:
```
Gruppe A (Almenundervisningslærer): Standard sværhedsgrad
Gruppe B (Specialundervisningslærer): Modificeret sværhedsgrad (mindre gruppe = mere støtte)
```

**Generator fordel**: Samme generator, forskellige indstillinger = øjeblikkelige parallelle versioner

**Eksempel**:
```
Matematiklektion: Additionsøvelse

Gruppe A arbejdsark:
- Interval: 1-100
- Opgaver: 25
- Format: Kun symbolsk
- Genereret: 42 sekunder

Gruppe B arbejdsark:
- Interval: 1-20
- Opgaver: 15
- Format: Billedtilstand
- Genereret: 42 sekunder

Total forberedelse: 84 sekunder til to differentierede versioner
Begge grupper øver samme færdighed (addition) på passende niveau
```

---

### Model 4: Alternativ Undervisning

**Struktur**: Almenundervisningslærer underviser stor gruppe, specialundervisningslærer trækker lille gruppe ud til forudundervisning eller genundervisning

**Arbejdsarkanvendelse**:
```
Stor gruppe (Almenundervisningslærer): Introduktion til nyt indhold + klassetrin-øvelse
Lille gruppe (Specialundervisningslærer): Forudundervisning med tilpasset arbejdsark ELLER genundervisning med forenklet arbejdsark
```

**Forudundervisningseksempel**:
```
Fredag (specialundervisning udskilning):
- Specialundervisningslærer: Forudunderviser næste uges ordforråd (forenklet ordsøgning)
- Elever: Eksponeres for begreber før mandag lektion
- Resultat: Mere succesfulde under mandag fællesundervisning

Mandag (almenundervisning hel klasse):
- Almenundervisningslærer: Underviser ordforråd (standard krydsord)
- Tidligere forudundervist elever: Allerede bekendt med begreberne (øget selvtillid)
```

## Ressourcerum Anvendelser

**Ressourcerum**: Separat rum hvor specialundervisningselever modtager specialiseret undervisning

**Planlægning**: Elever trækkes ud af almenundervisningsklassen i 30-60 minutter

**Fagplanudfordring**: Skal afstemmes med almenundervisningsfagplan samtidig med at tilbyde specialiseret støtte

---

### Ressourcerum Fagplan Koordinering

**Problem**: Ressourcerum underviser andet indhold = elev går glip af almenundervisning

**Løsning**: Parallel fagplan (samme emner, modificeret sværhedsgrad)

**Eksempel**:
```
Almenundervisningsklasse (Uge 15. oktober):
- Matematik: Tabeller 1-12
- Læsning: 4. klasses romananalyse
- Natur/teknik: Materiens tilstandsformer ordforråd

Ressourcerum (Samme uge):
- Matematik: Tabeller 1-5 (forenklet interval)
- Læsning: Modificeret forståelse (samme roman, tilpassede spørgsmål)
- Natur/teknik: Materiens tilstandsformer ordsøgning (samme ordforråd, anden aktivitet)

Resultat: Elev lærer samme indhold som kammerater (bare modificeret støtte)
```

**Generator arbejdsgang**:
```
Specialundervisningslærer: Koordinerer med almenundervisningslærere

Mandag morgen:
- Almenundervisningslærer lægger ugens arbejdsark i delt Google Drev
- Specialundervisningslærer: Åbner arbejdsark, noterer færdigheder der undervises
- Specialundervisningslærer: Genererer modificerede versioner (reduceret sværhedsgrad)
- Total tid: 15 minutter (vs 2-3 timer at lave fra bunden)
```

---

### IUP Mål Progressionsmonitorering

**Ressourcerum formål**: Målrettet IUP-mål undervisning

**Generator anvendelse**: Konsistent progressionsmonitoreringsværktøj

**Eksempel IUP-mål**: "Elev skal løse encifrede additionsopgaver med 80% nøjagtighed"

**Ugentlig progressionsmonitorering**:
```
Uge 1: Generér additionsark (20 opgaver, 1-10)
        Elevresultat: 12/20 (60%)

Uge 2: Generér NYT additionsark (samme færdighed, andre opgaver)
        Elevresultat: 14/20 (70%)

Uge 3: Generér NYT arbejdsark
        Elevresultat: 16/20 (80%) ✓ MÅL OPNÅET

Dokumentation: 3 arbejdsark i IUP-mappe (bevis for fremskridt)
```

**Generator fordel**: Nye opgaver hver uge (ingen udenadslære, ægte færdighedsvurdering)

## Samarbejdende Planlægningsarbejdsgang

### Delt Google Drev Mappe

**Struktur**:
```
2024-2025 Samundervisningsmaterialer/
├── Matematik/
│   ├── Standard_Niveau/
│   └── Modificeret_Niveau/
├── Læsning/
│   ├── Standard_Niveau/
│   └── Modificeret_Niveau/
├── Evaluering/
│   ├── Standard_Prøver/
│   └── Tilpassede_Prøver/
└── Facitlister/
```

**Arbejdsgang**:
```
Søndag (Almenundervisningslærer):
- Genererer standard-niveau arbejdsark til ugen
- Uploader til "Standard_Niveau" mapper
- Total tid: 15 minutter

Søndag (Specialundervisningslærer):
- Gennemser almenundervisningsuploads
- Genererer modificerede versioner der matcher samme færdigheder
- Uploader til "Modificeret_Niveau" mapper
- Total tid: 15 minutter

Mandag-fredag:
- Begge lærere tilgår passende versioner til elever
- Ingen daglig koordinering nødvendig (materialer forudforberedt)
```

---

### Ugentligt Planlægningsmøde (Forkortet)

**Traditionel fællesplanlægning**: 90 minutter/uge (diskutér fagplan, lav materialer, koordinér)

**Med generatorer**: 30 minutter/uge (diskutér kun undervisning, materialer allerede genereret)

**Tid sparet**: 60 minutter/uge × 36 uger = **36 timer/år pr samundervisningspar**

**Mødeagenda** (30 minutter):
```
1. Gennemgå sidste uges elevfremskridt (10 min)
2. Diskutér kommende uges mål (10 min)
3. Tildel roller (hvem underviser hvad) (5 min)
4. Bekræft differentieringsstrategi (hvilke elever får hvilke materialer) (5 min)

Bemærk: Materialer allerede genereret (diskuteres ikke i mødet)
Resultat: Mødet fokuserer på UNDERVISNING, ikke materialeforberedelse
```

## Inklusionsbedste Praksis

### Usynlig Differentiering

**Mål**: Tilpasse til IUP-elever uden stigmatisering

**Strategi**: Brug samme generator, subtile forskelle

**Eksempel**:
```
Hel klasse: "Alle laver et matematikark"

Virkelighed:
- 22 elever: 25-opgave arbejdsark (standard)
- 8 IUP-elever: 15-opgave arbejdsark (modificeret)

Visuel lighed:
- Samme titel ("Additionsøvelse")
- Samme format (opgaver i rækker)
- Samme sidelayout
- Eneste forskel: Opgaveantal (mindre tydeligt end "MODIFICERET" stemplet på toppen)

Resultat: IUP-elever føler sig inkluderet (ikke "anderledes")
```

---

### Universelt Design for Læring (UDL)

**UDL princip**: Design for alle elever fra start (ikke eftermontér tilpasninger)

**Generator anvendelse**:
```
I stedet for:
1. Lav standard arbejdsark
2. Modificér til IUP-elever (ekstra arbejde)

UDL tilgang:
1. Generér 3 versioner samtidigt (15 min):
   - Niveau 1: Under klassetrin (kæmpende elever)
   - Niveau 2: På klassetrin (flertallet)
   - Niveau 3: Over klassetrin (avancerede elever)

2. Elever vælger niveau ELLER lærer tildeler diskret

Resultat: Alle elever har passende materialer, ingen "modificeret" stigma
```

## Konklusion

Samundervisning kræver **fælles ressourcer + koordineret planlægning** - generatorer giver øjeblikkelig differentieringsplatform.

**Samundervisningsmodeller**:
1. Den Ene Underviser, Den Anden Assisterer (almenundervisningslærer underviser, specialundervisningslærer giver modificerede arbejdsark)
2. Stationsundervisning (begge lærere leder stationer med differentierede materialer)
3. Parallel Undervisning (samme færdighed, to sværhedsgrader på 84 sekunder)
4. Alternativ Undervisning (forudundervisning/genundervisning med tilpassede arbejdsark)

**Ressourcerum anvendelser**:
- Parallel fagplan (samme emner som almenundervisning, modificeret sværhedsgrad)
- IUP-mål progressionsmonitorering (nye vurderinger ugentligt)
- Målrettet færdighedsøvelse (afstemt med almenundervisningsindhold)

**Samarbejdende arbejdsgang**:
- Delt Google Drev (Standard + Modificeret mapper)
- Forkortede planlægningsmøder (30 min vs 90 min, 36 timer sparet årligt)
- Usynlig differentiering (samme generator, subtile forskelle)

**Universelt Design**: Generér 3 niveauer samtidigt (under/på/over klassetrin)

**Forskning**: Fælles ressourcer reducerer forberedelsestid med 40% (Friend & Cook, 2017)
