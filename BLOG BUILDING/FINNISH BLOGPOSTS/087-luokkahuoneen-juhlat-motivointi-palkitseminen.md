## Johdanto: Tunnustuksen voima oppimisessa

**Motivaation tyypit** tutkimuksen mukaan (Deci & Ryan, 1985):
- **Ulkoinen motivaatio**: Ulkoiset palkkiot (tarrat, palkinnot)
- **Sisäinen motivaatio**: Sisäinen tyytyväisyys (ylpeys, osaaminen)

**Tutkimus** (Hattie, 2009): Tunnustuksella ja palautteella on **0,75 vaikutuskoko** – yksi vahvimmista oppimistuloksiin vaikuttavista tekijöistä.

**Tarvitaan tasapainoa**: Käytä ulkoista motivaatiota aluksi, siirry asteittain sisäiseen motivaatioon.
```
Syyskuu: "Tee 10 tehtävää, ansaitse palkinto" (ulkoinen motivaatio)
Tammikuu: "Rakastan ahkeraa työskentelyä!" (sisäinen motivaatio kehittynyt)
Tulos: Palkinnot käynnistävät tavan, sisäinen motivaatio ylläpitää sitä
```

**Keskeinen periaate**: Juhli YRITYSTÄ ja KEHITYSTÄ (ei vain täydellisiä tuloksia)

## Tehtäväpohjien täyttämisen välitavoitteet

**Seuraa kumulatiivista edistymistä** (pitkän aikavälin motivaatio)

### 100 tehtävän kerho

**Toteutus**:
```
Luokkahuoneen juliste: "100 tehtävän kerho"
Tavoite: Täytä 100 tehtävää lukuvuoden loppuun mennessä

Oppilaan seurantalomake:
Nimi: ________________
☐☐☐☐☐ 5
☐☐☐☐☐ 10
...
☐☐☐☐☐ 100 (tähtitarra saavutettu!)

Yksilöllinen edistyminen: Jokainen oppilas värittää ruutuja sitä mukaa kun tehtäviä valmistuu
```

**Välitavoitteet**:
```
25 tehtävää: Pronssikirja
50 tehtävää: Hopeakirja
75 tehtävää: Kultakirja
100 tehtävää: Platinakirja + erityinen etuoikeus

Kirjat: Luodaan 42 sekunnissa kukin, tulostetaan korttipohjalle
```

**Juhliminen**: Kuukausittainen tunnustustilaisuus (julkistetaan uudet välitavoitteiden saavuttajat)

## Yrityksen tunnustaminen

**Keskity PROSESSIIN, ei vain lopputulokseen**

### "Eniten kehittynyt" -kirjat

**Viikoittainen tunnustus**:
```
Ei: "Kuka sai 100 %?" (kykypohjainen)
Sen sijaan: "Kuka kehittyi eniten?" (yrityspohjainen)

Esimerkki:
Oppilas A: Maanantai 60 % → Perjantai 85 % (25 % kehitys)
Oppilas B: Maanantai 90 % → Perjantai 95 % (5 % kehitys)

Eniten kehittynyt: Oppilas A (juhlitaan kasvua, ei täydellisyyttä)

Kirja:
"Viikon eniten kehittynyt matematiikan oppilas - Viikko [päivämäärä]
[Oppilas A] kehittyi 25 prosenttiyksikköä ahkeran työn ja määrätietoisuuden kautta!"
```

---

### Sinnikkyyskirjat

**Haasteisiin tarttumisen tunnustaminen**:
```
Oppilas kamppailee 9×9 sudokun kanssa (vie 60 minuuttia kolmessa päivässä)
Lopulta valmis: Perjantai-iltapäivänä

Palkinto: "Sinnikkyyssampioin"
Kirja: "Ei koskaan luovuttanut haasteesta - sai 9×9 sudokun valmiiksi määrätietoisuudellaan!"

Arvo: Yritystä kehutaan (ei synnynnäistä lahjakkuutta)
Kasvun ajattelutapa: "Sinnikkyys palkitsee"
```

## Luokkahuoneen talous

**Tehtävät ansaitsevat "valuuttaa"**

### Luokkahuoneen kolikot

**Järjestelmä**:
```
Jokainen valmis tehtävä = 1 luokkahuonekolikko
Täydet pisteet (100 %) = bonus 0,50 kolikkoa

Oppilaan seuranta:
Nimi: ____________ Saldo: €______

Viikkoansainta: 5 tehtävää × 1 € = 5 €
Ahkera viikko: 10 tehtävää = 10 €
```

**Käyttövaihtoehdot**:
```
5 €: Valitse istumapaikka viikoksi
10 €: Kotitehtäväpassi (ohita yksi tehtävä)
15 €: Lounas opettajan kanssa
25 €: Ole opettajan apulainen päivän ajan
50 €: Erikoispalkinto palkintolaatikosta

Säästäminen: Oppilaat oppivat viivästettyä tyydytystä (säästä 50 € isompaan palkintooon vs. käytä 5 € heti)
```

```

Hyöty: Oppilaat kannustavat toisiaan (tiimityö, ei kilpailu)
```

## Perjantaijuhlat

**Viikon lopun tunnustusrituaali**

### Perjantaikokous (15 minuuttia)

**Rakenne**:
```
15:00-15:05: Opettaja ilmoittaa viikon välitavoitteiden saavuttajat
"Tällä viikolla 7 oppilasta saavutti 50 tehtävän virstanpylvään! Juhlitaan: [nimet]"
Luokka: Aplodit

15:05-15:10: Oppilaan hetki (yksi oppilas jakaa oppimisesta)
Valittu oppilas: "Tällä viikolla opin murtoluvut. Se oli vaikeaa mutta jatkoin yrittämistä. Suosikki tehtäväni oli murtolukupizzan väritystehtävä!"
Luokka: Taputus ja hurraus

15:10-15:15: Seuraavan viikon esikatselu + motivoiva lähtö
Opettaja: "Ensi viikolla opimme kertolaskua. Tiedän että teette kovasti töitä. Hyvää viikonloppua!"

Hyöty: Positiivinen viikon lopetus (oppilaat lähtevät ylpeinä)
```

## Henkilökohtaiset keskustelut

**Yksilöllinen tunnustus**

### Kuukausitapaaminen (5 minuuttia per oppilas)

**Keskustelun rakenne**:
```
Opettaja: "Katsotaan edistymiskaaviotasi. Syyskuussa teit 15 tehtävää. Lokakuussa 22! Se on kasvua!"

Oppilas: Näkee oman datan (konkreettinen todiste kehityksestä)

Opettaja: "Mistä olet eniten ylpeä?"
Oppilas: "Olen ylpeä että sain vaikean sudokun valmiiksi"

Opettaja: "Mikä on tavoitteesi marraskuulle?"
Oppilas: "Täytän 25 tehtävää"

Hyöty: Oppilas asettaa oman tavoitteen (omistajuus), opettaja tunnustaa edistymisen (vahvistus)
```

**Oppilaat juhlivat toisiaan**

**Toteutus**:
```
Ilmoitustaulu: "Huudahdukset"
Tarralappu saatavilla

Oppilaat kirjoittavat:
"Huudahdus [Oppilaan nimi] auttamisesta matematiikassa!"

Luetaan ääneen: Joka perjantai opettaja lukee huudahdukset
Tulos: Positiivinen vertaiskulttuuri (oppilaat vahvistavat toisiaan)
```

## Saavutusten viestintä vanhemmille

**Jaa menestystä perheiden kanssa**

### Viikkoedistymissähköposti

**Malli**:
```
Aihe: [Oppilaan nimi] loistava viikko!

Hyvä [Vanhempi],

Halusin jakaa [Oppilaan] saavutukset tältä viikolta:

✓ Täytti 5 tehtävää (tavoitteessa!)
✓ Sai 90 % matematiikan testistä (kehitystä 75 %:sta viime viikolla)
✓ Auttoi luokkakaveria sanastopulmassa (ystävällinen ja avulias)

[Oppilas] työskentelee ahkerasti ja tekee hienoa edistystä. Juhlittakaa kotona!

Hyvää viikonloppua,
[Opettajan nimi]
```

**Tiheys**: Viikoittain positiivisia päivityksiä (kuukausittain huolia)

**Tulos**: Vanhemmat kuulevat onnistumisista (eivät vain ongelmista)

## Palkitsemisen sudenkuopien välttäminen

**Yleisiä virheitä vältettäväksi**

### Sudenkuoppa 1: Vain täydellisten tulosten palkitseminen

**Ongelma**: Pysähtynyt ajattelutapa (vain "älykkäät" lapset saavat palkintoja)
```
Huono: "Kaikki jotka saivat 100 % saavat palkinnon"
Tulos: Kamppailevia oppilaita ei koskaan palkita (motivaatio laskee)
```

**Ratkaisu**: Palkitse KASVUA
```
Hyvä: "Kaikki jotka kehittyivät viime viikosta saavat tunnustuksen"
Tulos: Kaikki oppilaat voivat ansaita tunnustuksen (yrityspohjainen)
```

---

### Sudenkuoppa 2: Jatkuvat aineelliset palkinnot

**Ongelma**: Riippuvuus ulkoisesta motivaatiosta
```
Huono: Palkinto jokaisesta valmistuneesta tehtävästä
Tulos: "Teen vain jos saan palkinnon" (ei sisäistä motivaatiota)
```

**Ratkaisu**: Vähennä aineellisia palkintoja ajan myötä
```
Syyskuu: Palkinto per 5 tehtävää (käynnistys)
Joulukuu: Palkinto per 20 tehtävää (harvemmin)
Maaliskuu: Vain kirjat (sisäinen motivaatio ensisijainen)
Tulos: Asteittainen siirtymä sisäiseen motivaatioon
```

---

### Sudenkuoppa 3: Julkinen häpeä

**Ei koskaan**: Julkinen näyttö heikoimmista suoriutujista
```
Huono: "Nämä oppilaat eivät saaneet tehtäviään valmiiksi" (juliste nimillä)
Tulos: Häpeä, vähentynyt motivaatio, vahingoittuneet suhteet
```

**Aina**: Yksityinen tuki
```
Hyvä: Yksityinen keskustelu kamppailevan oppilaan kanssa
"Huomasin että sinulla on vaikeuksia saada valmiiksi. Mitä tukea tarvitset?"
Tulos: Ongelmanratkaisu, arvokkuuden säilyttäminen
```

## Erikoistapahtumat

**Satunnaiset suuremmat juhlat**

### Oppikokonaisuuden päätösjuhlat

**Suuren oppikokonaisuuden jälkeen**:
```
Oppikokonaisuus: Murtoluvut (4 viikkoa työtä)
Loppuarviointi: 80 %+ luokan hallinta

Juhla: "Murtolukujuhlat"
- Tuo: Pizza (leikattu murtolukuihin: "Kuka haluaa 3/8 pizzaa?")
- Toiminta: Murtolukubingo (kertauspeli)
- Kirja: "Murtolukumestari" kaikille jotka saavuttivat hallinnan

Kesto: 30 minuuttia (viimeinen tunti perjantaina)
Akateeminen yhteys: Juhlat vahvistavat sisältöä (ei vain hauskaa taukoa)
```

---

### 100. koulupäivä

**Merkkipaalu: koulun 100. päivä**
```
Toiminta: "100 tehtävän haaste"
Tavoite: Luokka täyttää yhteensä 100 tehtävää yhtenä päivänä

Toteutus:
- Vaihtelu: Matematiikka, sanahaut, väritystehtävät, ristsanatehtävät, palapelit (valinnat saatavilla)
- Seuranta: Iso kaavio (oppilaat lisäävät tukkimiehen kirjainmerkkejä kun saavat valmiiksi)
- Juhla: Jos tavoite täyttyy kello 14 mennessä, viimeinen tunti on peliaikaa

Tulos: 100 tehtävää täytetty (tuottava juhla)
Akateeminen arvo: Kertausharjoittelu (ei menetetettyä oppimisaikaa)
```
