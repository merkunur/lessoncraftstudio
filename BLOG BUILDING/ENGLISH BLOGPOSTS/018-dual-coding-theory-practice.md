## How Dual Coding Works

### The Two Cognitive Systems

**Paivio's theory**: Human brain processes information through two independent channels

**Verbal System** (left hemisphere):
- Processes: Words, letters, sounds, numbers
- Mode: Sequential (one word at a time)
- Storage: Linguistic memory traces
- Example: The word "dog" stored as phonemes /d/ /…î/ /g/

**Visual System** (right hemisphere):
- Processes: Images, shapes, colors, spatial relationships
- Mode: Parallel (entire image simultaneously)
- Storage: Visual memory traces
- Example: Picture of dog stored as perceptual features (4 legs, tail, fur, ears)

---

### Why Two Codes Beat One

**Single-code learning** (word "elephant" only):
```
Input: Text "elephant"
Processing: Verbal system only
Storage: One memory trace (linguistic)
Retrieval pathways: One (verbal)
24-hour recall: 40%
```

**Dual-code learning** (word + image):
```
Input: Text "elephant" + picture of elephant
Processing: Verbal system + Visual system
Storage: Two memory traces (linguistic + visual)
Retrieval pathways: Two (either verbal OR visual can trigger recall)
Cross-referencing: Word activates image; image activates word
24-hour recall: 92%
```

**The multiplier effect**: Not additive (40% + 40% = 80%), but synergistic (40% √ó 2.3 = 92%)

## The Picture Superiority Effect

### Standing's 10,000 Image Experiment (1973)

**Procedure**:
- Participants shown 10,000 images (one every 5 seconds)
- 2 days later: Recognition test (old vs new images)

**Result**: **83% accuracy** (8,300/10,000 images correctly recognized)

**Comparison**:
- 10,000 words (no images): 12% recognition after 2 days
- **Picture superiority ratio: 7√ó better recognition**

**Interpretation**: Visual memory capacity vastly exceeds verbal memory

---

### Why Pictures Are Superior

**Distinctiveness**: Every dog picture looks unique (German Shepherd vs Poodle)
**Verbal limitation**: All dogs described with same word: "dog"

**Concreteness**: Picture directly resembles real object
**Verbal abstraction**: Word "dog" is arbitrary symbol (no resemblance to actual dog)

**Parallel processing**: Brain captures entire image in 100-200 milliseconds
**Sequential text**: Reading "The dog is brown with floppy ears" takes 3-4 seconds

## Mayer's Multimedia Learning Principles

### Building on Dual Coding (Mayer, 2009)

**Mayer's contribution**: Identified 12 principles for optimal dual-code design

---

### Principle 1: Contiguity

**Rule**: Place images NEXT to corresponding text (not separated)

**Bad design**:
```
[All images at top of page]
[All text at bottom]
```
**Problem**: Working memory forgets image before reading text

**Good design**:
```
üê∂ ‚Üí dog
üê± ‚Üí cat
üê≠ ‚Üí mouse
```
**Advantage**: Simultaneous visual + verbal processing

**Platform implementation**: Word Scramble, Crossword, Bingo (images adjacent to words)

---

### Principle 2: Coherence

**Rule**: Exclude irrelevant images (decorative elements harm learning)

**Bad design**: Vocabulary worksheet with decorative stars, flowers, borders
**Result**: 15% lower retention (Harp & Mayer, 1998)

**Good design**: Only content-relevant images

**Platform implementation**: 3,000+ curated educational images, minimal decoration

---

### Principle 3: Signaling

**Rule**: Highlight essential information

**Implementation**: Bold key terms, arrows pointing to critical image features

**Example**: Shadow Match
```
[Image of elephant]
Arrow pointing to trunk: "Long trunk"
Label: "ELEPHANT"
```

---

### Principle 4: Redundancy Principle

**Rule**: Don't show identical information in three forms (image + text + narration)

**Why**: Overloads verbal channel (text + narration both compete for linguistic processing)

**Platform approach**: Image + text (no audio narration in worksheets)

---

### Principle 5: Spatial Contiguity

**Rule**: Integrate text INTO images (not as separate caption)

**Platform implementation**: Pattern Train, Grid Drawing (labels embedded in graphics)

## When to Remove Visual Support (Scaffolding Fade)

### Vygotsky's Zone of Proximal Development

**ZPD principle**: Gradually reduce support as mastery develops

**Week 1-3**: Maximum dual coding (every word has image)
**Week 4-6**: Partial support (50% words have images)
**Week 7-9**: Minimal support (10% words have images)
**Week 10+**: Abstract only (no images)

**Assessment criterion**: 90% accuracy for 2 consecutive weeks ‚Üí Reduce support
