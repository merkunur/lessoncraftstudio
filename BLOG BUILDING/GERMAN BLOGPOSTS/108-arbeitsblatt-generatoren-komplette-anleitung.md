## Schritt 4: Ihr Arbeitsblatt anpassen

**Parameter festlegen**

### Wortsuche-Beispiel im Detail

**Anpassungsmöglichkeiten**:

**1. Vokabeln eingeben**:
```
Eingabefeld: Wörter tippen oder einfügen

Beispiel (Sachkunde-Einheit):
- Photosynthese
- Chlorophyll
- Kohlendioxid
- Sauerstoff
- Sonnenlicht
- Wasser
- Energie
- Glukose

Tipp: Ein Wort pro Zeile oder durch Kommas getrennt
```

---

**2. Rastergröße wählen**:
```
Optionen:
- 10×10: Einfach (jüngere Schüler, einfache Wörter)
- 15×15: Mittel (Standard-Schwierigkeit)
- 20×20: Schwer (ältere Schüler, mehr Wörter)

Empfehlung: Mit 15×15 beginnen (anpassen falls zu leicht/schwer)
```

---

**3. Schwierigkeitsgrad einstellen**:
```
Leicht: Wörter nur horizontal und vertikal (keine Diagonalen)
Mittel: Horizontal, vertikal, diagonal
Schwer: Alle Richtungen einschließlich rückwärts

Auswahl basierend auf: Klassenstufe und Erfahrung mit Wortsuchrätseln
```

---

**4. Zusätzliche Optionen**:
```
☑ Lösungsblatt einschließen (empfohlen)
☑ Wortliste einschließen (Liste der zu findenden Wörter)
☐ Wortliste ausblenden (herausfordernder)

Titel: "Photosynthese-Vokabeln"
Anweisungen: "Finde alle 8 Wörter zum Thema Photosynthese"
```

## Schritt 5: Vorschau und Anpassung

**Vor dem Finalisieren prüfen**

### Echtzeit-Vorschau

**Was zu überprüfen ist**:
```
Layout-Check:
- Passen alle Wörter ins Raster? ✓
- Ist der Titel klar? ✓
- Sind die Anweisungen lesbar? ✓
- Wirkt das Gesamtbild professionell? ✓

Anpassungen nötig?
- Raster zu voll: Größe erhöhen (10×10 → 15×15)
- Zu einfach: Schwierigkeit erhöhen (mittel → schwer)
- Wort fehlt: Zur Liste hinzufügen, neu generieren

Iterativer Prozess: Anpassen → Vorschau → Anpassen → Vorschau bis perfekt

Zeit: 30 Sekunden bis 2 Minuten (je nach benötigten Änderungen)
```

## Schritt 8: Effizient drucken

**Druckeinstellungen sind wichtig**

### Optimale Druckeinstellungen

**Druckerpräferenzen**:
```
Qualität: Standard (nicht Entwurf, nicht Foto)
- Entwurf: Zu hell (schwer lesbar)
- Standard: Klar, verwendet angemessen Tinte
- Foto: Unnötig (verschwendet Tinte, langsam)

Farbe: Schwarz-weiß (außer speziell benötigt)
- Meiste Arbeitsblätter: S/W ausreichend
- Ausmalbilder: Farbe erforderlich
- Ersparnis: S/W kostet 80% weniger pro Seite

Duplex (beidseitig): Wo angemessen
- Übungsarbeitsblätter: Einseitig (Schüler schreiben darauf)
- Lesetexte: Duplex OK (spart Papier)
- Wichtig: Prüfen, welche Seite "oben" ist (Ausrichtung wichtig)

Papier: Standard 80g Kopierpapier
- Karton vermeiden: Teuer, für meiste Arbeitsblätter unnötig
- Recyclingpapier vermeiden: Staut manchmal, kann weniger scharf sein
```

---

### Tipps für Massendruck

**Für 30-Schüler-Klasse**:
```
Druckauftrag: 30 Kopien des Arbeitsblatts

Einstellungen:
1. Papierfach vollständig laden (200+ Blatt)
2. "Sortieren" auswählen (falls später geheftet)
3. "Alle auf einmal drucken" wählen (nicht einzeln)
4. Druckauftrag senden
5. Weggehen (laufen lassen)

Zeit: 30 Kopien = ~3 Minuten Druckzeit
Vermeiden: Am Drucker stehen (Zeit für andere Aufgaben nutzen)
```

## Häufige Fehler vermeiden

**Von anderen lernen**

### Fehler 1: Zu viele Parameter

**Problem**:
```
Neuer Nutzer: Passt jede einzelne Option an (überfordert von Auswahl)
Ergebnis: Entscheidungslähmung (dauert 10 Minuten für ein Arbeitsblatt)
```

**Lösung**:
```
Einfach anfangen: Erst Standardeinstellungen verwenden
Generieren: Ergebnis sehen
Anpassen: Nur ändern, was geändert werden muss
Iterieren: Schrittweise mehr Optionen erkunden

Erstes Arbeitsblatt: 2 Minuten (Lernphase)
Fünftes Arbeitsblatt: 30 Sekunden (sicher)
```

---

### Fehler 2: Falscher Schwierigkeitsgrad

**Problem**:
```
Lehrkraft: Erstellt "schweres" Arbeitsblatt für schwache Schüler
Ergebnis: Frustration, Aufgeben
```

**Lösung**:
```
Leichter beginnen: Bei Unsicherheit eine Stufe niedriger
Testen: Erst einem Schüler geben
Anpassen: Leichter oder schwerer je nach Reaktion
Verfeinern: Schwierigkeit mit der Zeit genau richtig einstellen
```

---

### Fehler 3: Dateien nicht organisiert halten

**Problem**:
```
Downloads: Alle im Standard-"Downloads"-Ordner
Monate später: "Wo ist die Photosynthese-Wortsuche, die ich erstellt habe?"
Ergebnis: Arbeitsblätter unnötig neu erstellen
```

**Lösung**:
```
Sofort: Heruntergeladenes Arbeitsblatt in passenden Ordner verschieben
Benennung: Beschreibende Namen verwenden (nicht nur "arbeitsblatt1.pdf")
Wiederverwenden: Nächstes Jahr finden und wiederverwenden (Zeit sparen)
```

## Häufige Probleme beheben

**Schnelle Lösungen**

### Problem: "Arbeitsblatt wird nicht generiert"

**Mögliche Ursachen**:
```
1. Zu viele Wörter für Rastergröße
   Lösung: Wortanzahl reduzieren oder Rastergröße erhöhen

2. Internetverbindung verloren
   Lösung: WLAN prüfen, Seite aktualisieren

3. Ungültige Zeichen in Eingabe
   Lösung: Sonderzeichen entfernen, Standardbuchstaben verwenden
```

---

### Problem: "Druckqualität ist schlecht"

**Lösungen**:
```
1. Druckereinstellungen prüfen (sollte "Standard" sein, nicht "Entwurf")
2. Ausreichende Tintenfüllstände sicherstellen (Patrone bei niedrig ersetzen)
3. Druckköpfe reinigen (Drucker-Wartungsmenü)
4. Qualitätspapier verwenden (billiges Papier kann verwaschen aussehen)
```

---

### Problem: "Heruntergeladene Datei nicht auffindbar"

**Lösung**:
```
1. Standard-Downloads-Ordner prüfen
2. Computer nach heute erstelltem PDF durchsuchen
3. Browser: Download-Verlauf prüfen (zeigt Dateispeicherort)
4. Künftig: Sofort in organisierten Ordner speichern
```
