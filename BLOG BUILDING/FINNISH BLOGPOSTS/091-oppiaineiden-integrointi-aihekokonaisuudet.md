## Johdanto: Oppiainerajojen ylittäminen

**Perinteinen lähestymistapa**: Oppiaineet eristyksissä toisistaan
```
8:00-9:00: Matematiikka (ei yhteyttä muihin aineisiin)
9:00-10:00: Äidinkieli (ei yhteyttä matematiikkaan)
10:00-11:00: Ympäristöoppi (ei yhteyttä äidinkieleen tai matematiikkaan)
Tulos: Pirstaleinen oppiminen (oppilaat eivät näe yhteyksiä)
```

**Tutkimus** (Vars, 2001): Integroitu opetus parantaa **muistamista 30-40 %** (yhteydet vahvistavat muistijälkeä)

**Aihekokonaisuusajattelu**: Yksi teema läpäisee kaikki oppiaineet
```
Viikon teema: "Sää"
Matematiikka: Graafinen esitys lämpötilasta (laskutoimitukset + tiedon analysointi)
Äidinkieli: Säähän liittyvät tekstit (ymmärtäminen + sanasto)
Ympäristöoppi: Sääilmiöt (meteorologian perusteet)
Kirjoittaminen: Säähavaintopäiväkirja (havainnointi + kuvaus)
Yhteiskuntaoppi: Ilmastovyöhykkeet (maantiedeyhteys)

Tulos: Syvällinen ymmärrys (oppilaat näkevät, miten aineet liittyvät toisiinsa)
```

**Keskeinen periaate**: Todellinen maailma ei erottele oppiaineita - oppimisenkaan ei pitäisi

## Matematiikka + Äidinkieli -integraatio

**Luonnolliset yhteydet**: Sanallisia tehtäviä, tiedon tulkinta, mittaaminen

### Tarina-tehtävien luominen

**Toiminta**: Oppilaat kirjoittavat matematiikan sanallisia tehtäviä kirjallisuuden pohjalta
```
Luettava teos: "Peppi Pitkätossu" (Astrid Lindgren)
Yhteys: Seikkailumatematiikka

Oppilaiden luomia tehtäviä:
1. "Peppi nosti hevosen 5 kertaa maanantaina ja 7 kertaa tiistaina. Kuinka monta nostoa yhteensä?"
2. "Villa Villekullassa on 3 huonetta yläkerrassa ja 4 huonetta alakerrassa. Montako huonetta yhteensä?"
3. "Jos Peppi löytää 10 kultarahaa viikossa, kuinka monta rahaa hänellä on 4 viikon jälkeen?"

Hyödyt:
- Luetun ymmärtäminen (ymmärrä tarina luodaksesi tehtäviä)
- Matematiikan soveltaminen (todellinen konteksti)
- Luova ajattelu (yhdistä kirjallisuus ja luvut)
```

---

### Tiedot teksteistä

**Toiminta**: Poimi tietoa tietokirjoista, luo matematiikan tehtäviä
```
Tietoartikkeli: "Eläinten nopeudet"
Poimittu tieto:
- Gepardi: 110 km/h
- Leijona: 80 km/h
- Norsu: 40 km/h
- Ihminen: 45 km/h (Usain Bolt)

Luodut tehtävät:
1. Pylväsdiagrammi: Merkitse eläinten nopeudet
2. Sanallisia tehtäviä: "Kuinka paljon nopeampi gepardi on kuin norsu?"
3. Kertolasku: "Jos gepardi juoksee 3 tuntia 110 km/h, kuinka pitkän matkan?"

Aika: 42 sekuntia luoda matematiikan tehtävä luonnontiedon tiedoista
Akateeminen arvo: Oppilaat harjoittelevat matematiikkaa JA oppivat luonnontiedon faktoja
```

## Ympäristöoppi + Matematiikka -integraatio

**LUMA-yhteys**: Luonnontiede perustuu matemaattiseen analyysiin

### Mittaustoiminnot

**Toiminta**: Tee kokeita, analysoi matematiikalla
```
Koe: Kasvin kasvu 4 viikon aikana
Mittaustehtävä:

Viikko 1: ___ cm
Viikko 2: ___ cm
Viikko 3: ___ cm
Viikko 4: ___ cm

Matemaattinen analyysi:
1. Kokonaiskasvu: Viikko 4 - Viikko 1 = ___
2. Keskimääräinen kasvu viikossa: Yhteensä ÷ 4 = ___
3. Suurin kasvu: Viikkojen ___ ja ___ välillä
4. Graafi: Luo kasvukäyrä

Luonnontiedon oppiminen: Kasvin elinkaari, fotosynteesi
Matematiikan oppiminen: Mittaaminen (cm), vähennyslasku, keskiarvot, grafiikka
```

---

### Tieteelliset taulukot

**Toiminta**: Kirjaa havaintoja, luo tietopohjaisia tehtäviä
```
Koe: Mikä paperilentokone lentää pisimmälle?

Tiedonkeruu:
Malli A: 4,5 m, 5,5 m, 3,7 m (keskiarvo: 4,6 m)
Malli B: 6,7 m, 7,6 m, 6,1 m (keskiarvo: 6,8 m)
Malli C: 3,0 m, 4,0 m, 3,4 m (keskiarvo: 3,5 m)

Tehtävä (luotu datasta):
1. Mikä malli lensi pisimmälle keskimäärin?
2. Mikä on mallin B lentojen vaihteluväli? (7,6 - 6,1 = ___)
3. Kuinka paljon pidemmälle malli B lensi kuin malli A keskimäärin?
4. Jos malli B lentää vielä 5 kertaa, ennusta kokonaismatka

Suunnitteluprosessi: Suunnittele, testaa, analysoi, paranna
Matematiikan taidot: Mittaaminen, keskiarvot, vaihteluväli, ennustaminen
```

## Yhteiskuntaoppi + Äidinkieli -integraatio

**Historiallinen konteksti kohtaa lukutaidon**

### Historiallinen fiktio

**Toiminta**: Lue historiallista fiktiota, täytä yhteiskuntaopin tehtäviä
```
Kirja: "Täällä Pohjantähden alla" (Väinö Linna) -soveltaen nuoremmille

Lukeminen: Oppilaat lukevat luvut 1-5 (torppariajan Suomi)

Luodut tehtävät:
1. Aikajana: Järjestä kirjan tapahtumat (luetun ymmärtäminen)
2. Kartta: Seuraa tapahtumien paikkoja (maantiede + juoni)
3. Sanasto: 1800-luvun termit (sanahaku: torppa, renki, käskyläinen, rälssi)
4. Ymmärtäminen: Miksi torpparit halusivat omaa maata? (historiallinen konteksti + päättely)

Integraation hyödyt:
- Historia tulee henkilökohtaiseksi (hahmot, joista oppilaat välittävät)
- Luetun ymmärtäminen vahvistuu (konteksti auttaa ymmärtämään)
- Tunneyheys (oppilaat muistavat paremmin)
```

---

### Ajankohtainen lukutaito

**Toiminta**: Uutisartikkelit muuttuvat luku- ja yhteiskuntaopintunnneiksi
```
Artikkeli: "Uusi kierrätysohjelma kaupungissamme"

Lukutaidot:
- Pääajatus: Mistä artikkeli kertoo?
- Tukevat yksityiskohdat: Mitä materiaaleja voi kierrättää?
- Kirjoittajan tarkoitus: Miksi tämä kirjoitettiin?

Yhteiskuntaopin taidot:
- Kansalaisosallistuminen: Miten asukkaat voivat osallistua?
- Ympäristövaikutus: Miksi kierrätys on tärkeää?
- Paikallishallinto: Kuka teki tämän päätöksen?

Tehtävä (luotu 42 sekunnissa):
Ymmärtämiskysymykset + sanasto + yhteys yhteiskuntaoppiin

Todellinen merkitys: Oppilaat näkevät lukemisen tarkoituksen luokkahuoneen ulkopuolella
```

## Kuvataide + Akateemiset aineet -integraatio

**Visuaalinen oppiminen tukee sisällön hallintaa**

### Matematiikkataideprojektit

**Toiminta**: Luo taidetta käyttäen matemaattisia käsitteitä
```
Symmetriapiirustukset:
Oppilaat luovat symmetrisiä kuvioita (perhonen, lumihiutale, mandala)

Matematiikan käsitteet:
- Symmetria-akseli
- Peilausheijastus
- Geometriset muodot
- Kuvioiden toisto

Tehtäväosuus:
"Tunnista symmetria-akselit: Kuinka monta symmetria-akselia kuviossasi on?"
"Nimeä muodot: Luettele kaikki geometriset muodot, joita käytit taideteoksessasi"

Tulos: Matematiikan käsitteet visualisoitu (syvempi ymmärrys luomisen kautta)
```

---

### Luonnontieteen kuvitukset

**Toiminta**: Piirrä tieteellisiä käsitteitä, merkitse sanastolla
```
Aihe: Kasvin osat ja toiminnot

Toiminta:
1. Oppilaat piirtävät yksityiskohtaisen kasvin (juuret, varsi, lehdet, kukka)
2. Merkitsevät jokaisen osan
3. Täyttävät tehtävän: "Yhdistä osa toimintoon"
   - Juuret: _____ (kiinnittävät kasvin, imevät vettä)
   - Varsi: _____ (kuljettavat vettä/ravinteita, tukevat)
   - Lehdet: _____ (fotosynteesi, kaasunvaihto)
   - Kukka: _____ (lisääntyminen, houkuttelevat pölyttäjiä)

Kuvataidon taidot: Havainnointi, yksityiskohtainen piirtäminen, merkitseminen
Ympäristöopin taidot: Kasvin rakenne, toimintojen ymmärtäminen
```

## Aihekokonaisuuksien suunnittelu

**Järjestä oppiminen keskeisen teeman ympärille**

### Viikkoesimerkkiteema: "Meren elämä"

**Maanantai-perjantai -integraatio**:
```
Maanantai (Matematiikka):
- Meren matemaattiset sanatehtävät (valaiden painot, meren syvyydet)
- Meren lämpötilojen kaavioiminen eri syvyyksillä
- Tehtävä: 20 meriaiheista laskutehtävää

Tiistai (Äidinkieli):
- Tietoartikkeli: "Koralliriutan ekosysteemit"
- Sanasto: koralli, polyypit, symbioosi, biodiversiteetti
- Ymmärtämistehtävä + sanahakuluudusto

Keskiviikko (Ympäristöoppi):
- Tutkitaan meren vyöhykkeitä (aurinko-, hämärä-, keskiyö-, syvänmeren vyöhyke)
- Tutkitaan merieläimiä ja niiden vyöhykkeitä
- Tehtävä: Yhdistä eläimet oikeisiin vyöhykkeisiin

Torstai (Yhteiskuntaoppi):
- Meren suojelu ja saastuminen
- Ihmisen vaikutus meren ekosysteemeihin
- Tehtävä: Ristisanatehtävä suojelusanastosta

Perjantai (Kuvataide + Kirjoittaminen):
- Piirrä ja väritä meren maisema (kuvataide-integraatio)
- Kirjoita tietokappale merieläimestä (kirjoittaminen)
- Tehtävä: Tietotekstin graafinen jäsennin

Tulos: Viikko yhtenäistä oppimista (meren tuntemus 5 näkökulmasta)
Luontiaika: 5 tehtävää × 42 s = 210 s (3,5 minuuttia)
```

## Todelliset sovellukset

**Näytä oppilaille, miten aineet toimivat yhdessä koulun ulkopuolella**

### Ruokakaupan matematiikka + Talous

**Toiminta**: Ostossimulaatio
```
Yhteiskuntaoppi: Taloustiede (tarpeet vs. halut, budjetointi)
Matematiikka: Yhteenlasku, kertolasku, rahat, prosentit

Tehtävä:
"Perheen ostosretki"
Budjetti: 100 €

Tuotteet:
Maito (4 €) × 2 = ____€
Leipä (3 €) × 1 = ____€
Omenat (2 €/kg) × 5 kg = ____€
...

Kysymykset:
1. Mikä on kokonaiskustannus?
2. Kuinka paljon budjetista jää jäljelle?
3. Arvonlisävero on 14 %. Mikä on lopullinen kokonaissumma?
4. Voitko ostaa jäätelön (7 €)? Miksi tai miksi et?

Elämäntaidot: Budjetointi, päätöksenteko, matematiikan soveltaminen
Akateemiset taidot: Laskutoimitukset, prosentit, ongelmanratkaisu
```

### Säätietojen analysointi

**Toiminta**: Seuraa paikallista säätä, integroi oppiaineet
```
Ympäristöoppi: Meteorologia (lämpötila, sadanta, ilmanpaine)
Matematiikka: Tiedonkeruu, kaavioiminen, keskiarvot
Äidinkieli: Sääraportit (tietokirjallisuuden ymmärtäminen)
Yhteiskuntaoppi: Miten sää vaikuttaa yhteisöihin

Päivittäinen seuranta (2 viikkoa):
Päivä | Ylin lämpö | Alin lämpö | Sadanta | Taivas
...

Yksikön lopun tehtävä (luotu):
1. Mikä oli ylimpien lämpötilojen keskiarvo?
2. Kuinka monena päivänä satoi?
3. Luo viivakaavio ylimmistä lämpötiloista
4. Kirjoita sääyhteenveto (kuumimmat/kylmimmät/sateisimmat päivät)

Oppiainerajat ylittävät yhteydet näkyvissä: Kaikki aineet liittyvät yhteen ilmiöön
```

## Oppisisältölukutaito

**Lukemista ja kirjoittamista jokaisessa aineessa**

### Matematiikan päiväkirjat

**Toiminta**: Oppilaat selittävät matemaattista ajatteluaan kirjallisesti
```
Matematiikan tehtävä: 25 × 4 = 100

Perinteinen: Näytä vain laskutoimitus
Integroitu: Kirjoita selitys

Oppilas kirjoittaa:
"Ratkaisin 25 × 4 ajattelemalla markkoja. Eurossa on 4 kpl 25 sentin kolikoita, joten 25 eurossa on 100 kpl 25 sentin kolikoita. Voin myös jakaa sen osiin (20 × 4) + (5 × 4) = 80 + 20 = 100."

Hyödyt:
- Matematiikan ymmärrys syvenee (selittäminen = todellinen ymmärrys)
- Kirjoitusharjoitus (ilmaise matemaattista päättelyä)
- Useita strategioita (oppilaat oppivat toistensa selityksistä)

Tehtävän ohje (luotu): "Ratkaise 18 × 5 ja selitä strategiasi 2-3 lauseessa"
```

---

### Luonnontieteen kirjoittaminen

**Toiminta**: Kirjoita selityksiä, menettelytapoja, havaintoja
```
Koe: Uppoaako vai kelluuko

Perinteinen: Testaa vain esineitä
Integroitu: Tieteellinen kirjoittaminen

Oppilastehtävä:
Hypoteesi: Ennustan, että ___ ___, koska ___
Menettely: Ensin ___, sitten ___, lopuksi ___
Havainnot: Huomasin ___
Tulokset: ___ (upposi/kellui), koska ___
Johtopäätös: Hypoteesini oli (oikea/väärä). Opin ___

Ympäristöopin taidot: Tieteellinen menetelmä, havainnointi, analyysi
Kirjoitustaidot: Ohjeellinen kirjoitus, syy-seuraussuhteet, pohdinta
```

## Aihekokonaisuuksien suunnittelu

**Strateginen lähestymistapa integraatioon**

### Kuukausittainen suunnitteluprosessi

**Vaihe 1: Valitse keskeinen teema** (merkityksellinen, kiinnostava, opetussuunnitelmaan sopiva)
```
Esimerkki: "Lento"
Soveltuu: Luokat 2-5
Kesto: 2 viikkoa
```

**Vaihe 2: Tunnista tavoitteet jokaiselle aineelle**
```
Matematiikka: Mittaaminen (etäisyys, nopeus), kaavioiminen
Äidinkieli: Tietokirjallisuuden ymmärtäminen, pääajatus, sanasto
Ympäristöoppi: Voimat (nosto, vastus, työntö, painovoima), aerodynamiikka
Yhteiskuntaoppi: Lentohistoria (Wright-veljekset)
Kirjoittaminen: Tietoteksti (miten lentokoneet toimivat)
```

**Vaihe 3: Luo integroituja tehtäviä** (10 minuuttia koko yksikölle)
```
Matematiikka: Paperilentokone-etäisyyden kirjaus + kaavioiminen (42 s)
Äidinkieli: Lentohistorian kappale + ymmärtäminen (42 s)
Ympäristöoppi: Lennon voimien kaavio + merkitseminen (42 s)
Yhteiskuntaoppi: Wright-veljesten aikajana + kysymykset (42 s)
Kirjoittaminen: Tietotekstin graafinen jäsennin (42 s)
Sanasto: Lentotermien sanahaku (42 s)
...

Yhteensä: 15 tehtävää × 42 s = 10,5 minuuttia
Tulos: Kahden viikon integroitu yksikkö valmis opetettavaksi
```

## Arviointi oppiaineiden yli

**Integroitu arviointi mittaa syvempää ymmärrystä**

### Huipentava projekti

**Toiminta**: Monialainen projekti osoittaa oppimisen
```
Teema: "Paikallishistoria"

Projektin vaatimukset:
1. Tutkimus (Yhteiskuntaoppi): Haastattele vanhempia ihmisiä, vieraile paikallismuseossa
2. Kirjoittaminen (Äidinkieli): Kirjoita 2-sivuinen raportti paikallisesta historiallisesta tapahtumasta
3. Matematiikka (Tieto): Luo aikajana päivämäärineen, laske vuosien välinen aika
4. Kuvataide (Visuaalinen): Luo juliste kuvituksilla ja valokuvillaa
5. Esitys (Puhuminen): Esitä tulokset luokalle (5 minuuttia)

Arviointikriteerit:
- Historiallinen tarkkuus (Yhteiskuntaopin tavoitteet)
- Kirjoituksen laatu (Äidinkielen tavoitteet)
- Matemaattinen tarkkuus (Matematiikan tavoitteet)
- Visuaalinen esitys (Kuvataiteen tavoitteet)
- Suullinen viestintä (Puhumisen tavoitteet)

Tulos: Yksi projekti arvioi 5+ oppiainetta (tehokas, aito)
```
