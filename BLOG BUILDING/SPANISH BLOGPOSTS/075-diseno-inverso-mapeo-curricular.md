## Introducción: Dos Formas de Planificar

**Planificación tradicional** (hacia adelante):
```
Paso 1: Elegir actividades interesantes (fichas divertidas, proyectos creativos)
Paso 2: Impartir las actividades
Paso 3: Crear el examen al final de la unidad
Problema: Las actividades pueden no alinearse con la evaluación
Resultado: Los alumnos hacen actividades pero no dominan los objetivos
```

**Diseño inverso** (planificación desde el resultado):
```
Paso 1: Identificar resultados deseados (¿qué deben SABER los alumnos?)
Paso 2: Determinar evidencias de aprendizaje (¿cómo DEMOSTRARÁN que aprendieron?)
Paso 3: Planificar experiencias de aprendizaje (¿qué actividades los llevarán allí?)
Resultado: Cada actividad construye intencionalmente hacia el dominio
```

**Investigación** (Wiggins & McTighe, 2005 - Diseño para la Comprensión): El diseño inverso aumenta el rendimiento estudiantil **15-25%** al garantizar coherencia entre objetivos, instrucción y evaluación

**Concepto clave**: Comienza con el FIN en mente (la evaluación), trabaja hacia atrás hacia la instrucción

## Las 3 Etapas del Diseño Inverso

### Etapa 1: Identificar Resultados Deseados

**Pregunta**: ¿Qué deben saber y saber hacer los estudiantes?

**Ejemplo de unidad**: Fracciones para 4º grado

**Objetivos curriculares**:
```
- Comprender que una fracción a/b es equivalente a (n×a)/(n×b)
- Comparar dos fracciones con diferentes numeradores y denominadores
- Sumar y restar fracciones con denominadores iguales
```

**Comprensiones duraderas** (ideas que permanecen):
```
1. Las fracciones representan partes de un todo
2. Diferentes fracciones pueden representar la misma cantidad (equivalencia)
3. Las fracciones se pueden comparar encontrando denominadores comunes
```

**Preguntas esenciales**:
```
1. ¿Cómo pueden dos fracciones diferentes ser iguales?
2. ¿Cuándo usamos fracciones en la vida real?
3. ¿Cómo sabemos qué fracción es mayor?
```

---

### Etapa 2: Determinar Evidencias Aceptables (Evaluación)

**Pregunta**: ¿Cómo demostrarán los estudiantes el dominio?

**Evaluación final** (diseñada ANTES de la instrucción):

**Parte 1: Demostración de habilidades** (20 puntos)
```
1. Dibuja un modelo mostrando que 2/4 = 1/2 (prueba visual)
2. Compara 3/5 y 2/3 (muestra tu trabajo usando denominadores comunes)
3. Suma 2/8 + 3/8 (resuelve con modelo)
4. Lista 3 usos de fracciones en la vida real (aplicación)
```

**Parte 2: Resolución de problemas** (10 puntos)
```
Problema: "Comiste 2/6 de una pizza. Tu amigo comió 3/9 de la misma pizza. ¿Quién comió más? Demuestra tu respuesta."
Rúbrica:
- 4 puntos: Respuesta correcta con prueba visual
- 3 puntos: Respuesta correcta, sin demostración
- 2 puntos: Respuesta incorrecta, intento razonable
- 0-1 puntos: Sin procedimiento mostrado
```

**Criterio de dominio**: 80% de precisión = dominio alcanzado (24/30 puntos)

**Ahora**: Diseña la instrucción para asegurar que los estudiantes PUEDAN hacer estas tareas

---

### Etapa 3: Planificar Experiencias de Aprendizaje (Instrucción)

**Pregunta**: ¿Qué actividades prepararán a los estudiantes para la evaluación?

**Semana 1: Introducción a las fracciones**
```
Lunes: Modelos de fracciones con pizzas (concreto)
Martes: Ficha - sombrear fracciones (pictórico)
Miércoles: Sopa de letras con fracciones (vocabulario: numerador, denominador, equivalente)
Jueves: Colorear fracciones (1/2 rojo, 1/4 azul)
Viernes: Evaluación formativa (5 problemas - verificar comprensión)

Meta: Los estudiantes identifican y dibujan fracciones
```

**Semana 2: Fracciones equivalentes**
```
Lunes: Doblado de papel (1/2 = 2/4 = 4/8 físicamente)
Martes: Ficha - práctica de fracciones equivalentes (15 problemas)
Miércoles: Crucigrama de fracciones (vocabulario: equivalente, simplificar)
Jueves: Rompecabezas matemático (encuentra fracciones equivalentes)
Viernes: Evaluación formativa (5 problemas - solo equivalencia)

Meta: Los estudiantes comprenden por qué 2/4 = 1/2
```

**Semana 3: Comparar fracciones**
```
Lunes: Tiras de fracciones (comparación visual)
Martes: Ficha - comparar fracciones con denominadores comunes (20 problemas)
Miércoles: Ficha - comparar fracciones (encontrar mínimo común denominador)
Jueves: Problemas de aplicación (¿cuál es mayor?)
Viernes: Evaluación formativa (5 problemas - comparación)

Meta: Los estudiantes comparan cualquier par de fracciones
```

**Semana 4: Suma y resta**
```
Lunes: Sumar fracciones con modelos
Martes: Ficha - suma con denominadores iguales (20 problemas)
Miércoles: Ficha - resta con denominadores iguales (20 problemas)
Jueves: Práctica de operaciones mixtas
Viernes: PRUEBA DE UNIDAD (la evaluación diseñada en Etapa 2)

Meta: Los estudiantes suman/restan fracciones, listos para la prueba
```

**Verificación de alineación**: Cada actividad desarrolla habilidades necesarias para la evaluación final

## Mapeo Curricular: Vista Anual Completa

**Mapa curricular**: Panorama general del contenido del año completo

**Propósito**: Asegurar cobertura integral (sin lagunas de aprendizaje)

---

### Mapa Curricular Mensual (Ejemplo: Matemáticas 4º Grado)

**Septiembre: Sentido numérico y valor posicional**
```
Objetivos: Suma/resta multi-dígito, redondeo
Fichas clave:
- Semana 1: Suma de 3 dígitos (20 problemas diarios)
- Semana 2: Resta de 3 dígitos (20 problemas diarios)
- Semana 3: Redondeo a decenas, centenas (sopa de letras con números)
- Semana 4: Problemas de aplicación (operaciones mixtas)
Evaluación: Prueba de 25 problemas (80% meta de dominio)
```

**Octubre: Multiplicación - hechos y estrategias**
```
Objetivos: Multiplicar hasta 100, modelos de área
Fichas clave:
- Semana 1: Tablas 0-5 (Bingo de imágenes para práctica)
- Semana 2: Tablas 6-10 (fichas de práctica diaria)
- Semana 3: Modelos de área (multiplicación en cuadrícula)
- Semana 4: Problemas de aplicación
Evaluación: Prueba cronometrada (100 operaciones en 5 minutos, 80% meta)
```

**Noviembre: División**
```
Objetivos: División hasta 100, residuos
Fichas clave:
- Semana 1: Hechos de división (inverso de multiplicación)
- Semana 2: División larga (2 dígitos ÷ 1 dígito)
- Semana 3: División con residuos
- Semana 4: Problemas de división
Evaluación: Prueba de 30 problemas de división
```

**Diciembre: Fracciones (introducción)**
```
Objetivos: Comprender fracciones como partes de un todo
Fichas clave:
- Semana 1: Identificar fracciones (sombrear modelos)
- Semana 2: Fracciones equivalentes (modelos visuales)
- Semana 3: Comparar fracciones (mismo denominador)
Evaluación: Prueba visual de fracciones (dibujar y comparar)
```

**Enero: Fracciones (operaciones)**
```
Objetivos: Sumar/restar fracciones con denominadores iguales
Fichas clave:
- Semanas 1-2: Sumar fracciones (modelos + simbólico)
- Semanas 3-4: Restar fracciones
Evaluación: Prueba de 25 problemas de operaciones con fracciones
```

**Febrero: Decimales**
```
Objetivos: Notación decimal, valor posicional hasta centésimos
Fichas clave:
- Semana 1: Leer/escribir decimales
- Semana 2: Comparar decimales
- Semana 3: Sumar decimales
- Semana 4: Restar decimales
Evaluación: Prueba de operaciones con decimales
```

**Marzo: Medición y datos**
```
Objetivos: Convertir unidades, gráficos de línea, análisis de datos
Fichas clave:
- Semana 1: Conversión de unidades (cm/m, ml/l)
- Semana 2: Gráficos de línea (crear desde datos)
- Semana 3: Análisis de datos (sopa de letras con términos de medición)
Evaluación: Prueba de medición + interpretación de datos
```

**Abril: Geometría**
```
Objetivos: Ángulos, figuras, líneas, simetría
Fichas clave:
- Semana 1: Medición de ángulos (práctica con transportador)
- Semana 2: Tipos de triángulos/cuadriláteros (clasificación)
- Semana 3: Líneas (paralelas, perpendiculares, secantes)
- Semana 4: Simetría (dibujar ejes de simetría)
Evaluación: Prueba de geometría + tareas de construcción
```

**Mayo: Repaso y enriquecimiento**
```
Enfoque: Repaso espiral de todo el contenido del año
Fichas clave:
- Semana 1: Repaso de operaciones numéricas (práctica mixta)
- Semana 2: Repaso de fracciones/decimales
- Semana 3: Repaso de medición/geometría
- Semana 4: Preparación para evaluaciones finales
Evaluación: Examen final integral
```

## Protocolo de Alineación Curricular

**Problema**: Las actividades no corresponden con los objetivos

**Solución**: Verificación explícita de alineación

### Lista de Verificación de Alineación

**Para cada ficha de trabajo**:
```
☐ Objetivo(s) abordado(s): [escribir código del objetivo]
☐ Meta de aprendizaje: Los estudiantes [objetivo específico y medible]
☐ Criterio de éxito: [cómo saber si se dominó]
☐ Conexión con evaluación: [¿qué pregunta de la prueba prepara esto?]
```

**Ejemplo**:
```
Ficha: Comparar fracciones (20 problemas)

☐ Objetivo: Comparar fracciones con diferentes denominadores
☐ Meta: Los estudiantes compararán dos fracciones con diferentes denominadores usando denominadores comunes con 80% de precisión
☐ Criterio de éxito: 16/20 problemas correctos
☐ Conexión con evaluación: Preguntas 7-10 de la prueba de unidad requieren comparación de fracciones

Alineación verificada ✓
```

## Instrucción Basada en la Evaluación

**Principio**: La práctica debe parecerse a la evaluación

**Desajuste tradicional**:
```
Práctica: Fichas abiertas (escribir respuesta)
Evaluación: Prueba de opción múltiple (marcar respuestas)
Resultado: Choque de formato el día del examen (carga cognitiva desperdiciada)
```

**Práctica alineada**:
```
Práctica: Mismo formato que la evaluación
Ejemplo: Si la prueba es de opción múltiple, practicar con opción múltiple
Resultado: Familiaridad con el formato (carga cognitiva enfocada en contenido)
```

## Desarrollo de Guía de Ritmo

**Guía de ritmo**: Cronograma para cubrir los objetivos

**Propósito**: Asegurar completar el currículo antes del fin de año

### Crear una Guía de Ritmo

**Paso 1: Contar días lectivos**
```
Año escolar: 180 días
- Vacaciones/feriados: 20 días
- Días de evaluación: 10 días
- Asambleas/excursiones: 10 días
- Días de reserva (enfermedad, imprevistos): 10 días
Días lectivos reales: 130 días
```

**Paso 2: Asignar días a objetivos**
```
Matemáticas 4º: 30 objetivos
130 días ÷ 30 objetivos = 4,3 días por objetivo (promedio)

Ajustar por complejidad:
- Objetivos simples (redondeo): 2 días
- Objetivos complejos (operaciones con fracciones): 8 días
```

**Paso 3: Crear horario semanal**
```
Semana 1 (15-19 ago): Valor posicional y redondeo
Semana 2 (22-26 ago): Suma multi-dígito
Semana 3 (29 ago-2 sep): Resta multi-dígito
...

Resultado: Cada objetivo tiene tiempo dedicado
```

**Beneficio del generador**: Crea materiales rápidamente para cada objetivo
```

Generar: 5 fichas de suma (3,5 minutos)

## Diferenciación en el Diseño Inverso

**Desafío**: No todos los estudiantes alcanzan el dominio al mismo ritmo

**Solución**: Objetivos escalonados

### Niveles de Dominio Escalonados

**Nivel 1: Dominio básico** (80% de estudiantes)
```
Objetivo: Comparar fracciones con denominadores iguales
Evaluación: 3/4 vs 2/4 (¿cuál es mayor?)
```

**Nivel 2: Competente** (60% de estudiantes)
```
Objetivo: Comparar fracciones con diferentes denominadores
Evaluación: 3/4 vs 2/3 (usar denominadores comunes)
```

**Nivel 3: Avanzado** (20% de estudiantes)
```
Objetivo: Comparar tres o más fracciones, explicar razonamiento
Evaluación: Ordenar 2/3, 3/4, 5/6 de menor a mayor, justificar
```

**Instrucción**: Proveer materiales para los tres niveles
```
Lunes: Todos los estudiantes - Comparación básica (denominadores iguales)
Martes: Todos los estudiantes - Introducción a diferentes denominadores
Miércoles: Nivel 1 - Más práctica con denominadores iguales (Estación 1)
           Nivel 2 - Práctica diferentes denominadores (Estación 2)
           Nivel 3 - Comparación avanzada multi-fracción (Estación 3)
```

**Uso del generador**: Crea 3 niveles instantáneamente
```
Ficha Nivel 1: Comparar fracciones (solo mismo denominador)
Ficha Nivel 2: Comparar fracciones (encontrar MCD)
Ficha Nivel 3: Ordenar 4+ fracciones (complejo)

Tiempo: 126 segundos (42 seg cada una) para los 3 niveles

## Protocolo de Reflexión y Revisión

**Después de completar la unidad**: Reflexionar y mejorar

### Análisis Post-Unidad

**Preguntas a formular**:
```
1. ¿Los estudiantes dominaron los objetivos? (verificar datos de evaluación)
   - Si sí: Mantener unidad como está
   - Si no: ¿Qué necesita repaso?

2. ¿Qué actividades fueron más efectivas?
   - Mayor participación: [listar actividades]
   - Mejores resultados de aprendizaje: [listar actividades]
   - Mantener estas, eliminar las inefectivas

3. ¿El ritmo funcionó?
   - Muy apresurado: Agregar 1-2 días el próximo año
   - Muy lento: Reducir 1-2 días el próximo año

4. ¿Qué cambiaré el próximo año?
   - [revisiones específicas]
```

**Ejemplo de reflexión**:
```
Unidad: Fracciones (4 semanas)
Resultados de evaluación: 85% de dominio (por encima de meta 80%) ✓

Actividad más efectiva: Doblado de papel (Semana 2, Lunes)
- 100% participación
- Los estudiantes tuvieron momento "¡Ajá!"
- MANTENER ESTO

Menos efectiva: Sopa de letras de fracciones (Semana 1, Miércoles)
- Estudiantes aburridos (demasiado simple)
- REEMPLAZAR con Sudoku de fracciones (más desafiante)

Ritmo: Justo adecuado (terminamos viernes como planeado)

Cambios para próximo año:
- Reemplazar sopa de letras con Sudoku
- Agregar una verificación formativa más en Semana 3 (algunos estudiantes aún confundidos el jueves)
```
