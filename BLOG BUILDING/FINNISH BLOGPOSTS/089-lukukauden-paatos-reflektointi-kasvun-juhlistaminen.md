## Johdanto: Merkityksellinen lukukauden päätös

**Kouluvuoden viimeisten viikkojen haaste**: Oppilaat ovat hajamielisiä, käytöshaasteita ilmenee, aikaa tuhlataan

**Tutkimus** (McCombs & Pope, 1994): Jäsennelty reflektointi parantaa oppimisen pysyvyyttä **35 %** (kertaaminen vahvistaa oppimista)

**Perinteinen lukukauden päätös**: Elokuvat, juhlat, ei opetuksellista sisältöä
```
Todellisuus:
- 2 viikkoa: Ei-opetuksellista toimintaa
- Oppimisen heikkeneminen: Oppilaat unohtavat opittua (kesän oppimiskato alkaa aikaisin)
- Käytösongelmat: Ei rakennetta = kaaos
```

**Merkityksellinen vaihtoehto**: Reflektointi + juhlistaminen + kertaus
```
Rakenne:
- Viikko 1: Koko vuoden oppimisen kertaus (järjestelmällisesti)
- Viikko 2: Reflektointitehtävät + portfolion kokoaminen
- Viimeiset päivät: Juhlistaminen opetuksellisella yhteydellä

Tulos: Oppiminen jatkuu, oppilaat lähtevät ylpeinä kasvustaan
```

## Kasvun portfolion kokoaminen

**Konkreettinen näyttö vuoden edistymisestä**

### Portfolion sisältö

**Syyskuu-toukokuu vertailu**:
```
Alkuosa: "Mistä aloitin"
- Syyskuun kirjoitusnäyte (ensimmäinen viikko)
- Syyskuun matematiikan arviointi (lähtötaso)
- Syyskuun väritystehtävä (hienomotoriikan lähtötaso)

Loppuosa: "Missä olen nyt"
- Toukokuun kirjoitusnäyte (viimeinen viikko)
- Toukokuun matematiikan arviointi (loppuarviointi)
- Toukokuun taidetyö (hienomotoriikan edistyminen)

Keskiosa: "Matka"
- Valokuvia vuodelta
- Suosikkityöarkit (oppilas valitsee)
- Saavutustodistukset
- Kasvukuvaaja (lukusujuvuus, jne.)
```

**Kokoamistoiminta** (Viikko 1 kahdesta viimeisestä viikosta):
```
Maanantai-tiistai: Oppilaat keräävät työnäytteitä kansioista
Keskiviikko: Järjestävät portfolion (aikajärjestyksessä)
Torstai: Koristelevat kannen, kirjoittavat reflektoinnin
Perjantai: Jakavat parin tai pienryhmän kanssa
```

## Kertaustehtävät (2 viimeistä viikkoa)

**Järjestelmällinen kertaus vuoden sisällöstä**

### Maanantai-perjantai kertausaikataulu

**Viikko 1: Sisällön kertaus**
```
Maanantai: Matematiikan kierrekertaus (kaikki vuoden taidot, 30 tehtävän työarkki)
Tiistai: Sanaston sanahaku (100 sanaa koko vuodelta)
Keskiviikko: Luonnontieteen ristisanatehtävä (keskeiset termit kaikista yksiköistä)
Torstai: Yhteiskuntaopin aikajana (pääasiat/henkilöt joita opiskeltiin)
Perjantai: Luetun ymmärtäminen (katkelmia suosikkiteksteistä)

Muoto: Työarkit + pelit + keskustelut
Tavoite: Virkistää muisti ennen kesää
```

**Viikko 2: Reflektointi + juhlistaminen**
```
Maanantai: Luo portfoliot (järjestä työnäytteet)
Tiistai: Kirjoita reflektointi-esseet (vuosikatsaus)
Keskiviikko: Tallenna videoviestit (neuvoja ensi vuoden luokalle)
Torstai: Nimikirjat (luokkakaverit allekirjoittavat muistokirjat)
Perjantai: Juhlistaminen (opetukselliset pelit, palkintoseremonia, juhlat)
```

## Kesän oppimispaketti

**Estä kesän oppimiskato**

### Paketin sisältö

**6-8 viikon tarjonta**:
```
Viikko 1: 5 kertaustyöarkkia (kaikki aineet)
Viikko 2: 5 työarkkia
...
Viikko 8: 5 työarkkia

Yhteensä: 40 työarkkia (aineiden sekoitus)

Luontiaika: 28 minuuttia (40 × 42 sekuntia)
Tulostusaika: 45 minuuttia (1 200 sivua 30 oppilaalle)

Sisällytä:
- Ohjeet vanhemmille (20 min/päivä, 3-5 päivää/viikko)
- Vastausavaimet (vanhemmat voivat tarkistaa)
- Lukupäiväkirja (seuraa kesän lukemista)
```
