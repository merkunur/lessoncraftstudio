## Introduction : La Crise du Dernier Moment

**Réalité enseignante** : Besoin d'un remplaçant à court préavis (maladie, urgence, formation professionnelle)

**Le problème** : Préparer des plans de remplacement à 5h30 du matin alors qu'on est malade
```
6h00 : Réveil avec la grippe
6h15 : Appel pour demander un remplaçant
6h30 : Tentative d'écrire des plans de cours (en se sentant mal)
7h00 : Abandon, laisser une note générique "faites-les lire"
Résultat : Journée scolaire gâchée (le remplaçant ne sait pas quoi enseigner)
```

**La solution** : Plans d'urgence préparés à l'avance (créés quand on est en bonne santé)

**Recherche** (Weems & Rogers, 2007) : Des plans de remplacement de qualité maintiennent **90% de l'apprentissage normal** contre **40%** avec des plans médiocres ou inexistants

**Meilleure pratique** : Avoir 3 à 5 jours complets de plans de remplacement prêts en permanence

## Classeur d'Urgence pour Remplaçant

**Classeur physique dans la classe** (le remplaçant peut le saisir immédiatement)

### Contenu du Classeur

**Section 1 : Informations sur la Classe**
```
- Plan de classe (étiqueté avec les prénoms)
- Emploi du temps quotidien (horaires pour chaque matière)
- Système de gestion du comportement (règles de classe, récompenses)
- Élèves à besoins particuliers (liste des aménagements)
- Procédures d'urgence (exercice d'incendie, confinement)
- Élèves aidants (élèves fiables que le remplaçant peut solliciter)
```

**Section 2 : Jour 1 Leçon Complète**
```
Matinée :
- 8h30-9h00 : Routine matinale + appel (le remplaçant fait indépendamment)
- 9h00-9h30 : Fiche de mathématiques (30 problèmes, corrigé inclus)
- 9h30-10h00 : Mots cachés (vocabulaire de l'unité en cours)

Mi-journée :
- 10h00-10h30 : Lecture (lecture autonome + fiche de compréhension)
- 10h30-11h00 : Récréation
- 11h00-11h30 : Coloriage (activité apaisante)

Après-midi :
- 11h30-12h00 : Déjeuner
- 12h00-12h30 : Mots cachés sciences (termes de l'unité en cours)
- 12h30-13h00 : Mots croisés (vocabulaire académique)
- 13h00-13h30 : Puzzle mathématique (pensée critique, engageant)
- 13h30-14h00 : Choix libre : Tracer des lignes, coloriage ou lecture autonome

Sortie : 14h00

Note au remplaçant : "Tous les corrigés sont dans la Section 5. Les élèves doivent vérifier leur propre travail une fois terminé."
```

**Sections 3-5 : Jours 2, 3, 4** (structure similaire, fiches différentes)

**Section 6 : Corrigés** (toutes les fiches)

## Fiches Auto-Explicatives

**Principe clé** : Le remplaçant n'a pas besoin d'enseigner du nouveau contenu

**Caractéristiques des fiches pour remplaçants** :

### Instructions Claires sur la Fiche

**Exemple d'en-tête de fiche de mathématiques** :
```
Prénom : _________________ Date : _______

CONSIGNES :
1. Résous chaque problème d'addition
2. Montre ton travail
3. Vérifie tes réponses avec le corrigé (sur le bureau de l'enseignant)
4. Si tu as 8 erreurs ou plus, demande de l'aide au remplaçant
5. Une fois terminé, dépose dans la corbeille de remise

Temps prévu : 30 minutes
```

**Adapté au remplaçant** : Instructions SUR la fiche (le remplaçant n'a pas besoin d'expliquer)

---

### Types d'Activités Familières

**Utiliser des fiches que les élèves connaissent déjà** :
```
✓ Calculs mathématiques (pratiqués chaque semaine - les élèves connaissent le format)
✓ Mots cachés (faits régulièrement - auto-explicatifs)
✓ Mots croisés (type de puzzle familier)
✓ Coloriage (tout le monde sait colorier)

✗ Nouvelles règles de jeu (le remplaçant ne peut pas enseigner une activité inconnue)
✗ Projets complexes (nécessitent un échafaudage de l'enseignant)
✗ Travail de groupe (défi de gestion du comportement pour le remplaçant)
```

## Gestion du Comportement avec Remplaçants

**Défi** : Les élèves testent les limites avec un remplaçant

**Solution** : Attentes claires + travail engageant

### Note Pré-Écrite aux Élèves

**À inclure dans le plan de remplacement** :
```
Chers élèves,

Je suis absent(e) aujourd'hui, mais j'attends de vous que vous montriez à notre remplaçant le même respect que vous me montrez. Voici ce que j'attends :

✓ Suivre toutes les règles de la classe
✓ Compléter tout le travail assigné
✓ Aider le remplaçant s'il a des questions
✓ Être gentil et respectueux

Je demanderai un rapport au remplaçant à mon retour. Les élèves qui respectent les attentes recevront [récompense]. Ceux qui ne le font pas auront [conséquence].

Passez une excellente journée !
[Nom de l'enseignant]
```

**Responsabilisation** : Les élèves savent que l'enseignant fera un suivi

---

### Travail Engageant (Prévient l'Ennui)

**Ennui = problèmes de comportement**

**Stratégie** : Mélanger facile + stimulant + amusant
```
Matinée : Mathématiques standard (familier, renforce la confiance)
Milieu de matinée : Mots cachés (ludique, engageant)
Avant le déjeuner : Coloriage (apaisant, peu stressant)
Après le déjeuner : Mots croisés (défi modéré)
Fin de journée : Puzzle mathématique OU choix libre (finition motivante)

Résultat : Élèves engagés (moins de temps pour les comportements inappropriés)
```

## Plans de Remplacement Alignés sur le Contenu

**Idéal** : Les plans de remplacement révisent le contenu actuel (pas du remplissage)

### Créer des Plans Alignés Mensuellement

**Septembre (début d'année)** :
```
Générer des plans de remplacement basés sur le contenu de septembre :
- Mathématiques : Révision addition/soustraction (compétences enseignées en septembre)
- Vocabulaire : Mots d'orthographe de septembre (dans mots cachés/croisés)
- Sciences : Termes scientifiques de septembre (dans mots cachés)

Résultat : Même avec un remplaçant, les élèves pratiquent un contenu pertinent
```

**Octobre** :
```
Mettre à jour les plans de remplacement :
- Mathématiques : Multiplication (focus d'octobre)
- Vocabulaire : Mots d'octobre
- Sciences : Termes de l'unité d'octobre

Remplacer les fiches de septembre par les versions d'octobre
Temps : 10 minutes pour générer 5 nouvelles fiches
```

**En continu** : Mettre à jour les plans de remplacement mensuellement (toujours avec le contenu actuel)

## Différenciation dans les Plans de Remplacement

**Défi** : Le remplaçant ne peut pas différencier comme l'enseignant habituel

**Solution** : Matériel pré-différencié

### Bacs de Fiches à Niveaux

**Configurer à l'avance** :
```
Bac 1 (Bleu) : Fiches modifiées
- 10 problèmes au lieu de 20
- Support visuel
- Pour : Élèves en difficulté

Bac 2 (Vert) : Fiches standard
- 20 problèmes
- Niveau de classe
- Pour : Élèves de niveau

Bac 3 (Rouge) : Fiches défi
- 30 problèmes
- Applications complexes
- Pour : Élèves avancés

Note au remplaçant : "Les élèves savent de quel bac de couleur prendre. Ils sélectionneront eux-mêmes le niveau approprié."

Résultat : La différenciation se fait automatiquement (pas de prise de décision nécessaire du remplaçant)
```

## Former les Élèves aux Jours de Remplacement

**Préparation** : Enseigner aux élèves comment se comporter avec les remplaçants

**Formation en septembre** :
```
Enseignant : "Parfois je serai absent(e) et nous aurons un remplaçant. Voici ce que j'attends :"

Jeu de rôle :
- Des élèves volontaires jouent le "remplaçant"
- La classe pratique :
  * Suivre les instructions du remplaçant
  * Prendre les fiches du bon bac
  * Demander de l'aide au remplaçant de manière appropriée
  * Ranger à la fin de la journée

Essai pratique : "Aujourd'hui, nous allons faire comme si c'était un jour de remplacement"
Élèves : Suivent le plan d'urgence comme pratique
Résultat : Quand le vrai remplaçant arrive, les élèves connaissent la routine
```
