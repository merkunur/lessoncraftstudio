## Johdanto: Pääset alkuun näppärästi

**Uusi tehtävägeneraattorien käyttäjä?** Tämä kattava opas vie sinut läpi koko prosessin.

**Mitä opit**:
```
1. Alkuasetukset (tili, omat valinnat)
2. Oikean generaattorin valinta (33 vaihtoehtoa)
3. Tehtävien räätälöinti (parametrit, vaikeustaso)
4. Luominen ja lataaminen (PDF-tiedosto)
5. Tehokas tulostaminen (asetukset, paperin säästö)
6. Materiaalien järjestäminen (kansiorakenne)
7. Edistyneet vinkit (massaluonti, pohjat)
8. Yleisimmät ongelmat ja ratkaisut
```

**Aika oppimiseen**: 30 minuuttia perusasioihin, jatkuva kehittyminen käytössä

**Pääperiaate**: Generaattorit ovat työkaluja – harjoittelu tekee mestarin

**Järjestely**:
```
Pääluokat (33 generaattoria yhteensä):

- Yhteen- ja vähennyslasku, kerto- ja jakolasku

Lukutaito (12 generaattoria):

- Kuvabingo, etsi ja löydä, yhdistäminen, värityskuvat
```

## Vaihe 4: Tehtävän räätälöinti

**Parametrien asettaminen**

### Sanahaun esimerkkiohje

**Muokkausvaihtoehdot**:

**1. Syötä sanastosanat**:
```
Syöttökenttä: Kirjoita tai liitä sanat

Esimerkki (luonnontieteet):
- yhteyttäminen
- klorofylli
- hiilidioksidi
- happi
- auringonvalo
- vesi
- energia
- glukoosi

Vinkki: Yksi sana per rivi tai pilkulla eroteltuina
```

---

**2. Valitse ruudukon koko**:
```
Vaihtoehdot:
- 10×10: Helppo (nuoremmat oppilaat, yksinkertaiset sanat)
- 15×15: Keskitaso (tavallinen vaikeustaso)
- 20×20: Vaikea (vanhemmat oppilaat, enemmän sanoja)

Suositus: Aloita 15×15:stä (säädä helpommaksi/vaikeammaksi tarpeen mukaan)
```

---

**3. Aseta vaikeustaso**:
```
Helppo: Sanat vain vaakasuoraan ja pystysuoraan (ei vinottaisia)
Keskitaso: Vaakasuora, pystysuora, vinottainen
Vaikea: Kaikki suunnat mukaan lukien takaperin

Valitse perustuen: Oppilaiden luokka-asteeseen ja kokemukseen sanahauista
```

---

**4. Lisäasetukset**:
```
☑ Sisällytä vastausavain (suositus)
☑ Sisällytä sanalista (löydettävät sanat näkyvissä)
☐ Piilota sanalista (haastavampaa)

Otsikko: "Yhteyttämisen sanasto"
Ohjeet: "Etsi kaikki 8 yhteyttämiseen liittyvää sanaa"
```

## Vaihe 5: Esikatselu ja hienosäätö

**Tarkista ennen viimeistelyä**

### Reaaliaikainen esikatselu

**Mitä tarkastella**:
```
Ulkoasun tarkistus:
- Mahtuvatko sanat ruudukkoon? ✓
- Onko otsikko selkeä? ✓
- Ovatko ohjeet luettavissa? ✓
- Onko yleisilme ammattimainen? ✓

Tarvitaanko säätöjä?
- Ruudukko liian täynnä: Kasvata kokoa (10×10 → 15×15)
- Liian helppo: Nosta vaikeustasoa (keskitaso → vaikea)
- Puuttuva sana: Lisää listaan, luo uudelleen

Iteratiivinen prosessi: Säädä → Esikatsele → Säädä → Esikatsele kunnes täydellinen

Aikaa: 30 sekunnista 2 minuuttiin (riippuen tarvittavista säädöistä)
```

## Vaihe 8: Tehokas tulostaminen

**Tulostusasetukset ovat tärkeitä**

### Optimaaliset tulostusasetukset

**Tulostimen asetukset**:
```
Laatu: Tavallinen (ei luonnos, ei valokuva)
- Luonnos: Liian vaalea (vaikea lukea)
- Tavallinen: Selkeä, käyttää sopivasti mustetta
- Valokuva: Tarpeeton (tuhlaa mustetta, hidas)

Väri: Mustavalkoinen (ellei erikseen tarvita)
- Useimmat tehtävät: Mustavalko riittää
- Värityskuvat: Väri vaaditaan
- Säästö: Mustavalko maksaa 80 % vähemmän per sivu

Kaksipuolinen tulostus: Tarvittaessa
- Harjoitustehtävät: Yksipuolinen (oppilaat kirjoittavat)
- Luetut tekstit: Kaksipuolinen OK (säästää paperia)
- Tärkeää: Tarkista kääntösuunta (oikea puoli ylös)

Paperi: Tavallinen kopiopaperi 80 g/m²
- Vältä kartonkia: Kallista, tarpeetonta useimmille tehtäville
- Vältä kierrätettyä: Saattaa jumiutua, voi olla vähemmän terävä
```

---

### Massatulostusvinkit

**30 oppilaan luokalle**:
```
Tulostustyö: 30 kopiota tehtävästä

Asetukset:
1. Täytä paperikasetti täyteen (200+ arkkia)
2. Valitse "Lajittele" (jos myöhemmin nitovat)
3. Valitse "Tulosta kaikki kerralla" (ei yksi kerrallaan)
4. Lähetä tulostustyö
5. Kävele pois (anna sen pyöriä)

Aika: 30 kopiota = n. 3 minuuttia tulostusaikaa
Vältä: Seisomista tulostimen vieressä (käytä aika muihin tehtäviin)
```

## Edistyneet tekniikat

**Nosta generaattorikäyttösi seuraavalle tasolle**

### Massaluonti

**Useiden tehtävien luominen kerralla**:
```
Skenaario: Tarvitaan 5 erilaista matematiikan tehtävää eriyttämiseen

Prosessi:
1. Luo Tehtävä A (helppo: 10 laskutehtävää)
2. Lataa ja tallenna
3. Säädä parametrejä (keskitaso: 15 laskutehtävää)
4. Luo Tehtävä B
5. Jatka C, D, E:lle

Aika: 5 tehtävää × 1 min = 5 minuuttia yhteensä
Tulos: Koko viikon eriyttävä harjoitusmateriaali valmiina
```

---

### Pohjien luominen

**Tallenna mukautetut asetukset**:
```
Jos luot usein samantyyppisiä tehtäviä:

Esimerkki: Viikoittainen sanaston sanahaku (aina 15×15, keskitaso)

Prosessi:
1. Aseta kaikki asetukset (ruudukon koko, vaikeustaso, muoto)
2. Tallenna pohjaksi (jos saatavilla paketissasi)
3. Ensi viikolla: Lataa pohja, syötä vain uudet sanat

Säästetty aika: 30 sekuntia per tehtävä (ei parametrien valintaa)
```

---

### Kausivaihtelut

**Sama sisältö, eri teema**:
```
Taito: Kaksinumeroinen yhteenlasku (20 tehtävää)

Syyskuu: Omena-teemainen otsikko ja grafiikat
Lokakuu: Kurpitsa-teemainen
Marraskuu: Kiitospäivä-teemainen

Sama matematiikka, raikas ulkoasu (pitää oppilaat kiinnostuneina läpi vuoden)

Tapa: Vaihda otsikko, säädä grafiikoiden teemaa (jos saatavilla)
Aika: Lisää 15 sekuntia per tehtävä (vähäinen vaiva, suuri vaikutus sitoutumiseen)
```

## Yleisimmät virheet ja niiden välttäminen

**Opi muiden erehdyksistä**

### Virhe 1: Liikaa parametreja

**Ongelma**:
```
Uusi käyttäjä: Säätää jokaista yksittäistä vaihtoehtoa (hämmentyy valinnoista)
Tulos: Analyysihalvaus (vie 10 minuuttia luoda yksi tehtävä)
```

**Ratkaisu**:
```
Aloita yksinkertaisesti: Käytä oletusasetuksia ensin
Luo: Katso tulos
Säädä: Muuta vain sitä mikä vaatii muutosta
Iteroi: Tutustu lisävaihtoehtoihin vähitellen

Ensimmäinen tehtävä: 2 minuuttia (oppiminen)
Viides tehtävä: 30 sekuntia (itsevarma)
```

---

### Virhe 2: Väärä vaikeustaso

**Ongelma**:
```
Opettaja: Luo "vaikean" tehtävän kamppaileville oppilaille
Tulos: Turhautuminen, luovuttaminen
```

**Ratkaisu**:
```
Aloita helpommasta: Jos epävarma, mene yksi taso alaspäin
Testaa: Anna ensin yhdelle oppilaalle
Säädä: Helpompi tai vaikeampi vastauksen perusteella
Hienosäädä: Saat vaikeustason juuri sopivaksi ajan kanssa
```

---

### Virhe 3: Tiedostojen järjestämättömyys

**Ongelma**:
```
Lataukset: Kaikki menevät "Lataukset"-oletuskansioon
Kuukausia myöhemmin: "Missä oli se yhteyttämisen sanahaku jonka tein?"
Tulos: Tehtävien turhaa uudelleenluontia
```

**Ratkaisu**:
```
Välittömästi: Siirrä ladattu tehtävä asianmukaiseen kansioon
Nimeäminen: Käytä kuvaavia nimiä (ei vain "tehtava1.pdf")
Uudelleenkäyttö: Löydä ja käytä uudelleen ensi vuonna (säästä aikaa)
```

## Yleisten ongelmien vianmääritys

**Pikaiset korjaukset**

### Ongelma: "Tehtävä ei luonnu"

**Mahdolliset syyt**:
```
1. Liikaa sanoja ruudukon koolle
   Korjaus: Vähennä sanojen määrää tai kasvata ruudukkoa

2. Internet-yhteys katkennut
   Korjaus: Tarkista WiFi, päivitä sivu

3. Virheelliset merkit syötteessä
   Korjaus: Poista erikoismerkit, käytä tavallisia kirjaimia
```

---

### Ongelma: "Tulostuslaatu on heikko"

**Korjaukset**:
```
1. Tarkista tulostimen asetukset (pitäisi olla "Tavallinen" ei "Luonnos")
2. Varmista riittävä mustetaso (vaihda kasetti jos vähissä)
3. Puhdista tulostuspäät (tulostimen huoltovalikko)
4. Käytä laadukasta paperia (halpa paperi voi näyttää sumuiselta)
```

---

### Ongelma: "En löydä ladattua tiedostoa"

**Ratkaisu**:
```
1. Tarkista oletus Lataukset-kansio
2. Etsi tietokoneelta tänään luotuja PDF-tiedostoja
3. Selain: Tarkista lataushistoria (näyttää tiedoston sijainnin)
4. Jatkossa: Tallenna välittömästi järjestettyyn kansioon
```

## Yhteenveto

Hallitse tehtävägeneraattorit **30 minuutissa** - alkuasetukset (5 min) + ensimmäinen tehtävä (5 min) + harjoittelu (20 min).

**Alkuasetukset**: Luo tili (2 min), aseta valinnat (luokka-aste, kieli, yksiköt), tutustu päävalikkoon (33 generaattoria luokittain järjestettynä)

**Luonti-ohje**: Valitse generaattori → muokkaa (syötä sisältö, ruudukon koko, vaikeustaso) → esikatsele (tarkista ulkoasu) → luo (2-5 sek) → lataa (PDF valmis)

**Tulosta tehokkaasti**: Tavallinen laatu, mustavalko ellei väriä tarvita, kaksipuolinen tarvittaessa, massatulostusasetukset (lajittele, tulosta kaikki kerralla)

**Vastausavaimet**: Automaattinen luonti (valintaruutu luonnin aikana), PDF:n sivu 2, ei ylimääräistä aikaa

**Edistynyt**: Massaluonti (5 tehtävää 5 minuutissa), pohjien luonti (tallenna mukautetut asetukset), kausivaihtelut (sama taito, eri teema)

**Vältä virheitä**: Aloita oletusasetuksista (säädä vähitellen), sovita vaikeustaso oppilaisiin (aloita helpommasta), järjestä tiedostot heti (älä menetä töitä)

**Vianmääritys**: Ruudukon koko vs. sanamäärä, tulostuslaadun asetukset, ladattujen tiedostojen etsiminen

**Tuki**: Ohjeet, tietopankki, sähköpostituki (24 h vastaus Ydin/Täysi-paketeille), video-läpikäynnit

**Hinnoittelu**: Ilmainen 0 € (vesileima), Ydin 144 €/vuosi (opettajille), Täysi 240 €/vuosi (kaupallinen)
