## Introduction : Les limites des évaluations traditionnelles

**L'évaluation traditionnelle** :
```
Format : QCM + questions à trous
Ce qu'elle mesure : Reconnaissance et rappel
Ce qu'elle omet : Application, créativité, résolution de problèmes, communication

Exemple :
Question : "Combien font 8 × 7 ?"
Élève : Écrit "56"
Preuve : L'élève connaît ce fait

MAIS : Le test ne montre pas si l'élève COMPREND ce que signifie la multiplication
ni s'il PEUT l'appliquer dans des situations réelles
```

**Recherche** (Wiggins, 1990) : L'évaluation authentique prédit mieux la **performance en contexte réel** que les tests traditionnels

**L'évaluation alternative** :
```
Format : Tâche de performance, projet, démonstration
Ce qu'elle mesure : Application, synthèse, résolution de problèmes
Ce qu'elle révèle : Compréhension profonde + compétences

Exemple :
Tâche : "Conçois un jardin de 8 mètres × 7 mètres. Calcule la surface. Dessine à l'échelle.
Planifie ce que tu vas planter."
Élève : Applique la multiplication, utilise le raisonnement spatial, prend des décisions,
communique visuellement

Preuve : L'élève COMPREND la multiplication comme calcul de surface ET peut l'appliquer
dans un contexte authentique
```

**Principe fondamental** : L'évaluation devrait refléter comment on utilise les connaissances dans la vie réelle

## Tâches de performance

**Les élèves démontrent leur apprentissage par l'ACTION**

### Tâche de performance en mathématiques

**Compétence visée** : Mesurer l'aire et le périmètre (CM1)

**Test traditionnel** :
```
Question 1 : Un rectangle mesure 6 cm × 4 cm. Quelle est l'aire ?
Question 2 : Quel est le périmètre ?

Mesure : L'élève peut-il calculer avec des formules
Manque : Compréhension de ce que signifient aire/périmètre dans la vie réelle
```

**Alternative par tâche de performance** :
```
Tâche : "Aménage la chambre de tes rêves"

Consignes :
1. Choisis les dimensions (la pièce doit mesurer entre 3 et 4,5 mètres de côté)
2. Calcule l'aire (montre ton travail)
3. Calcule le périmètre (montre ton travail)
4. Dessine la pièce à l'échelle (1 cm = 50 cm)
5. Ajoute les meubles avec leurs dimensions (lit, bureau, commode)
6. Explique : "Est-ce que les meubles rentrent ? Combien d'espace au sol reste-t-il ?"

Grille d'évaluation :
- Calcul d'aire exact (2 points)
- Calcul de périmètre exact (2 points)
- Dessin à l'échelle correct (3 points)
- Dimensions de meubles réalistes (2 points)
- Explication du raisonnement spatial (3 points)
Total : 12 points

Ce que cela révèle :
- Peut appliquer les formules correctement (calcul)
- Comprend le raisonnement spatial (placement des meubles)
- Peut communiquer sa pensée mathématique (explication)
- Fait des liens avec le monde réel (application pratique)
```

**Composant fiche de travail** : Modèle de planification (généré en 42 secondes)
```
Fiche de planification de chambre

Dimensions de ma chambre : ____ mètres × ____ mètres

Calcul de l'aire :
____ × ____ = ____ mètres carrés

Calcul du périmètre :
(____ + ____) × 2 = ____ mètres

Liste des meubles :
Article : _______ Taille : ____ × ____
Article : _______ Taille : ____ × ____
Article : _______ Taille : ____ × ____

Est-ce que tout rentre ? Explique :
_________________________________________________
```

### Tâche de performance en sciences

**Compétence visée** : Comprendre les besoins des plantes (CE2)

**Test traditionnel** :
```
Choix multiple :
"De quoi les plantes ont-elles besoin pour grandir ?"
a) Eau et terre
b) Eau et lumière du soleil
c) Eau, lumière du soleil et nutriments
d) Seulement de l'eau

Mesure : L'élève peut-il se rappeler le fait
Manque : Compréhension de POURQUOI les plantes ont besoin de ces éléments
```

**Alternative par tâche de performance** :
```
Tâche : "Expérience de croissance des plantes"

Semaine 1 : Mise en place de l'expérience
- Plante 3 graines de haricot dans des gobelets
- Gobelet A : Eau + lumière du soleil + terre
- Gobelet B : Eau + obscurité + terre
- Gobelet C : Pas d'eau + lumière du soleil + terre

Semaines 2-4 : Observer et noter (fiche de mesures)
Date | Hauteur gobelet A | Hauteur gobelet B | Hauteur gobelet C | Observations
...

Semaine 4 : Fiche d'analyse
1. Quelle plante a poussé le plus ? Pourquoi ?
2. Qu'est-il arrivé au gobelet B (sans lumière) ? Pourquoi ?
3. Qu'est-il arrivé au gobelet C (sans eau) ? Pourquoi ?
4. Tire des conclusions : De quoi les plantes ont-elles besoin ? Prouve-le avec tes données.

Notation :
- Mesures précises enregistrées (3 points)
- Observations détaillées (2 points)
- Conclusions appuyées par les données (4 points)
- Raisonnement scientifique clair (3 points)
Total : 12 points

Ce que cela révèle :
- Peut mener une expérience (démarche scientifique)
- Peut collecter des données précisément (compétences de mesure)
- Peut analyser les résultats (pensée critique)
- Comprend cause-effet (connaissance conceptuelle)
```

## Évaluation par portfolio

**Collecter des échantillons de travail au fil du temps pour montrer la progression**

### Portfolio de lecture

**Contenu** (septembre - mai) :
```
Section 1 : Carnets de lecture (suivre les livres lus)
- Septembre : 2 livres
- Janvier : 5 livres
- Mai : 8 livres
Progression visible : Volume augmenté

Section 2 : Échantillons de compréhension
- Septembre : Résumé simple d'un livre illustré
- Janvier : Analyse de personnage d'un roman
- Mai : Essai comparatif sur deux romans
Progression visible : Complexité augmentée

Section 3 : Vocabulaire
- Septembre : 20 mots-outils maîtrisés
- Janvier : 50 mots-outils + 30 mots de contenu
- Mai : Plus de 100 mots, peut les utiliser en contexte
Progression visible : Vocabulaire élargi

Section 4 : Enregistrements de fluidité
- Septembre : 40 mots/min, haché
- Janvier : 65 mots/min, plus fluide
- Mai : 90 mots/min, expressif
Progression visible : Fluidité améliorée

Réflexion de fin d'année (l'élève complète la fiche) :
"En comparant mon travail de septembre et de mai, j'ai progressé en..."
"Je suis le plus fier de..."
"L'année prochaine, je veux améliorer..."
```

**Valeur d'évaluation** : Le portfolio montre la PROGRESSION (pas seulement le niveau actuel)

Consultation des parents : Les parents voient des preuves tangibles de la progression annuelle
(pas seulement une note sur le bulletin)

## Évaluation par projet

**Des projets de plusieurs semaines démontrent une compréhension approfondie**

### Projet d'études sociales

**Unité** : La communauté locale (CE1)

**Évaluation traditionnelle** :
```
Test : 20 questions sur les métiers de la communauté, compétences cartographiques, citoyenneté
Format : Textes à trous, associations
Durée : 30 minutes
```

**Alternative par projet** :
```
Tâche : "Créer un guide de la communauté"

Semaine 1 : Recherche
- Interviewer 2 personnes travaillant dans la communauté (pompier, bibliothécaire, etc.)
- Fiche : Modèle de questions d'interview
- Documenter : Que font-ils ? Pourquoi leur travail est-il important ?

Semaine 2 : Création de carte
- Dessiner une carte de la communauté locale (points de repère principaux)
- Étiqueter : école, bibliothèque, caserne de pompiers, commissariat, parc
- Fiche : Grille de planification de carte (échelle fournie)

Semaine 3 : Rédaction
- Écrire un livret "À propos de notre communauté"
- Sections : Histoire, Personnes importantes, Lieux à visiter, Comment être un bon citoyen
- Fiche : Organisateur d'écriture pour chaque section

Semaine 4 : Présentation
- Partager le Guide de la communauté avec la classe
- Expliquer : Qu'est-ce qui rend notre communauté spéciale ?

Grille d'évaluation :
- Recherche complète et précise (4 points)
- Carte claire et étiquetée (4 points)
- Rédaction informative et organisée (5 points)
- Présentation claire et captivante (3 points)
Total : 16 points

Ce que cela évalue :
- Contenu d'études sociales (connaissance de la communauté)
- Compétences de recherche (interviews, sources)
- Géographie (création de carte)
- Rédaction (texte informatif)
- Communication (présentation)

Investissement en temps : 4 semaines
Profondeur : Beaucoup plus profond qu'un test de 30 minutes
```

## Tickets de sortie (évaluation formative rapide)

**Vérifications de compréhension de 3 à 5 minutes**

### Exemples de tickets de sortie quotidiens

**Après une leçon de mathématiques sur les fractions** :
```
Ticket de sortie (fiche - prend 42 sec à générer) :

Nom : _____________ Date : _______

Aujourd'hui nous avons appris à comparer les fractions.

1. Laquelle est plus grande : 1/2 ou 1/4 ? Explique comment tu le sais.
   _________________________________________________

2. Fais un dessin pour montrer pourquoi 3/4 est plus grand que 1/2.
   [Espace pour le dessin]

3. Une chose que je ne comprends toujours pas :
   _________________________________________________

Utilisation enseignant : Lire les tickets, grouper les élèves selon la compréhension
- Compréhension claire → Pratique autonome demain
- Confusion → Réenseignement en petit groupe demain
```

---

### Ticket de sortie en sciences

**Après une leçon sur les états de la matière** :
```
Ticket de sortie :

Sujet d'aujourd'hui : États de la matière

1. Donne un exemple de :
   Solide : _______  Liquide : _______  Gaz : _______

2. Que se passe-t-il quand tu chauffes de la glace ?
   _________________________________________________

3. Dessine : Montre l'eau comme solide, liquide et gaz
   [Trois cases pour les dessins]

4. Évalue ta compréhension : ☐ J'ai compris  ☐ J'ai besoin d'aide  ☐ Je suis confus

Utilisation enseignant : Trier par auto-évaluation
- "J'ai compris" → Activité d'enrichissement demain
- "J'ai besoin d'aide" → Plus d'exemples et de pratique
- "Je suis confus" → Réenseigner avec des visuels
```

## Démonstrations authentiques

**Application de compétences en contexte réel**

### Lecture : Recommandation de livre

**Tâche d'évaluation** :
```
Au lieu de : Fiche de lecture traditionnelle (résumer l'intrigue)

Essayer : Présentation de recommandation de livre

Tâche : "Convaincs tes camarades de lire ton livre"

Exigences :
1. Résumé captivant (ne dévoile pas la fin !)
2. Pourquoi tu l'as aimé (connexion personnelle)
3. Qui l'apprécierait (identification du public)
4. Lire un passage préféré à voix haute (démonstration de fluidité)
5. Créer un visuel (affiche, dessin, diorama)

Fiche : Guide de planification
- Titre du livre/auteur : ______________
- Genre : ______________
- Personnages principaux : ______________
- Raison 1 de le lire : ______________
- Raison 2 de le lire : ______________
- Raison 3 de le lire : ______________
- Passage à lire à voix haute (page ___) : Marquer avec un post-it

Évaluation :
- Compréhension démontrée ? (a compris l'histoire)
- Communication claire ? (compétences de présentation)
- Fluidité appropriée ? (lecture orale)
- Persuasion efficace ? (arguments convaincants)

Authentique : C'est ainsi que les gens partagent vraiment des livres (pas une fiche artificielle)
```

## Grilles d'évaluation pour l'évaluation alternative

**Des critères clairs assurent une notation équitable**

### Modèle de grille d'évaluation de projet

**Projet de foire scientifique** (généré comme fiche) :
```
Grille d'évaluation : Projet de foire scientifique

Question de recherche (4 points)
4 = Question claire et testable
3 = Question présente mais pourrait être plus claire
2 = Question trop large ou non testable
1 = Pas de question claire

Procédure (4 points)
4 = Étapes détaillées et reproductibles
3 = Étapes présentes, quelques détails manquants
2 = Procédure peu claire
1 = Aucune procédure documentée

Collecte de données (4 points)
4 = Données précises, tableau/graphique organisé
3 = Données présentes, organisation à améliorer
2 = Données incomplètes
1 = Aucune donnée collectée

Conclusion (4 points)
4 = Conclusion claire appuyée par les données
3 = Conclusion présente, lien faible avec les données
2 = Conclusion ne correspond pas aux données
1 = Pas de conclusion

Présentation (4 points)
4 = Claire, captivante, bien organisée
3 = Compréhensible, organisation adéquate
2 = Difficile à suivre
1 = Confuse ou incomplète

Total : _____ / 20 points

Bénéfice pour l'élève : Sait exactement ce qui est attendu avant de commencer le projet
```
