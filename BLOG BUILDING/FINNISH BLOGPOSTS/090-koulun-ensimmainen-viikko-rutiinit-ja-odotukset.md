## Johdanto: Ratkaiseva Ensimmäinen Viikko

**Tutkimus** (Wong & Wong, 2009): Ensimmäiset kaksi viikkoa määrittävät **koko lukuvuoden** menestyksen

**Mitä ensimmäisellä viikolla tapahtuu**:
```
Vahva alku:
- Selkeät toimintatavat opetetaan
- Odotukset asetetaan
- Rutiinit harjoitellaan
- Yhteisö rakennetaan
Tulos: Sujuva lukuvuosi (95% vähemmän käytösongelmia)

Heikko alku:
- Toimintatapoja ei opeteta
- Epäselvät odotukset
- Kaaos sallitaan
Tulos: Käytösongelmien kanssa painiminen koko vuoden
```

**Perusperiaate**: "Ensivaikutelmaan ei tule toista mahdollisuutta"

## Päivä 3: Lähtötason Arviointi

**Tavoite**: Arvioida mistä oppilaat aloittavat

### Aamu: Lukemisen Arviointi

**Aktiviteetti**: Lukusujuvuuden tarkistus + ymmärtäminen
```
Lue tämä teksti hiljaa (opettaja kellottaa):

[Luokka-tason teksti, 100 sanaa]

Aika: _____ sekuntia = _____ sanaa/min

Vastaa nyt näihin kysymyksiin:
1. Mistä teksti kertoi? (pääajatus)
2. Miksi [hahmo] [teko]? (päättely)
3. Mitä [sana] tarkoittaa? (sanasto)

Opettaja: Kirjaa tiedot (lähtötaso edistymisen seurantaa varten)
```

## Päivä 5: Harjoittelu ja Kertaus

**Tavoite**: Vakiinnuta ma-to opitut rutiinit

### Aamu: Koko Päivän Simulaatio

**Aktiviteetti**: Harjoittele kokonainen päivä ikään kuin olisi viikko 2
```
Aamu:
- Täytä aamutehtävä (matematiikan lämmittely)
- Ota esiin kotitehtävät (ei vielä ole, mutta harjoittele liikettä)
- Palauta koriin

Keskipäivä:
- Siirtyminen lukemiseen (harjoittele siivousrutiinia)
- Tee lukunautin moniste itsenäisesti
- Palauta kun valmis

Iltapäivä:
- Siirtyminen matematiikkaan (harjoittele uudestaan)
- Täytä matematiikan moniste
- Tarkista vastausavaimella (harjoittele toimintatapaa)

Päivän loppu:
- Pakkaa reppu (harjoittele)
- Siivoa työtila (harjoittele)
- Asetu jonoon kotiinlähtöä varten (harjoittele)

Palaute: "Tänään teimme jokaisen rutiinin täydellisesti!"
```
