## Passo 4: Personalizzare la Scheda Didattica

**Impostazione parametri**

### Esempio Dettagliato: Cerca Parole

**Opzioni di personalizzazione**:

**1. Inserisci vocaboli**:
```
Campo testo: Digita o incolla parole

Esempio (unità fotosintesi):
- fotosintesi
- clorofilla
- anidride carbonica
- ossigeno
- luce solare
- acqua
- energia
- glucosio

Suggerimento: Una parola per riga oppure separate da virgola
```

---

**2. Scegli dimensione griglia**:
```
Opzioni:
- 10×10: Facile (scuola primaria, parole semplici)
- 15×15: Medio (difficoltà standard)
- 20×20: Difficile (scuola secondaria, più parole)

Raccomandazione: Inizia con 15×15 (regola se troppo facile/difficile)
```

---

**3. Imposta livello difficoltà**:
```
Facile: Solo orizzontale e verticale (no diagonali)
Medio: Orizzontale, verticale, diagonale
Difficile: Tutte direzioni incluso al contrario

Scegli in base a: Età alunni ed esperienza con cerca parole
```

---

**4. Opzioni aggiuntive**:
```
☑ Includi soluzioni (consigliato)
☑ Includi elenco parole (lista parole da trovare)
☐ Nascondi elenco (più impegnativo)

Titolo: "Vocabolario Fotosintesi"
Istruzioni: "Trova tutte le 8 parole relative alla fotosintesi"
```

## Passo 8: Stampare in Modo Efficiente

**Le impostazioni di stampa fanno la differenza**

### Impostazioni Stampa Ottimali

**Preferenze stampante**:
```
Qualità: Standard (non bozza, non foto)
- Bozza: Troppo chiaro (difficile da leggere)
- Standard: Nitido, usa inchiostro appropriato
- Foto: Non necessario (spreca inchiostro, lento)

Colore: Bianco e nero (salvo specifiche necessità)
- Maggior parte schede: B/N sufficiente
- Pagine da colorare: Colore necessario
- Risparmio: B/N costa 80% in meno per pagina

Fronte-retro: Quando appropriato
- Schede esercizi: Solo fronte (studenti scrivono)
- Letture: Fronte-retro OK (risparmia carta)
- Importante: Controlla orientamento capovolgimento

Carta: Carta comune 80 g/m²
- Evita cartoncino: Costoso, non necessario per schede normali
- Evita riciclata bassa qualità: Può incepparsi
```

---

## Errori Comuni da Evitare

**Impara dagli errori degli altri**

### Errore 1: Troppi Parametri

**Problema**:
```
Utente nuovo: Regola ogni singola opzione (sopraffatto dalle scelte)
Risultato: Paralisi da analisi (10 minuti per generare una scheda)
```

**Soluzione**:
```
Inizia semplice: Usa impostazioni predefinite prima
Genera: Vedi risultato
Regola: Cambia solo ciò che serve cambiare
Itera: Esplora gradualmente più opzioni

Prima scheda: 2 minuti (apprendimento)
Quinta scheda: 30 secondi (sicurezza)
```

---

### Errore 2: Livello Difficoltà Sbagliato

**Problema**:
```
Insegnante: Crea scheda "difficile" per studenti in difficoltà
Risultato: Frustrazione, abbandono
```

**Soluzione**:
```
Inizia più facile: Se incerto, vai un livello sotto
Testa: Dai prima a un solo studente
Regola: Più facile o difficile in base risposta
Affina: Trova difficoltà giusta col tempo
```

---

### Errore 3: File Non Organizzati

**Problema**:
```
Download: Tutti vanno in cartella "Download" predefinita
Mesi dopo: "Dov'è quel cerca parole fotosintesi che ho fatto?"
Risultato: Ricreare schede inutilmente
```

**Soluzione**:
```
Immediatamente: Sposta scheda scaricata in cartella appropriata
Nomenclatura: Usa nomi descrittivi (non solo "scheda1.pdf")
Riutilizzo: Trova e riutilizza anno prossimo (risparmia tempo)
```

## Risoluzione Problemi Comuni

**Soluzioni rapide**

### Problema: "La scheda non si genera"

**Possibili cause**:
```
1. Troppe parole per dimensione griglia
   Soluzione: Riduci numero parole o aumenta dimensione griglia

2. Connessione internet persa
   Soluzione: Controlla WiFi, ricarica pagina

3. Caratteri non validi nell'input
   Soluzione: Rimuovi simboli speciali, usa lettere standard
```

---

### Problema: "Qualità stampa scadente"

**Soluzioni**:
```
1. Controlla impostazioni stampante (dovrebbe essere "Standard" non "Bozza")
2. Verifica livelli inchiostro (sostituisci cartuccia se basso)
3. Pulisci testine stampa (menu manutenzione stampante)
4. Usa carta di qualità (carta economica può risultare sfocata)
```

---

### Problema: "Non trovo file scaricato"

**Soluzione**:
```
1. Controlla cartella Download predefinita
2. Cerca nel computer PDF creati oggi
3. Browser: Controlla cronologia download (mostra posizione file)
4. In futuro: Salva immediatamente in cartella organizzata
```
