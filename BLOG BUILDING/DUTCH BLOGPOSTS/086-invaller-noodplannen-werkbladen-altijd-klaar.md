## Inleiding: De Invaller Crisis

**De realiteit voor leerkrachten**: Je wordt 's nachts ziek en moet morgenochtend een invaller regelen.

**Het probleem**: Lesplannen maken om 6 uur 's ochtends terwijl je je beroerd voelt
```
6:00 uur: Wakker worden met griep
6:15 uur: School bellen voor invaller
6:30 uur: Proberen lesplannen te schrijven (je voelt je verschrikkelijk)
7:00 uur: Opgeven, vage notitie achterlaten "laat ze maar lezen"
Resultaat: Verloren lesdag (invaller weet niet wat te doen)
```

**De oplossing**: Vooraf gemaakte noodplannen (klaargemaakt wanneer je gezond bent)

**Onderzoek** (Weems & Rogers, 2007): Goede invallerplannen behouden **90% van het reguliere leren** versus **40%** met slechte of geen plannen

**Beste praktijk**: Heb altijd 3-5 dagen complete invallerplannen klaarliggen

## De Invaller Noodmap

**Fysieke map in de klas** (invaller kan deze meteen pakken)

### Inhoud van de Map

**Sectie 1: Klasinformatie**
```
- Plattegrond met namen (wie zit waar)
- Dagschema (tijden voor elk vak)
- Gedragsregels (klassensysteem, beloningen)
- Leerlingen met speciale behoeften (aanpassingen)
- Noodprocedures (brandalarm, lockdown)
- Behulpzame leerlingen (betrouwbare kinderen waar invaller kan vragen)
```

**Sectie 2: Dag 1 Complete Les**
```
Ochtend:
- 8:30-9:00: Ochtendroutine + aanwezigheid (invaller doet zelfstandig)
- 9:00-9:30: Rekenwerkblad (30 sommen, antwoordenblad bijgevoegd)
- 9:30-10:00: Woordzoeker (woordenschat huidige thema)

Middag:
- 10:00-10:30: Begrijpend lezen (zelfstandig lezen + begripswerkblad)
- 10:30-11:00: Pauze
- 11:00-11:30: Kleurplaat (rustgevende activiteit)

Namiddag:
- 11:30-12:00: Lunch
- 12:00-12:30: Natuur woordzoeker (termen uit huidige les)
- 12:30-13:00: Kruiswoordpuzzel (vakwoordenschat)
- 13:00-13:30: Rekenpuzzel (kritisch denken, boeiend)
- 13:30-14:00: Vrije keuze: Lijnen trekken, kleuren, of zelfstandig lezen

Einde: 14:00 uur

Opmerking voor invaller: "Alle werkbladen hebben antwoordenbladen in Sectie 5. Leerlingen controleren eigen werk na afloop."
```

**Sectie 3-5: Dagen 2, 3, 4** (vergelijkbare structuur, andere werkbladen)

**Sectie 6: Antwoordenbladen** (alle werkbladen)

## Werkbladen die Zichzelf Uitleggen

**Kernprincipe**: Invaller hoeft geen nieuwe stof te onderwijzen

**Kenmerken van invaller-vriendelijke werkbladen**:

### Duidelijke Instructies op het Werkblad

**Voorbeeld rekenblad kop**:
```
Naam: _________________ Datum: _______

INSTRUCTIES:
1. Los elke optelsom op
2. Laat je berekening zien
3. Controleer je antwoorden met het antwoordenblad (op bureau juf/meester)
4. Als je 8 of meer fout hebt, vraag hulp aan de invaller
5. Klaar? Leg in de inlevermand

Verwachte tijd: 30 minuten
```

**Invaller-vriendelijk**: Instructies STAAN op werkblad (invaller hoeft niets uit te leggen)

---

### Bekende Activiteitstypen

**Gebruik werkbladen die leerlingen al kennen**:
```
✓ Rekensommen (wekelijks geoefend - leerlingen kennen formaat)
✓ Woordzoekers (regelmatig gedaan - wijst zichzelf uit)
✓ Kruiswoordpuzzels (bekend puzzeltype)
✓ Kleuren (iedereen weet hoe te kleuren)

✗ Nieuwe spelregels (invaller kan onbekende activiteit niet uitleggen)
✗ Complexe projecten (vereisen begeleiding van vaste leerkracht)
✗ Groepswerk (gedragsmanagement uitdaging voor invaller)
```

## Gedragsmanagement voor Invallers

**Uitdaging**: Leerlingen testen grenzen bij invallers

**Oplossing**: Duidelijke verwachtingen + boeiend werk

### Vooraf Geschreven Briefje aan Leerlingen

**Voeg toe aan invallerplan**:
```
Beste klas,

Ik ben vandaag niet op school, maar ik verwacht dat jullie de invaller hetzelfde respect tonen als jullie mij tonen. Dit verwacht ik:

✓ Volg alle klasregels
✓ Maak al het werk af
✓ Help de invaller als er vragen zijn
✓ Wees vriendelijk en respectvol

Ik zal de invaller om een verslag vragen wanneer ik terugkom. Leerlingen die zich aan de verwachtingen houden krijgen [beloning]. Leerlingen die dat niet doen krijgen [consequentie].

Fijne dag!
[Naam leerkracht]
```

**Verantwoordelijkheid**: Leerlingen weten dat leerkracht erop terugkomt

---

### Boeiend Werk (Voorkomt Verveling)

**Verveling = gedragsproblemen**

**Strategie**: Mix van makkelijk + uitdagend + leuk
```
Ochtend: Standaard rekenen (bekend, bouwt vertrouwen)
Voor de lunch: Woordzoeker (spelachtig, boeiend)
Voor de middag: Kleuren (kalmerend, weinig stress)
Na de middag: Kruiswoordpuzzel (gematigde uitdaging)
Einde van de dag: Rekenpuzzel OF vrije keuze (motiverend einde)

Resultaat: Leerlingen betrokken (minder tijd voor wangedrag)
```

## Inhoudelijk Aansluitende Invallerplannen

**Ideaal**: Invallerplannen herhalen huidige lesstof (geen tijdverspilling)

### Maandelijks Aangepaste Plannen

**September (begin schooljaar)**:
```
Genereer invallerplannen gebaseerd op september-inhoud:
- Rekenen: Optellen/aftrekken herhaling (vaardigheden september)
- Woordenschat: September spelling (in woordzoeker/kruiswoordpuzzel)
- Natuur: September thema termen (in woordzoeker)

Resultaat: Ook met invaller oefenen leerlingen relevante lesstof
```

**Oktober**:
```
Update invallerplannen:
- Rekenen: Vermenigvuldigen (oktober focus)
- Woordenschat: Oktober woorden
- Natuur: Oktober thema termen

Vervang september werkbladen door oktober versies
Tijd: 10 minuten om 5 nieuwe werkbladen te maken
```

**Doorlopend**: Update maandelijks invallerplannen (altijd actuele lesstof)

## Differentiatie in Invallerplannen

**Uitdaging**: Invaller kan niet differentiëren zoals vaste leerkracht

**Oplossing**: Vooraf gedifferentieerde materialen

### Gelaagde Werkblad Bakken

**Van tevoren klaarzetten**:
```
Bak 1 (Blauw): Aangepaste werkbladen
- 10 sommen ipv 20
- Afbeeldingen ter ondersteuning
- Voor: Zwakke leerlingen

Bak 2 (Groen): Standaard werkbladen
- 20 sommen
- Groep-niveau
- Voor: Gemiddelde leerlingen

Bak 3 (Rood): Uitdagende werkbladen
- 30 sommen
- Complexe toepassingen
- Voor: Sterke leerlingen

Opmerking voor invaller: "Leerlingen weten welke kleur bak ze moeten pakken. Ze kiezen zelf het juiste niveau."

Resultaat: Differentiatie gebeurt automatisch (invaller hoeft geen beslissingen te nemen)
```

## Train Leerlingen voor Invallerdagen

**Voorbereiding**: Leer leerlingen hoe te gedragen bij invallers

**September training**:
```
Leerkracht: "Soms ben ik afwezig en hebben we een invaller. Dit verwacht ik:"

Rollenspel:
- Leerling vrijwilligers als "invaller"
- Klas oefent:
  * Instructies volgen van invaller
  * Werkbladen halen uit juiste bak
  * Hulp vragen op gepaste manier
  * Opruimen aan einde van dag

Oefenronde: "Vandaag doen we alsof het een invallerdag is"
Leerlingen: Volgen het invaller noodplan als oefening
Resultaat: Wanneer echte invaller komt, kennen leerlingen de routine
```
