# Prepositions App - Features Analysis

## Source File
`REFERENCE APPS/prepositions.html`

## App Overview
The Prepositions Worksheet Generator creates educational worksheets focused on teaching spatial relationships and prepositions through visual exercises. It supports 11 languages and offers two exercise modes: fill-in-the-blank and multiple choice.

## Core Features Found

### 1. Language Support (Line 579-593)
- 11 UI and content languages:
  - English (en)
  - German (de)
  - French (fr)
  - Spanish (es)
  - Portuguese (pt)
  - Italian (it)
  - Dutch (nl)
  - Swedish (sv)
  - Danish (da)
  - Norwegian (no)
  - Finnish (fi)
- Language affects both UI and worksheet content
- Image filenames used to generate preposition sentences in selected language

### 2. Page Setup (Lines 599-616)
**Paper Sizes:**
- Letter Portrait (8.5×11")
- Letter Landscape (11×8.5")
- A4 Portrait (210×297mm)
- A4 Landscape (297×210mm)
- Square (1200×1200)
- Custom dimensions

**Template System:**
- Selectable worksheet templates
- Page color customization
- Background themes with opacity control
- Border themes with opacity control

### 3. Exercise Configuration (Lines 665-683)
**Exercise Modes:**
- Fill-in-the-Blank mode
- Multiple Choice mode (with custom description option)

**Settings:**
- Number of exercises: 1-8 exercises per worksheet
- Select specific prepositions to practice
- Include/Exclude Name and Date fields

### 4. Item Selection (Lines 687-703)
**Generation Modes:**
- Manual Selection - Choose specific images from theme
- All Themes - Random selection from all available themes

**Image Theme Selection:**
- Browse dictionary of themed images
- Manual selection from theme gallery
- Visual selection count display (Selected: 0/8)

### 5. Shape Replacement (Lines 705-721)
**Shape Selection:**
- Manual Shape Selection mode

- Shape dictionary with image preview
- Selection count tracking

- Text content input field
- Add text to worksheet button

- Common formats: JPEG, PNG, GIF

7. **Manual or Random Selection** - Two generation modes for flexibility
