## KEY FEATURES (From matching.html analysis)

### 1. MATCHING MODES (4 different types)
- **Image ↔ Beginning Letter** (Immagine ↔ Lettera iniziale)
- **Image+Word ↔ Image+Word** (Immagine+Parola ↔ Immagine+Parola)
- **Image/Word ↔ Image/Word** (Immagine/Parola ↔ Immagine/Parola)
- **Image ↔ Custom Word** (Immagine ↔ Parola personalizzata)

### 2. IMAGE SELECTION METHODS
- Random Theme & Images
- Random from Chosen Theme
- Select Specific Images from library

### 3. PAGE SETUP OPTIONS
- **Page sizes:** Letter Portrait, Letter Landscape, A4 Portrait, A4 Landscape, Square (1200x1200), Custom
- **Background:** Fallback color, background themes, opacity control
- **Border:** Border themes, opacity control

### 4. WORKSHEET CONFIGURATION
- Include Name/Date fields (optional)
- Include item numbers (default: on)
- Show bullets/dots (default: on)
- Max number of pairs: adjustable

### 5. TEXT TOOLS
- Add custom text elements
- Font selection (multiple fonts)
- Color picker for text
- Font size adjustment
- Outline color and width (0-10)

### 6. IMAGE LIBRARY
- Theme-based browsing
- Search functionality
- Filter by theme
- 3000+ child-friendly images
- 11 language support for image filenames

### 7. CUSTOM IMAGE UPLOAD
- Multi-file upload
- Common formats (JPEG, PNG, GIF)
- Combine with library images

### 8. FULL CANVAS EDITABILITY
- Drag, rotate, scale any element
- Bring to front/forward
- Send backward/to back
- Alignment tools (align selected, align to page)
- Contextual toolbar for quick edits
- Undo/redo functionality
- Lock/unlock elements

### 9. EXPORT OPTIONS
- **Worksheet generation** (creates the matching exercise)
- **Answer Key generation** (shows correct matches)
- **JPEG export** (for both worksheet and answer key)
- **PDF export** (for both worksheet and answer key)
- **Grayscale option** (save ink when printing)

### 10. 11 LANGUAGES
**UI Languages:** English, German, French, Spanish, Italian, Portuguese, Dutch, Danish, Swedish, Norwegian, Finnish
**Image Library Languages:** Same 11 languages (important for matching exercises)
