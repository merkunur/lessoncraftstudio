## Johdanto: Oppilaiden itsenäisyyden rakentaminen

**Opettajan haaste**: "En voi auttaa kaikkia samanaikaisesti!"

**Todellisuus 30 oppilaan luokkahuoneessa**:
```
Opettaja työskentelee pienryhmän kanssa (5 oppilasta, 20 minuuttia)
Loput 25 oppilasta: Tarvitsevat tuottavaa itsenäistä työskentelyä

Ongelma: Oppilaat keskeyttävät
- "En ymmärrä ohjetta"
- "Olen valmis, mitä nyt?"
- "Voitko tarkistaa työni?"
- Tulos: Pienryhmäopetus keskeytyy jatkuvasti
```

**Ratkaisu**: Opeta oppilaat työskentelemään itsenäisesti
```
Toimivat järjestelmät:
- Selkeät odotukset (oppilaat tietävät mitä tehdä)
- Itsestään selvät materiaalit (voi suorittaa ilman opettajaa)
- Itsetarkistusmahdollisuudet (ei tarvitse odottaa opettajan arviointia)
- Valintataulut (kun valmis, seuraavat askeleet selvät)

Tulos: Opettaja voi keskittyä pienryhmään (25 oppilasta tuottavasti mukana)
```

**Tutkimus** (Zimmerman, 2002): Itsesäädelty oppiminen lisää oppimistuloksia **0,4-0,6 standardipoikkeamaa**

**Keskeinen periaate**: Itsenäisyys OPETETAAN, sitä ei oleteta

## Itsestään selvät tehtävämonisteet

**Suunnittele materiaaleja, jotka oppilaat voivat suorittaa yksin**

### Selkeät ohjeet tehtävämonisteessa

**Heikko suunnittelu** (vaatii opettajan selityksen):
```
Monisteen otsikko:
Nimi: ___________

[20 matematiikkatehtävää ilman kontekstia]

Ongelma: Oppilaat eivät tiedä mitä tehdä
- "Pitääkö näyttää laskutapa?"
- "Saanko käyttää laskinta?"
- "Kuinka paljon aikaa minulla on?"
→ Oppilaat kysyvät opettajalta (keskeyttää pienryhmän)
```

**Vahva suunnittelu** (itsestään selvä):
```
Nimi: ___________ Päivä: ________ Tavoiteaika: 20 minuuttia

OHJEET:
1. Ratkaise jokainen kertolasku
2. Näytä laskutapasi varattuun tilaan
3. ET saa käyttää laskinta
4. Kun valmis, tarkista vastaukset QR-koodilla (skannaa nurkka)
5. Jos sinulla on 5+ väärin, pyydä opettajalta apua
6. Jos saat kaikki oikein, siirry SYVENTÄVIEN TEHTÄVIEN laatikkoon

[20 matematiikkatehtävää, jokaisessa työtila]

[QR-koodi nurkassa] ← Skannaa tarkastaaksesi vastaukset

Selkeät odotukset: Oppilas tietää tarkalleen mitä tehdä (ei tarvitse kysyä opettajalta)
```

## Ajanhallintavälineet

**Auta oppilaita rytmittämään itsenäistä työskentelyä**

### Visuaalinen ajastin

**Luokkahuoneen järjestelmä**:
```
Työkalu: Iso digitaalinen ajastin heijastettuna taululle (tai fyysinen ajastin näkyvissä)

Opettaja ilmoittaa: "Teillä on 20 minuuttia matematiikan monisteeseen. Ajastin käynnistyy nyt."

Hyödyt:
- Oppilaat näkevät jäljellä olevan ajan (itsetahdistus)
- "Minulla on 10 minuuttia jäljellä ja 5 tehtävää" → Oppilas säätää tahtia
- Ei yllätyksiä kun aika loppuu (visuaalinen lähtölaskenta)

Itsenäinen taito: Oman ajan seuraamisen oppiminen (olennainen elämäntaito)
```

## Itsetarkistusjärjestelmät

**Välitön palaute ilman opettajaa**

### Vastausavaimen saatavuus

**Järjestelmävaihtoehdot**:

**Vaihtoehto 1: QR-koodi** (digitaalinen tarkistus)
```
Monisteen nurkka: QR-koodi
Oppilas skannaa: Vastausavain ilmestyy laitteelle
Itsetarkistus: Merkitsee omat virheensä punakynällä
Korjaa: Korjaa virheet
Palauttaa: Korjatun monisteen

Opettajan hyöty: Saa enimmäkseen oikein tehdyt työt (ei käytä aikaa yksinkertaisten virheiden merkitsemiseen)
Oppilaan hyöty: Välitön palaute (oppii virheistään heti)
```

**Vaihtoehto 2: Vastausavainpiste** (fyysinen tarkistus)
```
Luokkahuoneen järjestely: Vastausavainkansio takapöydällä

Oppilas suorittaa monisteen → kävelee pisteelle → tarkistaa vastaukset → palaa paikalle korjaamaan

Protokolla:
1. Suorita kaikki tehtävät ENSIN (ei tarkistamista yksi kerrallaan)
2. Käytä eri väristä kynää tarkistukseen (näyttää mitä korjattiin)
3. Korjaa virheet
4. Palauta vastausavain pisteelle

Hyöty: Oppilaat eivät jonota opettajan tarkistaessa töitä
```

**Vaihtoehto 3: Paritarkistus** (yhteistyö)
```
Parijärjestelmä: Oppilaat vaihtavat monisteet
Kumpikin tarkistaa parin työn vastausavaimella
Antaa palautetta: "Tehtävä 5 on väärin. Tarkista lainaaminen."

Kaksinkertainen hyöty:
- Molemmat oppilaat kertaavat materiaalia (tarkistaja vahvistaa omaa oppimistaan)
- Välitön palaute (ei odoteta opettajaa)
```

```
☐ Auta ystävää (jos hän pyytää apua)

EI saa:
☒ Häiritä opettajaa pienryhmäaikana
☒ Vaeltaa luokassa
☒ Istua tekemättä mitään

Selkeät odotukset: Oppilaat tietävät aina mitä tehdä seuraavaksi (ei keskeytyksiä)
```

## "Kysy kolmelta ennen minua" -protokolla

**Ongelmanratkaisu ennen opettajan kysymistä**

### Luokkahuoneen juliste

**Sääntö**: Ennen kuin kysyt opettajalta, kokeile näitä 3 asiaa:
```
1. Lue ohjeet huolellisesti uudelleen
   (Usein vastaus on ohjeissa)

2. Tarkista resursseista
   (Matematiikka: Viitekaavio, kaavasivu)
   (Lukeminen: Sanasto, sanapankki)
   (Kirjoittaminen: Tarkistuslista, arviointiperusteet)

3. Kysy naapurilta
   (Hiljaa: "Voitko auttaa minua ymmärtämään tehtävän 5?")

SITTEN: Jos edelleen jumissa, nosta käsi opettajalle

Hyödyt:
- Vähentää opettajan keskeytyksiä (moni kysymys ratkeaa itsenäisesti)
- Rakentaa ongelmanratkaisutaitoja (kekseliäisyys)
- Luo yhteistyöilmapiirin (vertaistuki)
```

## Itseohjautuva eteneminen

**Oppilaat etenevät kun ovat valmiita**

### Pistepohjäinen oppiminen

**Järjestelmä**: 4 pistettä, oppilaat työskentelevät omaan tahtiin
```
Piste 1: Yhteenlasku ryhmittelyllä
- Suorita moniste (20 tehtävää)
- Tarkista vastausavaimella (täytyy saada 16+ oikein = 80%)
- Kun läpäisty: Siirry pisteeseen 2

Piste 2: Vähennyslasku lainaamisella
- Suorita moniste
- Tarkista (tarvitaan 80%+)
- Kun läpäisty: Siirry pisteeseen 3

Piste 3: Sekoitettu yhteen-/vähennyslasku
- Suorita moniste
- Tarkista (tarvitaan 80%+)
- Kun läpäisty: Siirry pisteeseen 4

Piste 4: Sanallisia tehtäviä (soveltaminen)
- Suorita moniste
- Tarkista (tarvitaan 80%+)
- Kun läpäisty: Osio valmis! Aloita seuraava osio.

Eriyttäminen sisäänrakennettuna:
- Nopeat oppijat: Suorittavat kaikki 4 pistettä viikossa
- Keskitason oppijat: Suorittavat 1,5 viikossa
- Tukea tarvitsevat oppilaat: Suorittavat 2 viikossa
- Jokainen oppilas työskentelee sopivalla tahdilla (ei kiirehditä tai pidetä takana)
```

## Itsenäisyyden opettaminen vähitellen

**Tukea itsenäisyyttä ajan mittaan**

### Syyskuu: Vahva tuki

**Opettaja tekee**:
```
- Lukee ohjeet ääneen
- Käy esimerkkitehtävät yhdessä läpi
- Tarkistaa kaikkien työt
- Kertoo oppilaille mitä tehdä kun valmis

Itsenäinen työskentely: 10 minuuttia
Opettajan tuki: Vahva
```

---

### Marraskuu: Keskitason tuki

**Opettaja tekee**:
```
- Oppilaat lukevat ohjeet itsenäisesti (opettaja vastaa kysymyksiin)
- Oppilaat työskentelevät monisteen yksin
- Oppilaat itsetarkistavat vastausavaimella (opettaja pistotarkistaa)
- Oppilaat valitsevat 2-3 vaihtoehdosta kun valmiit

Itsenäinen työskentely: 20 minuuttia
Opettajan tuki: Keskitaso
```

---

### Maaliskuu: Vähäinen tuki

**Oppilaat tekevät**:
```
- Lukevat ja noudattavat ohjeita itsenäisesti
- Suorittavat koko monisteen ilman opettajan vuorovaikutusta
- Itsetarkistavat huolellisesti
- Valitsevat seuraavan toiminnan valintataulusta
- Työskentelevät tuottavasti 30+ minuuttia

Opettajan rooli: Työskentelee pienryhmän kanssa, tarkkailee luokkaa satunnaisesti
Itsenäisyys saavutettu: Oppilaat itseohjautuvia
```

**Eteneminen on vähittäistä**: Rakennetaan taitoja kuukausien aikana (ei odoteta välitöntä itsenäisyyttä)

## Fyysinen järjestely itsenäisyydelle

**Luokkahuoneen järjestely tukee itseohjautuvuutta**

### Materiaalipisteet

**Itsepalvelupisteet**:
```
Piste 1: "Lisämonisteet"
- Olit pois koulusta? Ota puuttuvat tehtävät
- Kadotit monisteen? Hanki kopio
- Oppilaan pääsy: Milloin tahansa (ei tarvitse kysyä opettajalta)

Piste 2: "Tarvikkeet"
- Kynät, pyyhekumit, värikynät, sakset
- Ota tarpeen mukaan (ei kysymistä)

Piste 3: "Palautuskorit"
- Yksi kori ainetta kohden (merkitty: Matematiikka, Lukeminen, Luonnontieto)
- Oppilas suorittaa työn → asettaa sopivaan koriin
- Ei odottamista opettajan kerätessä

Piste 4: "Tarkistetut työt"
- Arvioidut paperit palautetaan tähän
- Oppilaat noutavat itsenäisen työajan aikana

Hyöty: Oppilaat saavat tarvitsemansa itsenäisesti (opettajaa ei häiritä)
```

## Itsenäisen työskentelyn odotukset

**Opeta käyttäytyminen nimenomaisesti**

### Ankkurikaavio: "Miltä itsenäinen työskentely näyttää"

**Luo yhdessä oppilaiden kanssa**:
```
NÄYTTÄÄ TÄLTÄ:
- Työskentely pulpetilla tai määrätyllä työalueella
- Katse omassa paperissa
- Materiaalien hiljainen käyttö
- Itsenäinen työn tarkistaminen

KUULOSTAA TÄLTÄ:
- Kuiskausäänet (jos kysytään parilta apua)
- Kynät kirjoittavat
- Sivut kääntyvät
- Enimmäkseen hiljaista

EI NÄYTÄ TÄLTÄ:
- Luokassa kävely
- Kovaan ääneen puhuminen
- Opettajan keskeyttäminen
- Istuminen tekemättä mitään

Mallinnus: Opettaja näyttää sekä oikean että väärän (oppilaat tunnistavat eron)
Harjoittelu: Luokka harjoittelee itsenäistä työskentelyä opettajan tarkkaillessa
Vahvistaminen: Opettaja kehuu tiettyjä käyttäytymisiä ("Huomasin kaikkien työskentelevän hiljaa!")
```

**Opettaja tarkistaa keskeyttämättä pienryhmää**

### Nopea silmäily

**5 minuutin välein**:
```
Opettaja keskeyttää pienryhmäopetuksen hetkeksi (10 sekuntia)
Silmäilee luokkaa: Ovatko itsenäiset työntekijät tehtävässä?

Signaalit:
- Peukalo ylös: "Teet hienosti, jatka!" (ei sanoja tarvita)
- Osoittaa ajastinta: "Tarkista kuinka paljon aikaa sinulla on jäljellä"
- Nostaa monisteen: "Varmista että olet oikeassa tehtävässä"

Ei keskeytystä: Pienryhmä jatkaa, itsenäiset työntekijät ohjataan tarvittaessa
```

## Yleisten ongelmien ratkaiseminen

**Ratkaisut itsenäisyyden haasteisiin**

### Ongelma: "Oppilaat keskeyttävät edelleen pienryhmän aikana"

**Ratkaisu: Visuaalinen vihje**
```
Punainen kuppi opettajan pöydällä: "Älä keskeytä - vain hätätilanne"
Vihreä kuppi: "Voin ottaa vastaan nopeita kysymyksiä"

Sääntö: Kun punainen kuppi näkyy, oppilaiden TÄYTYY kokeilla "Kysy 3 ennen minua" -protokollaa
Tulos: Keskeytykset vähenevät 80%+
```

---

### Ongelma: "Oppilaat kiirehtivät läpi huolimattomasti"

**Ratkaisu: Laatustandardit**
```
Vähimmäisaikavaatimus: "Tämän monisteen tulisi viedä vähintään 15 minuuttia"

Plus: Itsetarkistus paljastaa virheet (oppilas näkee kiirehtimisen seuraukset)

Plus: "Jos valmistut alle 10 minuutissa, tarkistan työn. Jos 3+ virhettä, sinun täytyy tehdä uudelleen." (vastuullisuus)

Tulos: Oppilaat oppivat työskentelemään huolellisesti (kiirehtimistä ei palkita)
```

## Yhteenveto

Itsesäädelty oppiminen lisää oppimistuloksia **0,4-0,6 SD** (Zimmerman, 2002) - itsenäisyys voimaannuttaa oppilaita.

**Itsestään selvät monisteet**: Selkeät ohjeet arkkissa (odotettu aika, vaiheet, tarkistusmenetelmä), oppilaat suorittavat kysymättä

**Ajanhallinta**: Visuaaliset ajastimet (oppilaat näkevät lähtölaskennan), henkilökohtainen seuranta (aloitus-/lopetusajat), rakentaa tahdistustietoisuutta

**Itsetarkistus**: QR-koodivastausavaimet (välitön palaute), vastausavainpisteet (fyysinen tarkistus), paritarkistus (yhteistyö)

**Valintataulut**: Pakollisia vaiheita (tarkista, korjaa) + valitse sitten -vaihtoehdot (pulmat, lukeminen, syventävät), selkeät seuraavat askeleet

**Kysy 3 ennen minua**: Lue ohjeet uudelleen → tarkista resurssit → kysy parilta → sitten opettaja (vähentää keskeytyksiä 80%+)

**Itseohjautuvat pisteet**: Etene kun valmis (80%+ edetäksesi), eriyttäminen automaattista (nopeat/keskitaso/hitaat oppijat)

**Vähittäinen vapautus**: Syyskuun vahva tuki → maaliskuun vähäinen tuki (rakennetaan itsenäisyyttä kuukausien aikana)

**Fyysinen järjestely**: Materiaalipisteet (itsepalvelutarvikkeet, palautuskorit), oppilaat käyttävät itsenäisesti

**Odotukset**: Ankkurikaavio (näyttää/kuulostaa), mallinnus ja harjoittelu, vahvistetaan positiivisia käyttäytymisiä

**Tutkimus**: Itsesäätely = 0,4-0,6 SD kasvu (Zimmerman, 2002)
