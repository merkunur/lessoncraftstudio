## Innledning: De Avgjørende Første Dagene

**Forskning** (Wong & Wong, 2009): De to første ukene avgjør **suksess for hele skoleåret**

**Hva skjer i første uke**:
```
Sterk start:
- Tydelige rutiner innøves
- Forventninger etableres
- Prosedyrer praktiseres
- Klassefellesskap bygges
Resultat: Resten av året går smidigt (95% færre atferdsproblemer)

Svak start:
- Ingen rutiner innlæres
- Vage forventninger
- Kaos tolereres
Resultat: Atferdsproblemer hele året
```

**Nøkkelprinsipp**: "Du får aldri en ny sjanse til å gjøre et godt førsteinntrykk"

## Dag 1: Skape Trygghet og Oversikt

**Mål**: Trygghet, klasseromsoversikt, fellesskap

### Formiddag: Klasseromsjakt

**Aktivitet**: Elevene fullfører skattejakt-arbeidsark
```
Klasseromsjakt

Navn: _____________ Dato: _______

Finn disse tingene og skriv hvor de er:

1. Hvor leverer vi inn ferdig arbeid? ________
2. Hvor er ekstra blyanter? __________
3. Hvor stiller vi oss når vi skal ut? ___________
4. Hvor er klassebiblioteket? ______________
5. Hvor henger vi sekken? __________
6. Hvor er nødutgangen? _____________
7. Hvor finner vi dagens timeplan? _______
8. Hvor står klassemateriell? _________

Bonus: Hva er én klasseregl du ser på veggen?
_________________________________________________
```

**Formål**: Elevene lærer klasserommet aktivt (ikke passiv omvisning)

---

**Aktivitet**: Om Meg-arbeidsark
```

MINE FAVORITTER:
Farge: ________ Mat: ________ Dyr: ________
Bok: ____________ Fag: _________________

MIN FAMILIE:
Personer i familien min: _______________________
Kjæledyr: _____________________________________

I ÅR:
Jeg gleder meg til: _________________________
Jeg er litt nervøs for: _________________________
Mitt mål for året: ____________________

MORSOMT:
Noe spennende om meg: ____________
_________________________________________________
```

**Lærerbruk**: Lær å kjenne 30 elever raskt (30 ark lest på 30 minutter)

## Dag 2: Øve på Rutiner

**Mål**: Lær HVORDAN klasserommet fungerer

### Formiddag: Rutinesjekkliste

**Aktivitet**: Elevene fyller ut mens de øver
```
Klasseromrutiner - Øving

I dag skal vi øve på:

MORGENRUTINE:
☐ Heng opp sekk
☐ Ta fram lekser (hvis vi har)
☐ Kvess blyanter (2 blyanter)
☐ Les morgenbeskjed på tavla
☐ Begynn morgenstarten

UNDER TIMER:
☐ Rekk opp hånda for å svare
☐ Lytt når andre snakker
☐ Følg beskjeder første gang
☐ Spør om hjelp på riktig måte

OVERGANGER:
☐ Rydd arbeidsplassen
☐ Skyv inn stolen
☐ Still deg i kø stille
☐ Gå stille i gangen (nivå 0)

Jeg øvde på disse rutinene i dag! ✓
Lærer signatur: ____________
```

**Metode**: Vi ØVER faktisk på hver rutine (ikke bare snakker om den)

---

### Ettermiddag: Klasseoppgaver

**Aktivitet**: Søk om klasseoppgaver
```
Søknad om Klasseoppgave

Navn: _____________ Dato: _______

Jeg vil gjerne søke om denne oppgaven: __________

Tilgjengelige oppgaver:
- Køvakt (leder klassen i gangen)
- Dørvakt (holder døra for klassen)
- Arkutdeler (deler ut arbeidsark)
- Tavlevask (vasker tavla ved dagens slutt)
- Blyantkvesser (kveser klassens blyanter)
- Bibliotekar (organiserer klassebiblioteket)
- Fravær-ansvarlig (leverer fravær til kontoret)

Hvorfor jeg passer til denne oppgaven:
_________________________________________________
_________________________________________________

Tidligere erfaring (hjemme eller på skolen):
_________________________________________________

Lærer: Gjennomgå søknader, tildel oppgaver onsdag
```

## Dag 3: Kartlegge Utgangspunkt

**Mål**: Vurdere hvor elevene starter

### Formiddag: Lesevurdering

**Aktivitet**: Leseflyt-sjekk + forståelse
```
Les denne teksten stille for deg selv (lærer tar tid):

[Tekst på klassenivå, 100 ord]

Tid: _____ sekunder = _____ ord/minutt

Nå svar på disse spørsmålene:
1. Hva handlet teksten om? (hovedidé)
2. Hvorfor gjorde [person] [handling]? (slutning)
3. Hva betyr [ord]? (ordforråd)

Lærer: Registrer data (utgangspunkt for fremgangssporing)
```

---

### Ettermiddag: Matematikk-Kartlegging

**Aktivitet**: Repetisjonsarbeidsark (fjorårets pensum)
```
Matematikk-Repetisjon

20 oppgaver som dekker:
- Addisjon/subtraksjon (forrige trinn)
- Grunnleggende multiplikasjon (hvis aktuelt)
- Tekstoppgaver
- Former/måling

Formål: Ikke karakter! Hjelper bare læreren å forstå hva du husker.

Gjør ditt beste, men ikke stress hvis du ikke kan noe!
```

## Dag 4: Skape Fellesskap

**Mål**: Bygge positivt klassemiljø

### Formiddag: Finn Noen Som...

**Aktivitet**: Bingo-aktig bli-kjent aktivitet
```
Finn Noen Som...

Gå rundt i klasserommet, finn klassekamerater som passer til hver beskrivelse. Få signaturen deres!

☐ Har samme favorittfarge som deg: _______
☐ Har bursdag samme måned: ________
☐ Liker samme bokserie: _____________
☐ Har vært i et annet land: ___________
☐ Spiller et instrument: ____________
☐ Har mer enn 2 søsken: ______________
☐ Liker matte: ____________________________
☐ Elsker å lese: _________________________

Mål: Fyll hele arket, møt alle i klassen!
```

---

### Ettermiddag: Klassens Felles Mål

**Aktivitet**: Sammen lager vi klasseregler
```
Våre Klassemål Dette Året

Hvordan vil vi at klassen vår skal være?

Arbeidsark (individuelt):
Jeg håper klassen blir: __________________
Noe jeg kan gjøre for å bidra: ________________

Klassen: Del ideer, lærer skriver på plakat
Resultat: "Vår Klasseavtale"
Alle elever signerer (forpliktelse)
```

## Dag 5: Repetisjon og Befestelse

**Mål**: Befeste rutinene lært mandag-torsdag

### Formiddag: Fullstendig Dag-Simulering

**Aktivitet**: Øv på komplett dag som om det er uke 2
```
Morgen:
- Fullfør morgenstart (matte oppvarming-ark)
- Ta fram lekser (ingen ennå, men øv bevegelsen)
- Lever i kurven

Middag:
- Overgang til lesing (øv oppryddingsprosedyre)
- Jobb med leseark selvstendig
- Lever når du er ferdig

Ettermiddag:
- Overgang til matte (øv igjen)
- Fullfør matte-ark
- Sjekk selv med fasit (øv prosedyre)

Dagens slutt:
- Pakk sekken (øv)
- Rydd arbeidsplassen (øv)
- Still i kø for henting (øv)

Tilbakemelding: "I dag gjorde vi hver rutine perfekt!"
```

## Vikarplan For Første Uke

- Samle inn arbeidsark, legg til lærer
