## Johdanto: Opettajan Budjettipaineet

**Opettajan todellisuus**: Keskimääräinen opettaja käyttää **750 euroa vuodessa** omista varoistaan luokkahuoneen tarvikkeisiin

**Budjetti eriteltynä**:
```
Tyypillinen kulutus:
- Työkirjat: 300€ (kertakäyttöisiä, ei voi käyttää uudelleen)
- Opetusvälineet: 150€
- Luokkahuoneen sisustus: 100€
- Tarvikkeet: 200€
Yhteensä: 750€ omasta pussista
```

**Ongelma**: Työkirjat ovat kertakäyttöisiä (ei voi käyttää uudelleen seuraavana vuonna)
```
Hankinta: 30 matematiikan työkirjaa × 10€ = 300€
Käyttö: Syyskuusta toukokuun loppuun (oppilaat kirjoittavat kirjoihin)
Kesäkuu: Kirjat menevät kotiin (ei voi käyttää uudelleen)
Seuraava vuosi: Osta 30 uutta työkirjaa (300€ uudelleen)
5 vuoden kustannus: 1 500€ kulutusmateriaaleihin
```

**Ratkaisu**: Digitaalinen generointi + tulostaminen = uudelleenkäytettävät materiaalit
```
Kustannus: 144€/vuosi tilaus
Generoi: Rajoittamattomasti työarkkeja (vastaa yli 100 työkirjaa)
Uudelleenkäyttö: Tallenna PDF:t, tulosta uudelleen seuraavana vuonna (ei lisäkustannuksia)
5 vuoden kustannus: 720€ (säästä 780€ verrattuna työkirjoihin)
```

## Uudelleenkäytettävien Materiaalien Strategiat

**Vähennä paperin kulutusta**

### Kuivapyyhittävät Taskut

**Kertainvestointi**: 50€ 30 taskulle
```
Näin se toimii:
1. Tulosta työarkki (yksi kopio)
2. Laita kuivapyyhittävään taskuun
3. Oppilas täyttää kuivapyyhittävällä tussilla
4. Pyyhi, käytä uudelleen yli 100 kertaa

Esimerkki:
Aamuinen matematiikan lämmittely (5 tehtävää päivittäin):
- Ilman taskuja: Tulosta 30 kopiota × 180 päivää = 5 400 arkkia
- Taskulle: Tulosta 30 kopiota kerran, käytä koko vuosi = 30 arkkia
Säästetty paperi: 5 370 arkkia (268,50€ hintaan 0,05€/arkki)

ROI: 50€ investointi säästää 268,50€/vuosi (5,4× tuotto)
```

---

### Laminoidut Työarkit

**Usein käytetyille aktiviteeteille**:
```
Laminoi: Kertotaulut, aakkoset, lukusuorat
Hinta: 20€ laminointikone + 10€ laminointimuovit = 30€
Kestävyys: Kestää yli 5 vuotta

Käyttö: Oppilaat käyttävät kuivapyyhittävillä tusseilla
Esimerkki: "Jäljitä kirjaimet" -työarkki
- Tulosta kerran, laminoi, käytä yli 200 kertaa vuosien varrella
- Hinta per käyttökerta: 0,05€ ÷ 200 = 0,00025€ (käytännössä ilmainen)
```

## Paperin Säästöstrategiat

**Vähennä tulostuskustannuksia**

### Kaksipuoleinen Tulostus

**Yksinkertainen muutos, suuret säästöt**:
```
Yksipuoleinen: 180 työarkkia = 180 arkkia
Kaksipuoleinen: 180 työarkkia = 90 arkkia
Säästö: 90 arkkia per oppilas = 2 700 arkkia/luokka = 135€/vuosi

Asetukset: Tulostimen asetukset → Kaksipuoleinen tulostus → "Lyhyt reuna" (kääntö)
```

---

### Näyttö + Vihkopaperi

**Poista monistamisen tarve**:
```
Perinteinen: Tulosta 30 kopiota työarkista
Vaihtoehto: Näytä YKSI työarkki näytöllä, oppilaat kopioivat vihkoon

Esimerkki:
Matematiikan tehtävät: Näytä 20 tehtävää projektorilla
Oppilaat: Kirjoittavat tehtävät + vastaukset matematiikan vihkoon
Käytetty paperi: 0 arkkia

Milloin sopiva: Laskuharjoitukset, sanaston määritelmät, ymmärtämiskysymykset
Milloin ei sopiva: Monimutkaiset kaaviot, pitkät tekstit
```
