## Introduction : La Motivation par la Reconnaissance

**Types de motivation** (Deci & Ryan, 1985) :
- **Extrinsèque** : Récompenses externes (autocollants, prix)
- **Intrinsèque** : Satisfaction interne (fierté, sentiment de compétence)

**Recherche** (Hattie, 2009) : La reconnaissance et le feedback présentent une **taille d'effet de 0,75** (l'une des influences les plus puissantes sur la réussite scolaire)

**Équilibre nécessaire** : Utiliser l'extrinsèque au début, progresser vers l'intrinsèque au fil du temps
```
Septembre : "Complète 10 fiches, gagne un prix" (motivation extrinsèque)
Janvier : "J'adore travailler dur !" (motivation intrinsèque développée)
Résultat : Les récompenses lancent l'habitude, la motivation intrinsèque la maintient
```

**Principe clé** : Célébrer l'EFFORT et la PROGRESSION (pas seulement les scores parfaits)

## Reconnaissance Basée sur l'Effort

**Se concentrer sur le PROCESSUS, pas uniquement sur le résultat**

### Certificats "Meilleure Progression"

**Reconnaissance hebdomadaire** :
```
Pas : "Qui a obtenu 100% ?" (basé sur les capacités)
Mais plutôt : "Qui a montré la plus grande progression ?" (basé sur l'effort)

Exemple :
Élève A : Lundi 60% → Vendredi 85% (progression de 25%)
Élève B : Lundi 90% → Vendredi 95% (progression de 5%)

Meilleure Progression : Élève A (célébré pour sa croissance, pas la perfection)

Certificat :
"Meilleur Progrès en Mathématiques - Semaine du [date]
[Élève A] a progressé de 25 points grâce à son travail acharné et sa détermination !"
```

### Prix de la Persévérance

**Reconnaître la persistance face aux défis** :
```
Un élève se bat avec un Sudoku 9×9 (prend 60 minutes sur 3 jours)
Enfin terminé : Vendredi après-midi

Récompense : "Champion de la Persévérance"
Certificat : "N'a jamais abandonné le défi - a complété le Sudoku 9×9 grâce à sa détermination !"

Valeur : L'effort est valorisé (pas les capacités innées)
État d'esprit de croissance : "La persévérance paie"
```

## Système d'Économie de Classe

**Les fiches rapportent de la "monnaie"**

### Billets de Classe

**Système** :
```
Chaque fiche complétée = 1€ de classe
Score parfait (100%) = bonus de 0,50€

Suivi par élève :
Prénom : ____________ Solde : ______€

Potentiel hebdomadaire : 5 fiches × 1€ = 5€
Semaine d'effort intense : 10 fiches = 10€
```

**Options de dépense** :
```
5€ : Choisir sa place pour la semaine
10€ : Bon de dispense de devoirs (sauter un devoir)
15€ : Déjeuner avec l'enseignant(e)
25€ : Être l'assistant(e) du jour
50€ : Prix spécial de la boîte à surprises

Épargne : Les élèves apprennent la gratification différée (économiser 50€ pour un gros prix vs dépenser 5€ immédiatement)
```

## Célébrations du Vendredi

**Rituel de reconnaissance de fin de semaine**

### Rassemblement du Vendredi (15 minutes)

**Structure** :
```
15h00-15h05 : L'enseignant(e) annonce les lauréats de la semaine
"Cette semaine, 7 élèves ont atteint le jalon de 50 fiches ! Félicitons : [prénoms]"
Classe : Applaudissements

15h05-15h10 : Témoignage d'un élève (partage son apprentissage)
Élève sélectionné : "Cette semaine j'ai appris les fractions. C'était difficile mais j'ai persévéré. Ma fiche préférée était le coloriage de pizza fractionnée !"
Classe : Applaudissements et encouragements

15h10-15h15 : Aperçu de la semaine prochaine + message motivant
Enseignant(e) : "La semaine prochaine nous apprendrons les multiplications. Je sais que vous travaillerez dur. Bon week-end !"

Bénéfice : Fin de semaine positive (les élèves partent fiers d'eux)
```

## Entretiens Individuels

**Reconnaissance personnelle**

### Point Mensuel (5 minutes par élève)

**Structure d'entretien** :
```
Enseignant(e) : "Regardons ton tableau de progrès. En septembre tu as complété 15 fiches. En octobre, 22 ! Quelle progression !"

Élève : Voit ses propres données (preuve concrète d'amélioration)

Enseignant(e) : "De quoi es-tu le plus fier ?"
Élève : "Je suis fier d'avoir fini le Sudoku difficile"

Enseignant(e) : "Quel est ton objectif pour novembre ?"
Élève : "Complèter 25 fiches"

Bénéfice : L'élève fixe son propre objectif (appropriation), l'enseignant reconnaît les progrès (validation)
```

## Éviter les Pièges des Récompenses

**Erreurs courantes à éviter**

### Piège 1 : Ne Récompenser que les Scores Parfaits

**Problème** : État d'esprit fixe (seuls les "bons élèves" obtiennent des récompenses)
```
Mauvais : "Tous ceux qui ont obtenu 100% reçoivent un prix"
Résultat : Les élèves en difficulté ne sont jamais récompensés (démotivation)
```

**Solution** : Récompenser la PROGRESSION
```
Bon : "Tous ceux qui se sont améliorés depuis la semaine dernière reçoivent une reconnaissance"
Résultat : Tous les élèves peuvent obtenir une reconnaissance (basé sur l'effort)
```

---

### Piège 2 : Récompenses Matérielles Constantes

**Problème** : Dépendance à la motivation extrinsèque
```
Mauvais : Prix pour chaque fiche complétée
Résultat : "Je ne travaille que si j'ai un prix" (pas de motivation intrinsèque)
```

**Solution** : Réduire progressivement les récompenses tangibles
```
Septembre : Prix toutes les 5 fiches (lancement)
Décembre : Prix toutes les 20 fiches (moins fréquent)
Mars : Certificats uniquement (motivation intrinsèque primaire)
Résultat : Transition progressive vers la motivation interne
```

---

### Piège 3 : Humiliation Publique

**Jamais** : Affichage public des moins performants
```
Mauvais : "Ces élèves n'ont pas fini leur travail" (affiche avec noms)
Résultat : Honte, motivation réduite, relations endommagées
```

**Toujours** : Soutien privé
```
Bon : Conversation privée avec l'élève en difficulté
"J'ai remarqué que tu as du mal à terminer. De quel soutien as-tu besoin ?"
Résultat : Résolution de problème, maintien de la dignité
```

## Célébrations d'Événements Spéciaux

**Célébrations plus importantes occasionnelles**

### Fête de Fin d'Unité

**Après avoir complété une unité majeure** :
```
Unité : Fractions (4 semaines de travail)
Évaluation finale : 80%+ de maîtrise collective

Célébration : "Fête des Fractions"
- Apporter : Pizza (coupée en fractions : "Qui veut 3/8 de pizza ?")
- Activité : Bingo des Fractions (jeu de révision)
- Certificat : "Maître des Fractions" pour tous ceux qui ont atteint la maîtrise

Durée : 30 minutes (dernière période du vendredi)
Connexion académique : La fête renforce le contenu (pas seulement une pause ludique)
```

---

### 100 Jours d'École

**Jalon : 100ème jour d'école**
```
Activité : "Défi 100 Fiches"
Objectif : La classe complète 100 fiches collectivement en une journée

Organisation :
- Variété : Maths, mots mêlés, coloriages, mots croisés, puzzles (choix disponibles)
- Suivi : Grand tableau (les élèves ajoutent des bâtons à mesure qu'ils terminent)
- Célébration : Si objectif atteint à 14h, dernière heure = temps de jeux

Résultat : 100 fiches complétées (célébration productive)
Valeur académique : Pratique de révision (pas de temps d'apprentissage perdu)
```
