## Johdanto: Ikätasoinen pedagogiikka varhaiskasvatuksessa

**Varhaiskasvatuksen periaate**: Aktiviteettien on vastattava lapsen kehitystasoa – ei pelkästään ikää.

Suomalaisessa varhaiskasvatuksessa korostetaan lapsilähtöisyyttä ja kehityksen yksilöllisyyttä. Jokainen lapsi kehittyy omassa tahdissaan, ja oppimateriaalin on tuettava tätä kehitystä luonnollisesti.

**Varhaiskasvatuksen kehitysvaiheet**:

```
3-4-vuotiaat (Nuorempi esiopetus):
- Hienomotoriikka: Kynäotteen oppiminen, saksen käyttö
- Kognitiiviset taidot: Konkreettinen ajattelu, leikkien oppiminen
- Keskittymiskyky: 5-10 minuutin aktiviteetit
- Sosiaaliset taidot: Rinnakkaisleikistä yhteisleikkiin

5-6-vuotiaat (Esikouluikäiset):
- Hienomotoriikka: Kynänhallinnan kehittyminen, kirjainten jäljittely
- Kognitiiviset taidot: Symbolisen ajattelun alku (kirjaimet edustavat ääniä)
- Keskittymiskyky: 10-20 minuutin aktiviteetit
- Sosiaaliset taidot: Yhteistoiminnallinen oppiminen, vuorottelu
```

**Tehtävien suunnitteluperiaate**: LYHYT, VISUAALINEN, KONKREETTINEN (ei abstraktia tai pitkiä)

**Tutkimustieto**: Liian vaikeat aktiviteetit aiheuttavat **turhautumista ja motivaation laskua**, kun taas sopivan tasoiset tehtävät rakentavat **itseluottamusta ja taitoja**.

## 7 tärkeintä generaattoria varhaiskasvatukseen

**Järjestys kehitystasoisen sopivuuden mukaan** (eniten käytetty → erikoistunut):

### #1: Väritystehtävät (App 005) ⭐ PARAS VARHAISKASVATUKSELLE

**Miksi väritys on täydellinen 3-6-vuotiaille**:
- **Hienomotoriikan kehitys**: Kolmisormiote harjoituksessa (välttämätön kirjoitustaidolle)
- **Kahdenkäden koordinaatio**: Toinen käsi pitää paperia, toinen värittää (aivojen kehitys)
- **Silmän-käden koordinaatio**: Silmä ohjaa kättä (spatiaalinen hahmotus)
- **Keskittymisen harjoittelu**: 10-20 minuuttia jatkuvaa keskittymistä
- **Matala stressitaso**: Ei oikeita tai vääriä vastauksia (rakentaa itseluottamusta)

**Kehitysvaiheet**:

**3-4-vuotiaat**:
```
Aktiviteetti: Isot, yksinkertaiset kuvat (omena, aurinko, pallo)
Odotettu taso: Väritys menee viivojen yli (normaalia!)
Tavoite: Kynän pitämisen harjoittelu, paineen säätö
Aika: 5-10 minuuttia
```

**4-5-vuotiaat**:
```
Aktiviteetti: Keskitasoiset kuvat (talo ikkunoineen, kukka terälehdineen)
Odotettu taso: Jonkin verran viivojen sisäpuolella pysymistä
Tavoite: Kädenhallinta paranee, värien tunnistus
Aika: 10-15 minuuttia
```

**5-6-vuotiaat (Esikouluikäiset)**:
```
Aktiviteetti: Yksityiskohtaisemmat kuvat (kohtaus useilla elementeillä)
Odotettu taso: Suurimmaksi osaksi viivojen sisällä pysymistä
Tavoite: Tarkkuus, kokonaisen kuvan loppuunvärittäminen
Aika: 15-20 minuuttia
```

**Vinkki varhaiskasvatukseen**: Väritys = VALMISTAUTUMISTA kirjoittamiseen (samat lihakset, pienempi paine)

### #2: Viivojen piirtäminen (App 016) ⭐ KIRJOITUKSEN ESITAITOJEN RAKENTAJA

**Miksi viivojen piirtäminen kehittää kirjoitusvalmiutta**:
- **Kynänhallinta**: Polun seuraaminen kynällä (tarkkuusharjoittelu)
- **Visuaalinen seuraaminen**: Silmät seuraavat viivaa (vasemmalta oikealle lukemisen valmistelu)
- **Käden voima**: Jatkuva kynän paine (kirjoituskestävyys)

**Kehitysvaiheet**:

**3-4-vuotiaat: Suorat viivat**
```
Aktiviteetti: Yhdistä kaksi pistettä (suora viiva)
Esimerkki: Piirrä viiva pallosta koriin
Pituus: 5-8 cm
Vaikeus: Ensin vaakasuorat (helpoin), sitten pystysuorat, sitten vinot
```

**4-5-vuotiaat: Kaartuvat viivat**
```
Aktiviteetti: Seuraa mutkittelevaa polkua
Esimerkki: Auta pupua pääsemään porkkanan luo (kaartuva viiva)
Monimutkaisuus: Yksinkertaiset kaaret (ei teräviä käännöksiä)
```

**5-6-vuotiaat: Sokkelot**
```
Aktiviteetti: Navigoi yksinkertainen sokkelo (useita käännöksiä)
Esimerkki: Auta hiirtä saamaan juuston sokkelon läpi
Monimutkaisuus: 5-8 käännöstä, selkeä polku
```

**Tutkimus**: Visuaalis-motorinen integraatio (viivan jäljittely) ennustaa **kirjoitusvalmiutta** vahvalla korrelaatiolla.

### #3: Kuvabingo (App 012) ⭐ SANASTON KEHITTÄJÄ

**Miksi bingo toimii varhaiskasvatuksessa**:
- **Puhutun kielen kehitys**: Kuulee sanan, tunnistaa kuvan (reseptiivinen sanasto)
- **Visuaalinen erottelu**: Löydä vastaava kuva monien joukosta
- **Pelimuoto**: Hauskaa, sosiaalista (ei "työtä")
- **Lyhyt kesto**: 15-20 minuuttia (vastaa keskittymiskykyä)

**Sanastokategoriat esiopetukseen**:

**Syysteema** (syyskuu-lokakuu):
```
Kuvat: omena, kurpitsa, lehti, tammenterho, linnunpelätin, kalkkuna, maissi, piirakka, puu, orava
Aktiviteetti: Opettaja sanoo "Etsi kurpitsa!"
Taidot: Vuodenaikasanasto, kuunteleminen, visuaalinen etsintä
```

**Ammattilaiset** (marraskuu):
```
Kuvat: palomies, poliisi, lääkäri, opettaja, postinkantaja, maanviljelijä, kokki, lentäjä, hammaslääkäri, kirjastonhoitaja
Aktiviteetti: Kutsutaan ammatteja, oppilaat merkitsevät kuvia
Taidot: Ammatit, yhteisötuntemus
```

**Eläimet** (joulukuu):
```
Kuvat: kissa, koira, lintu, kala, kani, hevonen, lehmä, sika, kana, hiiri
Aktiviteetti: Tehkää eläinten ääniä, oppilaat arvaavat + merkitsevät
Taidot: Eläinten nimet, ääniyhteydet (moniaistillinen)
```

**Kesto**: 15-20 minuuttia per peli (täydellinen piirihetkeen)

### #4: Etsi esineet / Etsi ja löydä (App 026) ⭐ VISUAALINEN HAHMOTTAMINEN

**Miksi "etsi ja löydä" kehittää esitaitoja lukemiselle**:
- **Visuaalinen skannaus**: Vasemmalta oikealle silmänliike (lukemisvalmiuden kehitys)
- **Hahmo-tausta-erottelu**: Etsi tietty esine häiritseijöiden joukosta (kirjaintunnistuksen valmistelu)
- **Pitkäjänteinen keskittyminen**: Etsi useita esineitä (keskittymisharjoitus)

**Ikätasoiset "etsi ja löydä" -tehtävät**:

**3-4-vuotiaat: Yksinkertainen etsintä**
```
Tehtävä: "Etsi 3 palloa"
Tehtävämoniste: 15 esinettä yhteensä (3 palloa 12 muun esineen joukossa)
Kognitiivinen vaatimus: Yksinkertainen vastaavuus (kaikki pallot näyttävät samalta)
```

**4-5-vuotiaat: Kategoriaetsintä**
```
Tehtävä: "Etsi 5 asiaa, joita voi syödä"
Tehtävämoniste: 20 esinettä (ruokaa ja muuta sekaisin)
Kognitiivinen vaatimus: Kategorisointi (mitkä ovat ruokaa?)
```

**5-6-vuotiaat: Ominaisuusetsintä**
```
Tehtävä: "Etsi 4 PUNAISTA asiaa"
Tehtävämoniste: 25 esinettä eri väreissä
Kognitiivinen vaatimus: Suodatus ominaisuuden mukaan (värien erottelu)
```

**Aika**: 10-15 minuuttia (itsenäinen tai pariwork)

### #5: Kuviojuna (App 021) ⭐ MATEMATIIKAN VALMISTELU

**Miksi kuviot rakentavat matematiikan perustaa**:
- **Järjestysajattelu**: Järjestyksen ymmärtäminen (edellytys laskemiselle)
- **Ennustaminen**: Mikä tulee seuraavaksi? (looginen ajattelu)
- **Hienomotoriikka**: Leikkaaminen ja liimaaminen (saksitaidot)

**Kuvion monimutkaisuuden kehitys**:

**3-4-vuotiaat: AB-kuvio** (yksinkertaisin)
```
Kuvio: Punainen junan vaunu, sininen vaunu, punainen, sininen, punainen, sininen
Tehtävä: Liimaa seuraava vaunu (mikä väri?)
Vastaus: Sininen (toistaa AB-kuviota)
```

**4-5-vuotiaat: ABB-kuvio**
```
Kuvio: Ympyrä, neliö, neliö, ympyrä, neliö, neliö
Tehtävä: Jatka kuviota (mikä tulee seuraavaksi?)
Vastaus: Ympyrä
```

**5-6-vuotiaat: ABC-kuvio** (monimutkainen)
```
Kuvio: Kolmio, ympyrä, neliö, kolmio, ympyrä, neliö
Tehtävä: Jatka ja tunnista kuvion sääntö
Vastaus: Kolmio (ABC toistuu)
```

**Matematiikkayhteys**: Kuviot = perusta hyppylaskennalle (2, 4, 6, 8 on kuvio!)

### #6: Yhteenlasku (App 001) - VAIN KUVAMUOTO

**TÄRKEÄÄ**: 3-6-vuotiaiden tulisi käyttää VAIN **kuvamuotoa** (ei symbolista)

**Miksi kuvamuoto on ikätasoinen**:
- **Konkreettinen esitys**: Näe todelliset esineet (ei abstrakteja symboleja)
- **Yksi-yhteen-vastaavuus**: Laske jokainen esine (lukujen ymmärrys)
- **Visuaalinen subitisaatio**: Tunnista pienet määrät laskematta (1-5)

**Ikätasoiset yhteenlaskuaktiviteetit**:

**4-5-vuotiaat: Yhteenlasku esineillä 1-5**
```
Ongelma: [Kuva: 2 omenaa] + [Kuva: 3 omenaa] = ___
Oppilas: Laskee kaikki 5 omenaa
Vastaus: 5

Kognitiivinen taso: Esisymbolinen (ei vielä "2 + 3 = 5")
Tavoite: Määrän ymmärtämisen rakentaminen
```

**5-6-vuotiaat: Yhteenlasku esineillä 1-10**
```
Ongelma: [Kuva: 4 palloa] + [Kuva: 5 palloa] = ___
Oppilas: Laskee kaikki TAI ryhmittelee (4 ja vielä 5 tekee 9)
Vastaus: 9

Kognitiivinen taso: Siirtymä symboliseen (osa oppilaista valmiita "4 + 5 = 9")
Tavoite: Laskennallinen ajattelu konkreettisella tuella
```

**Kesto**: Vain 5-10 laskua (lyhyt keskittymiskyky)

### #7: Kuvien lajittelu (App 031) ⭐ LUOKITTELUTAIDOT

**Miksi lajittelu kehittää kognitiivisia taitoja**:
- **Kategorisointi**: Ryhmittele ominaisuuden mukaan (kriittinen ajattelu)
- **Ominaisuuksien tunnistaminen**: Tunnista yhtäläisyydet/erot
- **Sanasto**: Nimeä kategoriat (kielellinen kehitys)

**Lajitteluaktiviteetit 3-6-vuotiaille**:

**3-4-vuotiaat: Yksinkertainen kahden ryhmän lajittelu**
```
Tehtävä: Lajittele esineet "Iso" ja "Pieni" -ryhmiin
Esineet: 10 kuvaa (5 isoa eläintä, 5 pientä eläintä)
Kognitiivinen vaatimus: Koon erottelu
```

**4-5-vuotiaat: Moniluokkainen lajittelu**
```
Tehtävä: Lajittele esineet tyypin mukaan (eläimet, ruoka, lelut)
Esineet: 15 sekalaista kuvaa
Kognitiivinen vaatimus: Semanttiset kategoriat (mikä tämä asia ON?)
```

**5-6-vuotiaat: Ominaisuuslajittelu**
```
Tehtävä: Lajittele värin TAI muodon mukaan (oppilas valitsee ominaisuuden)
Esineet: 20 kuvaa useilla ominaisuuksilla
Kognitiivinen vaatimus: Joustava ajattelu (voi lajitella usealla tavalla)
```

## Ryhmänhallinta varhaiskasvatuksessa

**Haaste**: 3-6-vuotiaat tarvitsevat ENEMMÄN tukea kuin vanhemmat oppijat

**Ratkaisut**:

### Strategia 1: Mallinnan ensin (Minä teen, Me teemme, Sinä teet)

**Rakenne**:
```
Minä teen (2 min): Opettaja näyttää mallin taululle/projektorille
- "Katsokaa, kun värittelen tämän omenan. Pidän värikyniä näin..."

Me teemme (3 min): Ryhmä tekee yhden yhdessä
- "Nyt väritetään kaikki yhdessä omena. Ottakaa punainen kynä..."

Sinä teet (10 min): Oppilaat työskentelevät itsenäisesti
- "Nyt värittäkää oma omenanne! Kierrän auttamassa."
```

**Hyöty**: Selkeät odotukset (oppilaat tietävät, mitä tehdä)

---

### Strategia 2: Toimintapisteet (Kierto)

**Järjestely**: 4 pistettä, 5 oppilasta kussakin, kierto 15 minuutin välein

```
Piste 1 (Opettajan ohjaus): Uuden taidon esittely (viivojen piirtäminen)
Piste 2 (Itsenäinen): Väritys (aiemmin esitelty)
Piste 3 (Pari): Kuvabingo (oppilaat vuorottelevat kutsumisessa)
Piste 4 (Itsenäinen): Kuviojuna (leikkaa ja liimaa)

Kierto:
- 15 min: Piste 1
- Soitto soi: Siivous, kierto
- 15 min: Piste 2
- Toista kunnes kaikki ovat käyneet kaikilla pisteillä (60 min yhteensä)
```

**Hyöty**: Opettaja antaa intensiivistä tukea yhdelle pienelle ryhmälle kun muut työskentelevät itsenäisesti

---

### Strategia 3: Visuaaliset ohjeet (Ei lukukykyä vaadita)

**Tehtävämonisteen ohjeet vain kuvilla**:

```
Väritysohjeet:
[Kuva: Värikynä] → Väritä
[Kuva: Omena väritetty punaiseksi] → Väritä omena punaiseksi
[Kuva: Puu väritetty vihreäksi] → Väritä puu vihreäksi
[Kuva: Hymynaama] → Pidä hauskaa!

Tulos: Oppilaat seuraavat ohjeita ilman lukemista
```

## Yhteydenpito vanhempiin varhaiskasvatuksessa

**Vanhempien huoli**: "Onko lapseni valmis esikouluun/kouluun?"

**Vastaus**: Jaa työnäytteitä (näytä kehitys)

### Esikouluvalmius-portfolio

**Sisältö** (lähetetään kotiin toukokuussa):
```
Syyskuun työnäytteet:
- Ensimmäinen väritystehtävä (hienomotoriikan lähtötaso)
- Ensimmäinen viivojen piirto (hallinnan lähtötaso)

Toukokuun työnäytteet:
- Viimeaikainen väritystehtävä (näe kehitys!)
- Viimeaikainen sokkelo (monimutkainen visuaalis-motorinen integraatio)

Opettajan huomio:
"Katsokaa kuinka paljon [Lapsi] on kasvanut! Syyskuussa väritys oli haastavaa. Nyt [Lapsi] värittää tarkasti ja nauttii siitä. Tämä osoittaa, että [Lapsi] on valmis esikoulun/koulun kirjoitukseen!"

Taidot hallittu tänä vuonna:
✓ Kynäote (kolmisormiote)
✓ Saksitaidot (leikkaa viivojen mukaan)
✓ Kirjaintuntemus (tunnistaa 20/26 kirjainta)
✓ Lukujen ymmärrys (laskee 20:een)
✓ Kuvioiden tunnistus (AB- ja ABC-kuviot)

[Lapsi] ON VALMIS esikouluun/kouluun!
```

**Vanhempien palaute**: 96% sanoo portfolion auttavan näkemään lapsen kehityksen (vs pelkkä suullinen raportti)

## Vuodenaikaiset teemat 3-6-vuotiaille

**Pienet lapset**: Oppivat teemojen kautta (konkreettiset yhteydet)

**Syyskuu/Lokakuu: Syksy/Omenat**
```
Väritys: Omenat, kurpitsat, lehdet
Kuvabingo: Syyssanasto
Kuviojuna: Syksyn värit (punainen, oranssi, keltainen -kuvio)
Etsi ja löydä: Etsi syksyaiheisia esineitä
```

**Marraskuu: Kiitollisuus**
```
Väritys: Kalkkuna, piirakka, maissi
Kuvabingo: Ruuat, joita syömme juhlapöydässä
Kuvien lajittelu: Asiat, joista olemme kiitollisia
```

**Joulukuu: Talvi/Juhla-aika**
```
Väritys: Lumiukko, lapaset, lahjat
Viivojen piirto: Auta joulupukkia pääsemään taloihin
Kuviojuna: Piparkakku-raidat (punainen-valkoinen-punainen -kuvio)
```

**Tammikuu: Talven eläimet**
```
Väritys: Jääkarhu, pingviini, hylje
Kuvabingo: Arktisten alueiden eläimet
Etsi ja löydä: Etsi talven eläimiä
```

**Helmikuu: Ystävänpäivä/Ystävyys**
```
Väritys: Sydämet, kukat
Kuvien lajittelu: Ystävälliset vs epäystävälliset teot
Kuvabingo: Tunteet
```

**Maaliskuu: Kevät/Kasvit**
```
Väritys: Kukat, perhoset
Etsi ja löydä: Etsi kasvavia asioita
Yhteenlasku: Laske kukkien terälehtiä (kuvamuoto)
```

**Huhtikuu: Eläimet/Maatila**
```
Väritys: Maatilan eläimet
Kuvabingo: Eläinten äänet
Kuvien lajittelu: Maatila vs eläintarha -eläimet
```

**Toukokuu: Kesä/Ranta**
```
Väritys: Aurinko, rantalelut
Viivojen piirto: Sokkelot (monimutkainen lopun vuoden taso)
Kuviojuna: Rantateemaisia kuvioita
```

**Hyöty**: Temaattinen oppiminen tarjoaa kontekstin (ei irrallisia taitoja)
