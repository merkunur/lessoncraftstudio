## Johdanto: Tuloste vai digitaalinen – väärä vastakkainasettelu

**Todellisuus luokkahuoneessa**: Useimmat opettajat hyödyntävät MOLEMPIA muotoja (hybridiote)

**Tulostettujen tehtävämonisteiden edut**:
- Toimii aina (ei teknisiä ongelmia)
- Hienomotorinen harjoittelu (käsinkirjoitus, leikkaaminen, värittäminen)
- Ei ruutuaikaa (vanhempien toive)
- Portfolio-arviointi (konkreettiset työnäytteet)

**Digitaalisten tehtävämonisteiden edut**:
- Ei tulostuskustannuksia (säästää paperia ja mustetta)
- Välitön jakelu (Google Classroom)
- Etäopetus valmiina (varautuminen poikkeustilanteisiin)
- Saatavilla kaikkialla (poissaoleva oppilas saa tehtävät kotona)

**Tutkimus** (Singer & Alexander, 2017): Tulostetun ja digitaalisen materiaalin välillä ei ole merkittävää eroa alakouluikäisten ymmärtämisessä (molemmat tehokkaita)

**Keskeinen oivallus**: Valitse muoto tehtävätyypin ja opetustilanteen mukaan (ei ideologian perusteella)

## Milloin tulostaa

### Käyttötilanne 1: Hienomotoriikan kehitys

**Tehtävät, jotka vaativat fyysistä käsittelyä**:
- **Kuviojuna** (leikkaaminen ja liimaaminen)
- **Värittäminen** (värikynnän paineen hallinta)
- **Viivanveto** (kynäotteen harjoittelu)
- **Jäljentäminen** (kirjainten muodostaminen)

**Miksi tulostaminen on välttämätöntä**:
```
Digitaalinen värittäminen (tabletti):
- Kosketuspainallus (ei paineen vaihtelua)
- Ei otteen kehitystä
- Ei kahden käden yhteistyötä
Tulos: Hienomotoriset hyödyt jäävät saamatta

Tulostettu värittäminen:
- Kolmipisteote väriliidulla
- Paineen kontrolli (käden voima)
- Paperin tukeminen toisella kädellä (kahden käden yhteistyö)
Tulos: Täysi hienomotorinen kehitys
```

**Suositus**: Tulosta aina hienomotoriset tehtävät (opetus + kehitys)

---

### Käyttötilanne 2: Matematiikan laskuharjoitukset

**Tutkimus** (Mueller & Oppenheimer, 2014): Käsinkirjoitus parantaa oppimista **23%** verrattuna näppäimistöllä kirjoittamiseen

**Miksi tulostaa matematiikkaan**:
```
Digitaalinen matematiikka (kirjoita vastaukset):
- 7 + 8 = [kirjoita: 15]
- Kognitiivinen prosessi: Hae vastaus, kirjoita
- Lihasmuisti: Minimaalinen

Tulostettu matematiikka (käsinkirjoitus):
- 7 + 8 = ___ [kirjoita: 15]
- Kognitiivinen prosessi: Hae vastaus, suunnittele numeron muoto, kirjoita
- Lihasmuisti: Vahvempi (lihakset koodaavat vastausta)
Tulos: Syvempi oppiminen käsinkirjoituksen kautta
```

**Suositus**: Tulosta matematiikan tehtävät (käsinkirjoitus syventää oppimista)

---

### Käyttötilanne 3: Arvioinnin dokumentointi

**HOJKSin/tehostetun tuen vaatimus**: Pysyvät työnäytteet

**Tulosteen etu**:
```
Tulostettu arviointi:
- Fyysinen tehtävä portfoliossa
- Voi tehdä merkintöjä (opettajan huomiot: "laski sormilla")
- Näytettävissä vanhemmille (vanhempainilta)
- Virallinen dokumentointi (HOJKSin edistymisen todiste)

Digitaalinen arviointi:
- Voi poistua vahingossa
- Vaikeampi huomioida
- Vähemmän konkreettinen vanhempainilloissa
Tulos: Tuloste parempi viralliseen arviointiin
```

**Suositus**: Tulosta kaikki viralliset arvioinnit (dokumentointivaatimus)

---

### Käyttötilanne 4: Yhteydenpito kotiin

**Vanhempien toive**: 78% suosii fyysisiä kotitehtäviä (vs sähköposti/Google Classroom)

**Miksi vanhemmat suosivat tulosteita**:
```
Tulostettu kotitehtävä:
- Näkyvillä (keittiön pöydällä, vanhempi näkee sen)
- Konkreettinen (voi valvoa tekemistä)
- Ei kirjautumista (vanhempi ei tarvitse Google-tiliä)

Digitaalinen kotitehtävä:
- Piilossa (Google Classroomissa, vanhempi ei ehkä tarkista)
- Abstrakti (vanhempi ei ole varma, onko tehty)
- Kirjautumiseste (vanhempi tarvitsee salasanan, unohtaa)
Tulos: Tuloste parempi vanhempien osallistamiseen
```

**Suositus**: Tulosta tärkeät kotitehtävät (yhteydenpito kotiin)

## Milloin digitaalinen

### Käyttötilanne 1: Sanahaun sanastoharjoitukset (toistuva harjoittelu)

**Miksi digitaalinen on järkevää**:
```
Sanastosanahaku:
- Taito: Sanan tunnistaminen (ei vaadi kirjoittamista)
- Hienomotoriikka: Minimaalinen (vain sanojen ympyröinti)
- Uudelleenkäytettävyys: Oppilas voi tehdä useita kertoja (eri sanalista)

Digitaalinen jakelu:
- Jaa Google Classroomissa
- Oppilas avaa Chromebookilla
- Voi korostaa digitaalisesti TAI tulostaa halutessaan
- Ei paperin hukkaa, jos oppilas ei tarvitse lisäharjoitusta
Tulos: Joustava, kustannustehokas
```

**Suositus**: Jaa sanastoharjoitukset digitaalisesti (tulostus valinnaista)

---

### Käyttötilanne 2: Nopeille tekijöille lisätehtävät

**Ongelma**: Nopeat tekijät valmiita 5 minuutissa, tarvitsevat rikastuttavia tehtäviä

**Digitaalisen edut**:
```
Tulostetut lisätehtävät:
- Pitää tulostaa 30 kopiota etukäteen (hukkaa, jos vain 5 oppilasta ehtii)
- Säilytysongelmia (mihin laittaa 30 kryptogrammikopiota?)

Digitaaliset lisätehtävät:
- Jaa 3 haastastastavuustasoa Google Classroom -kansioon
- Vain aikaisin valmiit oppilaat käyttävät (ei hukkaa)
- Rajaton varasto (ei "tehtävät loppuivat")
- Oppilas tulostaa vain halutessaan
Tulos: Ei hukkaa, aina saatavilla
```

**Suositus**: Pidä lisätehtävät digitaalisina (tulostus tarvittaessa)

---

### Käyttötilanne 3: Etä-/poissaolevan oppilaan tehtävät

**Tilanne**: Oppilas poissa (sairas, lääkäriaika, perhesyy)

**Digitaalinen ratkaisu**:
```
Poissaoleva oppilas:
- Ei voi saada tulostetta (ei luokassa)
- Google Classroom: Jaa päivän tehtävät
- Poissaoleva oppilas: Pääsee käsiksi kotoa
- Tekee digitaalisesti TAI tulostaa kotona
- Palauttaa Google Classroomin kautta (opettaja arvioi etänä)
Tulos: Ei oppimisen menetystä poissaolon vuoksi
```

**COVID-opetus**: Digitaalinen jakelu = valmius poikkeustilanteisiin (voidaan siirtyä etäopetukseen yhdessä yössä)

**Suositus**: Jaa AINA tehtävät Google Classroomiin (vaikka tulostaisikin) = poissaolevat pääsevät käsiksi

---

### Käyttötilanne 4: Eriyttäminen ilman leimaamista

**Ongelma**: Eriytetyt tulostetut tehtävät = näkyvät erot (oppilaan nolostuminen)

**Digitaalisen edut**:
```
Tulostettu eriyttäminen:
- Vaikea oppilas saa "helpomman" tehtävän (5 tehtävää)
- Keskitason oppilas saa tavallisen (15 tehtävää)
- Edistynyt oppilas saa haastavan (25 tehtävää)
- Oppilaat NÄKEVÄT erot (mahdollinen nolostuminen)

Digitaalinen eriyttäminen:
- Opettaja antaa eri versiot Google Classroomin kautta (yksilölliset tehtävät)
- Jokainen oppilas näkee vain OMAN tehtävänsä
- Ei näkyviä vertailuja
- Oppilaat: "Me kaikki teemme matematiikan tehtävää"
Tulos: Eriyttäminen ilman leimaamista
```

**Suositus**: Käytä digitaalista eriytettyihin tehtäviin (yksityisyyden suoja)

## Hybridistrategia (molempien edut)

**Suositus**: Hyödynnä MOLEMPIA muotoja strategisesti

**Esimerkki viikoittaisesta aikataulusta**:

**Maanantai** (Tuloste):
- Matematiikan tehtävä (käsinkirjoitusharjoitus)
- Aamupäivän aloitustehtävä (siirtymärutiini, tuloste pulpeteissa)

**Tiistai** (Digitaalinen):
- Sanastosanahaku (jaa Google Classroomissa, tulostus valinnaista)
- Nopeille tekijöille ristikko (vain digitaalinen)

**Keskiviikko** (Tuloste):
- Kuviojuna (leikkaus/liimaus, fyysinen käsittely vaaditaan)
- Arviointitesti (dokumentointivaatimus)

**Torstai** (Digitaalinen):
- Luonnontieteen sanastoristikko (ei vaadi käsinkirjoitusta)
- Yhteiskuntaopin sanahaku (voi tehdä digitaalisesti)

**Perjantai** (Tuloste):
- Värittäminen (hienomotoriikka, rauhoittava toiminta)
- Kotitehtävä (yhteydenpito kotiin)

**Tulos**: 40% tulosteita, 60% digitaalista (kustannus-hyötyjen tasapaino)

## Paperin säästöstrategiat

**Ongelma**: Tulostuskustannukset kasvavat (paperi + muste kallista)

**Kustannusanalyysi**:
```
Perinteinen: 30 oppilasta × 5 tehtävää/päivä × 180 päivää = 27 000 sivua/vuosi
Kustannus: 135€ paperi + 400€ muste = 535€/vuosi tulostuskustannuksia
```

**Paperin säästöstrategiat**:

**Strategia 1: Kaksipuolinen tulostus**
- Vähentää paperinkäyttöä 50%
- Säästö: 67,50€ paperi + 200€ muste = 267,50€/vuosi

**Strategia 2: Valikoiva tulostaminen**
- Tulosta vain käsinkirjoitusta vaativat (matematiikka, arviointi)
- Digitaalinen sanahakulle, ristikoille, lisätehtäville
- Vähentää paperinkäyttöä 60%
- Säästö: 81€ paperi + 240€ muste = 321€/vuosi

**Strategia 3: Tussitaskut**
- Laita tulostettu tehtävä uudelleenkäytettävään tussitaskuun
- Oppilas tekee tusseilla
- Pyyhi, käytä uudelleen 100+ kertaa
- **Kertaluonteinen hinta**: 50€ (30 tussitaskua)
- **Vuotuinen säästö**: 435€ (taskujen hinnan jälkeen)
- **Vuosi 2+**: 535€ säästö (taskut kestävät vuosia)

**Strategia 4: Vastaa vihkoon**
- Tulosta YKSI kopio, näytä dokumenttikameralla
- Oppilaat kopioivat tehtävät vihkoon, ratkaisevat
- Käytä samaa kopiota koko päivän (tai viikon)
- Säästö: 99% kopioista (30 oppilasta = 1 kopio 30:n sijaan)

## Google Classroom -integraatio

**Alusta**: Vie PDF:ksi → lataa Google Classroomiin

**Aloitus** (kertasuoritus, 15 minuuttia):
```
1. Luo Google Classroom -luokka
2. Luo aiheet:
   - Matematiikan harjoitukset
   - Sanasto
   - Nopeille tekijöille haasteet
   - Rikastuttavat tehtävät
   - Arviointimateriaalit
3. Jaa luokkakoodi oppilaille
```

**Viikoittainen työnkulku** (10 minuuttia):
```
1. Luo 15 tehtävää (10,5 min generaattoreilla)
2. Vie kaikki PDF:ksi (sisältyy luontiaikaan)
3. Lataa Google Classroomiin:
   - Matematiikan tehtävät → Matematiikan harjoitukset -aihe
   - Sanahaut → Sanasto-aihe
   - Kryptogrammit → Nopeille tekijöille haasteet -aihe
   Latausaika yhteensä: 5 minuuttia (erälataus)
```

**Oppilaan pääsy**:
```
Oppilas kirjautuu Google Classroomiin:
- Näkee kaikki saatavilla olevat tehtävät
- Valitsee mitä tehdä (tai opettaja antaa tietyt)
- Lataa tehdäkseen digitaalisesti TAI tulostaa kotona
- Lähettää kuvan tehdystä työstä (jos vaaditaan)
```

**Vanhemman pääsy**:
```
Vanhempi: "Mitä kotitehtäviä lapsellani on?"
Opettaja: "Kaikki on Google Classroomissa, vanhempien välilehti"
Vanhempi: Kirjautuu, näkee kaikki tehtävät
Tulos: Läpinäkyvä viestintä
```

## Saavutettavuusnäkökulmat

**Digitaalisen edut erityisoppilaille**:

**Lukivaikeus**: Teksti puheeksi
```
Digitaalinen PDF:
- Oppilas avaa laitteella
- Aktivoi teksti puheeksi -toiminnon (lukee vihjeet ääneen)
- Tukee lukivaikeutta ilman erityisvalmistelua
```

**Näkövamma**: Zoomaus
```
Digitaalinen PDF:
- Oppilas zoomaa tekstin 200-300%
- Suurempi kuin tulosteella mahdollista (ilman isokokoisen tulosteen tekoa)
```

**Hienomotoriset vaikeudet**: Kirjoittaminen näppäimistöllä käsinkirjoituksen sijaan
```
Digitaalinen tehtävä:
- Oppilas kirjoittaa vastaukset (ei tarvitse kyniä)
- Tukee hienomotorisia haasteita
- Harjoittaa silti akateemista taitoa (ilman motorista estettä)
```

**Suositus**: Tarjoa digitaalinen vaihtoehto tukitoimia tarvitseville oppilaille

## Ympäristövaikutukset

**Kestävyysnäkökulma**: Paperinkulutus

**Vuotuinen paperinkäyttö** (30 oppilaan luokka, perinteinen tulostus):
```
27 000 sivua/vuosi ÷ 500 sivua/riiisi = 54 riisiä/vuosi
54 riisiä = ~6 puuta/vuosi (1 puu = ~8000 sivua)
```

**60% digitaalinen strategia**:
```
10 800 tulostettua sivua + 16 200 digitaalista = 27 000 yhteensä
10 800 sivua = 21,6 riisiä/vuosi
21,6 riisiä = ~2,4 puuta/vuosi
Säästö: 3,6 puuta/vuosi luokkaa kohti
```

**Koko koulun vaikutus** (30 luokkaa):
```
Perinteinen: 30 luokkaa × 6 puuta = 180 puuta/vuosi
Hybridi (60% digitaalinen): 30 luokkaa × 2,4 puuta = 72 puuta/vuosi
Säästö: 108 puuta/vuosi koko koulussa
```

**Ympäristökasvatuksen mahdollisuus**: Opeta oppilaille paperin säästämisestä
