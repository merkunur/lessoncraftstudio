## Introduction : Évaluer POUR Apprendre ou Évaluer LES Acquis ?

**Deux approches d'évaluation distinctes** :
- **Évaluation sommative** : Mesurer LES acquis (contrôle final, ce que l'élève a retenu)
- **Évaluation formative** : Évaluer POUR apprendre (vérifications régulières, ajuster l'enseignement)

**Recherche fondamentale** (Black & Wiliam, 1998) : L'évaluation formative améliore la réussite scolaire de **0,4 à 0,7 écart-type** (équivalent à passer du 50e au 66e-77e percentile)

**Le problème de l'évaluation traditionnelle** :
```
Semaines 1-3 : Enseigner l'addition
Semaine 4 : Contrôle (évaluation sommative)
Résultat : Découvrir lors du contrôle que 30% des élèves n'ont pas compris
Problème : Trop tard - on passe déjà à l'unité suivante
Conséquence : Les élèves accumulent des lacunes (effet boule de neige)
```

**La solution avec l'évaluation formative** :
```
Semaine 1 Jour 1 : Présenter le concept d'addition
Semaine 1 Jour 2 : Vérification rapide (5 problèmes)
Semaine 1 Jour 2 (après-midi) : Analyser les résultats
Semaine 1 Jour 3 : Reprendre avec les élèves en difficulté (remédiation)
Semaine 1 Jour 4 : Nouvelle vérification (suivi de progression)
Résultat : Identifier les incompréhensions PENDANT l'apprentissage (prévenir les lacunes)
```

**Différence essentielle** : Formative = PENDANT l'enseignement (on peut ajuster), Sommative = APRÈS l'enseignement (trop tard)

## Système de Suivi Quotidien de la Progression

**Principe** : Des vérifications brèves quotidiennes (5-10 minutes) permettent de suivre l'apprentissage en temps réel

### La Méthode des 5 Exercices

**Structure** :
```
Mini-fiche quotidienne (générée en 42 secondes) :
- 5 exercices ciblant la compétence actuelle
- Élèves : 5 minutes de résolution
- Enseignant : 5 minutes de correction
- Temps total : 10 minutes (ajustement pédagogique inclus)
```

**Exemple : Semaine sur l'addition à 2 chiffres**

**Vérification lundi** :
```
5 exercices : addition à 2 chiffres sans retenue
Résultats :
- 20 élèves : 5/5 correct (maîtrise)
- 7 élèves : 3-4/5 correct (en cours d'acquisition)
- 3 élèves : 0-2/5 correct (non acquis)

Action :
- Mardi : Classe entière avance vers l'addition avec retenue
- Mardi petit groupe : 3 élèves en difficulté reprennent l'addition sans retenue
```

**Vérification mardi** :
```
5 exercices : addition à 2 chiffres AVEC retenue
Résultats :
- 15 élèves : 5/5 correct (prêts à progresser)
- 10 élèves : 2-4/5 correct (besoin d'entraînement)
- 5 élèves : 0-1/5 correct (difficultés importantes)

Action :
- Mercredi : 15 élèves en pratique autonome (enrichissement)
- Mercredi : Enseignant travaille avec le groupe de 10 élèves
- Enseignant spécialisé prend en charge les 5 élèves en grande difficulté
```

**Bénéfice** : Détecter les difficultés AVANT qu'elles ne deviennent des lacunes (enseignement réactif)

## Analyse des Profils d'Erreurs

**Au-delà du juste/faux** : Analyser COMMENT les élèves se trompent

### Profils d'Erreurs en Mathématiques

**Erreurs dans l'addition avec retenue** :

**Profil 1 : Oubli de la retenue**
```
  47
+ 28
----
  65  (incorrect - l'élève a calculé 7+8=15, écrit 5, oublié de reporter 1)

Diagnostic : Comprend l'addition, oublie l'étape de la retenue
Remédiation : Pratique visuelle de la retenue (montrer explicitement le report)
```

**Profil 2 : Retenue mal placée**
```
  47
+ 28
----
  75  (incorrect - l'élève a reporté 1 correctement mais l'a ajouté au mauvais endroit)

Diagnostic : Comprend le besoin de retenue, erreur d'exécution
Remédiation : Révision des valeurs positionnelles (quelle colonne représente quoi ?)
```

**Profil 3 : Erreurs aléatoires (pas de profil)**
```
Parfois correct, parfois faux (incohérent)

Diagnostic : Erreurs d'inattention OU surcharge de la mémoire de travail
Remédiation : Réduire le nombre d'exercices (moins à la fois), stratégie de vérification
```

**Avantage du générateur** : Créer des pratiques ciblées pour des types d'erreurs spécifiques
```
Générer une fiche : 20 problèmes nécessitant spécifiquement une retenue
Objectif : L'élève oublie-t-il TOUJOURS la retenue, ou seulement parfois ?
Données : Si 0/20 correct = lacune de compétence, si 12/20 correct = problème d'inconsistance
```

## Suivi de Progression dans le Cadre d'une RASED ou d'un PPRE

**Dispositif à 3 paliers** :

**Palier 1** : Enseignement de base (tous les élèves)
**Palier 2** : Intervention ciblée (élèves en difficulté, 20-30% de la classe)
**Palier 3** : Intervention intensive (difficultés importantes, 5-10% de la classe)

---

### Protocole de Suivi Palier 2

**Fréquence** : Vérifications hebdomadaires (suivre l'efficacité de l'intervention)

**Exemple : Élève en difficulté avec les tables de multiplication**

**Semaine 1 référence** :
```
Générer : 25 faits de multiplication (0-10)
Score de l'élève : 8/25 correct (32%)
Décision : Qualification pour intervention Palier 2
```

**Semaines 2-8 : Intervention + suivi hebdomadaire**
```
Semaine 2 : Nouvelle fiche (25 faits, même compétence)
            Score : 10/25 (40%) - légère amélioration
            Continuer l'intervention

Semaine 3 : Score : 12/25 (48%) - progression continue
            Continuer l'intervention

Semaine 4 : Score : 11/25 (44%) - légère baisse
            Ajuster l'intervention (essayer une stratégie différente)

Semaine 5 : Score : 15/25 (60%) - amélioration après ajustement
            Continuer l'intervention

Semaine 6 : Score : 17/25 (68%) - progression régulière
            Continuer l'intervention

Semaine 7 : Score : 20/25 (80%) - approche de la maîtrise
            Continuer l'intervention (presque acquis)

Semaine 8 : Score : 22/25 (88%) - maîtrise atteinte !
            Décision : Sortie du Palier 2, retour au Palier 1 uniquement
```

**Valeur du générateur** : Créer une nouvelle évaluation chaque semaine (pas de mémorisation, mesure réelle de la compétence)

**Documentation** : 8 fiches dans le dossier (preuve de progression pour l'équipe RASED/PPRE)

---

### Protocole de Suivi Palier 3

**Fréquence** : Deux fois par semaine (suivi intensif)

**Évaluations plus courtes** : 10 problèmes (moins écrasant)

**Exemple : Élève avec lacunes importantes en mathématiques**

**Semaine 1** :
```
Lundi : 10 additions simples à 1 chiffre
        Score : 3/10 (30%)

Jeudi : 10 additions simples (problèmes différents)
        Score : 4/10 (40%) - progression dans la semaine !
```

**Semaine 2** :
```
Lundi : Score : 5/10 (50%)
Jeudi : Score : 6/10 (60%)
Tendance : Croissance régulière
```

**Bénéfice** : Les données bi-hebdomadaires montrent que l'intervention fonctionne (ou ne fonctionne pas si stagnation)

## Cycle d'Enseignement Piloté par les Données

**Principe** : Utiliser les données d'évaluation pour ajuster l'enseignement en continu

### Le Cycle Hebdomadaire

**Vendredi** : Analyser les données de la semaine
```
Examiner toutes les vérifications quotidiennes à 5 exercices :
- Quels élèves ont maîtrisé la compétence ? (prêts à avancer)
- Quels élèves ont besoin de plus de pratique ? (presque acquis)
- Quels élèves n'ont pas compris ? (besoin de réenseignement)
```

**Dimanche** : Planifier la semaine suivante selon les données
```
Groupe 1 (maîtrise) : Générer des fiches d'enrichissement (application avancée)
Groupe 2 (en cours) : Générer des fiches d'entraînement (même compétence, plus de répétitions)
Groupe 3 (difficulté) : Générer des fiches modifiées (simplifiées, avec étayage)
```

**Lundi-Vendredi** : Délivrer un enseignement différencié
```
Tous les groupes : Travaillent sur des supports de niveau approprié
Enseignant : Circule, apporte un soutien ciblé là où nécessaire
```

**Vendredi suivant** : Analyser à nouveau, regrouper si nécessaire

**Résultat** : Enseignement continuellement adapté aux besoins des élèves

## Stratégies de Correction Efficaces

**Problème** : Corriger 30 élèves quotidiennement = charge de travail importante

**Solutions** :

### Stratégie 1 : Auto-Correction des Élèves (avec corrigé)

**Processus** :
```
1. Élèves complètent la vérification à 5 exercices (5 min)
2. Enseignant projette le corrigé (1 min)
3. Élèves corrigent leur propre travail, marquent X sur les erreurs (2 min)
4. Élèves écrivent leur score en haut (30 secondes)
5. Remettre à l'enseignant

Enseignant : Parcourt rapidement les scores (2 min), note qui a besoin de soutien
Temps total : 10,5 minutes (vs 20+ minutes si l'enseignant corrige tout)
```

**Bénéfice** : Responsabilisation de l'élève (voir ses propres erreurs immédiatement), gain de temps

---

### Stratégie 2 : Correction par les Pairs (avec un camarade)

**Processus** :
```
1. Élèves complètent la fiche (5 min)
2. Échanger avec un camarade
3. Enseignant projette le corrigé
4. Camarades corrigent le travail l'un de l'autre (2 min)
5. Retour des fiches, discussion des erreurs (1 min)
6. Remettre à l'enseignant

Temps total : 8 minutes, discussion entre pairs incluse
```

---

### Stratégie 3 : Soumission Numérique (Espace Numérique de Travail)

**Processus** :
```
1. Élèves complètent la fiche (5 min)
2. Prendre en photo, soumettre sur l'ENT (2 min)
3. Enseignant corrige numériquement quand disponible (après la classe, temps de préparation)

Bénéfice : Pas de temps de classe consacré à la correction
```

## Planification des Interventions Basée sur les Données

**Les données orientent les décisions d'intervention**

### Scénario 1 : Difficulté de Classe Entière

**Données** : 80% de la classe a obtenu moins de 60% lors de la vérification de vendredi

**Diagnostic** : Enseignement inefficace (la majorité des élèves n'a pas compris)

**Action** :
- **Lundi** : Réenseigner TOUT le concept (approche différente)
- **Mardi** : Pratique (nouvelle fiche)
- **Mercredi** : Nouvelle vérification (évaluation à 5 exercices)
- **Résultat** : Si toujours faible, essayer UNE AUTRE méthode d'enseignement (demander de l'aide)

---

### Scénario 2 : Difficulté d'un Petit Groupe

**Données** : 5 élèves obtiennent systématiquement moins de 60%, le reste de la classe 85%+

**Diagnostic** : Petit groupe nécessite un soutien ciblé (pas un problème de classe entière)

**Action** :
- **Intervention en petit groupe** : 20 minutes quotidiennes avec l'enseignant spécialisé ou le RASED
- **Fiches modifiées** : Nombre réduit d'exercices (10 au lieu de 20), support visuel
- **Suivi quotidien de progression** : Suivre ces 5 élèves quotidiennement (vs hebdomadaire pour les autres)

---

### Scénario 3 : Élève Individuel en Grande Difficulté

**Données** : Un élève obtient 0-20% alors que tous les autres sont à 70%+

**Diagnostic** : Lacune importante de compétence OU autre facteur (vision, attention, traitement)

**Action** :
- **Signalement** : Réunion d'équipe éducative, évaluation possible pour adaptation pédagogique
- **Adaptation immédiate** : Supports modifiés, soutien individuel
- **Évaluation diagnostique** : Identifier OÙ la compréhension s'est interrompue (valeur positionnelle ? sens du nombre ?)
