## Introduksjon: Når sykdommen rammer klokka fem om morgenen

**Lærerens virkelighet**: Våkner syk og må skaffe vikar med få timers varsel (sykdom, akutt, nødsituasjon)

**Problemet**: Lage vikarplaner klokka halv seks mens du nesten ikke klarer å stå opp
```
06:00: Våkner med høy feber
06:15: Ringer etter vikar
06:30: Prøver desperat å skrive undervisningsopplegg (føler deg elendig)
07:00: Gir opp, legger igjen en lapp om "fri lesing"
Resultat: Bortkastet skoledag (vikaren vet ikke hva som skal undervises)
```

**Løsningen**: Ferdiglagde nødplaner (laget når du er frisk)

**Forskning** (Weems & Rogers, 2007): Kvalitetsplaner for vikar opprettholder **90% av normal læring** vs **40%** uten ordentlige planer

**Beste praksis**: Ha 3-5 komplette vikardager klare til enhver tid

## Vikarberedskap-permen

**Fysisk perm i klasserommet** (vikaren kan ta den med én gang)

### Innhold i permen

**Del 1: Klasseinformasjon**
```
- Klassekart (navngitte plasser)
- Dagsplan (tidspunkt for hver time)
- Regler og belønningssystem (klare forventninger)
- Elever med særskilte behov (tilpasninger)
- Beredskapsrutiner (brannøvelse, lockdown)
- Hjelpesomme elever (pålitelige elever vikaren kan spørre)
```

**Del 2: Dag 1 Komplett opplegg**
```
Morgen:
- 08:30-09:00: Morgenrutine + fravær (vikar gjør selvstendig)
- 09:00-09:30: Matteoppgave (30 oppgaver, fasit inkludert)
- 09:30-10:00: Finn ordet (ord fra gjeldende tema)

Formiddag:
- 10:00-10:30: Lesing (selvstendig lesing + oppgaver)
- 10:30-11:00: Friminutt
- 11:00-11:30: Fargelegging (rolig aktivitet)

Ettermiddag:
- 11:30-12:00: Lunsj
- 12:00-12:30: Naturfag finn-ordet (begreper fra nåværende enhet)
- 12:30-13:00: Kryssord (fagord)
- 13:00-13:30: Mattepuslespill (kritisk tenkning, engasjerende)
- 13:30-14:00: Valgfritt: Tegne linjer, fargelegging eller fri lesing

Hjemtid: 14:00

Melding til vikar: "Alle arbeidsark har fasit i Del 5. Elevene skal sjekke egne svar når de er ferdige."
```

**Del 3-5: Dag 2, 3, 4** (lignende struktur, ulike arbeidsark)

**Del 6: Fasiter** (alle arbeidsark)

## Selvforklarende arbeidsark

**Nøkkelprinsipp**: Vikaren trenger ikke å undervise nytt stoff

**Arbeidsark-egenskaper for vikarbruk**:

### Tydelige instruksjoner på arket

**Eksempel på matteoppgave-topptekst**:
```
Navn: _________________ Dato: _______

INSTRUKSJONER:
1. Løs hver plusseoppgave
2. Vis utregningen din
3. Sjekk svarene med fasiten (på lærerens pult)
4. Hvis du har mer enn 8 feil, spør vikaren om hjelp
5. Når du er ferdig, legg i innleveringsboksen

Forventet tid: 30 minutter
```

**Vikarvennlig**: Instruksjoner PÅ arket (vikaren trenger ikke forklare)

---

### Kjente aktivitetstyper

**Bruk arbeidsark elevene allerede kjenner**:
```
✓ Matteoppgaver (øves ukentlig - elevene kjenner formatet)
✓ Finn ordet (gjøres regelmessig - selvforklarende)
✓ Kryssord (kjent puslespilltype)
✓ Fargelegging (alle vet hvordan man farger)

✗ Nye spillregler (vikar kan ikke lære ukjent aktivitet)
✗ Komplekse prosjekter (krever lærer-veiledning)
✗ Gruppearbeid (atferdsutfordring for vikar)
```

## Klasseledelse for vikarer

**Utfordring**: Elever tester grenser med vikar

**Løsning**: Klare forventninger + engasjerende arbeid

### Ferdigskrevet melding til elevene

**Inkluder i vikarplanen**:
```
Kjære klasse,

Jeg er borte i dag, men jeg forventer at dere viser vikaren vår den samme respekten som dere viser meg. Dette forventer jeg:

✓ Følg alle klasseregler
✓ Fullfør alt arbeid som er tildelt
✓ Hjelp vikaren hvis de har spørsmål
✓ Vær vennlige og respektfulle

Jeg kommer til å spørre vikaren om rapport når jeg kommer tilbake. Elever som oppfører seg bra vil få [belønning]. Elever som ikke gjør det vil [konsekvens].

Ha en fin dag!
[Lærernavn]
```

**Ansvarlighet**: Elevene vet at læreren følger opp

---

### Engasjerende arbeid (forebygger kjedsomhet)

**Kjedsomhet = atferdsproblemer**

**Strategi**: Bland lett + utfordrende + gøy
```
Morgen: Standard matte (kjent, bygger selvtillit)
Før lunsj: Finn ordet (spillaktig, engasjerende)
Før lunsj: Fargelegging (beroligende, lite stress)
Etter lunsj: Kryssord (moderat utfordring)
Slutt av dag: Mattepuslespill ELLER valgfritt (motiverende avslutning)

Resultat: Elevene er engasjerte (mindre tid til upassende oppførsel)
```

## Faglig relevante vikarplaner

**Ideelt**: Vikarplaner repeterer nåværende pensum (ikke bare tidsfordriv)

### Lage tilpassede planer hver måned

**September (skolestart)**:
```
Lag vikarplaner basert på september-pensum:
- Matte: Pluss/minus repetisjon (ferdigheter fra september)
- Ordforråd: September staveord (i finn-ordet/kryssord)
- Naturfag: September fagbegreper (i finn-ordet)

Resultat: Selv med vikar øver elevene relevant innhold
```

**Oktober**:
```
Oppdater vikarplaner:
- Matte: Ganging (oktober-fokus)
- Ordforråd: Oktober-ord
- Naturfag: Oktober-enhet begreper

Erstatt september vikar-arbeidsark med oktober-versjoner
Tid: 10 minutter å generere 5 nye arbeidsark
```

**Løpende**: Oppdater vikarplaner månedlig (alltid aktuelt innhold)

## Differensiering i vikarplaner

**Utfordring**: Vikar kan ikke differensiere som vanlig lærer

**Løsning**: Ferdig-differensierte materialer

### Nivådelte oppgavekasser

**Sett opp på forhånd**:
```
Kasse 1 (Blå): Tilpassede arbeidsark
- 10 oppgaver i stedet for 20
- Bildestøtte
- For: Elever som strever

Kasse 2 (Grønn): Standard arbeidsark
- 20 oppgaver
- Klassenivå
- For: Elever på nivå

Kasse 3 (Rød): Utfordringsoppgaver
- 30 oppgaver
- Komplekse anvendelser
- For: Sterke elever

Melding til vikar: "Elevene vet hvilken fargekasse de skal ta fra. De velger passende nivå selv."

Resultat: Differensiering skjer automatisk (vikar trenger ikke ta beslutninger)
```

## Planlegging ved lengre fravær

**Scenario**: Lærer borte 2+ uker (operasjon, fødselspermisjon, familienødsituasjon)

### System for utvidet vikarperiode

**Uke 1-2 materialer**:
```
Generer 10 fulle skoledager (50+ arbeidsark)
Organisering:
- Mandag Uke 1 mappe (alt materiell)
- Tirsdag Uke 1 mappe
- ...
- Fredag Uke 2 mappe

Langtidsvikar: Åpner mandag-mappen, finner alt som trengs
Tid å lage: 2-3 timer (én helg for å forberede 2 uker)

Fordel: Lærer kan fokusere på bedring (ikke bekymre seg for klasserommet)
```

## Tren elevene for vikardager

**Forberedelse**: Lær elevene hvordan de skal oppføre seg med vikar

**September-opplæring**:
```
Lærer: "Noen ganger er jeg borte og vi får vikar. Dette forventer jeg:"

Rollespill:
- Elev-frivillig som "vikar"
- Klassen øver:
  * Følge instruksjoner fra vikar
  * Hente arbeidsark fra riktig kasse
  * Spørre vikar om hjelp på passende måte
  * Rydde opp ved slutten av dagen

Øvelsesdag: "I dag later vi som det er vikardag"
Elever: Følger nødplanen som øvelse
Resultat: Når ekte vikar kommer, kjenner elevene rutinen
```
