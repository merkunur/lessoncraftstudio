# Draw and Color App - Comprehensive Features Analysis
## Source: REFERENCE APPS/draw and color.html

## App Overview
Grid Drawing Worksheet Generator - Creates grid-based drawing worksheets where students replicate a picture by filling in cells based on clues.

---

## ACCORDION SECTION 1: Worksheet Setup (Lines 863-891)

### Grid Size Settings:
- **Rows**: 3-10 rows (line 866)
- **Columns**: 3-10 columns (line 868)
- Creates customizable grid for drawing

### Clue Cells Percentage:
- **Range**: 10-75% (line 870)
- Controls how many cells are pre-filled as clues
- Higher percentage = easier worksheet

### Mirror Clue Cells (Lines 874-886):
- **Options**: None (Random), Horizontal, Vertical (lines 876-878)
- Mirrors the clue cells for symmetrical patterns
- **Part to Reveal**: Choose which part shows clues (line 881-882)

### Name/Date Fields:
- **Include Name/Date Fields**: Checkbox option (line 887)
- Adds student name and date fields to worksheet

## CANVAS CONTROLS & FEATURES

### Zoom Controls (Lines 1049-1052):
- **Zoom Out**: Decrease view size
- **Zoom In**: Increase view size
- **Zoom Reset**: Return to 100%
- Percentage display shows current zoom level

### History Controls (Lines 1055-1056):
- **Undo**: Ctrl+Z keyboard shortcut
- **Redo**: Ctrl+Y keyboard shortcut
- Buttons enable/disable based on history

### Layer Controls (Lines 1008-1011):
- **Bring to Front**: Move selected object to top layer
- **Bring Forward**: Move selected object up one layer
- **Send Backward**: Move selected object down one layer
- **Send to Back**: Move selected object to bottom layer

### Lock/Unlock (Lines 1059-1063):
- **Unlock All**: Unlock all locked objects
- Lock individual objects to prevent editing
- Visual indicator when objects are locked
