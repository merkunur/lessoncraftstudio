# Big Small App - Features Analysis

## Source: REFERENCE APPS/big small.html

## App Overview
Big Small is a worksheet generator for size comparison exercises. Children learn to identify and compare objects by size (big, small, medium) and order them from smallest to largest or vice versa.

## PRICING
- **Tier:** Full Access ($240/year or $25/month)
- NOT in Core Bundle

## Exercise Types (5 types)
1. **Circle the small one** (findSmall) - Identify the smallest object
2. **Circle the big one** (findBig) - Identify the largest object
3. **Circle the medium one** (findMed) - Identify the medium-sized object (when 3 images)
4. **Number 1-2-3 small to big** (orderAsc) - Order objects from smallest to largest
5. **Number 1-2-3 big to small** (orderDesc) - Order objects from largest to smallest

## Exercise Settings
- **Number of exercises:** 1-10 per worksheet
- **Images per exercise:** 2 or 3 images
- **Image mode:**
  - Identical images (same image in different sizes)
  - Different images (various images in different sizes)
- **Answer indicators:** Optional circles/boxes for answers
- **Exercise numbers:** Optional numbering (1, 2, 3...)
- **Worksheet theme:** Auto-select images from theme if not manually picking

## Page Setup Features
- **Page sizes:** Letter Portrait, Letter Landscape, A4 Portrait, A4 Landscape, Square (1200x1200), Custom
- **Page color:** Custom color picker
- **Name/Date fields:** Optional header with student name and date lines
- **Background themes:** Theme-based backgrounds with opacity control
- **Border themes:** Theme-based decorative borders with opacity control

## Text Tools
- Add custom text anywhere on worksheet
- **Text color:** Custom color selection
- **Font size:** Adjustable (8+)
- **Font options:** Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana
- **Outline color:** Custom stroke color
- **Outline width:** 0-10 adjustable

## Image Library
- **3000+ child-friendly images**
- Theme-based organization
- Search functionality by keyword
- Select multiple images for exercises
- Preview selected images

## Custom Image Upload
- Multi-file upload support
- Accepts common image formats (JPEG, PNG, GIF)
- Session-based storage
- Combine with library images

## Editing Toolbar
- **Layers:** Bring to front, bring forward, send backward, send to back
- **Alignment:** Left, center, right, top, middle, bottom, distribute
- **Lock/Unlock objects**
- **Delete selected**
- **Zoom controls:** In, out, reset, percentage display
- **Undo/Redo:** Full history support (Ctrl+Z, Ctrl+Y)
- **Unlock All:** Quick unlock for all locked objects

## Download Options
- **Worksheet JPEG:** High-quality image download
- **Answer Key JPEG:** Answer key with correct answers marked
- **Worksheet PDF:** Print-ready PDF format
- **Answer Key PDF:** Answer key in PDF
- **Grayscale option:** Convert to grayscale for ink savings
