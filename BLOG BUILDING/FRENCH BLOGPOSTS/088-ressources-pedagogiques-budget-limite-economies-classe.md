## Introduction : La réalité budgétaire des enseignants

**Réalité terrain** : Un enseignant français dépense en moyenne **650€/an** de sa poche pour sa classe (enquête SNUipp-FSU, 2023)

**Répartition budgétaire typique** :
```
Dépenses courantes :
- Cahiers et fichiers consommables : 280€
- Matériel de manipulation : 140€
- Affichage et décoration : 90€
- Fournitures diverses : 140€
Total : 650€ sur budget personnel
```

**Problématique** : Les fichiers d'exercices sont à usage unique (impossibles à réutiliser)
```
Achat : 28 fichiers de maths × 10€ = 280€
Utilisation : Septembre-juin (élèves écrivent dedans)
Juin : Fichiers ramenés à la maison (inutilisables l'année suivante)
Année suivante : Rachat de 28 nouveaux fichiers (280€ à nouveau)
Coût sur 5 ans : 1 400€ en fichiers consommables
```

**Solution** : Génération numérique + impression = ressources réutilisables à l'infini
```
Coût : Abonnement 132€/an
Génération : Fiches illimitées (équivalent 90+ fichiers)
Réutilisation : Sauvegarde des PDF, réimpression gratuite année suivante
Coût sur 5 ans : 660€ (économie de 740€ vs fichiers)
```

## Stratégies de matériel réutilisable

**Réduire la consommation de papier**

### Pochettes effaçables

**Investissement ponctuel** : 45€ pour 28 pochettes
```
Fonctionnement :
1. Imprimer fiche (un seul exemplaire)
2. Insérer dans pochette effaçable
3. Élève complète au feutre effaçable
4. Effacer, réutiliser 100+ fois

Exemple :
Rituel maths du matin (fiche quotidienne 5 calculs) :
- Sans pochettes : 28 copies × 140 jours = 3 920 pages
- Avec pochettes : 28 copies une fois, réutilisation toute l'année = 28 pages
Économie papier : 3 892 pages (155,68€ à 0,04€/page)

Rentabilité : 45€ investis = 155,68€ économisés/an (retour 3,5×)
```

---

### Fiches plastifiées

**Pour activités fréquentes** :
```
Plastifier : Tables de multiplication, abécédaires, frises numériques
Coût : 18€ plastifieuse + 9€ pochettes = 27€
Durabilité : 5+ ans


Utilisation : Élèves utilisent feutres effaçables
Exemple : Fiche "Tracer les lettres"

- Coût par utilisation : 0,04€ ÷ 200 = 0,0002€ (quasi gratuit)
```

## Stratégies d'économie de papier

**Réduire coûts d'impression**

### Impression recto-verso

**Changement simple, économies importantes** :
```
Recto simple : 140 fiches = 140 pages
Recto-verso : 140 fiches = 70 pages
Économie : 70 pages par élève = 1 960 pages/classe = 78,40€/an

Réglage : Préférences imprimante → Impression recto-verso → "Bord court" (retournement)
```

---

### Projection + cahier

**Éliminer photocopies** :
```
Traditionnel : Imprimer 28 copies de la fiche
Alternative : Projeter UNE fiche au tableau, élèves recopient dans cahier

Exemple :
Problèmes mathématiques : Afficher 15 problèmes au vidéoprojecteur
Élèves : Recopient problèmes + réponses dans cahier de maths
Papier utilisé : 0 feuille

Quand approprié : Calculs, définitions vocabulaire, questions compréhension
Quand inapproprié : Schémas complexes, textes longs
```
