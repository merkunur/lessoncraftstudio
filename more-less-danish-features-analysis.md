# More Less App - Comprehensive Features Analysis (Danish Translation Target)

**App Name:** Compare Numbers Worksheet / More or Less Worksheet Maker
**File:** REFERENCE APPS/more less.html
**Purpose:** Number comparison worksheet generator for early math education
**Target Age:** Kindergarten to 2nd grade (Danish: 0. klasse to 2. klasse)

---

## CORE FUNCTIONALITY

This app creates **number comparison worksheets** where students compare quantities using greater than (>), less than (<), or equal to (=) symbols.
