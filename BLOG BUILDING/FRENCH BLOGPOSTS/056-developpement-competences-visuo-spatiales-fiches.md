## Introduction : Les compétences visuo-spatiales et la réussite en STIM

**Définition des compétences visuo-spatiales** : La capacité à manipuler mentalement, faire pivoter et comprendre les relations spatiales des objets.

**Composantes essentielles** :
- **Rotation mentale** : Visualiser un objet sous différents angles
- **Perception partie-tout** : Comprendre comment de petites parties forment un objet complet
- **Relations spatiales** : Saisir la position, la distance et la direction
- **Mémoire visuelle** : Mémoriser l'information visuelle avec précision
- **Perception figure-fond** : Distinguer un objet de son arrière-plan

**Corrélation avec les STIM** :
```
Résultat de recherche (Wai et al., 2009) :
Les capacités visuo-spatiales à 13 ans prédisent les carrières STIM à 33 ans
Corrélation : r = 0,52 (prédicteur PUISSANT)

Comparaison :
Capacités mathématiques → carrière STIM : r = 0,45
Capacités verbales → carrière STIM : r = 0,32
Capacités visuo-spatiales → carrière STIM : r = 0,52 (LE PLUS ÉLEVÉ)
```

**Pourquoi les compétences visuo-spatiales sont cruciales pour les STIM** :
- **Ingénierie** : Rotation mentale d'objets 3D (conception CAO)
- **Architecture** : Visualisation spatiale des bâtiments
- **Chimie** : Structures moléculaires tridimensionnelles
- **Physique** : Visualisation des forces et des trajectoires
- **Mathématiques** : Géométrie, graphiques, problèmes spatiaux

**Le problème de l'éducation traditionnelle** :
```
Focus scolaire :
- Compétences verbales : 60% du temps d'enseignement (lecture, écriture)
- Calcul mathématique : 30% (arithmétique, algèbre)
- Visuo-spatial : 10% (art, géométrie occasionnelle)

Résultat : Les compétences visuo-spatiales sont sous-développées (malgré leur fort pouvoir prédictif pour les STIM)
```

**Recherche** (Uttal et al., 2013) : Les compétences visuo-spatiales sont **hautement malléables** (peuvent être entraînées)
- Méta-analyse : L'entraînement spatial améliore les compétences de **47% en moyenne**
- **Transfert** : L'entraînement sur une tâche spatiale améliore d'autres tâches spatiales (généralisable)

**Solution** : Pratique systématique avec des fiches de travail visuo-spatiales

## Les 7 générateurs visuo-spatiaux

### Générateur #1 : Dessin sur quadrillage (App 024) ⭐ #1 POUR LE VISUO-SPATIAL

**Pourquoi le dessin sur quadrillage est L'outil ultime de développement spatial** :
- **Perception partie-tout** : Voir comment les cellules individuelles forment une image complète
- **Raisonnement proportionnel** : Petite cellule = petit espace de dessin (mise à l'échelle spatiale)
- **Systèmes de coordonnées** : La cellule B3 est comme (2,3) sur un plan cartésien
- **Mémoire spatiale** : Mémoriser le contenu de la cellule de référence pendant le dessin
- **Attention aux détails spatiaux** : Le placement exact est important

---

#### Compétences spatiales développées

**Compétence 1 : Mise à l'échelle proportionnelle**

**Concept** : Si la cellule de référence fait 2,5 cm et la cellule de dessin fait 5 cm, les caractéristiques doivent être agrandies 2×

**Le dessin sur quadrillage enseigne** :
```
Cellule de référence : Petit cercle (diamètre 0,6 cm)
Cellule de dessin : Espace plus grand (1,2 cm)
L'élève doit : Agrandir le cercle proportionnellement (dessiner un cercle de 1,2 cm de diamètre)
Raisonnement spatial : Comprendre les relations de taille
```

**Transfert aux STIM** :
- Dessins à l'échelle (plans architecturaux)
- Lecture de cartes (1 cm = 10 km)
- Diagrammes scientifiques (grossissement au microscope 400×)

---

**Compétence 2 : Intégration spatiale partie-tout**

**Concept** : Les parties individuelles disposées spatialement créent un tout

**Expérience du dessin sur quadrillage** :
```
Cellules individuellement : Semblent aléatoires (ligne courbe ici, point là)
Cellules assemblées : Forment un visage complet (œil, nez, bouche positionnés correctement)
L'élève apprend : L'arrangement spatial crée du sens
```

**Transfert aux STIM** :
- Schémas de circuits (composants positionnés correctement)
- Structures moléculaires (atomes en arrangement spatial)
- Diagrammes d'écosystèmes (organismes positionnés dans les couches d'habitat)

---

**Compétence 3 : Rotation mentale** (Mode miroir)

**Concept** : Visualiser un objet sous différentes orientations

**Dessin sur quadrillage en mode miroir** :
```
Standard : Copier le contenu de la cellule A1 dans la cellule de dessin A1 (sans rotation)
Retournement horizontal : Copier la cellule A1 dans la cellule J1, inversée gauche-droite (inversion mentale)
Retournement vertical : Copier la cellule A1 dans la cellule A10, inversée haut-bas
Les deux retournements : Rotation de 180° (défi spatial extrême)
```

**Recherche** (Uttal et al., 2013) : Entraînement à la rotation mentale → **amélioration spatiale de 47%**, se transfère à la géométrie mathématique (35% mieux)

**Paramètres** :
- Quadrillage : 7×7 à 10×10
- Mode miroir : Optionnel (défi avancé)
- Détection intelligente des cellules : Activée

**Temps d'activité** : 60-90 minutes

**Tarification** : Accès intégral uniquement (240$/an)

### Générateur #2 : Correspondance de quadrillage (App 027)

**Pourquoi la correspondance de quadrillage développe les compétences spatiales** :
- **Mémoire spatiale** : Mémoriser les détails des pièces du puzzle
- **Reconnaissance de rotation** : Identifier les pièces pivotées (rotation mentale)
- **Correspondance de position** : Trouver l'emplacement spatial correct
- **Comparaison visuelle** : Faire correspondre la pièce à plusieurs emplacements potentiels

**Défi spatial avec rotation** :
```
Sans rotation :
Pièce montre : Coin supérieur gauche de la pomme
Élève : Fait correspondre visuellement à la position correspondante du quadrillage (comparaison directe)

Avec rotation :
Pièce montre : Coin supérieur gauche de la pomme, PIVOTÉ de 90°
L'élève doit :
1. Faire pivoter mentalement la pièce à son orientation d'origine
2. Identifier de quelle partie de l'image il s'agit
3. Trouver la bonne position sur le quadrillage
Demande spatiale : 3× plus élevée
```

**Paramètres** :
- Quadrillage : 5×5 ou 6×6 (25-36 pièces)
- Rotation : Oui (avancé) ou Non (débutant)
- Complexité : Image très détaillée

**Temps d'activité** : 40-70 minutes

**Tarification** : Forfait de base ou Accès intégral

### Générateur #3 : Chemin illustré (Labyrinthe - App 011)

**Pourquoi les labyrinthes développent les compétences spatiales** :
- **Navigation spatiale** : Planifier un chemin dans un espace 2D
- **Cartographie mentale** : Garder une carte du labyrinthe en tête
- **Planification spatiale** : Anticiper, prévoir les impasses
- **Raisonnement directionnel** : Comprendre les relations haut/bas/gauche/droite

**Processus de planification spatiale** :
```
L'élève approche du labyrinthe :
1. Balayage visuel : Identifier le début, la fin, les chemins potentiels
2. Simulation mentale : "Si je vais à droite, puis en haut... c'est une impasse"
3. Planification spatiale : "Le chemin de gauche semble dégagé vers le point final"
4. Exécution : Tracer le chemin en suivant le plan spatial
5. Correction d'erreur : Rencontrer une impasse, replanifier spatialement
```

**Transfert aux STIM** :
- Recherche de chemin algorithmique (informatique)
- Conception de circuits (génie électrique)
- Navigation (géographie, physique)

**Paramètres** :
- Quadrillage : 5×5 à 7×7
- Difficulté : Moyenne-élevée (plusieurs mauvais chemins)

**Temps d'activité** : 15-25 minutes

**Tarification** : Forfait de base ou Accès intégral

### Générateur #4 : Correspondance d'ombres (App 009)

**Pourquoi la correspondance d'ombres développe les compétences spatiales** :
- **Perception figure-fond** : Séparer l'ombre de l'arrière-plan (spatial)
- **Constance de forme** : Reconnaître le même objet malgré le changement d'apparence (transformation spatiale)
- **Constance perceptive** : L'identité de l'objet reste malgré le changement visuel

**Raisonnement spatial** :
```
Tâche : Faire correspondre le chien à son ombre
Traitement spatial :
1. Identifier le contour du chien (bords spatiaux)
2. Scanner les ombres (recherche spatiale)
3. Superposition mentale : "Le contour du chien correspondrait-il à cette ombre ?" (superposition spatiale)
4. Correspondance trouvée (correspondance spatiale)
```

**Transfert aux STIM** :
- Dessins d'ingénierie (projections orthographiques)
- Astronomie (les ombres révèlent la forme de l'objet)
- Géométrie (congruence, similarité)

**Temps d'activité** : 15-20 minutes

**Tarification** : Forfait de base ou Accès intégral

### Générateur #5 : Cherche et trouve (App 026)

**Pourquoi la recherche visuelle développe les compétences spatiales** :
- **Balayage spatial** : Modèles de recherche systématiques (gauche→droite, haut→bas)
- **Mémoire spatiale** : Mémoriser les emplacements recherchés (ne pas rechercher à nouveau)
- **Attention spatiale** : Se concentrer sur une région spécifique tout en bloquant les distracteurs

**Stratégie de balayage spatial** :
```
Recherche désorganisée (inefficace) :
Élève : Regarde au hasard autour de la feuille, recherche les mêmes zones plusieurs fois
Résultat : Rate des cibles, perd du temps

Recherche spatiale systématique (efficace) :
Élève : Haut-gauche → haut-droite (rangée 1), rangée suivante gauche→droite, répéter
Résultat : Couverture complète, aucune cible manquée
Compétence spatiale : Balayage spatial organisé
```

**Paramètres** :
- Objets : 20-30
- Cibles : 8-12
- Stratégie enseignée : Balayage spatial systématique

**Temps d'activité** : 25-40 minutes

**Tarification** : Forfait de base ou Accès intégral

### Générateur #6 : Fiche de motifs (App 006) - MOTIFS SPATIAUX

**Pourquoi les motifs spatiaux développent les compétences** :
- **Séquençage spatial** : Comprendre l'ordre dans l'espace
- **Prédiction spatiale** : Anticiper le prochain arrangement spatial
- **Extraction de règle spatiale** : Identifier le principe organisateur

**Motifs spatiaux vs motifs numériques** :
```
Motif numérique : 2, 4, 6, 8 (séquence temporelle)
Motif spatial : ●■●■ (arrangement spatial)

Demande spatiale : Doit traiter position + identité simultanément
Résultat : Développe le raisonnement spatial
```

**Transfert aux STIM** :
- Pavages (géométrie)
- Structures cristallines (chimie)
- Séquences ADN (biologie - l'arrangement spatial compte)

**Temps d'activité** : 20-30 minutes

**Tarification** : Forfait de base ou Accès intégral

### Générateur #7 : Comparaison grand-petit (App 019)

**Pourquoi la comparaison de taille développe les compétences spatiales** :
- **Grandeur spatiale** : Comparer les dimensions spatiales
- **Taille relative** : Comprendre les relations spatiales comparatives
- **Estimation visuelle** : Estimer les différences spatiales

**Raisonnement spatial** :
```
Tâche : Quel éléphant est plus grand ?
Traitement spatial :
1. Mesurer l'angle visuel (combien d'espace l'objet occupe)
2. Comparer les étendues spatiales (éléphant A vs éléphant B)
3. Jugement spatial : A > B
Compétence : Comparaison de grandeur spatiale
```

**Transfert aux STIM** :
- Échelle (cartes, diagrammes)
- Mesure (estimation avant calcul)
- Proportions (rapports comme relations spatiales)

**Temps d'activité** : 10-15 minutes

**Tarification** : Forfait de base ou Accès intégral
