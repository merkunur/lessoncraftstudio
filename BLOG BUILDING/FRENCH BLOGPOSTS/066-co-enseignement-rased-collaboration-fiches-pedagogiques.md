## Introduction : Le défi de la coordination pédagogique

**Modèles de co-enseignement** : Enseignant titulaire + enseignant spécialisé = classe partagée

**Prévalence** : 60% des établissements scolaires pratiquent le co-enseignement (modèle inclusif)

**Défi de coordination** :
```
Enseignant titulaire : Planifie le programme, crée les supports
Enseignant spécialisé : Fournit les adaptations, différenciations
Problème : Deux enseignants, planification séparée → duplication d'effort OU désalignement
```

**Problème classique des fiches** :
```
Enseignant titulaire : Crée une fiche niveau classe (30 exercices, complexes)
Enseignant spécialisé : Doit adapter (réduire à 15 exercices, ajouter visuels)
Duplication : 2× travail pour essentiellement le même contenu
Résultat : Inefficacité
```

**Solution générateurs** : Une plateforme, différenciation instantanée

**Recherche** (Friend & Cook, 2017) : Un co-enseignement efficace nécessite **planification partagée** + **ressources communes** (réduit le temps de préparation de 40%)

## Modèles de co-enseignement & applications pratiques

### Modèle 1 : Un enseigne, un assiste

**Structure** : L'enseignant titulaire enseigne, l'enseignant spécialisé circule pour apporter du soutien

**Application pratique** :
```
Enseignant titulaire : Distribue une fiche de mathématiques (20 problèmes, niveau classe)
Enseignant spécialisé : Circule, identifie les élèves en difficulté
Stratégie : L'enseignant spécialisé distribue une version adaptée (10 problèmes, support visuel) aux élèves nécessitant une adaptation
```

**Flux de travail avec générateurs** :
```
Dimanche planification (10 minutes) :
Enseignant titulaire : Génère fiche mathématiques standard (20 problèmes, symboliques)
Enseignant spécialisé : Génère version adaptée (10 problèmes, mode images)
Les deux : Partagent via Google Drive

Lundi cours :
Enseignant titulaire : Distribue version standard à la plupart des élèves
Enseignant spécialisé : Distribue version adaptée aux élèves avec PPS/PAP (5-8 élèves)
Résultat : Différenciation sans réunions de planification séparées
```

**Temps économisé** : 30 minutes de réunion hebdomadaire (coordination possible via dossier partagé)

---

### Modèle 2 : Enseignement en ateliers

**Structure** : Classe divisée en ateliers, chaque enseignant anime un atelier

**Application pratique** :
```
Atelier 1 (Enseignant titulaire) : Mots croisés niveau classe
Atelier 2 (Enseignant spécialisé) : Mots croisés adaptés (moins de mots, indices visuels)
Atelier 3 (Autonomie) : Défis pour élèves rapides
Atelier 4 (Autonomie) : Manipulations concrètes
```

**Flux de travail avec générateurs** :
```
Enseignant titulaire génère :
- Atelier 1 mots croisés (15 mots, indices texte)
- Atelier 3 défis (cryptogrammes, sudoku avancés)

Enseignant spécialisé génère :
- Atelier 2 mots croisés (8 mots, indices images)
- Corrigés pour tous les ateliers

Temps : 12 minutes total (les deux enseignants en génération)
Ateliers prêts pour la semaine
```

---

### Modèle 3 : Enseignement parallèle

**Structure** : Classe divisée en deux, les deux enseignants enseignent le même contenu

**Application pratique** :
```
Groupe A (Enseignant titulaire) : Difficulté standard
Groupe B (Enseignant spécialisé) : Difficulté adaptée (groupe plus petit = plus de soutien)
```

**Avantage générateurs** : Même générateur, paramètres différents = versions parallèles instantanées

**Exemple** :
```
Leçon mathématiques : Pratique additions

Fiche Groupe A :
- Plage : 1-100
- Problèmes : 25
- Format : Symbolique uniquement
- Généré en : 42 secondes

Fiche Groupe B :
- Plage : 1-20
- Problèmes : 15
- Format : Mode images
- Généré en : 42 secondes

Préparation totale : 84 secondes pour deux versions différenciées
Les deux groupes pratiquent la même compétence (addition) au niveau approprié
```

---

### Modèle 4 : Enseignement alternatif

**Structure** : Enseignant titulaire enseigne grand groupe, enseignant spécialisé retire petit groupe pour pré-enseignement ou réenseignement

**Application pratique** :
```
Grand groupe (Enseignant titulaire) : Introduction nouveau contenu + pratique niveau classe
Petit groupe (Enseignant spécialisé) : Pré-enseignement avec fiche adaptée OU réenseignement avec fiche simplifiée
```

**Exemple de pré-enseignement** :
```
Vendredi (retrait enseignant spécialisé) :
- Enseignant spécialisé : Pré-enseigne vocabulaire semaine suivante (mots mêlés simplifié)
- Élèves : Exposés aux termes avant leçon lundi
- Résultat : Plus de succès lors de la leçon lundi en classe entière

Lundi (enseignant titulaire classe entière) :
- Enseignant titulaire : Enseigne vocabulaire (mots croisés standard)
- Élèves pré-enseignés : Déjà familiers avec les termes (confiance accrue)
```

## Applications RASED et dispositifs spécialisés

**RASED** : Réseau d'Aides Spécialisées aux Élèves en Difficulté - dispositif français de soutien spécialisé

**Organisation** : Élèves retirés de la classe pour 30-60 minutes de soutien ciblé

**Défi curriculaire** : Doit s'aligner avec le programme de la classe tout en fournissant un soutien spécialisé

---

### Coordination curriculaire RASED/ULIS

**Problème** : Enseignement RASED d'un contenu différent = élève manque instruction classe

**Solution** : Curriculum parallèle (mêmes thématiques, difficulté adaptée)

**Exemple** :
```
Classe ordinaire (Semaine du 15 octobre) :
- Mathématiques : Tables de multiplication 1-12
- Lecture : Étude roman CM1
- Sciences : Vocabulaire états de la matière

RASED (Même semaine) :
- Mathématiques : Tables multiplication 1-5 (plage simplifiée)
- Lecture : Compréhension adaptée (même roman, questions modifiées)
- Sciences : Mots mêlés états matière (même vocabulaire, activité différente)

Résultat : Élève apprend même contenu que ses pairs (juste support adapté)
```

**Flux de travail avec générateurs** :
```
Enseignant spécialisé : Coordination avec enseignants titulaires

Lundi matin :
- Enseignants titulaires : Déposent fiches semaine sur Google Drive partagé
- Enseignant spécialisé : Consulte fiches, note compétences enseignées
- Enseignant spécialisé : Génère versions adaptées (difficulté réduite)
- Temps total : 15 minutes (vs 2-3 heures création de zéro)
```

---

### Suivi progression objectifs PPS/PAP

**Objectif RASED/ULIS** : Instruction ciblée objectifs PPS (Projet Personnalisé de Scolarisation)

**Application générateurs** : Outil de suivi progression cohérent

**Exemple objectif PPS** : "L'élève résoudra des additions à un chiffre avec 80% de réussite"

**Suivi progression hebdomadaire** :
```
Semaine 1 : Génère fiche additions (20 problèmes, 1-10)
           Score élève : 12/20 (60%)

Semaine 2 : Génère NOUVELLE fiche additions (même compétence, problèmes différents)
           Score élève : 14/20 (70%)

Semaine 3 : Génère NOUVELLE fiche
           Score élève : 16/20 (80%) ✓ OBJECTIF ATTEINT

Documentation : 3 fiches dans dossier PPS (preuve de progression)
```

**Avantage générateurs** : Problèmes nouveaux chaque semaine (pas de mémorisation, évaluation vraie compétence)

**Structure** :
```

├── Mathematiques/
│   ├── Niveau_Standard/
│   └── Niveau_Adapte/
├── Lecture/
│   ├── Niveau_Standard/
│   └── Niveau_Adapte/
├── Evaluations/
│   ├── Evaluations_Standard/
│   └── Evaluations_Adaptees/
└── Corriges/
```

**Flux de travail** :
```
Dimanche (Enseignant titulaire) :
- Génère fiches niveau standard pour la semaine
- Téléverse dans dossiers "Niveau_Standard"
- Temps total : 15 minutes

Dimanche (Enseignant spécialisé) :
- Consulte téléversements enseignant titulaire
- Génère versions adaptées correspondant aux mêmes compétences
- Téléverse dans dossiers "Niveau_Adapte"
- Temps total : 15 minutes

Lundi-Vendredi :
- Les deux enseignants accèdent aux versions appropriées pour les élèves
- Pas de coordination quotidienne nécessaire (supports pré-préparés)
```

### Réunion planification hebdomadaire (raccourcie)

**Co-planification traditionnelle** : 90 minutes/semaine (discuter programme, créer supports, coordonner)

**Avec générateurs** : 30 minutes/semaine (discuter instruction uniquement, supports pré-générés)

**Temps économisé** : 60 minutes/semaine × 36 semaines = **36 heures/an par binôme co-enseignement**

**Ordre du jour réunion** (30 minutes) :
```
1. Révision progression élèves semaine dernière (10 min)
2. Discussion objectifs semaine suivante (10 min)
3. Attribution rôles (qui enseigne quoi) (5 min)
4. Confirmation stratégie différenciation (quels élèves reçoivent quels supports) (5 min)

Note : Supports déjà générés (pas discutés en réunion)
Résultat : Réunion centrée sur INSTRUCTION, pas préparation supports
```

## Meilleures pratiques inclusion scolaire

### Différenciation invisible

**Objectif** : Adapter pour élèves PPS/PAP sans stigmatisation

**Stratégie** : Utiliser même générateur, différences subtiles

**Exemple** :
```
Classe entière : "Tout le monde fait une fiche de mathématiques"

Réalité :
- 22 élèves : Fiche 25 problèmes (standard)
- 8 élèves PPS/PAP : Fiche 15 problèmes (adaptée)

Similarité visuelle :
- Même titre ("Pratique additions")
- Même format (problèmes en lignes)
- Même mise en page
- Seule différence : Nombre de problèmes (moins évident que "ADAPTÉ" tamponné dessus)

Résultat : Élèves PPS/PAP se sentent inclus (pas "différents")
```

---

### Conception universelle de l'apprentissage (CUA)

**Principe CUA** : Concevoir pour tous les apprenants dès le départ (pas adaptations a posteriori)

**Application générateurs** :
```
Au lieu de :
1. Créer fiche standard
2. Adapter pour élèves PPS/PAP (travail supplémentaire)

Approche CUA :
1. Générer 3 versions simultanément (15 min) :
   - Niveau 1 : Sous niveau classe (élèves en difficulté)
   - Niveau 2 : Niveau classe (majorité)
   - Niveau 3 : Au-dessus niveau classe (élèves avancés)

2. Élèves choisissent niveau OU enseignant attribue discrètement

Résultat : Tous les élèves ont supports appropriés, pas de stigmate "adapté"
```
