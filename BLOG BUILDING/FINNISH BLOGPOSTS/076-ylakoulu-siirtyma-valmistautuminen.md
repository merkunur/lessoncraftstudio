## Johdanto: Yläkoulun valmius­aukko

**Viidennen luokan todellisuus**: Alakoulun viimeinen vuosi (valmistautuminen yläkouluun)

**Yläkoulun erot**:
```
Alakoulu (1-6):
- Yksi luokanopettaja (kaikki aineet)
- Sama luokkahuone koko päivän
- Opettaja järjestää materiaalit
- Opettaja muistuttaa tehtävistä
- Jäsennelty tuki (käsi kädessä)

Yläkoulu (7-9):
- 6-7 aineenopettajaa
- Luokkahuoneen vaihto 45 min välein
- Oppilas hallitsee omat materiaalinsa
- Oppilas seuraa omia tehtäviään
- Itsenäinen vastuu (ui tai uppoa)
```

**Tutkimus** (Akos & Galassi, 2004): **40 % oppilaista** kamppailee yläkouluun siirtymisessä (organisointivaikeudet, arvosanojen lasku, lisääntynyt ahdistus)

**Siirtymän epäonnistumisen oireet**:
```
7-luokkalainen (syyskuu):
- Unohtaa kotitehtävät (jäänyt kaappiin)
- Unohtaa tehtävät (ei kirjannut muistivihkoon)
- Ei valmistautunut kokeeseen (ei opiskellut)
- Kadottaa materiaaleja (ei organisointijärjestelmää)
Tulos: Arvosanat laskevat, ahdistus kasvaa, tuntee olevansa ylikuormittunut
```

**Ratkaisu**: Opeta itsenäisyys VIIDENNEN LUOKAN AIKANA (ei siirtymän jälkeen)

## 5 kriittistä yläkouluvalmiustaitoa

**Taidot kehitettäväksi viidennen luokan aikana**:

1. Organisointi (materiaalien hallinta)
2. Ajanhallinta (tehtävien seuranta)
3. Opiskelutaidot (itsenäinen oppiminen)
4. Muistiinpanojen tekeminen (tiedon käsittely)
5. Oma-aloitteisuus (avun pyytäminen asianmukaisesti)

## Taito 1: Organisointijärjestelmät

**Tavoite**: Oppilas hallitsee omat materiaalinsa (ei opettajan muistutuksia)

### Mappijärjestelmän harjoittelu

**Syyskuun järjestelyt** (opettaja tarjoaa rakenteen):
```
Oppilaan mappi (3-tuumainen, 3-renkainen):
Osio 1: Matematiikka (välilehti, irtopaperia)
Osio 2: Äidinkieli (välilehti, irtopaperia)
Osio 3: Ympäristöoppi (välilehti, irtopaperia)
Osio 4: Historia (välilehti, irtopaperia)
Osio 5: Kotitehtävät (välilehti - kaikki tehtävät tänne)
Etutasku: Muistivihko + kynät/lyijykynät
Takatasku: Arvioidut työt kotiin vietäväksi

Opettaja: Tarkistaa mapit viikoittain (perjantaitarkastus)
```

**Tammikuun siirtymä** (vähennetään tukea):
```
Opettaja: "Tarkistan mapit nyt JOKA TOINEN perjantai"
Oppilaat: Täytyy ylläpitää järjestystä itsenäisesti 2 viikon ajan
Todellisuus: Joidenkin oppilaiden mapit menevät sekaisin (luonnollinen seuraus)
Opettaja: Sekaisilla mapeilla olevat oppilaat menettävät perjantain vapaa-ajan siivoamiseen
Tulos: Oppilaat oppivat, että organisointi on tärkeää
```

**Huhti-toukokuu** (täysi itsenäisyys):
```
Opettaja: Ei mappitarkastuksia
Oppilaat: Täysin vastuussa organisoinnista
Todellisuus: Toukokuuhun mennessä oppilailla on joko järjestelmä TAI he oppivat tarvitsevansa sitä yläkouluun
Etu: Parempi oppia 5. luokalla (turvallinen ympäristö) kuin 7. luokalla (ui tai uppoa)
```

---

### Työarkkien organisointiprotokolla

**Ongelma**: Irtonaiset työarkit katoavat

**Ratkaisu**: Välitön arkistointijärjestelmä

**Luokkahuoneen menettely**:
```
Opettaja jakaa työarkin:
Vaihe 1: Oppilas kirjoittaa nimen, päivämäärän (välittömästi)
Vaihe 2: Oppilas täyttää työarkin
Vaihe 3: Oppilas arkistoi oikeaan mappiosaan (ennen luokasta lähtöä)
Vaihe 4: Opettaja kerää TAI oppilas säilyttää viitteenä

Tulos: Työarkit eivät jää pöydälle (ei kadotettuja papereita)
```

**Generaattorin etu**: Yhdenmukainen muoto = helppo arkistointi
```
Kaikki matematiikan työarkit: Sama otsikko (MATEMATIIKKA - [Päivämäärä])
Kaikki sanastotyöarkit: Sama otsikko (SANASTO - [Päivämäärä])
Oppilaat: Tietävät heti, mihin osioon arkistoida
```

## Taito 2: Ajanhallinta ja tehtävien seuranta

**Tavoite**: Oppilas seuraa tehtäviä itsenäisesti

### Tehtäväkalenterin harjoittelu

**Syyskuu** (maksimituki):
```
Opettaja: Kirjoittaa tehtävät taululle (värikoodattuna aineen mukaan)
Oppilaat: Kopioivat kalenteriin samalla kun opettaja lukee ääneen
Opettaja: Kävelee ympäri, tarkistaa jokaisen kalenterin (varmistaa oikean kopioinnin)
Aika: 10 minuuttia päivässä (korkea tuki)
```

**Marraskuu** (kohtalainen tuki):
```
Opettaja: Kirjoittaa tehtävät taululle
Oppilaat: Kopioivat kalenteriin itsenäisesti
Opettaja: Pistotarkastaa 5 satunnaista kalenteria (ei kaikkia)
Aika: 7 minuuttia päivässä
Vastuu: Oppilaat vastuussa oikeasta kopioinnista
```

**Helmikuu** (vähäinen tuki):
```
Opettaja: Ilmoittaa tehtävät suullisesti (ei kirjoita taululle)
Oppilaat: Kirjoittavat kalenteriin suullisista ohjeista
Opettaja: Ei tarkastuksia (oppilaan vastuu)
Todellisuus: Jotkut oppilaat unohtavat kirjoittaa (luonnollinen seuraus)
Tulos: Oppilas oppii kalenterin tärkeyden (parempi 5. kuin 7. luokalla)
```

**Huhti-toukokuu** (yläkoulusimulaatio):
```
Opettaja: Julkaisee tehtävät vain Google Classroomiin (kuten yläkoulussa)
Oppilaat: Täytyy tarkistaa verkosta, siirtää kalenteriin
Opettaja: Nolla muistutuksia
Todellisuus: Jäljittelee yläkoulun odotuksia
Etu: Harjoitusympäristö (silti 5. luokan opettajan turvaverkko)
```

---

### Pitkäaikaisten tehtävien hallinta

**Haaste**: Moniviikkoiset projektit (ei päivittäisiä muistutuksia)

**Harjoitusjakso**:

**Projekti annettu**: 2 viikon aikataulu
```
Päivä 1: Opettaja ilmoittaa projektin eräpäivän 14 päivän päästä
Oppilaat: Kirjoittavat eräpäivän kalenteriin
Opettaja: "Jakakaa tämä osiin. Tehkää hieman joka ilta."
```

**Päivä 7**: Välitarkastus (puoliväli)
```
Opettaja: "Nostakaa käsi, jos olette ALOITTANEET projektinne"
Todellisuus: 30 % ei ole aloittanut (ups)
Opettaja: "Teillä on 7 päivää jäljellä. Aloittakaa tänä iltana!"
Opetus: Ei voi odottaa viime hetkeen
```

**Päivä 14**: Projekti erääntyy
```
Oppilaat palauttavat:
- 70 % valmis ajallaan (hienoa!)
- 20 % kiirehti eilisiltana (oppi läksyn: aloita aikaisemmin)
- 10 % keskeneräinen (iso opetus: parempi suunnittelu tarvitaan)

Opettaja: Purku ajanhallinnasta
Kysymys: "Mitä teette toisin ensi kerralla?"
Metakognitio: Oppilaat miettivät suunnittelustrategioita
```

## Taito 3: Opiskelutaitojen kehittäminen

**Tavoite**: Oppilas opiskelee itsenäisesti (ei vanhemman kysely­apua)

### Itsetestaamisjärjestelmä

**Perinteinen opiskelu** (tehoton):
```
Oppilas: Lukee muistiinpanot 3 kertaa
Todellisuus: Passiivinen kertaus (oppimisen illuusio)
Koepäivä: "Opiskelin mutta en muista!"
Ongelma: Ei aktiivista palauttamista
```

**Itsetestainen** (tehokas):
```
Oppilas luo harjoituskokeen muistiinpanoista
Oppilas tekee harjoituskokeen (muistiinpanot kiinni)
Oppilas tarkistaa vastaukset, yrittää virheelliset uudelleen
Todellisuus: Aktiivinen palauttaminen (vahvistaa muistia)
Koepäivä: "Muistan, koska testasin itseäni!"
```

**Työarkkisovellus**: Käytä generaattoreita opiskelumateriaaleihin

**Esimerkki: Sanakokeen valmistautuminen**

**Viikko 1**: Opettaja esittelee 20 sanastosanaa
**Viikko 2**: Koe perjantaina

**Oppilaan luomat opiskelumateriaalit**:
```
Maanantai: Oppilas luo sanaristikon (20 sanaa)
           Tekee sanaristikon (visuaalinen altistus 5-8 kertaa)

Tiistai: Oppilas luo ristikonarvoituksen (20 sanaa, määritelmät vihjeiksi)
         Tekee ristikonarvoituksen (määritelmien muistaminen)

Keskiviikko: Oppilas tekee muistikortteja (sana edessä, määritelmä takana)
             Itsetestaa (aktiivinen palauttaminen)

Torstai: Oppilas tekee harjoituskokeen (kaveri kysyy, tai vanhempi)
         Tunnistaa sanat joita ei vielä osaa (2-3 sanaa)
         Lisäharjoitus näille tietyille sanoille

Perjantai: OIKEA koe (itsevarma, hyvin valmistautunut)

Tulos: Itsenäinen opiskelu (ei vanhempaa tarvita)
```

**Opettajan rooli**: Opeta JÄRJESTELMÄ, oppilaat toteuttavat itsenäisesti

---

### Muistiinpanojen harjoittelu

**Yläkoulun todellisuus**: Opettajat luennoivat, oppilaat tekevät muistiinpanoja

**5. luokan valmistautuminen**: Jäsennelty muistiinpanoharjoitus

**Cornell-muistiinpanomenetelmä**:
```
Sivun asettelu:
┌─────────────────┬──────┐
│ Päämuistiinpanot│ Vihjt│
│                 │      │
│ (2/3 sivusta)   │ (1/3)│
│                 │      │
└─────────────────┴──────┘
│ Yhteenveto (alhaalla)  │
└────────────────────────┘

Tunnin aikana:
- Tee muistiinpanot Päämuistiinpanot-osioon (faktat, yksityiskohdat)

Tunnin jälkeen:
- Kirjoita kysymyksiä/avainsanoja Vihjeet-osioon
- Kirjoita 2-3 lauseen yhteenveto alareunaan

Opiskelussa:
- Peitä Päämuistiinpanot, yritä vastata Vihjeet-kysymyksiin
- Aktiivinen palauttaminen sisäänrakennettuna
```

**Harjoitusprotokolla**:
```
Viikoittainen muistiinpanojen tekeminen:
Opettaja pitää 10 minuutin miniluen­non (ympäristöoppi, historia)
Oppilaat: Tekevät Cornell-muistiinpanot
Opettaja: Kerää, tarkistaa täydellisyyden
Palaute: "Hyvä yhteenveto!" TAI "Lisää yksityiskohtia muistiinpanoihin"

Tulos: Toukokuuhun mennessä oppilaat tottuneet tekemään muistiinpanoja luennosta
```

## Taito 4: Kokeen valmistautumisen itsenäisyys

**Tavoite**: Oppilas valmistautuu kokeisiin ilman vanhemman apua

### Viikon koevalmistautumisprotokolla

**Oppilaan ohjattu opiskelu** (opetetaan suoraan):

**Maanantai**: Koe ilmoitetaan perjantaiksi
```
Oppilas luo opiskelusuunnitelman:
Maanantai-ilta: Kertaa muistiinpanot, korosta avainkohdat (20 min)
Tiistai-ilta: Tee harjoituskoe, tee se (25 min)
Keskiviikko-ilta: Opiskele virheelliset vastaukset uudelleen (15 min)
Torstai-ilta: Viimeinen kertaus + hyvä uni (10 min)
Perjantai: KOE (itsevarma!)
```

**Opettaja tarjoaa itsearviointityökaluja**:
```
Tiistai: Opiskelun tukiarkin (luo harjoitustehtäviä)
Oppilas: Tekee kotona, tarkistaa vastaukset vastausavaimella
Etu: Itsenäinen harjoittelu vahvistuksineen
```

**Generaattorin käyttö itsetarkastukseen**:
```
Oppilas kysyy opettajalta: "Voinko saada lisää harjoitustehtäviä murtolukuihin?"
Opettaja: Luo 20-tehtävän työarkin + vastausavaimen (42 sekuntia)
Oppilas: Vie kotiin, harjoittele itsenäisesti, tarkista oma työ
Tulos: Rajaton harjoittelu saatavilla (oppilaan aloitteesta)
```

## Taito 5: Oma-aloitteisuuden harjoittelu

**Tavoite**: Oppilas pyytää apua asianmukaisesti

**Yläkoulun todellisuus**: Ei voi nostaa kättä ja odottaa (opettajalla 120 oppilasta, ei välttämättä huomaa)

**5. luokan oma-aloitteisuusharjoittelu**:

### Asianmukainen avun hakeminen

**Syys-joulukuu** (opettaja hyvin saavutettavissa):
```
Oppilas jumissa: Nostaa käden heti
Opettaja: Tulee auttamaan 30 sekunnin sisällä
Tulos: Oppilas oppii, että apua on saatavilla
```

**Tammi-maaliskuu** (vähennetään saavutettavuutta):
```
Uusi sääntö: "Kokeile 3 strategiaa ennen käden nostamista"
Strategia 1: Lue ohjeet uudelleen
Strategia 2: Kysy naapurilta (hiljainen kuiskaus)
Strategia 3: Tarkista esimerkki vihosta
Jos vieläkin jumissa: NYT nosta käsi

Opettaja: Auttaa vain kun 3 strategiaa yritetty
Tulos: Oppilaasta tulee itsenäisempi ongelman­ratkaisija
```

**Huhti-toukokuu** (yläkoulusimulaatio):
```
Uusi sääntö: "Tule luokseni tunnin lopussa jos tarvitset apua"
Tunnin aikana: Opettaja työskentelee pienryhmän kanssa (ei saatavilla)
Tunnin jälkeen: Oppilaat jonossa kysymysten kanssa (kuten yläkoulussa)

Tulos: Oppilas oppii:
- Yrittämään ensin itsenäisesti
- Kirjoittamaan kysymyksen muistiin (ettei unohda)
- Odottamaan sopivaa aikaa kysyä
- Ajamaan omaa oppimistaan
```

## Asteittainen vastuun siirto

**Viitekehys** (Fisher & Frey, 2008):
```
Minä teen (Opettaja mallintaa) → Elo-syyskuu
Me teemme (Opettaja + oppilas yhdessä) → Loka-tammikuu
Te teette yhdessä (Oppilas + toveri) → Helmi-maaliskuu
Sinä teet yksin (Oppilas itsenäisesti) → Huhti-toukokuu

Kesäkuuhun mennessä: Oppilas täysin valmis yläkoulun itsenäisyyteen
```

## Yläkouluennakointitoiminnot (toukokuu)

**Lukuvuoden lopun yläkoulusimulaatio**:

**Viikko 1**: Kaappiharjoitus
```
Toiminta: Asenna luokkahuoneen "kaapit" (laatikot)
Oppilaat: Saavat 3 minuuttia "tuntien" välillä vaihtaa materiaaleja
Simulaatio: Kiirehditään vaihtamaan luokkia (kuten yläkoulussa)
Opetus: Organisointi on tärkeää (ei voi tuhlata aikaa etsimiseen)
```

**Viikko 2**: Monen opettajan järjestelmä
```
Toiminta: Jokainen aine opetetaan eri paikassa (liikuntasali, kirjasto, musiikkiluokka)
Oppilaat: Vaihtavat huonetta 45 minuutin välein
Simulaatio: Luokkien vaihtaminen
Opetus: Ota oikeat materiaalit jokaiseen luokkaan
```

**Viikko 3**: Kotitehtävät monesta lähteestä
```
Toiminta: Matematiikan kotitehtävät + äidinkielen kotitehtävät + ympäristöopin projekti
Oppilaat: Seuraavat 3 tehtävää samanaikaisesti
Simulaatio: Monta opettajaa antaa töitä
Opetus: Kalenteri välttämätön (ei voi muistaa suullisesti)
```
