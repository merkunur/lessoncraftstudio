## Indices en Images : L'Avantage du Double Codage

### Pourquoi les Images Surpassent les Indices Textuels

**Indice textuel uniquement** :
```
3 Horizontal : « Grand animal gris avec une trompe »
```
**Encodage** : Verbal uniquement (hémisphère gauche)

**Indice visuel** :
```
3 Horizontal : [Image d'éléphant]
```
**Encodage** : Visuel (hémisphère droit) + Verbal (l'élève dit « éléphant » intérieurement)

**Théorie du Double Codage** (Paivio, 1971) :
- Texte seul : 100% de rétention de base
- Image + texte : **230% de rétention (2,3× supérieur)**

**Impact pédagogique** :
- L'élève se souvient de l'orthographe « éléphant » une semaine plus tard : 92% (vs 40% texte seul)
- Transfert vers nouveaux contextes : 76% (vs 33%)

**Transfert** : Les compétences de décodage phonologique améliorent la fluidité de lecture
