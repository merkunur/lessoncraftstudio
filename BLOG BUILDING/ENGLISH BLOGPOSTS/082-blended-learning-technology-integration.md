## Introduction: The Blended Learning Model

**Blended learning**: Combining online/digital instruction with face-to-face teaching

**Research** (Means et al., 2013 - meta-analysis): Blended learning outperforms purely online OR purely face-to-face by **0.35 effect size** (equivalent to several months additional growth)

**Why blended works**:
```
Online component:
- Self-paced (students work at own speed)
- Accessible 24/7 (learn anytime, anywhere)
- Instant feedback (digital assessments auto-grade)

Face-to-face component:
- Teacher support (answer questions, clarify)
- Peer collaboration (social learning)
- Hands-on activities (manipulatives, experiments)

Combined: Best of both worlds
```

**Traditional classroom**: One-size-fits-all instruction (everyone same pace)

**Blended classroom**: Personalized pathways (students progress individually)

## The Flipped Classroom Model

**Traditional model**:
```
In class: Teacher lectures (30 min), students listen
Homework: Practice problems independently (no teacher support)
Problem: Students stuck at home (when they NEED help most)
```

**Flipped model**:
```
At home: Watch video lecture (15 min), complete intro worksheet
In class: Teacher available for support, students work on practice
Benefit: Teacher present when students NEED help (during application)
```

### Flipped Lesson Structure

**Monday night (at home)**:
```
Student watches: 10-minute video on fractions (teacher-created OR YouTube)

Worksheet: Video Companion Notes
Name: _____________ Date: _______

While watching video, answer:

1. What does the numerator represent? __________
2. What does the denominator represent? __________
3. Draw a model showing 3/4:
   [Drawing box]
4. In your own words, what is a fraction? __________

Turn in Tuesday: Teacher quickly checks (5 min to review 30 sheets)
Result: Teacher knows who understood video, who's confused
```

**Tuesday (in class)**:
```
Teacher: "Based on your homework, I see 5 students need review. Rest of class, start on practice worksheet. I'll work with the 5."

Group A (25 students): Independent practice (20 fraction problems)
Group B (5 students): Small group reteach with teacher (15 min)

After 15 min: Teacher circulates to all students (now everyone ready)

Result: Differentiated support (targeted help for who needs it)
```

## Station Rotation with Digital/Print Mix

**Model**: Students rotate between digital and print stations

### 4-Station Rotation (90 minutes)

**Station 1: Teacher-Led (Face-to-Face)**
```
Content: New concept introduction, guided practice
Group size: 6-8 students
Duration: 20 minutes per group
Materials: Whiteboard, manipulatives, discussion
```

**Station 2: Independent Worksheet Practice (Print)**
```
Content: Practice worksheets (generated in 42 sec each)
Group size: 6-8 students
Duration: 20 minutes
Materials: Math worksheets, answer keys for self-checking
Self-directed: Students work independently, check own answers
```

**Station 3: Digital Learning (Online)**
```
Content: Educational software (Khan Academy, IXL, Prodigy Math)
Group size: 6-8 students
Duration: 20 minutes
Materials: Chromebooks/tablets
Adaptive: Software adjusts difficulty based on student performance
```

**Station 4: Collaborative Problem-Solving (Print)**
```
Content: Complex math puzzles, group challenges
Group size: 6-8 students
Duration: 20 minutes
Materials: Math puzzle worksheets, Sudoku, cryptograms
Social learning: Students work in pairs/small groups
```

**Rotation schedule**:
```
Period 1 (9:00-9:20): Group A at Station 1, Group B at Station 2...
Period 2 (9:20-9:40): Groups rotate clockwise
Period 3 (9:40-10:00): Continue rotation
Period 4 (10:00-10:20): Final rotation (all groups hit all 4 stations)

Result: 80 minutes, every student gets 20 min teacher time + 20 min digital + 40 min independent/collaborative print work
```

## Self-Paced Learning Pathways

**Personalized progression**: Students advance when READY (not when calendar says)

### Mastery-Based Advancement

**Traditional**: All students move to next topic Friday (whether mastered or not)

**Self-paced**: Students move when achieve 80%+ mastery

**System**:
```
Topic: 2-digit addition

Pathway:
1. Watch intro video (5 min)
2. Complete practice worksheet #1 (15 problems) - Check with answer key
3. If 12+/15 correct (80%+): Move to worksheet #2
   If below 80%: Rewatch video, get teacher help, retry worksheet #1

4. Complete practice worksheet #2 (20 problems)
5. If 16+/20 correct (80%+): Move to assessment
   If below 80%: More practice (worksheet #3), teacher support

6. Assessment (25 problems)
7. If 20+/25 correct (80%+): MASTERED! Move to subtraction
   If below 80%: Reteach, retry assessment

Result: Student advances when READY (solid foundation before moving on)
```

**Teacher role**: Monitor dashboard, provide targeted support to stuck students

**Digital + print**: Videos digital, worksheets print or digital (student choice)

## Accessibility Features (Digital Worksheets)

**Advantage of digital**: Built-in accommodations

### Text-to-Speech

**Student with dyslexia**:
```
Print worksheet: Student struggles to read problems (reading barrier)
Digital PDF: Student activates text-to-speech (device reads problems aloud)
Result: Access to content without reading barrier
```

### Zoom/Magnification

**Student with visual impairment**:
```
Print worksheet: Fixed font size (may be too small)
Digital PDF: Student zooms to 200-300% (larger text)
Result: Can read comfortably without specialized large-print creation
```

### Typing Instead of Handwriting

**Student with dysgraphia (handwriting difficulty)**:
```
Print worksheet: Must write answers (motor challenge)
Digital PDF: Types answers (bypasses handwriting barrier)
Result: Demonstrates knowledge without motor interference
```

**Universal Design for Learning**: Digital provides options automatically
