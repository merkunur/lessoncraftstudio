## Exercise Modes

### Shadow Match Mode (Default)
- Match objects to their shadows (line 1105-1107)
- Select exactly 4 images for the worksheet
- Creates matching exercise with shadows

### Make It Whole Mode
- Match object halves together (line 1109-1111)
- Cut direction options:
  - Horizontal cut (line 1117-1118)
  - Vertical cut (line 1121-1122)
- Creates puzzle-style matching activity
