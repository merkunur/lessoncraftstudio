Busca imágenes específicas usando la barra de búsqueda. Escribe palabras clave en español como "perro", "manzana" o "coche". El sistema filtra instantáneamente la biblioteca completa de 3000+ imágenes. Haz clic en cualquier miniatura para seleccionarla. La imagen seleccionada aparece en el área de vista previa confirmando tu elección. Esta imagen se dividirá en piezas de rompecabezas para tus fichas de matemáticas.

Alternativamente, sube tus propias imágenes personalizadas usando la sección Subir Imágenes Personalizadas. Haz clic en el botón Elegir Archivos para abrir tu explorador de archivos. Selecciona una o múltiples imágenes simultáneamente en formatos JPEG, PNG o GIF. Tus imágenes cargadas aparecen en el área de vista previa de imágenes cargadas. ### Paso 2: Personaliza la Configuración del Rompecabezas - Ajusta Fichas Preescolar para Diferentes Niveles y Números

Configura el tamaño de cuadrícula de tu rompecabezas según el nivel de habilidad de tus estudiantes. Abre la sección Configuración del Rompecabezas en la barra lateral. El campo Filas acepta valores entre 2 y 4 filas. El campo Columnas también acepta valores entre 2 y 4 columnas. Una cuadrícula de 2×2 crea 4 piezas perfectas para preescolar. Una cuadrícula de 3×3 crea 9 piezas apropiadas para primer grado. Una cuadrícula de 4×4 crea 16 piezas desafiantes para segundo y tercer grado.

Selecciona el tipo de operación matemática en el menú desplegable Operación. Elige Suma para practicar solamente problemas de adición. Elige Resta para practicar solamente problemas de sustracción. Elige Suma y Resta para mezclar ambos tipos de problemas aleatoriamente. El sistema genera automáticamente ecuaciones apropiadas que resultan en números del 2 al 16 dependiendo del tamaño de cuadrícula que elegiste.

Configura el tamaño y formato de página en la sección Configuración de Página. Selecciona Carta vertical para el tamaño estándar estadounidense. Selecciona A4 vertical para el tamaño estándar internacional. Las opciones horizontales funcionan bien para rompecabezas más anchos. Elige un color de página personalizado si quieres fondos coloreados. Las fichas para imprimir con fondos en colores pastel reducen el deslumbramiento y son más fáciles para los ojos de los niños.

Cada ecuación es única y apropiada para el nivel de educación infantil.
