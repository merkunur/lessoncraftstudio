## Introdução: A Dislexia e a Barreira da Leitura

**Prevalência da dislexia**: 10-20% da população (aproximadamente 2 milhões de estudantes portugueses e brasileiros)

**Défice central**: **Processamento fonológico** (dificuldade em conectar letras a sons)

**O desafio da leitura para alunos com dislexia**:
```
Palavra: "GATO"
Leitor típico:
- Vê: G-A-T-O
- Sons: /g/ /a/ /t/ /o/
- Junta: "gato"
- Tempo: 0,5 segundos

Leitor disléxico:
- Vê: G-A-T-O (pode parecer confuso: "A-G-T-O" ou "T-A-G-O")
- Sons: /g/... /a/?... /t/? (lento, trabalhoso)
- Junta: Dificuldade em unir os sons
- Tempo: 5-10 segundos (10-20× mais lento)
```

**Consequência**: Até ao 3º ano, um aluno disléxico leu **30 milhões de palavras a menos** que os colegas (Stanovich, 1986 - Efeito Mateus)

**Problema das fichas tradicionais**:
```
Sopa de Letras: Encontrar 20 palavras numa grelha 15×15
Desafios do aluno disléxico:
- Não consegue reconhecer palavras escritas rapidamente (reconhecimento visual lento)
- Confunde letras semelhantes (b/d, p/q, m/n)
- Sobrecarga de memória de trabalho (manter palavra em mente enquanto procura)
Resultado: Encontra 3/20 palavras, frustração, evitamento
```

**Solução**: Geradores adequados para dislexia (apoio visual, scaffolding fonético, carga de texto reduzida)

**Investigação** (Shaywitz & Shaywitz, 2005): Ensino multissensorial (visual + auditivo + cinestésico) melhora a leitura disléxica em **73%** comparado com apenas texto

## Os 7 Geradores Adequados para Dislexia

### Gerador #1: Bingo de Imagens (App 012) ⭐ RECOMENDAÇÃO #1 PARA DISLEXIA

**Porque o Bingo de Imagens é A MELHOR ferramenta para dislexia**:
- **Leitura zero necessária** (reconhecimento de imagens contorna o défice de descodificação)
- **Prática de consciência fonológica** (professor diz a palavra, aluno ouve os sons)
- **Sucesso acessível** (taxa de conclusão 90%+, construção de confiança)
- **Desenvolvimento da linguagem oral** (aprendizagem de vocabulário através da audição)

---

#### Como Funciona para Alunos Disléxicos

**Configuração**: Cartões de bingo 3×3 ou 4×4 com imagens (animais, alimentos, objetos)

**Dinâmica do jogo**:
1. Professor anuncia a palavra: "Maçã!"
2. Aluno ouve /ˈmasɐ̃/ (input fonológico, sem leitura necessária)
3. Aluno procura a imagem da maçã (reconhecimento visual, sem descodificação)
4. Aluno marca a casa (ação cinestésica)

**Integração multissensorial**:
- **Auditivo**: Ouve o professor dizer "maçã" (consciência fonológica)
- **Visual**: Vê a imagem da maçã (significado semântico)
- **Cinestésico**: Marca a casa com o dedo/marcador (memória motora)

**Investigação** (Moats, 2000): Aprendizagem multissensorial de palavras melhora o vocabulário disléxico em **68%** vs. apenas visual

---

#### Extensão Fonética (Alinhamento com Método Fónico)

**Bingo de Imagens padrão**: Professor anuncia "maçã" (palavra completa)

**Versão com reforço fonético**:
```
Professor: "Encontrem a palavra que começa com /m/ ... /m/ /a/ ... maçã"
(Segmenta os sons, depois junta)

O aluno ouve:
1. Som inicial isolado: /m/
2. Inicial + medial: /m/ /a/
3. Palavra completa: "maçã"

Resultado: Treino de consciência fonémica integrado no jogo
```

**Princípio do método fónico**: Ensino explícito e multissensorial da fonética

**Tempo de atividade**: 15-25 minutos

**Preços**: Pacote Core ou Acesso Completo

### Gerador #2: Encontrar Sombras (App 009)

**Porque o Encontrar Sombras funciona para dislexia**:
- **Leitura não necessária** (tarefa de correspondência visual)
- **Reconhecimento de formas** (competência pré-literacia, contorna défice fonológico)
- **Acomodação da memória de trabalho** (apenas 6-8 pares, carga gerível)

**Benefício específico para dislexia**: Desenvolve discriminação visual sem sobrecarga de texto

**Integração fonética opcional**:
```
Após completar a correspondência:
Professor: "O que emparelhaste? Vamos dividir em sons."
Aluno aponta para o gato: "Gato... /g/ /a/ /t/ /o/"
Resultado: Conecta imagem visual à estrutura fonológica
```

**Tempo de atividade**: 15-20 minutos

**Preços**: Pacote Core ou Acesso Completo

### Gerador #3: Encontrar Objetos (Descobre os Objetos) (App 026) - APENAS PISTAS VISUAIS

**Porque o Descobre os Objetos funciona para dislexia**:
- **Pesquisa visual** (leitura não necessária)
- **Processamento auditivo** (professor dá pistas orais)
- **Apoio à memória de trabalho** (pode repetir pista, sem texto para descodificar)

**Modificação crítica para dislexia**:
❌ **NÃO USE PISTAS DE TEXTO** (por ex., "Encontra 5 palavras que começam com C")
✅ **USE PISTAS VISUAIS + INSTRUÇÕES ORAIS**

**Configuração adequada para dislexia**:
```
Ficha: 15-20 imagens (maçãs, gatos, cães, carros, etc.)
Professor: "Encontra 5 maçãs" (instrução oral)
Aluno: Pesquisa visual (sem leitura)
```

**Extensão fonética**:
```
Professor: "Encontra os objetos que começam com /k/... gato, carro"
Aluno: Identifica pelo som inicial (consciência fonémica)
```

**Configurações**:
- Objetos: 15-20 (não 30+, reduz sobrecarga)
- Entrega de pistas: Oral (orientado pelo professor)
- Sem pistas de texto na ficha

**Tempo de atividade**: 20-30 minutos

**Preços**: Pacote Core ou Acesso Completo

### Gerador #4: Colorir (App 001)

**Porque colorir funciona para dislexia**:
- **Leitura não necessária** (zero sobrecarga de texto)
- **Redução de stress** (movimento repetitivo acalma ansiedade)
- **Experiência de sucesso** (impossível falhar ao colorir)
- **Oportunidade de linguagem oral** (discutir o que estão a colorir)

**Contexto de dislexia**: Muitos alunos disléxicos desenvolvem **ansiedade de leitura** devido a falhas repetidas

**Colorir como intervenção**:
- Proporciona atividade escolar bem-sucedida (aumenta autoestima)
- Cria oportunidade de discussão (desenvolvimento da linguagem oral)
- Sem pressão de desempenho

**Extensão fonética**:
```
Aluno colore imagem de maçã.
Professor: "O que estás a colorir?"
Aluno: "Maçã"
Professor: "Com que som começa maçã?"
Aluno: "/m/"
Resultado: Consciência fonémica em contexto de baixo stress
```

**Tempo de atividade**: 15-30 minutos

**Preços**: Pacote Core ou Acesso Completo

### Gerador #5: Contar no Gráfico (App 013)

**Porque contar funciona para dislexia**:
- **Números vs. letras** (dislexia afeta principalmente o processamento de letras, números menos impactados)
- **Contagem visual** (tarefa concreta, sem fonética abstrata)
- **Confiança em matemática** (alunos disléxicos frequentemente destacam-se em matemática, aproveita ponto forte)

**Configuração adequada para dislexia**:
- Tipo de gráfico: Pictograma (imagens, não rótulos de palavras)
- Números: Numerais (1, 2, 3, não "um, dois, três" escritos por extenso)
- Instruções: Oral (professor explica tarefa verbalmente)

**Tempo de atividade**: 15-20 minutos

**Preços**: Pacote Core ou Acesso Completo

### Gerador #6: Correspondências (App 005) - IMAGEM PARA IMAGEM

**Porque correspondências funcionam para dislexia**:
- **Tarefa visual** (descodificação não necessária)
- **Sucesso claro** (correspondência correta = imediatamente óbvio)
- **Apoio à memória de trabalho** (pode referir-se visualmente a ambas as colunas)

**Configurações para dislexia**:
- Tipo: Correspondência imagem-para-imagem (NÃO palavra-para-imagem para iniciantes)
- Pares: 6-8 (não 12-15, reduz carga cognitiva)

**Progressão** (à medida que as competências fonéticas se desenvolvem):

**Fase 1** (disléxico iniciante): Imagem → Imagem
- Emparelhar imagem de maçã com imagem idêntica de maçã

**Fase 2** (descodificação emergente): Letra inicial → Imagem
- Emparelhar letra "C" com imagem de gato (som inicial /k/)

**Fase 3** (leitor em desenvolvimento): Palavra → Imagem
- Emparelhar "gato" escrito com imagem de gato (prática completa de descodificação)

**Tempo de atividade**: 12-20 minutos

**Preços**: Pacote Core ou Acesso Completo

### Gerador #7: Comparação Grande Pequeno (App 019)

**Porque comparação de tamanho funciona para dislexia**:
- **Conceito concreto** (visualmente óbvio)
- **Leitura não necessária** (discriminação visual pura)
- **Desenvolvimento da linguagem** (ensina vocabulário comparativo: maior, menor)

**Abordagem adequada para dislexia**:
```
Ficha: Pares de objetos (cão grande, cão pequeno)
Aluno: Rodeia o grande (sem leitura)

Extensão oral:
Professor: "Qual é maior?"
Aluno: Aponta (linguagem receptiva)
Professor: "Este é o cão GRANDE. Podes dizer GRANDE?"
Aluno: "Grande" (linguagem expressiva, prática fonológica)
```

**Tempo de atividade**: 10-15 minutos

**Preços**: Pacote Core ou Acesso Completo

## Princípios de Design Adequados para Dislexia

### Princípio 1: Superioridade das Imagens

**Problema**: Pistas baseadas em texto exigem descodificação (fraqueza do aluno disléxico)

**Solução**: Use imagens como fonte primária de informação

**Exemplo**:
```
Pista de texto (mau para dislexia): "Um animal grande e cinzento com tromba"
Pista visual (bom para dislexia): [Imagem de elefante]
```

**Investigação** (Paivio, 1971): Imagens processadas **60% mais rápido** que texto para leitores disléxicos

---

### Princípio 2: Ensino Multissensorial

**Abordagem metodológica**: Visual + Auditivo + Cinestésico

**Como os geradores apoiam aprendizagem multissensorial**:

**Bingo de Imagens**:
- **Visual**: Ver imagem de maçã
- **Auditivo**: Ouvir professor dizer "maçã"
- **Cinestésico**: Marcar casa com a mão

**Comboio de Padrões**:
- **Visual**: Ver padrão AB
- **Auditivo**: Professor diz "maçã, banana, maçã, banana"
- **Cinestésico**: Cortar e colar vagões

**Investigação** (Shaywitz & Shaywitz, 2005): Métodos multissensoriais melhoram leitura disléxica em **73%**

---

### Princípio 3: Integração da Consciência Fonológica

**Crítico para dislexia**: Ensino explícito de fonética (não método global)

**Como integrar fonética**:

**Versão fonética do Bingo de Imagens**:
```
Professor segmenta palavra antes de anunciar:
"Encontrem a palavra... /g/ /a/ /t/ /o/ ... gato"

Aluno:
1. Ouve sons individuais (isolamento de fonema)
2. Ouve palavra completa (síntese de fonema)
3. Emparelha com imagem (conexão semântica)
```

**Versão fonética do Encontrar Objetos**:
```
Professor: "Encontrem objetos que começam com /b/... bola, borboleta, banana"
Aluno: Identifica pelo som inicial (consciência fonémica)
```

---

### Princípio 4: Carga Reduzida de Memória de Trabalho

**Dislexia + memória de trabalho**: 30-40% abaixo dos pares neurotípicos

**Como reduzir a carga**:

**Sopa de Letras** (NÃO recomendado para dislexia):
- Exigência de memória de trabalho: Manter palavra + pesquisar grelha + rastrear posição
- Blocos necessários: 8-10 (excede capacidade disléxica)

**Encontrar Sombras** (adequado para dislexia):
- Exigência de memória de trabalho: Manter características do objeto + pesquisar 6 sombras
- Blocos necessários: 3-4 (dentro da capacidade disléxica)

**Regra de design**: Manter campo visual pequeno (6-8 itens, não 20-30)

## Exemplos de Objetivos PEI para Dislexia

### Objetivo 1: Consciência Fonológica

"O aluno identificará sons iniciais em 10 palavras CVC com 80% de precisão até [data]"

**Linha de base**: Identifica sons iniciais com 40% de precisão (défice fonológico)

**Intervenção**:
- Bingo de Imagens diário com segmentação fonética
- Professor modela: "/g/ /a/ /t/ /o/ ... gato"
- Aluno repete sons antes de encontrar imagem

**Monitorização de progresso**: Avaliação semanal (amostra de 10 palavras)

**Ferramenta de medição**: Bingo de Imagens (com extensão fonética)

---

### Objetivo 2: Reconhecimento de Palavras de Alta Frequência

"O aluno reconhecerá 50 palavras de alta frequência à vista (sem descodificação) com 90% de precisão até [data]"

**Linha de base**: Reconhece 15/100 palavras de alta frequência (memória ortográfica fraca)

**Intervenção**:
- Exposição diária a palavras à vista via Bingo de Imagens
- Emparelhar imagem + palavra escrita (imagem de maçã + "maçã" escrito abaixo)
- Prática multissensorial (ver + ouvir + dizer)

**Monitorização de progresso**: Avaliação com cartões semanalmente

**Ferramenta de medição**: Bingo de Imagens, Correspondências (emparelhamento palavra-para-imagem)

---

### Objetivo 3: Processamento Auditivo

"O aluno seguirá instruções orais de 2 passos com 80% de precisão até [data]"

**Linha de base**: Segue instruções de 2 passos com 45% de precisão (fraqueza no processamento auditivo)

**Intervenção**:
- Encontrar Objetos diariamente com instruções orais
- Professor: "Encontra 5 maçãs e rodeia-as"
- Aluno: Processa input auditivo, executa tarefa

**Monitorização de progresso**: Observação durante atividades

**Ferramenta de medição**: Encontrar Objetos (entrega de instruções orais)
