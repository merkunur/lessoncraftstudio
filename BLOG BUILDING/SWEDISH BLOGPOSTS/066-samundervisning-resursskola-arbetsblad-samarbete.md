## Introduktion: Utmaningen med samordnad undervisning

**Samundervisning**: Klasslärare + specialpedagog = gemensamt klassrum

**Förekomst**: Över 70% av svenska grundskolor använder samundervisning enligt Skolverkets statistik

**Samordningsutmaning**:
```
Klasslärare: Planerar kursplan, skapar material
Specialpedagog: Ger anpassningar, modifieringar
Problem: Två lärare, separat planering → dubbelarbete ELLER bristande samstämmighet
```

**Traditionellt arbetsbladsproblem**:
```
Klasslärare: Skapar arbetsblad på årskursnivå (30 uppgifter, utmanande)
Specialpedagog: Måste modifiera (reducera till 15 uppgifter, lägga till bilder)
Dubbelarbete: 2× arbete för i princip samma innehåll
Resultat: Ineffektivitet
```

**Generatorlösning**: En plattform, omedelbar differentiering

**Forskning** (Nilholm & Göransson, 2017): Effektiv samundervisning kräver **gemensam planering** + **delade resurser** (minskar förberedelsetid med 40%)

## Samundervisningsmodeller och arbetsbladsanvändning

### Modell 1: En undervisar, en assisterar

**Struktur**: Klasslärare undervisar, specialpedagog cirkulerar och ger stöd

**Arbetsbladsanvändning**:
```
Klasslärare: Delar ut mattearbetsblad (20 uppgifter, årskursnivå)
Specialpedagog: Cirkulerar, identifierar elever som behöver stöd
Strategi: Specialpedagog ger anpassad version (10 uppgifter, bildstöd) till elever med behov
```

**Generatorarbetsflöde**:
```
Söndagsplanering (10 minuter):
Klasslärare: Genererar standardarbetsblad (20 uppgifter, symboliskt)
Specialpedagog: Genererar modifierad version (10 uppgifter, bildläge)
Båda: Delar via Google Drive

Måndagslektion:
Klasslärare: Delar ut standardversion till de flesta elever
Specialpedagog: Delar ut modifierad version till elever med åtgärdsprogram (5-8 elever)
Resultat: Differentiering utan separata planeringsmöten
```

**Tidsvinst**: 30 minuters veckoplanering (kan samordna via delad mapp istället)

---

### Modell 2: Stationsundervisning

**Struktur**: Klassen delas i stationer, varje lärare leder en station

**Arbetsbladsanvändning**:
```
Station 1 (Klasslärare): Korsord på årskursnivå
Station 2 (Specialpedagog): Anpassat korsord (färre ord, bildledtrådar)
Station 3 (Självständigt): Utmaningar för snabba elever
Station 4 (Självständigt): Praktiskt material
```

**Generatorarbetsflöde**:
```
Klasslärare genererar:
- Station 1 korsord (15 ord, textledtrådar)
- Station 3 utmaningar (kryptogram, avancerad sudoku)

Specialpedagog genererar:
- Station 2 korsord (8 ord, bildledtrådar)
- Facit för alla stationer

Tid: 12 minuter totalt (båda lärarna genererar)
Stationer klara för veckan
```

---

### Modell 3: Parallellundervisning

**Struktur**: Klassen delas i två, båda lärarna undervisar samma innehåll

**Arbetsbladsanvändning**:
```
Grupp A (Klasslärare): Standard svårighetsgrad
Grupp B (Specialpedagog): Anpassad svårighetsgrad (mindre grupp = mer stöd)
```

**Generatorfördel**: Samma generator, olika inställningar = omedelbar parallellversion

**Exempel**:
```
Mattelektion: Additionsträning

Grupp A arbetsblad:
- Talområde: 1-100
- Uppgifter: 25
- Format: Endast symboler
- Genererad: 42 sekunder

Grupp B arbetsblad:
- Talområde: 1-20
- Uppgifter: 15
- Format: Bildläge
- Genererad: 42 sekunder

Total förberedelse: 84 sekunder för två differentierade versioner
Båda grupperna tränar samma förmåga (addition) på lämplig nivå
```

---

### Modell 4: Alternativ undervisning

**Struktur**: Klasslärare undervisar stor grupp, specialpedagog tar ut liten grupp för förkunskaper eller repetition

**Arbetsbladsanvändning**:
```
Stor grupp (Klasslärare): Nytt innehåll + träning på årskursnivå
Liten grupp (Specialpedagog): Förkunskaper med anpassat arbetsblad ELLER repetition med förenklat arbetsblad
```

**Förkunskapsexempel**:
```
Fredag (specialpedagog tar ut grupp):
- Specialpedagog: Förkunskaper nästa veckas ordförråd (förenklad ordletar)
- Elever: Exponeras för termer före måndagslektionen
- Resultat: Mer framgångsrika under måndagens helklasslektion

Måndag (klasslärare helklass):
- Klasslärare: Undervisar ordförråd (standard korsord)
- Elever med förkunskaper: Redan bekanta med termerna (ökat självförtroende)
```

## Resursskolans användning

**Resursskola/resursrum**: Separat utrymme där elever får specialiserad undervisning

**Schema**: Elever tas ut från ordinarie klass 30-60 minuter

**Kursplansutmaning**: Måste samstämma med ordinarie undervisning samtidigt som specialiserat stöd ges

---

### Resursundervisningens samordning

**Problem**: Resursundervisning av annat innehåll = eleven missar ordinarie undervisning

**Lösning**: Parallell kursplan (samma ämnen, anpassad svårighetsgrad)

**Exempel**:
```
Ordinarie klassrum (Vecka 15 oktober):
- Matte: Multiplikationstabellen 1-12
- Svenska: Läsförståelse åk 4
- NO: Materiens tillstånd ordförråd

Resursskola (Samma vecka):
- Matte: Multiplikationstabellen 1-5 (begränsat talområde)
- Svenska: Anpassad läsförståelse (samma text, anpassade frågor)
- NO: Ordletar materiens tillstånd (samma ordförråd, annan aktivitet)

Resultat: Eleven lär sig samma innehåll som klasskamrater (bara anpassat stöd)
```

**Generatorarbetsflöde**:
```
Specialpedagog: Samordnar med klasslärare

Måndag morgon:
- Klasslärare: Laddar upp veckans arbetsblad i delad Google Drive
- Specialpedagog: Öppnar arbetsbladen, noterar vilka förmågor som tränas
- Specialpedagog: Genererar anpassade versioner (reducerad svårighetsgrad)
- Total tid: 15 minuter (vs 2-3 timmar att skapa från grunden)
```

---

### Uppföljning av åtgärdsprogram

**Resursundervisningens syfte**: Riktad undervisning enligt åtgärdsprogram

**Generatoranvändning**: Konsekvent uppföljningsverktyg

**Exempel åtgärdsprogramsmål**: "Eleven ska lösa enkla additionsuppgifter med 80% korrekt"

**Veckovis uppföljning**:
```
Vecka 1: Generera additionsblad (20 uppgifter, 1-10)
         Elevresultat: 12/20 (60%)

Vecka 2: Generera NYTT additionsblad (samma förmåga, andra uppgifter)
         Elevresultat: 14/20 (70%)

Vecka 3: Generera NYTT arbetsblad
         Elevresultat: 16/20 (80%) ✓ MÅL UPPNÅTT

Dokumentation: 3 arbetsblad i åtgärdsprogramsmapp (bevis på progression)
```

**Generatorfördel**: Nya uppgifter varje vecka (ingen memorering, äkta kunskapsbedömning)

**Arbetsflöde**:
```

- Genererar arbetsblad på standardnivå för veckan

- Granskar klasslärares uppladdningar
- Genererar anpassade versioner som matchar samma förmågor
- Laddar upp till "Anpassad_nivå"-mappar
- Total tid: 15 minuter

Måndag-fredag:
- Båda lärarna hämtar lämpliga versioner för elever
- Ingen daglig samordning behövs (material förberett)
```

### Veckoplanering (Förkortad)

**Traditionell samplanering**: 90 minuter/vecka (diskutera kursplan, skapa material, samordna)

**Med generatorer**: 30 minuter/vecka (diskutera endast undervisning, material redan genererat)

**Tidsvinst**: 60 minuter/vecka × 36 veckor = **36 timmar/år per lärarpar**

**Mötesagenda** (30 minuter):
```
1. Utvärdera förra veckans elevprogression (10 min)
2. Diskutera kommande veckans mål (10 min)
3. Fördela roller (vem undervisar vad) (5 min)
4. Bekräfta differentieringsstrategi (vilka elever får vilket material) (5 min)

Notera: Material redan genererat (diskuteras inte på mötet)
Resultat: Mötet fokuserar på UNDERVISNING, inte materialförberedelse
```

## Inkludering i praktiken

### Osynlig differentiering

**Mål**: Anpassa för elever med åtgärdsprogram utan stigmatisering

**Strategi**: Använd samma generator, subtila skillnader

**Exempel**:
```
Helklass: "Alla gör ett mattearbetsblad"

Verklighet:
- 22 elever: 25-uppgifters arbetsblad (standard)
- 8 elever med åtgärdsprogram: 15-uppgifters arbetsblad (anpassat)

Visuell likhet:
- Samma titel ("Additionsträning")
- Samma format (uppgifter i rader)
- Samma sidlayout
- Enda skillnad: Antal uppgifter (mindre uppenbart än "ANPASSAT" stämplat överst)

Resultat: Elever med åtgärdsprogram känner sig inkluderade (inte "annorlunda")
```

---

### Universell utformning för lärande (UDL)

**UDL-princip**: Designa för alla elever från start (inte efteranpassa)

**Generatoranvändning**:
```
Istället för:
1. Skapa standardarbetsblad
2. Modifiera för elever med åtgärdsprogram (extra arbete)

UDL-metod:
1. Generera 3 versioner samtidigt (15 min):
   - Nivå 1: Under årskursnivå (elever i svårigheter)
   - Nivå 2: På årskursnivå (majoritet)
   - Nivå 3: Över årskursnivå (avancerade elever)

2. Elever väljer nivå ELLER lärare tilldelar diskret

Resultat: Alla elever har lämpligt material, inget "anpassat"-stigma
```
