## Introduction : Le Lien Entre Vocabulaire et Compréhension

**Recherche** (Nagy & Townsend, 2012) : Les connaissances lexicales expliquent **30 à 40 % de la variance** dans la compréhension écrite

**Le lien vocabulaire-compréhension** :
```
Élève lisant un texte : « Le scientifique examine le spécimen au microscope. »

Élève avec vocabulaire étendu :
- Connaît : scientifique, examine, spécimen, microscope
- Compréhension : Comprend l'intégralité de la phrase
- Peut répondre : « Qu'observe le scientifique ? » (le spécimen)

Élève avec vocabulaire limité :
- Connaît : le, au
- Mots inconnus : scientifique, examine, spécimen, microscope
- Compréhension : Incapable de comprendre la phrase (trop de mots inconnus)
- Ne peut pas répondre aux questions

Résultat : Vocabulaire = clé d'accès à la compréhension
```

**Le fossé lexical** : En CE2, les élèves ayant un vocabulaire riche connaissent 2 à 3 fois plus de mots que leurs camarades

**Problème de l'enseignement traditionnel du vocabulaire** :
```
Lundi : L'enseignant présente 10 nouveaux mots (définition, utilisation en phrase)
Mardi : Les élèves recopient les définitions
Mercredi : Dictée de vocabulaire
Expositions : 3-4 (insuffisant pour la mémorisation)
Résultat : 30 % de rétention une semaine plus tard
```

**Solution** : Expositions multiples via des activités ludiques sur fiches (objectif : 17 expositions par mot - Marzano, 2004)

## Les 6 Générateurs pour Enrichir le Vocabulaire

### Générateur n°1 : Mots Mêlés (App 003) ⭐ N°1 POUR L'EXPOSITION AU VOCABULAIRE

**Pourquoi les mots mêlés développent le vocabulaire** :
- **Cartographie orthographique** : Schémas visuels de lettres → représentation mentale
- **Exposition répétée** : Balayage du mot 5 à 8 fois lors de la recherche (répétition visuelle)
- **Renforcement de l'orthographe** : Visualisation répétée de l'orthographe correcte
- **Pratique sans pression** : Format ludique (pas d'anxiété de test)

---

#### Le Processus de Cartographie Orthographique

**La cartographie orthographique** (Ehri, 2005) : Processus de formation de la représentation mentale d'un mot écrit

**Les étapes** :
```
Exposition 1 : Voir le mot « photosynthèse » dans la grille
           L'élève scanne : P-H-O-T-O-S-Y-N-T-H-È-S-E
           Cerveau : Commence à cartographier la séquence de lettres

Exposition 3 : Revoir le mot (3ème fois dans la même grille)
           Cerveau : Le schéma de lettres devient familier

Expositions 5-8 : Multiples balayages pendant la recherche
            Cerveau : Séquence de lettres CARTOGRAPHIÉE (reconnaissance instantanée)

Résultat : Le mot passe de « décodage lent » à « mot outil » (reconnaissance automatique)
```

**Recherche** (Share, 1995 - Hypothèse d'auto-apprentissage) : **4 à 7 expositions** à un mot écrit créent une mémoire orthographique (statut de mot outil)

**Avantage des mots mêlés** : Fournit 5 à 8 expositions en une seule activité

#### Création de Listes de Vocabulaire

**Listes thématiques** (engager les élèves par la pertinence) :

**Vocabulaire scientifique** :
- photosynthèse, chlorophylle, organisme, habitat, écosystème, producteur, consommateur, énergie, oxygène, dioxyde de carbone

**Vocabulaire sciences sociales** :
- démocratie, gouvernement, citoyen, droits, responsabilité, liberté, justice, constitution, amendement, vote

**Vocabulaire mathématique** :
- numérateur, dénominateur, fraction, décimal, équation, variable, coefficient, polygone, périmètre, aire

**Vocabulaire littéraire** (roman en cours) :
- Extraire 10-15 mots clés d'un chapitre, créer une grille de mots mêlés
- Les élèves : Voient les mots dans la grille, puis les rencontrent dans la lecture
- Effet de pré-exposition : Reconnaissance plus rapide des mots pendant la lecture

**Durée de l'activité** : 20-35 minutes

**Tarification** : Formule gratuite, Formule Core, ou Accès complet

### Générateur n°2 : Mots Croisés (App 008)

**Pourquoi les mots croisés développent la compréhension** :
- **Connaissance définitionnelle** : Les indices testent la compréhension (pas seulement la reconnaissance)
- **Indices contextuels** : Nécessité de déduire le sens à partir de l'indice pour trouver la réponse
- **Pratique de récupération** : Rappel actif (plus efficace que la révision passive)

**Rédaction d'indices pour la compréhension** :

**Niveau 1 - Indices définitionnels** (rappel) :
```
Mot : Démocratie
Indice : « Un gouvernement dirigé par le peuple »
Demande cognitive : Rappeler la définition
```

**Niveau 2 - Indices par exemples** (application) :
```
Mot : Démocratie
Indice : « Le type de gouvernement en France »
Demande cognitive : Appliquer la définition à un exemple concret
```

**Niveau 3 - Indices par inférence** (analyse) :
```
Mot : Démocratie
Indice : « Gouvernement où les citoyens votent pour choisir leurs dirigeants »
Demande cognitive : Analyser les composantes pour identifier le terme
```

**Progression** : Utiliser des indices plus simples au début, progresser vers des indices d'inférence à mesure que le vocabulaire se renforce

**Durée de l'activité** : 30-50 minutes

**Tarification** : Formule Core ou Accès complet

---

### Générateur n°3 : Cryptogramme (App 023)

**Pourquoi les cryptogrammes développent le vocabulaire en contexte** :
- **Exposition au niveau de la phrase** : Mots utilisés dans un contexte authentique (pas isolés)
- **Pratique des indices contextuels** : Utiliser les mots environnants pour décoder
- **Engagement soutenu** : 40-60 minutes avec un texte riche en vocabulaire

**Exemples de messages cryptogrammes** (riches en vocabulaire) :

**Sciences** :
« La photosynthèse est le processus par lequel les plantes transforment l'énergie solaire en énergie chimique stockée dans les molécules de glucose »

**Avantages** :
- Plusieurs mots de vocabulaire en contexte (photosynthèse, transforment, chimique, glucose, molécules)
- Les élèves voient COMMENT les mots sont utilisés (pas seulement les définitions)
- Indices contextuels : « processus par lequel les plantes » aide à définir photosynthèse

**Durée de l'activité** : 45-70 minutes

**Tarification** : Formule Core ou Accès complet

---

### Générateur n°4 : Écriture (App 014)

**Pourquoi la pratique de l'écriture renforce la lecture** :
- **Réciprocité encodage-décodage** : L'écriture (encodage) renforce la lecture (décodage)
- **L'écriture manuscrite renforce la mémoire** : La mémoire motrice améliore la reconnaissance des mots
- **Développement des mots outils** : Écriture répétée = reconnaissance automatique

**Recherche** (Berninger et al., 2006) : Les élèves qui pratiquent l'écriture manuscrite montrent une reconnaissance des mots **23 % meilleure** par rapport aux élèves utilisant uniquement le clavier

**Applications des fiches d'écriture** :

**Phrases de vocabulaire** :
```
Consigne : Écrire des phrases en utilisant les mots de vocabulaire de la semaine

Mot : photosynthèse
L'élève écrit : « Les plantes utilisent la photosynthèse pour fabriquer de la nourriture à partir du soleil. »

Avantage :
- Écrit le mot (mémoire motrice)
- Utilise en contexte (compréhension)
- Voit l'orthographe correcte (cartographie orthographique)
```

**Pratique de l'orthographe** :
```
Semaines 1-3 : Tracer les mots outils (apprentissage du schéma moteur)
Semaines 4-6 : Copier les mots outils à partir du modèle
Semaine 7+ : Écrire de mémoire (pratique de récupération)

Résultat : Automaticité des mots outils (lecture sans décodage)
```

**Durée de l'activité** : 15-25 minutes

**Tarification** : Formule Core ou Accès complet

### Générateur n°5 : Loto des Images (App 012)

**Pourquoi le loto des images développe le vocabulaire oral** :
- **Compréhension auditive** : Entendre le mot, identifier l'image (vocabulaire réceptif)
- **Connexion sémantique** : Lier le mot à une signification visuelle (double codage)
- **Exposition orale répétée** : Entendre le mot 10-15 fois pendant le jeu

**Séquence de loto de vocabulaire** :

**Semaine 1 : Loto avec images uniquement**
```
Enseignant : « Trouvez le microscope » (dit le mot en montrant l'image)
Élève : Voit l'image du microscope, marque sa carte
Cerveau : Connecte le son /mi.kʁɔs.kɔp/ au visuel (connaissance sémantique)
```

**Semaine 2 : Loto oral uniquement** (aucune image montrée)
```
Enseignant : « Trouvez le microscope » (aucune image montrée)
Élève : Doit récupérer l'image mentale, puis la trouver sur la carte
Cerveau : Connexion mot-signification renforcée
```

**Semaine 3 : Loto par définition** (avancé)
```
Enseignant : « Trouvez l'outil que les scientifiques utilisent pour voir de minuscules objets »
Élève : Doit inférer « microscope » à partir de la définition, trouver l'image
Cerveau : Compréhension définitionnelle (pas seulement reconnaissance)
```

**Progression** : Support visuel → Oral uniquement → Indices par définition (augmentation de la demande cognitive)

**Durée de l'activité** : 15-25 minutes

**Tarification** : Formule Core ou Accès complet

---

### Générateur n°6 : Cherche et Trouve (Je Vois) (App 026)

**Pourquoi le « Je Vois » développe le vocabulaire** :
- **Discrimination visuelle** : Identifier les objets par leur nom (association mot-image)
- **Pratique du vocabulaire oral** : L'enseignant donne des instructions verbales
- **Pensée catégorielle** : « Trouvez tous les outils » (catégories sémantiques)

**Applications du « Je Vois » pour le vocabulaire** :

**Vocabulaire scientifique** :
```
Fiche : 25 images liées aux sciences (microscope, bécher, tube à essai, thermomètre, etc.)
Tâche 1 : « Trouvez 5 outils de laboratoire »
Tâche 2 : « Trouvez 3 équipements de sécurité »
Tâche 3 : « Trouvez 4 instruments de mesure »

Demande cognitive : Catégoriser par fonction (connaissance sémantique)
```

**Vocabulaire académique de niveau 2** (transversal) :
```
Images représentant : analyser, comparer, contraster, évaluer, synthétiser, inférer
(Métaphores visuelles pour vocabulaire académique abstrait)

Tâche : « Trouvez l'image montrant 'analyser' » (loupe examinant un objet)
Résultat : Support visuel concret pour mot abstrait
```

**Durée de l'activité** : 20-30 minutes

**Tarification** : Formule Core ou Accès complet

## Développement des Mots Outils

**Mots outils** : Mots haute fréquence reconnus automatiquement (sans décodage)

**Importance** : Les 100 premiers mots outils représentent **50 % de tous les textes écrits**

**Recherche** (Ehri, 2005) : L'automaticité des mots outils = fluidité de lecture (120+ MPM)

**Applications des générateurs pour les mots outils** :

**Semaine 1 : Mots Mêlés** (100 mots outils Dubois-Buyse)
- Exposition visuelle : 5-8 fois par mot
- La cartographie orthographique commence

**Semaine 2 : Pratique d'Écriture** (tracer puis copier les mots outils)
- Mémoire motrice : La main apprend la forme du mot
- Renforce la cartographie orthographique

**Semaine 3 : Mots Mêlés** (disposition DIFFÉRENTE, mêmes mots)
- Nouvelle recherche (positions non mémorisées)
- Confirme la cartographie orthographique (peut trouver les mots rapidement)

**Semaine 4 : Mots Croisés** (indices de mots outils)
- Pratique de récupération : Doit épeler correctement
- Consolide la reconnaissance automatique

**Résultat** : 4 semaines = maîtrise des mots outils (reconnaissance automatique)

## Cycle d'Enseignement du Vocabulaire (Modèle à 17 Expositions)

**Objectif** : 17 expositions par mot pour un apprentissage approfondi (Marzano, 2004)

**Semaine 1** (Expositions 1-7) :
- **Lundi** : Présentation du mot (définition, visuel, exemple) - Exposition 1
- **Mardi** : Lecture dans le manuel - Expositions 2-3
- **Mercredi** : Mots mêlés en devoirs - Expositions 4-7 (multiples balayages)

**Semaine 2** (Expositions 8-13) :
- **Lundi** : Discussion de révision - Exposition 8
- **Mardi** : Écriture (utilisation en phrase) - Exposition 9
- **Mercredi** : Mots croisés (indice définitionnel) - Expositions 10-12
- **Jeudi** : Loto des images (oral) - Exposition 13

**Semaine 3** (Expositions 14-17) :
- **Lundi** : Relecture de la section du manuel - Expositions 14-15
- **Mercredi** : Cryptogramme (en contexte) - Exposition 16
- **Vendredi** : Test d'unité - Exposition 17

**Résultat** : 17 expositions = connaissance lexicale approfondie (peut utiliser de manière autonome)

## Priorité au Vocabulaire Académique de Niveau 2

**Niveaux de vocabulaire** (Beck et al., 2002) :
- **Niveau 1** : Mots quotidiens (maison, courir, content)
- **Niveau 2** : Mots académiques transversaux (analyser, évaluer, comparer, contraster, inférer)
- **Niveau 3** : Termes spécialisés (photosynthèse, démocratie, dénominateur)

**Priorité au niveau 2** : Le plus important pour la réussite scolaire (utilisé dans toutes les matières)

**Liste de mots de niveau 2** (fort impact) :
- analyser, synthétiser, évaluer, comparer, contraster, inférer, conclure, démontrer, établir, illustrer, interpréter, prédire, résumer, justifier, clarifier

**Intégration transversale dans les fiches** :
```
Problème de maths : « COMPAREZ les fractions 1/2 et 3/4 »
Sciences : « ANALYSEZ les résultats de l'expérience »
Sciences sociales : « ÉVALUEZ l'efficacité de la nouvelle loi »
Lecture : « INFÉREZ pourquoi le personnage a fait ce choix »

Résultat : Les mots de niveau 2 rencontrés dans toutes les matières (expositions multiples)
```

**Application des générateurs** : Créer des mots mêlés, mots croisés avec mots de niveau 2 (accent sur vocabulaire académique)

## Soutien à la Fluidité de Lecture

**Définition de la fluidité** : Précision + Vitesse + Prosodie (expression)

**Référence** (Hasbrouck & Tindal, 2017) :
- **CP (printemps)** : 60 MPM
- **CE1 (printemps)** : 90 MPM
- **CE2 (printemps)** : 110 MPM
- **CM1 (printemps)** : 120 MPM
- **CM2 (printemps)** : 130 MPM

**Rôle du vocabulaire dans la fluidité** :
```
Lecteur avec vocabulaire étendu :
- Reconnaît 95 %+ des mots automatiquement (mots outils)
- Vitesse de lecture : 120 MPM
- Attention : Concentrée sur la compréhension (pas le décodage)

Lecteur avec vocabulaire limité :
- Doit décoder 30 %+ des mots (lent)
- Vitesse de lecture : 60 MPM
- Attention : Consumée par le décodage (aucune capacité pour la compréhension)

Intervention : Développer le vocabulaire de mots outils → augmenter l'automaticité → améliorer la fluidité
```

**Contribution des mots mêlés** : Développe la reconnaissance automatique (cartographie orthographique)
