## Step 8: Printing Efficiently

**Print settings matter**

### Optimal Print Settings

**Printer preferences**:
```
Quality: Standard (not draft, not photo)
- Draft: Too light (hard to read)
- Standard: Clear, uses appropriate ink
- Photo: Unnecessary (wastes ink, slow)

Color: Black and white (unless specifically needed)
- Most worksheets: B&W sufficient
- Coloring pages: Color required
- Savings: B&W costs 80% less per page

Duplex (double-sided): When appropriate
- Practice worksheets: Single-sided (students write on)
- Reading passages: Duplex OK (saves paper)
- Important: Check which side is "up" (flip orientation matters)

Paper: Standard 20# copy paper
- Avoid cardstock: Expensive, unnecessary for most worksheets
- Avoid recycled: Sometimes jams, can be less crisp
```

---

### Bulk Printing Tips

**For 30-student class**:
```
Print job: 30 copies of worksheet

Settings:
1. Load paper tray fully (200+ sheets)
2. Select "Collate" (if stapling later)
3. Choose "Print all at once" (not one-at-a-time)
4. Send print job
5. Walk away (let it run)

Time: 30 copies = ~3 minutes print time
Avoid: Standing by printer (use time for other tasks)
```

## Common Mistakes to Avoid

**Learn from others' errors**

### Mistake 1: Too Many Parameters

**Problem**:
```
New user: Adjusts every single option (overwhelmed by choices)
Result: Paralysis by analysis (takes 10 minutes to generate one worksheet)
```

**Solution**:
```
Start simple: Use default settings first
Generate: See result
Adjust: Only change what needs changing
Iterate: Gradually explore more options

First worksheet: 2 minutes (learning)
Fifth worksheet: 30 seconds (confident)
```

---

### Mistake 2: Wrong Difficulty Level

**Problem**:
```
Teacher: Creates "hard" worksheet for struggling students
Result: Frustration, giving up
```

**Solution**:
```
Start easier: If unsure, go one level down
Test: Give to one student first
Adjust: Easier or harder based on response
Refine: Get difficulty just right over time
```

---

### Mistake 3: Not Keeping Organized Files

**Problem**:
```
Downloads: All go to default "Downloads" folder
Months later: "Where is that photosynthesis word search I made?"
Result: Recreating worksheets unnecessarily
```

**Solution**:
```
Immediately: Move downloaded worksheet to appropriate folder
Naming: Use descriptive names (not just "worksheet1.pdf")
Reuse: Find and reuse next year (save time)
```

## Troubleshooting Common Issues

**Quick fixes**

### Issue: "Worksheet won't generate"

**Possible causes**:
```
1. Too many words for grid size
   Fix: Reduce word count or increase grid size

2. Internet connection lost
   Fix: Check WiFi, refresh page

3. Invalid characters in input
   Fix: Remove special symbols, use standard letters
```

---

### Issue: "Print quality is poor"

**Fixes**:
```
1. Check printer settings (should be "Standard" not "Draft")
2. Ensure adequate ink levels (replace cartridge if low)
3. Clean print heads (printer maintenance menu)
4. Use quality paper (cheap paper can look fuzzy)
```

---

### Issue: "Can't find downloaded file"

**Solution**:
```
1. Check default Downloads folder
2. Search computer for PDF created today
3. Browser: Check download history (shows file location)
4. Moving forward: Immediately save to organized folder
```
