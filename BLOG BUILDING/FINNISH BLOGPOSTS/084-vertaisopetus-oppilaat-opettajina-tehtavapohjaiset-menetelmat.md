## Johdanto: Opettamalla Oppii Parhaiten

**Tutkimus** (Chi ym., 2001): Oppilaat, jotka opettavat luokkatovereitaan, oppivat ainesta **50 % paremmin** kuin oppilaat, joita opetetaan

**Miksi opettaminen tehostaa oppimista**:
```
Passiivinen oppiminen (ollaan opetettavana):
Oppilas: Kuuntelee, tekee muistiinpanoja, yrittää muistaa
Tiedon säilyminen: 20-30 %

Aktiivinen oppiminen (opettaa muita):
Oppilas: On ymmärrettävä syvällisesti pystyäkseen selittämään
Oppilas: Järjestelee tiedon opetusta varten
Oppilas: Vastaa kysymyksiin, selventää epäselvyyksiä
Tiedon säilyminen: 70-90 %

Tulos: "Opettamalla oppii kahdesti" (japanilainen sananlasku)
```

**Vertaisopetuksen hyödyt**:
- **Opettaja-oppilas**: Syventää ymmärrystä (selittäminen vahvistaa tietoa)
- **Opetettava oppilas**: Saa yksilöllistä tukea (toveri selittää eri tavalla kuin opettaja)
- **Luokanopettaja**: Moninkertaistaa vaikutuksensa (30 oppilasta → 15 paria = 15 pientä opettajaa)

## Rakenteellinen Vertaisopetuksen Malli

**Ei pelkkää "osaava auttaa heikompaa"** (epätasapainoinen roolitus)

**Vastavuoroinen malli**: Jokainen on vuorollaan opettaja JA opetettava (tasavertaiset roolit)

### Maanantai-Keskiviikko-Perjantai -Rakenne

**Maanantai**: Oppilas A opettaa oppilasta B (matematiikka)
**Keskiviikko**: Oppilas B opettaa oppilasta A (sanasto)
**Perjantai**: Molemmat työskentelevät yhdessä (ongelmanratkaisu)

**Esimerkki**:
```
Maanantai - Matematiikkatehtävät (20 yhteenlaskutehtävää):
Oppilas A: Teki tehtävät eilen, sai 18/20 oikein (90 %)
Oppilas B: Vaikeuksia kymmenten siirtämisessä

Prosessi:
1. Oppilas A näyttää mallin: "Näin ratkaisen 47+28"
2. Oppilas B tarkkailee, kysyy kysymyksiä
3. Oppilas B yrittää seuraavaa tehtävää oppilaan A ohjauksessa
4. Oppilaat vuorottelevat (B ratkaisee, A tarkistaa)
5. Molemmat tekevät 10 tehtävää yhdessä tukien

Tulos: Oppilas B oppii toverin selityksestä (usein selkeämpää kuin opettajan)
        Oppilas A vahvistaa omaa oppimistaan (opettaminen = syvä prosessointi)
```

## Tehtäväpohjainen Ohjausprotokolla

**Selkeä rakenne** (ei vain "auttakaa toisianne")

### 5-Vaiheinen Ohjausmenetelmä

**Vaihe 1: Mallinnus**
```
Ohjaaja: "Katso, kun teen ensimmäisen tehtävän"
Ohjaaja ratkaisee: 3/4 + 2/4 = 5/4
Ohjaaja selittää: "Laskin yhteen yläpuolen luvut (3+2=5) ja säilytin alapuolen luvun samana (4)"

Ohjattava: Tarkkailee, kuuntelee (ei keskeytä mallinuksen aikana)
```

**Vaihe 2: Ohjattu Harjoittelu**
```
Ohjaaja: "Kokeile nyt seuraavaa: 1/4 + 3/4"
Ohjattava: Ratkaisee tehtävän, ajattelee ääneen
Ohjattava: "Lasken yhteen yläpuolen: 1+3=4, säilytän alapuolen: 4, vastaus on 4/4"

Ohjaaja: Tarkistaa työn, antaa palautetta
Ohjaaja: "Hienoa! 4/4 on sama kuin yksi kokonainen."
```

**Vaihe 3: Itsenäinen Harjoittelu Tarkistuksineen**
```
Ohjattava: Tekee seuraavat 3 tehtävää itsenäisesti
Ohjaaja: Tarkkailee, ei keskeytä (antaa ohjattavan työskennellä)
Ohjaaja: Kolmen tehtävän jälkeen tarkistaa vastaukset vastausavaimen kanssa

Jos oikein: "Hyvä! Jatketaanko seuraavaan kolmeen?"
Jos virheitä: "Katsotaan yhdessä tehtävä numero 7. Mitä tässä tapahtui?"
```

**Vaihe 4: Roolien Vaihto (jos vastavuoroinen malli)**
```
10 tehtävän jälkeen: Roolit vaihtuvat
Entinen ohjattava → ohjaaja
Entinen ohjaaja → ohjattava

Molemmat hyötyvät: Kukin saa opettaa JA tulla opetetuksi
```

**Vaihe 5: Reflektointi**
```
Jakson lopussa (2 minuuttia):
"Mitä opit opettaessasi/opetettaessa tänään?"
"Mikä oli vaikeaa ohjaajana olemisessa?"
"Mikä oli hyödyllistä siinä, miten parisi opetti sinua?"

Metakognitio: Oppimisprosessin pohdintaa
```

## Oppilasohjaajaiden Kouluttaminen

**Ei voi olettaa oppilaiden osaavan ohjata** (on opetettava selkeästi)

### Viikko 1: Opettaja Mallintaa Ohjausta

**Järjestely**: Opettaja roolileikkii sekä ohjaajaa että ohjattavaa

```
Opettaja (ohjaajana): "Näytän nyt, miten hyvä ohjaaja toimii. Tarkkailkaa huolellisesti."

Opettaja mallintaa:
✓ Näyttää ensin mallin (mallinnus)
✓ Selittää ajattelunsa ääneen ("Ensin minun täytyy...")
✓ Pyytää ohjattavaa kokeilemaan (ohjattu harjoittelu)
✓ Antaa tarkkaa kiitosta ("Muistit siirtää kymmenyksen!")
✓ Korjaa ystävällisesti ("Tarkistetaan tuo vaihe uudelleen")

Oppilaat: Tarkkailee, tekee muistiinpanoja "hyvän ohjaajan toiminnasta"
```

**Mitä EI pidä tehdä** (myös mallinnetaan):
```
Opettaja (HUONONA ohjaajana):
✗ Ottaa kynän, tekee tehtävän ohjattavan puolesta ("Minä teen tämän sinulle")
✗ Sanoo "Väärin" selittämättä miksi
✗ Kärsimätön, huokaa ärsyyntyneenä
✗ Puhuu koko ajan, ei anna ohjattavan selittää ajatteluaan

Oppilaat: Tunnistavat tämän lähestymistavan ongelmat
Opetus: Ohjaaminen on opettamista, ei pelkkää vastausten antamista
```

---

### Viikko 2: Ohjattu Harjoittelu Tuen Kanssa

**Toiminta**: Oppilaat harjoittelevat ohjausta opettajan tuella

```
Parit oppilaat: Pari A ja Pari B
Annetaan tehtävälehti: 10 matematiikkatehtävää

Ohjeet:
"Pari A ohjaa Paria B viiden ensimmäisen tehtävän läpi käyttäen 5-vaiheista prosessia. Kierrän luokassa ja valmennan."

Opettaja: Kiertää, antaa reaaliaikaista palautetta
- "Hyvä mallinnus! Pyydä nyt pariasi kokeilemaan yhtä."
- "Huomasin, että annoit vastauksen. Kysy sen sijaan ohjaava kysymys."
- "Hienoa! Annoit parisi kamppailla hieman ennen auttamista."

Viiden tehtävän jälkeen: Parit vaihtavat rooleja
```

## Vertaisohjauksen Tehtäväpaketit

**Erityisesti suunniteltu ohjaukseen**

### Ohjauspaketin Rakenne

**Tehtävälehti 1: Mallitehtävät** (3 tehtävää ratkaisuineen)
```
Tehtävä 1: 47 + 28 = 75

  47
+ 28
----
  75

Askel askeleelta:
1. Laske ykköset: 7+8=15 (kirjoita 5, siirrä 1)
2. Laske kymmenet: 4+2=6, lisää siirretty 1=7
3. Vastaus: 75

Ohjaaja: Käytä tätä esimerkkinä opettaessasi
```

**Tehtävälehti 2: Ohjattu Harjoittelu** (5 tehtävää, ohjaaja neuvoo)
```
Tehtävä 1: 52 + 39 = ____
Ohjattava: Työskentelee ohjaajan neuvoessa

Tehtävät 2-5: Samanlaista vaikeustasoa
```

**Tehtävälehti 3: Itsenäinen Harjoittelu** (10 tehtävää, ohjattava yksin)
```
Ohjattava tekee itsenäisesti
Ohjaaja tarkistaa vastausavaimen kanssa valmistumisen jälkeen
```

**Tehtävälehti 4: Haaste** (3 vaikeampaa tehtävää, yhteistoiminnallisesti)
```
Molemmat oppilaat työskentelevät yhdessä (tasavertaiset kumppanit)
Ei nimettyjä ohjaaja/ohjattava-rooleja (todellinen yhteistyö)
```

## Eri-ikäisten Vertaisopetus

**5. luokan oppilaat ohjaavat 2. luokkalaisia** (molemmat hyötyvät)

### Toteutusprotokolla

**Järjestely**:
```
Aikataulu: Joka perjantai, 30 minuuttia
Parit: Yksi 5. luokkalainen yhden 2. luokkalaisen kanssa
Paikka: 2. luokan luokkahuone (ohjattavien tila = mukava)
Materiaali: 2. luokan tason tehtävät (sopiva ohjattavalle)
```

**5. luokkalaisten koulutus**:
```
Viikko ennen aloittamista:
5. luokan opettaja: "Ensi viikolla ohjaat 2. luokkalaisia. Valmistaudutaan."

Koulutuksen aiheet:
- Miten selittää yksinkertaisesti (käytä kuvia, ei monimutkaisia sanoja)
- Miten olla kärsivällinen (muista, kun itse olit 2. luokalla)
- Miten kannustaa (kiitä yrittämisestä, ei vain vastauksista)
- Mitä tehdä, jos hämmentyy (hae 2. luokan opettaja)

Harjoittelu: 5. luokkalaiset roolileikit toistensa ohjaamisesta
```

**Perjantain ohjaustuokio**:
```
14:00-14:05: 5. luokkalaiset saapuvat, tervehtivät ohjattaviaan
14:05-14:20: Ohjaus (tehtävien harjoittelu)
14:20-14:25: Lopetus, ylävitoset, positiivinen palaute
14:25-14:30: 5. luokkalaiset palaavat luokkaansa

2. luokkalaiset: Saavat yksilöllistä apua (5. luokkalainen omistettu heille)
5. luokkalaiset: Vahvistavat omia matemaattisia taitojaan (opettaminen = oppimista)
```

**Hyödyt**:
- 2. luokkalaiset: Yksilöllinen tuki, positiivinen roolimalli
- 5. luokkalaiset: Johtamistaidot, syvempi ymmärrys, empatia
- Opettajat: Suhteet rakentuvat luokka-asteiden välille, käytöksen parantuminen

## Vastavuoroinen Opetus Lukemisessa

**Strategia** (Palincsar & Brown, 1984): Oppilaat opettavat vuorotellen lukemisen ymmärtämisen strategioita

### Vastavuoroisen Opetuksen Roolit

**Luettava teksti + tehtävälehti 4 roolilla**:

**Rooli 1: Tiivistäjä**
```
Tehtävä: Tiivistä kappale 2-3 lauseeseen
Tehtävälehti: "Pääajatus oli..."
Taidot: Keskeisten asioiden tunnistaminen, tiivis viestintä
```

**Rooli 2: Kysyjä**
```
Tehtävä: Kysy 3 kysymystä kappaleesta
Tehtävälehti: "Mietin, että..."
Esimerkki: "Miksi henkilö päätti...?"
Taidot: Kysymysten muodostaminen, kriittinen ajattelu
```

**Rooli 3: Selventäjä**
```
Tehtävä: Tunnista hämmentävät kohdat, auta selittämään
Tehtävälehti: "Tämä kohta oli hämmentävä: ___ Luulen, että se tarkoittaa..."
Taidot: Ymmärryksen seuranta, selittäminen
```

**Rooli 4: Ennustaja**
```
Tehtävä: Ennusta, mitä seuraavaksi tapahtuu
Tehtävälehti: "Luulen, että seuraavaksi..."
Taidot: Todisteiden käyttö, päätelmien tekeminen
```

**Kierto**: Kukin oppilas ottaa kaikki 4 roolia neljän luettavan kappaleen aikana
Tulos: Jokainen opettaa lukemisen ymmärtämisen strategioita
