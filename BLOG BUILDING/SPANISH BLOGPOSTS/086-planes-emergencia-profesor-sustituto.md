## Introducción: La Crisis del Profesor Sustituto

**Realidad docente**: Necesitas un profesor sustituto con poca antelación (enfermedad, emergencia, formación profesional)

**El problema**: Preparar planes para el suplente a las 5:30 AM mientras estás enfermo
```
6:00 AM: Despiertas con gripe
6:15 AM: Llamas para solicitar sustituto
6:30 AM: Intentas escribir planes de clase (sintiéndote fatal)
7:00 AM: Te rindes, dejas nota genérica "que lean algo"
Resultado: Día escolar desperdiciado (el suplente no sabe qué enseñar)
```

**La solución**: Planes de emergencia preparados con antelación (creados cuando estás sano)

**Investigación** (Weems y Rogers, 2007): Los planes de calidad para sustitutos mantienen el **90% del aprendizaje regular** frente al **40%** con planes deficientes o inexistentes

**Mejor práctica**: Tener 3-5 días completos de planes para suplente listos en todo momento

```

- 8:30-9:00: Rutina matinal + asistencia (el suplente lo hace de forma independiente)
- 9:00-9:30: Ficha de matemáticas (30 problemas, clave de respuestas incluida)
- 9:30-10:00: Sopa de letras (vocabulario de la unidad actual)

Media jornada:
- 10:00-10:30: Lectura (lectura independiente + ficha de comprensión)
- 10:30-11:00: Recreo
- 11:00-11:30: Página para colorear (actividad tranquilizadora)

Tarde:
- 11:30-12:00: Almuerzo
- 12:00-12:30: Sopa de letras de ciencias (términos de la unidad de ciencias actual)
- 12:30-1:00: Crucigrama (vocabulario académico)
- 1:00-1:30: Puzzle matemático (pensamiento crítico, atractivo)
- 1:30-2:00: Elección libre: Trazado de líneas, colorear o lectura independiente

Salida: 2:00 PM

Nota para el suplente: "Todas las fichas tienen claves de respuesta en la Sección 5. Los estudiantes deben revisar su propio trabajo cuando terminen."
```

**Secciones 3-5: Días 2, 3, 4** (estructura similar, fichas diferentes)

**Sección 6: Claves de Respuestas** (todas las fichas)

## Fichas Autoinstructivas

**Principio clave**: El suplente no necesita enseñar contenido nuevo

**Características de las fichas para planes de sustituto**:

### Instrucciones Claras en la Ficha

**Ejemplo de encabezado de ficha de matemáticas**:
```
Nombre: _________________ Fecha: _______

INSTRUCCIONES:
1. Resuelve cada problema de suma
2. Muestra tu trabajo
3. Revisa tus respuestas con la clave de respuestas (en el escritorio del profesor)
4. Si tienes 8 o más incorrectas, pide ayuda al sustituto
5. Cuando termines, coloca la ficha en la bandeja de entrega

Tiempo estimado: 30 minutos
```

**Amigable para el suplente**: Instrucciones EN la ficha (el suplente no necesita explicar)

---

### Tipos de Actividades Familiares

**Usa fichas que los estudiantes ya conocen**:
```
✓ Cálculo matemático (practicado semanalmente - los estudiantes conocen el formato)
✓ Sopas de letras (hechas regularmente - autoinstructivas)
✓ Crucigramas (tipo de puzzle familiar)
✓ Colorear (todos saben cómo colorear)

✗ Reglas de juegos nuevos (el suplente no puede enseñar actividad desconocida)
✗ Proyectos complejos (requieren andamiaje del profesor)
✗ Trabajo en grupo (desafío de gestión del comportamiento para el suplente)
```

## Gestión del Comportamiento para Suplentes

**Desafío**: Los estudiantes prueban límites con el sustituto

**Solución**: Expectativas claras + trabajo atractivo

### Nota Pre-escrita para los Estudiantes

**Incluir en el plan para suplente**:
```
Querida clase,

Hoy no estoy, pero espero que muestren al profesor sustituto el mismo respeto que me muestran a mí. Esto es lo que espero:

✓ Seguir todas las normas de clase
✓ Completar todo el trabajo asignado
✓ Ayudar al sustituto si tiene preguntas
✓ Ser amables y respetuosos

Pediré un informe al sustituto cuando regrese. Los estudiantes que cumplan las expectativas recibirán [recompensa]. Los que no lo hagan tendrán [consecuencia].

¡Que tengan un gran día!
[Nombre del profesor]
```

**Responsabilidad**: Los estudiantes saben que el profesor hará seguimiento

---

### Trabajo Atractivo (Previene el Aburrimiento)

**Aburrimiento = problemas de comportamiento**

**Estrategia**: Mezclar fácil + desafiante + divertido
```
Mañana: Matemáticas estándar (familiar, genera confianza)
Media mañana: Sopa de letras (tipo juego, atractivo)
Antes del almuerzo: Colorear (tranquilizador, bajo estrés)
Después del almuerzo: Crucigrama (desafío moderado)
Final del día: Puzzle matemático O elección libre (final motivador)

Resultado: Estudiantes comprometidos (menos tiempo para mal comportamiento)
```

## Planes para Suplente Alineados con el Contenido

**Ideal**: Los planes para suplente revisan el contenido actual (no trabajo de relleno)

### Creación de Planes Alineados Mensualmente

**Septiembre (inicio del año)**:
```

```

```
- Matemáticas: Multiplicación (enfoque de octubre)

Reemplaza las fichas del plan de septiembre con versiones de octubre
Tiempo: 10 minutos para generar 5 fichas nuevas
```

**Continuamente**: Actualiza planes para suplente mensualmente (siempre contenido actual)

## Diferenciación en Planes para Suplente

**Desafío**: El suplente no puede diferenciar como el profesor regular

**Solución**: Materiales pre-diferenciados

### Bandejas de Fichas por Niveles

**Configurar con antelación**:
```
Bandeja 1 (Azul): Fichas modificadas
- 10 problemas en lugar de 20
- Apoyo con imágenes
- Para: Estudiantes con dificultades

Bandeja 2 (Verde): Fichas estándar
- 20 problemas
- Nivel de grado
- Para: Estudiantes en nivel

Bandeja 3 (Roja): Fichas de desafío
- 30 problemas
- Aplicaciones complejas
- Para: Estudiantes avanzados

Nota para el suplente: "Los estudiantes saben de qué bandeja de color tomar. Seleccionarán el nivel apropiado por sí mismos."

Resultado: La diferenciación ocurre automáticamente (no se requiere toma de decisiones del suplente)
```

## Entrenar a los Estudiantes para Días con Suplente

**Preparación**: Enseñar a los estudiantes cómo comportarse con suplentes

**Entrenamiento de septiembre**:
```
Profesor: "A veces estaré fuera y tendremos un sustituto. Esto es lo que espero:"

Juego de roles:
- Estudiante voluntario como "suplente"
- La clase practica:
  * Seguir instrucciones del suplente
  * Obtener fichas de la bandeja correcta
  * Pedir ayuda al suplente apropiadamente
  * Limpiar al final del día

Práctica: "Hoy fingiremos que es un día con suplente"
Estudiantes: Siguen el plan de emergencia para suplente como práctica
Resultado: Cuando viene el suplente real, los estudiantes conocen la rutina
```
