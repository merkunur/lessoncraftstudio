# Find and Count - Features Analysis

## App Purpose
Find and Count is an educational worksheet generator that creates hidden object counting worksheets. Students find and count specific objects hidden in a grid layout.

## Main Features (from REFERENCE APPS/find and count.html)

### 1. Language Settings (Line 980-1002)
- 11 language support for image library
- Languages: English, German, French, Spanish, Portuguese, Italian, Dutch, Swedish, Danish, Norwegian, Finnish
- Changes filenames used for content generation

### 2. Page Setup (Line 1003-1051)
**Page Sizes:**
- Default Worksheet (800x1000)
- A4 Portrait (210×297mm)
- A4 Landscape (297×210mm)
- Letter Portrait (8.5×11") - default
- Letter Landscape (11×8.5")
- Square (1200x1200)
- Custom dimensions

**Grid Dimensions (Line 1034-1037):**
- Grid Rows: 5-10 (default 6)
- Grid Columns: 5-10 (default 6)
- Creates adjustable grid for object placement

**Page Color:** Customizable background color

**Background Themes:** Theme-based backgrounds with opacity control

**Border Themes:** Theme-based borders with opacity control

### 3. Text Tools (Line 1053-1075)
- Add custom text elements
- Edit text properties: color, font, size
- Full control over text content

### 4. Image Library (Line 1077-1093)
- 3000+ child-friendly images
- Theme-based organization
- Search functionality
- Language-specific image selection
- Individual image browsing

### 5. Upload Custom Images (Line 1094-1111)
- Multi-file upload support
- All common formats (JPEG, PNG, GIF)
- Combine with library images
- Personalize for students

### 6. Hidden Object Questions (Line 1112-1117)
- Configure which objects to find
- Select from added images
- Automatically generates counting questions

### 7. Canvas Editing Tools (Line 1127-1169)
**Layer Controls:**
- Bring to Front
- Bring Forward
- Send Backward
- Send to Back

**Alignment Tools:**
- Align Left/Center/Right
- Align Top/Middle/Bottom
- Center on page horizontally/vertically

**Object Controls:**
- Lock/Unlock objects
- Delete objects
- Full drag, rotate, scale editing

### 8. Generation Options (Line 1196-1200)
- Generate New Worksheet
- Generate Answer Key (automatic)
- Answer key shows quantities

### 9. Download Options (Line 1203-1214)
**Formats:**
- Worksheet (JPEG)
- Answer Key (JPEG)
- Worksheet (PDF)
- Answer Key (PDF)

**Options:**
- Grayscale mode (save ink)
- 300 DPI high-resolution exports

### 10. Additional Controls (Line 1178-1193)
**Zoom Controls:**
- Zoom In
- Zoom Out
- Reset Zoom
- Percentage display

**History Controls:**
- Undo (Ctrl+Z)
- Redo (Ctrl+Y)
- Unlock All (when objects locked)

## Key Educational Value
- Develops counting skills
- Improves visual discrimination
- Builds attention to detail
- Vocabulary development (object names)
- Perfect for scuola dell'infanzia and classe prima

## Unique Features
1. Adjustable grid size (5-10 rows/cols)
2. Automatic answer key generation with quantities
3. Hidden object question configuration
4. Full canvas editing after generation
5. 11-language image library support
