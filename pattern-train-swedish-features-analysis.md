# Pattern Train App - Features Analysis

## App Purpose
Creates pattern recognition worksheets where students identify and continue visual patterns (AB, AAB, ABB, ABC, AABB).

## Core Features (from REFERENCE APPS analysis)

### 1. Page Setup (Accordion Section 1)
- **Languages**: 11 languages (EN, DE, FR, ES, PT, IT, NL, SV, DA, NO, FI)
- **Paper Sizes**:
  - Letter Portrait (8.5×11")
  - Letter Landscape (11×8.5")
  - A4 Landscape (297×210mm)
  - A4 Portrait (210×297mm)
  - Square (1200x1200)
  - Custom size
- **Template**: Default train template background
- **Page Color**: Customizable background color
- **Background**: Theme-based backgrounds with opacity control
- **Border**: Theme-based borders with opacity control

### 2. Text Tools (Accordion Section 2)
- Add custom text to worksheet
- Text properties:
  - Color picker
  - Font size (min 8)
  - Font family (Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana)
  - Outline color
  - Outline width (0-10)

### 3. Pattern Configuration (Accordion Section 3)
- **Pattern Types**:
  - AB (2-element pattern)
  - AAB (3-element pattern)
  - ABB (3-element pattern)
  - ABC (3-element pattern)
  - AABB (4-element pattern)
- **Theme Selection**: Auto-pick images from theme OR manual selection
- **Clue Count**: 4-10 clues (number of pattern repetitions shown)
- **Name/Date Fields**: Optional checkbox to include name and date fields

### 4. Image Selection (Accordion Section 4)
- **Image Dictionary**: Browse images by theme
- **Upload Custom Images**: Multi-file upload (JPEG, PNG, GIF)
- **Assigned Images Display**: Shows which images are assigned to each pattern position

### 5. Canvas Editing Features
- **Full Editability**: Drag, rotate, scale, delete any element
- **Layering Controls**:
  - Bring to Front
  - Bring Forward
  - Send Backward
  - Send to Back
- **Alignment Tools**: Align selected objects and align to page

### 6. Download Options
- **Worksheet Tab**: Main pattern worksheet
- **Answer Key Tab**: Separate answer key showing completed pattern
- **Format Options**:
  - Worksheet (JPEG)
  - Answer Key (JPEG)
  - Worksheet (PDF)
  - Answer Key (PDF)
- **Grayscale Toggle**: Convert to grayscale for ink-saving

## Pattern Train Unique Value
- Creates BOTH worksheet AND answer key
- 5 different pattern types (AB, AAB, ABB, ABC, AABB)
- Adjustable difficulty via clue count (4-10)
- Uses images for visual patterns (not just shapes/colors)
- Combines pattern recognition with vocabulary (image filenames)
- Train template provides engaging thematic context
