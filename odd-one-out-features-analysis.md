## Core App Features (from REFERENCE APPS/odd one out.html)

### 1. Page Setup (Lines 1054-1073)
- **Page Size Options:**
  - Letter Portrait (8.5×11")
  - Letter Landscape (11×8.5")
  - A4 Portrait (210×297mm)
  - A4 Landscape (297×210mm)
  - Square (1200x1200)
  - Custom dimensions (user-defined width/height in pixels)
- **Page Color:** Custom color picker for background

### 2. Background & Border Customization (Lines 1075-1094)
- **Background Themes:** Selectable from theme library
- **Background Opacity Slider:** Adjust transparency
- **Border Themes:** Selectable from theme library
- **Border Opacity Slider:** Adjust transparency

### 3. Text Tools (Lines 1098-1117)
- **Add New Text:** Custom text input with placeholder
- **Text Properties:**
  - Color picker
  - Font size (8px minimum)
  - Font family: Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana
  - Outline color picker
  - Outline width (0-10, 0.5 step increments)

### 4. Exercise Configuration (Lines 1122-1159)
- **Number of Exercises:** 5, 6, 7, 8, 9, or 10 exercises per worksheet
- **Global Mode:**
  - **Identical Mode:** 3 identical images + 1 different (all from same theme)
  - **Similar Mode:** 3 images from Theme A + 1 image from Theme B (default)
- **Per-Exercise Override:** Each exercise can use global mode or override with specific mode
- **Clear Selections:** Button to clear selections for individual exercises
- **Optional Fields:**
  - Include Name/Date fields (checkbox)
  - Include exercise numbers (checkbox)

### 5. Image Library (Lines 1162-1180)
- **Theme A Selection:** Dropdown for primary theme (for Similar/Identical mode)
- **Theme B Selection:** Dropdown for secondary theme (for Similar mode odd one out)
- **Search Function:** Search images by keyword
- **Click to Add:** Click images from library to add to current exercise
- **Exercise Selections Display:**
  - Common Images preview (3 images)
  - Odd Image preview (1 image)

### 6. Upload Custom Images (Lines 1184-1196)
- **Multi-file Upload:** Upload custom images for personalization
- **Session Storage:** Uploaded images available during current session
- **Preview Area:** See uploaded images before adding to worksheet

### 7. Canvas Editing Tools (Toolbar - Lines 1214-1254)
- **Layer Control:**
  - Bring to Front
  - Bring Forward
  - Send Backward
  - Send to Back
- **Alignment (Selected Objects):**
  - Align selected objects to each other
- **Alignment (to Page):**
  - Align objects to page edges/center

### 8. Dual View Tabs (Lines 1254-1255)
- **Worksheet Tab:** Create and edit the main exercise worksheet
- **Answer Key Tab:** Automatically generated answer key (shows which is the odd one out)

### 9. Download Options (Line 1285)
- Download worksheet as PDF/JPEG
- Download answer key as PDF/JPEG
- High-quality 300 DPI export
- Grayscale option (save ink)

## Summary of Key Educational Value

**Purpose:** Visual discrimination and cognitive development exercises where students identify which image doesn't belong.

**Two Learning Modes:**
1. **Identical Mode:** Subtle differences (3 identical + 1 slightly different from same theme) - harder
2. **Similar Mode:** Category differences (3 from one category + 1 from another) - easier

**Flexibility:** Teachers can create 5-10 exercises per page, customize difficulty per exercise, and include/exclude name fields and numbering based on classroom needs.

**11 Languages:** All UI elements and image filenames support 11 languages for multilingual education.
