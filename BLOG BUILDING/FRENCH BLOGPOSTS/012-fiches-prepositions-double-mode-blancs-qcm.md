## Introduction : Le Défi des Prépositions

**Le concept grammatical le plus difficile pour les apprenants de français** : les prépositions (Swan & Smith, 2001)

**Pourquoi les prépositions posent problème** :
- Relations spatiales abstraites (dans, sur, sous, derrière, entre)
- Spécificités linguistiques (impossible de traduire littéralement : "on Monday" devient "le lundi" sans préposition en français)
- Dépendance au contexte (sur la table, dans la matinée, à 15 heures)
- Absence de règles cohérentes (pourquoi "dans le bus" mais "en voiture" ?)

**Fiche traditionnelle** :
```
Le chat est ___ la boîte.
```
**Problème** : L'élève lit la phrase, ne visualise rien, devine au hasard

---

**Fiche de prépositions visuelle** :
```
[Image : Chat assis sur le dessus d'une boîte]
Le chat est ___ la boîte.
Options : a) dans  b) sur  c) sous
```
**Avantage** : Le contexte visuel élimine toute ambiguïté

**Innovation du double mode** : Même fiche, deux niveaux de difficulté
- **Exercices à trous** (plus difficile) : L'élève produit la réponse
- **QCM** (plus facile) : L'élève sélectionne parmi les options

## Fonctionnement du Double Mode

### Mode 1 : Exercices à Trous (Pratique de Production)

**Ce que voit l'élève** :
```
[Image : Ballon sous une table]
Le ballon est ________ la table.
```

**Exigence cognitive** : ÉLEVÉE
- Doit retrouver la préposition correcte en mémoire
- Générer l'orthographe correcte
- Aucun étayage fourni

**Idéal pour** :
- Élèves FLE avancés (intermédiaire+)
- Francophones révisant la grammaire
- Évaluation sommative (mesure la vraie maîtrise)

**Recherche** : La pratique de production crée un encodage mémoriel 3× plus solide que la reconnaissance (Carrier & Pashler, 1992)

---

### Mode 2 : QCM (Pratique de Reconnaissance)

**Ce que voit l'élève** :
```
[Image : Ballon sous une table]
Le ballon est ________ la table.
a) sur    b) dans    c) sous    d) à côté de
```

**Exigence cognitive** : MODÉRÉE
- Reconnaître la bonne réponse parmi les options
- Éliminer les choix manifestement incorrects
- Exigence orthographique réduite

**Idéal pour** :
- Débutants FLE
- Jeunes apprenants (5-7 ans)
- Introduction initiale aux prépositions
- Pratique guidée avant les exercices à trous

**Recherche** : Les QCM permettent de se concentrer sur le sens plutôt que l'orthographe, réduisant la charge cognitive (Sweller, 1988)

---

**Résultat** : Développement systématique de la compétence reconnaissance → production

### Bénéfice 1 : Le Contexte Visuel Réduit l'Ambiguïté

**Problème des fiches textuelles** :
```
Le livre est ___ la table.
```
**L'élève pense** : "Ça pourrait être SUR la table ou SOUS la table... je vais dire SUR"
**Réponse attendue** : SOUS (l'élève s'est trompé)

**Solution avec fiche visuelle** :
```
[Image : Livre clairement positionné sous la table]
Le livre est ___ la table.
```
**L'élève voit** : Le livre est clairement SOUS
**Résultat** : Réponse correcte basée sur la compréhension, pas la chance

**Recherche** : Le contexte visuel améliore la précision des prépositions de 67% pour les apprenants FLE (Tyler et al., 2011)

**Transfert vers les apprentissages** :

## 47 Prépositions Couvertes

### Prépositions Spatiales (22)

**Localisation** : dans, sur, sous, au-dessus de, au-dessous de, derrière, devant, à côté de, près de, entre, parmi, loin de

**Direction** : vers, à, en, de, dans, sur, hors de, à travers, le long de

**Exemples** :
- Image : Chat DANS la boîte
- Image : Chat SUR la boîte
- Image : Chat SOUS la boîte
- Image : Chat ENTRE deux boîtes

---

### Prépositions Temporelles (8)

**Temps** : à, en, dans, avant, après, pendant, depuis, jusqu'à

**Exemples** :
- Image : Horloge montrant 15h → "à 15 heures"
- Image : Calendrier avec lundi entouré → "le lundi"
- Image : Calendrier mensuel → "en janvier"

**Diverses** : avec, sans, pour, de, par, comme, selon, contre, malgré, sauf, chez, vers, environ, outre, hormis, dès, moyennant

## Mise en Œuvre en Classe

### Stratégie 1 : TPR (Total Physical Response / Réponse Physique Totale)

**Protocole** :
1. Générer une fiche avec prépositions spatiales
2. Avant la fiche, activité physique
3. L'enseignant dit : "Mettez votre main SUR votre tête"
4. Les élèves exécutent l'action
5. Répéter avec : sous, derrière, devant, à côté de, entre

**Ensuite** : Compléter la fiche (renforcement visuel de l'expérience physique)

**Résultat** : 89% de précision vs 64% sans TPR (Asher, 1969)

---

### Stratégie 2 : Enseignement par les Pairs

**Configuration** :
1. Générer fiche QCM (plus facile)
2. Les élèves travaillent en binômes
3. Pour chaque question, le Partenaire A explique POURQUOI la réponse est correcte
4. Le Partenaire B confirme ou conteste
5. Inverser les rôles pour la question suivante

**Bénéfice** : La verbalisation approfondit la compréhension (apprentissage social de Vygotsky)

---

### Stratégie 3 : Difficulté Progressive

**Structure semaine par semaine** :

**Semaine 1** : 5 questions, QCM 3 options, prépositions spatiales simples (dans, sur, sous)

**Semaine 2** : 8 questions, 3 options, ajouter (derrière, devant, à côté de)

**Semaine 3** : 10 questions, 4 options, ajouter (entre, au-dessus, au-dessous)

**Semaine 4** : 10 questions, exercices à trous, révision de toutes les prépositions

**Évaluation** : 80%+ de réussite aux exercices à trous = maîtrise

---

### Stratégie 4 : Chasse au Trésor Réelle

**Activité d'extension** :

**Consigne** :
1. Générer une fiche avec 10 prépositions spatiales
2. Les élèves trouvent des exemples réels dans la classe
3. Photographier chaque exemple (tablette/téléphone)
4. Créer une affiche : Photo + Phrase avec préposition

**Exemple** :
- Photo : Crayon sur le bureau
- Phrase : "Le crayon est SUR le bureau."

**Transversalité** : Écriture, photographie, raisonnement spatial
