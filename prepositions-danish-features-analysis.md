## Exercise Types

### Fill-in-the-Blank Mode
- Shows an image with item and shape
- Student writes the correct preposition describing relationship
- Answer key shows correct preposition

### Multiple Choice Mode
- Custom description text at top
- Shows 3 images for each exercise
- Only one image correctly demonstrates the preposition
- Student circles/identifies correct image
- Answer key highlights correct answer
