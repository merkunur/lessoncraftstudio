## Johdanto: Siirtymätilanteissa hukkuva opetusaika

**Hukkaantunut aika**: Siirtymätilanteet kerryttävät merkittävästi menetettyä opetusaikaa

**Tutkimus** (Burns, 2007): Keskimääräinen luokkahuone menettää **20-30 minuuttia päivässä** siirtymätilanteisiin
```
Laskutoimitus:
8 siirtymää × 3 minuuttia = 24 minuuttia hukkaan päivittäin
24 minuuttia × 180 koulupäivää = 4 320 minuuttia vuodessa
4 320 minuuttia = 72 tuntia = 12 kokonaista koulupäivää

Todellisuus: Lähes 3 viikkoa opetusta menetetään vuosittain
siirtymätilanteissa!
```

**Tyypilliset haasteet siirtymätilanteissa**:
```
Ongelmat käyttäytymisessä:
- Oppilaat vaeltavat päämäärättömästi
- Keskustelevat kavereittensa kanssa työn sijaan
- "Mitä teen tavaroilleni?"
- "Minne minun pitää mennä?"
- Materiaalit eivät ole valmiina seuraavaa toimintaa varten

Tulos: Kaaos, hukattu aika, käyttäytymisongelmia
```

**Ratkaisu**: Opeta selkeät rutiinit + käytä tehtävämonistera siirtymätilanteen työkaluina
```
Rutiineilla:
- Oppilaat tietävät tarkalleen mitä tehdä
- Materiaalit valmiina nopeaa vaihtoa varten
- Minimaalinen opettajan ohjaus tarvitaan
- Sujuvat, tehokkaat siirtymätilanteet

Tulos: Säästät 15+ minuuttia päivässä = 45 tuntia lisäopetusaikaa vuodessa
```

**Keskeinen periaate**: Jokainen siirtymätilanne tarvitsee selkeän rutiinin (opetettu, harjoiteltu, vahvistettu)

## Siirtymärutiinien opettaminen

**Mallinna, harjoittele, vahvista**

### 3-vaiheinen siirtymämalli

**Sovellettavissa kaikkiin siirtymätilanteisiin**:
```
Vaihe 1: LOPETA nykyinen toiminta
- Opettajan merkki: Kellokello, valojen himmentäminen, taputtamisrytmi
- Oppilaat: Lopettavat työskentelyn, katsovat opettajaa
- Aika: 5 sekuntia

Vaihe 2: VALMISTAUDU seuraavaan toimintaan
- Oppilaat: Siivoa pois materiaalit, hae uudet materiaalit
- Paikka: Siirrä tarvittaessa uuteen paikkaan
- Aika: 30-60 sekuntia

Vaihe 3: ALOITA seuraava toiminta
- Oppilaat: Aloita työ (tehtävälomake, lukeminen jne.)
- Opettaja: Minimaalinen uudelleenohjaus tarvitaan (oppilaat tuntevat rutiinin)
- Aika: 5 sekuntia aloittamiseen

Kokonaissiirtymä: 40-70 sekuntia (vs. 3-5 minuuttia ilman rutiinia)
```

---

### Ensimmäinen kouluviikko: Siirtymärutiinien intensiivikurssi

**Opeta jokainen siirtymä erikseen**:
```
Maanantai: Opeta siirtymä #1 (aamutulo → matematiikka)
- Mallinna: Opettaja näyttää vaiheet
- Ohjaa: Luokka harjoittelee yhdessä (hitaassa tempossa)
- Harjoittele: Toista 3-5 kertaa kunnes sujuvaa

Tiistai: Opeta siirtymä #2 (matematiikka → äidinkieli)
- Mallinna, ohjaa, harjoittele (sama prosessi)

Keskiviikko: Opeta siirtymä #3 (äidinkieli → välitunti)
jne.

Perjantaina: Kaikki tärkeimmät siirtymät opetettu

Viikko 2: Harjoittele ja hioa (ajoitus, ongelmien ratkaisu)

Tulos: Sujuvat siirtymät viikon 2 loppuun mennessä
(sijoitus maksaa itsensä takaisin koko vuoden ajan)
```

## Tehtävämonisteet siirtymätilanteen työkaluina

**Välitön sitoutuminen = ei seisokkiaikaa**

### Aamun lämmittelytehtävät

**Aamutulo rutiini**:
```
Näkyvä ohje:

Kun saavut:
1. Ripusta reppu naulakkoon
2. Palauta kotitehtävät (jos on)
3. Hae lämmittelytehtävä korista
4. Istu pulpetillesi ja aloita työskentely
5. Työskentele kunnes opettaja antaa merkin (klo 8:30)

Lämmittelytehtävä:
- 5-10 nopeaa tehtävää
- Kertausmateriaalia (ei uutta sisältöä)
- Itsetarkistus (vastausavain pisteellä)

Aika: 10 minuuttia (8:20-8:30)
Oppilaat sitoutuneet välittömästi (ei odottelua, ei häiriökäyttäytymistä)
```

**Esimerkki lämmittelytehtävästä**:
```
Hyvää huomenta! Päivämäärä: __________

1. Laske: 7 × 8 = ____
2. Kirjoita oikein: _____________ (Opettaja kirjoittaa sanan taululle)
3. Pääajatus: Lue kappale, ilmaise pääajatus yhdellä lauseella
4. Sanasto: Käytä eilisen sanan lauseessa
5. Aivojumppa: Jos tänään on keskiviikko, mikä päivä on 10 päivän kuluttua?

Valmis etuajassa? Lue itsenäisesti kunnes klo 8:30.

Tarkoitus: Aktivoi aivot, kertaa taitoja, luo rauhallinen aloitus
```

### Lopetusliput (Oppitunnin lopetusmonisteet)

**Siirtymä kotiinlähtöön**:
```
Päivän viimeiset 5 minuuttia:

Rutiini:
1. Siivoa työpiste (1 min)
2. Täytä lopetuslippu (3 min)
3. Palauta lopetuslippu koriin
4. Istu hiljaa kunnes kotiinlähtö

Lopetuslippu:
Nimi: __________ Päivämäärä: __________

Tänään opin: ___________________________
Yksi kysymys joka minulla vielä on: __________________
Huomenna haluan muistaa: ___________

Tarkoitus: Reflektoi oppimista, sujuva kotiinlähtö (oppilaat istuvat, rauhallista)
```

## Ajastimet siirtymätilanteissa

**Visuaalinen ajanhallinta**

### Heijastettu ajastin

**Siirtymän lähtölaskenta**:
```
Opettaja ilmoittaa: "Sinulla on 2 minuuttia siirtyä matematiikasta
äidinkieleen. Hae lukuluokkasi ja ole istualla. Ajastin käynnistyy nyt."

[Näyttää 2:00 lähtölaskennan projektorilla]

Oppilaat näkevät: Jäljellä oleva aika (1:58... 1:57... 1:56...)

Opettaja: Minimaalinen uudelleenohjaus tarvitaan (ajastin luo kiireen tunnetta)

Klo 0:00: Oppilaiden tulisi olla istualla materiaalien kanssa valmiina

Vastuu: "Hienoa työtä! Kaikki siirtyivät alle 2 minuutissa. Eilen kesti
3 minuuttia. Olette tehostuneet!"

Hyöty: Rakentaa tehokkuutta, oppilaat seuraavat aikaa itse
```

---

### "Voita ajastin" -haaste

**Pelillistäminen**:
```
Maanantai lähtötaso: "Katsotaanpa kuinka kauan tämä siirtymä kestää.
Ajastin käynnistyy nyt."
Tulos: 3 minuuttia 15 sekuntia

Tiistai: "Voimmeko päihittää eilisen ajan? Tavoite: Alle 3 minuuttia."
Tulos: 2 minuuttia 45 sekuntia ✓

Perjantai: "Ennätyksemme on 2:10. Voimmeko rikk oa sen?"
Tulos: 1:55! Uusi ennätys!

Seuraa taulukolla: Visuaalinen edistyminen (oppilaat näkevät parannuksen)

Hyöty: Motivaatio siirtyä nopeasti, joukkuetavoite (luokka työskentelee yhdessä)
```

## Oppiainekohtaiset siirtymärutiinit

**Jokaisella oppiaineella on ennakoitava kulku**

### Matematiikasta äidinkieleen siirtymä

**Selkeät vaiheet**:
```
Rutiini (opetettu viikolla 1):

1. Matematiikan materiaalit pois (30 sek)
   - Tehtävälomake matematiikan kansioon
   - Kynät pulpettiin
   - Laskuvälineet hyllyyn

2. Äidinkielen materiaalit esiin (30 sek)
   - Lukuluku pulpetille
   - Lukupäiväkirja lukuluokan viereen
   - Kynä valmiina

3. Aloita lukeminen (välittömästi)
   - Ei odottelua että opettaja sanoo "aloita"
   - Lue itsenäisesti kunnes opettaja antaa ohjeen

Yhteensä: ~1 minuutti siirtymä

Opettajan rooli: Skannaa luokkaa, antaa peukun ylös valmiille oppilaille
```

---

### Äidinkielestä välitunnille siirtymä

**Ulkoiluun valmistautuminen**:
```
Rutiini:

1. Lopeta nykyinen sivu (1 min varoitus annettu)
2. Merkitse kohta kirjanmerkillä
3. Lukupäiväkirjaan merkintä: Tänään luettuja sivuja
4. Kirja pulpettiin
5. Jonoon ovelle (äänentaso 0)

Yhteensä: 2 minuuttia

Ovella: Opettaja tarkistaa
- Kaikki jonossa? ✓
- Hiljaiset äänet? ✓
- Valmiina välitunnille? ✓

Sitten: Päästetään välitunnille

Hyöty: Järjestynyt poistuminen (ei ryntäys), rauhallinen käytäväsiirtymä
```

## Fyysiset siirtymät (Liikkuminen paikkojen välillä)

**Järjestynyt liikkuminen tilojen välillä**

### Pulpetilta matolle -rutiini

**Pienryhmän aika siirtymä**:
```
Opettaja ilmoittaa: "Pienryhmä 2 minuutissa. Sateenkaarivät."

Rutiini:
Rivi 1 (punainen): Kävele matolle, istu puolikaaressa
Rivi 2 (oranssi): Kun rivi 1 istunut, kävele matolle
Rivi 3 (keltainen): Kun rivi 2 istunut, kävele matolle
(jne.)

Muu luokka: Itsenäinen työ pulpeteilla (tehtävälomake tai valintatehtävä)

Aika: 2 minuuttia yhteensä (porrastettu estää ruuhkan)

Matolla: Oppilaat istuvat hiljaa, valmiina ohjeisiin
(ei tarvetta rauhoittumisaikaan)
```

---

### Käytäväsiirtymät

**Hiljainen liikkuminen**:
```
Rutiini: "Äänentaso 0" käytävällä

Ennen huoneen poistumista:
- Jonoon (johtaja ovella)
- Opettajan tarkistus: Jono suora? Hiljaista?

Käytävällä:
- Kävele jonossa (käsivarren mittainen etäisyys)
- Äänentaso 0 (ei puhumista)
- Silmät eteenpäin

Saavuttaessa uuteen tilaan (kirjasto, liikuntasali jne.):
- Ensimmäinen oppilas pysähtyy ovelle
- Kaikki tulevat hiljaa sisään
- Istu/seiso ohjeiden mukaan välittömästi

Vahvistus: "Pöytä 3, kävelitte täydellisesti käytävällä. Kiitos!"

Seuraus: Tee uudelleen jos ei tehty oikein (opettaa vastuullisuutta)
```

## Siirtymämusiikki ja merkit

**Kuulomerkit**

### Johdonmukaiset merkit

**Huomion herättäjät**:
```
Merkki 1: Kellokello
Merkitys: "Pysähdy, katso, kuuntele" (välitön huomio)
Oppilaan vastaus: Jähmettyminen, silmät opettajaan, hiljaa

Merkki 2: Taputtamisrytmi
Opettaja: Tap-tap-tap-tap-tap (rytmi)
Oppilaat: Toistavat takaisin (tap-tap-tap-tap-tap)
Merkitys: "Huomiota tarvitaan"

Merkki 3: Valot pois/päälle
Merkitys: Hätätilanne tai tärkeä ilmoitus
Oppilaan vastaus: Ehdoton hiljaisuus

Johdonmukaisuus: Sama merkki = sama vastaus (automaattista ajan mittaan)
```

---

### Siirtymälaulut (Alakoulu)

**Musiikki ajastimena**:
```
Siivouslau lu: "Siivoa, siivoa" (2-minuuttinen laulu)
Sääntö: Kaiken täytyy olla siivottu ennen laulun loppumista

Siirtymälaulu: "Kävellen, kävellen" (1-minuuttinen laulu)
Sääntö: Pääse uuteen paikkaan ennen laulun loppumista

Hyöty: Hauskaa + ennakoitavaa (oppilaat tietävät kuinka paljon aikaa heillä on)
Toimii parhaiten: Esiopetus-2. luokka (vanhemmat oppilaat suosivat visuaalisia ajastimia)
```

## Yleiset siirtymäongelmat ja ratkaisut

**Ongelmanratkaisu**

### Ongelma: Oppilaat eivät aloita seuraavaa toimintaa välittömästi

**Ratkaisu**: "Ei kuollutta aikaa" -sääntö
```
Vanha: Siirtymä matematiikkaan, oppilaat odottavat että opettaja sanoo "aloittakaa"
Uusi: Siirtymä matematiikkaan, lämmittelytehtävä jo pulpetilla, aloita välittömästi

Avain: Aina tehtävä valmiina (ei odottamista ohjeita)
```

---

### Ongelma: Materiaalit eivät ole järjestyksessä, aikaa hukataan tavaroiden etsimiseen

**Ratkaisu**: Ennalta sijoitetut materiaalit
```
Ennen siirtymää: Opettaja (tai apulainen) jakaa seuraavan tehtävälomakkeen
jokaiselle pulpetille

Esimerkki:
- Äidinkielen aikana (klo 10:00): Oppilas-apulainen asettaa matematiikan
  tehtävälomakkeen jokaiselle pulpetille
- Matematiikan aikana (klo 10:30): Tehtävälomakkeet jo siellä (nolla jakoaikaa)

Vaihtoehto: Materiaalikori jokaisella pöydällä (oppilaat ottavat mitä tarvitsevat)
```

---

### Ongelma: Oppilaat juttelevat siirtymien aikana

**Ratkaisu**: Äänentasojärjestelmä + kehuminen
```
Äänentasojuliste:
0 = Hiljaisuus
1 = Kuiskaus
2 = Pöytäkeskustelu
3 = Esitysääni

Siirtymäsääntö: "Äänentaso 0 kaikkien siirtymien aikana"

Vahvistus: "Pöytä 2 pysyi äänentasossa 0. Hienosti tehty!"

Seuraus: Tee siirtymä uudelleen jos liian äänekästä (opettaa odotusta)
```

## Yhteenveto

Sujuvat siirtymät säästävät **20-30 minuuttia päivässä** (Burns, 2007) - selkeät rutiinit palauttavat lähes 3 viikkoa opetusta vuosittain.

**3-vaiheinen malli**: Lopeta (5 sek) → Valmistaudu (30-60 sek) → Aloita (5 sek), yhteensä 40-70 sekuntia per siirtymä

**Ensimmäinen viikon opetus**: Mallinna jokainen siirtymä, harjoittele 3-5 kertaa, hioa koko vuoden (sijoitus maksaa itsensä takaisin)

**Aamun lämmittelytehtävät**: Aamumonisteet (saapuminen → välitön sitoutuminen), kertausmateriaali (5-10 tehtävää), 10 minuutin rutiini

**Lopetusliput**: Viimeiset 5 minuuttia (reflektoi oppimista), sujuva kotiinlähtö (oppilaat istuvat rauhallisina)

**Ajastimet**: Heijastettu 2 minuutin ajastin (visuaalinen ajanhallinta), voita ajastin -haaste (pelillistäminen)

**Oppiainesirtymät**: Matematiikka → Äidinkieli (1 min rutiini), Äidinkieli → Välitunti (2 min järjestynyt poistuminen)

**Nopeille tekijöille -järjestelmä**: Tarkista vastaukset → rikastustehtävä → pulmat → lukeminen (aina jotain tekemistä)

**Fyysinen liikkuminen**: Sateenkaarivät (porrastettu mattasiirtymä), äänentaso 0 käytävillä (järjestynyt liikkuminen)

**Merkit**: Kellokello (pysähdy-katso-kuuntele), taputtamisrytmi (huomio), valot (hätätilanne), johdonmukaisuus = automaattinen vastaus

**Seuranta**: Kirjaa siirtymäajat viikoittain (tunnista hitaat siirtymät), toteuta ratkaisut (dataan perustuva parantaminen)

**Tutkimus**: 20-30 min säästetty päivässä rutiineilla (Burns, 2007)
