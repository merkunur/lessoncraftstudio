## APP-SPECIFIC FEATURES

### Exercise Modes (2 modes)
1. **Fill-in-the-Blank** - Students write the preposition
2. **Multiple Choice** - Students circle the correct image showing the preposition

### 8 Prepositions Available (with Portuguese translations)
| English | Portuguese |
|---------|------------|
| in | em |
| on top of | em cima de |
| under | embaixo de |
| next to | ao lado de |
| behind | atrás de |
| between | entre |
| above | acima de |
| in front of | na frente de |

### Configuration Options
- Number of exercises: 1-8 per worksheet
- Prepositions checklist: Select which prepositions to include
- Include Name/Date fields option
- Custom description for Multiple Choice mode

### Item Selection
- Manual Selection: Choose specific images from themed categories
- All Themes: Auto-generate from entire 3000+ image library
- Theme dropdown with search functionality
- Selection counter (Selected: 0/8)

### Shape Replacement (Reference Objects)
- Default shapes: circle, cube, cylinder, heart, hexagon, square, star, triangle
- Manual Shape Selection: Choose custom shape images
- All Themes mode for automatic selection
- Search functionality for shapes
