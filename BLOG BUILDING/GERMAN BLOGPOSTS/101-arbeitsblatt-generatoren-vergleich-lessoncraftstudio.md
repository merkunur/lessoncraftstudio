## Geschwindigkeitsvergleich

**Zeitaufwand für ein Arbeitsblatt**

### LessonCraftStudio: 42 Sekunden

**Ablauf**:
```
1. Generator auswählen (5 Sek.)
2. Parameter eingeben (20 Sek.)
   - Thema/Vokabular
   - Schwierigkeitsstufe
   - Layout-Einstellungen
3. Generieren (2 Sek.)
4. PDF herunterladen (15 Sek.)

Gesamt: 42 Sekunden durchschnittlich
Ergebnis: Vollständig angepasstes, druckfertiges Arbeitsblatt

Beispiel: Mathe-Textaufgaben
- Eingabe: 20 Aufgaben, Addition/Subtraktion, 2-stellige Zahlen
- Generierung: 42 Sekunden
- Resultat: Individuelles Arbeitsblatt mit einzigartigen Aufgaben
```

---

### Education.com: 2-3 Minuten

**Ablauf**:
```
1. Bibliothek durchsuchen (30-60 Sek. um passendes Blatt zu finden)
2. Vorschau ansehen (15 Sek.)
3. Falls verfügbar anpassen (30-45 Sek.)
4. Herunterladen (15 Sek.)

Gesamt: 2-3 Minuten
Einschränkung: Auswahl aus vorgefertigten Materialien (nicht vollständig individuell)
```

---

### Teachers Pay Teachers: 5-15 Minuten

**Ablauf**:
```
1. Nach Arbeitsblatt-Typ suchen (2-5 Min. Durchsicht)
2. Mehrere Optionen prüfen (3-5 Min.)
3. Kaufen (1 Min.)
4. Herunterladen (30 Sek.)
5. Oft noch Bearbeitung nötig (3-5 Min. in PowerPoint)

Gesamt: 5-15 Minuten
Vorteile: Hochwertig, von Lehrern erstellt
Nachteile: Zeitaufwendige Suche, Kosten pro Ressource
```

---

### Twinkl: 1-2 Minuten

**Ablauf**:
```
1. Datenbank durchsuchen (30 Sek.)
2. Vorschau (15 Sek.)
3. Vorlage anpassen (30-45 Sek.)
4. Herunterladen (15 Sek.)

Gesamt: 1-2 Minuten
Hinweis: Vorgefertigte Vorlagen (weniger flexibel als Generator)
```

---

### WorksheetWorks.com: 1 Minute

**Ablauf**:
```
1. Generator auswählen (10 Sek.)
2. Parameter festlegen (30 Sek.)
3. Generieren + herunterladen (20 Sek.)

Gesamt: ~1 Minute
Vergleichbare Geschwindigkeit: Ähnlich wie LessonCraftStudio
```

---

### Super Teacher Worksheets: 1-2 Minuten

**Ablauf**:
```
1. Nach Klassenstufe/Fach durchsuchen (30-60 Sek.)
2. Arbeitsblatt auswählen (15 Sek.)
3. Drucken/herunterladen (15 Sek.)

Gesamt: 1-2 Minuten
Einschränkung: Feste Arbeitsblätter (keine Anpassung)
```

**Geschwindigkeitssieger**: LessonCraftStudio & WorksheetWorks (schnellste mit Anpassungsoptionen)

## Anpassungsflexibilität

**Wie viel Kontrolle haben Lehrkräfte?**

### LessonCraftStudio: Umfangreiche Anpassung

**Optionen pro Arbeitsblatt**:
```
Inhalt: Eigenes Vokabular, Aufgaben usw. eingeben
Schwierigkeit: Komplexitätsstufen anpassen
Layout: Rastergröße, Anzahl der Aufgaben, Format
Bilder: Aus Bibliothek wählen (Bild-Generatoren)
Sprache: Mehrere Sprachen unterstützt

Beispiel: Wortsuche
- Eigene Wortliste (beliebige Wörter)
- Rastergröße (10×10, 15×15, 20×20)
- Schwierigkeit (leicht/mittel/schwer)
- Druckoptionen (mit/ohne Wortliste)

Flexibilität: Hoch (erstellen Sie genau was Sie brauchen)
```

**Anpassung**:

Insgesamt: Geringe Anpassung
```

Insgesamt: Moderate Anpassung

```
