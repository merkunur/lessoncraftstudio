## Johdanto: Paikkahahmotus ja menestys STEM-aloilla

**Visuaalis-spatiaalisten taitojen määritelmä**: Kyky käsitellä mielessä esineiden asentoa, pyörittää niitä ja ymmärtää niiden välisiä paikkasuhteita

**Keskeiset osa-alueet**:
- **Mentaalinen pyöritys**: Esineen hahmottaminen eri kulmista mielessä
- **Osa-kokonaisuushahmotus**: Ymmärrys siitä, miten pienet osat muodostavat kokonaisen hahmon
- **Paikkasuhteen hahmotus**: Sijainnin, etäisyyden ja suunnan ymmärtäminen
- **Visuaalinen muisti**: Visuaalisen tiedon tarkka muistaminen
- **Hahmo-tausta -erottelu**: Kohteen erottaminen taustasta

**Yhteys STEM-uraan**:
```
Tutkimustulos (Wai ym., 2009):
Visuaalis-spatiaalinen kyky 13-vuotiaana ennustaa STEM-uraa 33-vuotiaana
Korrelaatio: r = 0,52 (VAHVA ennustaja)

Vertailu:
Matemaattinen kyky → STEM-ura: r = 0,45
Kielellinen kyky → STEM-ura: r = 0,32
Visuaalis-spatiaalinen kyky → STEM-ura: r = 0,52 (KORKEIN)
```

**Miksi visuaalis-spatiaaliset taidot ovat tärkeitä STEM-aloilla**:
- **Insinöörityö**: 3D-mallien mentaalinen pyöritys (CAD-suunnittelu)
- **Arkkitehtuuri**: Rakennusten visuaalinen hahmottaminen
- **Kemia**: Kolmiulotteisten molekyylirakenteiden ymmärtäminen
- **Fysiikka**: Voimien ja liikeratojen visualisointi
- **Matematiikka**: Geometria, koordinaatisto, paikkasuhteisiin liittyvät ongelmat

**Perinteisen koulutuksen ongelma**:
```
Koulun painotukset:
- Kielelliset taidot: 60% opetusajasta (lukeminen, kirjoittaminen)
- Laskutaidot: 30% (aritmetiikka, algebra)
- Visuaalis-spatiaaliset taidot: 10% (kuvataide, satunnainen geometria)

Tulos: Paikkahahmotustaidot jäävät kehittymättä (huolimatta korkeasta STEM-ennustearvosta)
```

**Tutkimus** (Uttal ym., 2013): Visuaalis-spatiaaliset taidot ovat **erittäin kehitettävissä** (harjoittelulla parannettavissa)
- Meta-analyysi: Paikkahahmotuksen harjoittelu parantaa taitoja keskimäärin **47%**
- **Siirtovaikutus**: Yhden tehtävätyypin harjoittelu parantaa muita paikkahahmotustaitoja (yleistettävissä)

**Ratkaisu**: Järjestelmällinen harjoittelu visuaalis-spatiaalisten tehtävien avulla

## 7 paikkahahmotusta kehittävää tehtävätyyppiä

### Tehtävätyyppi 1: Ruutupiirustus (App 024) ⭐ PARAS PAIKKAHAHMOTUKSEN KEHITTÄJÄ

**Miksi ruutupiirustus on PARAS paikkahahmotuksen kehittäjä**:
- **Osa-kokonaisuushahmotus**: Yksittäiset ruudut muodostavat kokonaisen kuvan
- **Mittasuhteiden ymmärtäminen**: Pieni ruutu = pieni piirrosalue (mittakaavan hahmotus)
- **Koordinaatisto**: Ruutu B3 vastaa pistettä (2,3) tasossa
- **Visuaalinen muisti**: Malliruudun sisällön muistaminen piirtäessä
- **Tarkkuus paikkasuhteissa**: Tarkka sijoittelu on tärkeää

---

#### Kehittyvät taidot

**Taito 1: Mittakaavan hahmotus**

**Käsite**: Jos malliruutu on 1 tuuma ja piirrosruutu 2 tuumaa, yksityiskohdat on skaalattava 2×

**Ruutupiirustus opettaa**:
```
Malliruutu: Pieni ympyrä (halkaisija ¼ tuumaa)
Piirrosruutu: Isompi tila (½ tuumaa)
Lapsen tehtävä: Skaalata ympyrä oikein (piirtää ½ tuuman halkaisijalla)
Paikkahahmotus: Kokosuhteiden ymmärtäminen
```

**Sovellus STEM-aloilla**:
- Mittakaavataulukot (pohjapiirrokset)
- Kartanluku (1 tuuma = 10 mailia)
- Tieteelliset kaaviot (mikroskoopin 400× suurennus)

---

**Taito 2: Osa-kokonaisuuksien yhdistäminen**

**Käsite**: Yksittäiset osat oikeassa paikassa muodostavat kokonaisuuden

**Kokemus ruutupiirustuksessa**:
```
Ruudut erikseen: Näyttävät satunnaisilta (kaari täällä, piste tuolla)
Ruudut yhdistettynä: Muodostavat täydellisen kasvot (silmä, nenä, suu oikeissa paikoissa)
Lapsi oppii: Paikkasuhteet luovat merkityksen
```

**Sovellus STEM-aloilla**:
- Piirikaaviot (komponentit oikeissa paikoissa)
- Molekyylirakenteet (atomit paikallisessa järjestyksessä)
- Ekosysteemikaaviot (organismit sijoitettuina eri kerroksiin)

---

**Taito 3: Mentaalinen pyöritys** (Peilausmoodi)

**Käsite**: Esineen hahmottaminen eri asennosta

**Peilausmoodin ruutupiirustus**:
```
Perus: Kopioi ruutu A1 piirrosruutuun A1 (ei pyöritystä)
Vaaka-peilaus: Kopioi ruutu A1 ruutuun J1, peilattuna vasen-oikea (mentaalinen kääntö)
Pysty-peilaus: Kopioi ruutu A1 ruutuun A10, peilattuna ylös-alas
Molemmat peilaukset: 180° pyöritys (erittäin haastava)
```

**Tutkimus** (Uttal ym., 2013): Mentaalisen pyörityksen harjoittelu → **47% paikkahahmotuksen paraneminen**, siirtovaikutus geometriaan (35% parempi)

**Asetukset**:
- Ruudukko: 7×7 - 10×10
- Peilausmoodi: Valinnainen (edistynyt haaste)
- Älykäs ruudun tunnistus: Päällä

**Tehtävän kesto**: 60-90 minuuttia

**Hinnoittelu**: Vain täysi käyttöoikeus (240€/vuosi)

### Tehtävätyyppi 2: Ruutupalapelit (App 027)

**Miksi ruutupalapelit kehittävät paikkahahmotusta**:
- **Visuaalinen muisti**: Palasen yksityiskohtien muistaminen
- **Pyörimisen tunnistus**: Pyöräytettyjen palasten tunnistaminen (mentaalinen pyöritys)
- **Paikan yhdistäminen**: Oikean paikan löytäminen
- **Visuaalinen vertailu**: Palasen vertaaminen useisiin mahdollisiin paikkoihin

**Haaste pyörimisen kanssa**:
```
Ilman pyörimistä:
Palanen näyttää: Omenan vasen yläkulma
Lapsi: Vertaa visuaalisesti vastaavaan ruutukohtaan (suora vertailu)

Pyörimisen kanssa:
Palanen näyttää: Omenan vasen yläkulma, PYÖRITETTYNÄ 90°
Lapsen tehtävä:
1. Pyörittää palanen mielessään takaisin alkuasentoon
2. Tunnistaa, mikä osa kuvasta se on
3. Löytää oikean ruutupaikan
Paikkahahmotuksen vaatimus: 3× korkeampi
```

**Asetukset**:
- Ruudukko: 5×5 tai 6×6 (25-36 palaa)
- Pyöritys: Kyllä (edistynyt) tai Ei (aloittelijoille)
- Monimutkaisuus: Yksityiskohtainen kuva

**Tehtävän kesto**: 40-70 minuuttia

**Hinnoittelu**: Peruspakett tai täysi käyttöoikeus

### Tehtävätyyppi 3: Kuvapolku (Sokkelot - App 011)

**Miksi sokkelot kehittävät paikkahahmotusta**:
- **Paikkanavigointi**: Reitin suunnittelu 2D-tilassa
- **Mentaalinen kartoitus**: Sokkelon kartan pitäminen mielessä
- **Paikkaperusteinen suunnittelu**: Eteenpäin katsominen, umpikujien ennakointi
- **Suuntien hahmotus**: Ylös/alas/vasen/oikea -suhteiden ymmärtäminen

**Paikkaperusteisen suunnittelun prosessi**:
```
Lapsi lähestyy sokkeloa:
1. Visuaalinen kartoitus: Tunnistaa alku- ja loppupiste, mahdolliset reitit
2. Mentaalinen simulaatio: "Jos menen oikealle, sitten ylös... se on umpikuja"
3. Paikkaperusteinen suunnittelu: "Vasen reitti näyttää selkeältä loppupisteeseen"
4. Toteutus: Piirtää reitin suunnitelman mukaisesti
5. Virheiden korjaus: Törmää umpikujaan, suunnittelee uudelleen
```

**Sovellus STEM-aloilla**:
- Algoritmien reitinhaku (tietojenkäsittelytiede)
- Piirisuunnittelu (sähkötekniikka)
- Navigointi (maantiede, fysiikka)

**Asetukset**:
- Ruudukko: 5×5 - 7×7
- Vaikeusaste: Keskitaso-korkea (useita vääriä reittejä)

**Tehtävän kesto**: 15-25 minuuttia

**Hinnoittelu**: Peruspakett tai täysi käyttöoikeus

### Tehtävätyyppi 4: Varjojen yhdistäminen (App 009)

**Miksi varjojen yhdistäminen kehittää paikkahahmotusta**:
- **Hahmo-tausta -erottelu**: Varjon erottaminen taustasta (paikkaperusteinen)
- **Muodon pysyvyys**: Saman esineen tunnistaminen ulkonäön muutoksesta huolimatta (paikkaperusteinen muunnos)
- **Havainnon pysyvyys**: Esineen identiteetti säilyy visuaalisesta muutoksesta huolimatta

**Paikkaperusteinen päättely**:
```
Tehtävä: Yhdistä koira koiran varjoon
Paikkaperusteinen käsittely:
1. Tunnista koiran ääriviivat (paikkaperusteinen rajaus)
2. Tarkastele varjoja (paikkaperusteinen haku)
3. Mentaalinen päällekkäisyys: "Osuisiko koiran ääriviiva tähän varjoon?" (paikkaperusteinen limittäminen)
4. Yhteys löytyy (paikkaperusteinen vastaavuus)
```

**Sovellus STEM-aloilla**:
- Insinööripiirustukset (ortografiset projektiokuvat)
- Tähtitiede (varjot paljastavat esineen muodon)
- Geometria (yhteneväisyys, samankaltaisuus)

**Tehtävän kesto**: 15-20 minuuttia

**Hinnoittelu**: Peruspakett tai täysi käyttöoikeus

### Tehtävätyyppi 5: Esineiden etsiminen (I Spy) (App 026)

**Miksi visuaalinen haku kehittää paikkahahmotusta**:
- **Paikkaperusteinen kartoitus**: Järjestelmälliset hakumallit (vasen→oikea, ylä→ala)
- **Paikkaperusteinen muisti**: Läpikäytyjen kohtien muistaminen (ei etsitä uudelleen)
- **Paikkaperusteinen tarkkaavuus**: Keskittyminen tiettyyn alueeseen häiriötekijöistä huolimatta

**Järjestelmällinen etsintästrategia**:
```
Järjestämätön etsintä (tehoton):
Lapsi: Katsoo satunnaisesti ympärilleen, etsii samoista paikoista useita kertoja
Tulos: Kohteita jää löytymättä, ajan hukkaantuminen

Järjestelmällinen paikkahaku (tehokas):
Lapsi: Vasen ylä → oikea ylä (rivi 1), seuraava rivi vasen→oikea, toisto
Tulos: Täydellinen kattavuus, ei kohteita hukassa
Paikkataito: Järjestelmällinen paikkaperusteinen kartoitus
```

**Asetukset**:
- Esineitä: 20-30
- Etsittäviä: 8-12
- Opetettava strategia: Järjestelmällinen paikkaperusteinen kartoitus

**Tehtävän kesto**: 25-40 minuuttia

**Hinnoittelu**: Peruspakett tai täysi käyttöoikeus

### Tehtävätyyppi 6: Kuviosarjat (App 006) - PAIKKASARJAT

**Miksi paikkasarjat kehittävät taitoja**:
- **Paikkajärjestys**: Järjestyksen ymmärtäminen tilassa
- **Paikan ennustaminen**: Seuraavan paikkasuunnitelman ennakointi
- **Paikkasuunnitelman tunnistaminen**: Järjestävän periaatteen tunnistaminen

**Paikkasarjat vs. numerosarjat**:
```
Numerosarja: 2, 4, 6, 8 (ajallinen järjestys)
Paikkasarja: ●■●■ (paikkajärjestely)

Paikkavaatimus: Paikan + identiteetin käsittely yhtä aikaa
Tulos: Paikkaperusteisen päättelyn kehittyminen
```

**Sovellus STEM-aloilla**:
- Tesselloinnit (geometria)
- Kiderakenteet (kemia)
- DNA-sekvenssit (biologia - paikkajärjestys on tärkeä)

**Tehtävän kesto**: 20-30 minuuttia

**Hinnoittelu**: Peruspakett tai täysi käyttöoikeus

### Tehtävätyyppi 7: Iso-pieni -vertailu (App 019)

**Miksi kokovertailu kehittää paikkahahmotusta**:
- **Paikkamittakaava**: Paikkamittojen vertailu
- **Suhteellinen koko**: Vertailuun perustuvien paikkasuhteiden ymmärtäminen
- **Visuaalinen arviointi**: Paikkaperusteisten erojen arviointi

**Paikkaperusteinen päättely**:
```
Tehtävä: Kumpi norsu on suurempi?
Paikkaperusteinen käsittely:
1. Mittaa visuaalinen kulma (kuinka paljon tilaa esine vie)
2. Vertaa paikkamittoja (norsu A vs. norsu B)
3. Paikkaperusteinen arvio: A > B
Taito: Paikkamittakaavan vertailu
```

**Sovellus STEM-aloilla**:
- Mittakaava (kartat, kaaviot)
- Mittaaminen (arviointi ennen laskemista)
- Mittasuhteet (suhdeluvut paikkasuhteiden näkökulmasta)

**Tehtävän kesto**: 10-15 minuuttia

**Hinnoittelu**: Peruspakett tai täysi käyttöoikeus
