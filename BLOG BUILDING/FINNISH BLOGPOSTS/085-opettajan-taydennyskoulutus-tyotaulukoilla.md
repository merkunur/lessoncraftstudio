## Johdanto: Opettajat Oppimisen Ytimessä

**Tehokkaan täydennyskoulutuksen tunnuspiirteet** (Darling-Hammond ym., 2017):
1. Sisältökeskeisyys (konkreettiset opetusstrategiat)
2. Aktiivinen oppiminen (käytäntöä, ei pelkkää luentoa)
3. Yhteistoiminnallisuus (opettajat oppivat yhdessä)
4. Jatkuvuus (ei yksittäisiä kertakoulutuksia)
5. Työhön integroitu (välitön soveltaminen luokkahuoneessa)

**Tutkimustulos** (Yoon ym., 2007): Yli 30 tunnin aktiivinen täydennyskoulutus parantaa oppilaiden saavutuksia **21 prosenttiyksikköä**

**Perinteisen täydennyskoulutuksen ongelma**:
```
Yhden päivän koulutus:
Aamupäivä: Kouluttaja luennoi strategiasta (passiivista kuuntelua)
Lounas: Verkostoitumista
Iltapäivä: Lisää luentoa
Opettajat lähtevät: Innostuneita, mutta ilman käytännön työkaluja
Maanantai: Paluu samoihin rutiineihin (mikään ei muutu)

Tulos: Nolla toteutusta (innostus ei ole sama kuin soveltaminen)
```

**Generaattoripohjainen täydennyskoulutusratkaisu**:
```
Täydennyskoulutustyöpaja (3 tuntia):
Tunti 1: Lyhyt johdanto strategiaan (15 min luentoa, 45 min harjoittelua)
Tunti 2: Opettajat luovat materiaaleja (tuottavat 10 työtaulukkoa 7 minuutissa)
Tunti 3: Vertaisjakaminen + toteutuksen suunnittelu

Maanantai: Opettajat käyttävät luotuja materiaaleja välittömästi
Viikko 2: Seurantakoulutus (tulosten jakaminen, ongelmanratkaisu)

Tulos: Välitön soveltaminen (materiaalit valmiina, strategia harjoiteltu)
```

## Työpajamalli: Eriyttäminen Käytännössä

**Työpajan tavoite**: Opettajat oppivat luomaan tasoitettuja työtaulukoita

### Tunti 1: Strategian Oppiminen (60 minuuttia)

**15 minuutin mini-luento**:
```
Kouluttaja selittää:
- Mitä eriyttäminen on? (oppilaiden kohtaaminen heidän omalla tasollaan)
- Miksi tasoistetut työtaulukot? (yksi aihe, kolme vaikeustasoa)
- Miten tasoittaa: Monimutkaisuus, tuki, tehtävien määrä

Opettajat: Tekevät muistiinpanoja (eivät passiivisia - täyttävät lomaketta)
```

**45 minuutin käytännön harjoitus**:
```
Kouluttaja: "Nyt TE luotte tasoitettuja työtaulukoita"

Vaihe 1 (10 min): Generaattorin käyttöönotto
- Opettajat kirjautuvat sisään
- Kouluttaja demonstroi: "Näin säädetään asetuksia"

Vaihe 2 (15 min): Luo 3 tasoa
- Taso 1: Tukea tarvitsevat oppilaat (10 tehtävää, kuvatila)
- Taso 2: Keskitasoiset (15 tehtävää, sekoitettu)
- Taso 3: Edistyneet (25 tehtävää, monimutkainen)
- Aika: 126 sekuntia kaikkien 3 tason luomiseen

Vaihe 3 (20 min): Tulosten tarkastelu
- Opettajat tarkastelevat luomiaan työtaulukoita
- Parikeskustelu: "Miten nämä eroavat toisistaan?"
- Tunnistetaan kunkin tason ominaisuudet

Aktiivista oppimista: Opettajat TEKEVÄT strategiaa (eivät vain kuule siitä)
```

---

### Tunti 2: Materiaalien Luominen (60 minuuttia)

**Itsenäinen tuotantoaika**:
```
Kouluttaja: "Luokaa ensi viikon materiaalit luokkahuoneeseenne"

Opettajat tuottavat:
- Maanantai: 3-tasoinen matematiikan työtaulukko (2 min)
- Tiistai: 3-tasoinen sanasto (2 min)
- Keskiviikko: 3-tasoinen luetun ymmärtäminen (2 min)
- Torstai: 3-tasoinen luonnontiede (2 min)
- Perjantai: Arviointi (2 min)

Yhteensä: 10 minuuttia koko viikon eriyttäville materiaaleille

Jäljellä oleva aika (50 min):
- Materiaalien järjestely (tulostus, merkintä, arkistointi)
- Toteutuksen suunnittelu (mitkä oppilaat saavat minkä tason?)
- Mallivastausten luominen
```

**Opettajan kommentti työpajassa**:
```
"Olen halunnut eriyttää, mutta minulla ei ole koskaan ollut aikaa luoda kolmea versiota kaikesta. 10 minuutissa minulla on koko viikon materiaalit! Tämä muuttaa kaiken."
```

---

### Tunti 3: Yhteistoiminnallinen Suunnittelu (60 minuuttia)

**Vuosiluokkatiimin suunnittelu**:
```
2. luokan opettajat (4 opettajaa):
- Jaetaan tuotettuja materiaaleja
- Keskustellaan toteutusstrategioista
- Suunnitellaan: "Kokeillaan kaikki eriyttämistä ma-ke, tavataan torstaina keskustelemaan"
- Sovitaan: Seurantakoulutus 2 viikon kuluttua

Yhteistoiminnallista: Oppiminen vertaisilta (yhteistä ongelmanratkaisua)
```

**Sitoutuminen toimintaan**:
```
Jokainen opettaja täyttää:
"Tällä viikolla toteutan _____________ (strategian)
luokkahuoneessani _____________ (konkreettisen toimenpiteen) avulla.
Tiedän sen toimineen, kun _____________ (mitattava tulos)"

Esimerkki:
"Tällä viikolla toteutan tasoitetut matematiikan työtaulukot luokkahuoneessani antamalla Taso 1:n viidelle tukea tarvitsevalle oppilaalleni, Taso 2:n 18 keskitasoiselle oppilaalleni ja Taso 3:n seitsemälle edistyneelle oppilaalleni. Tiedän sen toimineen, kun kaikki oppilaat suorittavat työtaulukonsa onnistuneesti (80%+ tarkkuus)."

Vastuullisuus: Kirjallinen sitoumus (lisää toteutusta)
```

## Seurantakoulutus (2 Viikkoa Myöhemmin)

**30 minuutin tarkistus**:

### Tulosten Jakaminen

**Opettajat raportoivat toteutuksesta**:
```
Opettaja A: "Kokeilin tasoitettuja työtaulukoita. Tukea tarvitsevat oppilaani suorittivat 90% (yleensä 60%). He sanoivat, että se tuntui 'helpommalta' - todellisuudessa se oli vain oikean kokoista heille!"

Opettaja B: "Edistyneet oppilaani rakastivat Taso 3:a. He valmistuivat ja kysyivät, 'Saanko tehdä lisää?' Ensimmäinen kerta kun heitä on haastettu!"

Opettaja C: "Kamppailen sen kanssa, mitkä oppilaat kuuluvat mille tasolle. Jotkut loukkaantuivat saaessaan 'helppoja' tehtäviä."

Kouluttaja: Ratkaisee Ongelman C
"Kokeile antaa oppilaiden valita itse taso. Aseta kolme lokeroa nimikkeillä 'Harjoittelu, Perus, Haaste' - ei negatiivisia merkintöjä. Oppilaat valitsevat."

Vertaisoppimista: Opettajat ratkaisevat toistensa ongelmia
```

---

### Säätö ja Hiominen

**Iteratiivinen parantaminen**:
```
Kouluttaja: "Kokemustenne perusteella, mitä muuttaisitte?"

Opettajat:
- "Lisäisin neljännen 'kunnia'-tason todella edistyneille"
- "Sisällyttäisin mallivastaukset kaikille tasoille (itsearviointia varten)"
- "Luoisin visuaalisen vihjeen (värikoodattu), jotta oppilaat tietävät minkä tason"

Toiminta: Opettajat tuottavat parannettuja materiaaleja (soveltavat palautetta)
Aika: 15 minuuttia hiomiseen
```

## Ammatilliset Oppimisyhteisöt (AOY)

**Jatkuva yhteistoiminnallinen oppiminen**

### Kuukausittainen AOY-protokolla

**Viikko 1: Tuota ja Jaa**
```
Kokous (45 min):
- Jokainen opettaja tuottaa materiaaleja tulevaa kokonaisuutta varten
- Jaetaan tiimille (ladataan yhteiselle asemalle)
- Keskustellaan: "Mitä taitoja olemme tavoittelemassa?"

Tulos: Tiimilla on täydelliset kokonaisuusmateriaalit (yhteistoiminnallinen tehokkuus)
```

**Viikko 2: Toteuta**
```
Ei kokousta: Opettajat käyttävät materiaaleja luokkahuoneissa
Tiedon keräys: Säilytetään oppilaiden työnäytteitä
```

**Viikko 3: Analysoi Tulokset**
```
Kokous (45 min):
- Tuodaan: Oppilaiden työnäytteitä
- Analysoidaan: "Mikä työtaulukko oli tehokkain? Miksi?"
- Data: Verrataan oppilaiden suoritusta tiimin kesken

Kouluttaja: "Opettaja A:n oppilaat saivat 85% tästä työtaulukosta. Opettaja B:n oppilaat saivat 70%. Mikä oli erilaista?"

Keskustelu: Opetusstrategiat (ei vain materiaalit)
Oppiminen: Mikä toimii, mikä ei (näyttöön perustuva)
```

**Viikko 4: Hio Käytäntöä**
```
Kokous (45 min):
- Datan perusteella: Mitä muutoksia seuraavaan kokonaisuuteen?
- Toimintasuunnittelu: Konkreettiset strategiat kokeiltavaksi
- Tuota: Seuraavan kokonaisuuden materiaalit (sovelletaan oppimista)

Sykli toistuu: Jatkuvaa parantamista
```

## Mallitunnit

**Täydennyskoulutus mallioppimisen kautta**

### Mallitunnin Rakenne

**Ennakkohavainnointi** (10 minuuttia):
```
Demonstroiva opettaja: Selittää tuntisuunnitelman havainnoijille
"Tänään opetan murtolukuja käyttäen tasoitettuja työtaulukoita. Tarkatkaa, miten:
1. Johdatan käsitteen koko luokalle
2. Jaan eriyttävät työtaulukot
3. Seuraan jokaista ryhmää
4. Säädän tukea tarpeiden mukaan"

Havainnoijat: Saavat havainnointipainopisteet (mitä etsiä)
```

**Tunnin Havainnointi** (30 minuuttia):
```
Demonstroiva opettaja: Opettaa kollegoiden havainnoidessa

Havainnoijat merkitsevät:
- Miten opettaja johdattaa strategian?
- Miten oppilaat reagoivat eriyttäviin materiaaleihin?
- Mitä haasteita ilmenee?
- Mitkä hallintastrategiat ovat tehokkaita?
```

**Jälkipuinti** (20 minuuttia):
```
Reflektointikysymykset:
- "Mitä huomasitte oppilaiden sitoutumisesta?"
- "Miten eriyttäminen vaikutti oppimiseen?"
- "Mitä kokeilisitte omassa luokkahuoneessanne?"
- "Mitä kysymyksiä teillä on?"

Demonstroiva opettaja: Jakaa päätösten taustalla olevan ajattelun
Havainnoijat: Sisäistävät strategian (näkivät sen toiminnassa)
```

## Vertaisohjauspari

**Työhön integroitu tuki**

### Ohjaussykli

**Viikko 1: Yhteissuunnittelu**
```
Ohjaaja ja opettaja tapaavat (30 min):
- Tunnistetaan: Painopistetaito (esim. eriyttäminen)
- Tuotetaan: Materiaaleja yhdessä (työtaulukot viikolle)
- Suunnitellaan: Miltä eriyttäminen näyttää luokkahuoneessa?
```

**Viikko 2: Havainnointi**
```
Ohjaaja: Havainnoi opettajan strategian toteutusta (20 min)
Tekee muistiinpanoja: Konkreettista näyttöä strategian käytöstä
```

**Viikko 3: Palautekeskustelu**
```
Ohjaaja ja opettaja tapaavat (30 min):
- Ohjaaja: "Huomasin, että annoit kolme tasoa. Oppilaat vaikuttivat sitoutuneilta."
- Opettaja: "Kamppailen ajan kanssa - kestää liian kauan jakaa kolme versiota"
- Ohjaaja: "Kokeile värikoodausta. Sininen lokero Taso 1:lle, punainen Taso 2:lle, vihreä Taso 3:lle. Oppilaat ottavat oman värinsä."
- Opettaja: "Se säästäisi 5 minuuttia!"

Toimintasuunnitelma: Opettaja kokeilee ohjaajan ehdotusta ensi viikolla
```

**Viikko 4: Seuranta**
```
Ohjaaja: Havainnoi uudelleen (tarkistaa ehdotuksen toteutus)
Ohjaaja: "Värikoodaus toimi! Jako kesti 30 sekuntia 5 minuutin sijaan."

Sykli: Toistuu uuden painopistetaidon kanssa
```

## Uusien Opettajien Perehdytys

**Tuki aloitteleville opettajille**

### Perehdytysohjelman Rakenne

**Elokuun Esipalveluviikko**:
```
Päivä 1: Johdatus generaattoreihin (käytännön harjoittelu, 3 tuntia)
Päivä 2: Luo ensimmäisen kuukauden materiaalit (elo-syyskuu, 2 tuntia)
Päivä 3: Organisointijärjestelmät (arkistointi, digitaalinen tallennus, 1 tunti)

Tulos: Uudet opettajat aloittavat valmiilla materiaaleilla (vähentää ylikuormitusta)
```

**Kuukausittaiset Tarkistukset**:
```
Mentoropettaja tapaa uuden opettajan (30 min kuukausittain)

Syyskuu: "Miten työtaulukot toimivat? Tarvitsetko lisätukea?"
Lokakuu: "Tuotetaan eriyttäviä materiaaleja yhdessä"
Marraskuu: "Luodaan arviointeja 2. jaksolle"

Jatkuva tuki: Uusi opettaja ei ole koskaan yksin
```

## Rehtoreiden Täydennyskoulutus: Opettajien Tukeminen

**Täydennyskoulutus rehtoreille**

### Rehtorien Työpaja: Tehokkaan Eriyttämisen Tunnistaminen

**Tavoite**: Rehtorit oppivat, mitä etsiä oppituntikierroksilla

**Koulutussisältö**:
```
Mitä havainnoida:
✓ Useita työtaulukon versioita näkyvissä (eriyttäminen tapahtuu)
✓ Oppilaat sitoutuneita sopivalla tasolla (ei liian helppoa/vaikeaa)
✓ Opettaja kiertää strategisesti (antaa kohdennettua tukea)
✓ Materiaalit järjestetty/saavutettavissa (järjestelmät paikallaan)

Mitä kysyä opettajilta:
"Miten päätit, mitkä oppilaat saavat minkä tason?"
"Mikä data ohjasi eriyttämistäsi?"
"Miten seuraat oppilaiden edistymistä eri tasoilla?"

Palaute: Konkreettinen, toimiva (ei epämääräistä kehuja)
```
