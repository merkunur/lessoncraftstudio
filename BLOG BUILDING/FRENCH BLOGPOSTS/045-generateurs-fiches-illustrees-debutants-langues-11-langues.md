## Introduction : L'Effet de Supériorité de l'Image en Acquisition Linguistique

**Le défi des grands débutants** : Compréhension orale et écrite nulle dans la langue cible

**Problème des fiches traditionnelles** :
```
Consigne en français : "Un grand animal gris avec une trompe"
Apprenant A1 anglophone : Ne comprend pas (ignore "grand", "animal", "gris", "trompe")
Résultat : Fiche inutilisable sans médiation
```

**Solution** : Générateurs basés sur l'image (aucun texte requis pour accomplir la tâche)

**L'Effet de Supériorité de l'Image** (Paivio, 1971) :
- Images mémorisées **2,5× mieux** que les mots
- Traitement visuel plus rapide (150ms) vs verbal (500ms)
- **Court-circuite la barrière linguistique** (image d'éléphant = universelle)

**Avantage de la plateforme** : Interface disponible en 11 langues
- Français, anglais, espagnol, allemand, italien, portugais
- Néerlandais, suédois, danois, norvégien, finnois

**Les 7 générateurs fonctionnent 100% en mode visuel** (aucune consigne textuelle nécessaire pour l'apprenant)

## Générateur nº1 : Loto avec Images (App 012) ⭐ LA RECOMMANDATION PRIORITAIRE

**Pourquoi le Loto avec Images est L'activité parfaite pour grands débutants** :
- Zéro lecture requise (reconnaissance visuelle uniquement)
- Pratique orale intensive (l'enseignant énonce les mots dans la langue cible)
- Apprentissage social (joue avec locuteurs natifs ou autres apprenants)
- Accessibilité à la réussite (90%+ de taux de complétion)

---

### Fonctionnement pour Grands Débutants

**Mise en place** :
1. Générer des grilles de loto (3×3, 4×4 ou 5×5)
2. Thématique : Vocabulaire de base (animaux, aliments, couleurs, formes)
3. L'enseignant annonce dans la langue cible : "Pomme !"
4. Les apprenants : Cherchent l'image correspondante (reconnaissance visuelle, pas de lecture)

**Séquence d'acquisition lexicale** :

**Semaine 1** (Reconnaissance visuelle pure) :
- Enseignant dit : "Apple" (ou "Pomme" en FLE)
- Apprenant : Cherche image de pomme (ne connecte pas encore mot et image)
- Fréquence : Entend "apple/pomme" 10-15 fois pendant le jeu
- **Résultat** : Commence à associer le son /apl/ ou /pɔm/ à l'image

**Semaine 4** (Connexion son-image) :
- Enseignant dit : "Apple"
- Apprenant : Reconnaît immédiatement (voie neuronale formée)
- **C'est le vocabulaire réceptif** (comprend à l'écoute)

**Semaine 8** (Vocabulaire productif) :
- Apprenant peut dire : "I have apple!" (ou "J'ai pomme !")
- **C'est le vocabulaire expressif** (peut utiliser le mot à l'oral)

**Recherche** (Nation, 2001) : Les jeux de vocabulaire basés sur l'image accélèrent l'acquisition en L2 de **61% plus rapidement** que la mémorisation par cartes éclair

## Générateur nº2 : Associer Ombres et Objets (App 009)

**Pourquoi l'association ombre-objet fonctionne en didactique des langues** :
- Aucun texte requis (associer objet à sa silhouette)
- Enseigne le vocabulaire des formes (cercle, triangle, carré)
- Perception figure-fond (compétence visuelle, indépendante de la langue)

**Intégration linguistique** :

**Version silencieuse** (débutant) :
- Apprenant complète la fiche en autonomie
- Associe images aux ombres (tâche purement visuelle)

**Version verbale** (intermédiaire) :
- Enseignant : "Trouve l'ombre du chat"
- Apprenant : Identifie l'ombre, dit "C'est le chat" (pratique orale)

**Extension écrite** (débutant avancé) :
- Après l'association, l'apprenant étiquette chaque objet (écrit : "chat", "chien", "voiture")

**Durée de l'activité** : 15-25 minutes

**Tarification** : Formule Essentielle ou Accès Complet

## Générateur nº3 : Cherche et Trouve (J'espionne) (App 026)

**Pourquoi le Cherche et Trouve est excellent en didactique des langues** :
- L'enseignant fournit des consignes orales : "Trouve 5 pommes"
- Apprenant : Balayage visuel (pas de lecture), compréhension orale
- Répétition naturelle (l'enseignant répète la consigne 3-5 fois durant l'activité)

**Progression d'étayage linguistique** :

**Niveau 1** (Débutant) : Enseignant montre image + énonce le mot
```
Enseignant : [montre image de pomme] "Trouve les pommes. Cherche les pommes."
Apprenant : Balaie la fiche, entoure les pommes
Répétition : 5 expositions au mot "pomme"
```

**Niveau 2** (Débutant avancé) : Oral seul
```
Enseignant : "Trouve les pommes" (aucune image montrée)
Apprenant : Récupère image mentale de pomme, balaie la fiche
Compréhension : Doit comprendre instruction verbale
```

**Niveau 3** (Intermédiaire) : Enseignement par les pairs
```
Apprenant A : "Trouve les pommes" (pratique production orale)
Apprenant B : Balaie et entoure (compréhension orale)
Inversion des rôles : Les deux pratiquent compétences productives + réceptives
```

**Durée de l'activité** : 20-30 minutes

**Recherche** (Krashen, 1982) : L'input compréhensible (images + verbal) accélère l'acquisition linguistique **2,1× plus rapidement** que l'audio seul

**Tarification** : Formule Essentielle ou Accès Complet

## Générateur nº4 : Créateur d'Associations (App 005)

**Qu'est-ce que le Créateur d'Associations** : Associer images identiques (ou concepts liés)

**Applications en didactique des langues** :

**Association identique** (débutant) :
- Deux séries d'images identiques (pomme → pomme, chat → chat)
- Zéro langue requise (discrimination visuelle pure)

**Association catégorielle** (intermédiaire) :
- Associer objet à catégorie (pomme → image catégorie fruit)
- Requiert connaissances conceptuelles (médiatisé par la langue)

**Association mot-image** (avancé) :
- Une colonne : Mots écrits ("pomme", "banane", "orange")
- Autre colonne : Images
- Apprenant : Associe mot à image (compréhension écrite)

**Durée de l'activité** : 15-20 minutes

**Tarification** : Formule Essentielle ou Accès Complet

## Générateur nº5 : Comparaison Grand-Petit (App 019)

**Pourquoi la comparaison de taille fonctionne en didactique** :
- Concept universel (grand vs petit existe dans toutes les langues)
- Basé sur l'image (aucun français nécessaire pour comprendre la tâche)
- Enseigne les adjectifs comparatifs (plus grand, plus petit)

**Intégration linguistique** :

**Version non verbale** :
```
Fiche montre : [grand chien] vs [petit chien]
Apprenant : Entoure le grand (aucune langue nécessaire)
```

**Étayage verbal** :
```
Enseignant : "Lequel est grand ? Lequel est petit ?"
Apprenant : Pointe (compréhension réceptive)

Enseignant : "Ceci est le grand chien. Ceci est le petit chien." (modélisation)
Apprenant : Répète : "Grand chien, petit chien" (pratique productive)
```

**Extension comparative** (intermédiaire) :
```
Enseignant : "L'éléphant est plus grand que la souris."
Apprenant : Crée phrase : "Le chien est plus grand que le chat"
```

**Durée de l'activité** : 12-18 minutes

**Tarification** : Formule Essentielle ou Accès Complet

## Générateur nº6 : Train de Suites (App 030)

**Pourquoi les suites fonctionnent en didactique** :
- Aucune langue requise pour identifier suite (AB, ABB, ABC)
- Enseigne vocabulaire de séquençage (premier, ensuite, dernier)
- Découper/coller (motricité fine + intégration linguistique)

**Étayage linguistique** :

**Complétion silencieuse** (débutant) :
- Apprenant identifie suite visuellement
- Découpe et colle wagons dans l'ordre
- Aucune langue verbale nécessaire

**Narration verbale** (intermédiaire) :
```
Apprenant décrit : "Pomme, banane, pomme, banane" (identification de suite)
Enseignant demande : "Qu'est-ce qui vient ensuite ?"
Apprenant : "Pomme !" (prédiction + oral)
```

**Pratique des nombres ordinaux** (avancé) :
```
Enseignant : "Qu'y a-t-il dans le premier wagon ?"
Apprenant : "La pomme est dans le premier wagon." (vocabulaire ordinal)
```

**Durée de l'activité** : 15-25 minutes

**Tarification** : Formule Essentielle ou Accès Complet

## Générateur nº7 : Graphique de Comptage (App 013)

**Pourquoi les graphiques de comptage fonctionnent en didactique** :
- Reconnaissance numérique (symboles universels : 1, 2, 3)
- Comptage visuel (pas de barrière linguistique)
- Enseigne vocabulaire de quantité (un, deux, trois)

**Intégration linguistique** :

**Compter en L1** (débutant semaine 1) :
```
Apprenant : Compte pommes en espagnol : "Uno, dos, tres"
Écrit : 3 (dans graphique)
Introduction sans stress (utilise langue maternelle)
```

**Compter dans langue cible** (débutant semaine 4) :
```
Apprenant : Compte en français : "Un, deux, trois"
Écrit : 3
Pratique des mots-nombres en français
```

**Phrases modèles** (intermédiaire) :
```
Modèle : "Il y a ___ pommes."
Apprenant écrit : "Il y a 3 pommes." (phrase complète)
```
