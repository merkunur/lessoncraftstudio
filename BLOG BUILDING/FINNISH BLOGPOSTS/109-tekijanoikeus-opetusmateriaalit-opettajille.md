## Johdanto: Miksi Tekijänoikeudet Koskevat Opettajia?

**Tärkeä huomautus**: Tämä on yleistä tietoa, ei juridista neuvontaa. Käänny lakimiehen puoleen erityiskysymyksissä.

**Miksi tekijänoikeudet ovat tärkeitä opettajille?**:
```
Tyypillisiä tilanteita koulussa:
- Työkirjojen sivujen kopiointi oppilaille
- Kuvien lataaminen internetistä tehtäviin
- Ostetun materiaalin jakaminen kollegoille
- Materiaalien käyttö yksityisopetuksessa
- Työarkkien jakaminen verkossa

Riski: Tekijänoikeusloukkaus (taloudelliset seuraukset, työsuhteen vaarantuminen)
Tosiasia: Monet opettajat rikkovat tekijänoikeuksia tietämättään päivittäin
```

**Keskeinen periaate**: Se, että jotain on saatavilla verkossa, ei tarkoita, että sen käyttö olisi ilmaista tai laillista.

## Tekijänoikeuslain Perusteet

**Jokaisen opettajan tulisi tietää nämä**

### Mikä on Tekijänoikeuksien Suojaama?

**Automaattisesti suojattu sisältö**:
```
Nämä ovat tekijänoikeussuojattuja heti luomisesta:
- Kirjat ja oppikirjat
- Työkirjat ja tehtävämonisteet
- Kuvat ja kuvitukset
- Videot ja elokuvat
- Musiikki ja kappaleet
- Ohjelmistot ja sovellukset

Rekisteröintiä ei vaadita: Tekijänoikeus syntyy teoksen luomishetkellä

Symboli: © merkitsee tekijänoikeutta (mutta sen puuttuminen ei tarkoita suojaamatonta sisältöä)
```

---

### Vapaasti Käytettävä vs. Tekijänoikeussuojattu

**Vapaasti käytettävä sisältö**:
```
Tekijänoikeusvapaa materiaali:
- Vanhat teokset: Yleensä yli 70 vuotta tekijän kuoleman jälkeen
- Viranomaisteokset: Julkishallinnon tuottamat teokset
- Vapaaehtoisesti vapautetut: Tekijä on luopunut oikeuksista
- Creative Commons -lisensoitu: Tietyin ehdoin käytettävissä

Esimerkkejä:
- Klassikkokirjallisuus (esim. Kalevala)
- Viranomaisten tiedotemateriaalit
- CC-lisensoitu kuvamateriaali

Turvallista käyttää: Ei lupaa tarvita
```

**Tekijänoikeussuojattu sisältö**:
```
- Nykyaikaiset kirjat ja oppikirjat
- Kaupalliset työkirjat ja tehtäväkirjat
- Nykykuvat ja valokuvat
- Lisensioidut hahmot (Disney, Muumi jne.)

Vaatii: Luvan tai lisenssin käyttöön
```

## Opetuskäyttö ja Kohtuullinen Käyttö

**Rajoitettu poikkeus tekijänoikeuslakiin**

### Mitä Opetuskäyttö Tarkoittaa?

**Suomessa opetuskäytön rajat**:

**Sallittua opetuksessa**:
```
Tekijänoikeuslain mukainen opetuskäyttö:
- Pienet otteet teoksista opetuskäytössä
- Julkistetun teoksen esittäminen opetuksessa
- Lähde on aina mainittava
- Käyttö rajoittuu opetustilanteeseen
```

**Ei sallittua**:
```
Ylittää opetuskäytön rajat:
- Kokonaisten työkirjojen kopiointi
- Kaupallinen käyttö (yksityisopetus kaupallisena toimintana)
- Materiaalin jakaminen laajalle yleisölle
- Tekijän taloudellisten etujen loukkaaminen
```

## Tyypillisiä Opettajakohtaisia Tilanteita

**Onko se laillista?**

### Tilanne 1: Työkirjan Sivujen Kopiointi

**Kysymys**: "Voinko kopioida työkirjan sivuja koko luokalle?"

**Analyysi**:
```
Tilanne: Opettaja ostaa yhden työkirjan (20€), kopioi 25 oppilaalle

Tekijänoikeudenhaltijan näkökulma:
- Tarkoitettu myynti: 25 työkirjaa × 20€ = 500€
- Toteutunut myynti: 1 työkirja = 20€
- Menetetty tuotto: 480€

Opetuskäyttöanalyysi:
❌ Tarkoitus: Opetus (sallii opetuskäytön)
❌ Määrä: Huomattava osa teoksesta (ei sallittu)
❌ Taloudelliset vaikutukset: Selvästi haittaa myyntiä (ei sallittu)

Arvio: Yleensä EI sallittua opetuskäytön puitteissa
```

**Laillinen vaihtoehto**:
```
- Osta luokkakokoinen sarja (kallis mutta laillinen)
- Käytä tilauspalvelua (kuten tehtävägeneraattorit)
- Pyydä koulua/kuntaa hankkimaan materiaalit
```

---

### Tilanne 2: Kuvahaku Internetistä

**Kysymys**: "Voinko käyttää kuvia Google-hausta tehtävissäni?"

**Analyysi**:
```
Yleinen väärinkäsitys: "Jos se on Googlessa, se on ilmaista"
Todellisuus: Google-haku löytää tekijänoikeussuojattuja kuvia

Useimmat kuvat: Suojattuja tekijänoikeudella (kuvaaja/taiteilija omistaa)

Turvalliset vaihtoehdot:
✓ Tekijänoikeusvapaat: Pixabay, Unsplash (tarkista lisenssi)
✓ Creative Commons: Jotkut sallivat opetuskäytön (lue lisenssi)
✓ Omat kuvat: Omistat tekijänoikeuden
✓ Generaattorin kuvat: Tilauksen mukana tulevat käyttöoikeudet

Riskialttiit:
✗ Satunnaiset Google-kuvat: Tuntematon omistaja, todennäköisesti suojattu
✗ Disney-hahmot: Tiukasti suojattu (valvotaan aktiivisesti)
✗ Julkkiskuvat: Kuvaajan omistama, ei vapaasti käytettävissä
```

---

### Tilanne 3: Materiaalin Jakaminen Kollegoille

**Kysymys**: "Voinko jakaa ostamani opetusmateriaalin työkaverilleni?"

**Analyysi**:
```
Useimpien materiaalien lisenssit: "Yksittäisen opettajan käyttöön"

Mitä VOIT tehdä:
✓ Käyttää omassa opetuksessasi
✓ Muokata omille oppilaillesi
✓ Näyttää omalle luokallesi

Mitä ET VOI tehdä:
✗ Jakaa kollegalle (vaikka samasta koulusta)
✗ Laittaa koulun yhteiselle verkkoasemalle
✗ Lähettää sähköpostilla toiselle opettajalle

Miksi: Myyjä menettää potentiaalisen myynnin (taloudellinen haitta)

Jakamiseen: Osta lisälisenssejä (monilla palveluilla moniopettajahinnoittelu)
```

---

### Tilanne 4: YouTube-videot Opetuksessa

**Kysymys**: "Voinko näyttää YouTube-videoita luokalleni?"

**Analyysi**:
```
Yleisesti sallittua:
✓ Videon näyttäminen luokalle (opetuskäyttö)
✓ Käyttö opetustilanteessa (kasvokkain tapahtuva opetus)

Ehdot:
- Laillisesti saatavilla oleva video (ei piratoitu)
- Näytetään luokassa (ei julkinen näytös kouluajan ulkopuolella)
- Koko luokka voi katsoa (ei julkinen esitys)

Ei sallittua:
✗ Lataaminen ja kopioiden säilyttäminen (rikkoisi YouTuben käyttöehtoja)
✗ Mainosten poistaminen (tekijänoikeudenhaltijan edellyttämät)
✗ Julkinen näytös (esim. vanhempainilta ≠ opetustilanne)
```

## Oman Materiaalin Luominen

**Omistajuus ja oikeudet**

### Opettajan Luomat Tehtävät

**Jos luot alkuperäistä sisältöä**:
```
Omistat tekijänoikeuden:
- Itse suunnittelemasi tehtävät
- Omat valokuvat
- Omat oppituntisuunnitelmat

Oikeuksisi:
✓ Käyttö omassa opetuksessa
✓ Jakaminen kollegoille (jos haluat)
✓ Myynti (jos haluat)

Poikkeukset - ET omista jos:
- Luotu "työteoksena" (jotkut kunnat väittävät omistajuutta)
- Käytetty tekijänoikeussuojattuja elementtejä (Disney-hahmoja tehtävässäsi)
- Mukautettu tekijänoikeussuojatusta lähteestä
```

---

### Kaupalliset Käyttöoikeudet

**Sivutoimet ja yksityisopetus**:
```
Luokkahuonemateriaalit: Usein OK omaan opetukseen
Kaupallinen käyttö: Vaatii eri luvan

Esimerkki:
- Koulun tehtävä: Voi käyttää omassa opetuksessa
- Yksityisopetus: Tarvitsee kaupallisen lisenssin

Miksi: Eri tarkoitus (ei-kaupallinen vs. kaupallinen)

Tehtävägeneraattorit:
- Ilmainen/Core: Vain luokkahuonekäyttöön
- Full-paketti: Kaupallinen lisenssi mukana (240€/vuosi)
```

## Tehtävägeneraattorien Lisenssit

**Ymmärrä käyttöoikeutesi**

### Ilmainen Paketti (0€/vuosi)

**Mitä voit tehdä**:
```
✓ Luoda rajattomasti tehtäviä
✓ Käyttää omassa opetuksessa
✓ Tulostaa oppilaillesi
✓ Muokata generoituja tehtäviä

Rajoitukset:
✗ Vesileima pysyy tehtävissä
✗ Ei voi käyttää kaupalliseen yksityisopetukseen
✗ Ei voi myydä tehtäviä
```

---

### Core-paketti (144€/vuosi)

**Mitä voit tehdä**:
```
✓ Kaikki ilmaisen paketin ominaisuudet
✓ Ei vesileimaa (siisti, ammattimainen ulkoasu)
✓ Käyttö omassa opetuksessa
✓ Jakaminen kollegoille koulussasi

Rajoitukset:
✗ Ei voi myydä tehtäviä
✗ Ei voi käyttää kaupalliseen yksityisopetukseen
✗ Vain henkilökohtainen opetuskäyttö
```

---

### Full-paketti (240€/vuosi)

**Kaupallinen lisenssi mukana**:
```
✓ Kaikki Core-paketin ominaisuudet
✓ Kaupallinen käyttö sallittu:
   - Yksityisopetustoiminta
   - Kotikoulu-osuuskunnat
   - Konsultointitoiminta
   - Tehtäväpakettien myynti (lähteet merkittynä)

✓ White-label -vaihtoehdot
✓ API-käyttö integraatioihin

Sopii parhaiten: Opettajille, joilla on sivutoimintaa
```

## Lähteiden Merkintä

**Tunnustuksen antaminen**

### Milloin Lähteiden Merkintä Vaaditaan?

**Tilanteet**:
```
Muiden töiden käyttö:
- On merkittävä: Creative Commons -sisältö (jos lisenssi vaatii)
- Suositeltavaa merkitä: Tekijänoikeusvapaa sisältö (ei vaadittu mutta eettistä)
- Ei voi käyttää ilman lupaa: Täysin tekijänoikeussuojattu

Oma työ:
- Vapaaehtoista: "Luonut [Nimesi]"
- Jos myyt: Vesileima tai alatunniste suositeltava
```

---

### Oikea Lähteen Merkintätapa

**Esimerkki**:
```
Kuva: "Auringonlasku vuorilla" - Matti Meikäläinen
Lisenssi: Creative Commons BY 2.0
Lähde: flickr.com/photos/mattimeikalainen/12345

Tai:

Kuva: Matti Meikäläinen (CC BY 2.0)
```

## Koulun ja Kunnan Säännöt

**Tunne työnantajasi säännöt**

### Työteokset

**Jotkut kunnat väittävät omistajuutta**:
```
Käytäntö: "Työajalla/työnantajan resursseja käyttäen luotu materiaali kuuluu kunnalle"

Vaikutukset:
- Työajalla luodut tehtävät: Kunta voi väittää omistajuutta
- Kotona luodut oppituntisuunnitelmat: Mahdollisesti omasi (tarkista käytäntö)
- Koulun kopiokoneella tehdyt materiaalit: Kunta voi omistaa

Todellisuus: Vaihtelee kunnittain (lue työsopimuksesi)

Turvallinen lähestymistapa: Oleta että koululle luotu materiaali kuuluu koululle
```

Tyypillisesti kielletään:

Rikkomisen seuraukset:

## Oman Työn Suojaaminen

**Jos luot ja myyt materiaalia**

### Suojaa Tekijänoikeutesi

**Suojausvaiheet**:
```
1. Tekijänoikeus syntyy automaattisesti (rekisteröintiä ei tarvita)
2. Lisää tekijänoikeusmerkintä: "© 2025 [Nimesi]. Kaikki oikeudet pidätetään."
3. Harkitse rekisteröintiä: Tekijänoikeusrekisteri (vahvistusta varten)
4. Vesileima: Visuaalinen merkki että se on sinun
```

---

### Käyttöehdot

**Kun myyt materiaaleja**:
```
Selkeät lisenssiehdot:
- "Yhden opettajan käyttöön" TAI "Moniopettajalisenssi"
- "Vain henkilökohtaiseen käyttöön" TAI "Kaupallinen käyttö sallittu"
- "Ei saa jakaa, myydä uudelleen tai levittää"

Valvonta: DMCA-poistopyyntö jos piratoitu
```

**Korkean riskin toiminnot**

### Älä Tee Näin:

❌ Jaa yhtä ostosta koko koululle
❌ Myy materiaaleja tekijänoikeussuojatuilla kuvilla
❌ Nauhoita ja jaa tekijänoikeussuojattuja videoita

## Yhteenveto

Tekijänoikeudet suojaavat tekijöiden oikeuksia - opettajien on kunnioitettava henkistä omaisuutta samalla kun he hankkivat laadukkaita materiaaleja.

**Automaattinen suoja**: Kirjat, tehtävämonisteet, kuvat, videot suojattuja luomishetkestä lähtien (ei vaadi rekisteröintiä)

**Opetuskäyttö**: Rajoitettu poikkeus, ei automaattinen lupa kaikelle (lähde merkittävä, käyttö kohtuullista)

**Tyypilliset tilanteet**: Työkirjojen kopiointi yleensä ei sallittua (480€ menetetty tuotto), Google-kuvat useimmiten suojattuja (käytä Pixabay/Unsplash), materiaalien jakaminen kollegoille vaatii lisälisenssejä

**YouTube luokassa**: Yleensä sallittua (kasvokkain tapahtuva opetus), älä lataa tai poista mainoksia

**Opettajan luomat**: Omistat alkuperäisen sisällön (ellei työteoskäytäntö muuta), tarvitset kaupallisen lisenssin yksityisopetukseen

**Generaattorilisenssit**: Ilmainen (luokka, vesileima), Core 144€ (luokka, ei vesileimaa), Full 240€ (kaupallinen käyttö sallittu)

**Lähteiden merkintä**: Merkitse Creative Commons -sisältö (jos vaadittu), eettistä merkitä tekijänoikeusvapaa

**Kunnan säännöt**: Tarkista työteoskäytännöt (jotkut kunnat väittävät omistajuutta työajalla luotuihin materiaaleihin)

**Vältettävät riskit**: Älä kopioi työkirjoja, käytä piraattisivustoja, jaa yhtä lisenssiä, käytä Disney-hahmoja ilman lupaa

**Turvalliset vaihtoehdot**: Generaattorit (lisensoitu), tekijänoikeusvapaat kuvat, alkuperäinen sisältö, asianmukaiset lisenssit
