## Core App Features (from REFERENCE APPS/subtraction.html)

### 1. Exercise Modes (4 modes)
- **Cross Out (traditional)** - Children cross out images to solve subtraction
- **Image - Number** - Shows images and numbers together
- **Find Subtrahend** (images - ? = result) - Find the missing number being subtracted
- **Mixed** - Randomly alternates between Image-Number and Find Subtrahend

### 2. Exercise Configuration
- **Number of Exercises:** 1-10 problems per worksheet
- **Max Minuend:** 2-20 (the number being subtracted FROM)
- **Include Name/Date fields** - Optional header for student info
- **Include Exercise Numbers** - Numbers each problem (1, 2, 3...)
- **Child-friendly box** - Uses friendly visual boxes for expressions

### 3. 11 Languages Support
English, German, French, Spanish, Portuguese, Italian, Dutch, Swedish, Danish, Norwegian, Finnish

### 4. Page Setup Options
- **Page Sizes:**
  - Letter Portrait (8.5×11")
  - A4 Portrait (210×297mm)
  - A4 Landscape (297×210mm)
  - Letter Landscape (11×8.5")
  - Square (1200x1200)
  - Custom dimensions
- **Page Color** - Customizable background color

### 5. Background Themes
- Theme selection from library
- Opacity slider (0-100%)

### 6. Border Themes
- Theme selection from library
- Opacity slider (0-100%)

### 7. Text Tools
- Add custom text anywhere
- **Text Properties:**
  - Color picker
  - Font size (8+)
  - 7 Font options: Lexend Deca, Baloo 2, Nunito, Quicksand, Fredoka, Arial, Verdana
  - Outline color
  - Outline width (0-10)

### 8. Image Library
- 3000+ child-friendly images
- Theme-based organization
- Search functionality
- Select images for problems

### 9. Upload Custom Images
- Multi-file upload
- Accepts all common image formats (JPEG, PNG, GIF)
- Session-based storage
- Combine with library images

### 10. Canvas Editing Tools
- **Layers:** Bring to Front, Bring Forward, Send Backward, Send to Back
- **Alignment:** Left, Center, Right, Top, Middle, Bottom
- **Align to Page:** Center horizontally/vertically on page
- **Lock/Unlock** objects
- **Delete** selected objects
- **Zoom:** In, Out, Reset, percentage display
- **Undo/Redo** functionality

### 11. Download Options
- **Worksheet JPEG** - High-quality image
- **Answer Key JPEG** - Solution sheet
- **Worksheet PDF** - Print-ready document
- **Answer Key PDF** - Solution document
- **Grayscale toggle** - Ink-saving option
- **300 DPI professional quality**

### 12. Answer Key Generation
- Automatic answer key creation
- Separate tab for answer key view
- Downloadable in both JPEG and PDF formats
