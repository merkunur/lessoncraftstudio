## Johdanto: Ensimmäisen Vuoden Haasteet

**Todellisuustarkistus**: Ensimmäinen vuosi on vaativa.

**Yleisimmät aloittelevan opettajan haasteet**:
```
Ajanhallinta:
- Suunnittelu vie 3-4 tuntia päivässä (illat ja viikonloput)
- Ei koskaan tunnu olevan "valmiina"
- Uupumuksen riski on suuri

Luokkahuoneen hallinta:
- Oppilaat testaavat rajoja
- Käytännöt epäselviä (koska ei vielä vakiintuneita)
- Häiriötilanteet yleisiä

Materiaalien luominen:
- Tehtävämonisteiden tekeminen tyhjästä (tunteja per moniste)
- Laatu vaihtelevaa
- Tekijänoikeuskysymykset (muiden materiaalien käyttö)

Tulos: 50% uusista opettajista lopettaa viiden vuoden sisällä (Ingersoll, 2012)
```

**Ratkaisu**: Järjestelmät + valmiit materiaalit + realistiset odotukset
```
Toimivilla järjestelmillä:
- Suunnitteluaika lyhenee 70% (valmiit materiaalit)
- Luokkahuoneen hallinta sujuvaa (selkeät käytännöt)
- Laatu tasaista (ammattimaiset materiaalit)

Tulos: Ensimmäinen vuosi haastava mutta hallittavissa
```

**Keskeinen periaate**: Älä keksi pyörää uudelleen (käytä toimiksi todettuja järjestelmiä).

## Elokuu: Ennen Oppilaiden Saapumista

**Kouluvuoden ennakkovalmistelu**

### Viikot 1-2: Luokkahuoneen Fyysinen Järjestely

**Välttämättömät pisteet**:
```
Tarvittavat alueet:

1. Opettajan työpöytä
   - Tarvikkeet: Nitoja, teippi, sakset, kyniä
   - Arviointikirja/suunnittelukalenteri
   - Oheismateriaalit

2. Oppilaiden pulpetit/istuinjärjestys
   - Järjestely: Rivit, ryhmät tai U-muoto (aloita riveillä helpomman hallinnan vuoksi)
   - Nimiliput valmiina (ensimmäinen viikko)
   - Tarvikekassit (jos jaetut tarvikkeet)

3. Palautuspiste
   - Korit merkitty aineen mukaan (Matematiikka, Äidinkieli, Ympäristöoppi jne.)
   - Oppilaat tietävät minne palauttaa työt (ei tarvitse kysyä)

4. Materiaalivarasto
   - Ylimääräisten monisteiden laatikko (poissaolleille oppilaille)
   - Tarvikepiste (lyijykynät, pyyhekumit, värikynät)
   - Mallivastaukset (jos käytössä itsetarkistusjärjestelmä)

5. Luokkakirjasto
   - Kirjat järjestetty tason tai lajin mukaan
   - Lainausjärjestelmä (yksinkertainen lainakirja)

Aikainvestointi: 2 täyttä päivää (mutta luo pohjan koko vuodelle)
```

---

### Viikko 3: Ensimmäisen Kuukauden Suunnittelu

**Syyskuun kartoitus**:
```
Keskeiset suunnitteluvaiheet:

1. Käsiteltävät sisällöt
   - Tutki kunnan/koulun opetussuunnitelma
   - Tunnista syyskuun pakolliset sisällöt
   - Jaa viikkokokonaisuuksiin

2. Arviointiaikataulu
   - Viikko 1: Alkutason kartoitukset (mistä oppilaat lähtevät?)
   - Viikko 4: Ensimmäisen kokonaisuuden arviointi
   - Jatkuva: Viikoittaiset pienet kokeet

3. Generoi koko kuukauden tehtävät
   - Matematiikka: 4 viikkoa × 5 päivää = 20 monistetta
   - Lukeminen: 20 luetun ymmärtämisen tehtävää
   - Sanasto: 4 sanahakutehtävää (yksi per viikko)
   - Kokonaisgenerointiaika: 20 min per aine = 2 tuntia yhteensä

4. Sijaisopettajan hätäsuunnitelmat
   - Luo 3 täyden päivän sijaissuunnitelmat (sairauden varalta)
   - Säilytä selkeästi merkityssä kansiossa
   - Kerro koulun toimistolle sijaismateriaalien sijainti

Tulos: Syyskuu suunniteltu (vähentää ensimmäisen kuukauden stressiä merkittävästi)
```

---

### Viikko 4: Käytännöt ja Rutiinit

**Kirjoita ylös jokainen käytäntö**:
```
Luo käytäntökäsikirja (itsellesi ja sijaiselle):

Aamun rutiini:
1. Oppilaat saapuvat (8:20)
2. Ripustaa reput
3. Palauttaa kotitehtävät (jos on)
4. Ottaa aamutoiminnan tehtävän
5. Työskentelee 8:30 asti
6. Aamun kuulutukset

Siirtymäkäytännöt:
- Huomion herätys (soittokello)
- Siivoaa nykyisen toiminnan (30 sek)
- Ottaa seuraavan toiminnan välineet (30 sek)
- Aloittaa uuden toiminnan (heti)

Vessakäynnit/vesihanakäynnit:
- Nosta käsi kysyäksesi
- Vain 1 oppilas kerrallaan (vessakortti näkyvillä)
- Palaa 5 minuutin sisällä

Kotiinlähtö:
- 5 minuuttia ennen kellon soimista: Siivoa työpiste
- 3 minuuttia: Täytä poistumislippu
- 1 minuutti: Jonoon ovelle
- Kello soi: Kotiinlähtö

Dokumentoi jokainen rutiini (älä luota muistiin, kirjoita ylös)
```

## Syyskuu: Ensimmäisen Kuukauden Selviytyminen

**Viikko kerrallaan -prioriteetit**

### Viikko 1: Rutiinien Vakiinnuttaminen

**Älä keskity akateemisiin asioihin** (keskity käyttäytymisodotuksiin):
```
Maanantai:
- Tervehdi oppilaita lämpimästi (ensivaikutelma ratkaisee)
- Luokkahuoneen kiertokävely (aarretsetsaustehtävä: "Etsi teroitin")
- Opeta aamun rutiini
- Harjoittele aamun rutiinia 3× (mallinnus, ohjaus, itsenäinen)

Tiistai:
- Kertaa aamun rutiini (pitäisi sujua paremmin)
- Opeta siirtymä #1 (pulpetti → matto)
- Harjoittele 5× (käy tylsäksi, mutta luo tavan)
- Täytä "Minusta" -tehtävä (oppilaiden tutustuminen)

Keskiviikko:
- Opeta käytäntö #2 (töiden palauttaminen)
- Harjoittele kotiinlähtörutiini
- Luokan säännöt juliste (yhteistyössä oppilaiden kanssa)

Torstai:
- Opeta käytäväkäytäntö
- Harjoittele kävely taideryhmiin (musiikki, kuvataide, liikunta)
- Jaa luokkatehtävät (tehtävä: Työhakemus)

Perjantai:
- Kertaa kaikki tällä viikolla opitut käytännöt
- Juhli: "Opimme miten luokkamme toimii!"
- Vähäinen akateeminen opetus (viikko keskittyi hallintaan)

Tutkimus: Ensimmäiset 2 viikkoa määrittävät koko vuoden (Wong & Wong, 2009)
Investointi: Käytäntöihin käytetty aika nyt = sujuva purjehdus loppuvuoden
```

---

### Viikko 2: Akateemiset Rutiinit Alkavat

**Lisää akateemista työtä vähitellen**:
```
Maanantai-tiistai: Alkutason kartoitukset
- Lukeminen: Lukusujuvuustarkistus (1 minuutin ääneen luku)
- Matematiikka: Luokkatason tehtävä (20 laskua)
- Kirjoittaminen: Kirjoitusnäyte ("Minun kesälomani")

Tarkoitus: Tiedä mistä oppilaat lähtevät (ohjaa opetusta)
Huomaa: Ei arvostella! Vain diagnostista.

Keskiviikko-perjantai: Aloita opetus
- Lyhyet oppitunnit (15-20 min suoraa opetusta)
- Paljon harjoittelua (tehtävät luokassa opettajan tuella)
- Tarkista ymmärrys usein

Vältä: Uuden sisällön opettaminen 45 minuuttia putkeen (oppilaat eivät jaksa keskittyä)
Paras: 15 min opetus → 20 min harjoittelu → 10 min kertaus
```

---

### Viikot 3-4: Itsenäisyyden Kehittäminen

**Oppilaat alkavat työskennellä itsenäisemmin**:
```
Tavoite: Oppilaat tekevät tehtäviä ilman jatkuvaa opettajan tukea

Viikko 3:
- Opeta "Kysy 3:lta ennen minua" (lue ohjeet uudestaan → tarkista materiaalit → kysy kavereilta → sitten opettaja)
- Esittele itsetarkistus (mallivastaukset -piste)
- Harjoittele nopeasti valmistuvien rutiini (valintataulukko)

Viikko 4:
- Pienryhmäopetus alkaa (opettaja työskentelee 5 oppilaan kanssa)
- Muu luokka: Itsenäinen työskentely (tehtävät, lukeminen)
- Oppilaat hallitsevat omaa oppimistaan (vähän häiriöitä)

Virstanpylväs: Syyskuun loppuun mennessä oppilaat voivat työskennellä itsenäisesti 20-30 minuuttia
```

## Välttämättömät Tehtäväjärjestelmät Aloitteleville Opettajille

**Heti käyttöönotettavat perussisäljestelmät**

### Aamutoiminta (Lämmittelytehtävät)

**Miksi kriittinen aloitteleville opettajille**:
```
Ratkaistava ongelma: Oppilaat saapuvat eri aikoina (8:15-8:30)
Ilman aamutoimintaa: Aikaisin tulijat juttelevat, häiriökäyttäytyminen
Aamutoiminnalla: Kaikki tekemisessä heti saapuessaan

Järjestelmä:
1. Generoi 20 aamutoiminnan tehtävää (yksi per koulupäivä kuukaudessa)
2. Tulosta 30 kopiota per tehtävä (600 sivua yhteensä)
3. Aseta koriin oven viereen
4. Oppilaat ottavat automaattisesti saapuessaan

Sisältö: Kertaava materiaali (5 nopeaa tehtävää, vie 10 minuuttia)

Aikasäästö: Ei enää aikaisin saapuvien oppilaiden hallintaa (he ovat heti kiireisiä)
```

---

### Viikoittainen Taitoharjoittelu

**Ennakoitava rutiini**:
```
Maanantai: Matematiikan laskuharjoitus (20 laskua)
Tiistai: Sanahaku (käsiteltävä aihe)
Keskiviikko: Matematiikan sanallisia tehtäviä (soveltaminen)
Torstai: Luetun ymmärtäminen (yksi sivu)
Perjantai: Kertauspeli tai rikastuspulmatehtävä

Hyödyt aloittelevalle opettajalle:
- Ennakoitavissa (vähemmän suunnittelua)
- Oppilaat tuntevat rutiinin (vähemmän selittämistä)
- Johdonmukaista harjoittelua (taitojen kehittyminen)

Generointi: 5 tehtävää × 42 sek = 3,5 minuuttia per viikko
Vuosittainen aika: 3,5 min × 36 viikkoa = 126 minuuttia (2,1 tuntia koko vuodelle)

Vertaa: Luominen tyhjästä (40 min per tehtävä × 180 = 120 tuntia!)
```

---

### Arviointijärjestelmä

**Selvitä oppiivatko oppilaat**:
```
Viikoittain: Poistumisliput (3-5 kysymystä, viimeiset 5 minuuttia päivästä)
- "Tänään opimme ____. Näytä että ymmärsit."
- Nopea tarkistus: Ovatko oppilaat oppineet?

Kahden viikon välein: Lyhyt koe (10 kysymystä, 15 minuuttia)
- Kattaa 2 viikon opetuksen
- Tunnistaa kuka tarvitsee lisäopetusta

Kuukausittain: Aihealueen arviointi (20-30 kysymystä, 30-45 minuuttia)
- Kattava tarkistus
- Menee arviointikirjaan

Hyöty aloittelevalle opettajalle: Data näyttää mikä toimii (säädä opetusta sen mukaan)
```

## Aikaa Säästävät Strategiat

**Työskentele fiksummin, ei pidempään**

### Viikoittaiset Suunnitteluistunnot

**Sunnuntai-iltapäivän rutiini** (2 tuntia):
```
30 minuuttia: Katsele tulevan viikon sisällöt
30 minuuttia: Generoi kaikki viikon tehtävät (matematiikka, lukeminen, ympäristöoppi)
30 minuuttia: Luo mallivastaukset, järjestä materiaalit
30 minuuttia: Suunnittele oppitunnit (käytä tehtäviä harjoitteluna lyhyen opetuksen jälkeen)

Tulos: Viikko 100% suunniteltuna 2 tunnissa
Hyöty: Arki-illat vapaat (ei keskiyön suunnittelua)
```

---

### Kuukausittainen Massagenerointi

**Vieläkin tehokkaampi**:
```
Yksi lauantai per kuukausi: 4 tuntia
Generoi: Koko kuukauden tehtävät (kaikki aineet)

Syyskuun lauantai:
- Matematiikka: 20 monistetta
- Lukeminen: 20 monistetta
- Sanasto: 4 sanahakua
- Ympäristöoppi: 8 monistetta
Yhteensä: 60 tehtävää

Aika: 4 tuntia (vs 40+ tuntia luominen tyhjästä)

Hyöty: Kuukausi valmis (vähemmän viikoittaista stressiä)
```

## Yleiset Ensimmäisen Vuoden Virheet Vältettäviksi

**Opi muiden virheistä**

### Virhe 1: Liiallinen Oppituntien Suunnittelu

**Ongelma**:
```
Uusi opettaja: Suunnittelee monimutkaisen 45 minuutin oppitunnin
Todellisuus: Tunti kestää 15 minuuttia TAI 90 minuuttia (harvoin kuten suunniteltu)
Tulos: Hukattu aika tai ei tarpeeksi tekemistä
```

**Ratkaisu**:
```
Suunnittele lyhyemmin: 15-20 minuutin minitunnit
Varalla: Ylimääräiset tehtävät valmiina jos aikaa jää
Joustavuus: Säädä miten menee
```

---

### Virhe 2: Ei Käytäntöjä Kaikelle

**Ongelma**:
```
Opettaja: "Ottakaa esiin matematiikan kirja"
Kaaos: 30 oppilasta kaivaa pulpeteissaan (5 minuuttia hukkaan)
Päivittäinen vaikutus: 10 siirtymää × 5 min = 50 minuuttia hukkaan
```

**Ratkaisu**:
```
Käytäntö: "Teillä on 30 sekuntia ottaa matematiikan kirja. Ajastin alkaa nyt."
Tulos: Nopea, järjestyksessä oleva siirtymä
```

---

### Virhe 3: Yrittää Luoda Kaiken Itse

**Ongelma**:
```
Uusi opettaja: "Teen kaikki monisteet itse" (ylpeys/perfektionismi)
Todellisuus: 1 tunti per moniste × 180 päivää = 180 tuntia
Tulos: Uupumus, väsymys, lopettaminen
```

**Ratkaisu**:
```
Käytä generaattoreita: 42 sekuntia per tehtävä
Säästetty aika: 179 tuntia vuodessa (käytä muihin prioriteetteihin kuten arviointiin, vanhempien yhteydenpitoon)
```

---

### Virhe 4: Parhaat Kaverit Oppilaiden Kanssa

**Ongelma**:
```
Uusi opettaja: Haluaa oppilaiden pitävän hänestä (luonnollinen halu)
Virhe: Liian ystävällinen, epäselvät rajat
Tulos: Oppilaat eivät kunnioita auktoriteettia, luokkahuoneen hallinta-ongelmia
```

**Ratkaisu**:
```
Ensimmäinen lukukausi: Tiukka, selkeät odotukset, johdonmukaiset seuraukset
Toinen lukukausi: Voi rentoutua kun kunnioitus luotu
Sanonta: "Älä hymyile ennen joulua" (vanha neuvo, mutta on perää)

Tasapaino: Ystävällinen mutta luja (ei ilkeä, mutta ei myöskään tallattava)
```

Hyöty: Opi joltakulta joka on jo selvittänyt sen
