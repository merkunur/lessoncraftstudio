## Introduktion: Traditionell planering kontra bakåtdesign

**Traditionell planering** (framåtdesign):
```
Steg 1: Välj aktiviteter (roliga arbetsblad, praktiska projekt)
Steg 2: Genomför lektionerna
Steg 3: Skapa bedömning i slutet
Problem: Aktiviteterna kanske inte leder till målen
Resultat: Eleverna gör aktiviteter men når inte kunskapskraven
```

**Bakåtdesign** (Wiggins & McTighe, 2005):
```
Steg 1: Identifiera önskade resultat (vad ska eleverna KUNNA?)
Steg 2: Avgör acceptabelt bevis (hur ska de VISA sitt lärande?)
Steg 3: Planera läraktiviteter (vilka aktiviteter leder DIT?)
Resultat: Varje aktivitet bygger systematiskt mot måluppfyllelse
```

**Forskning** (Wiggins & McTighe, 2005 - Understanding by Design): Bakåtdesign ökar elevernas måluppfyllelse med **15-25%** genom att säkerställa samstämmighet mellan läroplansmål, undervisning och bedömning

**Nyckelinsikt**: Börja med SLUTET i åtanke (bedömningen), jobba bakåt mot undervisningen

## De 3 stadierna i bakåtdesign

### Stadium 1: Identifiera önskade resultat

**Fråga**: Vad ska eleverna veta och kunna göra?

**Exempelområde**: Bråk i årskurs 4

**Läroplansmål** (LGR22):
```
Ma 4.1: Förstå och förklara varför olika bråk kan representera samma del
Ma 4.2: Jämföra bråk med olika täljare och nämnare
Ma 4.3: Förstå addition och subtraktion av bråk med lika nämnare
```

**Beständiga förståelser** (stora idéer som eleverna behåller):
```
1. Bråk representerar delar av en helhet
2. Olika bråk kan representera samma mängd (likvärdighet)
3. Bråk kan jämföras genom att hitta gemensam nämnare
```

**Viktiga frågor**:
```
1. Hur kan två olika bråk vara lika stora?
2. När använder vi bråk i vardagen?
3. Hur vet vi vilket bråk som är större?
```

---

### Stadium 2: Avgör acceptabelt bevis (bedömning)

**Fråga**: Hur ska eleverna visa att de behärskar målen?

**Slutbedömning** (designad INNAN undervisningen):

**Del 1: Färdighetsdemonstration** (20 poäng)
```
1. Rita en modell som visar att 2/4 = 1/2 (visuellt bevis)
2. Jämför 3/5 och 2/3 (visa arbete med gemensam nämnare)
3. Beräkna 2/8 + 3/8 (lös med modell)
4. Ange 3 vardagssituationer där vi använder bråk (tillämpning)
```

**Del 2: Problemlösning** (10 poäng)
```
Problemtext: "Du åt 2/6 av en pizza. Din kompis åt 3/9 av samma pizza. Vem åt mest? Bevisa ditt svar."
Bedömningskriterier:
- 4 poäng: Korrekt svar med visuellt bevis
- 3 poäng: Korrekt svar, inget bevis
- 2 poäng: Felaktigt svar, rimligt försök
- 0-1 poäng: Inget arbete visat
```

**Behörighetskriterier**: 80% korrekt = måluppfyllelse (24/30 poäng)

**Nu**: Designa undervisningen för att säkerställa att eleverna KAN göra dessa uppgifter

---

### Stadium 3: Planera läraktiviteter (undervisning)

**Fråga**: Vilka aktiviteter förbereder eleverna för bedömningen?

**Vecka 1: Introduktion till bråk**
```
Måndag: Pizzamodeller för bråk (konkret)
Tisdag: Arbetsblad - skugga bråk (bildmässigt)
Onsdag: Ordletare med bråkvokabulär (täljare, nämnare, likvärdig)
Torsdag: Färglägg bråkdelar (1/2 röd, 1/4 blå)
Fredag: Formativ bedömning (5 uppgifter - kontrollera förståelse)

Mål: Eleverna kan identifiera och rita bråk
```

**Vecka 2: Likvärdiga bråk**
```
Måndag: Pappervikning (1/2 = 2/4 = 4/8 fysiskt)
Tisdag: Arbetsblad - likvärdiga bråk (15 uppgifter)
Onsdag: Korsord med bråk (vokabulär: likvärdig, förkorta)
Torsdag: Matematikpussel (hitta likvärdiga bråk i system)
Fredag: Formativ bedömning (5 uppgifter - enbart likvärdighet)

Mål: Eleverna förstår varför 2/4 = 1/2
```

**Vecka 3: Jämföra bråk**
```
Måndag: Bråkremsor (visuell jämförelse)
Tisdag: Arbetsblad - jämför bråk med gemensam nämnare (20 uppgifter)
Onsdag: Arbetsblad - jämför bråk (hitta gemensam nämnare)
Torsdag: Problemtexter (vilket är större?)
Fredag: Formativ bedömning (5 uppgifter - jämförelse)

Mål: Eleverna kan jämföra alla typer av bråk
```

**Vecka 4: Addition/subtraktion**
```
Måndag: Addera bråk med modeller
Tisdag: Arbetsblad - addition med lika nämnare (20 uppgifter)
Onsdag: Arbetsblad - subtraktion med lika nämnare (20 uppgifter)
Torsdag: Blandade räknesätt
Fredag: SLUTPROV (bedömningen som designades i Stadium 2)

Mål: Eleverna kan addera/subtrahera bråk, redo för prov
```

**Samstämmighetskontroll**: Varje aktivitet bygger färdigheter som krävs för slutbedömningen

### Månatlig läroöversikt (Årskurs 4 Matematik exempel)

**September: Talsystem & positionssystem**
```
Läroplansmål: Flersiffrig addition/subtraktion, avrundning
Nyckelarbetsblad:
- Vecka 1: 3-siffrig addition (20 uppgifter dagligen)
- Vecka 2: 3-siffrig subtraktion (20 uppgifter dagligen)
- Vecka 3: Avrundning till närmaste 10-tal, 100-tal (ordletare med tal)
- Vecka 4: Problemtexter (blandade räknesätt)
Bedömning: 25-uppgifters prov (80% målkrav)
```

**Oktober: Multiplikationstabeller & strategier**
```
Läroplansmål: Multiplikation inom 100, areamodeller
Nyckelarbetsblad:
- Vecka 1: Tabeller 0-5 (Bildbingo för övning)
- Vecka 2: Tabeller 6-10 (daglig övning)
- Vecka 3: Areamodeller (rutnätsmultiplikation)
- Vecka 4: Problemtexter (multiplikationstillämpningar)
Bedömning: Tidsbegränsat prov (100 uppgifter på 5 min, 80% målkrav)
```

**November: Division**
```
Läroplansmål: Division inom 100, rest
Nyckelarbetsblad:
- Vecka 1: Divisionsfakta (invers av multiplikation)
- Vecka 2: Långdivision (2-siffrig ÷ 1-siffrig)
- Vecka 3: Division med rest
- Vecka 4: Divisionsproblemtexter
Bedömning: 30-uppgifters divisionsprov
```

**December: Bråk (introduktion)**
```
Läroplansmål: Förstå bråk som delar av helhet
Nyckelarbetsblad:
- Vecka 1: Identifiera bråk (skugga modeller)
- Vecka 2: Likvärdiga bråk (visuella modeller)
- Vecka 3: Jämföra bråk (samma nämnare)
Bedömning: Visuellt bråkprov (rita och jämför)
```

**Januari: Bråk (räknesätt)**
```
Läroplansmål: Addera/subtrahera bråk med lika nämnare
Nyckelarbetsblad:
- Vecka 1-2: Addera bråk (modeller + symboliskt)
- Vecka 3-4: Subtrahera bråk
Bedömning: 25-uppgifters bråkräkningsprov
```

**Februari: Decimaltal**
```
Läroplansmål: Decimalnotation, positionssystem till hundradelar
Nyckelarbetsblad:
- Vecka 1: Läsa/skriva decimaltal
- Vecka 2: Jämföra decimaltal
- Vecka 3: Addera decimaltal
- Vecka 4: Subtrahera decimaltal
Bedömning: Decimalräkningsprov
```

**Mars: Mätning & data**
```
Läroplansmål: Enhetsomvandling, linjediagram, dataanalys
Nyckelarbetsblad:
- Vecka 1: Enhetsomvandling (cm/m, dl/l)
- Vecka 2: Linjediagram (skapa från data)
- Vecka 3: Dataanalys (ordletare med mättermer)
Bedömning: Mätprov + datatolk​ning
```

**April: Geometri**
```
Läroplansmål: Vinklar, former, linjer, symmetri
Nyckelarbetsblad:
- Vecka 1: Vinkelmätning (gradskiveövning)
- Vecka 2: Triangel-/fyrkantstyper (klassificering)
- Vecka 3: Linjer (parallella, vinkelräta, skärande)
- Vecka 4: Symmetri (rita symmetrilinjer)
Bedömning: Geometriprov + konstruktionsuppgifter
```

**Maj: Repetition & fördjupning**
```
Fokus: Spiral-repetition av hela årets innehåll
Nyckelarbetsblad:
- Vecka 1: Talräkning repetition (blandad övning)
- Vecka 2: Bråk/decimal repetition
- Vecka 3: Mätning/geometri repetition
- Vecka 4: Nationella prov förberedelse (om tillämpligt)
Bedömning: Omfattande slutprov
```

**Generatortillämpning**: Skapa 180+ arbetsblad (hela året) på **3 timmar**
```
September-maj: 36 veckor
Arbetsblad per vecka: 5 (daglig övning)
Totalt: 180 arbetsblad

Manuell tid: 180 × 40 min = 7200 min (120 timmar)
Generatortid: 180 × 42 sek = 126 min (2,1 timmar)
Tidsbesparing: 117,9 timmar (nästan 3 hela arbetsveckor)
```

## Läroplansmålsprotokoll

**Problem**: Aktiviteter matchar inte läroplansmålen

**Lösning**: Explicit samstämmighetskontroll

### Samstämmighetschecklista

**För varje arbetsblad**:
```
☐ Läroplansmål adresserat: [skriv målkod]
☐ Syfte: Eleverna ska [specifikt, mätbart mål]
☐ Framgångskriterier: [hur vet vi om det behärskas]
☐ Koppling till bedömning: [vilken provfråga förbereder detta för?]
```

**Exempel**:
```
Arbetsblad: Jämföra bråk (20 uppgifter)

☐ Läroplansmål: Ma 4.2 (Jämföra bråk)
☐ Syfte: Eleverna ska jämföra två bråk med olika nämnare genom gemensam nämnare med 80% korrekthet
☐ Framgångskriterier: 16/20 uppgifter korrekta
☐ Koppling till bedömning: Slutprovets fråga 7-10 kräver bråkjämförelse

Samstämmighet verifierad ✓
```

## Bedömningsdriven undervisning

**Princip**: Övningar ska likna bedömningen

**Traditionell brist på samstämmighet**:
```
Övning: Öppna arbetsblad (skriv svar)
Bedömning: Flervalstest (kryssa i rutor)
Resultat: Formatchock på provdagen (kognitiv belastning på fel saker)
```

**Samstämmig övning**:
```
Övning: Samma format som bedömningen
Exempel: Om provet är flerval, öva med flerval
Resultat: Formatbekantskap (kognitiv belastning fokuserad på innehåll)
```

## Utveckla planeringsschema

**Planeringsschema**: Tidsplan för att täcka läroplansmålen

**Syfte**: Säkerställa att hela läroplanen hinns med före årets slut

### Skapa ett planeringsschema

**Steg 1: Räkna undervisningsdagar**
```
Skolår: 180 dagar
- Lov/helgdagar: 20 dagar
- Provdagar: 10 dagar
- Samlingar/utflykter: 10 dagar
- Buffertdagar (sjukdom, snö): 10 dagar
Faktiska undervisningsdagar: 130 dagar
```

**Steg 2: Fördela dagar till läroplansmål**
```
Årskurs 4 matematik: 30 centrala mål
130 dagar ÷ 30 mål = 4,3 dagar per mål (genomsnitt)

Justera efter komplexitet:
- Enkla mål (avrundning): 2 dagar
- Komplexa mål (bråkräkning): 8 dagar
```

**Steg 3: Skapa veckoschema**
```
Vecka 1 (15-19 aug): Positionssystem & avrundning (Mål Ma 4.1-2)
Vecka 2 (22-26 aug): Flersiffrig addition (Mål Ma 4.3)
Vecka 3 (29 aug-2 sep): Flersiffrig subtraktion (Mål Ma 4.3)
...

Resultat: Varje mål har dedikerad tid
```

**Generatornytta**: Skapa snabbt material för varje mål
```
Vecka 2 fokus: Flersiffrig addition
Generera: 5 additionsarbetsblad (3,5 minuter)
Skriv ut: 30 kopior vardera (hanteras av skrivare)
Klart: Vecka 2:s material färdigt
```

## Differentiering i bakåtdesign

**Utmaning**: Inte alla elever når måluppfyllelse i samma takt

**Lösning**: Nivåindelade mål

### Nivåindelad måluppfyllelse

**Nivå 1: Grundläggande behärskning** (80% av eleverna)
```
Mål: Jämföra bråk med lika nämnare
Bedömning: 3/4 vs 2/4 (vilket är större?)
```

**Nivå 2: God behärskning** (60% av eleverna)
```
Mål: Jämföra bråk med olika nämnare
Bedömning: 3/4 vs 2/3 (använd gemensam nämnare)
```

**Nivå 3: Avancerad behärskning** (20% av eleverna)
```
Mål: Jämföra tre eller fler bråk, förklara resonemanget
Bedömning: Ordna 2/3, 3/4, 5/6 från minst till störst, motivera
```

**Undervisning**: Tillhandahåll material för alla tre nivåerna
```
Måndag: Alla elever - Grundläggande jämförelse (lika nämnare)
Tisdag: Alla elever - Introduktion olika nämnare
Onsdag: Nivå 1 - Mer övning med lika nämnare (Station 1)
         Nivå 2 - Öva olika nämnare (Station 2)
         Nivå 3 - Avancerad jämförelse flera bråk (Station 3)
```

**Generatoranvändning**: Skapa 3 nivåer omedelbart
```
Nivå 1 arbetsblad: Jämför bråk (endast samma nämnare)
Nivå 2 arbetsblad: Jämför bråk (hitta MGN)
Nivå 3 arbetsblad: Ordna 4+ bråk (komplext)

Tid: 126 sekunder (42 sek vardera) för alla 3 nivåer
```

## Reflektions- & revisionsprotokoll

**Efter avslutad enhet**: Reflektera och förbättra

### Analys efter enhet

**Frågor att ställa**:
```
1. Nådde eleverna målen? (kontrollera bedömningsdata)
   - Om ja: Behåll enheten som den är
   - Om nej: Vad behöver undervisas om?

2. Vilka aktiviteter var mest effektiva?
   - Högst engagemang: [lista aktiviteter]
   - Bäst läranderesultat: [lista aktiviteter]
   - Behåll dessa, eliminera ineffektiva

3. Fungerade planeringen?
   - För stressat: Lägg till 1-2 dagar nästa år
   - För långsamt: Minska 1-2 dagar nästa år

4. Vad kommer jag att ändra nästa år?
   - [specifika revideringar]
```

**Exempelreflektion**:
```
Enhet: Bråk (4 veckor)
Bedömningsresultat: 85% måluppfyllelse (över 80% målet) ✓

Mest effektiva aktivitet: Pappervikning (Vecka 2, Måndag)
- 100% engagemang
- Eleverna sa "Aha!" ögonblick
- BEHÅLL DENNA

Minst effektiv: Ordletare bråk (Vecka 1, Onsdag)
- Eleverna uttråkade (för enkelt)
- ERSÄTT med bråk-Sudoku (mer utmanande)

Planering: Lagom (avslutades fredag som planerat)

Ändringar nästa år:
- Ersätt ordletare med Sudoku
- Lägg till ytterligare en formativ kontroll vecka 3 (några elever fortfarande förvirrade torsdag)
```

## Sammanfattning

Bakåtdesign förbättrar måluppfyllelse **15-25%** (Wiggins & McTighe, 2005) - planera med slutet i åtanke.

**3 stadier**:
1. Identifiera önskade resultat (vad ska eleverna kunna?)
2. Avgör acceptabelt bevis (hur ska de visa lärande? - designa bedömning FÖRST)
3. Planera läraktiviteter (vilka aktiviteter bygger mot bedömningen?)

**Läroöversikt**: Helårsperspektiv (månadsöversikt, 36 veckor, heltäckande innehåll)

**Läroplansmålsamstämmighet**: Varje arbetsblad explicit anpassat till läroplansmål

**Bedömningsdriven undervisning**: Övningsformat matchar bedömningsformat

**Planeringsschema**: 130 undervisningsdagar, fördela efter målkomplexitet

**Nivåindelade mål**: Grundläggande/God/Avancerad nivå (differentiering inbyggd)

**Reflektionsprotokoll**: Analys efter enhet, revidera för nästa år

**Generatortillämpning**: Skapa 180 arbetsblad på 2,1 timmar (vs 120 timmar manuellt)

**Forskning**: Bakåtdesign = 15-25% måluppfyllelse-ökning (Wiggins & McTighe, 2005)

**Prissättning**: Core-paketet 1490 kr/år (sparar 107,9 timmar planering)
