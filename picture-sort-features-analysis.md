# Picture Sort App - Features Analysis

**Source:** REFERENCE APPS/picture sort.html
**Pricing Tier:** Full Access ($240/year or $25/month)

---

## App Overview

The Picture Sort Worksheet Generator creates sorting activities where children categorize images into two categories (left and right). The app generates both worksheets and answer keys.

---

## Accordion Sections (UI Controls)

### 1. Sorting Categories
- **Left Category Theme** - Select theme for left sorting category
- **Right Category Theme** - Select theme for right sorting category
- Automatic generation based on selected themes

### 2. Image Library
- **Theme Selection** - Browse images by theme
- **Search Function** - Search images by keyword (e.g., apple, car)
- **Available Images** - Click to add from 3000+ image library
- **Selected Images Preview** - Shows selected images with category assignment
- **Maximum 12 images** per worksheet
- **Category Assignment** - Assign each image to left or right category

### 3. Upload Custom Images
- **Multi-file upload** support
- **Accepts** JPEG, PNG, GIF formats
- **Session-based storage** - Uploaded images available during session
- **Combine** with library images

### 4. Page Setup
**Page Sizes:**
- Letter Portrait (8.5×11")
- Letter Landscape (11×8.5")
- A4 Portrait (210×297mm)
- A4 Landscape (297×210mm)
- Square (1200x1200)
- Custom dimensions

**Page Color:**
- Color picker for page background

**Background Options:**
- Background Theme selection
- Background Browser
- Background Opacity slider (0-100%)

**Border Options:**
- Border Theme selection
- Border Browser
- Border Opacity slider (0-100%)

**Additional:**
- Include Name/Date Fields checkbox

### 5. Text Tools
**Add New Text:**
- Content input field
- Add Text button

**Text Properties (when selected):**
- Color picker
- Font size (min 8)
- Font family options:
  - Lexend Deca
  - Baloo 2
  - Nunito
  - Quicksand
  - Fredoka
  - Arial
  - Verdana
- Outline color
- Outline width (0-10)
