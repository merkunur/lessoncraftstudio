## Inledning: När vikarien behövs akut

**Verkligheten för lärare**: Behöver vikarie med kort varsel (sjukdom, akutärende, fortbildning)

**Problemet**: Skapa vikarieplaner klockan 05:30 när man är sjuk
```
06:00: Vaknar med feber
06:15: Ringer efter vikarie
06:30: Försöker skriva lektionsplaneringar (känner sig förfärlig)
07:00: Ger upp, lämnar allmän lapp "låt dem läsa"
Resultat: Bortkastad skoldag (vikarien vet inte vad som ska läras ut)
```

**Lösningen**: Färdiga akutplaner (skapade när du är frisk)

**Forskning** (Weems & Rogers, 2007): Välförberedda vikarieplaner bibehåller **90% av ordinarie inlärning** jämfört med **40%** vid bristfällig planering

**Bästa praxis**: Ha 3-5 dagars kompletta vikarieplaner redo vid alla tillfällen

## Akutpärmen för vikarie

**Fysisk pärm i klassrummet** (vikarien kan plocka fram den direkt)

### Pärmens innehåll

**Avdelning 1: Klassrumsinformation**
```
- Sittplan (med elevnamn markerade)
- Dagligt schema (tider för varje ämne)
- Beteendesystem (klassregler, belöningar)
- Elever med särskilda behov (anpassningslista)
- Utrymningsprocedurer (brandövning, inlåsning)
- Hjälpsamma elever (pålitliga elever som vikarien kan fråga)
```

**Avdelning 2: Dag 1 komplett lektion**
```
Förmiddag:
- 08:30-09:00: Morgonrutin + närvaro (vikarien gör självständigt)
- 09:00-09:30: Matematikarbetsblad (30 uppgifter, facit medföljande)
- 09:30-10:00: Ordletare (ordförråd från aktuell enhet)

Middag:
- 10:00-10:30: Läsning (självständig läsning + läsförståelseblad)
- 10:30-11:00: Rast
- 11:00-11:30: Färgläggningssida (lugn aktivitet)

Eftermiddag:
- 11:30-12:00: Lunch
- 12:00-12:30: NO ordletare (aktuella NO-termer)
- 12:30-13:00: Korsord (akademiskt ordförråd)
- 13:00-13:30: Matematikpussel (kritiskt tänkande, engagerande)
- 13:30-14:00: Fritt val: Rita linjer, färglägg, eller självständig läsning

Avslutning: 14:00

Lapp till vikarie: "Alla arbetsblad har facit i Avdelning 5. Eleverna ska kontrollera sitt eget arbete när de är klara."
```

**Avdelning 3-5: Dag 2, 3, 4** (liknande struktur, olika arbetsblad)

**Avdelning 6: Facit** (alla arbetsblad)

## Självförklarande arbetsblad

**Nyckelprincip**: Vikarien behöver inte undervisa i nytt innehåll

**Arbetsbladsegenskaper för vikarieplaner**:

### Tydliga instruktioner på arbetsbladet

**Exempel på matematikarbetsblads rubrik**:
```
Namn: _________________ Datum: _______

INSTRUKTIONER:
1. Lös varje additionsuppgift
2. Visa dina uträkningar
3. Kontrollera dina svar med facit (på lärarens bord)
4. Om du får 8 eller fler fel, be vikarien om hjälp
5. När du är klar, lägg i inlämningskorgen

Förväntad tid: 30 minuter
```

**Vikarievänligt**: Instruktioner PÅ arbetsbladet (vikarien behöver inte förklara)

---

### Bekanta aktivitetstyper

**Använd arbetsblad eleverna redan känner till**:
```
✓ Matematikräkning (tränad varje vecka - eleverna känner formatet)
✓ Ordletare (gjorda regelbundet - självförklarande)
✓ Korsord (bekant pusseltyp)
✓ Färgläggning (alla kan färglägga)

✗ Nya spelregler (vikarien kan inte lära ut okänd aktivitet)
✗ Komplexa projekt (kräver lärarens stöttning)
✗ Grupparbete (beteendeutmaning för vikarie)
```

## Beteendehantering för vikarier

**Utmaning**: Elever testar gränser med vikarie

**Lösning**: Tydliga förväntningar + engagerande arbete

### Förskriven lapp till eleverna

**Inkludera i vikarieplan**:
```
Kära klass,

Jag är borta idag, men jag förväntar mig att ni visar vår vikarie samma respekt som ni visar mig. Det här förväntar jag mig:

✓ Följ alla klassregler
✓ Gör klart alla tilldelade uppgifter
✓ Hjälp vikarien om de har frågor
✓ Var snäll och respektfull

Jag kommer att fråga vikarien om rapport när jag kommer tillbaka. Elever som uppfyller förväntningarna får [belöning]. De som inte gör det får [konsekvens].

Ha en bra dag!
[Lärarens namn]
```

**Ansvarighet**: Eleverna vet att läraren kommer att följa upp

---

### Engagerande arbete (förhindrar tristess)

**Tristess = beteendeproblem**

**Strategi**: Blanda lätt + utmanande + roligt
```
Förmiddag: Standardmatematik (bekant, bygger förtroende)
Förmiddag: Ordletare (spelliknande, engagerande)
Före lunch: Färgläggning (lugnande, lågstress)
Efter lunch: Korsord (måttlig utmaning)
Slut på dagen: Matematikpussel ELLER fritt val (motiverande avslutning)

Resultat: Eleverna engagerade (mindre tid för dåligt beteende)
```

## Innehållsanpassade vikarieplaner

**Idealiskt**: Vikarieplaner repeterar aktuellt innehåll (inte bara sysselsättning)

### Skapa anpassade planer varje månad

**September (början av året)**:
```
Generera vikarieplaner baserade på septemberinnehåll:
- Matematik: Addition/subtraktion repetition (färdigheter undervisade i sept)
- Ordförråd: Septemberstavord (i ordletare/korsord)
- NO: September NO-termer (i ordletare)

Resultat: Även med vikarie övar eleverna relevant innehåll
```

**Oktober**:
```
Uppdatera vikarieplaner:
- Matematik: Multiplikation (oktoberfokus)
- Ordförråd: Oktoberord
- NO: Oktoberenhetstermer

Byt ut septemberarbetsbladen mot oktoberversioner
Tid: 10 minuter att generera 5 nya arbetsblad
```

**Löpande**: Uppdatera vikarieplaner månadsvis (alltid aktuellt innehåll)

## Differentiering i vikarieplaner

**Utmaning**: Vikarien kan inte differentiera som ordinarie lärare

**Lösning**: Fördifferentierat material

### Nivåanpassade arbetsbladslådor

**Förbered i förväg**:
```
Låda 1 (Blå): Modifierade arbetsblad
- 10 uppgifter istället för 20
- Bildstöd
- För: Elever med svårigheter

Låda 2 (Grön): Standardarbetsblad
- 20 uppgifter
- Årskursnivå
- För: Elever på nivån

Låda 3 (Röd): Utmaningsarbetsblad
- 30 uppgifter
- Komplexa tillämpningar
- För: Avancerade elever

Lapp till vikarie: "Eleverna vet vilken färglåda de ska ta från. De kommer själva välja lämplig nivå."

Resultat: Differentieringen sker automatiskt (inget beslutsfattande från vikarie krävs)
```

## Planering för långvarig frånvaro

**Scenario**: Lärare borta 2+ veckor (operation, föräldraledighet, familjekris)

### Utökat vikarieplan-system

**Vecka 1-2 material**:
```
Generera 10 hela dagars lektioner (50+ arbetsblad)
Organisation:
- Måndag Vecka 1 mapp (allt material)
- Tisdag Vecka 1 mapp
- ...
- Fredag Vecka 2 mapp

Långtidsvikarie: Öppnar måndagsmappen, hittar allt som behövs
Tid att skapa: 2-3 timmar (en helg för att förbereda 2 veckor)

Fördel: Läraren kan fokusera på återhämtning (inte oroa sig för klassrummet)
```

## Träna elever för vikariedagar

**Förberedelse**: Lär eleverna hur de ska bete sig med vikarier

**Septemberträning**:
```
Lärare: "Ibland är jag borta och vi har vikarie. Det här förväntar jag mig:"

Rollspel:
- Elevvolontärer som "vikarie"
- Klassen övar:
  * Följa instruktioner från vikarie
  * Hämta arbetsblad från rätt låda
  * Be vikarien om hjälp på lämpligt sätt
  * Städa upp vid dagens slut

Övningskörning: "Idag låtsas vi att det är vikariedag"
Elever: Följ akutvikarieplanerna som övning
Resultat: När riktig vikarie kommer känner eleverna till rutinen
```

## Sammanfattning

**Akutpärm**: 3-5 kompletta dagar (klassruminfo + dagliga lektioner + facit + hjälpsamma elever)

**Självförklarande arbetsblad**: Tydliga instruktioner PÅ arbetsbladet (vikarien behöver inte undervisa)

**Innehållsanpassat**: Uppdatera månadsvis (matchar alltid aktuell läroplan, inte sysselsättning)

**Generatoreffektivitet**: 3 dagar skapade på 45 minuter (15 generering + 30 utskrift)

**Digital backup**: Google Classroom akutmapp (noll varsel frånvaro täckt)

**Långvarig frånvaro**: 10 dagar på 2-3 timmar (utökad ledighet täckt)

**Vikarieåterkoppling**: Formulär för rapportering (beteende, genomförande, utmaningar)

**Elevträning**: Septemberrollspel (öva vikariedagsrutin)

**Forskning**: Kvalitetsplaner = 90% inlärning vs 40% utan (Weems & Rogers, 2007)
